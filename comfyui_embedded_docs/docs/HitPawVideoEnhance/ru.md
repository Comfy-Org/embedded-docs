Вот перевод документации на русский язык:

Узел HitPaw Video Enhance использует внешний API для улучшения качества видео. Он повышает разрешение видео с низким разрешением, удаляет визуальные артефакты и снижает уровень шума. Стоимость обработки рассчитывается за секунду входного видео.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `model` | DYNAMIC COMBO | Да | Доступно несколько вариантов | Модель ИИ, используемая для улучшения видео. Выбор модели открывает вложенный параметр `resolution`. Доступные модели и поддерживаемые ими разрешения различаются. |
| `model.resolution` | COMBO | Да | `"original"`<br>`"720p"`<br>`"1080p"`<br>`"2k/qhd"`<br>`"4k/uhd"`<br>`"8k"` | Целевое разрешение для улучшенного видео. Некоторые опции могут быть недоступны в зависимости от выбранной `model`. |
| `video` | VIDEO | Да | Н/Д | Входной видеофайл для улучшения. |

**Ограничения:**

* Длительность входного `video` должна составлять от 0,5 секунды до 60 минут (3600 секунд).
* Выбранное `resolution` должно быть больше размеров входного видео. Если видео квадратное, выбранное разрешение должно быть больше его ширины/высоты. Для неквадратных видео выбранное разрешение должно быть больше меньшего измерения видео. Если целевое разрешение меньше, будет вызвана ошибка. Выберите `"original"`, чтобы сохранить разрешение входного видео.

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `video` | VIDEO | Файл улучшенного видео. |
