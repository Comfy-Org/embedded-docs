Этот узел использует API Tencent Hunyuan3D для редактирования текстур 3D-модели. Вы предоставляете 3D-модель и текстовое описание желаемых изменений, а узел возвращает новую версию модели с перерисованными текстурами в соответствии с вашим запросом.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `model_3d` | FILE3D | Да | FBX, Любой | 3D-модель в формате FBX. Модель должна содержать менее 100 000 полигонов. |
| `prompt` | STRING | Да | | Описывает редактирование текстуры. Поддерживается до 1024 символов UTF-8. |
| `seed` | INT | Нет | от 0 до 2147483647 | Сид управляет тем, следует ли перезапускать узел; результаты недетерминированы независимо от сида. (по умолчанию: 0) |

**Примечание:** Входной параметр `model_3d` должен быть файлом в формате FBX. Другие форматы 3D-файлов не поддерживаются этим узлом.

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `GLB` | FILE3D | Обработанная 3D-модель в формате GLB. |
| `OBJ` | FILE3D | Обработанная 3D-модель в формате OBJ. |
| `texture_image` | IMAGE | Вновь сгенерированное изображение текстуры для 3D-модели. |
