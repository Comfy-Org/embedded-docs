این گره از API Tencent Hunyuan3D برای ویرایش بافت‌های یک مدل سه‌بعدی استفاده می‌کند. شما یک مدل سه‌بعدی و توضیح متنی از تغییرات مورد نظر خود ارائه می‌دهید و گره نسخه جدیدی از مدل را با بافت‌های بازطراحی‌شده بر اساس دستور شما بازمی‌گرداند.

## ورودی‌ها

| پارامتر | نوع داده | اجباری | محدوده | توضیحات |
|---------|----------|--------|--------|---------|
| `model_3d` | FILE3D | بله | FBX, هر نوع | مدل سه‌بعدی با فرمت FBX. مدل باید کمتر از ۱۰۰۰۰۰ وجه داشته باشد. |
| `prompt` | STRING | بله | | ویرایش بافت را توصیف می‌کند. حداکثر ۱۰۲۴ کاراکتر UTF-8 را پشتیبانی می‌کند. |
| `seed` | INT | خیر | ۰ تا ۲۱۴۷۴۸۳۶۴۷ | Seed مشخص می‌کند که آیا گره باید دوباره اجرا شود؛ نتایج بدون توجه به seed غیرقطعی هستند. (پیش‌فرض: ۰) |

**توجه:** ورودی `model_3d` باید یک فایل با فرمت FBX باشد. سایر فرمت‌های فایل سه‌بعدی توسط این گره پشتیبانی نمی‌شوند.

## خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-----------|----------|---------|
| `GLB` | FILE3D | مدل سه‌بعدی پردازش‌شده با فرمت GLB. |
| `OBJ` | FILE3D | مدل سه‌بعدی پردازش‌شده با فرمت OBJ. |
| `texture_image` | IMAGE | تصویر بافت تازه تولیدشده برای مدل سه‌بعدی. |
