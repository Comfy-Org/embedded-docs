Узел ModelMergeSD3_2B позволяет объединять две модели Stable Diffusion 3 2B путем смешивания их компонентов с регулируемыми весами. Он обеспечивает индивидуальный контроль над слоями внедрения и блоками трансформера, что позволяет создавать тонко настроенные комбинации моделей для специализированных задач генерации.
