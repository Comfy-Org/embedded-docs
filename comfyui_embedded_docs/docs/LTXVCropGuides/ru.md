Вот перевод документации на русский язык:

Узел LTXVCropGuides обрабатывает входные данные conditioning и latent для генерации видео, удаляя информацию о ключевых кадрах и корректируя размеры скрытого пространства. Он обрезает скрытое изображение и маску шума, исключая секции ключевых кадров, а также очищает индексы ключевых кадров как из положительных, так и из отрицательных входных данных conditioning. Это подготавливает данные для рабочих процессов генерации видео, не требующих управления ключевыми кадрами.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `положительный` | CONDITIONING | Да | - | Положительный вход conditioning, содержащий информацию для управления генерацией |
| `отрицательный` | CONDITIONING | Да | - | Отрицательный вход conditioning, содержащий информацию о том, чего следует избегать при генерации |
| `скрытое пространство` | LATENT | Да | - | Скрытое представление, содержащее образцы изображений и данные маски шума |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `положительный` | CONDITIONING | Обработанный положительный conditioning с очищенными индексами ключевых кадров и записями направляющего внимания |
| `отрицательный` | CONDITIONING | Обработанный отрицательный conditioning с очищенными индексами ключевых кадров и записями направляющего внимания |
| `скрытое пространство` | LATENT | Обрезанное скрытое представление с скорректированными образцами и маской шума, из которого удалены секции ключевых кадров |
