Узел LTXVCropGuides обрабатывает кондиционирующие и латентные входные данные для генерации видео, удаляя информацию о ключевых кадрах и корректируя размеры латентного представления. Он обрезает латентное изображение и маску шума, исключая секции ключевых кадров, одновременно очищая индексы ключевых кадров из позитивных и негативных кондиционирующих входных данных. Это подготавливает данные для рабочих процессов генерации видео, которые не требуют управления по ключевым кадрам.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `positive` | CONDITIONING | Да | - | Позитивный кондиционирующий вход, содержащий информацию для управления генерацией |
| `negative` | CONDITIONING | Да | - | Негативный кондиционирующий вход, содержащий информацию о том, чего следует избегать при генерации |
| `latent` | LATENT | Да | - | Латентное представление, содержащее сэмплы изображений и данные маски шума |

## Выходные параметры

| Выходной параметр | Тип данных | Описание |
|-------------|-----------|-------------|
| `positive` | CONDITIONING | Обработанный позитивный кондиционирующий вход с очищенными индексами ключевых кадров |
| `negative` | CONDITIONING | Обработанный негативный кондиционирующий вход с очищенными индексами ключевых кадров |
| `latent` | LATENT | Обрезанное латентное представление с корректированными сэмплами и маской шума |
