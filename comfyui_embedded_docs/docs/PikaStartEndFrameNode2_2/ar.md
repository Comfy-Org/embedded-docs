# ترجمة وثيقة عقدة PikaStartEndFrameNode v2.2

تقوم عقدة PikaFrames v2.2 بإنشاء فيديو من خلال دمج إطارك الأول والأخير. يمكنك تحميل صورتين لتحديد نقطتي البداية والنهاية، ويقوم الذكاء الاصطناعي بإنشاء انتقال سلس بينهما لإنتاج فيديو كامل.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|---------|--------------|--------|--------|-------|
| `image_start` | IMAGE | نعم | - | الصورة الأولى المراد دمجها. |
| `image_end` | IMAGE | نعم | - | الصورة الأخيرة المراد دمجها. |
| `prompt_text` | STRING | نعم | - | نص موجه يصف محتوى الفيديو المطلوب. |
| `negative_prompt` | STRING | نعم | - | نص يصف ما يجب تجنبه في الفيديو. |
| `seed` | INT | نعم | - | قيمة البذرة العشوائية لضمان اتساق التوليد. |
| `resolution` | STRING | نعم | - | دقة فيديو الإخراج. |
| `duration` | INT | نعم | - | مدة الفيديو المُنشأ. |

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|------------|--------------|-------|
| `output` | VIDEO | الفيديو المُنشأ الذي يدمج إطاري البداية والنهاية مع انتقالات الذكاء الاصطناعي. |
