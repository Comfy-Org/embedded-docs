Вот перевод документации на русский язык:

Узел PikaFrames v2.2 создает видео путем объединения первого и последнего кадров. Вы загружаете два изображения, чтобы задать начальную и конечную точки, а ИИ создает плавный переход между ними, формируя готовое видео.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `image_start` | IMAGE | Да | - | Первое изображение для объединения. |
| `image_end` | IMAGE | Да | - | Последнее изображение для объединения. |
| `prompt_text` | STRING | Да | - | Текстовый запрос, описывающий желаемое содержимое видео. |
| `negative_prompt` | STRING | Да | - | Текст, описывающий, чего следует избегать в видео. |
| `seed` | INT | Да | - | Значение случайного зерна для воспроизводимости генерации. |
| `resolution` | STRING | Да | - | Разрешение выходного видео. |
| `duration` | INT | Да | - | Длительность создаваемого видео. |

## Выходные данные

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `output` | VIDEO | Сгенерированное видео, объединяющее начальный и конечный кадры с переходами, созданными ИИ. |
