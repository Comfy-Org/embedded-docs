### نمای کلی

گره PikaFrames نسخه 2.2 با ترکیب اولین و آخرین فریم، یک ویدیو تولید می‌کند. شما دو تصویر را برای تعیین نقطه شروع و پایان آپلود می‌کنید و هوش مصنوعی یک انتقال روان بین آن‌ها ایجاد کرده و ویدیوی کاملی تولید می‌کند.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `image_start` | IMAGE | بله | - | اولین تصویر برای ترکیب. |
| `image_end` | IMAGE | بله | - | آخرین تصویر برای ترکیب. |
| `prompt_text` | STRING | بله | - | متن راهنما توصیف‌کننده محتوای ویدیوی مورد نظر. |
| `negative_prompt` | STRING | بله | - | متنی که مشخص می‌کند چه مواردی در ویدیو اجتناب شود. |
| `seed` | INT | بله | - | مقدار seed تصادفی برای یکنواختی تولید. |
| `resolution` | STRING | بله | - | رزولوشن ویدیوی خروجی. |
| `duration` | INT | بله | - | مدت زمان ویدیوی تولید شده. |

## خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-------------|-----------|-------------|
| `output` | VIDEO | ویدیوی تولید شده که فریم شروع و پایان را با انتقال‌های هوش مصنوعی ترکیب می‌کند. |
