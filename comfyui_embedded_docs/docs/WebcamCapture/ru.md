Вот перевод документации на русский язык:

Узел WebcamCapture захватывает изображения с веб-камеры и преобразует их в формат, который можно использовать в рабочих процессах ComfyUI. Он наследуется от узла LoadImage и предоставляет опции для управления размерами захвата и временем. При включении узел может захватывать новые изображения каждый раз при обработке очереди рабочего процесса.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `изображение` | WEBCAM | Да | - | Источник входного сигнала веб-камеры для захвата изображений |
| `ширина` | INT | Да | от 0 до MAX_RESOLUTION | Желаемая ширина захваченного изображения (по умолчанию: 0, используется собственное разрешение веб-камеры) |
| `высота` | INT | Да | от 0 до MAX_RESOLUTION | Желаемая высота захваченного изображения (по умолчанию: 0, используется собственное разрешение веб-камеры) |
| `захват_в_очереди` | BOOLEAN | Да | - | При включении захватывает новое изображение каждый раз при обработке очереди рабочего процесса (по умолчанию: True) |

**Примечание:** Если оба параметра `width` и `height` установлены в 0, узел использует собственное разрешение веб-камеры. Установка любого из размеров в ненулевое значение приведет к соответствующему масштабированию захваченного изображения.

## Выходы

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `IMAGE` | IMAGE | Захваченное изображение с веб-камеры, преобразованное в формат изображения ComfyUI |
