گره WebcamCapture تصاویر را از یک دستگاه وب‌کم دریافت کرده و به قالبی تبدیل می‌کند که در گردش‌های کاری ComfyUI قابل استفاده باشد. این گره از گره LoadImage ارث‌بری می‌کند و گزینه‌هایی برای کنترل ابعاد و زمان‌بندی دریافت تصویر ارائه می‌دهد. در صورت فعال بودن، این گره می‌تواند هر بار که صف گردش کار پردازش می‌شود، یک تصویر جدید دریافت کند.

## ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|---------|----------|-------|--------|---------|
| `تصویر` | WEBCAM | بله | - | منبع ورودی وب‌کم برای دریافت تصویر |
| `عرض` | INT | بله | 0 تا MAX_RESOLUTION | عرض مورد نظر برای تصویر دریافت‌شده (پیش‌فرض: 0، استفاده از وضوح بومی وب‌کم) |
| `ارتفاع` | INT | بله | 0 تا MAX_RESOLUTION | ارتفاع مورد نظر برای تصویر دریافت‌شده (پیش‌فرض: 0، استفاده از وضوح بومی وب‌کم) |
| `دریافت هنگام صف` | BOOLEAN | بله | - | در صورت فعال بودن، هر بار که صف گردش کار پردازش می‌شود، یک تصویر جدید دریافت می‌کند (پیش‌فرض: True) |

**نکته:** هنگامی که هر دو `width` و `height` روی 0 تنظیم شوند، گره از وضوح بومی وب‌کم استفاده می‌کند. تنظیم هر یک از ابعاد به مقداری غیر از صفر، تصویر دریافت‌شده را متناسب با آن تغییر اندازه می‌دهد.
