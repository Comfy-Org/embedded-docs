Узел RepeatImageBatch предназначен для повторения заданного изображения указанное количество раз, создавая пакет идентичных изображений. Эта функция полезна для операций, требующих множества экземпляров одного и того же изображения, таких как пакетная обработка или аугментация данных.

## Входные параметры

| Поле     | Тип данных | Описание                                                                 |
|----------|------------|--------------------------------------------------------------------------|
| `изображение`  | `IMAGE`    | Параметр `изображение` представляет изображение, которое будет повторяться. Он важен для определения содержимого, которое будет дублироваться в пакете. |
| `количество` | `INT`      | Параметр `количество` указывает количество раз, которое входное изображение должно быть повторено. Он напрямую влияет на размер выходного пакета, обеспечивая гибкое создание пакетов. |

## Выходные параметры

| Поле    | Тип данных | Описание                                                               |
|---------|------------|------------------------------------------------------------------------|
| `изображение` | `IMAGE`    | Выходные данные представляют собой пакет изображений, каждое из которых идентично входному изображению, повторенному в соответствии с указанным значением `количество`. |
