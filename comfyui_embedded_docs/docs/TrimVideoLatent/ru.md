Вот перевод документации на русский язык:

Узел TrimVideoLatent удаляет кадры из начала латентного представления видео. Он принимает латентный образец видео и обрезает указанное количество кадров с начала, возвращая оставшуюся часть видео. Это позволяет сократить видеопоследовательности путем удаления начальных кадров.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `samples` | LATENT | Да | - | Входное латентное представление видео, содержащее кадры для обрезки |
| `trim_amount` | INT | Да | от 0 до 99999 | Количество кадров, удаляемых с начала видео (по умолчанию: 0) |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `output` | LATENT | Обрезанное латентное представление видео с удаленным указанным количеством кадров с начала |
