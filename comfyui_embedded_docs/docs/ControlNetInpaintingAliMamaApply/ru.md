Вот перевод документации на русский язык:

Узел ControlNetInpaintingAliMamaApply применяет conditioning ControlNet для задач инпейнтинга, объединяя позитивный и негативный conditioning с управляющим изображением и маской. Он обрабатывает входное изображение и маску для создания модифицированного conditioning, который направляет процесс генерации, позволяя точно контролировать, какие области изображения будут подвергнуты инпейнтингу. Узел поддерживает регулировку силы воздействия и временные параметры для точной настройки влияния ControlNet на разных этапах процесса генерации.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `positive` | CONDITIONING | Да | - | Позитивный conditioning, направляющий генерацию к желаемому содержимому |
| `negative` | CONDITIONING | Да | - | Негативный conditioning, направляющий генерацию от нежелательного содержимого |
| `control_net` | CONTROL_NET | Да | - | Модель ControlNet, обеспечивающая дополнительный контроль над генерацией |
| `vae` | VAE | Да | - | VAE (вариационный автоэнкодер), используемый для кодирования и декодирования изображений |
| `image` | IMAGE | Да | - | Входное изображение, служащее управляющим сигналом для ControlNet |
| `mask` | MASK | Да | - | Маска, определяющая области изображения, подлежащие инпейнтингу |
| `strength` | FLOAT | Да | от 0.0 до 10.0 | Сила воздействия ControlNet (по умолчанию: 1.0) |
| `start_percent` | FLOAT | Да | от 0.0 до 1.0 | Начальная точка (в процентах) начала влияния ControlNet во время генерации (по умолчанию: 0.0) |
| `end_percent` | FLOAT | Да | от 0.0 до 1.0 | Конечная точка (в процентах) прекращения влияния ControlNet во время генерации (по умолчанию: 1.0) |

**Примечание:** Если в ControlNet включена опция `concat_mask`, маска инвертируется и применяется к изображению перед обработкой, а сама маска включается в дополнительные конкатенированные данные, отправляемые в ControlNet.

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `positive` | CONDITIONING | Модифицированный позитивный conditioning с применением ControlNet для инпейнтинга |
| `negative` | CONDITIONING | Модифицированный негативный conditioning с применением ControlNet для инпейнтинга |
