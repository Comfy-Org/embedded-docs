Вот перевод документации на русский язык:

Узел SamplingPercentToSigma преобразует значение процента дискретизации в соответствующее значение сигмы, используя параметры дискретизации модели. Он принимает процентное значение от 0.0 до 1.0 и сопоставляет его с соответствующим значением сигмы в шумовом расписании модели, с возможностью возврата либо рассчитанной сигмы, либо фактических максимальных/минимальных значений сигмы на границах.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `model` | MODEL | Да | - | Модель, содержащая параметры дискретизации, используемые для преобразования |
| `sampling_percent` | FLOAT | Да | от 0.0 до 1.0 | Процент дискретизации для преобразования в сигму (по умолчанию: 0.0) |
| `return_actual_sigma` | BOOLEAN | Да | - | Возвращает фактическое значение сигмы вместо значения, используемого для проверки интервалов. Это влияет только на результаты при значениях 0.0 и 1.0. (по умолчанию: False) |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `sigma_value` | FLOAT | Преобразованное значение сигмы, соответствующее входному проценту дискретизации |
