   - القيم في عمود Range: الأرقام، "auto"، أسماء الخيارات
   - الكود، مسارات الملفات

## نظرة عامة

تقوم هذه العقدة بإنشاء نموذج ثلاثي الأبعاد من صورة إلى خمس صور مرجعية باستخدام واجهة برمجة تطبيقات Rodin Gen-2.5. يمكنك الاختيار بين أوضاع الجودة السريعة أو العادية أو العالية جدًا لتحقيق التوازن بين سرعة الإنشاء والتكلفة.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|-----------|-----------|----------|-------|-------------|
| `images` | IMAGE | نعم | 1 إلى 5 صور | صورة إلى خمس صور إدخال. تُستخدم الصورة الأولى للمواد عند توفير صور متعددة. |
| `mode` | COMBO | نعم | `"Fast"`<br>`"Regular"`<br>`"Extreme-High"` | وضع جودة الإنشاء. تنتج أوضاع الجودة الأعلى نتائج أفضل ولكن بتكلفة أعلى. |
| `material` | COMBO | نعم | `"PBR"`<br>`"Matte"` | نوع المادة للنموذج ثلاثي الأبعاد المُنشأ. |
| `geometry_file_format` | COMBO | نعم | `"glb"`<br>`"obj"`<br>`"stl"`<br>`"usdz"` | تنسيق ملف الإخراج لهندسة النموذج ثلاثي الأبعاد. |
| `texture_mode` | COMBO | نعم | `"Original"`<br>`"Clean"`<br>`"Style"` | وضع إنشاء النسيج. "Original" يحافظ على أنسجة الإدخال، و"Clean" يزيلها، و"Style" يطبق نسيجًا منمقًا. |
| `seed` | INT | نعم | 0 إلى 2147483647 | بذرة عشوائية للحصول على نتائج قابلة للتكرار. استخدم نفس البذرة للحصول على نفس المخرجات. |
| `TAPose` | BOOLEAN | نعم | True / False | ما إذا كان سيتم تطبيق وضعية T-pose على النموذج المُنشأ. |
| `hd_texture` | BOOLEAN | نعم | True / False | ما إذا كان سيتم إنشاء خريطة نسيج عالية الدقة. |
| `texture_delight` | BOOLEAN | نعم | True / False | ما إذا كان سيتم إزالة الإضاءة من صور الإدخال قبل إنشاء النسيج. |
| `use_original_alpha` | BOOLEAN | نعم | True / False | ما إذا كان سيتم استخدام قناة ألفا الأصلية من صور الإدخال. |
| `addon_highpack` | BOOLEAN | نعم | True / False | ما إذا كان سيتم إنشاء نسخة عالية المضلعات من النموذج بالإضافة إلى النسخة القياسية. |
| `bbox_width` | INT | نعم | 1 إلى 1000 | عرض الصندوق المحيط للنموذج المُنشأ بالسنتيمتر. |
| `bbox_height` | INT | نعم | 1 إلى 1000 | ارتفاع الصندوق المحيط للنموذج المُنشأ بالسنتيمتر. |
| `bbox_length` | INT | نعم | 1 إلى 1000 | طول الصندوق المحيط للنموذج المُنشأ بالسنتيمتر. |
| `height_cm` | INT | نعم | 1 إلى 300 | ارتفاع النموذج المُنشأ بالسنتيمتر. |

**ملاحظة حول عدد الصور:** تقبل العقدة ما بين صورة واحدة و5 صور. إذا قمت بتوفير مجموعة من الصور (على سبيل المثال، مجموعة من 4 صور)، فسيتم التعامل مع كل صورة في المجموعة كصورة إدخال منفصلة. سيؤدي توفير أكثر من 5 صور إلى حدوث خطأ.
