## نمای کلی

این گره با استفاده از API Rodin Gen-2.5، یک مدل سه‌بعدی از یک تا پنج تصویر مرجع تولید می‌کند. می‌توانید بین حالت‌های کیفیت سریع، معمولی یا فوق‌بالا یکی را انتخاب کنید تا سرعت تولید و هزینه را متعادل کنید.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `images` | IMAGE | بله | ۱ تا ۵ تصویر | یک تا پنج تصویر ورودی. اولین تصویر برای مواد (متریال) زمانی که چندین تصویر ارائه می‌شود استفاده می‌گردد. |
| `mode` | COMBO | بله | `"Fast"`<br>`"Regular"`<br>`"Extreme-High"` | حالت کیفیت تولید. حالت‌های با کیفیت بالاتر نتایج بهتری تولید می‌کنند اما هزینه بیشتری دارند. |
| `material` | COMBO | بله | `"PBR"`<br>`"Matte"` | نوع ماده (متریال) برای مدل سه‌بعدی تولیدشده. |
| `geometry_file_format` | COMBO | بله | `"glb"`<br>`"obj"`<br>`"stl"`<br>`"usdz"` | فرمت فایل خروجی برای هندسه مدل سه‌بعدی. |
| `texture_mode` | COMBO | بله | `"Original"`<br>`"Clean"`<br>`"Style"` | حالت تولید بافت. "Original" بافت‌های ورودی را حفظ می‌کند، "Clean" آن‌ها را حذف می‌کند و "Style" یک بافت سبک‌دار اعمال می‌نماید. |
| `seed` | INT | بله | ۰ تا ۲۱۴۷۴۸۳۶۴۷ | یک دانه تصادفی برای نتایج قابل تکرار. برای دریافت خروجی یکسان از دانه یکسان استفاده کنید. |
| `TAPose` | BOOLEAN | بله | True / False | آیا حالت T-Pose به مدل تولیدشده اعمال شود یا خیر. |
| `hd_texture` | BOOLEAN | بله | True / False | آیا نقشه بافت با وضوح بالا تولید شود یا خیر. |
| `texture_delight` | BOOLEAN | بله | True / False | آیا نورپردازی از تصاویر ورودی قبل از تولید بافت حذف شود یا خیر. |
| `use_original_alpha` | BOOLEAN | بله | True / False | آیا از کانال آلفای اصلی تصاویر ورودی استفاده شود یا خیر. |
| `addon_highpack` | BOOLEAN | بله | True / False | آیا نسخه پرتعداد چندضلعی (high-polygon) مدل علاوه بر نسخه استاندارد تولید شود یا خیر. |
| `bbox_width` | INT | بله | ۱ تا ۱۰۰۰ | عرض جعبه محصور برای مدل تولیدشده بر حسب سانتی‌متر. |
| `bbox_height` | INT | بله | ۱ تا ۱۰۰۰ | ارتفاع جعبه محصور برای مدل تولیدشده بر حسب سانتی‌متر. |
| `bbox_length` | INT | بله | ۱ تا ۱۰۰۰ | طول جعبه محصور برای مدل تولیدشده بر حسب سانتی‌متر. |
| `height_cm` | INT | بله | ۱ تا ۳۰۰ | ارتفاع مدل تولیدشده بر حسب سانتی‌متر. |

**نکته درباره تعداد تصاویر:** این گره بین ۱ تا ۵ تصویر را می‌پذیرد. اگر یک دسته از تصاویر (مثلاً یک دسته ۴ تصویری) ارائه دهید، هر تصویر در دسته به‌عنوان یک تصویر ورودی جداگانه در نظر گرفته می‌شود. ارائه بیش از ۵ تصویر منجر به خطا خواهد شد.
