## Обзор

Этот узел генерирует 3D-модель на основе одного-пяти референсных изображений с использованием API Rodin Gen-2.5. Вы можете выбрать один из режимов качества: «Быстрый», «Обычный» или «Экстремально высокий», чтобы сбалансировать скорость генерации и стоимость.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `изображения` | IMAGE | Да | от 1 до 5 изображений | От одного до пяти входных изображений. Первое изображение используется для материалов, если указано несколько изображений. |
| `режим` | COMBO | Да | `"Fast"`<br>`"Regular"`<br>`"Extreme-High"` | Режим качества генерации. Режимы более высокого качества дают лучшие результаты, но стоят дороже. |
| `материал` | COMBO | Да | `"PBR"`<br>`"Matte"` | Тип материала для сгенерированной 3D-модели. |
| `формат_файла_геометрии` | COMBO | Да | `"glb"`<br>`"obj"`<br>`"stl"`<br>`"usdz"` | Формат выходного файла геометрии 3D-модели. |
| `режим_текстуры` | COMBO | Да | `"Original"`<br>`"Clean"`<br>`"Style"` | Режим генерации текстур. «Original» сохраняет исходные текстуры, «Clean» удаляет их, а «Style» применяет стилизованную текстуру. |
| `seed` | INT | Да | от 0 до 2147483647 | Случайное зерно для воспроизводимых результатов. Используйте одно и то же зерно, чтобы получить одинаковый результат. |
| `TAPose` | BOOLEAN | Да | True / False | Применять ли T-позу к сгенерированной модели. |
| `hd_texture` | BOOLEAN | Да | True / False | Создавать ли карту текстуры высокой четкости. |
| `texture_delight` | BOOLEAN | Да | True / False | Удалять ли освещение из входных изображений перед генерацией текстуры. |
| `use_original_alpha` | BOOLEAN | Да | True / False | Использовать ли исходный альфа-канал из входных изображений. |
| `addon_highpack` | BOOLEAN | Да | True / False | Создавать ли высокополигональную версию модели в дополнение к стандартной. |
| `bbox_width` | INT | Да | от 1 до 1000 | Ширина ограничивающего прямоугольника для сгенерированной модели в сантиметрах. |
| `bbox_height` | INT | Да | от 1 до 1000 | Высота ограничивающего прямоугольника для сгенерированной модели в сантиметрах. |
| `bbox_length` | INT | Да | от 1 до 1000 | Длина ограничивающего прямоугольника для сгенерированной модели в сантиметрах. |
| `height_cm` | INT | Да | от 1 до 300 | Высота сгенерированной модели в сантиметрах. |

**Примечание о количестве изображений:** Узел принимает от 1 до 5 изображений. Если вы предоставляете пакет изображений (например, пакет из 4 изображений), каждое изображение в пакете обрабатывается как отдельное входное изображение. Предоставление более 5 изображений приведет к ошибке.

## Выходные параметры

| Имя выходного параметра | Тип данных | Описание |
|-------------|-----------|-------------|
| `model_file` | FILE3D | Файл сгенерированной 3D-модели в выбранном формате геометрии. |
