این گره با استفاده از یک توضیح متنی (پرامپت) به همراه تصاویر فریم اول و آخر، یک ویدیو تولید می‌کند. این گره توضیحات شما و دو فریم کلیدی را دریافت کرده و یک دنباله ویدیویی کامل ایجاد می‌کند که بین آن‌ها انتقال می‌یابد. گره گزینه‌های مختلفی برای کنترل وضوح، نسبت تصویر، مدت‌زمان و سایر پارامترهای تولید ویدیو ارائه می‌دهد.

## ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `model` | COMBO | بله | `"seedance-1-5-pro-251215"`<br>`"seedance-1-0-pro-250528"`<br>`"seedance-1-0-lite-i2v-250428"` | مدل مورد استفاده برای تولید ویدیو (پیش‌فرض: `"seedance-1-0-lite-i2v-250428"`). |
| `prompt` | STRING | بله | - | توضیح متنی (پرامپت) مورد استفاده برای تولید ویدیو. |
| `first_frame` | IMAGE | بله | - | فریم اول مورد استفاده برای ویدیو. باید بین 300x300 و 6000x6000 پیکسل با نسبت تصویر بین 0.4 و 2.5 باشد. |
| `last_frame` | IMAGE | بله | - | فریم آخر مورد استفاده برای ویدیو. باید بین 300x300 و 6000x6000 پیکسل با نسبت تصویر بین 0.4 و 2.5 باشد. |
| `resolution` | COMBO | بله | `"480p"`<br>`"720p"`<br>`"1080p"` | وضوح (رزولوشن) ویدیوی خروجی. |
| `aspect_ratio` | COMBO | بله | `"adaptive"`<br>`"16:9"`<br>`"4:3"`<br>`"1:1"`<br>`"3:4"`<br>`"9:16"`<br>`"21:9"` | نسبت تصویر ویدیوی خروجی (پیش‌فرض: `"adaptive"`). |
| `duration` | INT | بله | 3 - 12 | مدت‌زمان ویدیوی خروجی بر حسب ثانیه (پیش‌فرض: 5). توجه: برای مدل `seedance-1-5-pro-251215`، حداقل مدت‌زمان پشتیبانی‌شده 4 ثانیه است. |
| `seed` | INT | خیر | 0 - 2147483647 | دانه (seed) مورد استفاده برای تولید (پیش‌فرض: 0). |
| `camera_fixed` | BOOLEAN | خیر | - | مشخص می‌کند که آیا دوربین ثابت باشد یا خیر. پلتفرم یک دستورالعمل برای ثابت کردن دوربین به پرامپت شما اضافه می‌کند، اما تأثیر واقعی آن را تضمین نمی‌کند (پیش‌فرض: False). |
| `watermark` | BOOLEAN | خیر | - | آیا واترمارک «AI generated» به ویدیو اضافه شود یا خیر (پیش‌فرض: False). |
| `generate_audio` | BOOLEAN | خیر | - | این پارامتر برای هر مدلی به جز `seedance-1-5-pro-251215` نادیده گرفته می‌شود (پیش‌فرض: False). |
