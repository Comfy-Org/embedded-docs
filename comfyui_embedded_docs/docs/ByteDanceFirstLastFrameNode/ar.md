# ترجمة وثيقة عقدة ComfyUI

## نظرة عامة

تقوم هذه العقدة بإنشاء فيديو باستخدام موجه نصي مع صور الإطار الأول والأخير. تأخذ وصفك والإطارين الرئيسيين لإنشاء تسلسل فيديو كامل ينتقل بينهما. توفر العقدة خيارات متنوعة للتحكم في دقة الفيديو ونسبة العرض إلى الارتفاع والمدة ومعايير التوليد الأخرى.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|---------|---------------|--------|--------|-------|
| `model` | COMBO | نعم | `"seedance-1-5-pro-251215"`<br>`"seedance-1-0-pro-250528"`<br>`"seedance-1-0-lite-i2v-250428"` | النموذج المستخدم لتوليد الفيديو (الافتراضي: `"seedance-1-0-lite-i2v-250428"`). |
| `prompt` | STRING | نعم | - | الموجه النصي المستخدم لتوليد الفيديو. |
| `first_frame` | IMAGE | نعم | - | الإطار الأول المستخدم في الفيديو. يجب أن يكون بين 300×300 و6000×6000 بكسل، مع نسبة عرض إلى ارتفاع بين 0.4 و2.5. |
| `last_frame` | IMAGE | نعم | - | الإطار الأخير المستخدم في الفيديو. يجب أن يكون بين 300×300 و6000×6000 بكسل، مع نسبة عرض إلى ارتفاع بين 0.4 و2.5. |
| `resolution` | COMBO | نعم | `"480p"`<br>`"720p"`<br>`"1080p"` | دقة الفيديو الناتج. |
| `aspect_ratio` | COMBO | نعم | `"adaptive"`<br>`"16:9"`<br>`"4:3"`<br>`"1:1"`<br>`"3:4"`<br>`"9:16"`<br>`"21:9"` | نسبة العرض إلى الارتفاع للفيديو الناتج (الافتراضي: `"adaptive"`). |
| `duration` | INT | نعم | 3 - 12 | مدة الفيديو الناتج بالثواني (الافتراضي: 5). ملاحظة: بالنسبة لنموذج `seedance-1-5-pro-251215`، أقل مدة مدعومة هي 4 ثوانٍ. |
| `seed` | INT | لا | 0 - 2147483647 | البذرة المستخدمة للتوليد (الافتراضي: 0). |
| `camera_fixed` | BOOLEAN | لا | - | يحدد ما إذا كان سيتم تثبيت الكاميرا. تقوم المنصة بإضافة تعليمات لتثبيت الكاميرا إلى الموجه الخاص بك، ولكنها لا تضمن التأثير الفعلي (الافتراضي: False). |
| `watermark` | BOOLEAN | لا | - | ما إذا كان سيتم إضافة علامة مائية "مُولد بالذكاء الاصطناعي" إلى الفيديو (الافتراضي: False). |
| `generate_audio` | BOOLEAN | لا | - | يتم تجاهل هذا المعامل لأي نموذج باستثناء `seedance-1-5-pro-251215` (الافتراضي: False). |
