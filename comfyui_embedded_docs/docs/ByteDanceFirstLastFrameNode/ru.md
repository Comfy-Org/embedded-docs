Вот перевод документации на русский язык:

Этот узел генерирует видео на основе текстового запроса с использованием изображений первого и последнего кадров. Он принимает ваше описание и два ключевых кадра для создания полной видеопоследовательности с плавным переходом между ними. Узел предоставляет различные опции для управления разрешением видео, соотношением сторон, длительностью и другими параметрами генерации.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `model` | COMBO | Да | `"seedance-1-5-pro-251215"`<br>`"seedance-1-0-pro-250528"`<br>`"seedance-1-0-lite-i2v-250428"` | Модель для генерации видео (по умолчанию: `"seedance-1-0-lite-i2v-250428"`). |
| `prompt` | STRING | Да | - | Текстовый запрос для генерации видео. |
| `first_frame` | IMAGE | Да | - | Первый кадр для видео. Должен быть от 300x300 до 6000x6000 пикселей с соотношением сторон от 0,4 до 2,5. |
| `last_frame` | IMAGE | Да | - | Последний кадр для видео. Должен быть от 300x300 до 6000x6000 пикселей с соотношением сторон от 0,4 до 2,5. |
| `resolution` | COMBO | Да | `"480p"`<br>`"720p"`<br>`"1080p"` | Разрешение выходного видео. |
| `aspect_ratio` | COMBO | Да | `"adaptive"`<br>`"16:9"`<br>`"4:3"`<br>`"1:1"`<br>`"3:4"`<br>`"9:16"`<br>`"21:9"` | Соотношение сторон выходного видео (по умолчанию: `"adaptive"`). |
| `duration` | INT | Да | 3 - 12 | Длительность выходного видео в секундах (по умолчанию: 5). Примечание: для модели `seedance-1-5-pro-251215` минимальная поддерживаемая длительность составляет 4 секунды. |
| `seed` | INT | Нет | 0 - 2147483647 | Сид для генерации (по умолчанию: 0). |
| `camera_fixed` | BOOLEAN | Нет | - | Указывает, следует ли фиксировать камеру. Платформа добавляет инструкцию по фиксации камеры к вашему запросу, но не гарантирует фактический эффект (по умолчанию: False). |
| `watermark` | BOOLEAN | Нет | - | Добавлять ли водяной знак "Создано ИИ" на видео (по умолчанию: False). |
| `generate_audio` | BOOLEAN | Нет | - | Этот параметр игнорируется для всех моделей, кроме `seedance-1-5-pro-251215` (по умолчанию: False). |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `output` | VIDEO | Сгенерированный видеофайл |
