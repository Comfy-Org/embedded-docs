Вот перевод документации на русский язык:

Этот узел задаёт направляющий латент для модели редактирования. Он принимает данные кондиционирования и опциональный латент на входе, затем изменяет кондиционирование, добавляя информацию об опорном латенте. Если модель это поддерживает, вы можете соединить несколько узлов ReferenceLatent последовательно, чтобы задать несколько опорных изображений.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `кондиционирование` | CONDITIONING | Да | - | Данные кондиционирования, которые будут изменены с добавлением информации об опорном латенте |
| `латент` | LATENT | Нет | - | Опциональные латентные данные, используемые в качестве опорных для модели редактирования |

## Выходы

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `output` | CONDITIONING | Изменённые данные кондиционирования, содержащие информацию об опорном латенте |
