## Обзор

Загружает модель MoGe (Monocular Geometry — монокулярная геометрия) из файла и подготавливает её к использованию в задачах оценки геометрии. Этот узел считывает файл модели из папки `geometry_estimation` и инициализирует модель MoGe с её обученными весами.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `model_name` | STRING | Да | Список доступных файлов моделей в папке `geometry_estimation` | Имя файла модели MoGe для загрузки. Выберите из доступных файлов моделей в вашей установке ComfyUI. |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `MOGE_MODEL` | MOGE_MODEL | Загруженный экземпляр модели MoGe, готовый к использованию в рабочих процессах оценки геометрии. |
