Этот узел организует список скрытых представлений (latent images) и соответствующих им данных кондиционирования (conditioning) по их разрешению. Он группирует вместе элементы, имеющие одинаковую высоту и ширину, создавая отдельные пакеты для каждого уникального разрешения. Этот процесс полезен для подготовки данных к эффективному обучению, так как позволяет моделям обрабатывать несколько элементов одного размера совместно.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `latents` | LATENT | Да | N/A | Список словарей скрытых представлений для группировки по разрешению. |
| `conditioning` | CONDITIONING | Да | N/A | Список списков кондиционирования (должен соответствовать длине списка `latents`). |

**Примечание:** Количество элементов в списке `latents` должно точно совпадать с количеством элементов в списке `conditioning`. Каждый словарь скрытого представления может содержать пакет образцов, а соответствующий список кондиционирования должен содержать соответствующее количество элементов кондиционирования для этого пакета.

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `latents` | LATENT | Список пакетных словарей скрытых представлений, по одному на корзину разрешений. |
| `conditioning` | CONDITIONING | Список списков кондиционирования, по одному на корзину разрешений. |
