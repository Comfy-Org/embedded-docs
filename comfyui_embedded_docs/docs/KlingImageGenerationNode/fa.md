### نمای کلی

گره تولید تصویر Kling، تصاویری را از طریق توضیحات متنی با امکان استفاده از تصویر مرجع برای هدایت تولید، ایجاد می‌کند. این گره بر اساس توضیحات متنی و تنظیمات مرجع شما، یک یا چند تصویر تولید کرده و آن‌ها را به عنوان خروجی بازمی‌گرداند.

## ورودی‌ها

| پارامتر | نوع داده | اجباری | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `prompt` | STRING | بله | - | توضیحات متنی مثبت |
| `negative_prompt` | STRING | بله | - | توضیحات متنی منفی |
| `image_type` | COMBO | بله | `"subject_reference"`<br>`"style_reference"` | انتخاب نوع مرجع تصویر (پیشرفته). زمانی که تصویر مرجع ارائه شود، الزامی است. |
| `image_fidelity` | FLOAT | بله | 0.0 - 1.0 | شدت مرجع برای تصاویر بارگذاری‌شده توسط کاربر (پیش‌فرض: 0.5، پیشرفته) |
| `human_fidelity` | FLOAT | بله | 0.0 - 1.0 | شباهت مرجع سوژه (پیش‌فرض: 0.45، پیشرفته) |
| `model_name` | COMBO | بله | `"kling-v3"`<br>`"kling-v2"`<br>`"kling-v1-5"` | انتخاب مدل برای تولید تصویر (پیش‌فرض: "kling-v3") |
| `aspect_ratio` | COMBO | بله | `"16:9"`<br>`"9:16"`<br>`"1:1"`<br>`"4:3"`<br>`"3:4"`<br>`"21:9"`<br>`"9:21"` | نسبت ابعاد تصاویر تولیدشده (پیش‌فرض: "16:9") |
| `n` | INT | بله | 1 - 9 | تعداد تصاویر تولیدشده (پیش‌فرض: 1) |
| `image` | IMAGE | خیر | - | تصویر مرجع اختیاری |
| `seed` | INT | خیر | 0 - 2147483647 | دانه (seed) تعیین می‌کند که آیا گره باید دوباره اجرا شود؛ نتایج بدون توجه به دانه، قطعی نیستند (پیش‌فرض: 0) |

**محدودیت‌های پارامتر:**

- پارامتر `image` اختیاری است. هنگامی که تصویر مرجع ارائه می‌شود، پارامتر `image_type` باید روی `"subject_reference"` یا `"style_reference"` تنظیم شود.
- زمانی که تصویر مرجع ارائه نشود، پارامترهای `image_type`، `image_fidelity` و `human_fidelity` استفاده نمی‌شوند.
- توضیحات مثبت و منفی حداکثر طول `MAX_PROMPT_LENGTH_IMAGE_GEN` کاراکتر دارند.
- پارامتر `seed` اختیاری است و نتایج قطعی را تضمین نمی‌کند.
