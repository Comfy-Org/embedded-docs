## نظرة عامة

تقوم عقدة توليد الصور Kling Image Generation بإنشاء صور من النصوص الوصفية مع إمكانية استخدام صورة مرجعية للتوجيه. تنتج العقدة صورة واحدة أو أكثر بناءً على النص الوصفي وإعدادات المرجع، ثم تُرجع الصور المُنشأة كمخرجات.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|---------|---------------|--------|---------|-------------|
| `نص التوجيه الإيجابي` | STRING | نعم | - | النص الوصفي الإيجابي |
| `نص التوجيه السلبي` | STRING | نعم | - | النص الوصفي السلبي |
| `نوع الصورة` | COMBO | نعم | `"subject_reference"`<br>`"style_reference"` | اختيار نوع المرجع للصورة (متقدم). مطلوب عند توفير صورة مرجعية. |
| `شدة المرجع للصورة المرفوعة` | FLOAT | نعم | 0.0 - 1.0 | شدة المرجع للصور المرفوعة من المستخدم (الافتراضي: 0.5، متقدم) |
| `تشابه الموضوع` | FLOAT | نعم | 0.0 - 1.0 | درجة تشابه المرجع مع الموضوع (الافتراضي: 0.45، متقدم) |
| `اسم النموذج` | COMBO | نعم | `"kling-v3"`<br>`"kling-v2"`<br>`"kling-v1-5"` | اختيار النموذج لتوليد الصور (الافتراضي: "kling-v3") |
| `نسبة العرض إلى الارتفاع` | COMBO | نعم | `"16:9"`<br>`"9:16"`<br>`"1:1"`<br>`"4:3"`<br>`"3:4"`<br>`"21:9"`<br>`"9:21"` | نسبة الأبعاد للصور المُنشأة (الافتراضي: "16:9") |
| `عدد الصور` | INT | نعم | 1 - 9 | عدد الصور المُنشأة (الافتراضي: 1) |
| `صورة` | IMAGE | لا | - | صورة مرجعية اختيارية |
| `البذرة` | INT | لا | 0 - 2147483647 | البذرة تتحكم في إعادة تشغيل العقدة؛ النتائج غير حتمية بغض النظر عن البذرة (الافتراضي: 0) |

**قيود المعاملات:**

- معامل `image` اختياري. عند توفير صورة مرجعية، يجب تعيين معامل `image_type` إما إلى `"subject_reference"` أو `"style_reference"`.
- عند عدم توفير صورة مرجعية، لا تُستخدم معاملات `image_type` و `image_fidelity` و `human_fidelity`.
- النص الوصفي والنص الوصفي السلبي لهما حد أقصى للطول يبلغ `MAX_PROMPT_LENGTH_IMAGE_GEN` حرفًا.
- معامل `seed` اختياري ولا يضمن نتائج حتمية.
