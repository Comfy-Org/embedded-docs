گره VAEDecodeHunyuan3D، بازنمایی‌های نهفته (latent) را با استفاده از یک رمزگشای VAE به داده‌های وکسل سه‌بعدی تبدیل می‌کند. این گره نمونه‌های نهفته را از طریق مدل VAE با تنظیمات قابل‌پیکربندی برای تکه‌تکه‌سازی و وضوح پردازش می‌کند تا داده‌های حجمی مناسب برای کاربردهای سه‌بعدی تولید کند.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|----------|----------|--------|--------|----------|
| `samples` | LATENT | بله | - | بازنمایی نهفته‌ای که باید به داده‌های وکسل سه‌بعدی رمزگشایی شود |
| `vae` | VAE | بله | - | مدل VAE مورد استفاده برای رمزگشایی نمونه‌های نهفته |
| `num_chunks` | INT | بله | 1000-500000 | تعداد تکه‌هایی که پردازش برای مدیریت حافظه به آنها تقسیم می‌شود (پیش‌فرض: 8000) |
| `octree_resolution` | INT | بله | 16-512 | وضوح ساختار هشت‌درختی (octree) مورد استفاده برای تولید وکسل سه‌بعدی (پیش‌فرض: 256) |

## خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-----------|----------|----------|
| `voxels` | VOXEL | داده‌های وکسل سه‌بعدی تولید شده از بازنمایی نهفته رمزگشایی‌شده |
