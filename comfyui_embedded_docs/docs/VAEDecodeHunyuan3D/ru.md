Вот перевод документации на русский язык:

Узел VAEDecodeHunyuan3D преобразует скрытые представления в 3D-данные вокселей с помощью декодера VAE. Он обрабатывает скрытые образцы через модель VAE с настраиваемыми параметрами разбиения на чанки и разрешения для генерации объемных данных, подходящих для 3D-приложений.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `samples` | LATENT | Да | - | Скрытое представление, которое необходимо декодировать в 3D-данные вокселей |
| `vae` | VAE | Да | - | Модель VAE, используемая для декодирования скрытых образцов |
| `num_chunks` | INT | Да | 1000-500000 | Количество чанков, на которые разбивается обработка для управления памятью (по умолчанию: 8000) |
| `octree_resolution` | INT | Да | 16-512 | Разрешение октодерева, используемого для генерации 3D-вокселей (по умолчанию: 256) |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `voxels` | VOXEL | Сгенерированные 3D-данные вокселей из декодированного скрытого представления |
