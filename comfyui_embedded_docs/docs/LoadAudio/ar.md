# ترجمة وثيقة LoadAudio

## نظرة عامة

تقوم عقدة LoadAudio بتحميل الملفات الصوتية من دليل الإدخال وتحويلها إلى تنسيق يمكن معالجته بواسطة عقد الصوت الأخرى في ComfyUI. تقوم بقراءة الملفات الصوتية واستخراج بيانات الشكل الموجي ومعدل العينة، مما يجعلها متاحة لمهام معالجة الصوت اللاحقة.

## المدخلات

| Parameter | Data Type | Required | Range | Description |
|-----------|-----------|----------|-------|-------------|
| `صوت` | AUDIO | نعم | جميع الملفات الصوتية والفيديو المدعومة في دليل الإدخال | الملف الصوتي المراد تحميله من دليل الإدخال |

**ملاحظة:** تقبل العقدة فقط الملفات الصوتية والفيديو الموجودة في دليل الإدخال الخاص بـ ComfyUI. يجب أن يكون الملف موجودًا وقابلًا للوصول لضمان نجاح التحميل.

## المخرجات

| Output Name | Data Type | Description |
|-------------|-----------|-------------|
| `AUDIO` | AUDIO | بيانات صوتية تحتوي على معلومات الشكل الموجي ومعدل العينة |
