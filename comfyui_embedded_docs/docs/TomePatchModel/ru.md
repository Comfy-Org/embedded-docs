Вот перевод документации на русский язык:

Узел TomePatchModel применяет метод слияния токенов (Token Merging, ToMe) к диффузионной модели для снижения вычислительных затрат во время инференса. Он работает путем выборочного объединения похожих токенов в механизме внимания, что позволяет модели обрабатывать меньшее количество токенов, сохраняя при этом качество изображения. Данная техника помогает ускорить генерацию без существенной потери качества.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `модель` | MODEL | Да | - | Диффузионная модель, к которой применяется слияние токенов |
| `соотношение` | FLOAT | Нет | 0.0 - 1.0 | Доля токенов для слияния (по умолчанию: 0.3) |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `модель` | MODEL | Модифицированная модель с примененным слиянием токенов |
