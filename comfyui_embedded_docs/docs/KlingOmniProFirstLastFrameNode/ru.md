Этот узел использует новейшую модель Kling AI для генерации видео из начального кадра, опционального конечного кадра или референсных изображений. Он может создать одно видео или многокадровый раскадровку с отдельными промптами и длительностью для каждого сегмента. Узел обрабатывает эти входные данные для создания видео заданной длины и разрешения с возможностью генерации аудио.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `model_name` | COMBO | Да | `"kling-v3-omni"`<br>`"kling-video-o1"` | Конкретная модель Kling AI для генерации видео. |
| `prompt` | STRING | Да | - | Текстовый промпт, описывающий содержимое видео. Может включать как положительные, так и отрицательные описания. Игнорируется при включенных раскадровках. |
| `duration` | INT | Да | от 3 до 15 | Желаемая длительность генерируемого видео в секундах (по умолчанию: 5). |
| `first_frame` | IMAGE | Да | - | Начальное изображение для видеопоследовательности. |
| `end_frame` | IMAGE | Нет | - | Опциональный конечный кадр видео. Не может использоваться одновременно с `reference_images`. Не работает с раскадровками. |
| `reference_images` | IMAGE | Нет | - | До 6 дополнительных референсных изображений. |
| `resolution` | COMBO | Нет | `"4k"`<br>`"1080p"`<br>`"720p"` | Выходное разрешение генерируемого видео (по умолчанию: "1080p"). |
| `storyboards` | DYNAMIC_COMBO | Нет | `"disabled"`<br>`"1 storyboard"`<br>`"2 storyboards"`<br>`"3 storyboards"`<br>`"4 storyboards"`<br>`"5 storyboards"`<br>`"6 storyboards"` | Генерация серии видеосегментов с отдельными промптами и длительностью. Поддерживается только для `kling-v3-omni`. При включении для каждой раскадровки требуется ввод промпта и длительности. |
| `generate_audio` | BOOLEAN | Нет | True / False | Генерация аудио для видео (по умолчанию: False). Поддерживается только для `kling-v3-omni`. |
| `seed` | INT | Нет | от 0 до 2147483647 | Сид определяет, должен ли узел перезапускаться; результаты недетерминированы независимо от сида (по умолчанию: 0). |

**Важные ограничения:**

* Входной параметр `end_frame` не может использоваться одновременно с входным параметром `reference_images`.
* Входной параметр `end_frame` не может использоваться одновременно с раскадровками.
* Модель `kling-video-o1` не поддерживает длительность более 10 секунд, генерацию аудио, разрешение 4k или раскадровки.
* Если вы не предоставите `end_frame` или `reference_images` для модели `kling-video-o1`, `duration` может быть установлена только на 5 или 10 секунд.
* Все входные изображения (`first_frame`, `end_frame` и любые `reference_images`) должны иметь минимальный размер 300 пикселей как по ширине, так и по высоте.
* Соотношение сторон всех входных изображений должно быть в пределах от 1:2.5 до 2.5:1.
* Через входной параметр `reference_images` можно передать максимум 6 изображений.
* Текст промпта должен содержать от 1 до 2500 символов (0 символов допускается при включенных раскадровках).
* При включенных раскадровках общая длительность всех сегментов раскадровки должна равняться глобальному значению `duration`.
* Каждый промпт раскадровки должен содержать от 1 до 512 символов.

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `output` | VIDEO | Сгенерированный видеофайл. |
