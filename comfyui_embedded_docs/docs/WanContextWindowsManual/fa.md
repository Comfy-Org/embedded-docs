گره WAN Context Windows (Manual) به شما امکان می‌دهد تا به‌صورت دستی پنجره‌های زمینه (context windows) را برای مدل‌های شبه-WAN با پردازش دو بعدی پیکربندی کنید. این گره با مشخص کردن طول پنجره، هم‌پوشانی، روش زمان‌بندی و تکنیک تلفیق، تنظیمات سفارشی پنجره زمینه را در طول نمونه‌برداری اعمال می‌کند. این کار کنترل دقیقی بر نحوه پردازش اطلاعات توسط مدل در مناطق مختلف زمینه به شما می‌دهد.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|---------|----------|--------|--------|---------|
| `model` | MODEL | بله | - | مدلی که در طول نمونه‌برداری، پنجره‌های زمینه روی آن اعمال می‌شود. |
| `context_length` | INT | بله | 1 تا 1048576 | طول پنجره زمینه (پیش‌فرض: 81). |
| `context_overlap` | INT | بله | 0 تا 1048576 | هم‌پوشانی پنجره زمینه (پیش‌فرض: 30). |
| `context_schedule` | COMBO | بله | `"static_standard"`<br>`"uniform_standard"`<br>`"uniform_looped"`<br>`"batched"` | گام (stride) پنجره زمینه. |
| `context_stride` | INT | بله | 1 تا 1048576 | گام پنجره زمینه؛ فقط برای زمان‌بندی‌های یکنواخت (uniform) قابل اعمال است (پیش‌فرض: 1). |
| `closed_loop` | BOOLEAN | بله | - | آیا حلقه پنجره زمینه بسته شود؛ فقط برای زمان‌بندی‌های حلقه‌ای (looped) قابل اعمال است (پیش‌فرض: False). |
| `fuse_method` | COMBO | بله | `"pyramid"`<br>`"gaussian"`<br>`"average"`<br>`"overlap"` | روش مورد استفاده برای تلفیق پنجره‌های زمینه (پیش‌فرض: "pyramid"). |
| `freenoise` | BOOLEAN | بله | - | آیا نویز FreeNoise برای به‌هم‌آمیزی بهتر پنجره‌ها اعمال شود (پیش‌فرض: False). |

**نکته:** پارامتر `context_stride` فقط بر زمان‌بندی‌های یکنواخت تأثیر می‌گذارد و `closed_loop` فقط برای زمان‌بندی‌های حلقه‌ای قابل اعمال است. طول پنجره و مقادیر هم‌پوشانی به‌طور خودکار تنظیم می‌شوند تا حداقل مقادیر معتبر در طول پردازش تضمین شوند. پارامتر `fuse_method` اکنون گزینه‌های اضافی فراتر از فقط "pyramid" را شامل می‌شود.
