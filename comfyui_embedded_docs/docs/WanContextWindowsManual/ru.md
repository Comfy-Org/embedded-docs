Вот перевод документации на русский язык:

Узел WAN Context Windows (Manual) позволяет вручную настраивать контекстные окна для моделей типа WAN с двумерной обработкой. Он применяет пользовательские настройки контекстных окон во время сэмплирования, задавая длину окна, перекрытие, метод планирования и технику слияния. Это дает вам точный контроль над тем, как модель обрабатывает информацию в разных областях контекста.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `model` | MODEL | Да | - | Модель, к которой применяются контекстные окна во время сэмплирования. |
| `context_length` | INT | Да | от 1 до 1048576 | Длина контекстного окна (по умолчанию: 81). |
| `context_overlap` | INT | Да | от 0 до 1048576 | Перекрытие контекстного окна (по умолчанию: 30). |
| `context_schedule` | COMBO | Да | `"static_standard"`<br>`"uniform_standard"`<br>`"uniform_looped"`<br>`"batched"` | Шаг контекстного окна. |
| `context_stride` | INT | Да | от 1 до 1048576 | Шаг контекстного окна; применяется только к равномерным расписаниям (по умолчанию: 1). |
| `closed_loop` | BOOLEAN | Да | - | Замыкать ли цикл контекстного окна; применяется только к циклическим расписаниям (по умолчанию: False). |
| `fuse_method` | COMBO | Да | `"pyramid"`<br>`"gaussian"`<br>`"average"`<br>`"overlap"` | Метод слияния контекстных окон (по умолчанию: "pyramid"). |
| `freenoise` | BOOLEAN | Да | - | Применять ли перетасовку шума FreeNoise для улучшения смешивания окон (по умолчанию: False). |

**Примечание:** Параметр `context_stride` влияет только на равномерные расписания, а `closed_loop` применяется только к циклическим расписаниям. Значения длины контекста и перекрытия автоматически корректируются для обеспечения минимальных допустимых значений во время обработки. Параметр `fuse_method` теперь включает дополнительные опции помимо "pyramid".

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-----------|
| `model` | MODEL | Модель с примененной конфигурацией контекстных окон. |
