## Обзор

Изменение существующего изображения на основе текстового запроса. Этот узел отправляет ваши изображения и текстовое описание в Grok API, который редактирует изображения в соответствии с вашими инструкциями и возвращает результат.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `prompt` | STRING | Да | Н/Д | Текстовый запрос, используемый для генерации изображения. Должен содержать не менее 1 символа после удаления пробелов. |
| `model` | MODEL | Да | См. описание | Модель Grok для работы с изображениями. Этот параметр имеет несколько дополнительных опций, которые появляются после выбора модели. Доступные модели: `grok-imagine-image-quality`, `grok-imagine-image-pro`, `grok-imagine-image`. Каждая модель имеет разные возможности (см. примечание ниже). |
| `seed` | INT | Да | от 0 до 2147483647 | Значение для определения необходимости повторного запуска узла; фактические результаты недетерминированы независимо от seed. (по умолчанию: 0) |

**Примечание об ограничениях параметра `model`:**
- Параметр `model` представляет собой динамический комбинированный список, включающий дополнительные опции для `resolution`, `number_of_images`, `images` и `aspect_ratio`.
- **`grok-imagine-image-quality`**: Поддерживает до 3 входных изображений и позволяет задавать произвольное соотношение сторон.
- **`grok-imagine-image-pro`**: Поддерживает только 1 входное изображение и не позволяет задавать произвольное соотношение сторон.
- **`grok-imagine-image`**: Поддерживает до 3 входных изображений и позволяет задавать произвольное соотношение сторон.
- **Для редактирования требуется хотя бы одно входное изображение**. Узел выдаст ошибку, если изображения не предоставлены.
- **Произвольное соотношение сторон** (дополнительная опция `aspect_ratio`) разрешено только при подключении нескольких изображений к входу. Если предоставлено только одно изображение, соотношение сторон должно быть установлено на "auto".

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `IMAGE` | IMAGE | Отредактированное изображение(я), возвращенное Grok API. Если сгенерировано одно изображение, оно возвращается напрямую. Если сгенерировано несколько изображений, они объединяются в один пакетный тензор. |
