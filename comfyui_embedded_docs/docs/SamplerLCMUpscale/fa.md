### مرور کلی

گره SamplerLCMUpscale یک روش نمونه‌برداری تخصصی ارائه می‌دهد که نمونه‌برداری مدل سازگار با نهفتگی (LCM) را با قابلیت‌های افزایش وضوح تصویر ترکیب می‌کند. این گره به شما امکان می‌دهد در طول فرآیند نمونه‌برداری با استفاده از روش‌های درون‌یابی مختلف، تصاویر را بزرگ‌نمایی کنید؛ این ویژگی برای تولید خروجی‌هایی با وضوح بالاتر و حفظ کیفیت تصویر مفید است.

## ورودی‌ها

| پارامتر | نوع داده | اجباری | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `scale_ratio` | FLOAT | خیر | 0.1 - 20.0 | ضریب مقیاس‌گذاری برای افزایش وضوح (پیش‌فرض: 1.0) |
| `scale_steps` | INT | خیر | 1- - 1000 | تعداد مراحل فرآیند افزایش وضوح. برای محاسبه خودکار از 1- استفاده کنید (پیش‌فرض: 1-) |
| `upscale_method` | COMBO | بله | "bislerp"<br>"nearest-exact"<br>"bilinear"<br>"area"<br>"bicubic" | روش درون‌یابی مورد استفاده برای افزایش وضوح تصویر |
