Вот перевод документации на русский язык:

Узел SamplerLCMUpscale предоставляет специализированный метод сэмплирования, который сочетает сэмплирование на основе модели латентной согласованности (LCM) с возможностями масштабирования изображения. Он позволяет увеличивать разрешение изображений в процессе сэмплирования, используя различные методы интерполяции, что делает его полезным для генерации выходных данных более высокого разрешения с сохранением качества изображения.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `коэффициент_масштабирования` | FLOAT | Нет | 0.1 - 20.0 | Коэффициент масштабирования, применяемый при увеличении разрешения (по умолчанию: 1.0) |
| `шаги_масштабирования` | INT | Нет | -1 - 1000 | Количество шагов, используемых в процессе масштабирования. Используйте -1 для автоматического расчета (по умолчанию: -1) |
| `метод_масштабирования` | COMBO | Да | "bislerp"<br>"nearest-exact"<br>"bilinear"<br>"area"<br>"bicubic" | Метод интерполяции, используемый для масштабирования изображения |

## Выходные параметры

| Имя выходного параметра | Тип данных | Описание |
|-------------|-----------|-------------|
| `sampler` | SAMPLER | Возвращает настроенный объект сэмплера, который можно использовать в конвейере сэмплирования |
