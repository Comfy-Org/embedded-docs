گره ByteDance Image با استفاده از مدل‌های ByteDance از طریق یک API و بر اساس متن راهنما (prompt)، تصاویر تولید می‌کند. این گره به شما امکان می‌دهد یک مدل را انتخاب کنید، ابعاد تصویر را مشخص کنید و پارامترهای مختلف تولید مانند seed و guidance scale را کنترل کنید. این گره به سرویس تولید تصویر ByteDance متصل می‌شود و تصویر ایجاد شده را بازمی‌گرداند.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|---------|----------|--------|--------|---------|
| `مدل` | STRING | بله | `"seedream-3-0-t2i-250415"` | مدل ByteDance که برای تولید تصویر استفاده می‌شود. در حال حاضر فقط یک گزینه مدل موجود است. |
| `پرامپت` | STRING | بله | - | متن راهنما (prompt) که برای تولید تصویر استفاده می‌شود. باید حداقل ۱ کاراکتر پس از حذف فاصله‌های خالی داشته باشد. |
| `پیش‌تنظیم اندازه` | STRING | بله | توضیحات را ببینید | یک اندازه پیشنهادی را انتخاب کنید. گزینه Custom را برای استفاده از عرض و ارتفاع زیر انتخاب کنید. اندازه‌های از پیش تعیین شده توسط لیست `RECOMMENDED_PRESETS` تعریف می‌شوند. |
| `عرض` | INT | بله | ۵۱۲ تا ۲۰۴۸ (گام ۶۴) | عرض سفارشی برای تصویر. این مقدار فقط زمانی استفاده می‌شود که `پیش‌تنظیم اندازه` روی `Custom` تنظیم شده باشد. پیش‌فرض: ۱۰۲۴. |
| `ارتفاع` | INT | بله | ۵۱۲ تا ۲۰۴۸ (گام ۶۴) | ارتفاع سفارشی برای تصویر. این مقدار فقط زمانی استفاده می‌شود که `پیش‌تنظیم اندازه` روی `Custom` تنظیم شده باشد. پیش‌فرض: ۱۰۲۴. |
| `seed` | INT | خیر | ۰ تا ۲۱۴۷۴۸۳۶۴۷ (گام ۱) | Seed مورد استفاده برای تولید. پیش‌فرض: ۰. |
| `guidance_scale` | FLOAT | خیر | ۱.۰ تا ۱۰.۰ (گام ۰.۰۱) | مقدار بالاتر باعث می‌شود تصویر بیشتر از متن راهنما پیروی کند. پیش‌فرض: ۲.۵. |
| `واترمارک` | BOOLEAN | خیر | True / False | آیا واترمارک "AI generated" به تصویر اضافه شود یا خیر. پیش‌فرض: False. این یک پارامتر پیشرفته است. |

**نکته در مورد پارامترهای اندازه:** پارامترهای `width` و `height` فقط زمانی استفاده می‌شوند که `size_preset` روی `Custom` تنظیم شده باشد. اگر یک اندازه از پیش تعیین شده انتخاب شود، ابعاد آن اندازه، مقادیر عرض و ارتفاع سفارشی را نادیده می‌گیرد. هنگام استفاده از ابعاد سفارشی، هر دو عرض و ارتفاع باید بین ۵۱۲ و ۲۰۴۸ پیکسل باشند.
