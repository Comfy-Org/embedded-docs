Узел "ByteDance Image" генерирует изображения с использованием моделей ByteDance через API на основе текстовых подсказок. Он позволяет выбрать модель, указать размеры изображения и управлять различными параметрами генерации, такими как seed и guidance scale. Узел подключается к сервису генерации изображений ByteDance и возвращает созданное изображение.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `model` | STRING | Да | `"seedream-3-0-t2i-250415"` | Модель ByteDance, используемая для генерации изображения. В настоящее время доступен только один вариант модели. |
| `промпт` | STRING | Да | - | Текстовая подсказка, используемая для генерации изображения. Длина должна быть не менее 1 символа после удаления пробелов. |
| `размер_пресет` | STRING | Да | См. описание | Выберите рекомендуемый размер. Выберите Custom, чтобы использовать ширину и высоту ниже. Доступные пресеты определяются списком `RECOMMENDED_PRESETS`. |
| `ширина` | INT | Да | от 512 до 2048 (шаг 64) | Пользовательская ширина изображения. Это значение используется только когда `размер_пресет` установлен в `Custom`. По умолчанию: 1024. |
| `высота` | INT | Да | от 512 до 2048 (шаг 64) | Пользовательская высота изображения. Это значение используется только когда `размер_пресет` установлен в `Custom`. По умолчанию: 1024. |
| `сид` | INT | Нет | от 0 до 2147483647 (шаг 1) | Seed, используемый для генерации. По умолчанию: 0. |
| `коэффициент_направленности` | FLOAT | Нет | от 1.0 до 10.0 (шаг 0.01) | Более высокое значение заставляет изображение точнее следовать подсказке. По умолчанию: 2.5. |
| `водяной_знак` | BOOLEAN | Нет | True / False | Добавлять ли водяной знак "AI generated" на изображение. По умолчанию: False. Это расширенный параметр. |

**Примечание о параметрах размера:** Параметры `width` и `height` используются только когда `size_preset` установлен в `Custom`. Если выбран предустановленный размер, его размеры переопределяют пользовательские значения ширины и высоты. При использовании пользовательских размеров ширина и высота должны быть в диапазоне от 512 до 2048 пикселей.

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `IMAGE` | IMAGE | Сгенерированное изображение, возвращенное из API ByteDance в виде тензора. |
