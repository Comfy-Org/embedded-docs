# ترجمة وثيقة عقدة ByteDance Image

عقدة ByteDance Image تُنشئ الصور باستخدام نماذج ByteDance عبر واجهة برمجة التطبيقات (API) بناءً على النصوص التوجيهية. تتيح لك اختيار نموذج، وتحديد أبعاد الصورة، والتحكم في معاملات التوليد المختلفة مثل البذرة ومقياس التوجيه. تتصل العقدة بخدمة توليد الصور من ByteDance وتُعيد الصورة المُنشأة.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|---------|---------------|--------|---------|-------|
| `model` | STRING | نعم | `"seedream-3-0-t2i-250415"` | نموذج ByteDance المستخدم لتوليد الصورة. يتوفر حالياً خيار نموذج واحد فقط. |
| `prompt` | STRING | نعم | - | النص التوجيهي المستخدم لتوليد الصورة. يجب أن يكون بطرف حرف واحد على الأقل بعد إزالة المسافات البيضاء. |
| `size_preset` | STRING | نعم | انظر الوصف | اختر حجماً موصى به. اختر "مخصص" لاستخدام قيم العرض والارتفاع أدناه. يتم تعريف الإعدادات المسبقة المتاحة بواسطة قائمة `RECOMMENDED_PRESETS`. |
| `width` | INT | نعم | 512 إلى 2048 (خطوة 64) | العرض المخصص للصورة. تُستخدم هذه القيمة فقط عندما يكون `size_preset` مضبوطاً على "مخصص". القيمة الافتراضية: 1024. |
| `height` | INT | نعم | 512 إلى 2048 (خطوة 64) | الارتفاع المخصص للصورة. تُستخدم هذه القيمة فقط عندما يكون `size_preset` مضبوطاً على "مخصص". القيمة الافتراضية: 1024. |
| `seed` | INT | لا | 0 إلى 2147483647 (خطوة 1) | البذرة المستخدمة في التوليد. القيمة الافتراضية: 0. |
| `guidance_scale` | FLOAT | لا | 1.0 إلى 10.0 (خطوة 0.01) | القيمة الأعلى تجعل الصورة تتبع النص التوجيهي بشكل أدق. القيمة الافتراضية: 2.5. |
| `watermark` | BOOLEAN | لا | صواب / خطأ | إضافة علامة مائية "مُولد بالذكاء الاصطناعي" إلى الصورة. القيمة الافتراضية: خطأ. هذا معامل متقدم. |

**ملاحظة حول معاملات الحجم:** تُستخدم معاملات `width` و `height` فقط عندما يكون `size_preset` مضبوطاً على "مخصص". إذا تم اختيار حجم مسبق، فإن أبعاد الإعداد المسبق تلغي قيم العرض والارتفاع المخصصة. يجب أن يكون كل من العرض والارتفاع بين 512 و 2048 بكسل عند استخدام أبعاد مخصصة.
