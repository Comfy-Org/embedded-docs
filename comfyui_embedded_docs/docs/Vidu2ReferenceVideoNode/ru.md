Вот перевод документации на русский язык:

Узел Vidu2 «Преобразование референсов в видео» создаёт видео на основе текстового запроса и нескольких референсных изображений. Вы можете задать до семи объектов, каждый со своим набором референсных изображений, и ссылаться на них в запросе с помощью `@subject{subject_id}`. Узел генерирует видео с настраиваемой длительностью, соотношением сторон и движением.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `model` | COMBO | Да | `"viduq2"` | Модель ИИ, используемая для генерации видео. |
| `subjects` | AUTOGROW | Да | Н/Д | Для каждого объекта укажите до 3 референсных изображений (всего 7 изображений на все объекты). Ссылайтесь на них в запросах через `@subject{subject_id}`. |
| `prompt` | STRING | Да | Н/Д | Текстовое описание, используемое для управления генерацией видео. Когда параметр `audio` включён, видео будет содержать сгенерированную речь и фоновую музыку на основе этого запроса. |
| `audio` | BOOLEAN | Нет | Н/Д | При включении видео будет содержать сгенерированную речь и фоновую музыку на основе запроса (по умолчанию: `False`). |
| `duration` | INT | Нет | от 1 до 10 | Длительность генерируемого видео в секундах (по умолчанию: `5`). |
| `seed` | INT | Нет | от 0 до 2147483647 | Число, используемое для управления случайностью генерации и получения воспроизводимых результатов (по умолчанию: `1`). |
| `aspect_ratio` | COMBO | Нет | `"16:9"`<br>`"9:16"`<br>`"4:3"`<br>`"3:4"`<br>`"1:1"` | Форма видеокадра. |
| `resolution` | COMBO | Нет | `"720p"`<br>`"1080p"` | Пиксельное разрешение выходного видео. |
| `movement_amplitude` | COMBO | Нет | `"auto"`<br>`"small"`<br>`"medium"`<br>`"large"` | Управляет амплитудой движения объектов в кадре. |

**Ограничения:**

* Длина `prompt` должна составлять от 1 до 2000 символов.
* Можно задать несколько объектов, но общее количество референсных изображений по всем объектам не должно превышать 7.
* Каждый отдельный объект может иметь не более 3 референсных изображений.
* Соотношение ширины к высоте каждого референсного изображения должно быть в пределах от 1:4 до 4:1.
* Каждое референсное изображение должно быть не менее 128 пикселей по ширине и высоте.

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `output` | VIDEO | Сгенерированный видеофайл. |
