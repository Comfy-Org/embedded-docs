### مرور کلی

گره تولید ویدیو از مرجع Vidu2، یک ویدیو از روی متن prompt و چندین تصویر مرجع می‌سازد. می‌توانید تا هفت سوژه تعریف کنید که هرکدام مجموعه تصاویر مرجع خود را دارند و در prompt با استفاده از `@subject{subject_id}` به آن‌ها ارجاع دهید. این گره ویدیویی با مدت‌زمان، نسبت تصویر و حرکت قابل تنظیم تولید می‌کند.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|---------|----------|--------|--------|----------|
| `model` | COMBO | بله | `"viduq2"` | مدل هوش مصنوعی مورد استفاده برای تولید ویدیو. |
| `subjects` | AUTOGROW | بله | نامشخص | برای هر سوژه، حداکثر ۳ تصویر مرجع (در مجموع ۷ تصویر برای همه سوژه‌ها) ارائه دهید. در promptها با `@subject{subject_id}` به آن‌ها ارجاع دهید. |
| `prompt` | STRING | بله | نامشخص | توضیحات متنی که برای هدایت تولید ویدیو استفاده می‌شود. وقتی پارامتر `audio` فعال باشد، ویدیو شامل گفتار تولیدشده و موسیقی پس‌زمینه بر اساس این prompt خواهد بود. |
| `audio` | BOOLEAN | خیر | نامشخص | در صورت فعال‌سازی، ویدیو شامل گفتار تولیدشده و موسیقی پس‌زمینه بر اساس prompt خواهد بود (پیش‌فرض: `False`). |
| `duration` | INT | خیر | ۱ تا ۱۰ | طول ویدیوی تولیدشده بر حسب ثانیه (پیش‌فرض: `۵`). |
| `seed` | INT | خیر | ۰ تا ۲۱۴۷۴۸۳۶۴۷ | عددی برای کنترل تصادفی‌بودن تولید به منظور نتایج قابل تکرار (پیش‌فرض: `۱`). |
| `aspect_ratio` | COMBO | خیر | `"16:9"`<br>`"9:16"`<br>`"4:3"`<br>`"3:4"`<br>`"1:1"` | شکل قاب ویدیو. |
| `resolution` | COMBO | خیر | `"720p"`<br>`"1080p"` | وضوح پیکسلی ویدیوی خروجی. |
| `movement_amplitude` | COMBO | خیر | `"auto"`<br>`"small"`<br>`"medium"`<br>`"large"` | دامنه حرکت اشیاء در قاب را کنترل می‌کند. |

**محدودیت‌ها:**

* `prompt` باید بین ۱ تا ۲۰۰۰ کاراکتر باشد.
* می‌توانید چندین سوژه تعریف کنید، اما تعداد کل تصاویر مرجع در همه سوژه‌ها نباید از ۷ تجاوز کند.
* هر سوژه مجزا حداکثر می‌تواند ۳ تصویر مرجع داشته باشد.
* نسبت عرض به ارتفاع هر تصویر مرجع باید بین ۱:۴ و ۴:۱ باشد.
* هر تصویر مرجع باید حداقل ۱۲۸ پیکسل در عرض و ارتفاع داشته باشد.
