Вот перевод документации на русский язык:

Этот узел создаёт хук LoRA (Low-Rank Adaptation), который применяется только к компоненту модели, полностью оставляя компонент CLIP без изменений. Он загружает файл LoRA и применяет его с указанной силой к модели, устанавливая силу воздействия на CLIP равной нулю. Узел может объединяться в цепочку с предыдущими хуками для создания сложных конвейеров модификации.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `lora_name` | STRING | Да | Доступно несколько опций | Имя файла LoRA для загрузки из папки loras |
| `strength_model` | FLOAT | Да | от -20.0 до 20.0 | Множитель силы применения LoRA к компоненту модели (по умолчанию: 1.0) |
| `prev_hooks` | HOOKS | Нет | - | Опциональные предыдущие хуки для объединения в цепочку с этим хуком |

## Выходы

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `hooks` | HOOKS | Созданный хук LoRA, который может быть применён к обработке модели |
