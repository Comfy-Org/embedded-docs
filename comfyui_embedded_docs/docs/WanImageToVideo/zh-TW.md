WanImageToVideo 節點負責為影片生成任務準備條件調控和潛在表示。它會建立一個用於影片生成的空白潛在空間，並可選擇性地整合起始影像和 CLIP 視覺輸出，以引導影片生成過程。該節點會根據提供的影像和視覺資料，同時修改正向和負向的條件調控輸入。

## 輸入參數

| 參數名稱 | 資料類型 | 必填 | 數值範圍 | 參數說明 |
|-----------|-----------|----------|-------|-------------|
| `正向` | CONDITIONING | 是 | - | 用於引導生成的正向條件調控輸入 |
| `負向` | CONDITIONING | 是 | - | 用於引導生成的負向條件調控輸入 |
| `vae` | VAE | 是 | - | 用於將影像編碼至潛在空間的 VAE 模型 |
| `寬度` | INT | 是 | 16 至 MAX_RESOLUTION | 輸出影片的寬度（預設值：832，間距：16） |
| `高度` | INT | 是 | 16 至 MAX_RESOLUTION | 輸出影片的高度（預設值：480，間距：16） |
| `長度` | INT | 是 | 1 至 MAX_RESOLUTION | 影片中的影格數量（預設值：81，間距：4） |
| `批次大小` | INT | 是 | 1 至 4096 | 單次批次生成的影片數量（預設值：1） |
| `clip_vision_output` | CLIP_VISION_OUTPUT | 否 | - | 用於附加條件調控的選用 CLIP 視覺輸出 |
| `起始影像` | IMAGE | 否 | - | 用於初始化影片生成的選用起始影像 |

**注意事項：** 當提供 `start_image` 時，節點會對影像序列進行編碼，並對條件調控輸入應用遮罩處理。若提供 `clip_vision_output` 參數，則會同時向正向和負向輸入添加基於視覺的條件調控。
