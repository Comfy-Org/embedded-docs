Вот перевод документации на русский язык:

Узел WanImageToVideo подготавливает представления conditioning и latent для задач генерации видео. Он создает пустое латентное пространство для генерации видео и может опционально использовать начальные изображения и выходные данные CLIP vision для направления процесса генерации. Узел модифицирует как положительные, так и отрицательные входные данные conditioning на основе предоставленного изображения и данных vision.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `положительный` | CONDITIONING | Да | - | Положительный вход conditioning для направления генерации |
| `отрицательный` | CONDITIONING | Да | - | Отрицательный вход conditioning для направления генерации |
| `vae` | VAE | Да | - | Модель VAE для кодирования изображений в латентное пространство |
| `ширина` | INT | Да | от 16 до MAX_RESOLUTION | Ширина выходного видео (по умолчанию: 832, шаг: 16) |
| `высота` | INT | Да | от 16 до MAX_RESOLUTION | Высота выходного видео (по умолчанию: 480, шаг: 16) |
| `длина` | INT | Да | от 1 до MAX_RESOLUTION | Количество кадров в видео (по умолчанию: 81, шаг: 4) |
| `размер_пакета` | INT | Да | от 1 до 4096 | Количество видео для генерации в одном пакете (по умолчанию: 1) |
| `clip_vision_output` | CLIP_VISION_OUTPUT | Нет | - | Опциональный выход CLIP vision для дополнительного conditioning |
| `начальное_изображение` | IMAGE | Нет | - | Опциональное начальное изображение для инициализации генерации видео |

**Примечание:** Когда указан `start_image`, узел кодирует последовательность изображений и применяет маскирование к входам conditioning. Параметр `clip_vision_output`, если указан, добавляет conditioning на основе vision как к положительным, так и к отрицательным входам.

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `положительный` | CONDITIONING | Модифицированный положительный conditioning с включенными данными изображения и vision |
| `отрицательный` | CONDITIONING | Модифицированный отрицательный conditioning с включенными данными изображения и vision |
| `latent` | LATENT | Пустой тензор латентного пространства, готовый для генерации видео |
