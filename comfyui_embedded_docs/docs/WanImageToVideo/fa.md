گره WanImageToVideo، نمایش‌های شرطی (conditioning) و نهفته (latent) را برای وظایف تولید ویدیو آماده می‌کند. این گره یک فضای نهفته خالی برای تولید ویدیو ایجاد می‌کند و می‌تواند به‌صورت اختیاری تصاویر شروع و خروجی‌های CLIP vision را برای هدایت فرآیند تولید ویدیو ترکیب کند. این گره ورودی‌های شرطی مثبت و منفی را بر اساس تصویر و داده‌های بینایی ارائه‌شده اصلاح می‌کند.

## ورودی‌ها

| پارامتر | نوع داده | اجباری | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `positive` | CONDITIONING | بله | - | ورودی شرطی مثبت برای هدایت تولید |
| `negative` | CONDITIONING | بله | - | ورودی شرطی منفی برای هدایت تولید |
| `vae` | VAE | بله | - | مدل VAE برای رمزگذاری تصاویر به فضای نهفته |
| `width` | INT | بله | 16 تا MAX_RESOLUTION | عرض ویدیوی خروجی (پیش‌فرض: 832، گام: 16) |
| `height` | INT | بله | 16 تا MAX_RESOLUTION | ارتفاع ویدیوی خروجی (پیش‌فرض: 480، گام: 16) |
| `length` | INT | بله | 1 تا MAX_RESOLUTION | تعداد فریم‌های ویدیو (پیش‌فرض: 81، گام: 4) |
| `batch_size` | INT | بله | 1 تا 4096 | تعداد ویدیوهای تولیدشده در یک دسته (پیش‌فرض: 1) |
| `clip_vision_output` | CLIP_VISION_OUTPUT | خیر | - | خروجی اختیاری CLIP vision برای شرطی‌سازی اضافی |
| `start_image` | IMAGE | خیر | - | تصویر شروع اختیاری برای مقداردهی اولیه تولید ویدیو |

**توجه:** هنگامی که `start_image` ارائه می‌شود، گره دنباله تصویر را رمزگذاری کرده و ماسک‌گذاری را روی ورودی‌های شرطی اعمال می‌کند. پارامتر `clip_vision_output` نیز در صورت ارائه، شرطی‌سازی مبتنی بر بینایی را به ورودی‌های مثبت و منفی اضافه می‌کند.

## خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-------------|-----------|-------------|
| `positive` | CONDITIONING | شرطی مثبت اصلاح‌شده با داده‌های تصویر و بینایی ترکیب‌شده |
| `negative` | CONDITIONING | شرطی منفی اصلاح‌شده با داده‌های تصویر و بینایی ترکیب‌شده |
