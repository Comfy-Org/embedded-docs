Этот узел предназначен для извлечения определенного подмножества латентных сэмплов из заданного батча на основе указанного индекса и длины. Он позволяет выборочно обрабатывать латентные сэмплы, упрощая операции с меньшими сегментами батча для повышения эффективности или целенаправленного воздействия.

## Входы

| Параметр       | Тип данных | Описание |
|---------------|-------------|-------------|
| `samples`     | `LATENT`    | Коллекция латентных сэмплов, из которой будет извлечено подмножество. Этот параметр имеет решающее значение для определения исходного батча сэмплов для обработки. |
| `batch_index` | `INT`       | Указывает начальный индекс в батче, с которого начнется подмножество сэмплов. Этот параметр позволяет целенаправленно извлекать сэмплы из определенных позиций в батче. |
| `length`      | `INT`       | Определяет количество сэмплов, которые будут извлечены, начиная с указанного индекса. Этот параметр контролирует размер обрабатываемого подмножества, обеспечивая гибкое управление сегментами батча. |

## Выходы

| Параметр | Тип данных | Описание |
|-----------|-------------|-------------|
| `latent`  | `LATENT`    | Извлеченное подмножество латентных сэмплов, теперь доступное для дальнейшей обработки или анализа. |
