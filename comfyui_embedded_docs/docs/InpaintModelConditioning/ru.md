Узел `InpaintModelConditioning` предназначен для упрощения процесса кондиционирования (conditioning) в моделях для инпейнтинга (inpainting), позволяя интегрировать и обрабатывать различные входные данные кондиционирования для настройки результата инпейнтинга. Он охватывает широкий спектр функций: от загрузки конкретных контрольных точек (checkpoints) модели и применения моделей стиля или ControlNet до кодирования и комбинирования элементов кондиционирования, выступая в качестве комплексного инструмента для настройки задач инпейнтинга.

## Входные данные

| Параметр | Comfy dtype    | Описание |
|----------|----------------|----------|
| `положительный`| `CONDITIONING` | Представляет положительную информацию или параметры кондиционирования, которые применяются к модели инпейнтинга. Этот входной параметр важен для определения контекста или ограничений, в которых должна выполняться операция инпейнтинга, что существенно влияет на конечный результат. |
| `отрицательный`| `CONDITIONING` | Представляет отрицательную информацию или параметры кондиционирования, которые применяются к модели инпейнтинга. Этот входной параметр необходим для указания условий или контекстов, которых следует избегать в процессе инпейнтинга, тем самым влияя на конечный результат. |
| `vae`     | `VAE`          | Указывает модель VAE, используемую в процессе кондиционирования. Этот входной параметр важен для определения конкретной архитектуры и параметров используемой модели VAE. |
| `пиксели`  | `IMAGE`        | Представляет данные пикселей изображения, подлежащего инпейнтингу. Этот входной параметр необходим для предоставления визуального контекста, требуемого для задачи инпейнтинга. |
| `маска`    | `MASK`         | Указывает маску, применяемую к изображению, обозначающую области для инпейнтинга. Этот входной параметр важен для определения конкретных областей изображения, требующих инпейнтинга. |

## Выходные данные

| Параметр | Тип данных     | Описание |
|----------|----------------|----------|
| `положительный`| `CONDITIONING` | Модифицированная положительная информация кондиционирования после обработки, готовая к применению к модели инпейнтинга. Этот выходной параметр необходим для направления процесса инпейнтинга в соответствии с заданными положительными условиями. |
| `отрицательный`| `CONDITIONING` | Модифицированная отрицательная информация кондиционирования после обработки, готовая к применению к модели инпейнтинга. Этот выходной параметр необходим для направления процесса инпейнтинга в соответствии с заданными отрицательными условиями. |
| `latent`  | `LATENT`       | Латетное представление, полученное в результате процесса кондиционирования. Этот выходной параметр важен для понимания базовых признаков и характеристик изображения, подвергаемого инпейнтингу. |
