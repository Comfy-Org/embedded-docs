## نمای کلی

این گره به شما امکان می‌دهد عملکرد نسبی هر GPU را هنگام استفاده از چندین کارت گرافیک با سرعت‌های مختلف مشخص کنید. این گره یک گروه از گزینه‌های GPU ایجاد می‌کند که می‌توان از آن برای توزیع کار بین دستگاه‌ها استفاده کرد، اگرچه توزیع واقعی بار کاری مبتنی بر سرعت در نسخه فعلی پیاده‌سازی نشده است.

## ورودی‌ها

| پارامتر | نوع داده | اجباری | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `device_index` | INT | بله | 0 تا 64 | شماره ایندکس دستگاه GPU برای پیکربندی (پیش‌فرض: 0) |
| `relative_speed` | FLOAT | بله | 0.0 تا نامحدود | سرعت نسبی این GPU نسبت به سایرین، که برای توزیع بار کاری استفاده می‌شود (پیش‌فرض: 1.0، گام: 0.01) |
| `gpu_options` | GPU_OPTIONS | خیر | - | یک گروه گزینه‌های GPU موجود برای افزودن گزینه‌های این دستگاه به آن. در صورت عدم ارائه، یک گروه جدید ایجاد می‌شود |

## خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-------------|-----------|-------------|
| `GPU_OPTIONS` | GPU_OPTIONS | یک گروه از گزینه‌های GPU شامل تنظیمات دستگاه پیکربندی‌شده، که می‌تواند برای عملیات چند GPU به گره‌های دیگر منتقل شود |

**توجه:** پارامتر `relative_speed` تعریف شده است اما هنوز توسط زمان‌بند داخلی برای توزیع کار بین GPUها استفاده نمی‌شود. در پیاده‌سازی فعلی، کار بدون توجه به سرعت نسبی دستگاه‌ها، به طور مساوی بین همه دستگاه‌ها توزیع می‌شود.
