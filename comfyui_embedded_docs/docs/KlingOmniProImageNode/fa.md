گره تصویر کِلینگ اومنی (پرو) با استفاده از جدیدترین مدل هوش مصنوعی Kling، تصاویر را ایجاد یا ویرایش می‌کند. این گره بر اساس توضیحات متنی، تصویر تولید می‌کند و می‌تواند به‌صورت اختیاری از تصاویر مرجع برای هدایت سبک یا محتوا استفاده کند. گره یک درخواست به API خارجی ارسال می‌کند که وظیفه را پردازش کرده و تصویر(های) نهایی را بازمی‌گرداند.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
| :--- | :--- | :--- | :--- | :--- |
| `model_name` | COMBO | بله | `"kling-v3-omni"`<br>`"kling-image-o1"` | مدل خاص Kling AI که برای تولید تصویر استفاده می‌شود. |
| `prompt` | STRING | بله | - | یک توضیح متنی که محتوای تصویر را توصیف می‌کند. این می‌تواند شامل توضیحات مثبت و منفی باشد. متن باید بین ۱ تا ۲۵۰۰ کاراکتر باشد. |
| `resolution` | COMBO | بله | `"1K"`<br>`"2K"<br>`"4K"` | وضوح هدف برای تصویر تولید شده. توجه: وضوح 4K برای مدل `kling-image-o1` پشتیبانی نمی‌شود. |
| `aspect_ratio` | COMBO | بله | `"16:9"`<br>`"9:16"`<br>`"1:1"`<br>`"4:3"`<br>`"3:4"`<br>`"3:2"`<br>`"2:3"`<br>`"21:9"` | نسبت ابعاد (عرض به ارتفاع) مورد نظر برای تصویر تولید شده. |
| `تعداد سری` | COMBO | بله | `"disabled"`<br>`"2"`<br>`"3"`<br>`"4"`<br>`"5"`<br>`"6"`<br>`"7"`<br>`"8"`<br>`"9"` | تولید یک سری از تصاویر. این ویژگی برای مدل `kling-image-o1` پشتیبانی نمی‌شود. (پیش‌فرض: "غیرفعال") |
| `reference_images` | IMAGE | خیر | - | حداکثر ۱۰ تصویر مرجع اضافی. هر تصویر باید حداقل ۳۰۰ پیکسل در عرض و ارتفاع داشته باشد و نسبت ابعاد آن باید بین ۱:۲.۵ و ۲.۵:۱ باشد. |
| `seed` | INT | خیر | ۰ تا ۲۱۴۷۴۸۳۶۴۷ | seed تعیین می‌کند که آیا گره باید دوباره اجرا شود؛ نتایج بدون توجه به seed غیرقطعی هستند. (پیش‌فرض: ۰) |

## خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
| :--- | :--- | :--- |
| `image` | IMAGE | تصویر(های) نهایی تولید یا ویرایش شده توسط مدل Kling AI. اگر سری درخواست شده باشد، چندین تصویر به صورت دسته‌ای بازگردانده می‌شوند. |
