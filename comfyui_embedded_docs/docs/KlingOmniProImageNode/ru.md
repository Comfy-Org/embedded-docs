Вот перевод документации на русский язык:

Узел Kling Omni Image (Pro) создаёт или редактирует изображения с использованием новейшей модели Kling AI. Он генерирует изображения на основе текстового описания и может опционально использовать референсные изображения для управления стилем или содержимым. Узел отправляет запрос к внешнему API, который обрабатывает задачу и возвращает готовое изображение (изображения).

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
| :--- | :--- | :--- | :--- | :--- |
| `model_name` | COMBO | Да | `"kling-v3-omni"`<br>`"kling-image-o1"` | Конкретная модель Kling AI для генерации изображений. |
| `prompt` | STRING | Да | - | Текстовый запрос, описывающий содержимое изображения. Может включать как положительные, так и отрицательные описания. Текст должен содержать от 1 до 2500 символов. |
| `resolution` | COMBO | Да | `"1K"`<br>`"2K"`<br>`"4K"` | Целевое разрешение генерируемого изображения. Примечание: разрешение 4K не поддерживается для модели `kling-image-o1`. |
| `aspect_ratio` | COMBO | Да | `"16:9"`<br>`"9:16"`<br>`"1:1"`<br>`"4:3"`<br>`"3:4"`<br>`"3:2"`<br>`"2:3"`<br>`"21:9"` | Желаемое соотношение сторон (ширина к высоте) для генерируемого изображения. |
| `series_amount` | COMBO | Да | `"disabled"`<br>`"2"`<br>`"3"`<br>`"4"`<br>`"5"`<br>`"6"`<br>`"7"`<br>`"8"`<br>`"9"` | Генерация серии изображений. Эта функция не поддерживается для модели `kling-image-o1`. (по умолчанию: "disabled") |
| `reference_images` | IMAGE | Нет | - | До 10 дополнительных референсных изображений. Каждое изображение должно быть не менее 300 пикселей по ширине и высоте, а его соотношение сторон должно быть в пределах от 1:2.5 до 2.5:1. |
| `seed` | INT | Нет | от 0 до 2147483647 | Сид управляет необходимостью повторного запуска узла; результаты недетерминированы независимо от сида. (по умолчанию: 0) |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
| :--- | :--- | :--- |
| `image` | IMAGE | Готовое изображение (изображения), сгенерированное или отредактированное моделью Kling AI. Если была запрошена серия, возвращается несколько изображений в виде пакета. |
