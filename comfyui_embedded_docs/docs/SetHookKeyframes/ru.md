Вот перевод документации на русский язык:

Узел Set Hook Keyframes позволяет применить планирование по ключевым кадрам к существующим группам хуков. Он принимает группу хуков и, при необходимости, применяет информацию о времени ключевых кадров для управления моментом выполнения различных хуков в процессе генерации. Если ключевые кадры предоставлены, узел клонирует группу хуков и устанавливает временные метки ключевых кадров для всех хуков внутри группы.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `хуки` | HOOKS | Да | - | Группа хуков, к которой будет применено планирование по ключевым кадрам |
| `ключевые_кадры_хука` | HOOK_KEYFRAMES | Нет | - | Необязательная группа ключевых кадров, содержащая информацию о времени выполнения хуков |

## Выходы

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `хуки` | HOOKS | Модифицированная группа хуков с примененным планированием по ключевым кадрам (клонирована, если были предоставлены ключевые кадры) |
