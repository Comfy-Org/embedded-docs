## Обзор

Этот узел форматирует текст с использованием метода форматирования строк Python. Он работает как шаблон, где вы задаете текстовый шаблон с заполнителями, а затем предоставляете значения для их заполнения. Поддерживаются все опции и возможности форматирования Python.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `f_string` | STRING | Да | N/A | Шаблон строки форматирования с заполнителями (по умолчанию: `{a}`). Поддерживает многострочный ввод. |
| `values` | STRING | Да | N/A | Динамический ввод для предоставления значений, заполняющих заполнители в строке форматирования. При необходимости можно добавить несколько входов значений. |

**Примечание о входе `values`:** Этот вход является динамическим и может быть расширен для включения нескольких именованных значений. Каждый вход значения помечен буквой (a, b, c и т.д.) и соответствует заполнителю в строке форматирования (например, `{a}`, `{b}`, `{c}`). Вы можете добавлять или удалять входы значений по мере необходимости.

## Выходы

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `STRING` | STRING | Отформатированная текстовая строка, в которой все заполнители заменены соответствующими значениями. |
