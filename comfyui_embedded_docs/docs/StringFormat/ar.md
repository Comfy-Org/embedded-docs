نظرة عامة

تنسّق هذه العقدة النصوص باستخدام أسلوب تنسيق السلاسل النصية في بايثون. تعمل كقالب حيث تُحدد نمط النص مع عناصر نائبة، ثم تُقدم القيم لملء هذه العناصر النائبة. تدعم جميع خيارات وميزات التنسيق في بايثون.

المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|-----------|-----------|----------|-------|-------------|
| `f_string` | STRING | نعم | غير متاح | قالب سلسلة التنسيق مع العناصر النائبة (الافتراضي: `{a}`). يدعم الإدخال متعدد الأسطر. |
| `values` | STRING | نعم | غير متاح | إدخال ديناميكي لتوفير القيم لملء العناصر النائبة في سلسلة التنسيق. يمكن إضافة مدخلات قيم متعددة حسب الحاجة. |

**ملاحظة حول مدخل `values`:** هذا المدخل ديناميكي ويمكن توسيعه ليشمل قيمًا مسماة متعددة. يُسمى كل مدخل قيمة بحرف (a, b, c, إلخ) ويتوافق مع عنصر نائب في سلسلة التنسيق (مثل `{a}`، `{b}`، `{c}`). يمكنك إضافة أو إزالة مدخلات القيم حسب الحاجة.
