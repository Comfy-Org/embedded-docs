Узел RebatchImages предназначен для переорганизации пакета изображений в новую конфигурацию с изменением размера пакета в соответствии с заданными параметрами. Этот процесс необходим для управления и оптимизации обработки данных изображений в пакетных операциях, обеспечивая группировку изображений по требуемому размеру пакета для эффективной обработки.

## Входные параметры

| Поле | Тип данных | Описание |
|------|------------|----------|
| `изображения` | `IMAGE` | Список изображений для перепакетирования. Этот параметр определяет входные данные, которые будут подвергнуты процессу переорганизации. |
| `размер_пакета` | `INT` | Задает желаемый размер выходных пакетов. Этот параметр напрямую влияет на то, как входные изображения группируются и обрабатываются, определяя структуру выходных данных. |

## Выходные параметры

| Поле | Тип данных | Описание |
|------|------------|----------|
| `image` | `IMAGE` | Выходные данные представляют собой список пакетов изображений, переорганизованных в соответствии с заданным размером пакета. Это обеспечивает гибкую и эффективную обработку данных изображений в пакетных операциях. |
