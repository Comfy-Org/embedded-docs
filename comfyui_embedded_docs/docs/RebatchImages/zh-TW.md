## 概述

RebatchImages 節點旨在將一批影像重新組織為新的批次配置，並根據指定調整批次大小。此過程對於管理和優化批次操作中的影像資料處理至關重要，確保影像根據所需的批次大小進行分組，以實現高效處理。
