> این مستند با هوش مصنوعی تهیه شده است. اگر خطایی دیدید یا پیشنهادی برای بهبود دارید، خوشحال می‌شویم مشارکت کنید! [ویرایش در GitHub](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/TencentSmartTopologyNode/fa.md)

این گره، رتوپولوژی هوشمند را بر روی یک مدل سه‌بعدی انجام می‌دهد و به‌طور خودکار یک مش جدید و تمیز با تعداد چندضلعی بهینه‌شده ایجاد می‌کند. این گره برای پردازش مدل به یک API مربوط به Tencent Hunyuan 3D متصل می‌شود و از فرمت‌های فایل GLB و OBJ تا سقف ۲۰۰ مگابایت پشتیبانی می‌کند. گره، مدل پردازش‌شده را به‌عنوان یک فایل OBJ بازمی‌گرداند.

## ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `model_3d` | FILE3D | بله | - | مدل سه‌بعدی ورودی (GLB یا OBJ). فایل باید در فرمت GLB یا OBJ باشد و از ۲۰۰ مگابایت تجاوز نکند. |
| `polygon_type` | STRING | بله | `"مثلث"`<br>`"چهارضلعی"` | نوع ترکیب سطح. |
| `face_level` | STRING | بله | `"متوسط"`<br>`"زیاد"`<br>`"کم"` | سطح کاهش چندضلعی‌ها. |
| `seed` | INT | خیر | ۰ تا ۲۱۴۷۴۸۳۶۴۷ | seed مشخص می‌کند که آیا گره باید دوباره اجرا شود یا خیر؛ نتایج بدون توجه به seed غیرقطعی هستند. (پیش‌فرض: ۰) |

**توجه:** پارامتر `seed` برای راه‌اندازی اجرای مجدد گره استفاده می‌شود، اما خروجی نهایی برای یک مقدار seed یکسان تضمین نمی‌شود که یکسان باشد.
