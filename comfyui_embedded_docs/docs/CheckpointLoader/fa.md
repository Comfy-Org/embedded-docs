گره CheckpointLoader یک ایست بازرسی مدل از پیش‌آموزش‌دیده را به همراه فایل پیکربندی آن بارگذاری می‌کند. این گره یک فایل پیکربندی و یک فایل ایست بازرسی را به عنوان ورودی دریافت کرده و اجزای مدل بارگذاری‌شده شامل مدل اصلی، مدل CLIP و مدل VAE را برای استفاده در گردش کار بازمی‌گرداند.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|---------|----------|--------|--------|---------|
| `نام تنظیمات` | STRING | بله | فایل‌های پیکربندی موجود | فایل پیکربندی که معماری و تنظیمات مدل را تعریف می‌کند |
| `نام چک‌پوینت` | STRING | بله | فایل‌های ایست بازرسی موجود | فایل ایست بازرسی حاوی وزن‌ها و پارامترهای آموزش‌دیده مدل |

**توجه:** این گره نیازمند انتخاب هر دو فایل پیکربندی و فایل ایست بازرسی است. فایل پیکربندی باید با معماری فایل ایست بازرسی در حال بارگذاری مطابقت داشته باشد.

## خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-----------|----------|---------|
| `MODEL` | MODEL | مؤلفه مدل اصلی بارگذاری‌شده آماده برای استنتاج |
| `CLIP` | CLIP | مؤلفه مدل CLIP بارگذاری‌شده برای رمزگذاری متن |
| `VAE` | VAE | مؤلفه مدل VAE بارگذاری‌شده برای رمزگذاری و رمزگشایی تصویر |

**نکته مهم:** این گره به عنوان منسوخ‌شده (deprecated) علامت‌گذاری شده است و ممکن است در نسخه‌های آینده حذف شود. برای گردش کارهای جدید، از گره‌های بارگذاری جایگزین استفاده کنید.
