تستخدم عقدة `Veo3FirstLastFrameNode` نموذج Google Veo 3 لإنشاء فيديو بناءً على موجه نصي، مع توفير إطار أول وإطار أخير يحددان بداية ونهاية تسلسل الفيديو.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|-----------|-----------|----------|-------|-------------|
| `الوصف النصي` | STRING | نعم | غير متاح | وصف نصي للفيديو (القيمة الافتراضية: سلسلة فارغة). |
| `الوصف السلبي` | STRING | لا | غير متاح | موجه نصي سلبي لتوجيه ما يجب تجنبه في الفيديو (القيمة الافتراضية: سلسلة فارغة). |
| `الدقة` | COMBO | نعم | `"720p"`<br>`"1080p"`<br>`"4k"` | دقة الفيديو الناتج. |
| `نسبة العرض إلى الارتفاع` | COMBO | لا | `"16:9"`<br>`"9:16"` | نسبة العرض إلى الارتفاع للفيديو الناتج (القيمة الافتراضية: "16:9"). |
| `المدة` | INT | لا | 4 إلى 8 | مدة الفيديو الناتج بالثواني (القيمة الافتراضية: 8). |
| `البذرة` | INT | لا | 0 إلى 4294967295 | البذرة (Seed) لتوليد الفيديو (القيمة الافتراضية: 0). |
| `الإطار الأول` | IMAGE | نعم | غير متاح | إطار البداية للفيديو. |
| `الإطار الأخير` | IMAGE | نعم | غير متاح | إطار النهاية للفيديو. |
| `النموذج` | COMBO | لا | `"veo-3.1-generate"`<br>`"veo-3.1-fast-generate"`<br>`"veo-3.1-lite"` | نموذج Veo 3 المحدد لاستخدامه في التوليد (القيمة الافتراضية: "veo-3.1-generate"). |
| `توليد الصوت` | BOOLEAN | لا | غير متاح | توليد صوت للفيديو (القيمة الافتراضية: True). |

**ملاحظة:** لا يدعم نموذج `veo-3.1-lite` دقة 4K. إذا قمت بتحديد `veo-3.1-lite` ودقة `4k`، فسيحدث خطأ.
