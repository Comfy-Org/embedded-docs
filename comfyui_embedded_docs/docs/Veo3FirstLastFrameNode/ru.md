Вот перевод документации на русский язык:

Узел Veo3FirstLastFrameNode использует модель Google Veo 3 для генерации видео на основе текстового запроса, с указанными первым и последним кадрами, которые определяют начало и конец видеопоследовательности.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `prompt` | STRING | Да | Н/Д | Текстовое описание видео (по умолчанию: пустая строка). |
| `negative_prompt` | STRING | Нет | Н/Д | Негативный текстовый запрос для указания того, чего следует избегать в видео (по умолчанию: пустая строка). |
| `resolution` | COMBO | Да | `"720p"`<br>`"1080p"`<br>`"4k"` | Разрешение выходного видео. |
| `aspect_ratio` | COMBO | Нет | `"16:9"`<br>`"9:16"` | Соотношение сторон выходного видео (по умолчанию: "16:9"). |
| `duration` | INT | Нет | от 4 до 8 | Длительность выходного видео в секундах (по умолчанию: 8). |
| `seed` | INT | Нет | от 0 до 4294967295 | Зерно для генерации видео (по умолчанию: 0). |
| `first_frame` | IMAGE | Да | Н/Д | Начальный кадр видео. |
| `last_frame` | IMAGE | Да | Н/Д | Конечный кадр видео. |
| `model` | COMBO | Нет | `"veo-3.1-generate"`<br>`"veo-3.1-fast-generate"`<br>`"veo-3.1-lite"` | Конкретная модель Veo 3 для использования при генерации (по умолчанию: "veo-3.1-generate"). |
| `generate_audio` | BOOLEAN | Нет | Н/Д | Генерировать аудиодорожку для видео (по умолчанию: True). |

**Примечание:** Модель `veo-3.1-lite` не поддерживает разрешение 4K. Если вы выберете `veo-3.1-lite` и разрешение `4k`, возникнет ошибка.

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `output` | VIDEO | Сгенерированный видеофайл. |
