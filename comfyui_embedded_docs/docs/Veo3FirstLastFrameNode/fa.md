گره Veo3FirstLastFrameNode از مدل Veo 3 گوگل برای تولید ویدئو بر اساس یک متن راهنما (prompt) استفاده می‌کند، به‌طوری که فریم اول و آخر مشخص‌شده، شروع و پایان دنباله ویدئو را تعریف می‌کنند.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `پرامپت` | STRING | بله | نامشخص | توضیحات متنی ویدئو (پیش‌فرض: رشته خالی). |
| `پرامپت منفی` | STRING | خیر | نامشخص | متن راهنمای منفی برای هدایت مواردی که باید در ویدئو اجتناب شود (پیش‌فرض: رشته خالی). |
| `رزولوشن` | COMBO | بله | `"720p"`<br>`"1080p"`<br>`"4k"` | وضوح تصویر ویدئوی خروجی. |
| `نسبت تصویر` | COMBO | خیر | `"16:9"`<br>`"9:16"` | نسبت ابعاد ویدئوی خروجی (پیش‌فرض: "16:9"). |
| `مدت زمان` | INT | خیر | ۴ تا ۸ | مدت زمان ویدئوی خروجی بر حسب ثانیه (پیش‌فرض: ۸). |
| `seed` | INT | خیر | ۰ تا ۴۲۹۴۹۶۷۲۹۵ | دانه (seed) برای تولید ویدئو (پیش‌فرض: ۰). |
| `اولین فریم` | IMAGE | بله | نامشخص | فریم شروع ویدئو. |
| `آخرین فریم` | IMAGE | بله | نامشخص | فریم پایان ویدئو. |
| `مدل` | COMBO | خیر | `"veo-3.1-generate"`<br>`"veo-3.1-fast-generate"`<br>`"veo-3.1-lite"` | مدل خاص Veo 3 برای استفاده در تولید (پیش‌فرض: "veo-3.1-generate"). |
| `تولید صدا` | BOOLEAN | خیر | نامشخص | تولید صدا برای ویدئو (پیش‌فرض: True). |

**توجه:** مدل `veo-3.1-lite` از وضوح 4K پشتیبانی نمی‌کند. اگر `veo-3.1-lite` و وضوح `4k` را انتخاب کنید، خطایی رخ خواهد داد.
