# ترجمة وثيقة عقد SaveAudioMP3

يقوم عقد SaveAudioMP3 بحفظ البيانات الصوتية كملف MP3. يستقبل العقد الإدخال الصوتي ويصدره إلى دليل الإخراج المحدد مع إمكانية تخصيص اسم الملف وجودة الصوت. يتولى العقد تلقائيًا معالجة تسمية الملف وتحويل التنسيق لإنشاء ملف MP3 قابل للتشغيل.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|---------|--------------|--------|--------|-------|
| `الصوت` | AUDIO | نعم | - | البيانات الصوتية المراد حفظها كملف MP3 |
| `بادئة اسم الملف` | STRING | لا | - | البادئة لاسم ملف الإخراج (الافتراضي: "audio/ComfyUI") |
| `الجودة` | STRING | لا | "V0"<br>"128k"<br>"320k" | إعداد جودة الصوت لملف MP3 (الافتراضي: "V0") |
| `prompt` | PROMPT | لا | - | بيانات الموجه الداخلية (يتم توفيرها تلقائيًا بواسطة النظام) |
| `extra_pnginfo` | EXTRA_PNGINFO | لا | - | معلومات PNG إضافية (يتم توفيرها تلقائيًا بواسطة النظام) |

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|------------|--------------|--------|
| *لا يوجد* | - | لا يقوم هذا العقد بإرجاع أي بيانات مخرجة، بل يقوم بحفظ الملف الصوتي في دليل الإخراج |
