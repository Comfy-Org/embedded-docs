Узел SaveAudioMP3 сохраняет аудиоданные в формате MP3. Он принимает аудиовходные данные и экспортирует их в указанный выходной каталог с настраиваемым именем файла и параметрами качества. Узел автоматически обрабатывает именование файлов и преобразование формата для создания воспроизводимого MP3-файла.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `аудио` | AUDIO | Да | - | Аудиоданные, которые необходимо сохранить в виде MP3-файла |
| `префикс_имени_файла` | STRING | Нет | - | Префикс для имени выходного файла (по умолчанию: "audio/ComfyUI") |
| `качество` | STRING | Нет | "V0"<br>"128k"<br>"320k" | Настройка качества звука для MP3-файла (по умолчанию: "V0") |
| `prompt` | PROMPT | Нет | - | Внутренние данные запроса (автоматически предоставляются системой) |
| `extra_pnginfo` | EXTRA_PNGINFO | Нет | - | Дополнительная информация PNG (автоматически предоставляется системой) |

## Выходные данные

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| *Нет* | - | Этот узел не возвращает никаких выходных данных, но сохраняет аудиофайл в выходной каталог |
