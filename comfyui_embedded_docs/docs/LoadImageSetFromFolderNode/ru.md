Вот перевод документации на русский язык:

Узел LoadImageSetFromFolderNode загружает несколько изображений из указанной папки для целей обучения. Он автоматически определяет распространённые форматы изображений и может опционально изменять их размер с помощью различных методов перед возвратом в виде пакета.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `folder` | STRING | Да | Доступно несколько вариантов | Папка для загрузки изображений. |
| `resize_method` | STRING | Нет | "None"<br>"Stretch"<br>"Crop"<br>"Pad" | Метод изменения размера изображений (по умолчанию: "None"). |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `IMAGE` | IMAGE | Пакет загруженных изображений в виде единого тензора. |
