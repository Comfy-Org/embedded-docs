# ترجمة وثيقة عقدة ComfyUI

## نظرة عامة

تقوم عقدة `LoadImageSetFromFolderNode` بتحميل صور متعددة من مجلد محدد لأغراض التدريب. تكتشف هذه العقدة تلقائيًا صيغ الصور الشائعة، ويمكنها اختياريًا تغيير حجم الصور باستخدام طرق مختلفة قبل إعادتها كدفعة واحدة.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|---------|---------------|--------|---------|-------|
| `folder` | STRING | نعم | خيارات متعددة متاحة | المجلد المراد تحميل الصور منه. |
| `resize_method` | STRING | لا | "None"<br>"Stretch"<br>"Crop"<br>"Pad" | الطريقة المستخدمة لتغيير حجم الصور (الافتراضي: "None"). |

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|-------------|---------------|---------|
| `IMAGE` | IMAGE | دفعة الصور المحملة كموتر واحد. |
