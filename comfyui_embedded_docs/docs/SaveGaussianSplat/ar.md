# SaveGaussianSplat

يقوم هذا العقدة بحفظ ملف Gaussian splat ثلاثي الأبعاد في دليل المخرجات. يتولى عملية حفظ الملف ويوفر بيانات معاينة لمنفذ العرض ثلاثي الأبعاد.

## المدخلات

| المعامل | الوصف | نوع البيانات | إلزامي | النطاق |
|-----------|-------------|-----------|----------|-------|
| `model_3d` | ملف Gaussian splat ثلاثي الأبعاد. | FILE3D | نعم | SplatAny<br>PLY<br>SPLAT<br>SPZ<br>KSPLAT |
| `filename_prefix` | البادئة لاسم الملف المحفوظ (الافتراضي: "3d/ComfyUI"). | STRING | نعم | أي بادئة اسم ملف صالحة |
| `viewport_state` | حالة منفذ العرض الحالية التي تحتوي على معلومات الكاميرا والنموذج. | LOAD3D | نعم | - |
| `model_3d_info` | معلومات إضافية للنموذج ثلاثي الأبعاد لمنفذ العرض. | LOAD3DMODELINFO | لا | - |
| `camera_info` | معلومات الكاميرا لمعاينة منفذ العرض. | LOAD3DCAMERA | لا | - |
| `width` | عرض المعاينة (الافتراضي: 1024). | INT | نعم | 1 إلى 4096 |
| `height` | ارتفاع المعاينة (الافتراضي: 1024). | INT | نعم | 1 إلى 4096 |

## المخرجات

| اسم المخرج | الوصف | نوع البيانات |
|-------------|-------------|-----------|
| `model_3d` | ملف Gaussian splat ثلاثي الأبعاد المحفوظ. | FILE3D |
| `model_3d_info` | معلومات النموذج ثلاثي الأبعاد لمنفذ العرض. | LOAD3DMODELINFO |
| `camera_info` | معلومات الكاميرا لمعاينة منفذ العرض. | LOAD3DCAMERA |
| `width` | عرض المعاينة. | INT |
| `height` | ارتفاع المعاينة. | INT |
