# SaveGaussianSplat

يقوم هذا العقدة بحفظ ملف Gaussian splat ثلاثي الأبعاد في دليل الإخراج الخاص بـ ComfyUI وتوفير بيانات معاينة لعارض العرض ثلاثي الأبعاد.

## المدخلات

| المعامل | الوصف | نوع البيانات | إلزامي | النطاق |
|-----------|-------------|-----------|----------|-------|
| `model_3d` | ملف Gaussian splat ثلاثي الأبعاد. | FILE3D | نعم | - |
| `filename_prefix` | بادئة اسم الملف المحفوظ (الافتراضي: "3d/ComfyUI") | STRING | نعم | - |
| `viewport_state` | حالة منفذ العرض الحالية من عقدة تحميل ثلاثي الأبعاد. | LOAD3D | نعم | - |
| `model_3d_info` | معلومات إضافية للنموذج ثلاثي الأبعاد للمعاينة. | LOAD3DMODELINFO | لا | - |
| `camera_info` | معلومات الكاميرا للمعاينة. | LOAD3DCAMERA | لا | - |
| `width` | عرض المعاينة بالبكسل (الافتراضي: 1024) | INT | نعم | الحد الأدنى: 1، الحد الأقصى: 4096، الخطوة: 1 |
| `height` | ارتفاع المعاينة بالبكسل (الافتراضي: 1024) | INT | نعم | الحد الأدنى: 1، الحد الأقصى: 4096، الخطوة: 1 |

## المخرجات

| اسم المخرج | الوصف | نوع البيانات |
|-------------|-------------|-----------|
| `model_3d` | ملف Gaussian splat ثلاثي الأبعاد المحفوظ. | FILE3D |
| `model_3d_info` | معلومات النموذج ثلاثي الأبعاد للمعاينة. | LOAD3DMODELINFO |
| `camera_info` | معلومات الكاميرا للمعاينة. | LOAD3DCAMERA |
| `width` | عرض المعاينة. | INT |
| `height` | ارتفاع المعاينة. | INT |
