# SaveGaussianSplat

این گره یک فایل سه‌بعدی Gaussian splat را در دایرکتوری خروجی ComfyUI ذخیره می‌کند و داده‌های پیش‌نمایش را برای نمایشگر سه‌بعدی فراهم می‌کند.

## ورودی‌ها

| پارامتر | توضیحات | نوع داده | الزامی | محدوده |
|-----------|-------------|-----------|----------|-------|
| `model_3d` | یک فایل سه‌بعدی Gaussian splat. | FILE3D | بله | - |
| `filename_prefix` | پیشوند نام فایل ذخیره‌شده (پیش‌فرض: "3d/ComfyUI") | STRING | بله | - |
| `viewport_state` | وضعیت فعلی نمای دید از یک گره بارگذار سه‌بعدی. | LOAD3D | بله | - |
| `model_3d_info` | اطلاعات اضافی مدل سه‌بعدی برای پیش‌نمایش. | LOAD3DMODELINFO | خیر | - |
| `camera_info` | اطلاعات دوربین برای پیش‌نمایش. | LOAD3DCAMERA | خیر | - |
| `width` | عرض پیش‌نمایش بر حسب پیکسل (پیش‌فرض: 1024) | INT | بله | حداقل: 1، حداکثر: 4096، گام: 1 |
| `height` | ارتفاع پیش‌نمایش بر حسب پیکسل (پیش‌فرض: 1024) | INT | بله | حداقل: 1، حداکثر: 4096، گام: 1 |

## خروجی‌ها

| نام خروجی | توضیحات | نوع داده |
|-------------|-------------|-----------|
| `model_3d` | فایل سه‌بعدی Gaussian splat ذخیره‌شده. | FILE3D |
| `model_3d_info` | اطلاعات مدل سه‌بعدی برای پیش‌نمایش. | LOAD3DMODELINFO |
| `camera_info` | اطلاعات دوربین برای پیش‌نمایش. | LOAD3DCAMERA |
| `width` | عرض پیش‌نمایش. | INT |
| `height` | ارتفاع پیش‌نمایش. | INT |

> این مستند با هوش مصنوعی تهیه شده است. اگر خطایی دیدید یا پیشنهادی برای بهبود دارید، خوشحال می‌شویم مشارکت کنید! [ویرایش در GitHub](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/SaveGaussianSplat/fa.md)
