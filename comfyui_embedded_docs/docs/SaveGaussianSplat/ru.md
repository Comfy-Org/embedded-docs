# SaveGaussianSplat

Этот узел сохраняет 3D-файл гауссовского сплаттинга в выходной каталог ComfyUI и предоставляет данные предварительного просмотра для 3D-вьювера.

## Входы

| Параметр | Описание | Тип данных | Обязательный | Диапазон |
|----------|----------|------------|--------------|----------|
| `model_3d` | 3D-файл гауссовского сплаттинга. | FILE3D | Да | - |
| `filename_prefix` | Префикс для имени сохраняемого файла (по умолчанию: "3d/ComfyUI") | STRING | Да | - |
| `viewport_state` | Текущее состояние области просмотра из узла загрузчика 3D. | LOAD3D | Да | - |
| `model_3d_info` | Дополнительная информация о 3D-модели для предварительного просмотра. | LOAD3DMODELINFO | Нет | - |
| `camera_info` | Информация о камере для предварительного просмотра. | LOAD3DCAMERA | Нет | - |
| `width` | Ширина предварительного просмотра в пикселях (по умолчанию: 1024) | INT | Да | min: 1, max: 4096, step: 1 |
| `height` | Высота предварительного просмотра в пикселях (по умолчанию: 1024) | INT | Да | min: 1, max: 4096, step: 1 |

## Выходы

| Имя выхода | Описание | Тип данных |
|------------|----------|------------|
| `model_3d` | Сохраненный 3D-файл гауссовского сплаттинга. | FILE3D |
| `model_3d_info` | Информация о 3D-модели для предварительного просмотра. | LOAD3DMODELINFO |
| `camera_info` | Информация о камере для предварительного просмотра. | LOAD3DCAMERA |
| `width` | Ширина предварительного просмотра. | INT |
| `height` | Высота предварительного просмотра. | INT |
