# عقدة تحويل الصورة إلى نموذج ثلاثي الأبعاد من Tencent

تقوم هذه العقدة باستخدام واجهة برمجة تطبيقات Hunyuan3D Pro من Tencent لإنشاء نموذج ثلاثي الأبعاد من صورة واحدة أو أكثر من الصور المدخلة. تقوم بمعالجة الصور، وإرسالها إلى واجهة البرمجة، وإرجاع ملفات النموذج ثلاثي الأبعاد المُنشأة بتنسيقي GLB وOBJ، بالإضافة إلى خرائط النسيج الاختيارية.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|---------|---------------|--------|--------|-------|
| `model` | COMBO | نعم | `"3.0"`<br>`"3.1"` | إصدار نموذج Hunyuan3D المراد استخدامه. خيار المضلعات المنخفضة غير متاح لإصدار `3.1`. |
| `image` | IMAGE | نعم | - | الصورة المدخلة الأساسية المستخدمة لإنشاء النموذج ثلاثي الأبعاد. يجب أن لا يقل حجمها عن 128×128 بكسل. |
| `image_left` | IMAGE | لا | - | صورة اختيارية للجانب الأيسر من الجسم لإنشاء عرض متعدد الزوايا. يجب أن لا يقل حجمها عن 128×128 بكسل. |
| `image_right` | IMAGE | لا | - | صورة اختيارية للجانب الأيمن من الجسم لإنشاء عرض متعدد الزوايا. يجب أن لا يقل حجمها عن 128×128 بكسل. |
| `image_back` | IMAGE | لا | - | صورة اختيارية للجانب الخلفي من الجسم لإنشاء عرض متعدد الزوايا. يجب أن لا يقل حجمها عن 128×128 بكسل. |
| `face_count` | INT | نعم | 3000 - 1500000 | العدد المستهدف من الأسطح للنموذج ثلاثي الأبعاد المُنشأ (الافتراضي: 500000). |
| `generate_type` | DYNAMICCOMBO | نعم | `"Normal"`<br>`"LowPoly"`<br>`"Geometry"` | نوع النموذج ثلاثي الأبعاد المراد إنشاؤه. يؤدي اختيار خيار إلى إظهار معاملات إضافية ذات صلة. |
| `generate_type.pbr` | BOOLEAN | لا | - | يُفعّل إنشاء مواد العرض الفيزيائي الواقعي (PBR). هذا المعامل يظهر فقط عندما تكون قيمة `generate_type` مضبوطة على "Normal" أو "LowPoly" (الافتراضي: False). |
| `generate_type.polygon_type` | COMBO | لا | `"triangle"`<br>`"quadrilateral"` | نوع المضلع المستخدم للشبكة. هذا المعامل يظهر فقط عندما تكون قيمة `generate_type` مضبوطة على "LowPoly". |
| `seed` | INT | نعم | 0 - 2147483647 | قيمة بذرة لعملية الإنشاء. تتحكم البذرة في ما إذا كانت العقدة ستعيد التشغيل؛ النتائج غير حتمية بغض النظر عن البذرة (الافتراضي: 0). |

**ملاحظة:** يجب أن لا يقل الحد الأدنى للعرض والارتفاع لجميع الصور المدخلة عن 128 بكسل. يتم تقليل حجم الصور تلقائياً إذا تجاوز طول أطول جانب فيها 4900 بكسل.

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|-------------|---------------|--------|
| `model_file` | STRING | مخرج قديم للتوافق مع الإصدارات السابقة. |
| `GLB` | FILE3DGLB | النموذج ثلاثي الأبعاد المُنشأ بتنسيق ملف GLB (تنسيق نقل GL الثنائي). |
| `OBJ` | FILE3DOBJ | النموذج ثلاثي الأبعاد المُنشأ بتنسيق ملف OBJ (Wavefront). |
| `texture_image` | IMAGE | صورة النسيج للنموذج ثلاثي الأبعاد المُنشأ. |
| `optional_metallic` | IMAGE | خريطة المعدنية لمواد العرض الفيزيائي الواقعي. تُرجع صورة سوداء إذا لم تكن متوفرة. |
| `optional_normal` | IMAGE | خريطة العمودية لمواد العرض الفيزيائي الواقعي. تُرجع صورة سوداء إذا لم تكن متوفرة. |
| `optional_roughness` | IMAGE | خريطة الخشونة لمواد العرض الفيزيائي الواقعي. تُرجع صورة سوداء إذا لم تكن متوفرة. |
