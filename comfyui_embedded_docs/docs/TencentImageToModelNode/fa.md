این گره از API پروانه‌ی Hunyuan3D شرکت Tencent برای تولید مدل سه‌بعدی از یک یا چند تصویر ورودی استفاده می‌کند. تصاویر را پردازش کرده، به API ارسال می‌کند و فایل‌های مدل سه‌بعدی تولیدشده را در قالب‌های GLB و OBJ به همراه نقشه‌های بافت اختیاری بازمی‌گرداند.

## ورودی‌ها

| پارامتر | نوع داده | اجباری | محدوده | توضیحات |
|---------|----------|--------|--------|---------|
| `model` | COMBO | بله | `"3.0"`<br>`"3.1"` | نسخه مدل Hunyuan3D که باید استفاده شود. گزینه LowPoly برای مدل `3.1` در دسترس نیست. |
| `image` | IMAGE | بله | - | تصویر ورودی اصلی برای تولید مدل سه‌بعدی. حداقل باید 128×128 پیکسل باشد. |
| `image_left` | IMAGE | خیر | - | تصویر اختیاری از سمت چپ شیء برای تولید چندنمایی. حداقل باید 128×128 پیکسل باشد. |
| `image_right` | IMAGE | خیر | - | تصویر اختیاری از سمت راست شیء برای تولید چندنمایی. حداقل باید 128×128 پیکسل باشد. |
| `image_back` | IMAGE | خیر | - | تصویر اختیاری از پشت شیء برای تولید چندنمایی. حداقل باید 128×128 پیکسل باشد. |
| `face_count` | INT | بله | 3000 - 1500000 | تعداد وجه‌های هدف برای مدل سه‌بعدی تولیدشده (پیش‌فرض: 500000). |
| `generate_type` | DYNAMICCOMBO | بله | `"Normal"`<br>`"LowPoly"`<br>`"Geometry"` | نوع مدل سه‌بعدی که باید تولید شود. انتخاب یک گزینه، پارامترهای مرتبط اضافی را نمایش می‌دهد. |
| `generate_type.pbr` | BOOLEAN | خیر | - | تولید متریال مبتنی بر رندر فیزیکی (PBR) را فعال می‌کند. این پارامتر فقط زمانی قابل مشاهده است که `generate_type` روی "Normal" یا "LowPoly" تنظیم شده باشد (پیش‌فرض: False). |
| `generate_type.polygon_type` | COMBO | خیر | `"triangle"`<br>`"quadrilateral"` | نوع چندضلعی برای مش. این پارامتر فقط زمانی قابل مشاهده است که `generate_type` روی "LowPoly" تنظیم شده باشد. |
| `seed` | INT | بله | 0 - 2147483647 | یک مقدار seed برای فرآیند تولید. Seed تعیین می‌کند که آیا گره باید دوباره اجرا شود یا خیر؛ نتایج صرف‌نظر از seed غیرقطعی هستند (پیش‌فرض: 0). |

**توجه:** تمام تصاویر ورودی باید حداقل عرض و ارتفاع 128 پیکسل داشته باشند. اگر طولانی‌ترین ضلع تصاویر از 4900 پیکسل بیشتر شود، به‌طور خودکار کوچک‌سازی می‌شوند.

## خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-----------|----------|---------|
| `model_file` | STRING | یک خروجی قدیمی برای سازگاری با نسخه‌های قبلی. |
| `GLB` | FILE3DGLB | مدل سه‌بعدی تولیدشده در قالب فایل GLB (فرمت انتقال باینری GL). |
| `OBJ` | FILE3DOBJ | مدل سه‌بعدی تولیدشده در قالب فایل OBJ (Wavefront). |
| `texture_image` | IMAGE | تصویر بافت برای مدل سه‌بعدی تولیدشده. |
| `optional_metallic` | IMAGE | نقشه فلزینگی برای متریال‌های PBR. در صورت عدم وجود، یک تصویر سیاه بازمی‌گرداند. |
| `optional_normal` | IMAGE | نقشه نرمال برای متریال‌های PBR. در صورت عدم وجود، یک تصویر سیاه بازمی‌گرداند. |
| `optional_roughness` | IMAGE | نقشه زبری برای متریال‌های PBR. در صورت عدم وجود، یک تصویر سیاه بازمی‌گرداند. |
