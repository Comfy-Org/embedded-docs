Узел PairConditioningSetPropertiesAndCombine модифицирует и объединяет пары кондиционирования, применяя новые данные кондиционирования к существующим позитивным и негативным входам кондиционирования. Он позволяет регулировать силу применяемого кондиционирования и контролировать способ установки области кондиционирования. Этот узел особенно полезен для сложных рабочих процессов манипуляции кондиционированием, где требуется объединить несколько источников кондиционирования вместе.

## Входы

| Параметр | Тип данных | Обязательно | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `positive` | CONDITIONING | Да | - | Исходный позитивный вход кондиционирования |
| `negative` | CONDITIONING | Да | - | Исходный негативный вход кондиционирования |
| `positive_NEW` | CONDITIONING | Да | - | Новое позитивное кондиционирование для применения |
| `negative_NEW` | CONDITIONING | Да | - | Новое негативное кондиционирование для применения |
| `strength` | FLOAT | Да | 0.0 до 10.0 | Коэффициент силы для применения нового кондиционирования (по умолчанию: 1.0) |
| `set_cond_area` | STRING | Да | "default"<br>"mask bounds" | Управляет способом применения области кондиционирования |
| `mask` | MASK | Нет | - | Опциональная маска для ограничения области применения кондиционирования |
| `hooks` | HOOKS | Нет | - | Опциональная группа хуков для расширенного контроля |
| `timesteps` | TIMESTEPS_RANGE | Нет | - | Опциональная спецификация диапазона временных шагов |

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `positive` | CONDITIONING | Объединенный позитивный выход кондиционирования |
| `negative` | CONDITIONING | Объединенный негативный выход кондиционирования |
