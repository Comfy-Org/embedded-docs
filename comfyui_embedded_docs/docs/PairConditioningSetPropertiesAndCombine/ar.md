هذه هي الترجمة العربية للوثيقة التقنية الخاصة بعقدة ComfyUI، مع الالتزام التام بقواعد الترجمة المحددة:

تقوم عقدة `PairConditioningSetPropertiesAndCombine` بتعديل ودمج أزواج التكييف من خلال تطبيق بيانات تكييف جديدة على مدخلات التكييف الإيجابي والسلبي الحالية. تتيح لك ضبط قوة التكييف المُطبق والتحكم في كيفية تعيين منطقة التكييف. هذه العقدة مفيدة بشكل خاص لسير عمل معالجة التكييف المتقدمة حيث تحتاج إلى مزج مصادر تكييف متعددة معًا.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|-----------|-----------|----------|-------|-------------|
| `إيجابي` | CONDITIONING | نعم | - | مدخل التكييف الإيجابي الأصلي |
| `سلبي` | CONDITIONING | نعم | - | مدخل التكييف السلبي الأصلي |
| `إيجابي جديد` | CONDITIONING | نعم | - | التكييف الإيجابي الجديد المراد تطبيقه |
| `سلبي جديد` | CONDITIONING | نعم | - | التكييف السلبي الجديد المراد تطبيقه |
| `القوة` | FLOAT | نعم | 0.0 إلى 10.0 | عامل القوة لتطبيق التكييف الجديد (الافتراضي: 1.0) |
| `تعيين منطقة الشرط` | STRING | نعم | "default"<br>"mask bounds" | يتحكم في كيفية تطبيق منطقة التكييف (الافتراضي: "default") |
| `القناع` | MASK | لا | - | قناع اختياري لتقييد منطقة تطبيق التكييف |
| `خطافات` | HOOKS | لا | - | مجموعة خطاف اختيارية للتحكم المتقدم |
| `خطوات الزمن` | TIMESTEPS_RANGE | لا | - | مواصفات نطاق الخطوات الزمنية الاختيارية |
