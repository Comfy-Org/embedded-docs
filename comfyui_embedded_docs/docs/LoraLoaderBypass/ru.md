Вот перевод документации на русский язык:

Узел LoraLoaderBypass применяет LoRA (низкоранговую адаптацию) к диффузионной модели и модели CLIP в специальном режиме "обхода" (bypass). В отличие от стандартного загрузчика LoRA, этот метод не изменяет веса базовой модели постоянно. Вместо этого он вычисляет выходные данные, добавляя эффект LoRA к обычному прямому проходу модели, что полезно для обучения или при работе с моделями, чьи веса выгружены.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `model` | MODEL | Да | - | Диффузионная модель, к которой будет применена LoRA. |
| `clip` | CLIP | Да | - | Модель CLIP, к которой будет применена LoRA. |
| `lora_name` | COMBO | Да | *Список доступных файлов LoRA* | Имя файла LoRA для применения. Варианты загружаются из папки `loras`. |
| `strength_model` | FLOAT | Да | от -100.0 до 100.0 | Насколько сильно изменять диффузионную модель. Это значение может быть отрицательным (по умолчанию: 1.0). |
| `strength_clip` | FLOAT | Да | от -100.0 до 100.0 | Насколько сильно изменять модель CLIP. Это значение может быть отрицательным (по умолчанию: 1.0). |

**Примечание:** Если оба параметра `strength_model` и `strength_clip` установлены в 0, узел вернет исходные, неизмененные входные данные `model` и `clip` без обработки.

## Выходные параметры

| Имя выходного параметра | Тип данных | Описание |
|-------------|-----------|-------------|
| `MODEL` | MODEL | Диффузионная модель с примененной LoRA в режиме обхода. |
| `CLIP` | CLIP | Модель CLIP с примененной LoRA в режиме обхода. |
