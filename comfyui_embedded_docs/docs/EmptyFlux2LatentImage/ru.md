Узел EmptyFlux2LatentImage создает пустое латентное представление. Он генерирует тензор, заполненный нулями, который служит отправной точкой для процесса шумоподавления модели Flux. Размеры латентного пространства определяются входными значениями ширины и высоты, уменьшенными в 16 раз.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `width` | INT | Да | от 16 до 8192 | Ширина итогового изображения. Ширина латентного пространства будет равна этому значению, деленному на 16. Значение по умолчанию: 1024. |
| `height` | INT | Да | от 16 до 8192 | Высота итогового изображения. Высота латентного пространства будет равна этому значению, деленному на 16. Значение по умолчанию: 1024. |
| `batch_size` | INT | Нет | от 1 до 4096 | Количество латентных образцов, генерируемых за один пакет. Значение по умолчанию: 1. |

**Примечание:** Входные значения `width` и `height` должны быть кратны 16, так как узел внутренне делит их на этот коэффициент для создания размеров латентного пространства.

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `samples` | LATENT | Латентный тензор, заполненный нулями. Форма: `[batch_size, 128, height // 16, width // 16]`. |
