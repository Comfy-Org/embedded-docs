فایل‌های ورودی را برای استفاده با API جمینای بارگذاری و قالب‌بندی می‌کند. این گره به کاربران امکان می‌دهد فایل‌های متنی (.txt) و PDF (.pdf) را به عنوان زمینه ورودی برای مدل جمینای وارد کنند. فایل‌ها به فرمت مناسب مورد نیاز API تبدیل می‌شوند و می‌توانند به صورت زنجیره‌ای به هم متصل شوند تا چندین فایل در یک درخواست واحد گنجانده شوند.

## ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|---------|----------|-------|--------|---------|
| `file` | COMBO | بله | چندین گزینه موجود است | فایل‌های ورودی برای استفاده به عنوان زمینه مدل. در حال حاضر فقط فایل‌های متنی (.txt) و PDF (.pdf) را می‌پذیرد. فایل‌ها باید کوچک‌تر از حداکثر محدودیت اندازه فایل ورودی باشند. |
| `GEMINI_INPUT_FILES` | GEMINI_INPUT_FILES | خیر | نامعتبر | یک فایل (یا فایل‌های) اضافی اختیاری برای دسته‌بندی همراه با فایل بارگذاری‌شده از این گره. امکان زنجیره‌سازی فایل‌های ورودی را فراهم می‌کند تا یک پیام واحد بتواند شامل چندین فایل ورودی باشد. |

**توجه:** پارامتر `file` فقط فایل‌های متنی (.txt) و PDF (.pdf) را نمایش می‌دهد که کوچک‌تر از حداکثر محدودیت اندازه فایل ورودی هستند. فایل‌ها به طور خودکار فیلتر و بر اساس نام مرتب می‌شوند.
