Вот перевод документации на русский язык:

Узел TextGenerateLTX2Prompt — это специализированная версия узла генерации текста. Он принимает текстовый запрос пользователя и автоматически форматирует его с помощью специальных системных инструкций перед отправкой языковой модели для улучшения или завершения. Узел может работать в двух режимах: только текст или с привязкой к изображению, используя разные системные промпты для каждого случая.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `clip` | CLIP | Да | | Модель CLIP, используемая для кодирования текста. |
| `подсказка` | STRING | Да | | Исходный текстовый ввод от пользователя, который будет улучшен или завершён. |
| `максимальная длина` | INT | Да | | Максимальное количество токенов, которое языковая модель может сгенерировать. |
| `режим выборки` | COMBO | Да | `"greedy"`<br>`"top_k"`<br>`"top_p"`<br>`"temperature"` | Стратегия сэмплирования, используемая для выбора следующего токена при генерации текста. |
| `изображение` | IMAGE | Нет | | Необязательное входное изображение. При его наличии узел использует другой системный промпт, содержащий заполнитель для контекста изображения. |
| `мышление` | BOOLEAN | Нет | | При включении модель выводит свой процесс рассуждения перед окончательным ответом. |
| `use_default_template` | BOOLEAN | Нет | | При включении узел использует стандартный шаблон чата для форматирования. |
| `видео` | VIDEO | Нет | | Необязательный видеовход, который может использоваться как дополнительный контекст для генерации. |
| `аудио` | AUDIO | Нет | | Необязательный аудиовход, который может использоваться как дополнительный контекст для генерации. |

**Примечание:** Поведение узла меняется в зависимости от наличия входного параметра `image`. Если изображение предоставлено, сгенерированный промпт будет отформатирован для задачи преобразования изображения в видео. Если изображение отсутствует, форматирование будет выполнено для задачи преобразования текста в видео.

## Выходные данные

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `output` | STRING | Улучшенная или завершённая текстовая строка, сгенерированная языковой моделью. |
