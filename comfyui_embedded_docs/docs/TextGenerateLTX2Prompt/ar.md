عقدة `TextGenerateLTX2Prompt` هي نسخة متخصصة من عقدة توليد النصوص. تأخذ هذه العقدة مطالبة نصية من المستخدم وتقوم بتنسيقها تلقائيًا مع تعليمات نظام محددة قبل إرسالها إلى نموذج اللغة لتحسينها أو إكمالها. يمكن للعقدة العمل في وضعين: نص فقط أو مع مرجع صورة، وذلك باستخدام مطالبات نظام مختلفة لكل حالة.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|-----------|-----------|----------|-------|-------------|
| `clip` | CLIP | نعم | | نموذج CLIP المستخدم لتشفير النص. |
| `المطالبة` | STRING | نعم | | الإدخال النصي الخام من المستخدم والذي سيتم تحسينه أو إكماله. |
| `الحد الأقصى للطول` | INT | نعم | | الحد الأقصى لعدد الرموز (tokens) المسموح لنموذج اللغة بتوليدها. |
| `وضع العينة` | COMBO | نعم | `"greedy"`<br>`"top_k"`<br>`"top_p"`<br>`"temperature"` | استراتيجية أخذ العينات المستخدمة لاختيار الرمز التالي أثناء توليد النص. |
| `الصورة` | IMAGE | لا | | صورة إدخال اختيارية. عند توفيرها، تستخدم العقدة مطالبة نظام مختلفة تتضمن حاملًا لسياق الصورة. |
| `التفكير` | BOOLEAN | لا | | عند التفعيل، سيقوم النموذج بإخراج عملية التفكير الخاصة به قبل الإجابة النهائية. |
| `استخدام القالب الافتراضي` | BOOLEAN | لا | | عند التفعيل، ستستخدم العقدة قالب الدردشة الافتراضي للتنسيق. |
| `فيديو` | VIDEO | لا | | إدخال فيديو اختياري يمكن استخدامه كسياق إضافي للتوليد. |
| `صوت` | AUDIO | لا | | إدخال صوتي اختياري يمكن استخدامه كسياق إضافي للتوليد. |

**ملاحظة:** يتغير سلوك العقدة بناءً على وجود مدخل `image`. إذا تم توفير صورة، فسيتم تنسيق المطالبة المُنشأة لمهمة تحويل الصورة إلى فيديو. إذا لم يتم توفير صورة، فسيكون التنسيق لمهمة تحويل النص إلى فيديو.
