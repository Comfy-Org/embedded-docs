گره TextGenerateLTX2Prompt یک نسخه تخصصی از گره تولید متن است. این گره، متن ورودی کاربر را دریافت کرده و به‌طور خودکار آن را با دستورالعمل‌های سیستمی خاص قالب‌بندی می‌کند، سپس برای بهبود یا تکمیل به مدل زبانی ارسال می‌کند. این گره می‌تواند در دو حالت کار کند: فقط متن یا همراه با تصویر مرجع، که برای هر حالت از پرامپت‌های سیستمی متفاوتی استفاده می‌کند.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|---------|----------|--------|--------|---------|
| `clip` | CLIP | بله | | مدل CLIP مورد استفاده برای رمزگذاری متن. |
| `پرامپت` | STRING | بله | | متن خام ورودی از کاربر که بهبود یا تکمیل خواهد شد. |
| `حداکثر طول` | INT | بله | | حداکثر تعداد توکن‌هایی که مدل زبانی مجاز به تولید است. |
| `حالت نمونه‌گیری` | COMBO | بله | `"greedy"`<br>`"top_k"`<br>`"top_p"`<br>`"temperature"` | استراتژی نمونه‌برداری برای انتخاب توکن بعدی در طول تولید متن. |
| `تصویر` | IMAGE | خیر | | یک تصویر ورودی اختیاری. در صورت ارائه، گره از پرامپت سیستمی متفاوتی استفاده می‌کند که شامل جایگزینی برای زمینه تصویر است. |
| `تفکر` | BOOLEAN | خیر | | در صورت فعال‌سازی، مدل فرآیند استدلال خود را قبل از پاسخ نهایی خروجی می‌دهد. |
| `use_default_template` | BOOLEAN | خیر | | در صورت فعال‌سازی، گره از قالب چت پیش‌فرض برای قالب‌بندی استفاده می‌کند. |
| `ویدیو` | VIDEO | خیر | | یک ورودی ویدیویی اختیاری که می‌تواند به‌عنوان زمینه اضافی برای تولید استفاده شود. |
| `صدا` | AUDIO | خیر | | یک ورودی صوتی اختیاری که می‌تواند به‌عنوان زمینه اضافی برای تولید استفاده شود. |

**توجه:** رفتار گره بر اساس وجود ورودی `image` تغییر می‌کند. اگر تصویری ارائه شود، پرامپت تولید شده برای یک وظیفه تصویر به ویدیو قالب‌بندی می‌شود. اگر تصویری ارائه نشود، قالب‌بندی برای یک وظیفه متن به ویدیو خواهد بود.
