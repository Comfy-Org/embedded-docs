گره SetUnionControlNetType به شما امکان می‌دهد نوع شبکه کنترلی مورد استفاده برای شرطی‌سازی را مشخص کنید. این گره یک شبکه کنترلی موجود را دریافت کرده و نوع آن را بر اساس انتخاب شما تنظیم می‌کند و یک کپی اصلاح‌شده از شبکه کنترلی با پیکربندی نوع مشخص‌شده ایجاد می‌کند.

## ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|---------|----------|-------|--------|---------|
| `control_net` | CONTROL_NET | بله | - | شبکه کنترلی که باید با یک تنظیم نوع جدید اصلاح شود |
| `type` | STRING | بله | `"auto"`<br>تمام کلیدهای موجود UNION_CONTROLNET_TYPES | نوع شبکه کنترلی برای اعمال. از "auto" برای تشخیص خودکار نوع استفاده کنید یا یک نوع شبکه کنترلی خاص را از گزینه‌های موجود انتخاب نمایید |
