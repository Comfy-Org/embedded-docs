Узел SetUnionControlNetType позволяет указать тип сети управления, используемой для кондиционирования. Он принимает существующую сеть управления и устанавливает её тип на основе вашего выбора, создавая модифицированную копию сети управления с заданной конфигурацией типа.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `control_net` | CONTROL_NET | Да | - | Сеть управления, которую необходимо изменить с новым типом |
| `type` | STRING | Да | `"auto"`<br>Все доступные ключи UNION_CONTROLNET_TYPES | Тип сети управления для применения. Используйте "auto" для автоматического определения типа или выберите конкретный тип сети управления из доступных опций |

## Выходы

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `control_net` | CONTROL_NET | Модифицированная сеть управления с применённым заданным типом |
