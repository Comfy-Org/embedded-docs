## Обзор

Узел Krea 2 Style Reference позволяет добавить референсное изображение для влияния на стиль генерации изображений Krea 2. Вы можете объединять несколько стилевых референсов в цепочку (до 10 штук) и передавать объединённый результат в узел Krea 2 Image. Каждое предоставленное изображение загружается в хранилище ComfyAPI и передаётся в виде URL-адреса.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `image` | IMAGE | Да | - | Референсное изображение, стиль которого влияет на генерацию. |
| `strength` | FLOAT | Да | от -2.0 до 2.0 (шаг: 0.05) | Сила влияния референса; отрицательные значения инвертируют влияние стиля (по умолчанию: 1.0). |
| `style_reference` | STYLE_REF | Нет | - | Опциональная входящая цепочка стилевых референсов; данный узел добавляет ещё один элемент. |

**Примечание об ограничениях:** В общей сложности можно объединить не более 10 стилевых референсов в цепочку. При попытке добавить 11-й референс узел выдаст ошибку.

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `style_reference` | STYLE_REF | Список записей стилевых референсов, каждая из которых содержит URL-адрес и значение силы влияния. Передайте этот выход в узел Krea 2 Image. |
