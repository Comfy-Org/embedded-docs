Вот перевод документации на русский язык:

Узел ComboOptionTestNode — это логический узел, предназначенный для тестирования и передачи выбранных значений из выпадающих списков. Он принимает два входа в виде выпадающих списков, каждый с предопределённым набором опций, и выводит выбранные значения напрямую, без изменений.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `combo` | COMBO | Да | `"option1"`<br>`"option2"`<br>`"option3"` | Первый выбор из набора трёх тестовых опций. |
| `combo2` | COMBO | Да | `"option4"`<br>`"option5"`<br>`"option6"` | Второй выбор из другого набора трёх тестовых опций. |

## Выходы

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `output_1` | COMBO | Выводит значение, выбранное из первого выпадающего списка (`combo`). |
| `output_2` | COMBO | Выводит значение, выбранное из второго выпадающего списка (`combo2`). |
