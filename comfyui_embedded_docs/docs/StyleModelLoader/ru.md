Этот узел обнаруживает модели, расположенные в папке `ComfyUI/models/style_models`, а также считывает модели из дополнительных путей, указанных в файле extra_model_paths.yaml. Иногда может потребоваться **обновить интерфейс ComfyUI**, чтобы он смог прочитать файлы моделей из соответствующей папки.

Узел StyleModelLoader предназначен для загрузки модели стиля из указанного пути. Его основная задача — поиск и инициализация моделей стилей, которые можно использовать для применения определённых художественных стилей к изображениям, что позволяет настраивать визуальные результаты на основе загруженной модели стиля.

## Входные параметры

| Имя параметра      | Comfy dtype     | Python dtype | Описание                                                                                       |
|---------------------|-----------------|--------------|---------------------------------------------------------------------------------------------------|
| `style_model_name`  | COMBO[STRING] | `str`        | Указывает имя загружаемой модели стиля. Это имя используется для поиска файла модели в заданной структуре каталогов, что позволяет динамически загружать различные модели стилей в зависимости от ввода пользователя или потребностей приложения. |

## Выходные параметры

| Имя параметра  | Comfy dtype   | Python dtype | Описание                                                                                       |
|-----------------|---------------|--------------|---------------------------------------------------------------------------------------------------|
| `style_model`   | `STYLE_MODEL` | `StyleModel` | Возвращает загруженную модель стиля, готовую к применению стилей к изображениям. Это обеспечивает динамическую настройку визуальных результатов путём применения различных художественных стилей. |
