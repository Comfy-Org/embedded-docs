Узел PrimitiveBoundingBox создает простую прямоугольную область, определяемую её положением и размером. Он принимает координаты X и Y для верхнего левого угла, а также значения ширины и высоты, и выводит структуру данных ограничивающего прямоугольника, которая может использоваться другими узлами в рабочем процессе.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `x` | INT | Да | от 0 до 8192 | Координата X верхнего левого угла ограничивающего прямоугольника (по умолчанию: 0). |
| `y` | INT | Да | от 0 до 8192 | Координата Y верхнего левого угла ограничивающего прямоугольника (по умолчанию: 0). |
| `ширина` | INT | Да | от 1 до 8192 | Ширина ограничивающего прямоугольника (по умолчанию: 512). |
| `высота` | INT | Да | от 1 до 8192 | Высота ограничивающего прямоугольника (по умолчанию: 512). |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `bounding_box` | BOUNDING_BOX | Структура данных, содержащая свойства `x`, `y`, `ширина` и `высота` заданного прямоугольника. |
