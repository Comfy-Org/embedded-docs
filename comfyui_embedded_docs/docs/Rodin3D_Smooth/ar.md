بصفتي خبيرًا في الترجمة التقنية المتخصصة في توثيق عُقد ComfyUI، إليك الترجمة العربية للوثيقة المطلوبة:

## نظرة عامة

تقوم عقدة Rodin 3D Smooth بتوليد أصول ثلاثية الأبعاد باستخدام واجهة برمجة تطبيقات Rodin من خلال معالجة الصور المدخلة وتحويلها إلى نماذج ثلاثية الأبعاد ملساء. تستقبل العقدة صورًا متعددة كمدخلات وتنتج ملف نموذج ثلاثي الأبعاد قابل للتنزيل. تدير العقدة عملية التوليد بأكملها بما في ذلك إنشاء المهمة، والاستعلام عن الحالة، وتنزيل الملفات تلقائيًا.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|---------|--------------|--------|--------|-------|
| `Images` | IMAGE | نعم | - | الصور المدخلة لاستخدامها في توليد النموذج ثلاثي الأبعاد. يمكن توفير صور متعددة. |
| `Seed` | INT | نعم | - | قيمة البذرة العشوائية لضمان اتساق التوليد. |
| `Material_Type` | STRING | نعم | - | نوع المادة المراد تطبيقها على النموذج ثلاثي الأبعاد. |
| `Polygon_count` | STRING | نعم | - | عدد المضلعات المستهدف للنموذج ثلاثي الأبعاد المُولَّد. يحدد جودة الشبكة ومستوى التفاصيل. |

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|------------|--------------|-------|
| `3D Model Path` | STRING | مسار الملف للنموذج ثلاثي الأبعاد الذي تم تنزيله (للتّوافق مع الإصدارات السابقة فقط). |
| `GLB` | FILE3DGLB | النموذج ثلاثي الأبعاد المُولَّد بتنسيق GLB. |
