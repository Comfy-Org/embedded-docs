# Rodin 3D Smooth 節點

Rodin 3D Smooth 節點透過處理輸入影像並將其轉換為平滑的 3D 模型，使用 Rodin API 來生成 3D 資產。它接收多張影像作為輸入，並產生可下載的 3D 模型檔案。此節點會自動處理整個生成過程，包括任務建立、狀態輪詢和檔案下載。

## 輸入

| 參數 | 資料類型 | 必要 | 範圍 | 說明 |
|-----------|-----------|----------|-------|-------------|
| `圖片` | IMAGE | 是 | - | 用於 3D 模型生成的輸入影像。可提供多張影像。 |
| `種子值` | INT | 是 | - | 用於生成一致性的隨機種子值。 |
| `材質類型` | STRING | 是 | - | 應用於 3D 模型的材質類型。 |
| `多邊形數量` | STRING | 是 | - | 生成的 3D 模型的目標多邊形數量。決定網格品質和細節層級。 |
