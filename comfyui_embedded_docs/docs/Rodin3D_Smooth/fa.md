### مرور کلی

گره Rodin 3D Smooth با استفاده از API رادین، دارایی‌های سه‌بعدی تولید می‌کند. این گره با پردازش تصاویر ورودی، آن‌ها را به مدل‌های سه‌بعدی صاف تبدیل می‌کند. چندین تصویر را به عنوان ورودی دریافت کرده و یک فایل مدل سه‌بعدی قابل دانلود تولید می‌کند. این گره کل فرآیند تولید شامل ایجاد وظیفه، نظارت بر وضعیت و دانلود خودکار فایل را مدیریت می‌کند.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|---------|----------|--------|--------|---------|
| `Images` | IMAGE | بله | - | تصاویر ورودی برای تولید مدل سه‌بعدی. می‌توان چندین تصویر ارائه داد. |
| `Seed` | INT | بله | - | مقدار دانه تصادفی برای یکنواختی تولید. |
| `Material_Type` | STRING | بله | - | نوع ماده‌ای که روی مدل سه‌بعدی اعمال می‌شود. |
| `Polygon_count` | STRING | بله | - | تعداد چندضلعی هدف برای مدل سه‌بعدی تولیدشده. کیفیت مش و سطح جزئیات را تعیین می‌کند. |

## خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-----------|----------|---------|
| `3D Model Path` | STRING | مسیر فایل مدل سه‌بعدی دانلودشده (فقط برای سازگاری با نسخه‌های قبلی). |
| `GLB` | FILE3DGLB | مدل سه‌بعدی تولیدشده در قالب GLB. |
