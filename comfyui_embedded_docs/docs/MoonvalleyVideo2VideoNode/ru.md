Вот перевод документации на русский язык:

Узел Moonvalley Marey Video to Video преобразует входное видео в новое видео на основе текстового описания. Он использует API Moonvalley для генерации видео, соответствующих вашему запросу, при сохранении характеристик движения или позы из исходного видео. Вы можете управлять стилем и содержимым выходного видео с помощью текстовых запросов и различных параметров генерации.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `prompt` | STRING | Да | - | Описывает видео для генерации (многострочный ввод) |
| `negative_prompt` | STRING | Нет | - | Текст негативного запроса (по умолчанию: обширный список негативных описаний) |
| `seed` | INT | Да | от 0 до 4294967295 | Значение случайного зерна (по умолчанию: 9) |
| `video` | VIDEO | Да | - | Эталонное видео, используемое для генерации выходного видео. Должно быть не менее 5 секунд. Видео длиннее 5 с будут автоматически обрезаны. Поддерживается только формат MP4. |
| `control_type` | COMBO | Нет | "Motion Transfer"<br>"Pose Transfer" | Выбор типа управления (по умолчанию: "Motion Transfer") |
| `motion_intensity` | INT | Нет | от 0 до 100 | Используется только если `control_type` равен "Motion Transfer" (по умолчанию: 100) |
| `steps` | INT | Да | от 1 до 100 | Количество шагов инференса (по умолчанию: 33) |

**Примечание:** Параметр `motion_intensity` применяется только когда `control_type` установлен в "Motion Transfer". При использовании "Pose Transfer" этот параметр игнорируется.

## Выходные параметры

| Имя выходного параметра | Тип данных | Описание |
|-------------|-----------|-------------|
| `output` | VIDEO | Сгенерированное выходное видео |
