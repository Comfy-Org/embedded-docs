## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|---------|----------|--------|--------|----------|
| `prompt` | STRING | بله | - | توصیف ویدئوی مورد نظر برای تولید (ورودی چندخطی) |
| `negative_prompt` | STRING | خیر | - | متن توصیف منفی (پیش‌فرض: فهرست گسترده‌ای از توصیف‌گرهای منفی) |
| `seed` | INT | بله | 0 تا 4294967295 | مقدار دانه تصادفی (پیش‌فرض: 9) |
| `video` | VIDEO | بله | - | ویدئوی مرجع برای تولید ویدئوی خروجی. حداقل ۵ ثانیه طول می‌خواهد. ویدئوهای بلندتر از ۵ ثانیه خودکار برش می‌خورند. فقط فرمت MP4 پشتیبانی می‌شود. |
| `control_type` | COMBO | خیر | "انتقال حرکت"<br>"انتقال حالت" | انتخاب نوع کنترل (پیش‌فرض: "انتقال حرکت") |
| `motion_intensity` | INT | خیر | 0 تا 100 | فقط در صورت انتخاب "انتقال حرکت" کاربرد دارد (پیش‌فرض: 100) |
| `steps` | INT | بله | 1 تا 100 | تعداد مراحل استنتاج (پیش‌فرض: 33) |

**نکته:** پارامتر `motion_intensity` فقط زمانی اعمال می‌شود که `control_type` روی "انتقال حرکت" تنظیم شده باشد. در حالت "انتقال حالت"، این پارامتر نادیده گرفته می‌شود.
