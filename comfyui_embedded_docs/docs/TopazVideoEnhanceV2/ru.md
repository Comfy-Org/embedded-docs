# Topaz Video Enhance V2

Узел **Topaz Video Enhance V2** позволяет масштабировать и улучшать видео с помощью моделей ИИ от Topaz Labs. Он может увеличивать разрешение, регулировать частоту кадров через интерполяцию, а также применять творческие или реалистичные улучшения, чтобы вдохнуть новую жизнь в ваш видеоматериал.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `видео` | VIDEO | Да | - | Входное видео для обработки. Должно быть в формате контейнера MP4. |
| `upscaler_model` | COMBO | Да | `"Astra 2"`<br>`"Starlight (Astra) Fast"`<br>`"Starlight (Astra) Creative"`<br>`"Starlight Precise 2.5"`<br>`"Disabled"` | Модель ИИ, используемая для масштабирования видео. Выбор "Disabled" означает, что масштабирование применяться не будет. |
| `upscaler_model.upscaler_resolution` | COMBO | Условно | `"FullHD (1080p)"`<br>`"4K (2160p)"` | Целевое выходное разрешение для масштабатора. Этот параметр обязателен, когда выбрана модель масштабирования (не "Disabled"). |
| `upscaler_model.creativity` | FLOAT / COMBO | Условно | Astra 2: от 0.0 до 1.0 (шаг 0.1)<br>Starlight Creative: `"low"`<br>`"middle"`<br>`"high"` | Степень творческой обработки при масштабировании. Доступно только для моделей "Astra 2" и "Starlight (Astra) Creative". Для Astra 2 это ползунок (по умолчанию: 0.5). Для Starlight Creative это комбинированный список (по умолчанию: "low"). |
| `upscaler_model.prompt` | STRING | Нет | - | Необязательный описательный (не инструктивный) промпт сцены. Доступен только для модели "Astra 2". Ограничен 500 входными кадрами (~15с при 30fps) при установке. По умолчанию: пусто. |
| `upscaler_model.sharp` | FLOAT | Нет | от 0.0 до 1.0 (шаг 0.01) | Предварительная резкость: 0.0=размытие по Гауссу, 0.5=сквозная передача (по умолчанию), 1.0=повышение резкости USM. Доступно только для модели "Astra 2". По умолчанию: 0.5. |
| `upscaler_model.realism` | FLOAT | Нет | от 0.0 до 1.0 (шаг 0.01) | Приближает выходной результат к фотографическому реализму. Оставьте 0 для использования настроек модели по умолчанию. Доступно только для модели "Astra 2". По умолчанию: 0.0. |
| `interpolation_model` | COMBO | Да | `"Disabled"`<br>`"apo-8"` | Модель ИИ, используемая для интерполяции кадров. Выбор "Disabled" означает, что интерполяция применяться не будет. |
| `interpolation_model.interpolation_frame_rate` | INT | Условно | от 15 до 240 | Выходная частота кадров. Обязателен, когда модель интерполяции — "apo-8". По умолчанию: 60. |
| `interpolation_model.interpolation_slowmo` | INT | Нет | от 1 до 16 | Коэффициент замедления, применяемый к входному видео. Например, 2 делает выходное видео в два раза медленнее и удваивает длительность. По умолчанию: 1. |
| `interpolation_model.interpolation_duplicate` | BOOLEAN | Нет | True/False | Анализировать входное видео на наличие дублирующихся кадров и удалять их. По умолчанию: False. |
| `interpolation_model.interpolation_duplicate_threshold` | FLOAT | Нет | от 0.001 до 0.1 (шаг 0.001) | Чувствительность обнаружения дублирующихся кадров. По умолчанию: 0.01. |
| `уровень динамического сжатия` | COMBO | Нет | `"Low"`<br>`"Mid"`<br>`"High"` | Уровень CQP для сжатия видео. По умолчанию: "Low". |

**Важные ограничения:**
- Как минимум один из параметров `upscaler_model` или `interpolation_model` должен быть включён (не "Disabled"), иначе будет выдана ошибка.
- Входное видео должно быть в формате контейнера MP4.
- Модель "Astra 2" с промптом ограничена 500 входными кадрами (~15 секунд при 30fps). Без промпта ограничение выше.
- Когда `upscaler_model` не равен "Disabled", подпараметр `upscaler_resolution` обязателен.
- Когда `interpolation_model` не равен "Disabled", подпараметр `interpolation_frame_rate` обязателен.
