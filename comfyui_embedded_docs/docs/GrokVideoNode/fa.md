گره Grok Video یک ویدیوی کوتاه از توضیحات متنی تولید می‌کند. این گره می‌تواند با استفاده از یک prompt ویدیویی از ابتدا بسازد یا یک تصویر ورودی را بر اساس prompt متحرک کند. این گره درخواستی را به یک API خارجی ارسال کرده و ویدیوی تولیدشده را بازمی‌گرداند.

## ورودی‌ها

| پارامتر | نوع داده | اجباری | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `model` | COMBO | بله | `"grok-imagine-video"`<br>`"grok-imagine-video-beta"` | مدل مورد استفاده برای تولید ویدیو. |
| `prompt` | STRING | بله | - | توضیحات متنی ویدیوی مورد نظر. |
| `resolution` | COMBO | بله | `"480p"`<br>`"720p"` | وضوح تصویر ویدیوی خروجی. |
| `aspect_ratio` | COMBO | بله | `"auto"`<br>`"16:9"`<br>`"4:3"`<br>`"3:2"`<br>`"1:1"`<br>`"2:3"`<br>`"3:4"`<br>`"9:16"` | نسبت ابعاد ویدیوی خروجی (پیش‌فرض: "auto"). |
| `duration` | INT | بله | 1 تا 15 | مدت زمان ویدیوی خروجی بر حسب ثانیه (پیش‌فرض: 6). |
| `seed` | INT | بله | 0 تا 2147483647 | دانه (seed) برای تعیین اینکه آیا گره دوباره اجرا شود؛ نتایج واقعی بدون توجه به seed غیرقطعی هستند (پیش‌فرض: 0). |
| `image` | IMAGE | خیر | - | یک تصویر ورودی اختیاری برای متحرک‌سازی. |

**توجه:** اگر یک `image` ارائه شود، تنها یک تصویر پشتیبانی می‌شود. ارائه چندین تصویر باعث بروز خطا خواهد شد. `prompt` باید حداقل ۱ کاراکتر پس از حذف فاصله‌های خالی باشد.

## خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-------------|-----------|-----------|
| `output` | VIDEO | ویدیوی تولیدشده. |
