Вот перевод документации на русский язык:

Узел VAEEncodeAudio преобразует аудиоданные в латентное представление с помощью вариационного автоэнкодера (VAE). Он принимает аудиовход и обрабатывает его через VAE для создания сжатых латентных сэмплов, которые можно использовать для дальнейшей генерации аудио или задач манипуляции. При необходимости узел автоматически передискретизирует аудио для соответствия ожидаемой частоте дискретизации VAE перед кодированием.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `audio` | AUDIO | Да | - | Аудиоданные для кодирования, содержащие информацию о форме волны и частоте дискретизации |
| `vae` | VAE | Да | - | Модель вариационного автоэнкодера, используемая для кодирования аудио в латентное пространство |

**Примечание:** Аудиовход автоматически передискретизируется для соответствия ожидаемой частоте дискретизации VAE (по умолчанию: 44100 Гц), если исходная частота дискретизации отличается от этого значения.

## Выходы

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `LATENT` | LATENT | Закодированное аудиопредставление в латентном пространстве, содержащее сжатые сэмплы |
