Вот перевод документации на русский язык:

Узел AudioEncoderLoader загружает модель аудиокодера из файла, находящегося в вашей папке с аудиокодерами. На вход он принимает имя файла модели аудиокодера, а на выходе возвращает загруженную модель, которая затем может быть использована для задач обработки аудио в вашем рабочем процессе.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `имя_аудио_энкодера` | STRING | Да | Список доступных файлов аудиокодеров в папке audio_encoders | Выбирает, какой файл модели аудиокодера загрузить |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `audio_encoder` | AUDIO_ENCODER | Загруженная модель аудиокодера, готовая к использованию в рабочих процессах обработки аудио |
