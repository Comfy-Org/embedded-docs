گره **PairConditioningSetProperties** به شما امکان می‌دهد ویژگی‌های هر دو جفت شرطی‌سازی مثبت و منفی را هم‌زمان تغییر دهید. این گره تنظیمات شدت، ناحیه شرطی‌سازی، و کنترل‌های اختیاری ماسک یا زمان‌بندی را روی هر دو ورودی شرطی‌سازی اعمال کرده و داده‌های شرطی‌سازی مثبت و منفی اصلاح‌شده را بازمی‌گرداند.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|---------|----------|--------|--------|---------|
| `مثبت_جدید` | CONDITIONING | بله | - | ورودی شرطی‌سازی مثبت برای اصلاح |
| `منفی_جدید` | CONDITIONING | بله | - | ورودی شرطی‌سازی منفی برای اصلاح |
| `شدت` | FLOAT | بله | 0.0 تا 10.0 | ضریب شدت اعمال‌شده به شرطی‌سازی (پیش‌فرض: 1.0) |
| `تعیین ناحیه شرطی` | STRING | بله | "default"<br>"mask bounds" | نحوه محاسبه ناحیه شرطی‌سازی را تعیین می‌کند (پیش‌فرض: "default") |
| `ماسک` | MASK | خیر | - | ماسک اختیاری برای محدود کردن ناحیه شرطی‌سازی |
| `hooks` | HOOKS | خیر | - | گروه هوک اختیاری برای اصلاحات پیشرفته شرطی‌سازی |
| `گام‌های زمانی` | TIMESTEPS_RANGE | خیر | - | محدوده گام زمانی اختیاری برای محدود کردن زمان اعمال شرطی‌سازی |

## خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-----------|----------|---------|
| `positive` | CONDITIONING | شرطی‌سازی مثبت اصلاح‌شده با ویژگی‌های اعمال‌شده |
| `negative` | CONDITIONING | شرطی‌سازی منفی اصلاح‌شده با ویژگی‌های اعمال‌شده |
