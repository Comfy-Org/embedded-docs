**عقدة ضبط خصائص التكييف الزوجي (PairConditioningSetProperties)**

تتيح لك عقدة **PairConditioningSetProperties** تعديل خصائص أزواج التكييف الإيجابي والسلبي في وقت واحد. تقوم بتطبيق تعديلات القوة، وإعدادات منطقة التكييف، وعناصر التحكم الاختيارية في القناع أو التوقيت على مدخلي التكييف، مع إرجاع بيانات التكييف الإيجابي والسلبي المعدلة.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|---------|--------------|--------|--------|-------|
| `positive_NEW` | CONDITIONING | نعم | - | مدخل التكييف الإيجابي المراد تعديله |
| `negative_NEW` | CONDITIONING | نعم | - | مدخل التكييف السلبي المراد تعديله |
| `strength` | FLOAT | نعم | 0.0 إلى 10.0 | مضاعف القوة المطبق على التكييف (القيمة الافتراضية: 1.0) |
| `set_cond_area` | STRING | نعم | "default"<br>"mask bounds" | يحدد كيفية حساب منطقة التكييف (القيمة الافتراضية: "default") |
| `mask` | MASK | لا | - | قناع اختياري لتقييد منطقة التكييف |
| `hooks` | HOOKS | لا | - | مجموعة خطافات اختيارية لتعديلات التكييف المتقدمة |
| `timesteps` | TIMESTEPS_RANGE | لا | - | نطاق زمني اختياري للحد من وقت تطبيق التكييف |

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|------------|--------------|--------|
| `positive` | CONDITIONING | التكييف الإيجابي المعدل مع الخصائص المطبقة |
| `negative` | CONDITIONING | التكييف السلبي المعدل مع الخصائص المطبقة |
