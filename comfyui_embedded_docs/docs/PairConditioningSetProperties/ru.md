Узел PairConditioningSetProperties позволяет одновременно изменять свойства пар позитивного и негативного кондиционирования. Он применяет корректировки силы, настройки области кондиционирования и опциональные элементы управления маскированием или временными параметрами к обоим входам кондиционирования, возвращая модифицированные данные позитивного и негативного кондиционирования.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `positive_NEW` | CONDITIONING | Да | - | Вход позитивного кондиционирования для модификации |
| `negative_NEW` | CONDITIONING | Да | - | Вход негативного кондиционирования для модификации |
| `strength` | FLOAT | Да | 0.0 до 10.0 | Множитель силы, применяемый к кондиционированию (по умолчанию: 1.0) |
| `set_cond_area` | STRING | Да | "default"<br>"mask bounds" | Определяет способ расчета области кондиционирования |
| `mask` | MASK | Нет | - | Опциональная маска для ограничения области кондиционирования |
| `hooks` | HOOKS | Нет | - | Опциональная группа хуков для расширенных модификаций кондиционирования |
| `timesteps` | TIMESTEPS_RANGE | Нет | - | Опциональный диапазон временных шагов для ограничения применения кондиционирования |

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `positive` | CONDITIONING | Модифицированное позитивное кондиционирование с примененными свойствами |
| `negative` | CONDITIONING | Модифицированное негативное кондиционирование с примененными свойствами |
