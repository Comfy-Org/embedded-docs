Вот перевод документации на русский язык:

Узел **PairConditioningSetProperties** позволяет одновременно изменять свойства пар положительного и отрицательного кондиционирования. Он применяет корректировки силы, настройки области кондиционирования, а также дополнительные элементы управления маскированием или временными интервалами к обоим входам кондиционирования, возвращая измененные данные положительного и отрицательного кондиционирования.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `положительный_НОВЫЙ` | CONDITIONING | Да | - | Входное положительное кондиционирование для изменения |
| `отрицательный_НОВЫЙ` | CONDITIONING | Да | - | Входное отрицательное кондиционирование для изменения |
| `сила` | FLOAT | Да | 0.0 до 10.0 | Множитель силы, применяемый к кондиционированию (по умолчанию: 1.0) |
| `установить_область_условия` | STRING | Да | "default"<br>"mask bounds" | Определяет способ расчета области кондиционирования (по умолчанию: "default") |
| `маска` | MASK | Нет | - | Необязательная маска для ограничения области кондиционирования |
| `хуки` | HOOKS | Нет | - | Необязательная группа хуков для расширенных модификаций кондиционирования |
| `временные_шаги` | TIMESTEPS_RANGE | Нет | - | Необязательный диапазон временных шагов для ограничения времени применения кондиционирования |

## Выходы

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `отрицательный` | CONDITIONING | Измененное положительное кондиционирование с примененными свойствами |
| `negative` | CONDITIONING | Измененное отрицательное кондиционирование с примененными свойствами |
