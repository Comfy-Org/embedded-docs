Вот перевод документации узла RegexExtract на русский язык:

Узел RegexExtract выполняет поиск шаблонов в тексте с помощью регулярных выражений. Он может найти первое совпадение, все совпадения, определенные группы из совпадений или все группы по всем совпадениям. Узел поддерживает различные флаги регулярных выражений для учета регистра, многострочного поиска и режима dotall.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `string` | STRING | Да | - | Входной текст для поиска шаблонов |
| `regex_pattern` | STRING | Да | - | Шаблон регулярного выражения для поиска |
| `mode` | COMBO | Да | "First Match"<br>"All Matches"<br>"First Group"<br>"All Groups" | Режим извлечения определяет, какие части совпадений возвращаются (по умолчанию: "First Match") |
| `case_insensitive` | BOOLEAN | Нет | - | Игнорировать ли регистр при поиске (по умолчанию: True) |
| `multiline` | BOOLEAN | Нет | - | Считать ли строку многострочной (по умолчанию: False) |
| `dotall` | BOOLEAN | Нет | - | Должна ли точка (.) соответствовать символам новой строки (по умолчанию: False) |
| `group_index` | INT | Нет | 0-100 | Индекс захватываемой группы для извлечения при использовании групповых режимов (по умолчанию: 1) |

**Примечание:** При использовании режимов "First Group" или "All Groups" параметр `group_index` указывает, какую захватываемую группу извлекать. Группа 0 представляет все совпадение, а группы 1+ представляют нумерованные захватываемые группы в вашем шаблоне регулярного выражения.

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `output` | STRING | Извлеченный текст на основе выбранного режима и параметров |
