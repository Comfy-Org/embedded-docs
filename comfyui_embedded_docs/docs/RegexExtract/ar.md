أنت خبير في الترجمة التقنية متخصص في توثيق عُقد ComfyUI من الإنجليزية إلى العربية.

## قواعد الترجمة

1. **المحتوى الذي يجب عدم ترجمته:**
   - أسماء المعاملات بين علامات الاقتباس الخلفية: `image`, `seed`, `model`
   - أنواع البيانات بالأحرف الكبيرة: IMAGE, STRING, INT, FLOAT, MODEL, CONDITIONING, إلخ
   - القيم في عمود Range: الأرقام، "auto"، أسماء الخيارات
   - الكود، مسارات الملفات

2. **المحتوى الذي يجب ترجمته:**
   - عناوين الأقسام: ## نظرة عامة, ## المدخلات, ## المخرجات
   - جميع النصوص الوصفية والتوضيحية
   - أوصاف المعاملات

3. **جودة الترجمة:**
   - استخدام اللغة العربية الفصحى المعاصرة
   - الحفاظ على نبرة احترافية ولكن سهلة الفهم
   - ضمان الدقة التقنية
   - استخدام المصطلحات التقنية العربية القياسية

4. **التنسيق:**
   - الحفاظ على جميع تنسيقات Markdown
   - الحفاظ على بنية الجداول
   - عدم إضافة أي ملاحظة أو رابط في بداية الوثيقة (سيتم إضافتها تلقائيًا)

تقوم عقدة RegexExtract بالبحث عن أنماط في النص باستخدام التعبيرات النمطية. يمكنها العثور على أول تطابق، أو جميع التطابقات، أو مجموعات محددة من التطابقات، أو جميع المجموعات عبر تطابقات متعددة. تدعم العقدة عدة علامات تعبير نمطي للتحكم في حساسية حالة الأحرف، والمطابقة متعددة الأسطر، وسلوك النقطة.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|-----------|-----------|----------|-------|-------------|
| `string` | STRING | نعم | - | النص المدخل للبحث عن الأنماط |
| `regex_pattern` | STRING | نعم | - | نمط التعبير النمطي للبحث عنه |
| `mode` | COMBO | نعم | "أول تطابق"<br>"جميع التطابقات"<br>"المجموعة الأولى"<br>"جميع المجموعات" | يحدد وضع الاستخراج أجزاء التطابقات التي سيتم إرجاعها (الافتراضي: "أول تطابق") |
| `case_insensitive` | BOOLEAN | لا | - | تجاهل حالة الأحرف عند المطابقة (الافتراضي: صحيح) |
| `multiline` | BOOLEAN | لا | - | معالجة النص كأسطر متعددة (الافتراضي: خطأ) |
| `dotall` | BOOLEAN | لا | - | مطابقة النقطة (.) لأسطر جديدة (الافتراضي: خطأ) |
| `group_index` | INT | لا | 0-100 | فهرس مجموعة الالتقاط لاستخراجها عند استخدام أوضاع المجموعة (الافتراضي: 1) |

**ملاحظة:** عند استخدام وضعي "المجموعة الأولى" أو "جميع المجموعات"، يحدد معامل `group_index` مجموعة الالتقاط المراد استخراجها. تمثل المجموعة 0 التطابق بأكمله، بينما تمثل المجموعات 1+ مجموعات الالتقاط المرقمة في نمط التعبير النمطي الخاص بك.
