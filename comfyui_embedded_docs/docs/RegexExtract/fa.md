گره RegexExtract با استفاده از عبارات با قاعده (Regular Expressions) به جستجوی الگوها در متن می‌پردازد. این گره می‌تواند اولین تطابق، تمام تطابق‌ها، گروه‌های خاصی از تطابق‌ها، یا تمام گروه‌های موجود در چندین تطابق را پیدا کند. این گره از پرچم‌های مختلف regex برای حساسیت به حروف بزرگ و کوچک، تطابق چندخطی و رفتار dotall پشتیبانی می‌کند.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|---------|----------|--------|--------|---------|
| `رشته` | STRING | بله | - | متن ورودی برای جستجوی الگوها |
| `الگوی regex` | STRING | بله | - | الگوی عبارت با قاعده برای جستجو |
| `حالت` | COMBO | بله | "اولین تطابق"<br>"تمام تطابق‌ها"<br>"اولین گروه"<br>"تمام گروه‌ها" | حالت استخراج تعیین می‌کند که کدام بخش‌های تطابق‌ها برگردانده شوند (پیش‌فرض: "اولین تطابق") |
| `بدون حساسیت به حروف` | BOOLEAN | خیر | - | آیا هنگام تطابق، حروف بزرگ و کوچک نادیده گرفته شوند (پیش‌فرض: True) |
| `چندخطی` | BOOLEAN | خیر | - | آیا رشته به عنوان چند خط در نظر گرفته شود (پیش‌فرض: False) |
| `dotall` | BOOLEAN | خیر | - | آیا نقطه (.) با خطوط جدید تطابق دارد (پیش‌فرض: False) |
| `شماره گروه` | INT | خیر | 0-100 | شاخص گروه ضبط‌شده برای استخراج هنگام استفاده از حالت‌های گروه (پیش‌فرض: 1) |

**توجه:** هنگام استفاده از حالت‌های "اولین گروه" یا "تمام گروه‌ها"، پارامتر `group_index` مشخص می‌کند که کدام گروه ضبط‌شده استخراج شود. گروه 0 نشان‌دهنده کل تطابق است، در حالی که گروه‌های 1+ نشان‌دهنده گروه‌های ضبط‌شده شماره‌گذاری شده در الگوی regex شما هستند.
