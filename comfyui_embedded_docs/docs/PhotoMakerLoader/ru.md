Вот перевод документации на русский язык:

Узел PhotoMakerLoader загружает модель PhotoMaker из доступных файлов моделей. Он считывает указанный файл модели и подготавливает кодировщик ID PhotoMaker для использования в задачах генерации изображений на основе идентичности. Данный узел помечен как экспериментальный и предназначен для тестирования.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `название_модели_photomaker` | STRING | Да | Доступно несколько вариантов | Имя файла модели PhotoMaker для загрузки. Доступные варианты определяются файлами моделей, находящимися в папке `photomaker`. |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `photomaker_model` | PHOTOMAKER | Загруженная модель PhotoMaker, содержащая кодировщик ID, готовая к использованию в операциях кодирования идентичности. |
