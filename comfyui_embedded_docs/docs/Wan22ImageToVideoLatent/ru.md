Узел `Wan22ImageToVideoLatent` создаёт скрытые представления видео на основе изображений. Он генерирует пустое скрытое пространство видео с заданными размерами и может опционально кодировать начальную последовательность изображений в первые кадры. Если указано начальное изображение, узел кодирует его в скрытое пространство и создаёт соответствующую маску шума для областей, подлежащих реставрации.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `vae` | VAE | Да | - | Модель VAE, используемая для кодирования изображений в скрытое пространство |
| `width` | INT | Да | от 32 до MAX_RESOLUTION | Ширина выходного видео в пикселях (по умолчанию: 1280, шаг: 32) |
| `height` | INT | Да | от 32 до MAX_RESOLUTION | Высота выходного видео в пикселях (по умолчанию: 704, шаг: 32) |
| `length` | INT | Да | от 1 до MAX_RESOLUTION | Количество кадров в видеопоследовательности (по умолчанию: 49, шаг: 4) |
| `batch_size` | INT | Да | от 1 до 4096 | Количество генерируемых пакетов (по умолчанию: 1) |
| `start_image` | IMAGE | Нет | - | Необязательная начальная последовательность изображений для кодирования в скрытое представление видео |

**Примечание:** Если указан `start_image`, узел кодирует последовательность изображений в начальные кадры скрытого пространства и генерирует соответствующую маску шума. Параметры `width` и `height` должны быть кратны 16 для корректных размеров скрытого пространства. Параметр `length` определяет количество кадров в скрытом представлении видео; временная размерность скрытого пространства вычисляется как `((length - 1) // 4) + 1`.

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `samples` | LATENT | Сгенерированное скрытое представление видео |
| `noise_mask` | LATENT | Маска шума, указывающая, какие области следует очистить от шума в процессе генерации |
