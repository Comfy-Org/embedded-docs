## Обзор

Отслеживание объектов в видеокадрах с помощью трекера на основе памяти SAM3. Этот узел обрабатывает последовательность видеокадров и поддерживает идентификацию объектов между кадрами, используя либо начальные маски, либо текстовые подсказки для определения того, что необходимо отслеживать.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `images` | IMAGE | Да | Пакетные видеокадры | Видеокадры в виде пакетных изображений |
| `model` | MODEL | Да | Модель SAM3 | Модель SAM3 для отслеживания |
| `initial_mask` | MASK | Нет | Одна маска на объект | Маска(и) для первого кадра отслеживания (по одной на объект). Обязательно, если не указан `conditioning`. |
| `conditioning` | CONDITIONING | Нет | Текстовое условие | Текстовое условие для обнаружения новых объектов во время отслеживания. Обязательно, если не указан `initial_mask`. |
| `detection_threshold` | FLOAT | Нет | от 0.0 до 1.0 (по умолчанию: 0.5) | Порог оценки для обнаружения по текстовой подсказке |
| `max_objects` | INT | Нет | от 0 до 64 (по умолчанию: 0) | Максимальное количество отслеживаемых объектов. Начальные маски учитываются в этом лимите. Значение 0 использует внутреннее ограничение в 64. |
| `detect_interval` | INT | Нет | от 1 до безлимита (по умолчанию: 1) | Запуск обнаружения каждые N кадров (1 = каждый кадр). Более высокие значения экономят вычислительные ресурсы. |

**Примечание:** Необходимо указать либо `initial_mask`, либо `conditioning`. Если оба параметра опущены, узел выдаст ошибку.

## Выходные параметры

| Имя выходного параметра | Тип данных | Описание |
|-------------|-----------|-------------|
| `track_data` | SAM3TrackData | Данные отслеживания, содержащие маски объектов и метаданные по всем видеокадрам |
