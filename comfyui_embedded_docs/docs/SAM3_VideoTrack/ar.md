# نظرة عامة

تتبع الكائنات عبر إطارات الفيديو باستخدام متعقب SAM3 القائم على الذاكرة. تقوم هذه العقدة بمعالجة سلسلة من إطارات الفيديو وتحافظ على هويات الكائنات عبر الإطارات، باستخدام إما الأقنعة الأولية أو المطالبات النصية لتحديد ما يجب تتبعه.

# المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|---------|---------------|--------|---------|-------|
| `الصور` | IMAGE | نعم | إطارات فيديو مجمعة | إطارات الفيديو كصور مجمعة |
| `النموذج` | MODEL | نعم | نموذج SAM3 | نموذج SAM3 المستخدم للتتبع |
| `القناع الأولي` | MASK | لا | قناع واحد لكل كائن | قناع (أقنعة) للإطار الأول المراد تتبعه (قناع واحد لكل كائن). مطلوب إذا لم يتم توفير `التهيئة`. |
| `التهيئة` | CONDITIONING | لا | تكييف نصي | تكييف نصي لاكتشاف الكائنات الجديدة أثناء التتبع. مطلوب إذا لم يتم توفير `القناع الأولي`. |
| `عَتَبة الكشف` | FLOAT | لا | 0.0 إلى 1.0 (الافتراضي: 0.5) | عتبة النتيجة للكشف بالمطالبة النصية |
| `أقصى عدد للكائنات` | INT | لا | 0 إلى 64 (الافتراضي: 0) | الحد الأقصى للكائنات المتتبعة. تُحتسب الأقنعة الأولية ضمن هذا الحد. القيمة 0 تستخدم الحد الداخلي البالغ 64. |
| `فاصل الكشف` | INT | لا | 1 إلى غير محدود (الافتراضي: 1) | تشغيل الكشف كل N إطار (1 = كل إطار). القيم الأعلى توفر موارد الحوسبة. |

**ملاحظة:** يجب توفير إما `initial_mask` أو `conditioning`. إذا تم حذف كليهما، ستُظهر العقدة خطأ.
