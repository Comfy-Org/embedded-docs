Вот перевод документации на русский язык:

Kling Virtual Try On Node. Загрузите изображение человека и изображение одежды, чтобы примерить эту одежду на человека. Вы можете объединить несколько изображений предметов одежды в одно изображение с белым фоном.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `human_image` | IMAGE | Да | - | Изображение человека, на которого будет примеряться одежда |
| `cloth_image` | IMAGE | Да | - | Изображение одежды для примерки на человека |
| `model_name` | STRING | Да | `"kolors-virtual-try-on-v1"` | Модель виртуальной примерки для использования (по умолчанию: "kolors-virtual-try-on-v1") |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `output` | IMAGE | Результирующее изображение, показывающее человека с примеренным предметом одежды |
