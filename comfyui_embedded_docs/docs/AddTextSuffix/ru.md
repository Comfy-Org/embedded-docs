Этот узел добавляет указанный суффикс в конец входной текстовой строки. Он принимает исходный текст и суффикс в качестве входных данных, а затем возвращает объединенный результат.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `text` | STRING | Да | | Исходный текст, к которому будет добавлен суффикс. |
| `suffix` | STRING | Нет | | Суффикс, добавляемый к тексту (по умолчанию: ""). |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `text` | STRING | Результирующий текст после добавления суффикса. |
