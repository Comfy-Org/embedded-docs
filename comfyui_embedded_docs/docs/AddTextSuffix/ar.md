تقوم هذه العقدة بإلحاق لاحقة محددة بنهاية سلسلة نصية مدخلة. تأخذ النص الأصلي واللاحقة كمدخلات، ثم تُرجع النتيجة المدمجة.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|----------|---------------|--------|--------|-------|
| `text` | STRING | نعم | | النص الأصلي الذي ستُضاف إليه اللاحقة. |
| `suffix` | STRING | لا | | اللاحقة المراد إضافتها إلى النص (القيمة الافتراضية: ""). |

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|-------------|---------------|--------|
| `text` | STRING | النص الناتج بعد إلحاق اللاحقة. |
