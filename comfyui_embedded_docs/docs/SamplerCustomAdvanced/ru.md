Узел SamplerCustomAdvanced выполняет продвинутую выборку в скрытом пространстве с использованием пользовательских конфигураций шума, направляющего сигнала (guidance) и параметров сэмплирования. Он обрабатывает скрытое представление изображения через направленный процесс сэмплирования с настраиваемой генерацией шума и расписанием сигм, формируя как итоговый результат выборки, так и его очищенную от шума версию (при наличии).

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `noise` | NOISE | Да | - | Генератор шума, задающий исходный шумовой паттерн и зерно (seed) для процесса сэмплирования |
| `guider` | GUIDER | Да | - | Модель направляющего сигнала, которая управляет процессом выборки в сторону желаемых результатов |
| `sampler` | SAMPLER | Да | - | Алгоритм сэмплирования, определяющий способ перемещения по скрытому пространству в ходе генерации |
| `sigmas` | SIGMAS | Да | - | Расписание сигм, контролирующее уровни шума на каждом шаге сэмплирования |
| `latent_image` | LATENT | Да | - | Исходное скрытое представление, служащее отправной точкой для сэмплирования |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `output` | LATENT | Итоговое скрытое представление, полученное после завершения процесса сэмплирования |
| `denoised_output` | LATENT | Очищенная от шума версия выходных данных (при наличии), в противном случае возвращается то же значение, что и в `output` |
