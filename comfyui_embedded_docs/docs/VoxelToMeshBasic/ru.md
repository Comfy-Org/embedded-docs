Вот перевод документации на русский язык:

Узел VoxelToMeshBasic преобразует трёхмерные воксельные данные в полигональную сетку. Он обрабатывает воксельные объёмы, применяя пороговое значение для определения того, какие части объёма станут сплошными поверхностями в результирующей сетке. Узел выводит полную структуру сетки с вершинами и гранями, которая может использоваться для 3D-рендеринга и моделирования.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `voxel` | VOXEL | Да | - | Трёхмерные воксельные данные для преобразования в сетку |
| `threshold` | FLOAT | Да | -1.0 до 1.0 | Пороговое значение, используемое для определения того, какие воксели станут частью поверхности сетки (по умолчанию: 0.6) |

## Выходы

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `MESH` | MESH | Сгенерированная трёхмерная сетка, содержащая вершины и грани |
