### توضیحات گره

گره **یکسان‌ساز صدا (۳ باند)** به شما امکان می‌دهد فرکانس‌های باس، میانی و زیروبمی یک موج صوتی را تنظیم کنید. این گره سه فیلتر مجزا اعمال می‌کند: یک فیلتر پایین‌گذر برای باس، یک فیلتر میان‌گذر برای فرکانس‌های میانی و یک فیلتر بالاگذر برای زیروبمی. هر باند را می‌توان به‌طور مستقل با تنظیمات بهره، فرکانس و پهنای باند کنترل کرد.

## ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|---------|----------|-------|--------|---------|
| `audio` | AUDIO | بله | - | داده‌های صوتی ورودی شامل موج و نرخ نمونه‌برداری. |
| `low_gain_dB` | FLOAT | خیر | 24.0- تا 24.0 | بهره برای فرکانس‌های پایین (باس). مقادیر مثبت باعث تقویت و مقادیر منفی باعث کاهش می‌شوند. (پیش‌فرض: 0.0) |
| `low_freq` | INT | خیر | 20 تا 500 | فرکانس قطع فیلتر پایین‌گذر بر حسب هرتز (Hz). (پیش‌فرض: 100) |
| `mid_gain_dB` | FLOAT | خیر | 24.0- تا 24.0 | بهره برای فرکانس‌های میانی. مقادیر مثبت باعث تقویت و مقادیر منفی باعث کاهش می‌شوند. (پیش‌فرض: 0.0) |
| `mid_freq` | INT | خیر | 200 تا 4000 | فرکانس مرکزی فیلتر میان‌گذر بر حسب هرتز (Hz). (پیش‌فرض: 1000) |
| `mid_q` | FLOAT | خیر | 0.1 تا 10.0 | ضریب کیفیت (پهنای باند) فیلتر میان‌گذر. مقادیر پایین‌تر باند وسیع‌تری ایجاد می‌کنند و مقادیر بالاتر باند باریک‌تری. (پیش‌فرض: 0.707) |
| `high_gain_dB` | FLOAT | خیر | 24.0- تا 24.0 | بهره برای فرکانس‌های بالا (زیروبمی). مقادیر مثبت باعث تقویت و مقادیر منفی باعث کاهش می‌شوند. (پیش‌فرض: 0.0) |
| `high_freq` | INT | خیر | 1000 تا 15000 | فرکانس قطع فیلتر بالاگذر بر حسب هرتز (Hz). (پیش‌فرض: 5000) |

**نکته:** پارامترهای `low_gain_dB`، `mid_gain_dB` و `high_gain_dB` تنها زمانی اعمال می‌شوند که مقدار آن‌ها صفر نباشد. اگر بهره روی 0.0 تنظیم شود، مرحله فیلتر مربوطه نادیده گرفته می‌شود.
