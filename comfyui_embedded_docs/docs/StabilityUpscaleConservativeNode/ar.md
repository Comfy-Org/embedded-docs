# ترجمة وثيقة عقد ComfyUI

## نظرة عامة

رفع دقة الصورة مع الحد الأدنى من التعديلات إلى دقة 4K. تستخدم هذه العقدة تقنية الرفع التحفظي من Stability AI لزيادة دقة الصورة مع الحفاظ على المحتوى الأصلي وإجراء تغييرات طفيفة فقط.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|---------|---------------|--------|---------|-------|
| `الصورة` | IMAGE | نعم | - | الصورة المدخلة المراد رفع دقتها |
| `النص الوصفي` | STRING | نعم | - | ما ترغب في رؤيته في الصورة الناتجة. يؤدي الوصف القوي والواضح الذي يحدد العناصر والألوان والموضوعات إلى نتائج أفضل. (الافتراضي: سلسلة فارغة) |
| `الإبداع` | FLOAT | نعم | 0.2-0.5 | يتحكم في احتمالية إنشاء تفاصيل إضافية غير معتمدة بشكل كبير على الصورة الأولية. (الافتراضي: 0.35) |
| `البذرة` | INT | نعم | 0-4294967294 | البذرة العشوائية المستخدمة لإنشاء الضوضاء. (الافتراضي: 0) |
| `نص سلبي` | STRING | لا | - | كلمات مفتاحية لما لا ترغب في رؤيته في الصورة الناتجة. هذه ميزة متقدمة. (الافتراضي: سلسلة فارغة) |
