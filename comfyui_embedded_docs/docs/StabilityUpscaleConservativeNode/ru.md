Вот перевод документации на русский язык:

Увеличивает разрешение изображения до 4K с минимальными изменениями. Этот узел использует консервативное масштабирование от Stability AI для повышения разрешения изображения с сохранением исходного содержимого и внесением лишь незначительных изменений.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `изображение` | IMAGE | Да | - | Входное изображение для масштабирования |
| `промпт` | STRING | Да | - | Что вы хотите видеть на выходном изображении. Чёткий, описательный запрос, который ясно определяет элементы, цвета и объекты, приведёт к лучшим результатам. (по умолчанию: пустая строка) |
| `креативность` | FLOAT | Да | 0.2-0.5 | Управляет вероятностью создания дополнительных деталей, не сильно обусловленных исходным изображением. (по умолчанию: 0.35) |
| `seed` | INT | Да | 0-4294967294 | Случайное зерно, используемое для создания шума. (по умолчанию: 0) |
| `негативный промпт` | STRING | Нет | - | Ключевые слова того, что вы не хотите видеть на выходном изображении. Это расширенная функция. (по умолчанию: пустая строка) |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `изображение` | IMAGE | Масштабированное изображение с разрешением 4K |
