Узел `TextEncodeAceStepAudio1.5` подготавливает текстовые метаданные и метаданные, связанные с аудио, для использования с моделью AceStepAudio 1.5. Он принимает описательные теги, текст песни и музыкальные параметры, а затем использует модель CLIP для преобразования их в формат кондиционирования, подходящий для генерации аудио.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `clip` | CLIP | Да | Н/Д | Модель CLIP, используемая для токенизации и кодирования входного текста. |
| `tags` | STRING | Да | Н/Д | Описательные теги для аудио, такие как жанр, настроение или инструменты. Поддерживает многострочный ввод и динамические подсказки. |
| `lyrics` | STRING | Да | Н/Д | Текст песни для аудиодорожки. Поддерживает многострочный ввод и динамические подсказки. |
| `seed` | INT | Нет | от 0 до 18446744073709551615 | Случайное значение зерна для воспроизводимой генерации. Имеет виджет управления после генерации. По умолчанию: 0. |
| `bpm` | INT | Нет | от 10 до 300 | Ударов в минуту (BPM) для генерируемого аудио. По умолчанию: 120. |
| `duration` | FLOAT | Нет | от 0.0 до 2000.0 | Желаемая длительность аудио в секундах. По умолчанию: 120.0. |
| `timesignature` | COMBO | Нет | `"2"`<br>`"3"`<br>`"4"`<br>`"6"` | Музыкальный размер. |
| `language` | COMBO | Нет | `"ar"`<br>`"az"`<br>`"bg"`<br>`"bn"`<br>`"ca"`<br>`"cs"`<br>`"da"`<br>`"de"`<br>`"el"`<br>`"en"`<br>`"es"`<br>`"fa"`<br>`"fi"`<br>`"fr"`<br>`"he"`<br>`"hi"`<br>`"hr"`<br>`"ht"`<br>`"hu"`<br>`"id"`<br>`"is"`<br>`"it"`<br>`"ja"`<br>`"ko"`<br>`"la"`<br>`"lt"`<br>`"ms"`<br>`"ne"`<br>`"nl"`<br>`"no"`<br>`"pa"`<br>`"pl"`<br>`"pt"`<br>`"ro"`<br>`"ru"`<br>`"sa"`<br>`"sk"`<br>`"sr"`<br>`"sv"`<br>`"sw"`<br>`"ta"`<br>`"te"`<br>`"th"`<br>`"tl"`<br>`"tr"`<br>`"uk"`<br>`"ur"`<br>`"vi"`<br>`"yue"`<br>`"zh"`<br>`"unknown"` | Язык входного текста. | `keyscale` | COMBO | Нет | `"C major"`<br>`"C minor"`<br>`"C# major"`<br>`"C# minor"`<br>`"Db major"`<br>`"Db minor"`<br>`"D major"`<br>`"D minor"`<br>`"D# major"`<br>`"D# minor"`<br>`"Eb major"`<br>`"Eb minor"`<br>`"E major"`<br>`"E minor"`<br>`"F major"`<br>`"F minor"`<br>`"F# major"`<br>`"F# minor"`<br>`"Gb major"`<br>`"Gb minor"`<br>`"G major"`<br>`"G minor"`<br>`"G# major"`<br>`"G# minor"`<br>`"Ab major"`<br>`"Ab minor"`<br>`"A major"`<br>`"A minor"`<br>`"A# major"`<br>`"A# minor"`<br>`"Bb major"`<br>`"Bb minor"`<br>`"B major"`<br>`"B minor"` | Музыкальная тональность и лад (мажор или минор). |
| `generate_audio_codes` | BOOLEAN | Нет | Н/Д | Включить LLM, генерирующую аудиокоды. Это может быть медленно, но повысит качество генерируемого аудио. Отключите эту опцию, если вы передаете модели аудиоссылку. По умолчанию: True. |
| `cfg_scale` | FLOAT | Нет | от 0.0 до 100.0 | Шкала управления без классификатора. Более высокие значения заставляют вывод точнее следовать подсказке. По умолчанию: 2.0. |
| `temperature` | FLOAT | Нет | от 0.0 до 2.0 | Температура сэмплирования. Более низкие значения делают вывод более детерминированным. По умолчанию: 0.85. |
| `top_p` | FLOAT | Нет | от 0.0 до 2000.0 | Вероятность сэмплирования ядра (top-p). По умолчанию: 0.9. |

| `min_p` | FLOAT | Нет | от 0.0 до 1.0 | Минимальный порог вероятности для сэмплирования токенов (min-p). По умолчанию: 0.000. |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `CONDITIONING` | CONDITIONING | Данные кондиционирования, которые содержат закодированный текст и аудиопараметры для модели AceStepAudio 1.5. |
