Узел UNetTemporalAttentionMultiply применяет коэффициенты умножения к различным типам механизмов внимания во временной модели UNet. Он изменяет модель, регулируя веса слоев самовнимания и перекрестного внимания, различая структурные и временные компоненты. Это позволяет точно настраивать степень влияния каждого типа внимания на выходные данные модели.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `model` | MODEL | Да | - | Входная модель для модификации с помощью множителей внимания |
| `self_structural` | FLOAT | Нет | 0.0 - 10.0 | Множитель для структурных компонентов самовнимания (по умолчанию: 1.0) |
| `self_temporal` | FLOAT | Нет | 0.0 - 10.0 | Множитель для временных компонентов самовнимания (по умолчанию: 1.0) |
| `cross_structural` | FLOAT | Нет | 0.0 - 10.0 | Множитель для структурных компонентов перекрестного внимания (по умолчанию: 1.0) |
| `cross_temporal` | FLOAT | Нет | 0.0 - 10.0 | Множитель для временных компонентов перекрестного внимания (по умолчанию: 1.0) |

## Выходные параметры

| Имя выходного параметра | Тип данных | Описание |
|-------------|-----------|-------------|
| `model` | MODEL | Модифицированная модель с отрегулированными весами внимания |
