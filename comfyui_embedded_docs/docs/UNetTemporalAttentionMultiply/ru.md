Узел UNetTemporalAttentionMultiply применяет коэффициенты умножения к различным типам механизмов внимания в темпоральной модели UNet. Он модифицирует модель, регулируя веса слоев самовнимания и перекрестного внимания, различая структурные и темпоральные компоненты. Это позволяет точно настраивать степень влияния каждого типа внимания на выходные данные модели.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `model` | MODEL | Да | - | Входная модель для модификации с помощью множителей внимания |
| `self_structural` | FLOAT | Нет | 0.0 - 10.0 | Множитель для структурных компонентов самовнимания (по умолчанию: 1.0) |
| `self_temporal` | FLOAT | Нет | 0.0 - 10.0 | Множитель для темпоральных компонентов самовнимания (по умолчанию: 1.0) |
| `cross_structural` | FLOAT | Нет | 0.0 - 10.0 | Множитель для структурных компонентов перекрестного внимания (по умолчанию: 1.0) |
| `cross_temporal` | FLOAT | Нет | 0.0 - 10.0 | Множитель для темпоральных компонентов перекрестного внимания (по умолчанию: 1.0) |

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `model` | MODEL | Модифицированная модель с настроенными весами внимания |
