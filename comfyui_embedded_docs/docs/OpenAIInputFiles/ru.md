Загружает и форматирует входные файлы для API OpenAI. Этот узел подготавливает текстовые (.txt) и PDF (.pdf) файлы для включения в качестве контекстных входных данных для узла OpenAI Chat Node. Файлы будут прочитаны моделью OpenAI при генерации ответа. Несколько узлов OpenAI Input Files можно объединить в цепочку, чтобы включить несколько файлов в одно сообщение.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `файл` | COMBO | Да | Доступно несколько опций (все файлы .txt и .pdf в папке input размером менее 32 МБ) | Входные файлы для включения в качестве контекста для модели. На данный момент принимаются только текстовые (.txt) и PDF (.pdf) файлы. Размер файлов не должен превышать 32 МБ. |
| `OPENAI_INPUT_FILES` | OPENAI_INPUT_FILES | Нет | Н/Д | Дополнительный файл(ы) для объединения в пакет с файлом, загруженным из этого узла. Позволяет объединять входные файлы в цепочку, чтобы одно сообщение могло содержать несколько входных файлов. |

**Ограничения по файлам:**

- Поддерживаются только файлы .txt и .pdf
- Максимальный размер файла: 32 МБ
- Файлы загружаются из папки input ComfyUI

## Выходные данные

| Имя выходного параметра | Тип данных | Описание |
|-------------------------|------------|----------|
| `OPENAI_INPUT_FILES` | OPENAI_INPUT_FILES | Отформатированные входные файлы, готовые к использованию в качестве контекста для вызовов API OpenAI. |
