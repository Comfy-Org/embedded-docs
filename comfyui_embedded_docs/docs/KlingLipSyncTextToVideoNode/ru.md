Вот перевод документации на русский язык:

Kling Lip Sync Text to Video Node синхронизирует движения губ в видеофайле в соответствии с текстовой подсказкой. Узел принимает входное видео и генерирует новое видео, в котором движения губ персонажа согласованы с предоставленным текстом. Узел использует синтез речи для создания естественно выглядящей синхронизации речи.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `video` | VIDEO | Да | - | Входной видеофайл для синхронизации губ |
| `text` | STRING | Да | - | Текстовое содержимое для генерации видео с синхронизацией губ. Требуется, когда режим установлен в text2video. Максимальная длина — 120 символов. |
| `voice` | COMBO | Нет | "Melody"<br>"Bella"<br>"Aria"<br>"Ethan"<br>"Ryan"<br>"Dorothy"<br>"Nathan"<br>"Lily"<br>"Aaron"<br>"Emma"<br>"Grace"<br>"Henry"<br>"Isabella"<br>"James"<br>"Katherine"<br>"Liam"<br>"Mia"<br>"Noah"<br>"Olivia"<br>"Sophia" | Выбор голоса для аудио синхронизации губ (по умолчанию: "Melody") |
| `voice_speed` | FLOAT | Нет | 0.8-2.0 | Скорость речи. Допустимый диапазон: 0.8~2.0, точность до одного десятичного знака. (по умолчанию: 1) |

**Требования к видео:**

- Размер видеофайла не должен превышать 100 МБ
- Высота/ширина должны быть от 720px до 1920px
- Длительность должна быть от 2 до 10 секунд

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `output` | VIDEO | Сгенерированное видео с синхронизированным по губам аудио |
| `video_id` | STRING | Уникальный идентификатор сгенерированного видео |
| `duration` | STRING | Информация о длительности сгенерированного видео |
