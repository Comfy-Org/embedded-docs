گره SaveLoRA یک مدل LoRA (انطباق رتبه-پایین) را در یک فایل ذخیره می‌کند. این گره یک مدل LoRA را به عنوان ورودی دریافت کرده و آن را در قالب فایل `.safetensors` در دایرکتوری خروجی می‌نویسد. می‌توانید یک پیشوند برای نام فایل و یک شماره گام اختیاری برای درج در نام نهایی فایل مشخص کنید.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|---------|----------|--------|--------|---------|
| `lora` | LORA_MODEL | بله | نامشخص | مدل LoRA که باید ذخیره شود. از مدل دارای لایه‌های LoRA استفاده نکنید. |
| `prefix` | STRING | بله | نامشخص | پیشوندی که برای فایل LoRA ذخیره‌شده استفاده می‌شود (پیش‌فرض: "loras/ComfyUI_trained_lora"). |
| `steps` | INT | خیر | نامشخص | اختیاری: تعداد گام‌هایی که LoRA آموزش دیده است، برای نام‌گذاری فایل ذخیره‌شده استفاده می‌شود. |

**توجه:** ورودی `lora` باید یک مدل LoRA خالص باشد. از ارائه یک مدل پایه که لایه‌های LoRA به آن اعمال شده است خودداری کنید.

## خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-----------|----------|---------|
| *هیچکدام* | نامشخص | این گره هیچ داده‌ای به گردش کار خروجی نمی‌دهد. این یک گره خروجی است که یک فایل را روی دیسک ذخیره می‌کند. |
