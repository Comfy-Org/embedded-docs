أنت خبير في الترجمة التقنية متخصص في توثيق عُقد ComfyUI من الإنجليزية إلى العربية.

## قواعد الترجمة

1. **المحتوى الذي يجب عدم ترجمته:**
   - أسماء المعاملات بين علامات الاقتباس الخلفية: `image`, `seed`, `model`
   - أنواع البيانات بالأحرف الكبيرة: IMAGE, STRING, INT, FLOAT, MODEL, CONDITIONING, إلخ
   - القيم في عمود Range: الأرقام، "auto"، أسماء الخيارات
   - الكود، مسارات الملفات

2. **المحتوى الذي يجب ترجمته:**
   - عناوين الأقسام: ## نظرة عامة, ## المدخلات, ## المخرجات
   - جميع النصوص الوصفية والتوضيحية
   - أوصاف المعاملات

3. **جودة الترجمة:**
   - استخدام اللغة العربية الفصحى المعاصرة
   - الحفاظ على نبرة احترافية ولكن سهلة الفهم
   - ضمان الدقة التقنية
   - استخدام المصطلحات التقنية العربية القياسية

4. **التنسيق:**
   - الحفاظ على جميع تنسيقات Markdown
   - الحفاظ على بنية الجداول
   - عدم إضافة أي ملاحظة أو رابط في بداية الوثيقة (سيتم إضافتها تلقائيًا)

الرجاء ترجمة الوثيقة التالية إلى العربية، دون تضمين الملاحظة الأولية للوثيقة:

يقوم بإنشاء فيديوهات من النصوص التوجيهية باستخدام واجهة برمجة تطبيقات Veo 3 من Google. تدعم هذه العقدة نماذج Veo 3 المتعددة، بما في ذلك الإصدارات السريعة والخفيفة، وتتيح لك تحديد دقة الفيديو ومدته وإنشاء الصوت.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|-----------|-----------|----------|-------|-------------|
| `النص_الوصفي` | STRING | نعم | - | وصف نصي للفيديو (الافتراضي: "") |
| `نسبة الأبعاد` | COMBO | نعم | "16:9"<br>"9:16" | نسبة العرض إلى الارتفاع للفيديو الناتج (الافتراضي: "16:9") |
| `الدقة` | COMBO | لا | "720p"<br>"1080p"<br>"4k" | دقة الفيديو الناتج. دقة 4K غير متوفرة لنموذجي veo-3.1-lite و veo-3.0. (الافتراضي: "720p") |
| `النص السلبي` | STRING | لا | - | نص توجيهي سلبي لتوجيه ما يجب تجنبه في الفيديو (الافتراضي: "") |
| `المدة بالثواني` | INT | لا | 4-8 | مدة الفيديو الناتج بالثواني، بخطوات مقدارها 2 (الافتراضي: 8) |
| `تحسين النص` | BOOLEAN | لا | - | هذه المعلمة مهملة ولا يتم استخدامها. (الافتراضي: True) |
| `إنشاء الأشخاص` | COMBO | لا | "ALLOW"<br>"BLOCK" | ما إذا كان سيتم السماح بإنشاء أشخاص في الفيديو (الافتراضي: "ALLOW") |
| `البذرة` | INT | لا | 0-4294967295 | بذرة توليد الفيديو (0 لعشوائي) (الافتراضي: 0) |
| `الصورة` | IMAGE | لا | - | صورة مرجعية اختيارية لتوجيه إنشاء الفيديو |
| `النموذج` | COMBO | لا | "veo-3.1-generate"<br>"veo-3.1-fast-generate"<br>"veo-3.1-lite"<br>"veo-3.0-generate-001"<br>"veo-3.0-fast-generate-001" | نموذج Veo 3 المستخدم لإنشاء الفيديو (الافتراضي: "veo-3.0-generate-001") |
| `إنشاء الصوت` | BOOLEAN | لا | - | إنشاء صوت للفيديو. مدعوم من جميع نماذج Veo 3. (الافتراضي: False) |

**ملاحظة:** المعامل `enhance_prompt` مهمل ويتم تجاهل قيمته. تقوم العقدة دائمًا بتحسين النص التوجيهي داخليًا. بالإضافة إلى ذلك، يتم تطبيق المعامل `resolution` فقط عند استخدام نموذج من إصدار veo-3.1؛ ويتم تجاهله لنماذج veo-3.0.
