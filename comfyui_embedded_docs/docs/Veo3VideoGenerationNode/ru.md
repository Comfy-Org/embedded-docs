Генерирует видео по текстовым описаниям с использованием API Google Veo 3. Этот узел поддерживает несколько моделей Veo 3, включая быстрые и облегченные варианты, а также позволяет задавать разрешение видео, длительность и генерацию звука.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `prompt` | STRING | Да | - | Текстовое описание видео (по умолчанию: "") |
| `aspect_ratio` | COMBO | Да | "16:9"<br>"9:16" | Соотношение сторон выходного видео (по умолчанию: "16:9") |
| `разрешение` | COMBO | Нет | "720p"<br>"1080p"<br>"4k" | Разрешение выходного видео. 4K недоступно для моделей veo-3.1-lite и veo-3.0. (по умолчанию: "720p") |
| `negative_prompt` | STRING | Нет | - | Отрицательный текстовый промпт, указывающий, чего следует избегать в видео (по умолчанию: "") |
| `duration_seconds` | INT | Нет | 4-8 | Длительность выходного видео в секундах, с шагом 2 (по умолчанию: 8) |
| `enhance_prompt` | BOOLEAN | Нет | - | Этот параметр устарел и игнорируется. (по умолчанию: True) |
| `person_generation` | COMBO | Нет | "ALLOW"<br>"BLOCK" | Разрешить или запретить генерацию людей в видео (по умолчанию: "ALLOW") |
| `seed` | INT | Нет | 0-4294967295 | Сид для генерации видео (0 для случайного) (по умолчанию: 0) |
| `image` | IMAGE | Нет | - | Дополнительное референсное изображение для управления генерацией видео |
| `model` | COMBO | Нет | "veo-3.1-generate"<br>"veo-3.1-fast-generate"<br>"veo-3.1-lite"<br>"veo-3.0-generate-001"<br>"veo-3.0-fast-generate-001" | Модель Veo 3 для генерации видео (по умолчанию: "veo-3.0-generate-001") |
| `generate_audio` | BOOLEAN | Нет | - | Генерировать звуковое сопровождение для видео. Поддерживается всеми моделями Veo 3. (по умолчанию: False) |

**Примечание:** Параметр `enhance_prompt` устарел, его значение игнорируется. Узел всегда внутренне улучшает промпт. Кроме того, параметр `resolution` применяется только при использовании модели veo-3.1; для моделей veo-3.0 он игнорируется.

## Выходные данные

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `output` | VIDEO | Сгенерированный видеофайл |
