### مرور کلی

با استفاده از API Veo 3 گوگل، ویدئوهایی از توضیحات متنی تولید می‌کند. این گره از چندین مدل Veo 3 از جمله انواع سریع و سبک پشتیبانی می‌کند و به شما امکان می‌دهد رزولوشن، مدت زمان و تولید صدا را مشخص کنید.

## ورودی‌ها

| پارامتر | نوع داده | اجباری | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `پرامپت` | STRING | بله | - | توضیحات متنی ویدئو (پیش‌فرض: "") |
| `نسبت تصویر` | COMBO | بله | "16:9"<br>"9:16" | نسبت ابعاد ویدئوی خروجی (پیش‌فرض: "16:9") |
| `وضوح تصویر` | COMBO | خیر | "720p"<br>"1080p"<br>"4k" | رزولوشن ویدئوی خروجی. رزولوشن 4K برای مدل‌های veo-3.1-lite و veo-3.0 در دسترس نیست. (پیش‌فرض: "720p") |
| `پرامپت منفی` | STRING | خیر | - | توضیحات متنی منفی برای مشخص کردن مواردی که باید در ویدئو اجتناب شود (پیش‌فرض: "") |
| `مدت زمان (ثانیه)` | INT | خیر | 4-8 | مدت زمان ویدئوی خروجی بر حسب ثانیه، با گام‌های ۲ تایی (پیش‌فرض: ۸) |
| `enhance_prompt` | BOOLEAN | خیر | - | این پارامتر منسوخ شده و نادیده گرفته می‌شود. (پیش‌فرض: True) |
| `تولید افراد` | COMBO | خیر | "ALLOW"<br>"BLOCK" | آیا تولید افراد در ویدئو مجاز است یا خیر (پیش‌فرض: "ALLOW") |
| `seed` | INT | خیر | 0-4294967295 | دانه (seed) برای تولید ویدئو (0 برای تصادفی) (پیش‌فرض: 0) |
| `تصویر` | IMAGE | خیر | - | تصویر مرجع اختیاری برای هدایت تولید ویدئو |
| `مدل` | COMBO | خیر | "veo-3.1-generate"<br>"veo-3.1-fast-generate"<br>"veo-3.1-lite"<br>"veo-3.0-generate-001"<br>"veo-3.0-fast-generate-001" | مدل Veo 3 برای تولید ویدئو (پیش‌فرض: "veo-3.0-generate-001") |
| `تولید صدا` | BOOLEAN | خیر | - | تولید صدا برای ویدئو. توسط تمام مدل‌های Veo 3 پشتیبانی می‌شود. (پیش‌فرض: False) |

**نکته:** پارامتر `enhance_prompt` منسوخ شده و مقدار آن نادیده گرفته می‌شود. گره همیشه به‌صورت داخلی پرامپت را بهبود می‌بخشد. همچنین، پارامتر `resolution` فقط هنگام استفاده از مدل veo-3.1 اعمال می‌شود و برای مدل‌های veo-3.0 نادیده گرفته می‌شود.
