Узел "Replace Text" выполняет простую замену текста. Он ищет указанный фрагмент текста во входных данных и заменяет каждое его вхождение новым текстом. Операция применяется ко всем текстовым входам, переданным узлу.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `text` | STRING | Да | - | Текст для обработки. |
| `find` | STRING | Да | - | Текст для поиска (по умолчанию: пустая строка). |
| `replace` | STRING | Да | - | Текст для замены (по умолчанию: пустая строка). |

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `text` | STRING | Обработанный текст, в котором все вхождения текста `find` заменены на текст `replace`. |
