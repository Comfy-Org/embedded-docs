# ترجمة وثيقة عقدة "استبدال النص" (Replace Text)

عقدة استبدال النص تقوم بعملية استبدال نصي بسيطة. تبحث عن جزء نصي محدد داخل النص المُدخل وتستبدل كل ظهور لهذا النص بنص جديد. يتم تطبيق العملية على جميع مدخلات النص المقدمة للعقدة.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|---------|--------------|--------|--------|-------|
| `text` | STRING | نعم | - | النص المراد معالجته. |
| `find` | STRING | نعم | - | النص المراد البحث عنه (القيمة الافتراضية: سلسلة فارغة). |
| `replace` | STRING | نعم | - | النص المراد الاستبدال به (القيمة الافتراضية: سلسلة فارغة). |

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|------------|--------------|-------|
| `text` | STRING | النص المعالج مع استبدال جميع ظهورات النص الموجود في `find` بالنص الموجود في `replace`. |
