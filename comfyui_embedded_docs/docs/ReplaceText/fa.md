### نمای کلی

گره «جایگزینی متن» یک عملیات ساده‌ی جایگزینی متن را انجام می‌دهد. این گره درون متن ورودی، یک عبارت متنی مشخص را جستجو کرده و هر نمونه از آن را با یک عبارت متنی جدید جایگزین می‌کند. این عملیات بر روی تمام ورودی‌های متنی ارائه‌شده به گره اعمال می‌شود.

### ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `text` | STRING | بله | - | متنی که باید پردازش شود. |
| `یافتن` | STRING | بله | - | متنی که باید جستجو شود (پیش‌فرض: رشته‌ی خالی). |
| `جایگزین` | STRING | بله | - | متنی که باید جایگزین شود (پیش‌فرض: رشته‌ی خالی). |

### خروجی‌ها
