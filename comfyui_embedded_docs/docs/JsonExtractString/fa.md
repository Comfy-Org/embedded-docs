گره JsonExtractString یک رشته متنی حاوی داده‌های JSON را می‌خواند و مقدار مرتبط با یک کلید مشخص را استخراج می‌کند. این گره مقدار استخراج‌شده را به یک رشته تبدیل می‌کند. اگر JSON نامعتبر باشد، کلید پیدا نشود، یا مقدار null باشد، گره یک رشته خالی برمی‌گرداند.

## ورودی‌ها

| پارامتر | نوع داده | اجباری | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `json_string` | STRING | بله | نامشخص | متنی حاوی داده‌های JSON که باید تجزیه شود. |
| `key` | STRING | بله | نامشخص | کلید مشخصی که مقدار رشته‌ای آن را می‌خواهید از شیء JSON استخراج کنید. |

**نکته:** این گره فقط مقادیر را از اشیاء JSON (دیکشنری‌ها) استخراج می‌کند. اگر JSON تجزیه‌شده یک شیء نباشد یا کلید مشخص‌شده در آن وجود نداشته باشد، خروجی یک رشته خالی خواهد بود.

## خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-------------|-----------|-------------|
| `output` | STRING | مقدار رشته‌ای استخراج‌شده از JSON برای کلید مشخص‌شده، یا یک رشته خالی در صورت شکست استخراج. |
