ModelMergeBlocks предназначен для продвинутых операций слияния моделей, позволяя объединять две модели с настраиваемыми коэффициентами смешивания для различных частей моделей. Этот узел облегчает создание гибридных моделей путем выборочного объединения компонентов из двух исходных моделей на основе заданных параметров.

## Входные данные

| Параметр | Тип данных | Описание |
|-----------|-------------|-------------|
| `model1`  | `MODEL`     | Первая модель для слияния. Она служит базовой моделью, к которой применяются патчи из второй модели. |
| `model2`  | `MODEL`     | Вторая модель, из которой извлекаются патчи и применяются к первой модели в соответствии с заданными коэффициентами смешивания. |
| `input`   | `FLOAT`     | Задает коэффициент смешивания для входного слоя моделей. Определяет, насколько сильно входной слой второй модели сливается с первой моделью. |
| `middle`  | `FLOAT`     | Определяет коэффициент смешивания для средних слоев моделей. Этот параметр контролирует уровень интеграции средних слоев моделей. |
| `out`     | `FLOAT`     | Определяет коэффициент смешивания для выходного слоя моделей. Влияет на конечный результат, регулируя вклад выходного слоя второй модели. |

## Выходные данные

| Параметр | Тип данных | Описание |
|-----------|-------------|-------------|
| `model`   | MODEL     | Результирующая объединенная модель, представляющая собой гибрид двух входных моделей с примененными патчами в соответствии с заданными коэффициентами смешивания. |
