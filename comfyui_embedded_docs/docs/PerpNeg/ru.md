Вот перевод документации на русский язык:

Узел PerpNeg применяет перпендикулярное негативное управление к процессу сэмплирования модели. Этот узел изменяет функцию конфигурации модели для корректировки предсказаний шума с использованием негативного кондиционирования и масштабирующих коэффициентов. Данный узел устарел и заменён узлом PerpNegGuider для улучшенной функциональности.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `модель` | MODEL | Да | - | Модель, к которой применяется перпендикулярное негативное управление |
| `пустое_условие` | CONDITIONING | Да | - | Пустое кондиционирование, используемое для расчётов негативного управления |
| `масштаб_отриц` | FLOAT | Нет | 0.0 - 100.0 | Масштабирующий коэффициент для негативного управления (по умолчанию: 1.0) |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `модель` | MODEL | Модифицированная модель с применённым перпендикулярным негативным управлением |

**Примечание**: Этот узел устарел и заменён узлом PerpNegGuider. Он помечен как экспериментальный и не должен использоваться в рабочих процессах.
