گره PerpNeg، هدایت منفی عمود بر (perpendicular negative guidance) را به فرآیند نمونه‌گیری یک مدل اعمال می‌کند. این گره، تابع پیکربندی مدل را برای تنظیم پیش‌بینی نویز با استفاده از شرط‌گذاری منفی و عوامل مقیاس‌دهی تغییر می‌دهد. این گره منسوخ شده و برای عملکرد بهبودیافته با گره PerpNegGuider جایگزین شده است.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|---------|----------|--------|--------|---------|
| `model` | MODEL | بله | - | مدلی که هدایت منفی عمود بر روی آن اعمال می‌شود |
| `empty_conditioning` | CONDITIONING | بله | - | شرط‌گذاری خالی که برای محاسبات هدایت منفی استفاده می‌شود |
| `neg_scale` | FLOAT | خیر | 0.0 - 100.0 | عامل مقیاس‌دهی برای هدایت منفی (پیش‌فرض: 1.0) |
