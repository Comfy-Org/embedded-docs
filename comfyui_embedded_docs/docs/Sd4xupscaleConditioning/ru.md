Данный узел специализируется на повышении разрешения изображений с помощью 4-кратного масштабирования, используя элементы кондиционирования для улучшения результата. Он применяет методы диффузии для увеличения изображений, позволяя при этом регулировать коэффициент масштабирования и уровень шумовой аугментации для точной настройки процесса улучшения.

## Входные параметры

| Параметр             | Comfy dtype    | Описание |
|----------------------|----------------|-------------|
| `images`             | `IMAGE`        | Входные изображения для масштабирования. Этот параметр критически важен, так как напрямую влияет на качество и разрешение выходных изображений. |
| `positive`           | `CONDITIONING` | Позитивные элементы кондиционирования, направляющие процесс масштабирования к желаемым атрибутам или характеристикам выходных изображений. |
| `negative`           | `CONDITIONING` | Негативные элементы кондиционирования, которых следует избегать в процессе масштабирования, помогая отвести результат от нежелательных атрибутов или характеристик. |
| `scale_ratio`        | `FLOAT`        | Определяет коэффициент, на который увеличивается разрешение изображения. Более высокий коэффициент масштабирования приводит к увеличению выходного изображения, позволяя добиться большей детализации и четкости. |
| `noise_augmentation` | `FLOAT`        | Управляет уровнем шумовой аугментации, применяемой в процессе масштабирования. Это может использоваться для внесения вариативности и повышения устойчивости выходных изображений. |

## Выходные параметры

| Параметр     | Тип данных    | Описание |
|---------------|--------------|-------------|
| `positive`    | `CONDITIONING` | Уточненные позитивные элементы кондиционирования, полученные в результате процесса масштабирования. |
| `negative`    | `CONDITIONING` | Уточненные негативные элементы кондиционирования, полученные в результате процесса масштабирования. |
| `latent`      | `LATENT`       | Скрытое представление, сгенерированное в процессе масштабирования, которое может быть использовано в дальнейшей обработке или обучении модели. |
