این گره از API گروک برای ویرایش یک ویدیوی موجود بر اساس یک توضیح متنی استفاده می‌کند. ویدیوی شما را بارگذاری می‌کند، درخواستی به مدل هوش مصنوعی برای تغییر آن مطابق با توضیحات شما ارسال می‌کند، و ویدیوی تازه تولید شده را بازمی‌گرداند.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `model` | COMBO | بله | `"grok-imagine-video"`<br>`"grok-imagine-video-beta"` | مدل هوش مصنوعی مورد استفاده برای ویرایش ویدیو (پیش‌فرض: `"grok-imagine-video"`). |
| `prompt` | STRING | بله | نامشخص | توضیح متنی ویدیوی مورد نظر. |
| `video` | VIDEO | بله | نامشخص | ویدیوی ورودی که باید ویرایش شود. حداکثر مدت زمان پشتیبانی شده ۸.۷ ثانیه و حداکثر حجم فایل ۵۰ مگابایت است. |
| `seed` | INT | خیر | ۰ تا ۲۱۴۷۴۸۳۶۴۷ | یک مقدار دانه (seed) برای تعیین اینکه آیا گره باید دوباره اجرا شود. نتایج واقعی بدون توجه به مقدار دانه، غیرقطعی هستند (پیش‌فرض: ۰). |

**محدودیت‌ها:**

* ویدیوی ورودی `video` باید بین ۱ تا ۸.۷ ثانیه مدت زمان داشته باشد.
* حجم فایل ویدیوی ورودی `video` نباید از ۵۰ مگابایت تجاوز کند.
* `prompt` نباید خالی باشد.
