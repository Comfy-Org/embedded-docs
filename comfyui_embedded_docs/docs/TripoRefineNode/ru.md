Вот перевод документации на русский язык:

Узел TripoRefineNode дорабатывает черновые 3D-модели, созданные специально моделями Tripo v1.4. Он принимает идентификатор задачи модели и обрабатывает его через API Tripo для создания улучшенной версии модели. Этот узел предназначен для работы исключительно с черновыми моделями, созданными моделями Tripo v1.4.

## Входные данные

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `идентификатор_задачи_модели` | MODEL_TASK_ID | Да | - | Должен быть идентификатором задачи модели Tripo v1.4 |

**Примечание:** Этот узел принимает только черновые модели, созданные моделями Tripo v1.4. Использование моделей других версий может привести к ошибкам.

## Выходные данные

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `model_file` | STRING | Путь к файлу или ссылка на доработанную модель (только для обратной совместимости) |
| `model task_id` | MODEL_TASK_ID | Идентификатор задачи для операции доработки модели |
| `GLB` | FILE3DGLB | Доработанная 3D-модель в формате GLB |
