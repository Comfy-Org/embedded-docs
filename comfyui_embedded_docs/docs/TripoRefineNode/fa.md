### بررسی اجمالی

گره TripoRefineNode مدل‌های سه‌بعدی پیش‌نویس ایجاد شده به‌طور خاص توسط مدل‌های Tripo نسخه 1.4 را بهبود می‌بخشد. این گره یک شناسه وظیفه مدل را دریافت کرده و آن را از طریق API Tripo پردازش می‌کند تا نسخه بهبودیافته‌ای از مدل تولید کند. این گره به‌گونه‌ای طراحی شده است که منحصراً با مدل‌های پیش‌نویس تولید شده توسط مدل‌های Tripo v1.4 کار کند.

### ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `model_task_id` | MODEL_TASK_ID | بله | - | باید یک مدل Tripo نسخه 1.4 باشد |

**توجه:** این گره فقط مدل‌های پیش‌نویس ایجاد شده توسط مدل‌های Tripo v1.4 را می‌پذیرد. استفاده از مدل‌های سایر نسخه‌ها ممکن است منجر به خطا شود.

### خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-------------|-----------|-------------|
| `model_file` | STRING | مسیر فایل یا ارجاع به مدل بهبودیافته (فقط برای سازگاری با نسخه‌های قبلی) |
| `model task_id` | MODEL_TASK_ID | شناسه وظیفه برای عملیات مدل بهبودیافته |
| `GLB` | FILE3DGLB | مدل سه‌بعدی بهبودیافته در قالب GLB |
