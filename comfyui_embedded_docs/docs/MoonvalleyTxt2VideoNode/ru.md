Вот перевод документации узла Moonvalley Marey Text to Video на русский язык:

Узел Moonvalley Marey Text to Video генерирует видеоконтент на основе текстовых описаний с помощью API Moonvalley. Он принимает текстовый запрос и преобразует его в видео с настраиваемыми параметрами разрешения, качества и стиля. Узел обрабатывает весь процесс — от отправки запроса на генерацию до загрузки итогового видео.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `prompt` | STRING | Да | - | Текстовое описание генерируемого видеоконтента |
| `negative_prompt` | STRING | Нет | - | Текст негативного запроса (по умолчанию: обширный список исключаемых элементов, таких как синтетика, смена сцены, артефакты, шум и т.д.) |
| `resolution` | STRING | Нет | "16:9 (1920 x 1080)"<br>"9:16 (1080 x 1920)"<br>"1:1 (1152 x 1152)"<br>"4:3 (1536 x 1152)"<br>"3:4 (1152 x 1536)"<br>"21:9 (2560 x 1080)" | Разрешение выходного видео (по умолчанию: "16:9 (1920 x 1080)") |
| `prompt_adherence` | FLOAT | Нет | 1.0-20.0 | Масштаб управления генерацией (по умолчанию: 4.0) |
| `seed` | INT | Нет | 0-4294967295 | Значение случайного зерна (по умолчанию: 9) |
| `steps` | INT | Нет | 1-100 | Количество шагов инференса (по умолчанию: 33) |

## Выходные данные

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `video` | VIDEO | Сгенерированное видео на основе текстового запроса |
