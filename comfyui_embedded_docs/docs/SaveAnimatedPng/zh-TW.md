# SaveAnimatedPNG 節點

SaveAnimatedPNG 節點專為從一系列影格建立和儲存動畫 PNG 影像而設計。它負責將單獨的影像影格組合成連貫的動畫，並允許自訂影格持續時間、循環播放和中繼資料包含。

## 輸入

| 欄位               | 資料類型      | 說明                                                                         |
|-------------------|-------------|-------------------------------------------------------------------------------------|
| `images`          | `IMAGE`     | 要處理並儲存為動畫 PNG 的影像列表。列表中的每個影像代表動畫中的一個影格。 |
| `檔名前綴` | `STRING`    | 指定輸出檔案的基本名稱，將用作產生的動畫 PNG 檔案的前綴。 |
| `每秒影格數`             | `FLOAT`     | 動畫的每秒影格數，控制影格顯示的速度。 |
| `壓縮等級`  | `INT`       | 應用於動畫 PNG 檔案的壓縮級別，影響檔案大小和影像清晰度。 |
