# ترجمة وثيقة عقدة Create Hook LoRA

عقدة إنشاء خطاف LoRA (Create Hook LoRA) تقوم بإنشاء كائنات خطاف لتطبيق تعديلات LoRA (التكيف منخفض الرتبة) على النماذج. تقوم بتحميل ملف LoRA محدد وإنشاء خطافات يمكنها ضبط قوة النموذج و CLIP، ثم تدمج هذه الخطافات مع أي خطافات موجودة مسبقًا تمرر إليها. تدير العقدة تحميل LoRA بكفاءة من خلال تخزين ملفات LoRA المحملة مسبقًا في ذاكرة التخزين المؤقت لتجنب العمليات المتكررة.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|---------|---------------|--------|---------|-------|
| `اسم LoRA` | STRING | نعم | خيارات متعددة متاحة | اسم ملف LoRA المراد تحميله من دليل loras |
| `قوة النموذج` | FLOAT | نعم | -20.0 إلى 20.0 | مضاعف القوة لتعديلات النموذج (الافتراضي: 1.0) |
| `قوة القص` | FLOAT | نعم | -20.0 إلى 20.0 | مضاعف القوة لتعديلات CLIP (الافتراضي: 1.0) |
| `الخطافات السابقة` | HOOKS | لا | غير متاح | مجموعة خطافات موجودة مسبقًا اختيارية لدمجها مع خطافات LoRA الجديدة |

**قيود المعاملات:**

- إذا تم ضبط كل من `strength_model` و `strength_clip` على 0، ستتخطى العقدة إنشاء خطافات LoRA جديدة وستعيد الخطافات الموجودة دون تغيير
- تقوم العقدة بتخزين آخر ملف LoRA تم تحميله في ذاكرة التخزين المؤقت لتحسين الأداء عند استخدام نفس LoRA بشكل متكرر

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|-------------|---------------|---------|
| `HOOKS` | HOOKS | مجموعة خطافات تحتوي على خطافات LoRA المدمجة وأي خطافات سابقة |
