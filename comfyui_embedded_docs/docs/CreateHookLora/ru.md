Вот перевод документации на русский язык:

Узел Create Hook LoRA создает объекты-хуки для применения модификаций LoRA (Low-Rank Adaptation) к моделям. Он загружает указанный LoRA-файл и создает хуки, которые могут настраивать силу воздействия на модель и CLIP, а затем объединяет эти хуки с любыми существующими хуками, переданными в узел. Узел эффективно управляет загрузкой LoRA, кэшируя ранее загруженные LoRA-файлы, чтобы избежать повторных операций.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `lora_name` | STRING | Да | Доступно несколько опций | Имя LoRA-файла для загрузки из каталога loras |
| `strength_model` | FLOAT | Да | -20.0 до 20.0 | Множитель силы для модификаций модели (по умолчанию: 1.0) |
| `strength_clip` | FLOAT | Да | -20.0 до 20.0 | Множитель силы для модификаций CLIP (по умолчанию: 1.0) |
| `prev_hooks` | HOOKS | Нет | N/A | Необязательная существующая группа хуков для объединения с новыми LoRA-хуками |

**Ограничения параметров:**

- Если оба параметра `strength_model` и `strength_clip` установлены в 0, узел пропустит создание новых LoRA-хуков и вернет существующие хуки без изменений
- Узел кэширует последний загруженный LoRA-файл для оптимизации производительности при повторном использовании одного и того же LoRA

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `HOOKS` | HOOKS | Группа хуков, содержащая объединенные LoRA-хуки и все предыдущие хуки |
