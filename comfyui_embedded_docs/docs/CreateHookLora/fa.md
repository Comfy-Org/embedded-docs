### خلاصه

گره Create Hook LoRA، اشیاء هوک را برای اعمال تغییرات LoRA (تطبیق رتبه پایین) بر روی مدل‌ها تولید می‌کند. این گره یک فایل LoRA مشخص را بارگذاری کرده و هوک‌هایی ایجاد می‌کند که می‌توانند قدرت مدل و CLIP را تنظیم کنند، سپس این هوک‌ها را با هر هوک موجود دیگری که به آن ارسال شده است ترکیب می‌کند. این گره با ذخیره‌سازی (کش کردن) فایل‌های LoRA که قبلاً بارگذاری شده‌اند، از انجام عملیات تکراری جلوگیری کرده و کارایی را بهبود می‌بخشد.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|---------|----------|--------|--------|----------|
| `lora_name` | STRING | بله | چندین گزینه موجود | نام فایل LoRA که باید از دایرکتوری loras بارگذاری شود |
| `strength_model` | FLOAT | بله | 20.0- تا 20.0 | ضریب قدرت برای تنظیمات مدل (پیش‌فرض: 1.0) |
| `strength_clip` | FLOAT | بله | 20.0- تا 20.0 | ضریب قدرت برای تنظیمات CLIP (پیش‌فرض: 1.0) |
| `prev_hooks` | HOOKS | خیر | ناموجود | گروه هوک موجود اختیاری که با هوک‌های جدید LoRA ترکیب می‌شود |

**محدودیت‌های پارامتر:**

- اگر هر دو `strength_model` و `strength_clip` روی 0 تنظیم شوند، گره از ایجاد هوک‌های جدید LoRA صرف‌نظر کرده و هوک‌های موجود را بدون تغییر برمی‌گرداند
- این گره آخرین فایل LoRA بارگذاری شده را ذخیره می‌کند تا در صورت استفاده مکرر از یک LoRA، عملکرد بهینه شود
