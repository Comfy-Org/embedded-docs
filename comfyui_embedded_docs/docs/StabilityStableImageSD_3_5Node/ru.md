# Узел Stability Stable Image SD 3.5

Этот узел генерирует изображения синхронно с использованием модели Stable Diffusion 3.5 от Stability AI. Он создает изображения на основе текстовых подсказок, а также может изменять существующие изображения, если они переданы на вход. Узел поддерживает различные соотношения сторон и предустановки стилей для настройки выходного результата.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `prompt` | STRING | Да | - | Что вы хотите видеть на выходном изображении. Четкая, описательная подсказка, которая ясно определяет элементы, цвета и объекты, приведет к лучшим результатам. (по умолчанию: пустая строка) |
| `model` | COMBO | Да | `sd3.5-large`<br>`sd3.5-large-turbo`<br>`sd3.5-medium` | Модель Stable Diffusion 3.5 для генерации. |
| `aspect_ratio` | COMBO | Да | `16:9`<br>`1:1`<br>`21:9`<br>`2:3`<br>`3:2`<br>`4:5`<br>`5:4`<br>`9:16`<br>`9:21` | Соотношение сторон генерируемого изображения. (по умолчанию: 1:1) |
| `style_preset` | COMBO | Нет | `3d-model`<br>`analog-film`<br>`anime`<br>`cinematic`<br>`comic-book`<br>`digital-art`<br>`enhance`<br>`fantasy-art`<br>`isometric`<br>`line-art`<br>`low-poly`<br>`modeling-compound`<br>`neon-punk`<br>`origami`<br>`photographic`<br>`pixel-art`<br>`tile-texture`<br>`None` | Необязательный желаемый стиль генерируемого изображения. Выберите "None" для отсутствия предустановки стиля. |
| `cfg_scale` | FLOAT | Да | 1.0 до 10.0 | Насколько строго процесс диффузии следует тексту подсказки (более высокие значения сохраняют изображение ближе к вашей подсказке). (по умолчанию: 4.0) |
| `seed` | INT | Да | 0 до 4294967294 | Случайное зерно, используемое для создания шума. (по умолчанию: 0) |
| `image` | IMAGE | Нет | - | Необязательное входное изображение для генерации "изображение-в-изображение". При его наличии узел переключается в режим "изображение-в-изображение", а параметр `aspect_ratio` игнорируется. |
| `negative_prompt` | STRING | Нет | - | Ключевые слова того, что вы не хотите видеть на выходном изображении. Это расширенная функция. (по умолчанию: пустая строка) |
| `image_denoise` | FLOAT | Нет | 0.0 до 1.0 | Уровень шумоподавления входного изображения; 0.0 дает изображение, идентичное входному, 1.0 — как если бы изображение не было предоставлено. (по умолчанию: 0.5) Этот параметр используется только при наличии `image`. |

**Примечание:** При наличии `image` узел переключается в режим генерации "изображение-в-изображение", а параметр `aspect_ratio` автоматически определяется из входного изображения. Если `image` не предоставлено, параметр `image_denoise` игнорируется.

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `image` | IMAGE | Сгенерированное или измененное изображение. |
