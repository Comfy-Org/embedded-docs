این گره با استفاده از مدل Stable Diffusion 3.5 شرکت Stability AI، تصاویر را به‌صورت همزمان تولید می‌کند. این گره بر اساس توضیحات متنی (prompt) تصاویر را می‌سازد و در صورت ارائه تصویر ورودی، می‌تواند تصاویر موجود را نیز تغییر دهد. این گره از نسبت‌های ابعاد مختلف و سبک‌های از پیش تعیین‌شده برای سفارشی‌سازی خروجی پشتیبانی می‌کند.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|---------|----------|--------|--------|---------|
| `پرامپت` | STRING | بله | - | آنچه می‌خواهید در تصویر خروجی ببینید. یک توضیح قوی و توصیفی که عناصر، رنگ‌ها و سوژه‌ها را به‌وضوح تعریف کند، به نتایج بهتری منجر می‌شود. (پیش‌فرض: رشته خالی) |
| `مدل` | COMBO | بله | `sd3.5-large`<br>`sd3.5-large-turbo`<br>`sd3.5-medium` | مدل Stable Diffusion 3.5 مورد استفاده برای تولید تصویر. |
| `نسبت تصویر` | COMBO | بله | `16:9`<br>`1:1`<br>`21:9`<br>`2:3`<br>`3:2`<br>`4:5`<br>`5:4`<br>`9:16`<br>`9:21` | نسبت ابعاد تصویر تولیدشده. (پیش‌فرض: 1:1) |
| `پیش‌تنظیم سبک` | COMBO | خیر | `3d-model`<br>`analog-film`<br>`anime`<br>`cinematic`<br>`comic-book`<br>`digital-art`<br>`enhance`<br>`fantasy-art`<br>`isometric`<br>`line-art`<br>`low-poly`<br>`modeling-compound`<br>`neon-punk`<br>`origami`<br>`photographic`<br>`pixel-art`<br>`tile-texture`<br>`None` | سبک دلخواه تصویر تولیدشده (اختیاری). برای عدم استفاده از سبک از پیش تعیین‌شده، گزینه "None" را انتخاب کنید. |
| `مقیاس cfg` | FLOAT | بله | 1.0 تا 10.0 | میزان پایبندی فرایند انتشار به متن توضیح (مقادیر بالاتر تصویر را به توضیح شما نزدیک‌تر نگه می‌دارد). (پیش‌فرض: 4.0) |
| `بذر` | INT | بله | 0 تا 4294967294 | دانه تصادفی (seed) مورد استفاده برای ایجاد نویز. (پیش‌فرض: 0) |
| `تصویر` | IMAGE | خیر | - | تصویر ورودی اختیاری برای تولید تصویر به تصویر (image-to-image). در صورت ارائه، گره به حالت تصویر به تصویر تغییر وضعیت می‌دهد و پارامتر `نسبت تصویر` نادیده گرفته می‌شود. |
| `پرامپت منفی` | STRING | خیر | - | کلمات کلیدی آنچه نمی‌خواهید در تصویر خروجی ببینید. این یک ویژگی پیشرفته است. (پیش‌فرض: رشته خالی) |
| `کاهش نویز تصویر` | FLOAT | خیر | 0.0 تا 1.0 | میزان نویززدایی تصویر ورودی؛ مقدار 0.0 تصویری یکسان با ورودی و مقدار 1.0 به‌گونه‌ای است که گویی هیچ تصویری ارائه نشده است. (پیش‌فرض: 0.5) این پارامتر تنها زمانی استفاده می‌شود که یک `تصویر` ارائه شده باشد. |

**توجه:** هنگامی که یک `image` ارائه می‌شود، گره به حالت تولید تصویر به تصویر تغییر وضعیت می‌دهد و پارامتر `aspect_ratio` به‌طور خودکار از تصویر ورودی تعیین می‌شود. هنگامی که هیچ `image`ای ارائه نشود، پارامتر `image_denoise` نادیده گرفته می‌شود.
