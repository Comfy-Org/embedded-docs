# ترجمة وثيقة عقدة ComfyUI

هذه الوثيقة تم إنشاؤها بواسطة الذكاء الاصطناعي. إذا وجدت أي أخطاء أو لديك اقتراحات للتحسين، فلا تتردد في المساهمة! [تحرير على GitHub](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/StabilityStableImageSD_3_5Node/en.md)

تقوم هذه العقدة بتوليد الصور بشكل متزامن باستخدام نموذج Stable Diffusion 3.5 من Stability AI. تنشئ الصور بناءً على النصوص التوجيهية، ويمكنها أيضًا تعديل الصور الموجودة عند تقديمها كمدخلات. تدعم العقدة نسب أبعاد متنوعة وإعدادات أنماط مسبقة لتخصيص المخرجات.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|---------|---------------|--------|---------|-------|
| `prompt` | STRING | نعم | - | ما ترغب في رؤيته في الصورة الناتجة. نص توجيهي قوي ووصفي يحدد بوضوح العناصر والألوان والموضوعات سيؤدي إلى نتائج أفضل. (الافتراضي: سلسلة فارغة) |
| `model` | COMBO | نعم | `sd3.5-large`<br>`sd3.5-large-turbo`<br>`sd3.5-medium` | نموذج Stable Diffusion 3.5 المستخدم في التوليد. |
| `aspect_ratio` | COMBO | نعم | `16:9`<br>`1:1`<br>`21:9`<br>`2:3`<br>`3:2`<br>`4:5`<br>`5:4`<br>`9:16`<br>`9:21` | نسبة أبعاد الصورة المولدة. (الافتراضي: 1:1) |
| `style_preset` | COMBO | لا | `3d-model`<br>`analog-film`<br>`anime`<br>`cinematic`<br>`comic-book`<br>`digital-art`<br>`enhance`<br>`fantasy-art`<br>`isometric`<br>`line-art`<br>`low-poly`<br>`modeling-compound`<br>`neon-punk`<br>`origami`<br>`photographic`<br>`pixel-art`<br>`tile-texture`<br>`None` | النمط المطلوب اختياريًا للصورة المولدة. اختر "None" لعدم استخدام نمط مسبق. |
| `cfg_scale` | FLOAT | نعم | 1.0 إلى 10.0 | مدى التزام عملية الانتشار بالنص التوجيهي (القيم الأعلى تجعل الصورة أقرب إلى النص التوجيهي). (الافتراضي: 4.0) |
| `seed` | INT | نعم | 0 إلى 4294967294 | البذرة العشوائية المستخدمة في إنشاء الضوضاء. (الافتراضي: 0) |
| `image` | IMAGE | لا | - | صورة إدخال اختيارية لتوليد صورة-إلى-صورة. عند تقديمها، تتحول العقدة إلى وضع الصورة-إلى-صورة ويتم تجاهل معامل `aspect_ratio`. |
| `negative_prompt` | STRING | لا | - | كلمات مفتاحية لما لا ترغب في رؤيته في الصورة الناتجة. هذه ميزة متقدمة. (الافتراضي: سلسلة فارغة) |
| `image_denoise` | FLOAT | لا | 0.0 إلى 1.0 | إزالة الضوضاء من صورة الإدخال؛ القيمة 0.0 تنتج صورة مطابقة للإدخال، والقيمة 1.0 تعني كما لو لم يتم تقديم أي صورة على الإطلاق. (الافتراضي: 0.5) يُستخدم هذا المعامل فقط عند تقديم `image`. |

**ملاحظة:** عند تقديم `image`، تتحول العقدة إلى وضع توليد الصورة-إلى-صورة ويتم تحديد معامل `aspect_ratio` تلقائيًا من صورة الإدخال. عند عدم تقديم `image`، يتم تجاهل معامل `image_denoise`.
