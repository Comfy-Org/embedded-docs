Этот узел применяет визуальный стиль из эталонного изображения к вашему входному изображению. Он использует внешний AI-сервис для обработки изображений, позволяя вам контролировать силу переноса стиля и сохранение структуры исходного изображения.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `image` | IMAGE | Да | - | Изображение, к которому применяется перенос стиля. |
| `reference_image` | IMAGE | Да | - | Эталонное изображение для извлечения стиля. |
| `prompt` | STRING | Нет | - | Необязательный текстовый запрос для управления переносом стиля. |
| `style_strength` | INT | Нет | от 0 до 100 | Процент силы стиля (по умолчанию: 100). |
| `structure_strength` | INT | Нет | от 0 до 100 | Сохраняет структуру исходного изображения (по умолчанию: 50). |
| `flavor` | COMBO | Нет | "faithful"<br>"gen_z"<br>"psychedelia"<br>"detaily"<br>"clear"<br>"donotstyle"<br>"donotstyle_sharp" | Вариант переноса стиля. |
| `engine` | COMBO | Нет | "balanced"<br>"definio"<br>"illusio"<br>"3d_cartoon"<br>"colorful_anime"<br>"caricature"<br>"real"<br>"super_real"<br>"softy" | Выбор обрабатывающего движка. |
| `portrait_mode` | COMBO | Нет | "disabled"<br>"enabled" | Включить портретный режим для улучшения черт лица. |
| `portrait_style` | COMBO | Нет | "standard"<br>"pop"<br>"super_pop" | Визуальный стиль, применяемый к портретным изображениям. Этот параметр доступен только когда `portrait_mode` установлен в "enabled". |
| `portrait_beautifier` | COMBO | Нет | "none"<br>"beautify_face"<br>"beautify_face_max" | Интенсивность улучшения черт лица на портретах. Этот параметр доступен только когда `portrait_mode` установлен в "enabled". |
| `fixed_generation` | BOOLEAN | Нет | - | Если отключено, каждое создание будет вносить элемент случайности, что приведет к более разнообразным результатам (по умолчанию: True). |

**Ограничения:**

* Требуется ровно одно `image` и одно `reference_image`.
* Оба изображения должны иметь соотношение сторон от 1:3 до 3:1.
* Минимальная высота и ширина обоих изображений должна составлять 160 пикселей.
* Параметры `portrait_style` и `portrait_beautifier` активны и обязательны только когда `portrait_mode` установлен в "enabled".
