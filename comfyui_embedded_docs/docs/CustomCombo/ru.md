Вот перевод документации на русский язык:

Узел Custom Combo позволяет создать собственное выпадающее меню с вашим списком текстовых опций. Это узел, ориентированный на работу с интерфейсом, который предоставляет внутреннее представление для обеспечения совместимости в рамках вашего рабочего процесса. При выборе опции из выпадающего меню узел выводит этот текст в виде строки и его индексную позицию.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `choice` | COMBO | Да | Определяется пользователем | Текстовая опция, выбранная из пользовательского выпадающего меню. Список доступных опций определяется пользователем в интерфейсе узла. |
| `index` | INT | Нет | 0 | Целочисленное значение, которое можно использовать для указания индекса. По умолчанию: 0. |

**Примечание:** Проверка входных данных для этого узла намеренно отключена. Это позволяет вам задавать любые пользовательские текстовые опции в интерфейсе без проверки внутренней системой того, что ваш выбор соответствует предопределённому списку.

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `STRING` | STRING | Текстовая строка опции, выбранной из пользовательского комбинированного списка. |
| `INDEX` | INT | Индексная позиция выбранной опции в выпадающем списке. |
