自訂組合節點允許您建立一個包含自訂文字選項清單的下拉式選單。這是一個專注於前端操作的節點，同時提供後端表示以確保在工作流程中的相容性。當您從下拉選單中選擇一個選項時，該節點會將該文字以字串形式輸出，並同時輸出其索引位置。

## 輸入

| 參數 | 資料類型 | 必要 | 範圍 | 說明 |
|-----------|-----------|----------|-------|-------------|
| `選項` | COMBO | 是 | 使用者自訂 | 從自訂下拉選單中選取的文字選項。可用的選項清單由使用者在節點的前端介面中定義。 |
| `index` | INT | 否 | 0 | 一個可用於指定索引的整數值。預設值：0。 |

**注意：** 此節點輸入的驗證功能已刻意停用。這讓您可以在前端定義任何自訂文字選項，而無需後端檢查您的選取是否來自預先定義的清單。

| 輸出名稱 | 資料類型 | 說明 |
|-------------|-----------|-------------|
| `INDEX` | STRING | 從自訂組合框中選取之選項的文字字串。 |
| `INDEX` | INT | 所選選項在下拉清單中的索引位置。 |
