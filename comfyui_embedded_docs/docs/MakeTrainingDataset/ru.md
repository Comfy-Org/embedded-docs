Этот узел подготавливает данные для обучения путем кодирования изображений и текста. Он принимает список изображений и соответствующий список текстовых подписей, затем использует модель VAE для преобразования изображений в латентные представления и модель CLIP для преобразования текста в данные кондиционирования. Полученные парные латентные представления и кондиционирование выводятся в виде списков, готовых к использованию в рабочих процессах обучения.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `images` | IMAGE | Да | N/A | Список изображений для кодирования. |
| `vae` | VAE | Да | N/A | Модель VAE для кодирования изображений в латентные представления. |
| `clip` | CLIP | Да | N/A | Модель CLIP для кодирования текста в кондиционирование. |
| `texts` | STRING | Нет | N/A | Список текстовых подписей. Может иметь длину n (соответствует количеству изображений), 1 (повторяется для всех) или быть опущен (используется пустая строка). |

**Ограничения параметров:**

* Количество элементов в списке `texts` должно быть равно 0, 1 или точно соответствовать количеству элементов в списке `images`. Если равно 0, для всех изображений используется пустая строка. Если равно 1, этот единственный текст повторяется для всех изображений.

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `latents` | LATENT | Список словарей латентных представлений. |
| `conditioning` | CONDITIONING | Список списков кондиционирования. |
