این گره با رمزگذاری تصاویر و متن، داده‌ها را برای آموزش آماده می‌کند. این گره یک لیست از تصاویر و یک لیست متناظر از زیرنویس‌های متنی دریافت می‌کند، سپس از یک مدل VAE برای تبدیل تصاویر به بازنمایی‌های نهفته (latent) و از یک مدل CLIP برای تبدیل متن به داده‌های شرطی‌سازی (conditioning) استفاده می‌کند. جفت‌های حاصل از داده‌های نهفته و شرطی‌سازی به صورت لیست خروجی داده می‌شوند و برای استفاده در گردش‌های کاری آموزش آماده هستند.

## ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `تصاویر` | IMAGE | بله | نامشخص | لیست تصاویر برای رمزگذاری. |
| `vae` | VAE | بله | نامشخص | مدل VAE برای رمزگذاری تصاویر به داده‌های نهفته. |
| `clip` | CLIP | بله | نامشخص | مدل CLIP برای رمزگذاری متن به داده‌های شرطی‌سازی. |
| `متون` | STRING | خیر | نامشخص | لیست زیرنویس‌های متنی. می‌تواند طول n (مطابق با تصاویر)، 1 (تکرار برای همه)، یا حذف شده (استفاده از رشته خالی) باشد. |

**محدودیت‌های پارامتر:**

* تعداد آیتم‌های لیست `texts` باید 0، 1، یا دقیقاً برابر با تعداد آیتم‌های لیست `images` باشد. اگر 0 باشد، برای همه تصاویر از یک رشته خالی استفاده می‌شود. اگر 1 باشد، آن متن واحد برای همه تصاویر تکرار می‌شود.

## خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-------------|-----------|-------------|
| `latents` | LATENT | لیست دیکشنری‌های داده‌های نهفته. |
| `conditioning` | CONDITIONING | لیست لیست‌های داده‌های شرطی‌سازی. |
