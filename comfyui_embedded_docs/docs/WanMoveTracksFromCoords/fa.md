گره WanMoveTracksFromCoords، مسیرهای حرکتی را از یک رشته JSON حاوی مختصات ایجاد می‌کند. این گره داده‌های مختصات را به فرمت تنسور تبدیل می‌کند که توسط سایر گره‌های پردازش ویدیو قابل استفاده است و می‌تواند به‌صورت اختیاری یک ماسک برای کنترل دید مسیرها در طول زمان اعمال کند.

## ورودی‌ها

| پارامتر | نوع داده | اجباری | محدوده | توضیحات |
|---------|----------|--------|--------|---------|
| `track_coords` | STRING | خیر | نامشخص | یک رشته JSON حاوی داده‌های مختصات مسیرها. مقدار پیش‌فرض یک لیست خالی است (`"[]"`). |
| `track_mask` | MASK | خیر | نامشخص | یک ماسک اختیاری. در صورت ارائه، گره از آن برای تعیین دید هر مسیر در هر فریم استفاده می‌کند. |

**نکته:** ورودی `track_coords` ساختار JSON خاصی را انتظار دارد. باید لیستی از مسیرها باشد که هر مسیر شامل لیستی از فریم‌هاست و هر فریم یک شیء با مختصات `x` و `y` است. تعداد فریم‌ها باید در تمام مسیرها یکسان باشد.

## خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-----------|----------|---------|
| `tracks` | TRACKS | داده‌های مسیر تولیدشده، شامل مختصات مسیر و اطلاعات دید برای هر مسیر. |
| `track_length` | INT | تعداد کل فریم‌ها در مسیرهای تولیدشده. |
