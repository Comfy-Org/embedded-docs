Узел PolyexponentialScheduler предназначен для генерации последовательности уровней шума (сигм) на основе полиэкспоненциального расписания шума. Это расписание представляет собой полиномиальную функцию от логарифма сигмы, что обеспечивает гибкую и настраиваемую прогрессию уровней шума на протяжении всего процесса диффузии.

## Входные параметры

| Параметр     | Тип данных | Описание                                                                                                                                                                                                                                                                                                                                                      |
|--------------|------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| `steps`      | INT        | Определяет количество шагов в процессе диффузии, влияя на детализацию генерируемых уровней шума.                                                                                                                                                                                                                                                                  |
| `sigma_max`  | FLOAT      | Максимальный уровень шума, задающий верхнюю границу расписания шума.                                                                                                                                                                                                                                                                                           |
| `sigma_min`  | FLOAT      | Минимальный уровень шума, задающий нижнюю границу расписания шума.                                                                                                                                                                                                                                                                                             |
| `rho`        | FLOAT      | Параметр, управляющий формой полиэкспоненциального расписания шума, влияющий на то, как уровни шума прогрессируют между минимальным и максимальным значениями.                                                                                                                                                                                                 |

## Выходные параметры

| Параметр  | Тип данных | Описание                                                                 |
|-----------|------------|--------------------------------------------------------------------------|
| `sigmas`  | SIGMAS     | Выходные данные представляют собой последовательность уровней шума (сигм), настроенную под указанное полиэкспоненциальное расписание шума. |
