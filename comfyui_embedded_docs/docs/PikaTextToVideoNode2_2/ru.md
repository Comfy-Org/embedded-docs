Узел Pika Text2Video v2.2 отправляет текстовый запрос в Pika API версии 2.2 для генерации видео. Он преобразует ваше текстовое описание в видео с помощью сервиса генерации видео на базе ИИ от Pika. Узел позволяет настраивать различные аспекты процесса генерации видео, включая соотношение сторон, длительность и разрешение.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `prompt_text` | STRING | Да | - | Основное текстовое описание, описывающее, что вы хотите сгенерировать в видео |
| `negative_prompt` | STRING | Да | - | Текст, описывающий, чего не должно быть в сгенерированном видео |
| `seed` | INT | Да | - | Число, управляющее случайностью генерации для воспроизводимых результатов |
| `resolution` | STRING | Да | - | Настройка разрешения выходного видео |
| `duration` | INT | Да | - | Длительность видео в секундах |
| `aspect_ratio` | FLOAT | Нет | 0.4 - 2.5 | Соотношение сторон (ширина / высота) (по умолчанию: 1.7777777777777777) |

## Выходные параметры

| Имя выходного параметра | Тип данных | Описание |
|-------------|-----------|-------------|
| `output` | VIDEO | Сгенерированный видеофайл, полученный от Pika API |
