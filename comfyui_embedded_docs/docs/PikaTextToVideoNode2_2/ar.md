# ترجمة وثيقة عقدة Pika Text2Video v2.2

عقدة Pika Text2Video v2.2 ترسل مطالبة نصية إلى إصدار 2.2 من واجهة برمجة تطبيقات Pika لإنشاء فيديو. تقوم بتحويل وصفك النصي إلى فيديو باستخدام خدمة توليد الفيديو بالذكاء الاصطناعي من Pika. تتيح لك العقدة تخصيص جوانب مختلفة من عملية توليد الفيديو بما في ذلك نسبة العرض إلى الارتفاع والمدة والدقة.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|---------|---------------|--------|---------|-------|
| `prompt_text` | STRING | نعم | - | الوصف النصي الرئيسي الذي يصف ما تريد توليده في الفيديو |
| `negative_prompt` | STRING | نعم | - | نص يصف ما لا تريد ظهوره في الفيديو المُنشأ |
| `seed` | INT | نعم | - | رقم يتحكم في عشوائية التوليد للحصول على نتائج قابلة للتكرار |
| `resolution` | STRING | نعم | - | إعداد الدقة لفيديو الإخراج |
| `duration` | INT | نعم | - | طول الفيديو بالثواني |
| `aspect_ratio` | FLOAT | لا | 0.4 - 2.5 | نسبة العرض إلى الارتفاع (العرض / الارتفاع) (الافتراضي: 1.7777777777777777) |

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|-------------|---------------|---------|
| `output` | VIDEO | ملف الفيديو المُنشأ الذي تم إرجاعه من واجهة برمجة تطبيقات Pika |
