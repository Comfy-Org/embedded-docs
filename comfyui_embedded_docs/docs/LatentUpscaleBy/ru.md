Узел `LatentUpscaleBy` предназначен для масштабирования латентных представлений изображений. Он позволяет регулировать коэффициент масштабирования и метод увеличения разрешения, обеспечивая гибкость при повышении качества латентных сэмплов.

## Входные параметры

| Параметр       | Тип данных       | Описание |
|----------------|------------------|----------|
| `образцы`      | `LATENT`         | Латентное представление изображений, подлежащих масштабированию. Этот параметр определяет входные данные, которые будут обработаны в процессе увеличения разрешения. |
| `метод_апскейла` | COMBO[STRING]   | Указывает метод, используемый для масштабирования латентных сэмплов. Выбор метода может существенно повлиять на качество и характеристики увеличенного вывода. |
| `масштабировать_по`     | `FLOAT`          | Определяет коэффициент, на который масштабируются латентные сэмплы. Этот параметр напрямую влияет на разрешение вывода, обеспечивая точный контроль над процессом масштабирования. |

## Выходные параметры

| Параметр  | Тип данных | Описание |
|-----------|------------|----------|
| `latent`  | `LATENT`   | Увеличенное латентное представление, готовое к дальнейшей обработке или задачам генерации. Этот вывод необходим для повышения разрешения сгенерированных изображений или для последующих операций модели. |
