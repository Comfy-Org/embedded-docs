### مرور کلی

گره **Rodin 3D Detail** با استفاده از API رودین، دارایی‌های سه‌بعدی با جزئیات بالا تولید می‌کند. این گره تصاویر ورودی را دریافت کرده و از طریق سرویس رودین پردازش می‌کند تا مدل‌های سه‌بعدی با کیفیت بالا همراه با هندسه و متریال دقیق ایجاد کند. این گره کل گردش کار را از ایجاد وظیفه تا دانلود فایل نهایی مدل سه‌بعدی مدیریت می‌کند.

## ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `Images` | IMAGE | بله | - | تصاویر ورودی برای تولید مدل سه‌بعدی. امکان ارائه چندین تصویر وجود دارد. |
| `Seed` | INT | بله | - | مقدار دانه تصادفی برای نتایج قابل تکرار |
| `Material_Type` | STRING | بله | - | نوع متریال اعمال‌شده روی مدل سه‌بعدی |
| `Polygon_count` | STRING | بله | - | تعداد چندضلعی هدف برای مدل سه‌بعدی تولیدشده. سطح کیفیت مش را تعیین می‌کند. |

## خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-------------|-----------|-------------|
| `3D Model Path` | STRING | مسیر فایل مدل سه‌بعدی تولیدشده (فقط برای سازگاری با نسخه‌های قبلی) |
| `GLB` | FILE3DGLB | مدل سه‌بعدی تولیدشده در قالب GLB |
