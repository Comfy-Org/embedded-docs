أنت خبير في الترجمة التقنية متخصص في توثيق عُقد ComfyUI من الإنجليزية إلى العربية.

## قواعد الترجمة

1. **المحتوى الذي يجب عدم ترجمته:**
   - أسماء المعاملات بين علامات الاقتباس الخلفية: `image`, `seed`, `model`
   - أنواع البيانات بالأحرف الكبيرة: IMAGE, STRING, INT, FLOAT, MODEL, CONDITIONING, إلخ
   - القيم في عمود Range: الأرقام، "auto"، أسماء الخيارات
   - الكود، مسارات الملفات

2. **المحتوى الذي يجب ترجمته:**
   - عناوين الأقسام: ## نظرة عامة, ## المدخلات, ## المخرجات
   - جميع النصوص الوصفية والتوضيحية
   - أوصاف المعاملات

3. **جودة الترجمة:**
   - استخدام اللغة العربية الفصحى المعاصرة
   - الحفاظ على نبرة احترافية ولكن سهلة الفهم
   - ضمان الدقة التقنية
   - استخدام المصطلحات التقنية العربية القياسية

4. **التنسيق:**
   - الحفاظ على جميع تنسيقات Markdown
   - الحفاظ على بنية الجداول
   - عدم إضافة أي ملاحظة أو رابط في بداية الوثيقة (سيتم إضافتها تلقائيًا)

الرجاء ترجمة الوثيقة التالية إلى العربية، دون تضمين الملاحظة الأولية للوثيقة:

تقوم هذه العقدة بإنشاء فيديو من وصف نصي باستخدام نموذج Wan 2.7. ترسل طلبك إلى واجهة برمجة تطبيقات خارجية، والتي تقوم بمعالجة المطالبة وإرجاع ملف فيديو. يمكنك اختياريًا توفير مقطع صوتي للتأثير على حركة الفيديو وتوقيته.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|-----------|-----------|----------|-------|-------------|
| `model` | COMBO | نعم | `"wan2.7-t2v"` | النموذج المحدد لاستخدامه في إنشاء الفيديو. |
| `model.prompt` | STRING | نعم | - | وصف للعناصر والميزات البصرية التي ترغب في ظهورها في الفيديو. يدعم اللغتين الإنجليزية والصينية. |
| `model.negative_prompt` | STRING | لا | - | وصف للعناصر أو الميزات التي ترغب في تجنبها في الفيديو المُنشأ. |
| `model.resolution` | COMBO | نعم | `"720P"`<br>`"1080P"` | دقة الفيديو الناتج. |
| `model.ratio` | COMBO | نعم | `"16:9"`<br>`"9:16"`<br>`"1:1"`<br>`"4:3"`<br>`"3:4"` | نسبة العرض إلى الارتفاع للفيديو الناتج. |
| `model.duration` | INT | نعم | 2 إلى 15 | طول الفيديو بالثواني (الافتراضي: 5). |
| `audio` | AUDIO | لا | - | ملف صوتي لتوجيه إنشاء الفيديو، مثل مزامنة الشفاه أو مطابقة الحركة مع الإيقاع. إذا لم يتم توفيره، سيقوم النموذج بإنشاء موسيقى خلفية أو مؤثرات صوتية مناسبة. يجب أن تتراوح مدة الصوت بين 1.5 و 60 ثانية. |
| `seed` | INT | لا | 0 إلى 2147483647 | رقم يُستخدم للتحكم في عشوائية التوليد، لضمان نتائج قابلة للتكرار (الافتراضي: 0). |
| `prompt_extend` | BOOLEAN | لا | - | عند تفعيله، سيتم تحسين المطالبة بمساعدة الذكاء الاصطناعي (الافتراضي: True). |
| `watermark` | BOOLEAN | لا | - | عند تفعيله، ستتم إضافة علامة مائية مولدة بالذكاء الاصطناعي إلى النتيجة (الافتراضي: False). |

**ملاحظة:** معامل `audio` اختياري. إذا تم توفيره، يجب أن تتراوح مدته بين 1.5 و 60 ثانية. إذا تم حذفه، سيقوم النموذج تلقائيًا بإنشاء الصوت.
