این گره با استفاده از مدل Wan 2.7، یک ویدیو از توضیحات متنی تولید می‌کند. درخواست شما را به یک API خارجی ارسال می‌کند که پرامپت را پردازش کرده و یک فایل ویدیویی بازمی‌گرداند. می‌توانید به‌صورت اختیاری یک کلیپ صوتی برای تأثیرگذاری بر حرکت و زمان‌بندی ویدیو ارائه دهید.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `model` | COMBO | بله | `"wan2.7-t2v"` | مدل خاص مورد استفاده برای تولید ویدیو. |
| `model.prompt` | STRING | بله | - | توضیحی از عناصر و ویژگی‌های بصری که در ویدیو می‌خواهید. از انگلیسی و چینی پشتیبانی می‌کند. |
| `model.negative_prompt` | STRING | خیر | - | توضیحی از عناصر یا ویژگی‌هایی که می‌خواهید در ویدیوی تولیدشده اجتناب کنید. |
| `model.resolution` | COMBO | بله | `"720P"`<br>`"1080P"` | وضوح تصویر ویدیوی خروجی. |
| `model.ratio` | COMBO | بله | `"16:9"`<br>`"9:16"`<br>`"1:1"`<br>`"4:3"`<br>`"3:4"` | نسبت ابعاد ویدیوی خروجی. |
| `model.duration` | INT | بله | 2 تا 15 | طول ویدیو بر حسب ثانیه (پیش‌فرض: 5). |
| `audio` | AUDIO | خیر | - | یک فایل صوتی برای هدایت تولید ویدیو، مانند هماهنگی لب یا تطبیق حرکت با ضرب‌آهنگ. اگر ارائه نشود، مدل موسیقی پس‌زمینه یا افکت‌های صوتی متناسب تولید می‌کند. مدت زمان صدا باید بین 1.5 تا 60 ثانیه باشد. |
| `seed` | INT | خیر | 0 تا 2147483647 | عددی برای کنترل تصادفی‌بودن تولید، که نتایج قابل‌تکرار را تضمین می‌کند (پیش‌فرض: 0). |
| `prompt_extend` | BOOLEAN | خیر | - | در صورت فعال‌سازی، پرامپت با کمک هوش مصنوعی بهبود می‌یابد (پیش‌فرض: True). |
| `watermark` | BOOLEAN | خیر | - | در صورت فعال‌سازی، یک واترمارک تولیدشده با هوش مصنوعی به نتیجه اضافه می‌شود (پیش‌فرض: False). |

**توجه:** پارامتر `audio` اختیاری است. در صورت ارائه، مدت زمان آن باید بین 1.5 تا 60 ثانیه باشد. در صورت عدم ارائه، مدل به‌طور خودکار صدا تولید می‌کند.
