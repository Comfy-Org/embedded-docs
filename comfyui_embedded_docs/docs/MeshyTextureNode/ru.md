Вот перевод документации на русский язык:

Узел Meshy: Texture применяет текстуры, созданные ИИ, к 3D-модели. Он принимает идентификатор задачи (task ID) от предыдущего узла генерации или преобразования 3D-моделей Meshy и использует либо текстовое описание, либо референсное изображение для создания новых текстур для модели. Узел выводит текстурированную модель в форматах GLB и FBX.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `model` | COMBO | Да | `"latest"` | Версия ИИ-модели для текстурирования. В настоящее время доступна только версия "latest". |
| `meshy_task_id` | MESHY_TASK_ID | Да | - | Уникальный идентификатор (task ID) от предыдущей задачи генерации или преобразования 3D-моделей Meshy. Предоставляет базовую 3D-модель для текстурирования. |
| `enable_original_uv` | BOOLEAN | Нет | - | Использовать оригинальные UV-координаты модели вместо генерации новых. При включении (по умолчанию: `True`) Meshy сохраняет существующие текстуры загруженной модели. Если у модели нет оригинальных UV-координат, качество результата может быть ниже. |
| `pbr` | BOOLEAN | Нет | - | Включает вывод материала на основе физически корректного рендеринга (PBR) для текстурированной модели (по умолчанию: `False`). |
| `text_style_prompt` | STRING | Нет | - | Опишите желаемый стиль текстуры объекта с помощью текста. Максимум 600 символов. Не может использоваться одновременно с `image_style`. |
| `image_style` | IMAGE | Нет | - | 2D-изображение для управления процессом текстурирования. Не может использоваться одновременно с `text_style_prompt`. |

**Ограничения параметров:**

* Необходимо указать либо `text_style_prompt`, либо `image_style`, но нельзя указать оба одновременно.
* Длина `text_style_prompt` ограничена максимум 600 символами.

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `model_file` | STRING | Имя файла сгенерированной GLB-модели. Этот выход предоставляется для обратной совместимости. |
| `meshy_task_id` | MODEL_TASK_ID | Уникальный идентификатор задачи для данной работы по текстурированию, который можно использовать для ссылки на результат. |
| `GLB` | FILE3DGLB | Текстурированная 3D-модель, сохраненная в формате GLB. |
| `FBX` | FILE3DFBX | Текстурированная 3D-модель, сохраненная в формате FBX. |
