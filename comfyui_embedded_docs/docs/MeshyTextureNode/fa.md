### مرور کلی

گره Meshy: Texture Texture بافت‌های تولیدشده توسط هوش مصنوعی را روی یک مدل سه‌بعدی اعمال می‌کند. این گره یک شناسه وظیفه (Task ID) از یک گره قبلی تولید یا تبدیل سه‌بعدی Meshy دریافت می‌کند و با استفاده از یک توضیح متنی یا یک تصویر مرجع، بافت‌های جدیدی برای مدل ایجاد می‌کند. خروجی این گره، مدل بافت‌خورده در قالب‌های فایل GLB و FBX است.

## ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `مدل` | COMBO | بله | `"latest"` | نسخه مدل هوش مصنوعی مورد استفاده برای بافت‌دهی. در حال حاضر، فقط نسخه "latest" در دسترس است. |
| `meshy_task_id` | MESHY_TASK_ID | بله | - | شناسه منحصربه‌فرد (شناسه وظیفه) از یک وظیفه قبلی تولید یا تبدیل سه‌بعدی Meshy. این شناسه، مدل سه‌بعدی پایه را برای بافت‌دهی فراهم می‌کند. |
| `فعال‌سازی UV اصلی` | BOOLEAN | خیر | - | از UV اصلی مدل به جای تولید UV جدید استفاده کنید. وقتی فعال باشد (پیش‌فرض: `True`)، Meshy بافت‌های موجود از مدل آپلودشده را حفظ می‌کند. اگر مدل UV اصلی نداشته باشد، کیفیت خروجی ممکن است به خوبی نباشد. |
| `PBR` | BOOLEAN | خیر | - | خروجی مواد مبتنی بر رندر فیزیکی (PBR) را برای مدل بافت‌خورده فعال می‌کند (پیش‌فرض: `False`). |
| `توضیح سبک بافت متنی` | STRING | خیر | - | سبک بافت مورد نظر شیء را با استفاده از متن توصیف کنید. حداکثر ۶۰۰ کاراکتر. نمی‌توان همزمان با `سبک تصویر` استفاده کرد. |
| `سبک تصویر` | IMAGE | خیر | - | یک تصویر دو بعدی برای هدایت فرآیند بافت‌دهی. نمی‌توان همزمان با `توضیح سبک بافت متنی` استفاده کرد. |

**محدودیت‌های پارامتر:**

* شما باید یا `text_style_prompt` یا `image_style` را ارائه دهید، اما نمی‌توانید هر دو را همزمان ارائه کنید.
* `text_style_prompt` به حداکثر ۶۰۰ کاراکتر محدود شده است.

## خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-------------|-----------|-------------|
| `model_file` | STRING | نام فایل مدل GLB تولیدشده. این خروجی برای سازگاری با نسخه‌های قبلی ارائه شده است. |
| `meshy_task_id` | MODEL_TASK_ID | شناسه منحصربه‌فرد وظیفه برای این کار بافت‌دهی که می‌توان از آن برای ارجاع به نتیجه استفاده کرد. |
| `GLB` | FILE3DGLB | مدل سه‌بعدی بافت‌خورده ذخیره‌شده در قالب فایل GLB. |
| `FBX` | FILE3DFBX | مدل سه‌بعدی بافت‌خورده ذخیره‌شده در قالب فایل FBX. |
