گره AlignYourStepsScheduler مقادیر سیگما را برای فرآیند نویززدایی بر اساس انواع مختلف مدل تولید می‌کند. این گره سطوح نویز مناسب را برای هر مرحله از فرآیند نمونه‌برداری محاسبه کرده و تعداد کل مراحل را با توجه به پارامتر denoise تنظیم می‌کند. این کار به هماهنگ‌سازی مراحل نمونه‌برداری با نیازهای خاص مدل‌های انتشار مختلف کمک می‌کند.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|---------|----------|--------|--------|---------|
| `model_type` | STRING | بله | `"SD1"`<br>`"SDXL"`<br>`"SVD"` | نوع مدل مورد استفاده برای محاسبه سیگما را مشخص می‌کند (پیش‌فرض: "SD1") |
| `steps` | INT | بله | 1 تا 10000 | تعداد کل مراحل نمونه‌برداری برای تولید (پیش‌فرض: 10) |
| `denoise` | FLOAT | بله | 0.0 تا 1.0 | میزان نویززدایی تصویر را کنترل می‌کند؛ مقدار 1.0 از تمام مراحل استفاده می‌کند و مقادیر پایین‌تر از مراحل کمتری استفاده می‌کنند (پیش‌فرض: 1.0) |
