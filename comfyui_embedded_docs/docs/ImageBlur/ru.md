Узел `ImageBlur` применяет гауссово размытие к изображению, позволяя смягчить края и уменьшить детализацию и шум. Он предоставляет контроль над интенсивностью и распространением размытия через параметры.

## Входы

| Поле          | Тип данных | Описание                                                                   |
|----------------|-------------|-------------------------------------------------------------------------------|
| `image`        | `IMAGE`     | Входное изображение для размытия. Это основной объект для применения эффекта размытия. |
| `blur_radius`  | `INT`       | Определяет радиус эффекта размытия. Больший радиус приводит к более выраженному размытию. |
| `sigma`        | `FLOAT`     | Управляет распространением размытия. Более высокое значение sigma означает, что размытие будет затрагивать более широкую область вокруг каждого пикселя. |

## Выходы

| Поле | Тип данных | Описание                                                              |
|-------|-------------|--------------------------------------------------------------------------|
| `image`| `IMAGE`     | На выходе получается размытая версия входного изображения, где степень размытия определяется входными параметрами. |
