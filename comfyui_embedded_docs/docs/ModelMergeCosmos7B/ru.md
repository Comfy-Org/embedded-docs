Узел `ModelMergeCosmos7B` объединяет две модели ИИ с помощью взвешенного смешивания определённых компонентов. Он обеспечивает тонкий контроль над тем, как различные части моделей комбинируются, путём настройки индивидуальных весов для эмбеддингов позиций, блоков трансформера и финальных слоёв.
