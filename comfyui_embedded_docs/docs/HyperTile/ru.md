Вот перевод документации узла HyperTile на русский язык:

Узел HyperTile применяет технику тайлинга к механизму внимания в диффузионных моделях для оптимизации использования памяти при генерации изображений. Он разделяет скрытое пространство на более мелкие тайлы, обрабатывает их по отдельности, а затем собирает результаты обратно. Это позволяет работать с изображениями большего размера без нехватки памяти.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `model` | MODEL | Да | - | Диффузионная модель, к которой применяется оптимизация HyperTile |
| `tile_size` | INT | Нет | 1 - 2048 | Целевой размер тайла для обработки (по умолчанию: 256). Фактический размер тайла округляется вниз до кратного 8, с минимумом 32. |
| `swap_size` | INT | Нет | 1 - 128 | Управляет тем, как тайлы переставляются в процессе обработки для повышения эффективности (по умолчанию: 2) |
| `max_depth` | INT | Нет | 0 - 10 | Максимальный уровень глубины (масштаб разрешения) для применения тайлинга. Значение 0 применяет тайлинг только на самом высоком разрешении (по умолчанию: 0) |
| `scale_depth` | BOOLEAN | Нет | True / False | При включении размер тайла пропорционально масштабируется на более глубоких уровнях. Это может помочь сохранить качество на низких разрешениях (по умолчанию: False) |

## Выходы

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `model` | MODEL | Модифицированная модель с применённой оптимизацией HyperTile |
