HyperTile 節點對擴散模型中的注意力機制應用平鋪技術，以優化圖像生成過程中的記憶體使用。它將潛在空間分割成較小的區塊並分別處理，然後重新組合結果。這使得在處理較大圖像尺寸時不會耗盡記憶體。
