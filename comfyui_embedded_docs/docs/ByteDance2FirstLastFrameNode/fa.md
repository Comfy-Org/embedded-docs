این گره از مدل Seedance 2.0 شرکت ByteDance برای تولید ویدیو استفاده می‌کند. ویدیو را بر اساس یک توضیح متنی و یک تصویر فریم اول الزامی ایجاد می‌کند. می‌توانید به‌صورت اختیاری یک تصویر فریم آخر برای هدایت پایان دنباله ویدیو ارائه دهید.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `مدل` | COMBO | بله | `"Seedance 2.0"`<br>`"Seedance 2.0 Fast"` | مدل مورد استفاده برای تولید ویدیو. Seedance 2.0 برای حداکثر کیفیت و Seedance 2.0 Fast برای سرعت بهینه‌سازی شده است. انتخاب یک مدل، ورودی‌های اضافی برای `prompt`، `resolution`، `ratio`، `duration` و `generate_audio` را آشکار می‌کند. |
| `اولین فریم` | IMAGE | خیر | - | تصویری که به‌عنوان فریم اول ویدیو استفاده می‌شود. |
| `آخرین فریم` | IMAGE | خیر | - | تصویری که به‌عنوان فریم آخر ویدیو استفاده می‌شود. |
| `first_frame_asset_id` | STRING | خیر | - | یک asset_id از Seedance برای استفاده به‌عنوان فریم اول. این پارامتر نمی‌تواند هم‌زمان با ورودی تصویر `اولین فریم` استفاده شود. پیش‌فرض یک رشته خالی است. |
| `last_frame_asset_id` | STRING | خیر | - | یک asset_id از Seedance برای استفاده به‌عنوان فریم آخر. این پارامتر نمی‌تواند هم‌زمان با ورودی تصویر `آخرین فریم` استفاده شود. پیش‌فرض یک رشته خالی است. |
| `seed` | INT | خیر | 0 تا 2147483647 | یک مقدار seed. تغییر این seed باعث اجرای مجدد گره می‌شود، اما نتایج غیرقطعی هستند. پیش‌فرض 0 است. |
| `واترمارک` | BOOLEAN | خیر | - | آیا به ویدیوی تولید شده واترمارک اضافه شود یا خیر. پیش‌فرض False است. |

**محدودیت‌های پارامتر:**
*   باید **یا** یک تصویر `first_frame` **یا** یک `first_frame_asset_id` ارائه دهید. ارائه هر دو باعث خطا می‌شود.
*   نمی‌توانید هم یک تصویر `last_frame` و هم یک `last_frame_asset_id` برای یک فریم ارائه دهید.
*   ورودی `model` یک کامبوی پویا است. پس از انتخاب یک مدل، باید فیلد `prompt` آشکار شده (یک توضیح متنی) را پر کرده و سایر پارامترهای آشکار شده (`resolution`، `ratio`، `duration`، `generate_audio`) را پیکربندی کنید.
