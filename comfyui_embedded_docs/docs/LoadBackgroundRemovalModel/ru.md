## Обзор

Загружает модель удаления фона из файла. Этот узел подготавливает модель к использованию для удаления фона из изображений.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `bg_removal_name` | STRING | Да | Список доступных файлов моделей | Модель, используемая для удаления фона из изображений. Выберите из списка доступных файлов моделей удаления фона. |

## Выходные параметры

| Имя выходного параметра | Тип данных | Описание |
|-------------|-----------|-------------|
| `bg_model` | BACKGROUND_REMOVAL | Загруженная модель удаления фона, готовая к использованию другими узлами для обработки изображений. |
