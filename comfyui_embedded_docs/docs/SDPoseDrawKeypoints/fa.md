گره SDPoseDrawKeypoints داده‌های تخمین ژست (نقاط کلیدی) را دریافت کرده و آن‌ها را به‌عنوان یک اسکلت بصری روی یک بوم خالی ترسیم می‌کند. این گره به شما امکان می‌دهد بخش‌های مختلف ژست مانند بدن، دست‌ها، صورت و پاها را به‌صورت انتخابی با ضخامت خطوط و اندازه نقاط قابل تنظیم رسم کنید. تصویر خروجی را می‌توان برای مصورسازی یا به‌عنوان ورودی سایر گره‌هایی که به تصویر ژست نیاز دارند استفاده کرد.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|---------|----------|--------|--------|---------|
| `keypoints` | POSE_KEYPOINT | بله | - | داده‌های نقاط کلیدی ژست که باید رسم شوند. این داده‌ها معمولاً از یک گره تشخیص ژست دریافت می‌شوند. |
| `draw_body` | BOOLEAN | خیر | - | کنترل می‌کند که آیا اسکلت اصلی بدن رسم شود (پیش‌فرض: True). |
| `draw_hands` | BOOLEAN | خیر | - | کنترل می‌کند که آیا نقاط کلیدی دست‌ها رسم شوند (پیش‌فرض: True). |
| `draw_face` | BOOLEAN | خیر | - | کنترل می‌کند که آیا نقاط کلیدی صورت رسم شوند (پیش‌فرض: True). |
| `draw_feet` | BOOLEAN | خیر | - | کنترل می‌کند که آیا نقاط کلیدی پاها رسم شوند (پیش‌فرض: False). |
| `stick_width` | INT | خیر | ۱ تا ۱۰ | ضخامت خطوط مورد استفاده برای رسم اسکلت بدن (پیش‌فرض: ۴). |
| `face_point_size` | INT | خیر | ۱ تا ۱۰ | اندازه نقاط مورد استفاده برای رسم نقاط کلیدی صورت (پیش‌فرض: ۳). |
| `score_threshold` | FLOAT | خیر | ۰.۰ تا ۱.۰ | حداقل امتیاز اطمینان مورد نیاز برای رسم یک نقطه کلیدی. نقاط کلیدی با امتیاز کمتر از این مقدار نادیده گرفته می‌شوند (پیش‌فرض: ۰.۳). |

**توجه:** اگر ورودی `keypoints` خالی یا `None` باشد، گره یک تصویر خالی ۶۴×۶۴ پیکسل خروجی می‌دهد.

## خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-----------|----------|---------|
| `output` | IMAGE | تصویری با نقاط کلیدی ژست رسم‌شده. ابعاد تصویر با `canvas_height` و `canvas_width` مشخص‌شده در داده‌های نقاط کلیدی ورودی مطابقت دارد. |
