Вот перевод документации на русский язык:

Узел StableCascade_SuperResolutionControlnet подготавливает входные данные для обработки суперразрешения Stable Cascade. Он принимает входное изображение и кодирует его с помощью VAE для создания входных данных для controlnet, а также генерирует заполнители латентных представлений для этапа C и этапа B конвейера Stable Cascade.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `изображение` | IMAGE | Да | - | Входное изображение для обработки суперразрешения |
| `vae` | VAE | Да | - | Модель VAE, используемая для кодирования входного изображения |

## Выходные данные

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `controlnet_input` | IMAGE | Закодированное представление изображения, подходящее для входа controlnet |
| `stage_c` | LATENT | Заполнитель латентного представления для этапа C обработки Stable Cascade |
| `stage_b` | LATENT | Заполнитель латентного представления для этапа B обработки Stable Cascade |
