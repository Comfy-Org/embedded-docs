### ## نمای کلی

گره `StableCascade_SuperResolutionControlnet` ورودی‌های لازم برای پردازش ابررزولوشن (Super-Resolution) در Stable Cascade را آماده می‌کند. این گره یک تصویر ورودی دریافت کرده و با استفاده از یک VAE آن را رمزگذاری می‌کند تا ورودی کنترل‌نت (controlnet) را ایجاد کند. همچنین به‌طور همزمان، نمایش‌های نهفته (latent) جایگزین برای مرحله C و مرحله B از خط لوله Stable Cascade تولید می‌کند.

### ## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `image` | IMAGE | بله | - | تصویر ورودی که باید برای ابررزولوشن پردازش شود |
| `vae` | VAE | بله | - | مدل VAE مورد استفاده برای رمزگذاری تصویر ورودی |

### ## خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-------------|-----------|-------------|
| `controlnet_input` | IMAGE | نمایش رمزگذاری‌شده تصویر که برای ورودی کنترل‌نت مناسب است |
| `stage_c` | LATENT | نمایش نهفته جایگزین برای مرحله C پردازش Stable Cascade |
| `stage_b` | LATENT | نمایش نهفته جایگزین برای مرحله B پردازش Stable Cascade |
