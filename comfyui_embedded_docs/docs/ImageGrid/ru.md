Вот перевод документации на русский язык в соответствии с вашими правилами:

Узел Image Grid объединяет несколько изображений в единую организованную сетку или коллаж. Он принимает список изображений и размещает их в заданное количество столбцов, изменяя размер каждого изображения для соответствия определённому размеру ячейки и добавляя необязательные отступы между ними. Результатом является одно новое изображение, содержащее все входные изображения, расположенные в виде сетки.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `images` | IMAGE | Да | - | Список изображений для размещения в сетке. Для работы узла требуется как минимум одно изображение. |
| `columns` | INT | Нет | 1 - 20 | Количество столбцов в сетке (по умолчанию: 4). |
| `cell_width` | INT | Нет | 32 - 2048 | Ширина каждой ячейки сетки в пикселях (по умолчанию: 256). |
| `cell_height` | INT | Нет | 32 - 2048 | Высота каждой ячейки сетки в пикселях (по умолчанию: 256). |
| `padding` | INT | Нет | 0 - 50 | Величина отступа между изображениями в сетке в пикселях (по умолчанию: 4). |

## Выходные параметры

| Имя выходного параметра | Тип данных | Описание |
|-------------|-----------|-------------|
| `image` | IMAGE | Единственное выходное изображение, содержащее все входные изображения, расположенные в виде сетки. |
