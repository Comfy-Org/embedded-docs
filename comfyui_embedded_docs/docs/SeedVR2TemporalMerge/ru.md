# SeedVR2TemporalMerge

Этот узел объединяет семплированные временные чанки скрытого представления SeedVR2 в одно полное скрытое представление полной длины. Он использует перекрестное сглаживание с окном Ханна в области перекрытия между чанками для создания плавных переходов или выполняет простое объединение, когда перекрытие не задано.

## Входы

| Параметр | Описание | Тип данных | Обязательный | Диапазон |
|----------|----------|------------|--------------|----------|
| `latents` | Семплированные временные чанки в порядке следования. | LATENT | Да | Список скрытых представлений |
| `temporal_overlap` | Значение temporal_overlap из выхода узла Split SeedVR2 Latent. 0 = простое объединение. (по умолчанию: 0) | INT | Да | от 0 до 16384 |

**Примечание:** Значение `temporal_overlap` должно быть больше или равно 0. Последний чанк в последовательности может содержать меньше временных кадров, чем остальные чанки. Все чанки должны иметь совпадающие размерности, за исключением временной оси (T), и первый чанк должен быть 5-мерным (B, C, T, H, W).

## Выходы

| Имя выхода | Описание | Тип данных |
|------------|----------|------------|
| `latent` | Объединенное скрытое представление полной длины. | LATENT |
