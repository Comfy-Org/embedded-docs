# SeedVR2TemporalMerge

Этот узел объединяет временные фрагменты скрытого видеоданных, которые были ранее разделены узлом Split SeedVR2 Latent. Он использует перекрестное затухание с окном Ханна на перекрывающихся областях для создания плавного перехода между фрагментами или выполняет простое объединение, если перекрытие не задано.

## Входные параметры

| Параметр | Описание | Тип данных | Обязательный | Диапазон |
|----------|----------|------------|--------------|----------|
| `latents` | Выбранные временные фрагменты в порядке следования. | LATENT | Да | Список скрытых представлений |
| `temporal_overlap` | Выходное значение `temporal_overlap` узла Split SeedVR2 Latent. 0 = простое объединение. | INT | Да | от 0 до 16384 (по умолчанию: 0) |

**Примечание:** Параметр `temporal_overlap` должен быть подключен от выхода `temporal_overlap` узла Split SeedVR2 Latent. Все входные скрытые представления должны иметь одинаковый размер пакета, количество каналов, высоту и ширину. Только последний фрагмент может иметь меньшую временную размерность, чем остальные.

## Выходные параметры

| Имя выхода | Описание | Тип данных |
|------------|----------|------------|
| `latent` | Объединенное скрытое представление полной длины. | LATENT |
