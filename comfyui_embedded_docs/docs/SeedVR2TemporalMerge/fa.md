# SeedVR2TemporalMerge

این گره، تکه‌های زمانی نهان (latent temporal chunks) نمونه‌برداری‌شده از SeedVR2 را دوباره به یک نهان تمام‌طول واحد ترکیب می‌کند. برای ایجاد انتقال‌های نرم، از یک هم‌هم‌پوشانی پنجره هان (Hann window crossfade) در ناحیه هم‌پوشانی بین تکه‌ها استفاده می‌کند، یا زمانی که هم‌پوشانی مشخص نشده باشد، الحاق ساده (plain concatenation) انجام می‌دهد.

## ورودی‌ها

| پارامتر | توضیحات | نوع داده | الزامی | محدوده |
|---------|---------|----------|--------|--------|
| `latents` | تکه‌های زمانی نهان نمونه‌برداری‌شده به ترتیب توالی. | LATENT | بله | فهرستی از latents |
| `temporal_overlap` | خروجی temporal_overlap گره Split SeedVR2 Latent. مقدار 0 به معنای الحاق ساده است. (پیش‌فرض: 0) | INT | بله | 0 تا 16384 |

**نکته:** مقدار `temporal_overlap` باید بزرگتر یا مساوی 0 باشد. آخرین تکه در توالی ممکن است فریم‌های زمانی کمتری نسبت به سایر تکه‌ها داشته باشد. همه تکه‌ها باید دارای ابعاد یکسان باشند، به جز محور زمانی (T)، و اولین تکه باید 5 بعدی (B, C, T, H, W) باشد.
