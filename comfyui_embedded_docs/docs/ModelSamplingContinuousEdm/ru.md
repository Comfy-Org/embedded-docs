Этот узел предназначен для улучшения возможностей сэмплирования модели путем интеграции методов непрерывного сэмплирования EDM (диффузионные модели на основе энергии). Он позволяет динамически регулировать уровни шума в процессе сэмплирования модели, обеспечивая более точный контроль над качеством и разнообразием генерации.

## Входы

| Параметр    | Тип данных    | Python dtype        | Описание |
|-------------|---------------|----------------------|----------|
| `модель`     | `MODEL`       | `torch.nn.Module`   | Модель, которую необходимо расширить возможностями непрерывного сэмплирования EDM. Служит основой для применения передовых методов сэмплирования. |
| `выборка`  | COMBO[STRING] | `str`               | Определяет тип применяемого сэмплирования: 'eps' для эпсилон-сэмплирования или 'v_prediction' для предсказания скорости, что влияет на поведение модели в процессе сэмплирования. |
| `sigma_max` | `FLOAT`       | `float`             | Максимальное значение сигмы для уровня шума, позволяющее задать верхнюю границу при внесении шума в процессе сэмплирования. |
| `sigma_min` | `FLOAT`       | `float`             | Минимальное значение сигмы для уровня шума, устанавливающее нижний предел внесения шума, что влияет на точность сэмплирования модели. |

## Выходы

| Параметр  | Тип данных | Python dtype        | Описание |
|-----------|------------|----------------------|----------|
| `модель`   | MODEL      | `torch.nn.Module`   | Улучшенная модель с интегрированными возможностями непрерывного сэмплирования EDM, готовая к дальнейшему использованию в задачах генерации. |
