Вот перевод документации на русский язык:

Узел PreviewAudio создает временное аудио-предпрослушивание, которое можно воспроизвести непосредственно в интерфейсе. Он принимает аудиоданные на вход и генерирует виджет предпросмотра, позволяя пользователям прослушивать аудиовыходы без сохранения постоянных файлов.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `audio` | AUDIO | Да | - | Аудиоданные для предпросмотра |

## Выходы

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `ui` | UI | Отображает виджет аудиоплеера в интерфейсе для предпрослушивания аудио |
