Crea Controles Recraft para personalizar la generación Recraft. Este nodo permite configurar ajustes de color que se utilizarán durante el proceso de generación de imágenes Recraft.

## Entradas

| Parámetro | Tipo de Dato | Obligatorio | Rango | Descripción |
|-----------|--------------|-------------|-------|-------------|
| `colores` | COLOR | No | - | Ajustes de color para los elementos principales |
| `color_de_fondo` | COLOR | No | - | Ajuste de color de fondo |

## Salidas

| Nombre de Salida | Tipo de Dato | Descripción |
|------------------|--------------|-------------|
| `recraft_controls` | CONTROLS | Los controles Recraft configurados que contienen los ajustes de color |
