Вот перевод документации на русский язык:

Создаёт элементы управления Recraft для настройки генерации Recraft. Этот узел позволяет настроить параметры цвета, которые будут использоваться в процессе генерации изображений Recraft.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `colors` | COLOR | Нет | - | Цветовые настройки для основных элементов |
| `background_color` | COLOR | Нет | - | Настройка цвета фона |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `recraft_controls` | CONTROLS | Настроенные элементы управления Recraft, содержащие цветовые параметры |
