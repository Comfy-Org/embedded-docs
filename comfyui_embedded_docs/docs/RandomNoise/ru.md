Вот перевод документации на русский язык:

Узел RandomNoise генерирует случайные шумовые паттерны на основе значения seed. Он создает воспроизводимый шум, который можно использовать для различных задач обработки изображений и генерации. Одно и то же значение seed всегда будет давать одинаковый шумовой паттерн, что позволяет получать согласованные результаты при многократном запуске.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `noise_seed` | INT | Да | от 0 до 18446744073709551615 | Значение seed, используемое для генерации случайного шумового паттерна (по умолчанию: 0). Одно и то же значение seed всегда будет давать одинаковый шумовой результат. |

## Выходы

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `noise` | NOISE | Сгенерированный случайный шумовой паттерн на основе предоставленного значения seed. |
