Вот перевод документации на русский язык:

Узел CLIPMergeAdd объединяет две модели CLIP путем добавления патчей из второй модели в первую. Он создает копию первой модели CLIP и выборочно включает ключевые патчи из второй модели, исключая идентификаторы позиций и параметры шкалы логитов. Это позволяет объединять компоненты моделей CLIP, сохраняя структуру базовой модели.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `clip1` | CLIP | Да | - | Базовая модель CLIP, которая будет клонирована и использована в качестве основы для объединения |
| `clip2` | CLIP | Да | - | Вторичная модель CLIP, предоставляющая ключевые патчи для добавления к базовой модели |

## Выходы

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `CLIP` | CLIP | Объединенная модель CLIP, содержащая структуру базовой модели с добавленными патчами из вторичной модели |
