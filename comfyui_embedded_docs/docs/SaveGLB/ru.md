Узел SaveGLB сохраняет данные 3D-сетки или 3D-файлы в выходной каталог. Он принимает данные сетки или различные форматы 3D-файлов (GLB, GLTF, OBJ, FBX, STL, USDZ) и экспортирует их с указанным префиксом имени файла. При сохранении данных сетки узел может обрабатывать несколько сеток и автоматически добавляет метаданные рабочего процесса в файлы, если метаданные включены.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `mesh` | MESH или FILE3D | Да | - | Сетка или 3D-файл для сохранения. Принимает данные сетки или форматы 3D-файлов, включая GLB, GLTF, OBJ, FBX, STL и USDZ |
| `filename_prefix` | STRING | Нет | - | Префикс для имени выходного файла (по умолчанию: "3d/ComfyUI") |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `ui` | UI | Отображает сохраненные 3D-файлы в пользовательском интерфейсе с информацией об имени файла, подпапке и типе |
