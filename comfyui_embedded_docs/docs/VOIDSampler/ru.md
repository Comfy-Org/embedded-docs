## Обзор

Узел VOIDSampler предоставляет специализированный метод сэмплирования DDIM, разработанный специально для моделей inpainting VOID. Он реализует тот же процесс шумоподавления, который использовался при обучении моделей VOID, без масштабирования шума, применяемого стандартными KSampler. Этот узел предназначен для использования с узлами SamplerCustom или SamplerCustomAdvanced и должен сочетаться с RandomNoise или VOIDWarpedNoiseSource.

## Входные параметры

Данный узел не имеет настраиваемых входных параметров. Это автономный сэмплер, применяющий фиксированный алгоритм сэмплирования DDIM.

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| *Нет входных параметров* | - | - | - | Этот узел не принимает никаких входных параметров. |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `SAMPLER` | SAMPLER | Объект сэмплера, реализующий алгоритм VOID DDIM, готовый для подключения к узлам SamplerCustom или SamplerCustomAdvanced. |
