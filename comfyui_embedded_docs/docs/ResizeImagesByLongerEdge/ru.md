Вот перевод документации на русский язык:

Узел "Resize Images by Longer Edge" (Изменение размера изображений по длинной стороне) изменяет размер одного или нескольких изображений так, чтобы их самая длинная сторона соответствовала заданной целевой длине. Он автоматически определяет, какая сторона (ширина или высота) длиннее, и масштабирует другую сторону пропорционально, сохраняя исходное соотношение сторон. Это полезно для стандартизации размеров изображений на основе их наибольшего измерения.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `image` | IMAGE | Да | - | Входное изображение или пакет изображений для изменения размера. |
| `longer_edge` | INT | Да | 1 - 8192 | Целевая длина для длинной стороны. Короткая сторона будет масштабирована пропорционально. (по умолчанию: 1024) |

## Выходы

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `image` | IMAGE | Измененное изображение или пакет изображений. На выходе будет то же количество изображений, что и на входе, при этом длинная сторона каждого из них будет соответствовать указанной длине `longer_edge`. |
