Узел LossGraphNode создаёт визуальный график значений потерь (loss) при обучении с течением времени и отображает его в виде изображения для предварительного просмотра. Он принимает данные о потерях из процессов обучения и генерирует линейный график, показывающий, как изменяются потери на различных этапах обучения. Полученный график включает подписи осей и минимальные/максимальные значения потерь.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `потери` | LOSS_MAP | Да | - | Карта потерь из узла обучения. |
| `префикс_имени_файла` | STRING | Да | - | Префикс для имени файла изображения графика потерь. (по умолчанию: "loss_graph") |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `ui.images` | IMAGE | Сгенерированное изображение графика потерь, отображаемое в режиме предварительного просмотра. |
