LossGraphNode 會建立訓練損失值隨時間變化的視覺化圖表，並將其儲存為影像檔案。它從訓練過程中讀取損失資料，並生成顯示損失隨訓練步驟變化的折線圖。生成的圖表包含座標軸標籤、最小/最大損失值，並會自動以時間戳記儲存到暫存輸出目錄。

## 輸入參數

| 參數名稱 | 資料類型 | 是否必填 | 數值範圍 | 參數說明 |
|-----------|-----------|----------|-------|-------------|
| `loss` | LOSS | 是 | 多種選項可用 | 包含要繪製的損失值之損失資料（預設值：空字典） |
| `filename_prefix` | STRING | 是 | - | 輸出影像檔名的前綴詞（預設值："loss_graph"） |

**注意：** `loss` 參數需要包含帶有損失值的 "loss" 鍵的有效損失字典。節點會自動調整損失值以適應圖表尺寸，並生成顯示損失隨訓練步驟進展的折線圖。
