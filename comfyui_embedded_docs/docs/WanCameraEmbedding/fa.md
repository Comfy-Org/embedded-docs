### مرور کلی

گره WanCameraEmbedding با استفاده از جاسازی‌های پلوکر (Plücker embeddings) بر اساس پارامترهای حرکت دوربین، جاسازی‌های مسیر دوربین را تولید می‌کند. این گره دنباله‌ای از حالت‌های دوربین را ایجاد می‌کند که حرکات مختلف دوربین را شبیه‌سازی کرده و آن‌ها را به تنسورهای جاسازی مناسب برای خطوط لوله تولید ویدیو تبدیل می‌کند.
