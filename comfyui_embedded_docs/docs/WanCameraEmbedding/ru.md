Вот перевод документации узла WanCameraEmbedding на русский язык:

Узел WanCameraEmbedding генерирует вложения траектории камеры, используя вложения Плюккера на основе параметров движения камеры. Он создает последовательность положений камеры, имитирующих различные движения камеры, и преобразует их в тензоры вложений, подходящие для конвейеров генерации видео.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `camera_pose` | COMBO | Да | "Static"<br>"Pan Up"<br>"Pan Down"<br>"Pan Left"<br>"Pan Right"<br>"Zoom In"<br>"Zoom Out"<br>"Anti Clockwise (ACW)"<br>"ClockWise (CW)" | Тип движения камеры для симуляции (по умолчанию: "Static") |
| `width` | INT | Да | от 16 до MAX_RESOLUTION | Ширина выходного изображения в пикселях (по умолчанию: 832, шаг: 16) |
| `height` | INT | Да | от 16 до MAX_RESOLUTION | Высота выходного изображения в пикселях (по умолчанию: 480, шаг: 16) |
| `length` | INT | Да | от 1 до MAX_RESOLUTION | Длина последовательности траектории камеры (по умолчанию: 81, шаг: 4) |
| `speed` | FLOAT | Нет | от 0.0 до 10.0 | Скорость движения камеры (по умолчанию: 1.0, шаг: 0.1) |
| `fx` | FLOAT | Нет | от 0.0 до 1.0 | Параметр фокусного расстояния по оси x (по умолчанию: 0.5, шаг: 0.000000001) |
| `fy` | FLOAT | Нет | от 0.0 до 1.0 | Параметр фокусного расстояния по оси y (по умолчанию: 0.5, шаг: 0.000000001) |
| `cx` | FLOAT | Нет | от 0.0 до 1.0 | Координата x главной точки (по умолчанию: 0.5, шаг: 0.01) |
| `cy` | FLOAT | Нет | от 0.0 до 1.0 | Координата y главной точки (по умолчанию: 0.5, шаг: 0.01) |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `camera_embedding` | TENSOR | Сгенерированный тензор вложения камеры, содержащий последовательность траектории |
| `width` | INT | Значение ширины, использованное для обработки |
| `height` | INT | Значение высоты, использованное для обработки |
| `length` | INT | Значение длины, использованное для обработки |
