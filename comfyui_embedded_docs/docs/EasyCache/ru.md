Вот перевод документации на русский язык:

Узел EasyCache реализует встроенную систему кэширования для моделей, повышающую производительность за счёт повторного использования ранее вычисленных шагов в процессе сэмплирования. Он добавляет функциональность EasyCache к модели с настраиваемыми пороговыми значениями, определяющими моменты начала и завершения использования кэша в ходе сэмплирования.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `модель` | MODEL | Да | - | Модель, к которой добавляется EasyCache. |
| `порог_повторного_использования` | FLOAT | Нет | 0.0 - 3.0 | Порог для повторного использования кэшированных шагов (по умолчанию: 0.2). |
| `начальный_процент` | FLOAT | Нет | 0.0 - 1.0 | Относительный шаг сэмплирования для начала использования EasyCache (по умолчанию: 0.15). |
| `конечный_процент` | FLOAT | Нет | 0.0 - 1.0 | Относительный шаг сэмплирования для завершения использования EasyCache (по умолчанию: 0.95). |
| `подробный` | BOOLEAN | Нет | - | Включает подробное логирование (по умолчанию: False). |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `модель` | MODEL | Модель с добавленной функциональностью EasyCache. |
