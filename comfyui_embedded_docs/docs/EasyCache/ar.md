# ترجمة وثيقة عقدة EasyCache

عقدة EasyCache تنفذ نظام تخزين مؤقت أصلي للنماذج لتحسين الأداء عن طريق إعادة استخدام الخطوات المحسوبة سابقًا أثناء عملية أخذ العينات. تضيف وظيفة EasyCache إلى نموذج مع عتبات قابلة للتكوين لتحديد متى تبدأ وتتوقف عن استخدام التخزين المؤقت خلال الجدول الزمني لأخذ العينات.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|---------|---------------|--------|---------|-------|
| `model` | MODEL | نعم | - | النموذج المراد إضافة EasyCache إليه. |
| `reuse_threshold` | FLOAT | لا | 0.0 - 3.0 | عتبة إعادة استخدام الخطوات المخزنة مؤقتًا (الافتراضي: 0.2). |
| `start_percent` | FLOAT | لا | 0.0 - 1.0 | خطوة أخذ العينات النسبية لبدء استخدام EasyCache (الافتراضي: 0.15). |
| `end_percent` | FLOAT | لا | 0.0 - 1.0 | خطوة أخذ العينات النسبية لإنهاء استخدام EasyCache (الافتراضي: 0.95). |
| `verbose` | BOOLEAN | لا | - | ما إذا كان سيتم تسجيل معلومات مفصلة (الافتراضي: False). |

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|-------------|---------------|-------|
| `model` | MODEL | النموذج مع إضافة وظيفة EasyCache إليه. |
