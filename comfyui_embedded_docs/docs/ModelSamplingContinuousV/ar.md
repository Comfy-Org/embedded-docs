عقدة ModelSamplingContinuousV تقوم بتعديل سلوك أخذ العينات للنموذج من خلال تطبيق معاملات أخذ العينات المستمرة للتنبؤ-V. تقوم بإنشاء نسخة مطابقة للنموذج المُدخل وتكوينه بإعدادات نطاق سيغما مخصصة للتحكم المتقدم في عملية أخذ العينات. يتيح ذلك للمستخدمين ضبط عملية أخذ العينات بدقة باستخدام قيم سيغما دنيا وقصوى محددة.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|---------|--------------|--------|--------|-------|
| `model` | MODEL | نعم | - | النموذج المُدخل المراد تعديله باستخدام أخذ العينات المستمرة للتنبؤ-V |
| `sampling` | STRING | نعم | `"v_prediction"` | طريقة أخذ العينات المراد تطبيقها (حاليًا يتم دعم التنبؤ-V فقط) |
| `sigma_max` | FLOAT | نعم | 0.0 - 1000.0 | القيمة القصوى لسيغما في عملية أخذ العينات (الافتراضي: 500.0) |
| `sigma_min` | FLOAT | نعم | 0.0 - 1000.0 | القيمة الدنيا لسيغما في عملية أخذ العينات (الافتراضي: 0.03) |
