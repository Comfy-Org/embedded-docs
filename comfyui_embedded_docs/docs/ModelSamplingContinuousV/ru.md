Вот перевод документации на русский язык:

Узел ModelSamplingContinuousV изменяет поведение семплирования модели, применяя параметры непрерывного V-предсказания. Он создает клон входной модели и настраивает его с пользовательскими параметрами диапазона сигма для расширенного управления семплированием. Это позволяет пользователям точно настраивать процесс семплирования с заданными минимальными и максимальными значениями сигма.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `модель` | MODEL | Да | - | Входная модель, которая будет изменена с помощью непрерывного V-предсказания |
| `выборка` | STRING | Да | `"v_prediction"` | Метод семплирования (в настоящее время поддерживается только V-предсказание) |
| `sigma_max` | FLOAT | Да | 0.0 - 1000.0 | Максимальное значение сигма для семплирования (по умолчанию: 500.0) |
| `sigma_min` | FLOAT | Да | 0.0 - 1000.0 | Минимальное значение сигма для семплирования (по умолчанию: 0.03) |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `модель` | MODEL | Измененная модель с примененным непрерывным V-предсказанием |
