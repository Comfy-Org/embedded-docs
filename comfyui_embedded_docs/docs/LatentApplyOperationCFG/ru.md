Узел `LatentApplyOperationCFG` применяет латентную операцию для модификации процесса управления conditioning в модели. Он работает путем перехвата выходных данных conditioning в процессе семплирования с классификатор-свободным управлением (CFG) и применения указанной операции к латентным представлениям перед их использованием для генерации.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `модель` | MODEL | Да | - | Модель, к которой будет применена операция CFG |
| `операция` | LATENT_OPERATION | Да | - | Латентная операция, применяемая в процессе семплирования CFG |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `модель` | MODEL | Модифицированная модель с примененной операцией CFG к ее процессу семплирования |
