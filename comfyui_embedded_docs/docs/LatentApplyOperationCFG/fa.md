گره LatentApplyOperationCFG یک عملیات نهفته را برای اصلاح فرآیند هدایت شرطی‌سازی در یک مدل اعمال می‌کند. این گره با رهگیری خروجی‌های شرطی‌سازی در طول فرآیند نمونه‌برداری CFG (راهنمایی بدون طبقه‌بند) و اعمال عملیات مشخص‌شده بر روی نمایش‌های نهفته قبل از استفاده برای تولید، کار می‌کند.
