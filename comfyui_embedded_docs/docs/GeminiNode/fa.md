این گره به کاربران امکان می‌دهد با مدل‌های هوش مصنوعی Gemini گوگل تعامل داشته و پاسخ‌های متنی دریافت کنند. می‌توانید انواع مختلفی از ورودی‌ها از جمله متن، تصویر، صدا، ویدئو و فایل را به عنوان زمینه (context) برای مدل فراهم کنید تا پاسخ‌های مرتبط‌تر و معنادارتری تولید کند. این گره به‌طور خودکار تمام ارتباطات API و تجزیه پاسخ‌ها را مدیریت می‌کند.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|---------|----------|--------|--------|---------|
| `prompt` | STRING | بله | - | ورودی‌های متنی به مدل، که برای تولید پاسخ استفاده می‌شود. می‌توانید دستورالعمل‌های دقیق، سوالات یا زمینه را برای مدل وارد کنید. پیش‌فرض: رشته خالی. |
| `model` | COMBO | بله | `gemini-2.5-pro-preview-05-06`<br>`gemini-2.5-flash-preview-04-17`<br>`gemini-2.5-pro`<br>`gemini-2.5-flash`<br>`gemini-3-pro-preview`<br>`gemini-3-1-pro`<br>`gemini-3-1-flash-lite` | مدل Gemini که برای تولید پاسخ‌ها استفاده می‌شود. پیش‌فرض: gemini-3-1-pro. |
| `seed` | INT | بله | 0 تا 18446744073709551615 | هنگامی که seed روی یک مقدار مشخص ثابت شود، مدل نهایت تلاش خود را می‌کند تا پاسخ یکسانی برای درخواست‌های تکراری ارائه دهد. خروجی قطعی تضمین نمی‌شود. همچنین تغییر مدل یا تنظیمات پارامترهایی مانند دما (temperature) می‌تواند باعث تغییر در پاسخ حتی با استفاده از مقدار seed یکسان شود. به‌طور پیش‌فرض، از یک مقدار seed تصادفی استفاده می‌شود. پیش‌فرض: 42. |
| `images` | IMAGE | خیر | - | تصویر(های) اختیاری برای استفاده به عنوان زمینه برای مدل. برای گنجاندن چندین تصویر، می‌توانید از گره Batch Images استفاده کنید. پیش‌فرض: None. |
| `audio` | AUDIO | خیر | - | صدای اختیاری برای استفاده به عنوان زمینه برای مدل. پیش‌فرض: None. |
| `video` | VIDEO | خیر | - | ویدئوی اختیاری برای استفاده به عنوان زمینه برای مدل. پیش‌فرض: None. |
| `files` | GEMINI_INPUT_FILES | خیر | - | فایل(های) اختیاری برای استفاده به عنوان زمینه برای مدل. ورودی‌های گره Gemini Generate Content Input Files را می‌پذیرد. پیش‌فرض: None. |
| `system_prompt` | STRING | خیر | - | دستورالعمل‌های بنیادین که رفتار هوش مصنوعی را تعیین می‌کند. پیش‌فرض: رشته خالی. این یک پارامتر پیشرفته است. |
