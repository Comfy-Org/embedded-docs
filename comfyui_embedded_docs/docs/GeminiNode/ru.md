Вот перевод документации на русский язык в соответствии с вашими правилами:

Этот узел позволяет пользователям взаимодействовать с моделями ИИ Gemini от Google для генерации текстовых ответов. Вы можете предоставлять различные типы входных данных, включая текст, изображения, аудио, видео и файлы, в качестве контекста для модели, чтобы генерировать более релевантные и осмысленные ответы. Узел автоматически обрабатывает всё взаимодействие с API и разбор ответов.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `prompt` | STRING | Да | - | Текстовые входные данные для модели, используемые для генерации ответа. Вы можете включить подробные инструкции, вопросы или контекст для модели. По умолчанию: пустая строка. |
| `model` | COMBO | Да | `gemini-2.5-pro-preview-05-06`<br>`gemini-2.5-flash-preview-04-17`<br>`gemini-2.5-pro`<br>`gemini-2.5-flash`<br>`gemini-3-pro-preview`<br>`gemini-3-1-pro`<br>`gemini-3-1-flash-lite` | Модель Gemini, используемая для генерации ответов. По умолчанию: gemini-3-1-pro. |
| `seed` | INT | Да | от 0 до 18446744073709551615 | Когда значение seed фиксировано, модель прилагает максимум усилий для предоставления одинакового ответа на повторные запросы. Детерминированный вывод не гарантируется. Кроме того, изменение модели или настроек параметров, таких как температура, может вызвать вариации в ответе, даже если вы используете одно и то же значение seed. По умолчанию используется случайное значение seed. По умолчанию: 42. |
| `images` | IMAGE | Нет | - | Необязательное изображение(я) для использования в качестве контекста для модели. Чтобы включить несколько изображений, вы можете использовать узел Batch Images. По умолчанию: None. |
| `audio` | AUDIO | Нет | - | Необязательное аудио для использования в качестве контекста для модели. По умолчанию: None. |
| `video` | VIDEO | Нет | - | Необязательное видео для использования в качестве контекста для модели. По умолчанию: None. |
| `files` | GEMINI_INPUT_FILES | Нет | - | Необязательный файл(ы) для использования в качестве контекста для модели. Принимает входные данные от узла Gemini Generate Content Input Files. По умолчанию: None. |
| `system_prompt` | STRING | Нет | - | Фундаментальные инструкции, определяющие поведение ИИ. По умолчанию: пустая строка. Это расширенный параметр. |

## Выходные параметры

| Имя выходного параметра | Тип данных | Описание |
|-------------|-----------|-------------|
| `STRING` | STRING | Текстовый ответ, сгенерированный моделью Gemini. |
