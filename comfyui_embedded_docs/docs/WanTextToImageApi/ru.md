Узел Wan Text to Image генерирует изображения на основе текстовых описаний. Он использует модели ИИ для создания визуального контента из текстовых запросов, поддерживая ввод как на английском, так и на китайском языке. Узел предоставляет различные элементы управления для настройки размера, качества и стилевых предпочтений выходного изображения.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `model` | COMBO | Да | "wan2.5-t2i-preview" | Используемая модель (по умолчанию: "wan2.5-t2i-preview") |
| `prompt` | STRING | Да | - | Запрос, описывающий элементы и визуальные особенности. Поддерживает английский и китайский языки (по умолчанию: пусто) |
| `negative_prompt` | STRING | Нет | - | Негативный запрос, описывающий, чего следует избегать (по умолчанию: пусто) |
| `width` | INT | Нет | 768-1440 | Ширина изображения в пикселях (по умолчанию: 1024, шаг: 32) |
| `height` | INT | Нет | 768-1440 | Высота изображения в пикселях (по умолчанию: 1024, шаг: 32) |
| `seed` | INT | Нет | 0-2147483647 | Сид для генерации (по умолчанию: 0) |
| `prompt_extend` | BOOLEAN | Нет | - | Улучшать ли запрос с помощью ИИ (по умолчанию: True) |
| `watermark` | BOOLEAN | Нет | - | Добавлять ли водяной знак, указывающий на создание ИИ, к результату (по умолчанию: False) |

## Выходные данные

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `output` | IMAGE | Сгенерированное изображение на основе текстового запроса |
