## نظرة عامة

تقوم عقدة "نص إلى صورة Wan" بإنشاء صور بناءً على أوصاف نصية. تستخدم نماذج الذكاء الاصطناعي لإنتاج محتوى بصري من التعليمات النصية المكتوبة، وتدعم إدخال النصوص باللغتين الإنجليزية والصينية. توفر العقدة عناصر تحكم متنوعة لضبط حجم الصورة الناتجة وجودتها وتفضيلات الأسلوب.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|---------|--------------|--------|--------|-------|
| `model` | COMBO | نعم | "wan2.5-t2i-preview" | النموذج المستخدم (الافتراضي: "wan2.5-t2i-preview") |
| `prompt` | STRING | نعم | - | تعليمة تصف العناصر والخصائص البصرية. تدعم اللغتين الإنجليزية والصينية (الافتراضي: فارغ) |
| `negative_prompt` | STRING | لا | - | تعليمة سلبية تصف ما يجب تجنبه (الافتراضي: فارغ) |
| `width` | INT | لا | 768-1440 | عرض الصورة بالبكسل (الافتراضي: 1024، الخطوة: 32) |
| `height` | INT | لا | 768-1440 | ارتفاع الصورة بالبكسل (الافتراضي: 1024، الخطوة: 32) |
| `seed` | INT | لا | 0-2147483647 | البذرة المستخدمة في التوليد (الافتراضي: 0) |
| `prompt_extend` | BOOLEAN | لا | - | هل سيتم تحسين التعليمة بمساعدة الذكاء الاصطناعي (الافتراضي: صحيح) |
| `watermark` | BOOLEAN | لا | - | هل ستتم إضافة علامة مائية منشأة بالذكاء الاصطناعي إلى النتيجة (الافتراضي: خطأ) |
