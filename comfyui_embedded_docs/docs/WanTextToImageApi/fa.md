### مرور کلی

گره «تبدیل متن به تصویر Wan» بر اساس توضیحات متنی، تصویر تولید می‌کند. این گره با استفاده از مدل‌های هوش مصنوعی، محتوای بصری را از نوشته‌های متنی خلق کرده و از ورودی متن به دو زبان انگلیسی و چینی پشتیبانی می‌کند. همچنین کنترل‌های متنوعی برای تنظیم اندازه، کیفیت و اولویت‌های سبک تصویر خروجی ارائه می‌دهد.

## ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|---------|----------|-------|--------|---------|
| `مدل` | COMBO | بله | "wan2.5-t2i-preview" | مدل مورد استفاده (پیش‌فرض: "wan2.5-t2i-preview") |
| `پرامپت` | STRING | بله | - | توضیحاتی درباره عناصر و ویژگی‌های بصری. از زبان انگلیسی و چینی پشتیبانی می‌کند (پیش‌فرض: خالی) |
| `پرامپت منفی` | STRING | خیر | - | توضیح منفی برای مشخص کردن مواردی که باید از آن‌ها اجتناب شود (پیش‌فرض: خالی) |
| `عرض` | INT | خیر | 768-1440 | عرض تصویر بر حسب پیکسل (پیش‌فرض: 1024، گام: 32) |
| `ارتفاع` | INT | خیر | 768-1440 | ارتفاع تصویر بر حسب پیکسل (پیش‌فرض: 1024، گام: 32) |
| `بذر` | INT | خیر | 0-2147483647 | دانه (seed) مورد استفاده برای تولید (پیش‌فرض: 0) |
| `افزایش پرامپت` | BOOLEAN | خیر | - | آیا توضیحات با کمک هوش مصنوعی بهبود یابد (پیش‌فرض: True) |
| `واترمارک` | BOOLEAN | خیر | - | آیا واترمارک تولیدشده با هوش مصنوعی به نتیجه اضافه شود (پیش‌فرض: False) |
