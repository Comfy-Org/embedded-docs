Генерирует SVG синхронно из входного изображения. Этот узел преобразует растровые изображения в формат векторной графики, обрабатывая каждое изображение во входном пакете и объединяя результаты в единый выходной SVG.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `изображение` | IMAGE | Да | - | Входное изображение для преобразования в формат SVG |

## Выходы

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `SVG` | SVG | Сгенерированный вывод векторной графики, объединяющий все обработанные изображения |
