Вот перевод документации на русский язык:

Узел ModelSamplingSD3 применяет параметры семплирования Stable Diffusion 3 к модели. Он изменяет поведение семплирования модели путем настройки параметра сдвига (`shift`), который управляет характеристиками распределения семплирования. Узел создает модифицированную копию входной модели с примененной указанной конфигурацией семплирования.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `модель` | MODEL | Да | - | Входная модель, к которой применяются параметры семплирования SD3 |
| `сдвиг` | FLOAT | Да | 0.0 - 100.0 | Управляет параметром сдвига семплирования (по умолчанию: 3.0) |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `модель` | MODEL | Модифицированная модель с примененными параметрами семплирования SD3 |
