# ترجمة وثيقة عقدة تمديد فيديو Kling

عقدة تمديد فيديو Kling تتيح لك تمديد مقاطع الفيديو التي تم إنشاؤها بواسطة عُقد Kling الأخرى. تأخذ العقدة فيديو موجودًا محددًا بواسطة معرف الفيديو الخاص به، وتُنتج محتوى إضافيًا بناءً على نصوص التوجيه الخاصة بك. تعمل العقدة عن طريق إرسال طلب التمديد الخاص بك إلى واجهة برمجة تطبيقات Kling، ثم إعادة الفيديو الممدد مع معرفه ومدته الجديدين.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|---------|---------------|--------|--------|-------|
| `prompt` | STRING | لا | - | نص توجيه إيجابي لتوجيه تمديد الفيديو |
| `negative_prompt` | STRING | لا | - | نص توجيه سلبي للعناصر التي يجب تجنبها في الفيديو الممدد |
| `cfg_scale` | FLOAT | لا | 0.0 - 1.0 | يتحكم في قوة توجيه النص (القيمة الافتراضية: 0.5) |
| `video_id` | STRING | نعم | - | معرف الفيديو المراد تمديده. يدعم مقاطع الفيديو المُنشأة بواسطة عمليات تحويل النص إلى فيديو، وتحويل الصورة إلى فيديو، وعمليات تمديد الفيديو السابقة. لا يمكن أن تتجاوز المدة الإجمالية 3 دقائق بعد التمديد. |

**ملاحظة:** يجب أن يشير `video_id` إلى فيديو تم إنشاؤه بواسطة عُقد Kling أخرى، ولا يمكن أن تتجاوز المدة الإجمالية بعد التمديد 3 دقائق.

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|-------------|---------------|--------|
| `output` | VIDEO | الفيديو الممدد الذي تم إنشاؤه بواسطة واجهة برمجة تطبيقات Kling |
| `video_id` | STRING | المعرف الفريد للفيديو الممدد |
| `duration` | STRING | مدة الفيديو الممدد |
