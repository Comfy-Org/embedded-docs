گره توسعه ویدیوی Kling به شما امکان می‌دهد ویدیوهای ایجاد شده توسط سایر گره‌های Kling را گسترش دهید. این گره یک ویدیوی موجود را که با شناسه ویدیوی آن مشخص شده است دریافت کرده و بر اساس دستورات متنی شما محتوای اضافی تولید می‌کند. عملکرد این گره به این صورت است که درخواست توسعه شما را به API Kling ارسال کرده و ویدیوی توسعه‌یافته را همراه با شناسه و مدت زمان جدید آن بازمی‌گرداند.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `prompt` | STRING | خیر | - | دستور متنی مثبت برای هدایت توسعه ویدیو |
| `negative_prompt` | STRING | خیر | - | دستور متنی منفی برای عناصری که باید در ویدیوی توسعه‌یافته اجتناب شوند |
| `cfg_scale` | FLOAT | خیر | 0.0 - 1.0 | قدرت هدایت دستور را کنترل می‌کند (پیش‌فرض: 0.5) |
| `video_id` | STRING | بله | - | شناسه ویدیویی که باید توسعه یابد. از ویدیوهای تولید شده توسط متن به ویدیو، تصویر به ویدیو و عملیات توسعه ویدیوی قبلی پشتیبانی می‌کند. مدت زمان کل پس از توسعه نمی‌تواند از ۳ دقیقه تجاوز کند. |

**توجه:** `video_id` باید به ویدیویی که توسط سایر گره‌های Kling ایجاد شده است اشاره کند و مدت زمان کل پس از توسعه نمی‌تواند از ۳ دقیقه تجاوز کند.

## خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-------------|-----------|-------------|
| `output` | VIDEO | ویدیوی توسعه‌یافته تولید شده توسط API Kling |
| `video_id` | STRING | شناسه یکتای ویدیوی توسعه‌یافته |
| `duration` | STRING | مدت زمان ویدیوی توسعه‌یافته |
