Вот перевод документации на русский язык:

Узел Kling Video Extend позволяет расширять видео, созданные другими узлами Kling. Он принимает существующее видео, идентифицируемое по его идентификатору, и генерирует дополнительный контент на основе ваших текстовых подсказок. Узел работает, отправляя ваш запрос на расширение в API Kling и возвращая расширенное видео вместе с его новым идентификатором и длительностью.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `prompt` | STRING | Нет | - | Положительная текстовая подсказка для направления расширения видео |
| `negative_prompt` | STRING | Нет | - | Отрицательная текстовая подсказка для элементов, которых следует избегать в расширенном видео |
| `cfg_scale` | FLOAT | Нет | 0.0 - 1.0 | Управляет силой влияния подсказки (по умолчанию: 0.5) |
| `video_id` | STRING | Да | - | Идентификатор видео, которое необходимо расширить. Поддерживает видео, созданные с помощью преобразования текста в видео, изображения в видео, а также предыдущих операций расширения видео. Общая длительность после расширения не может превышать 3 минуты. |

**Примечание:** `video_id` должен ссылаться на видео, созданное другими узлами Kling, а общая длительность после расширения не может превышать 3 минуты.

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `output` | VIDEO | Расширенное видео, сгенерированное API Kling |
| `video_id` | STRING | Уникальный идентификатор расширенного видео |
| `duration` | STRING | Длительность расширенного видео |
