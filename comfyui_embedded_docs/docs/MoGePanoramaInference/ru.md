## Обзор

Этот узел выполняет оценку глубины на равнопромежуточных панорамных изображениях. Он работает путем разделения панорамы на 12 перспективных видов, запуска модели оценки глубины MoGe на каждом виде, а затем объединения результатов обратно в единую полную карту глубины для исходной панорамы.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `moge_model` | MOGE_MODEL | Да | | Модель MoGe для выполнения инференса. |
| `image` | IMAGE | Да | | Равнопромежуточное панорамное изображение (любое соотношение сторон). |
| `resolution_level` | INT | Да | от 0 до 9 | Уровень детализации для каждого вида. Более высокие значения создают более детальные карты глубины (по умолчанию: 9). |
| `split_resolution` | INT | Да | от 256 до 1024 | Разрешение каждого перспективного вида после разделения панорамы (по умолчанию: 512). |
| `merge_resolution` | INT | Да | от 256 до 8192 | Разрешение по длинной стороне финальной объединенной равнопромежуточной карты глубины (по умолчанию: 1920). |
| `batch_size` | INT | Да | от 1 до 12 | Количество перспективных видов для обработки в каждом пакете инференса. Общее количество видов — 12 (по умолчанию: 4). |

## Выходные параметры

| Имя выходного параметра | Тип данных | Описание |
|-------------|-----------|-------------|
| `moge_geometry` | MOGE_GEOMETRY | Словарь, содержащий оцененную геометрию: `points` (3D-облако точек), `depth` (карта глубины), `mask` (маска допустимой области) и `image` (входное изображение). |
