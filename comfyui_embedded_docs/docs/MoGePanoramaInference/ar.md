# نظرة عامة

تقوم هذه العقدة بإجراء تقدير العمق على الصور البانورامية متساوية المستطيلات. تعمل عن طريق تقسيم البانوراما إلى 12 منظرًا منظوريًا، وتشغيل نموذج تقدير العمق MoGe على كل منظر، ثم دمج النتائج مرة أخرى في خريطة عمق واحدة كاملة للبانوراما الأصلية.

# المدخلات

| المعامل | نوع البيانات | إجباري | النطاق | الوصف |
|---------|--------------|--------|--------|-------|
| `moge_model` | MOGE_MODEL | نعم | | نموذج MoGe المراد استخدامه للاستدلال. |
| `image` | IMAGE | نعم | | صورة بانورامية متساوية المستطيلات (أي نسبة أبعاد). |
| `resolution_level` | INT | نعم | 0 إلى 9 | مستوى تفاصيل كل منظر. القيم الأعلى تنتج خرائط عمق أكثر تفصيلاً (الافتراضي: 9). |
| `split_resolution` | INT | نعم | 256 إلى 1024 | دقة كل منظر منظوري بعد تقسيم البانوراما (الافتراضي: 512). |
| `merge_resolution` | INT | نعم | 256 إلى 8192 | دقة الجانب الطويل لخريطة العمق متساوية المستطيلات النهائية المدمجة (الافتراضي: 1920). |
| `batch_size` | INT | نعم | 1 إلى 12 | عدد المناظير المنظورية المراد معالجتها في كل دفعة استدلال. إجمالي عدد المناظير هو 12 (الافتراضي: 4). |

# المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|------------|--------------|-------|
| `moge_geometry` | MOGE_GEOMETRY | قاموس يحتوي على الهندسة المقدرة: `points` (سحابة نقاط ثلاثية الأبعاد)، `depth` (خريطة العمق)، `mask` (قناع المنطقة الصالحة)، و `image` (الصورة المدخلة). |
