## مرور کلی

این گره تخمین عمق را بر روی تصاویر پانورامای استوانه‌ای (equirectangular) انجام می‌دهد. عملکرد آن به این صورت است که پانوراما را به ۱۲ نمای پرسپکتیو تقسیم می‌کند، مدل تخمین عمق MoGe را بر روی هر نما اجرا می‌کند و سپس نتایج را در یک نقشه عمق کامل و یکپارچه برای پانورامای اصلی ادغام می‌کند.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|---------|----------|--------|--------|---------|
| `moge_model` | MOGE_MODEL | بله | | مدل MoGe مورد استفاده برای استنتاج. |
| `image` | IMAGE | بله | | تصویر پانورامای استوانه‌ای (با هر نسبت ابعادی). |
| `resolution_level` | INT | بله | ۰ تا ۹ | سطح جزئیات هر نما. مقادیر بالاتر نقشه‌های عمق دقیق‌تری تولید می‌کنند (پیش‌فرض: ۹). |
| `split_resolution` | INT | بله | ۲۵۶ تا ۱۰۲۴ | وضوح هر نمای پرسپکتیو پس از تقسیم پانوراما (پیش‌فرض: ۵۱۲). |
| `merge_resolution` | INT | بله | ۲۵۶ تا ۸۱۹۲ | وضوح ضلع بلند نقشه عمق استوانه‌ای نهایی ادغام‌شده (پیش‌فرض: ۱۹۲۰). |
| `batch_size` | INT | بله | ۱ تا ۱۲ | تعداد نماهای پرسپکتیو پردازش‌شده در هر دسته استنتاج. تعداد کل نماها ۱۲ عدد است (پیش‌فرض: ۴). |
