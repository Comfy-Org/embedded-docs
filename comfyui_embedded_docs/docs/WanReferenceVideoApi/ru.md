Вот перевод документации на русский язык:

Узел Wan Reference to Video использует внешний вид и голос из одного или нескольких эталонных видео, а также текстовый запрос для генерации нового видео. Он сохраняет согласованность с персонажами из эталонного материала, создавая новый контент на основе вашего описания.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `model` | COMBO | Да | `"wan2.6-r2v"` | Конкретная модель ИИ для генерации видео. |
| `prompt` | STRING | Да | - | Описание элементов и визуальных характеристик для нового видео. Поддерживает английский и китайский языки. Используйте идентификаторы, такие как `character1` и `character2`, для обозначения персонажей из эталонных видео. |
| `negative_prompt` | STRING | Нет | - | Описание элементов или характеристик, которых следует избегать в сгенерированном видео. |
| `reference_videos` | AUTOGROW | Да | - | Список видео, используемых в качестве эталонов для внешнего вида и голоса персонажей. Необходимо предоставить хотя бы одно видео. Каждому видео можно присвоить имя, например `character1`, `character2` или `character3`. |
| `size` | COMBO | Да | `"720p: 1:1 (960x960)"`<br>`"720p: 16:9 (1280x720)"`<br>`"720p: 9:16 (720x1280)"`<br>`"720p: 4:3 (1088x832)"`<br>`"720p: 3:4 (832x1088)"`<br>`"1080p: 1:1 (1440x1440)"`<br>`"1080p: 16:9 (1920x1080)"`<br>`"1080p: 9:16 (1080x1920)"`<br>`"1080p: 4:3 (1632x1248)"`<br>`"1080p: 3:4 (1248x1632)"` | Разрешение и соотношение сторон для выходного видео. |
| `duration` | INT | Да | от 5 до 10 | Длительность сгенерированного видео в секундах. Значение должно быть кратно 5 (по умолчанию: 5). |
| `seed` | INT | Нет | от 0 до 2147483647 | Значение случайного зерна для воспроизводимых результатов. Значение 0 генерирует случайное зерно. |
| `shot_type` | COMBO | Да | `"single"`<br>`"multi"` | Указывает, является ли сгенерированное видео одним непрерывным кадром или содержит несколько кадров с монтажными склейками. |
| `watermark` | BOOLEAN | Нет | - | При включении добавляет водяной знак, созданный ИИ, в итоговое видео (по умолчанию: False). |

**Ограничения:**

* Каждое видео, предоставленное в `reference_videos`, должно иметь длительность от 2 до 30 секунд.
* Параметр `duration` ограничен конкретными значениями (5 или 10 секунд).

## Выходные данные

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `output` | VIDEO | Новый сгенерированный видеофайл. |
