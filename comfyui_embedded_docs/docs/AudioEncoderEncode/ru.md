Вот перевод документации на русский язык:

Узел AudioEncoderEncode обрабатывает аудиоданные, кодируя их с помощью модели аудиокодера. Он принимает аудиовход и преобразует его в закодированное представление, которое может быть использовано для дальнейшей обработки в конвейере кондиционирования. Этот узел преобразует необработанные аудиосигналы в формат, подходящий для приложений машинного обучения, работающих со звуком.

## Входные параметры

| Параметр | Тип данных | Тип входа | По умолчанию | Диапазон | Описание |
|----------|------------|-----------|--------------|----------|----------|
| `audio_encoder` | AUDIO_ENCODER | Обязательный | - | - | Модель аудиокодера, используемая для обработки аудиовхода |
| `audio` | AUDIO | Обязательный | - | - | Аудиоданные, содержащие информацию о сигнале и частоте дискретизации |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `output` | AUDIO_ENCODER_OUTPUT | Закодированное аудиопредставление, сгенерированное аудиокодером |
