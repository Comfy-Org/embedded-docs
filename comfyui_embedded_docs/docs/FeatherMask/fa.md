گره `FeatherMask` یک افکت پَرمانند (محو‌سازی لبه) را به لبه‌های یک ماسک مشخص اعمال می‌کند و با تنظیم شفافیت لبه‌ها بر اساس فواصل تعیین‌شده از هر لبه، آن‌ها را به‌صورت نرم و تدریجی محو می‌کند. این کار باعث ایجاد لبه‌های نرم‌تر و ترکیب‌شده‌تر می‌شود.

## ورودی‌ها

| پارامتر | نوع داده | توضیحات |
|-----------|--------------|-------------|
| `ماسک`    | MASK         | ماسکی که افکت پَرمانند روی آن اعمال خواهد شد. این ماسک ناحیه‌ای از تصویر را که تحت تأثیر محو‌سازی لبه قرار می‌گیرد، مشخص می‌کند. |
| `چپ`    | INT          | فاصله از لبه چپ را مشخص می‌کند که درون آن افکت پَرمانند اعمال شود. |
| `بالا`     | INT          | فاصله از لبه بالا را مشخص می‌کند که درون آن افکت پَرمانند اعمال شود. |
| `راست`   | INT          | فاصله از لبه راست را مشخص می‌کند که درون آن افکت پَرمانند اعمال شود. |
| `پایین`  | INT          | فاصله از لبه پایین را مشخص می‌کند که درون آن افکت پَرمانند اعمال شود. |

## خروجی‌ها

| پارامتر | نوع داده | توضیحات |

| `ماسک`    | MASK         | خروجی، نسخه‌ای تغییر‌یافته از ماسک ورودی است که افکت پَرمانند روی لبه‌های آن اعمال شده است. |
