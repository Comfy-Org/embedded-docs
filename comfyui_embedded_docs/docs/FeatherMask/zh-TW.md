`FeatherMask` 節點會對指定遮罩的邊緣套用羽化效果，透過根據各邊指定的距離調整邊緣不透明度，使遮罩邊緣產生平滑的過渡。這能創造出更柔和、更自然的邊緣效果。

## 輸入

| 參數 | 資料類型 | 說明 |
|-----------|--------------|-------------|
| `mask`    | MASK         | 將套用羽化效果的遮罩。此遮罩決定了影像中受羽化影響的區域。 |
| `left`    | INT          | 指定左側邊緣內套用羽化效果的距離。 |
| `top`     | INT          | 指定頂部邊緣內套用羽化效果的距離。 |
| `right`   | INT          | 指定右側邊緣內套用羽化效果的距離。 |
| `bottom`  | INT          | 指定底部邊緣內套用羽化效果的距離。 |

## 輸出

| 參數 | 資料類型 | 說明 |
|-----------|--------------|-------------|
| `mask`    | MASK         | 輸出為輸入遮罩的修改版本，其邊緣已套用羽化效果。 |
