Узел `FeatherMask` применяет эффект размытия (feathering) к краям заданной маски, плавно изменяя прозрачность краев в зависимости от указанных расстояний от каждого края. Это создает более мягкий и плавный эффект перехода.

## Входные параметры

| Параметр | Тип данных | Описание |
|-----------|--------------|-------------|
| `mask`    | MASK         | Маска, к которой будет применен эффект размытия. Определяет область изображения, на которую повлияет размытие. |
| `left`    | INT          | Задает расстояние от левого края, в пределах которого будет применяться эффект размытия. |
| `top`     | INT          | Задает расстояние от верхнего края, в пределах которого будет применяться эффект размытия. |
| `right`   | INT          | Задает расстояние от правого края, в пределах которого будет применяться эффект размытия. |
| `bottom`  | INT          | Задает расстояние от нижнего края, в пределах которого будет применяться эффект размытия. |

## Выходные параметры

| Параметр | Тип данных | Описание |
|-----------|--------------|-------------|
| `mask`    | MASK         | Результат — измененная версия входной маски с эффектом размытия, примененным к ее краям. |
