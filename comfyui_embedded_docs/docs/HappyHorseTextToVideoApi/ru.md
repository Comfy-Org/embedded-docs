## Обзор

Генерирует видео на основе текстового запроса с использованием модели HappyHorse. Этот узел отправляет ваш запрос и настройки в API HappyHorse, ожидает генерации видео, а затем загружает результат.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `model` | DICT | Да | См. описание | Словарь, содержащий выбор модели и связанные с ней параметры. Модель должна быть `"happyhorse-1.0-t2v"`. Этот словарь включает следующие подпараметры:<br><br>**`prompt`** (STRING): Текстовое описание видео, которое вы хотите сгенерировать. Поддерживает английский и китайский языки. (по умолчанию: "").<br>**`resolution`** (COMBO): Разрешение выходного видео. Варианты: `"720P"`, `"1080P"`.<br>**`ratio`** (COMBO): Соотношение сторон выходного видео. Варианты: `"16:9"`, `"9:16"`, `"1:1"`, `"4:3"`, `"3:4"`.<br>**`duration`** (INT): Длительность видео в секундах. (по умолчанию: 5, мин: 3, макс: 15, шаг: 1). |
| `seed` | INT | Да | от 0 до 2147483647 | Зерно для генерации. Использование одного и того же зерна с одинаковыми входными данными приведет к одинаковому результату. (по умолчанию: 0). |
| `watermark` | BOOLEAN | Нет | True / False | Добавлять ли водяной знак, указывающий на создание ИИ, к результату. (по умолчанию: False). |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `VIDEO` | VIDEO | Сгенерированный видеофайл. |
