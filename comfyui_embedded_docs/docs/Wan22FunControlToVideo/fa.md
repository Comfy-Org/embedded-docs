### مرور کلی

گره `Wan22FunControlToVideo` با استفاده از معماری مدل ویدیویی Wan، آماده‌سازی شرطی‌سازی (conditioning) و بازنمایی‌های نهفته (latent) را برای تولید ویدیو انجام می‌دهد. این گره ورودی‌های شرطی‌سازی مثبت و منفی را به همراه تصاویر مرجع اختیاری و ویدیوهای کنترلی پردازش می‌کند تا بازنمایی‌های فضای نهفته لازم برای سنتز ویدیو ایجاد شود. همچنین این گره ابعاد مکانی و زمانی را برای تولید داده‌های شرطی‌سازی مناسب مدل‌های ویدیویی مقیاس‌دهی می‌کند.

## ورودی‌ها

| پارامتر | نوع داده | اجباری | محدوده | توضیحات |
|---------|----------|--------|--------|---------|
| `positive` | CONDITIONING | بله | - | ورودی شرطی‌سازی مثبت برای هدایت تولید ویدیو |
| `negative` | CONDITIONING | بله | - | ورودی شرطی‌سازی منفی برای هدایت تولید ویدیو |
| `vae` | VAE | بله | - | مدل VAE مورد استفاده برای رمزگذاری تصاویر به فضای نهفته |
| `width` | INT | بله | 16 تا MAX_RESOLUTION | عرض ویدیوی خروجی بر حسب پیکسل (پیش‌فرض: 832، گام: 16) |
| `height` | INT | بله | 16 تا MAX_RESOLUTION | ارتفاع ویدیوی خروجی بر حسب پیکسل (پیش‌فرض: 480، گام: 16) |
| `length` | INT | بله | 1 تا MAX_RESOLUTION | تعداد فریم‌های دنباله ویدیویی (پیش‌فرض: 81، گام: 4) |
| `batch_size` | INT | بله | 1 تا 4096 | تعداد دنباله‌های ویدیویی برای تولید (پیش‌فرض: 1) |
| `ref_image` | IMAGE | خیر | - | تصویر مرجع اختیاری برای ارائه راهنمایی بصری |
| `control_video` | IMAGE | خیر | - | ویدیوی کنترلی اختیاری برای هدایت فرآیند تولید |

**نکته:** پارامتر `length` به صورت تکه‌های ۴ فریمی پردازش می‌شود و گره به طور خودکار مقیاس‌دهی زمانی را برای فضای نهفته انجام می‌دهد. هنگامی که `ref_image` ارائه می‌شود، از طریق نهفته‌های مرجع بر شرطی‌سازی تأثیر می‌گذارد. هنگامی که `control_video` ارائه می‌شود، مستقیماً بر بازنمایی نهفته الحاقی (concat latent) مورد استفاده در شرطی‌سازی تأثیر می‌گذارد. پارامتر `start_image` به عنوان ورودی در طرح این گره نمایش داده نشده است اما در منطق اجرا به آن ارجاع داده می‌شود.

## خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-----------|----------|---------|
| `positive` | CONDITIONING | شرطی‌سازی مثبت اصلاح‌شده با داده‌های نهفته مخصوص ویدیو شامل نهفته الحاقی، ماسک و نهفته‌های مرجع اختیاری |
| `negative` | CONDITIONING | شرطی‌سازی منفی اصلاح‌شده با داده‌های نهفته مخصوص ویدیو شامل نهفته الحاقی، ماسک و نهفته‌های مرجع اختیاری |
| `latent` | LATENT | تنسور نهفته خالی با ابعاد مناسب برای تولید ویدیو بر اساس اندازه دسته، کانال‌های نهفته و مقیاس‌دهی مکانی/زمانی |
