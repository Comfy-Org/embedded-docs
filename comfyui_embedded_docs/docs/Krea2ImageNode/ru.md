## Обзор

Узел Krea 2 Image генерирует изображения с использованием модели искусственного интеллекта Krea 2. Он поддерживает два варианта модели: Medium для выразительных иллюстраций и Large для выразительного фотореализма. При необходимости вы можете добавить мудборд и до 10 референсов стиля изображения, чтобы влиять на сгенерированное изображение.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `prompt` | STRING | Да | Н/Д | Текстовый запрос для изображения. |
| `model` | DICT | Да | См. ниже | Krea 2 Medium лучше всего подходит для выразительных иллюстраций; Krea 2 Large лучше всего подходит для выразительного фотореализма. |
| `seed` | INT | Да | от 0 до 2147483647 | Случайное зерно для воспроизводимости результатов (по умолчанию: 0). |

Параметр `model` представляет собой словарь со следующими подпараметрами:

| Подпараметр | Тип данных | Обязательный | Диапазон | Описание |
|---------------|-----------|----------|-------|-------------|
| `model` | STRING | Да | `"krea 2 medium"`<br>`"krea 2 large"` | Выбирает вариант модели Krea 2. |
| `aspect_ratio` | STRING | Да | Н/Д | Соотношение сторон для сгенерированного изображения. |
| `resolution` | STRING | Да | Н/Д | Разрешение для сгенерированного изображения. |
| `creativity` | FLOAT | Да | Н/Д | Управляет уровнем креативности генерации. |
| `moodboard_id` | STRING | Нет | Н/Д | UUID мудборда Krea для влияния на изображение. Должен быть действительным UUID. |
| `moodboard_strength` | FLOAT | Нет | Н/Д | Сила влияния мудборда (по умолчанию: 0.35). |
| `style_reference` | LIST | Нет | от 0 до 10 элементов | Список референсов стиля изображения. Каждый референс должен содержать `url` (STRING) и `strength` (FLOAT). |

**Ограничения:**
- `moodboard_id` должен быть действительным UUID (например, `"123e4567-e89b-12d3-a456-426614174000"`). Скопируйте его с сайта Krea.
- `style_reference` принимает максимум 10 референсов стиля изображения.
- `prompt` должен содержать не менее 1 символа.

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `image` | IMAGE | Сгенерированное изображение в виде тензора. |
