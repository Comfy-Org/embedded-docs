## نمای کلی

گره Krea 2 Image با استفاده از مدل هوش مصنوعی Krea 2 تصاویر را تولید می‌کند. این گره از دو نوع مدل پشتیبانی می‌کند: Medium برای تصاویر گویا و رسا و Large برای فوتورئالیسم گویا. می‌توانید به‌صورت اختیاری یک moodboard و حداکثر ۱۰ مرجع سبک تصویر برای تأثیرگذاری بر تصویر تولیدشده اضافه کنید.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|---------|----------|--------|--------|---------|
| `prompt` | STRING | بله | نامشخص | متن راهنمای تصویر. |
| `model` | DICT | بله | زیر را ببینید | Krea 2 Medium برای تصاویر گویا و رسا مناسب‌ترین است؛ Krea 2 Large برای فوتورئالیسم گویا بهترین گزینه است. |
| `seed` | INT | بله | ۰ تا ۲۱۴۷۴۸۳۶۴۷ | دانه تصادفی برای بازتولیدپذیری (پیش‌فرض: ۰). |

پارامتر `model` یک دیکشنری با زیرپارامترهای زیر است:

| زیرپارامتر | نوع داده | الزامی | محدوده | توضیحات |
|------------|----------|--------|--------|---------|
| `model` | STRING | بله | `"krea 2 medium"`<br>`"krea 2 large"` | نوع مدل Krea 2 را انتخاب می‌کند. |
| `aspect_ratio` | STRING | بله | نامشخص | نسبت ابعاد تصویر تولیدشده. |
| `resolution` | STRING | بله | نامشخص | وضوح تصویر تولیدشده. |
| `creativity` | FLOAT | بله | نامشخص | سطح خلاقیت تولید را کنترل می‌کند. |
| `moodboard_id` | STRING | خیر | نامشخص | UUID یک moodboard از Krea برای تأثیرگذاری بر تصویر. باید یک UUID معتبر باشد. |
| `moodboard_strength` | FLOAT | خیر | نامشخص | قدرت تأثیر moodboard (پیش‌فرض: ۰.۳۵). |
| `style_reference` | LIST | خیر | ۰ تا ۱۰ آیتم | فهرستی از مراجع سبک تصویر. هر مرجع باید دارای `url` (STRING) و `strength` (FLOAT) باشد. |

**محدودیت‌ها:**
- `moodboard_id` باید یک UUID معتبر باشد (مثلاً `"123e4567-e89b-12d3-a456-426614174000"`). آن را از وب‌سایت Krea کپی کنید.
- `style_reference` حداکثر ۱۰ مرجع سبک تصویر را می‌پذیرد.
- `prompt` باید حداقل ۱ کاراکتر داشته باشد.
