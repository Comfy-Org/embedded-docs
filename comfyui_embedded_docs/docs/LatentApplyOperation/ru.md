Вот перевод документации на русский язык:

Узел LatentApplyOperation применяет указанную операцию к латентным сэмплам. Он принимает латентные данные и операцию в качестве входных данных, обрабатывает латентные сэмплы с помощью заданной операции и возвращает изменённые латентные данные. Этот узел позволяет преобразовывать или манипулировать латентными представлениями в вашем рабочем процессе.

## Входные данные

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `образцы` | LATENT | Да | - | Латентные сэмплы для обработки операцией |
| `операция` | LATENT_OPERATION | Да | - | Операция, применяемая к латентным сэмплам |

## Выходные данные

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `output` | LATENT | Изменённые латентные сэмплы после применения операции |
