Вот перевод документации на русский язык в соответствии с вашими правилами:

Узел LoadImageSetNode загружает несколько изображений из входной директории для пакетной обработки и целей обучения. Он поддерживает различные форматы изображений и может опционально изменять их размер с помощью разных методов. Этот узел обрабатывает все выбранные изображения как пакет и возвращает их в виде единого тензора.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `images` | IMAGE | Да | Несколько файлов изображений | Выберите несколько изображений из входной директории. Поддерживаются форматы PNG, JPG, JPEG, WEBP, BMP, GIF, JPE, APNG, TIF и TIFF. Позволяет пакетный выбор изображений. |
| `resize_method` | STRING | Нет | "None"<br>"Stretch"<br>"Crop"<br>"Pad" | Необязательный метод изменения размера загруженных изображений (по умолчанию: "None"). Выберите "None" для сохранения исходных размеров, "Stretch" для принудительного изменения размера, "Crop" для сохранения соотношения сторон путем обрезки или "Pad" для сохранения соотношения сторон путем добавления полей. |

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `IMAGE` | IMAGE | Тензор, содержащий все загруженные изображения в виде пакета для дальнейшей обработки. |
