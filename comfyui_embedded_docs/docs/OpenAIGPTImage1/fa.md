### نمای کلی

تصاویر را به صورت همزمان از طریق نقطه پایانی GPT Image شرکت OpenAI تولید می‌کند. این گره می‌تواند تصاویر جدیدی از روی توضیحات متنی ایجاد کند یا تصاویر موجود را با دریافت یک تصویر ورودی و ماسک اختیاری ویرایش نماید. از چندین مدل GPT Image از جمله gpt-image-1، gpt-image-1.5 و gpt-image-2 پشتیبانی می‌کند.

## ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|---------|----------|-------|--------|---------|
| `prompt` | STRING | بله | - | توضیحات متنی برای GPT Image (پیش‌فرض: "") |
| `seed` | INT | خیر | 0 تا 2147483647 | دانه تصادفی برای تولید (پیش‌فرض: 0) - هنوز در بک‌اند پیاده‌سازی نشده است |
| `quality` | COMBO | خیر | "low"<br>"medium"<br>"high" | کیفیت تصویر، بر هزینه و زمان تولید تأثیر می‌گذارد (پیش‌فرض: "low") |
| `background` | COMBO | خیر | "auto"<br>"opaque"<br>"transparent" | بازگرداندن تصویر با یا بدون پس‌زمینه (پیش‌فرض: "auto") |
| `size` | COMBO | خیر | "auto"<br>"1024x1024"<br>"1024x1536"<br>"1536x1024"<br>"2048x2048"<br>"2048x1152"<br>"1152x2048"<br>"3840x2160"<br>"2160x3840"<br>"Custom" | اندازه تصویر. برای استفاده از عرض و ارتفاع سفارشی گزینه "Custom" را انتخاب کنید (فقط GPT Image 2) (پیش‌فرض: "auto") |
| `n` | INT | خیر | 1 تا 8 | تعداد تصاویر تولیدی (پیش‌فرض: 1) |
| `image` | IMAGE | خیر | - | تصویر مرجع اختیاری برای ویرایش تصویر |
| `mask` | MASK | خیر | - | ماسک اختیاری برای نقاشی درون‌تصویری (نواحی سفید جایگزین می‌شوند) |
| `model` | COMBO | خیر | "gpt-image-1"<br>"gpt-image-1.5"<br>"gpt-image-2" | مدل GPT Image مورد استفاده (پیش‌فرض: "gpt-image-2") |
| `custom_width` | INT | خیر | 1024 تا 3840 | فقط زمانی استفاده می‌شود که `size` برابر "Custom" باشد. باید مضربی از 16 باشد (فقط GPT Image 2) (پیش‌فرض: 1024) |
| `custom_height` | INT | خیر | 1024 تا 3840 | فقط زمانی استفاده می‌شود که `size` برابر "Custom" باشد. باید مضربی از 16 باشد (فقط GPT Image 2) (پیش‌فرض: 1024) |

**محدودیت‌های پارامتر:**

- هنگامی که `image` ارائه می‌شود، گره به حالت ویرایش تصویر تغییر می‌کند
- `mask` فقط زمانی قابل استفاده است که `image` ارائه شده باشد
- هنگام استفاده از `mask`، فقط تصاویر تکی پشتیبانی می‌شوند (اندازه دسته باید 1 باشد)
- `mask` و `image` باید اندازه یکسانی داشته باشند
- وضوح سفارشی (`size` = "Custom") فقط توسط مدل gpt-image-2 پشتیبانی می‌شود
- عرض و ارتفاع سفارشی باید مضرب 16 باشند
- نسبت ابعاد وضوح سفارشی نباید از 3:1 تجاوز کند
- مجموع پیکسل‌های وضوح سفارشی باید بین 655,360 و 8,294,400 باشد
- پس‌زمینه شفاف برای مدل gpt-image-2 پشتیبانی نمی‌شود
- اندازه‌های بزرگتر از 1536x1024 (مانند 2048x2048، 3840x2160) فقط توسط مدل gpt-image-2 پشتیبانی می‌شوند
