عقدة WanFunInpaintToVideo تنشئ تسلسلات فيديو عن طريق الترميم (inpainting) بين صور البداية والنهاية. تستخدم العقدة توجيهًا إيجابيًا وسلبيًا مع صور إطارات اختيارية لتوليد كمونات الفيديو (video latents). تدير العقدة عملية توليد الفيديو بمعاملات أبعاد وطول قابلة للتكوين.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|-----------|-----------|----------|-------|-------------|
| `positive` | CONDITIONING | نعم | - | توجيهات إيجابية لتوليد الفيديو |
| `negative` | CONDITIONING | نعم | - | توجيهات سلبية لتجنبها في توليد الفيديو |
| `vae` | VAE | نعم | - | نموذج VAE لعمليات الترميز/فك الترميز |
| `width` | INT | نعم | 16 إلى MAX_RESOLUTION | عرض فيديو الإخراج بالبكسل (الافتراضي: 832، الخطوة: 16) |
| `height` | INT | نعم | 16 إلى MAX_RESOLUTION | ارتفاع فيديو الإخراج بالبكسل (الافتراضي: 480، الخطوة: 16) |
| `length` | INT | نعم | 1 إلى MAX_RESOLUTION | عدد الإطارات في تسلسل الفيديو (الافتراضي: 81، الخطوة: 4) |
| `batch_size` | INT | نعم | 1 إلى 4096 | عدد مقاطع الفيديو المراد توليدها في الدفعة الواحدة (الافتراضي: 1) |
| `clip_vision_output` | CLIP_VISION_OUTPUT | لا | - | مخرجات رؤية CLIP اختيارية لتوجيه إضافي |
| `start_image` | IMAGE | لا | - | صورة إطار البداية اختيارية لتوليد الفيديو |
| `end_image` | IMAGE | لا | - | صورة إطار النهاية اختيارية لتوليد الفيديو |

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|-------------|-----------|-------------|
| `positive` | CONDITIONING | مخرجات التوجيه الإيجابي المعالجة |
| `negative` | CONDITIONING | مخرجات التوجيه السلبي المعالجة |
| `latent` | LATENT | تمثيل الكمون (latent) للفيديو المُولَّد |
