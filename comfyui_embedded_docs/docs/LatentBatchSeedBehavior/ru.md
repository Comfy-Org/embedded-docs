Узел `LatentBatchSeedBehavior` предназначен для изменения поведения сида (seed) в пакете латентных образцов. Он позволяет либо рандомизировать, либо фиксировать сид в рамках пакета, тем самым влияя на процесс генерации: либо внося вариативность, либо обеспечивая единообразие в создаваемых результатах.

## Входные параметры

| Параметр        | Тип данных | Описание |
|-----------------|------------|----------|
| `samples`       | `LATENT`   | Параметр 'samples' представляет собой пакет латентных образцов для обработки. Его модификация зависит от выбранного поведения сида, что влияет на единообразие или вариативность создаваемых результатов. |
| `seed_behavior` | COMBO[STRING] | Параметр 'seed_behavior' определяет, должен ли сид для пакета латентных образцов быть рандомизированным или фиксированным. Этот выбор существенно влияет на процесс генерации, либо внося вариативность, либо обеспечивая единообразие в рамках пакета. |

## Выходные параметры

| Параметр | Тип данных | Описание |
|----------|------------|----------|
| `latent` | `LATENT`   | На выходе получается модифицированная версия входных латентных образцов с корректировками, выполненными на основе заданного поведения сида. Выходные данные либо сохраняют, либо изменяют индекс пакета, чтобы отразить выбранное поведение сида. |
