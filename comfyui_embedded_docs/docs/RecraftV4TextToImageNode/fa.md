این گره با استفاده از مدل‌های هوش مصنوعی Recraft V4 یا V4 Pro، تصاویری را از توضیحات متنی تولید می‌کند. درخواست شما را به یک API خارجی ارسال کرده و تصاویر تولیدشده را بازمی‌گرداند. می‌توانید با تعیین مدل، اندازه تصویر و تعداد تصاویر، خروجی را کنترل کنید.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|---------|----------|--------|--------|---------|
| `prompt` | STRING | بله | نامشخص | متن راهنما برای تولید تصویر. حداکثر ۱۰,۰۰۰ کاراکتر. |
| `negative_prompt` | STRING | خیر | نامشخص | توضیحات متنی اختیاری از عناصر نامطلوب در تصویر. |
| `model` | COMBO | بله | `"recraftv4"`<br>`"recraftv4_pro"` | مدل مورد استفاده برای تولید. انتخاب مدل، اندازه‌های تصویر در دسترس را تعیین می‌کند. |
| `size` | COMBO | بله | بسته به مدل متفاوت است | اندازه تصویر تولیدشده. گزینه‌های موجود به مدل انتخاب‌شده بستگی دارد. برای `recraftv4`، پیش‌فرض "1024x1024" است. برای `recraftv4_pro`، پیش‌فرض "2048x2048" است. |
| `n` | INT | بله | ۱ تا ۶ | تعداد تصاویر برای تولید (پیش‌فرض: ۱). |
| `seed` | INT | بله | ۰ تا ۱۸۴۴۶۷۴۴۰۷۳۷۰۹۵۵۱۶۱۵ | دانه (seed) برای تعیین اینکه گره دوباره اجرا شود؛ نتایج واقعی بدون توجه به دانه، غیرقطعی هستند (پیش‌فرض: ۰). |
| `recraft_controls` | CUSTOM | خیر | نامشخص | کنترل‌های اضافی اختیاری بر تولید از طریق گره Recraft Controls. |

**توجه:** پارامتر `size` یک ورودی پویا است که گزینه‌های موجود آن بر اساس `model` انتخاب‌شده تغییر می‌کند. مقدار `seed` خروجی‌های تصویر قابل تکرار را تضمین نمی‌کند.

|-----------|----------|---------|
| `output` | IMAGE | تصویر یا دسته‌ای از تصاویر تولیدشده. |
