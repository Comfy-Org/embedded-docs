Вот перевод документации на русский язык:

Этот узел генерирует изображения на основе текстовых описаний с использованием моделей ИИ Recraft V4 или V4 Pro. Он отправляет ваш запрос во внешний API и возвращает сгенерированные изображения. Вы можете управлять результатом, указывая модель, размер изображения и количество создаваемых изображений.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `prompt` | STRING | Да | Н/Д | Запрос для генерации изображения. Максимум 10 000 символов. |
| `negative_prompt` | STRING | Нет | Н/Д | Необязательное текстовое описание нежелательных элементов на изображении. |
| `model` | COMBO | Да | `"recraftv4"`<br>`"recraftv4_pro"` | Модель для генерации. Выбор модели определяет доступные размеры изображений. |
| `size` | COMBO | Да | Зависит от модели | Размер генерируемого изображения. Доступные варианты зависят от выбранной модели. Для `recraftv4` значение по умолчанию — "1024x1024". Для `recraftv4_pro` значение по умолчанию — "2048x2048". |
| `n` | INT | Да | от 1 до 6 | Количество генерируемых изображений (по умолчанию: 1). |
| `seed` | INT | Да | от 0 до 18446744073709551615 | Значение seed для определения необходимости повторного запуска узла; фактические результаты недетерминированы независимо от seed (по умолчанию: 0). |
| `recraft_controls` | CUSTOM | Нет | Н/Д | Необязательные дополнительные элементы управления генерацией через узел Recraft Controls. |

**Примечание:** Параметр `size` является динамическим входным параметром, доступные варианты которого меняются в зависимости от выбранной `model`. Значение `seed` не гарантирует воспроизводимость результатов изображений.

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `output` | IMAGE | Сгенерированное изображение или пакет изображений. |
