Узел UNetCrossAttentionMultiply применяет коэффициенты умножения к механизму перекрестного внимания в модели UNet. Он позволяет масштабировать компоненты запроса, ключа, значения и вывода слоев перекрестного внимания для экспериментов с различными режимами и эффектами внимания.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `model` | MODEL | Да | - | Модель UNet для модификации коэффициентами масштабирования внимания |
| `q` | FLOAT | Нет | 0.0 - 10.0 | Коэффициент масштабирования для компонентов запроса в перекрестном внимании (по умолчанию: 1.0) |
| `k` | FLOAT | Нет | 0.0 - 10.0 | Коэффициент масштабирования для компонентов ключа в перекрестном внимании (по умолчанию: 1.0) |
| `v` | FLOAT | Нет | 0.0 - 10.0 | Коэффициент масштабирования для компонентов значения в перекрестном внимании (по умолчанию: 1.0) |
| `out` | FLOAT | Нет | 0.0 - 10.0 | Коэффициент масштабирования для компонентов вывода в перекрестном внимании (по умолчанию: 1.0) |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `model` | MODEL | Модифицированная модель UNet с масштабированными компонентами перекрестного внимания |
