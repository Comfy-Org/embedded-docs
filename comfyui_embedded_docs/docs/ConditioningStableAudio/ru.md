Узел ConditioningStableAudio добавляет информацию о времени как к положительным, так и к отрицательным входам conditioning для генерации аудио. Он задает параметры времени начала и общей длительности, которые помогают управлять тем, когда и как долго должен генерироваться аудиоконтент. Этот узел изменяет существующие данные conditioning, добавляя метаданные времени, специфичные для аудио.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `positive` | CONDITIONING | Да | - | Положительный вход conditioning, который будет изменен с добавлением информации о времени аудио |
| `negative` | CONDITIONING | Да | - | Отрицательный вход conditioning, который будет изменен с добавлением информации о времени аудио |
| `seconds_start` | FLOAT | Да | от 0.0 до 1000.0 | Время начала в секундах для генерации аудио (по умолчанию: 0.0) |
| `seconds_total` | FLOAT | Да | от 0.0 до 1000.0 | Общая длительность в секундах для генерации аудио (по умолчанию: 47.0) |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `positive` | CONDITIONING | Измененный положительный conditioning с примененной информацией о времени аудио |
| `negative` | CONDITIONING | Измененный отрицательный conditioning с примененной информацией о времени аудио |
