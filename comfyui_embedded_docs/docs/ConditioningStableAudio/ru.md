Узел ConditioningStableAudio добавляет информацию о времени как к позитивному, так и к негативному условиям для генерации аудио. Он устанавливает параметры времени начала и общей продолжительности, которые помогают управлять тем, когда и как долго должно генерироваться аудиосодержимое. Этот узел модифицирует существующие данные условий, добавляя к ним метаданные с временными характеристиками, специфичными для аудио.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `positive` | CONDITIONING | Да | - | Позитивное условие, которое будет модифицировано с добавлением информации о времени для аудио |
| `negative` | CONDITIONING | Да | - | Негативное условие, которое будет модифицировано с добавлением информации о времени для аудио |
| `seconds_start` | FLOAT | Да | 0.0 до 1000.0 | Время начала в секундах для генерации аудио (по умолчанию: 0.0) |
| `seconds_total` | FLOAT | Да | 0.0 до 1000.0 | Общая продолжительность в секундах для генерации аудио (по умолчанию: 47.0) |

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `positive` | CONDITIONING | Модифицированное позитивное условие с примененной информацией о времени для аудио |
| `negative` | CONDITIONING | Модифицированное негативное условие с примененной информацией о времени для аудио |
