Вот перевод документации на русский язык:

Узел USOStyleReference применяет патчи стилевой привязки к модели, используя закодированные признаки изображения из выходных данных CLIP vision. Он создает модифицированную версию входной модели, внедряя информацию о стиле, извлеченную из визуальных входных данных, что обеспечивает возможность переноса стиля или генерации на основе референса.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `model` | MODEL | Да | - | Базовая модель, к которой применяется патч стилевой привязки |
| `model_patch` | MODEL_PATCH | Да | - | Патч модели, содержащий информацию о стилевой привязке |
| `clip_vision_output` | CLIP_VISION_OUTPUT | Да | - | Закодированные визуальные признаки, извлеченные в результате обработки CLIP vision |

## Выходные параметры

| Имя выходного параметра | Тип данных | Описание |
|-------------|-----------|-------------|
| `model` | MODEL | Модифицированная модель с примененными патчами стилевой привязки |
