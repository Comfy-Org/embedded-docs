### نمای کلی

تصاویر را به صورت همزمان از طریق نقطه پایانی DALL·E 3 شرکت OpenAI تولید می‌کند. این گره یک متن ورودی (پرامپت) دریافت کرده و با استفاده از مدل DALL·E 3 شرکت OpenAI تصاویر متناظر را ایجاد می‌کند. شما می‌توانید کیفیت، سبک و ابعاد تصویر را مشخص کنید.

### ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `prompt` | STRING | بله | - | متن ورودی برای DALL·E (پیش‌فرض: "") |
| `seed` | INT | خیر | 0 تا 2147483647 | هنوز در بک‌اند پیاده‌سازی نشده است (پیش‌فرض: 0) |
| `quality` | COMBO | خیر | "standard"<br>"hd" | کیفیت تصویر (پیش‌فرض: "standard") |
| `style` | COMBO | خیر | "natural"<br>"vivid" | حالت Vivid باعث می‌شود مدل به سمت تولید تصاویر فوق‌واقع‌گرایانه و دراماتیک متمایل شود. حالت Natural باعث می‌شود مدل تصاویر طبیعی‌تر و با ظاهری کمتر فوق‌واقع‌گرایانه تولید کند. (پیش‌فرض: "natural") |
| `size` | COMBO | خیر | "1024x1024"<br>"1024x1792"<br>"1792x1024" | اندازه تصویر (پیش‌فرض: "1024x1024") |
