گره PixVerse Template به شما امکان می‌دهد از میان قالب‌های موجود برای تولید ویدیوی PixVerse انتخاب کنید. این گره نام قالب انتخاب‌شده را به شناسه قالب متناظر که API PixVerse برای ساخت ویدیو نیاز دارد، تبدیل می‌کند.

## ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|---------|----------|-------|--------|---------|
| `template` | STRING | بله | چندین گزینه موجود | قالبی که برای تولید ویدیوی PixVerse استفاده می‌شود. گزینه‌های موجود با قالب‌های از پیش تعریف‌شده در سیستم PixVerse مطابقت دارند. |

## خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-----------|----------|---------|
| `pixverse_template` | STRING | شناسه قالبی که با نام قالب انتخاب‌شده مطابقت دارد و می‌تواند توسط سایر گره‌های PixVerse برای تولید ویدیو استفاده شود. |
