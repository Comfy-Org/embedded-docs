Вот перевод документации на русский язык:

Узел PixVerse Template позволяет выбрать один из доступных шаблонов для генерации видео PixVerse. Он преобразует выбранное название шаблона в соответствующий идентификатор шаблона, который требуется API PixVerse для создания видео.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `шаблон` | STRING | Да | Доступно несколько опций | Шаблон, используемый для генерации видео PixVerse. Доступные опции соответствуют предопределенным шаблонам в системе PixVerse. |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `pixverse_template` | STRING | Идентификатор шаблона, соответствующий выбранному названию шаблона, который может использоваться другими узлами PixVerse для генерации видео. |
