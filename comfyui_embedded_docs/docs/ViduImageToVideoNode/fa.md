گره تولید ویدیو از تصویر Vidu، یک ویدیوی کوتاه از یک تصویر شروع و یک توضیح متنی اختیاری ایجاد می‌کند. این گره از یک مدل هوش مصنوعی برای تولید محتوای ویدیویی استفاده می‌کند که از فریم تصویر ارائه‌شده ادامه می‌یابد و ویدیوی حاصل را بازمی‌گرداند.

## ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `model` | COMBO | بله | `viduq1` | نام مدل (پیش‌فرض: viduq1) |
| `image` | IMAGE | بله | - | تصویری که به عنوان فریم شروع ویدیوی تولیدشده استفاده می‌شود |
| `prompt` | STRING | خیر | - | توضیح متنی برای تولید ویدیو (پیش‌فرض: خالی) |
| `duration` | INT | خیر | 5-5 | مدت زمان ویدیوی خروجی بر حسب ثانیه (پیش‌فرض: 5، ثابت در ۵ ثانیه) |
| `seed` | INT | خیر | 0-2147483647 | دانه (seed) برای تولید ویدیو (0 برای تصادفی) (پیش‌فرض: 0) |
| `resolution` | COMBO | خیر | `1080p` | مقادیر پشتیبانی‌شده ممکن است بسته به مدل و مدت زمان متفاوت باشد (پیش‌فرض: 1080p) |
| `movement_amplitude` | COMBO | خیر | `auto`<br>`small`<br>`medium`<br>`large` | دامنه حرکت اشیاء در قاب (پیش‌فرض: auto) |

**محدودیت‌ها:**

- فقط یک تصویر ورودی مجاز است (نمی‌توان چند تصویر را پردازش کرد).
- تصویر ورودی باید نسبت تصویری بین 1:4 و 4:1 داشته باشد.
