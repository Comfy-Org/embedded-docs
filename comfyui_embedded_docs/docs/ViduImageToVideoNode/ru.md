Вот перевод документации узла ComfyUI на русский язык:

Узел Vidu Image To Video Generation создает короткое видео из начального изображения и необязательного текстового описания. Он использует модель ИИ для генерации видеоконтента, продолжающегося от предоставленного кадра изображения, и возвращает результирующее видео.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `model` | COMBO | Да | `viduq1` | Название модели (по умолчанию: viduq1) |
| `image` | IMAGE | Да | - | Изображение, используемое в качестве начального кадра генерируемого видео |
| `prompt` | STRING | Нет | - | Текстовое описание для генерации видео (по умолчанию: пусто) |
| `duration` | INT | Нет | 5-5 | Длительность выходного видео в секундах (по умолчанию: 5, фиксированное значение 5 секунд) |
| `seed` | INT | Нет | 0-2147483647 | Зерно для генерации видео (0 для случайного) (по умолчанию: 0) |
| `resolution` | COMBO | Нет | `1080p` | Поддерживаемые значения могут различаться в зависимости от модели и длительности (по умолчанию: 1080p) |
| `movement_amplitude` | COMBO | Нет | `auto`<br>`small`<br>`medium`<br>`large` | Амплитуда движения объектов в кадре (по умолчанию: auto) |

**Ограничения:**

- Допускается только одно входное изображение (обработка нескольких изображений невозможна).
- Входное изображение должно иметь соотношение сторон от 1:4 до 4:1.

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `output` | VIDEO | Сгенерированное выходное видео |
