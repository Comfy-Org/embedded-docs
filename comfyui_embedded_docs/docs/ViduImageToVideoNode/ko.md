Vidu 이미지-투-비디오 생성 노드는 시작 이미지와 선택적 텍스트 설명을 사용하여 짧은 비디오를 생성합니다. 제공된 이미지 프레임에서 이어지는 비디오 콘텐츠를 AI 모델로 생성하며, 결과 비디오를 반환합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `model` | COMBO | 예 | `viduq1` | 모델 이름 (기본값: viduq1) |
| `image` | IMAGE | 예 | - | 생성된 비디오의 시작 프레임으로 사용할 이미지 |
| `prompt` | STRING | 아니요 | - | 비디오 생성을 위한 텍스트 설명 (기본값: 비어 있음) |
| `duration` | INT | 아니요 | 5-5 | 출력 비디오의 길이(초) (기본값: 5, 5초로 고정) |
| `seed` | INT | 아니요 | 0-2147483647 | 비디오 생성을 위한 시드 (0은 무작위) (기본값: 0) |
| `resolution` | COMBO | 아니요 | `1080p` | 지원되는 값은 모델 및 길이에 따라 다를 수 있음 (기본값: 1080p) |
| `movement_amplitude` | COMBO | 아니요 | `auto`<br>`small`<br>`medium`<br>`large` | 프레임 내 객체의 움직임 진폭 (기본값: auto) |

**제약 사항:**

- 하나의 입력 이미지만 허용됩니다(여러 이미지 처리 불가).
- 입력 이미지의 가로세로 비율은 1:4에서 4:1 사이여야 합니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `output` | VIDEO | 생성된 비디오 출력 |
