Вот перевод документации на русский язык:

Узел Vidu Reference Video генерирует видео на основе нескольких эталонных изображений и текстового запроса. Он использует модели ИИ для создания согласованного видеоконтента на основе предоставленных изображений и описания. Узел поддерживает различные настройки видео, включая длительность, соотношение сторон, разрешение и управление движением.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `model` | COMBO | Да | `"viduq1"` | Название модели для генерации видео (по умолчанию: "viduq1") |
| `images` | IMAGE | Да | - | Изображения, используемые в качестве эталонов для генерации видео с согласованными объектами (максимум 7 изображений) |
| `prompt` | STRING | Да | - | Текстовое описание для генерации видео |
| `длительность` | INT | Нет | 5-5 | Длительность выходного видео в секундах (по умолчанию: 5) |
| `сид` | INT | Нет | 0-2147483647 | Зерно для генерации видео (0 для случайного) (по умолчанию: 0) |
| `соотношение сторон` | COMBO | Нет | `"16:9"`<br>`"9:16"`<br>`"1:1"` | Соотношение сторон выходного видео (по умолчанию: "16:9") |
| `разрешение` | COMBO | Нет | `"1080p"` | Поддерживаемые значения могут различаться в зависимости от модели и длительности (по умолчанию: "1080p") |
| `амплитуда движения` | COMBO | Нет | `"auto"`<br>`"small"`<br>`"medium"`<br>`"large"` | Амплитуда движения объектов в кадре (по умолчанию: "auto") |

**Ограничения:**

- Поле `prompt` является обязательным и не может быть пустым
- Допускается не более 7 эталонных изображений
- Каждое изображение должно иметь соотношение сторон от 1:4 до 4:1
- Каждое изображение должно иметь минимальные размеры 128x128 пикселей
- Длительность фиксирована и составляет 5 секунд

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `output` | VIDEO | Сгенерированное видео на основе эталонных изображений и запроса |
