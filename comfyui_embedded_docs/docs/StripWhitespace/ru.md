Вот перевод документации на русский язык:

Этот узел удаляет все лишние пробелы, табуляции и символы новой строки в начале и конце текстовой строки. Он принимает текстовый ввод и возвращает очищенную версию с обрезанными начальными и конечными пробельными символами.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `text` | STRING | Да | Н/Д | Текстовая строка, из которой необходимо удалить начальные и конечные пробельные символы. |

## Выходные данные

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `text` | STRING | Обработанный текст, из которого удалены все начальные и конечные пробельные символы. |
