### مرور کلی

این گره یک تصویر و مقدار وزن را برای استفاده با گره **Luma Generate Image** نگهداری می‌کند. یک زنجیره مرجع ایجاد می‌کند که می‌تواند به سایر گره‌های Luma ارسال شود تا بر تولید تصویر تأثیر بگذارد. این گره می‌تواند یک زنجیره مرجع جدید شروع کند یا به یک زنجیره موجود اضافه شود.

### ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|---------|----------|-------|--------|---------|
| `image` | IMAGE | بله | - | تصویری که به عنوان مرجع استفاده می‌شود. |
| `weight` | FLOAT | بله | 0.0 - 1.0 | وزن مرجع تصویر (پیش‌فرض: 1.0). |
| `luma_ref` | LUMA_REF | خیر | - | زنجیره مرجع Luma موجود برای افزودن به آن (اختیاری). |

### خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-----------|----------|---------|
