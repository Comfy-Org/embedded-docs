گره ViduExtendVideoNode فریم‌های جدیدی تولید می‌کند تا طول یک ویدیوی موجود را افزایش دهد. این گره از یک مدل هوش مصنوعی مشخص برای ایجاد ادامه‌ای یکپارچه بر اساس ویدیوی منبع و یک提示 متنی اختیاری استفاده می‌کند.

## ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `مدل` | COMBO | بله | `"viduq2-pro"`<br>`"viduq2-turbo"` | مدل هوش مصنوعی مورد استفاده برای افزایش طول ویدیو. انتخاب یک مدل، تنظیمات خاص مدت‌زمان و وضوح آن را نمایش می‌دهد. |
| `model.duration` | INT | بله | 1 تا 7 | مدت‌زمان ویدیوی افزایش‌یافته بر حسب ثانیه (پیش‌فرض: 4). این تنظیم پس از انتخاب یک مدل ظاهر می‌شود. |
| `model.resolution` | COMBO | بله | `"720p"`<br>`"1080p"` | وضوح ویدیوی خروجی. این تنظیم پس از انتخاب یک مدل ظاهر می‌شود. |
| `ویدیو` | VIDEO | بله | - | ویدیوی منبع برای افزایش طول. |
| `پرامپت` | STRING | خیر | - | یک提示 متنی اختیاری برای هدایت محتوای ویدیوی افزایش‌یافته (حداکثر 2000 کاراکتر، پیش‌فرض: خالی). |
| `seed` | INT | خیر | 0 تا 2147483647 | یک مقدار seed برای کنترل تصادفی‌بودن تولید (پیش‌فرض: 1). |
| `فریم پایانی` | IMAGE | خیر | - | یک تصویر اختیاری برای استفاده به‌عنوان فریم انتهایی هدف برای افزایش طول. در صورت ارائه، نسبت تصویر آن باید بین 1:4 و 4:1 باشد و ابعاد آن حداقل 128×128 پیکسل باشد. |

**توجه:** ویدیوی منبع `video` باید مدت‌زمانی بین 4 تا 55 ثانیه داشته باشد.
