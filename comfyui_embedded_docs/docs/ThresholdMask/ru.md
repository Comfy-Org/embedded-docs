Узел ThresholdMask преобразует маску в бинарную маску путем применения порогового значения. Он сравнивает каждый пиксель входной маски с заданным пороговым значением и создает новую маску, где пиксели выше порога становятся 1 (белыми), а пиксели ниже или равные порогу становятся 0 (черными).

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `mask` | MASK | Да | - | Входная маска для обработки |
| `value` | FLOAT | Да | 0.0 - 1.0 | Пороговое значение для бинаризации (по умолчанию: 0.5) |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `mask` | MASK | Результирующая бинарная маска после пороговой обработки |
