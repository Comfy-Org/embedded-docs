FreSca 節點在取樣過程中對引導訊號應用頻率相關的縮放。它透過傅立葉濾波將引導訊號分離為低頻和高頻成分，然後對每個頻率範圍應用不同的縮放因子，最後再重新組合。這使得能夠更細緻地控制引導訊號對生成輸出不同面向的影響。
