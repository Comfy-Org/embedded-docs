FreSca 節點在取樣過程中對引導信號施加頻率相關的縮放處理。它透過傅立葉濾波將引導信號分離為低頻和高頻分量，然後對每個頻率範圍應用不同的縮放因子，最後重新組合它們。這樣可以更細緻地控制引導信號對生成輸出的不同層面所產生的影響。

## 輸入參數

| 參數名稱 | 資料類型 | 是否必填 | 數值範圍 | 參數說明 |
|-----------|-----------|----------|-------|-------------|
| `model` | MODEL | 是 | - | 要施加頻率縮放處理的模型 |
| `scale_low` | FLOAT | 否 | 0-10 | 低頻分量的縮放因子（預設值：1.0） |
| `scale_high` | FLOAT | 否 | 0-10 | 高頻分量的縮放因子（預設值：1.25） |
| `freq_cutoff` | INT | 否 | 1-10000 | 圍繞中心點被視為低頻的頻率索引數量（預設值：20） |
