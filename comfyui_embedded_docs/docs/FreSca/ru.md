Вот перевод документации на русский язык:

Узел FreSca применяет частотно-зависимое масштабирование к направляющему сигналу (guidance) в процессе семплирования. Он разделяет направляющий сигнал на низкочастотные и высокочастотные компоненты с помощью фильтрации Фурье, затем применяет различные коэффициенты масштабирования к каждому частотному диапазону перед их повторным объединением. Это позволяет более тонко контролировать влияние направляющего сигнала на различные аспекты генерируемого результата.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `model` | MODEL | Да | - | Модель, к которой применяется частотное масштабирование |
| `scale_low` | FLOAT | Нет | 0 - 10 | Коэффициент масштабирования для низкочастотных компонентов (по умолчанию: 1.0) |
| `scale_high` | FLOAT | Нет | 0 - 10 | Коэффициент масштабирования для высокочастотных компонентов (по умолчанию: 1.25) |
| `freq_cutoff` | INT | Нет | 1 - 10000 | Количество частотных индексов вокруг центра, считающихся низкочастотными (по умолчанию: 20) |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `model` | MODEL | Модифицированная модель с частотно-зависимым масштабированием, применённым к её функции направляющего сигнала |
