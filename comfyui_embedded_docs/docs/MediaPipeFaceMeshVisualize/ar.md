نظرة عامة

يرسم نقاط معالم الوجه وخطوط الاتصال (شبكة الوجه) فوق الصورة المدخلة. تستخدم هذه العقدة بيانات المعالم التي تنتجها عقدة كشف الوجه لتصور السمات الوجهية المكتشفة، مثل العينين والأنف والفم ومحيط الوجه.

المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|-----------|-----------|----------|-------|-------------|
| `face_landmarks` | FACE_LANDMARKS | نعم | | بيانات معالم الوجه من عقدة الكشف. |
| `الصورة` | IMAGE | لا | | الصورة التي سيتم رسم الشبكة عليها. إذا لم يتم توصيلها، سيتم استخدام لوحة سوداء بنفس حجم نتيجة الكشف. |
| `connections` | COMBO | نعم | `"all"`<br>`"fill"`<br>`"custom"` | يحدد أي أجزاء من شبكة الوجه سيتم رسمها. `"all"` يرسم الشبكة الكاملة (البيضاوي، العيون، الحواجب، الشفاه، القزحيات، الأنف). `"fill"` يرسم مضلعًا صلبًا لبيضاوي الوجه (قناع الصورة الظلية). `"custom"` يتيح لك تشغيل أو إيقاف كل ميزة على حدة. (الافتراضي: `"all"`) |
| `color` | COLOR | نعم | | لون خطوط الشبكة والنقاط. (الافتراضي: `#00ff00`) |
| `thickness` | INT | نعم | 0 إلى 8 | سُمك خطوط الشبكة بالبكسل. يؤدي تعيين هذا إلى 0 إلى تعطيل رسم الخطوط. (الافتراضي: 1) |
| `point_size` | INT | نعم | 0 إلى 16 | نصف قطر نقاط المعالم بالبكسل. يؤدي تعيين هذا إلى 0 إلى تعطيل رسم النقاط. (الافتراضي: 2) |

**ملاحظة حول معامل `connections`:** عند تحديد `"custom"`، تظهر مدخلات منطقية إضافية لكل ميزة وجهية (مثل `face_oval`، `lips`، `left_eye`، `right_eye`، `left_eyebrow`، `right_eyebrow`، `left_iris`، `right_iris`، `nose`، `tesselation`). سيتم رسم الميزات التي تقوم بتمكينها فقط.
