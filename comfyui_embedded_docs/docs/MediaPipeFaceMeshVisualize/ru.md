## Обзор

Рисует точки лицевых ориентиров и соединительные линии (лицевую сетку) поверх входного изображения. Этот узел использует данные ориентиров, полученные от узла обнаружения лиц, для визуализации обнаруженных черт лица, таких как глаза, нос, рот и контур лица.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `face_landmarks` | FACE_LANDMARKS | Да | | Данные лицевых ориентиров от узла обнаружения. |
| `image` | IMAGE | Нет | | Изображение, на котором будет нарисована сетка. Если не подключено, будет использован черный холст того же размера, что и результат обнаружения. |
| `connections` | COMBO | Да | `"all"`<br>`"fill"`<br>`"custom"` | Определяет, какие части лицевой сетки рисовать. `"all"` рисует полную сетку (овал, глаза, брови, губы, радужки, нос). `"fill"` рисует сплошной многоугольник овала лица (маска силуэта). `"custom"` позволяет включать/отключать каждую черту индивидуально. (по умолчанию: `"all"`) |
| `color` | COLOR | Да | | Цвет линий и точек сетки. (по умолчанию: `#00ff00`) |
| `thickness` | INT | Да | от 0 до 8 | Толщина линий сетки в пикселях. Значение 0 отключает рисование линий. (по умолчанию: 1) |
| `point_size` | INT | Да | от 0 до 16 | Радиус точек ориентиров в пикселях. Значение 0 отключает рисование точек. (по умолчанию: 2) |

**Примечание к параметру `connections`:** При выборе значения `"custom"` появляются дополнительные логические входы для каждой черты лица (например, `face_oval`, `lips`, `left_eye`, `right_eye`, `left_eyebrow`, `right_eyebrow`, `left_iris`, `right_iris`, `nose`, `tesselation`). Будут нарисованы только включенные черты.

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `IMAGE` | IMAGE | Входное изображение с нарисованной на нем сеткой лицевых ориентиров. |
