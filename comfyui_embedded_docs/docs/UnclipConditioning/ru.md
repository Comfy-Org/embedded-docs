Этот узел предназначен для интеграции выходных данных CLIP vision в процесс кондиционирования, регулируя влияние этих выходных данных на основе заданных параметров силы и шумового увеличения. Он обогащает кондиционирование визуальным контекстом, улучшая процесс генерации.

## Входы

| Параметр              | Comfy dtype            | Описание |
|------------------------|------------------------|-------------|
| `conditioning`         | `CONDITIONING`         | Базовые данные кондиционирования, к которым будут добавлены выходные данные CLIP vision, служащие основой для дальнейших модификаций. |
| `clip_vision_output`   | `CLIP_VISION_OUTPUT`   | Выходные данные модели CLIP vision, предоставляющие визуальный контекст, который интегрируется в кондиционирование. |
| `strength`             | `FLOAT`                | Определяет интенсивность влияния выходных данных CLIP vision на кондиционирование. |
| `noise_augmentation`   | `FLOAT`                | Указывает уровень шумового увеличения, применяемого к выходным данным CLIP vision перед их интеграцией в кондиционирование. |

## Выходы

| Параметр             | Comfy dtype            | Описание |
|-----------------------|------------------------|-------------|
| `conditioning`         | `CONDITIONING`         | Обогащенные данные кондиционирования, теперь содержащие интегрированные выходные данные CLIP vision с примененными силой и шумовым увеличением. |
