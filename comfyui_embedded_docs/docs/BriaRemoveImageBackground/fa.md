### مرور کلی

این گره با استفاده از سرویس Bria RMBG 2.0، پس‌زمینه تصویر را حذف می‌کند. تصویر را برای پردازش به یک API خارجی ارسال کرده و نتیجه را با پس‌زمینه حذف‌شده بازمی‌گرداند.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `image` | IMAGE | بله | - | تصویر ورودی که پس‌زمینه آن حذف خواهد شد. |
| `moderation` | COMBO | خیر | `"false"`<br>`"true"` | تنظیمات نظارت محتوا. وقتی روی `"true"` تنظیم شود، گزینه‌های نظارت اضافی در دسترس قرار می‌گیرند. |
| `visual_input_moderation` | BOOLEAN | خیر | - | نظارت محتوای بصری روی تصویر ورودی را فعال می‌کند. این پارامتر فقط زمانی در دسترس است که `moderation` روی `"true"` تنظیم شده باشد. پیش‌فرض: `False`. |
| `visual_output_moderation` | BOOLEAN | خیر | - | نظارت محتوای بصری روی تصویر خروجی را فعال می‌کند. این پارامتر فقط زمانی در دسترس است که `moderation` روی `"true"` تنظیم شده باشد. پیش‌فرض: `True`. |
| `seed` | INT | خیر | 0 تا 2147483647 | یک مقدار دانه (seed) که تعیین می‌کند آیا گره باید دوباره اجرا شود. نتایج بدون توجه به مقدار دانه، غیرقطعی هستند. پیش‌فرض: `0`. |

**توجه:** پارامترهای `visual_input_moderation` و `visual_output_moderation` به پارامتر `moderation` وابسته هستند. این پارامترها فقط در صورتی فعال و الزامی هستند که `moderation` روی `"true"` تنظیم شده باشد.
