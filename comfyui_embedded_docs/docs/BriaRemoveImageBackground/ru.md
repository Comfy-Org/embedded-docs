Этот узел удаляет фон из изображения с помощью сервиса Bria RMBG 2.0. Он отправляет изображение во внешний API для обработки и возвращает результат с удалённым фоном.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `image` | IMAGE | Да | - | Входное изображение, из которого будет удалён фон. |
| `moderation` | COMBO | Нет | `"false"`<br>`"true"` | Настройки модерации. Если установлено значение `"true"`, становятся доступны дополнительные параметры модерации. |
| `visual_input_moderation` | BOOLEAN | Нет | - | Включает модерацию визуального содержимого входного изображения. Этот параметр доступен только при значении `moderation` равном `"true"`. По умолчанию: `False`. |
| `visual_output_moderation` | BOOLEAN | Нет | - | Включает модерацию визуального содержимого выходного изображения. Этот параметр доступен только при значении `moderation` равном `"true"`. По умолчанию: `True`. |
| `seed` | INT | Нет | от 0 до 2147483647 | Значение seed, которое определяет, следует ли повторно запускать узел. Результаты недетерминированы независимо от значения seed. По умолчанию: `0`. |

**Примечание:** Параметры `visual_input_moderation` и `visual_output_moderation` зависят от параметра `moderation`. Они активны и обязательны только в том случае, если `moderation` установлено в `"true"`.

## Выходные данные

| Имя выходного параметра | Тип данных | Описание |
|-------------------------|------------|----------|
| `image` | IMAGE | Обработанное изображение с удалённым фоном. |
