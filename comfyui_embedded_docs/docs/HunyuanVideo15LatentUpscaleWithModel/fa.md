گره **Hunyuan Video 15 Latent Upscale With Model** وضوح نمایش نهفته (latent) یک تصویر را افزایش می‌دهد. این گره ابتدا نمونه‌های نهفته را با استفاده از یک روش درون‌یابی مشخص به اندازه دلخواه بزرگ‌نمایی می‌کند، سپس نتیجه بزرگ‌شده را با استفاده از یک مدل تخصصی بزرگ‌نمایی Hunyuan Video 1.5 برای بهبود کیفیت، پالایش می‌کند.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|---------|----------|--------|--------|---------|
| `model` | LATENT_UPSCALE_MODEL | بله | نامعتبر | مدل بزرگ‌نمایی نهفته Hunyuan Video 1.5 که برای پالایش نمونه‌های بزرگ‌شده استفاده می‌شود. |
| `samples` | LATENT | بله | نامعتبر | نمایش نهفته تصویر که باید بزرگ‌نمایی شود. |
| `upscale_method` | COMBO | خیر | `"nearest-exact"`<br>`"bilinear"`<br>`"area"`<br>`"bicubic"`<br>`"bislerp"` | الگوریتم درون‌یابی مورد استفاده برای مرحله بزرگ‌نمایی اولیه (پیش‌فرض: `"bilinear"`). |
| `width` | INT | خیر | 0 تا 16384 | عرض هدف برای نهفته بزرگ‌شده، بر حسب پیکسل. مقدار 0 باعث محاسبه خودکار عرض بر اساس ارتفاع هدف و نسبت ابعاد اصلی می‌شود. عرض نهایی خروجی مضربی از 16 خواهد بود (پیش‌فرض: 1280). |
| `height` | INT | خیر | 0 تا 16384 | ارتفاع هدف برای نهفته بزرگ‌شده، بر حسب پیکسل. مقدار 0 باعث محاسبه خودکار ارتفاع بر اساس عرض هدف و نسبت ابعاد اصلی می‌شود. ارتفاع نهایی خروجی مضربی از 16 خواهد بود (پیش‌فرض: 720). |
| `crop` | COMBO | خیر | `"disabled"`<br>`"center"` | نحوه برش نهفته بزرگ‌شده برای تطبیق با ابعاد هدف را تعیین می‌کند. |

**نکته در مورد ابعاد:** اگر هر دو `width` و `height` روی 0 تنظیم شوند، گره نمونه‌های ورودی `samples` را بدون تغییر برمی‌گرداند. اگر تنها یکی از ابعاد روی 0 تنظیم شود، بعد دیگر برای حفظ نسبت ابعاد اصلی محاسبه می‌شود. ابعاد نهایی همیشه طوری تنظیم می‌شوند که حداقل 64 پیکسل باشند و بر 16 بخش‌پذیر باشند.
