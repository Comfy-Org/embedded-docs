Вот перевод документации на русский язык:

Узел Hunyuan Video 15 Latent Upscale With Model увеличивает разрешение скрытого представления изображения. Сначала он увеличивает скрытые сэмплы до заданного размера, используя выбранный метод интерполяции, а затем уточняет результат с помощью специализированной модели увеличения Hunyuan Video 1.5 для повышения качества.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `model` | LATENT_UPSCALE_MODEL | Да | Н/Д | Модель скрытого увеличения Hunyuan Video 1.5, используемая для уточнения увеличенных сэмплов. |
| `samples` | LATENT | Да | Н/Д | Скрытое представление изображения, которое необходимо увеличить. |
| `upscale_method` | COMBO | Нет | `"nearest-exact"`<br>`"bilinear"`<br>`"area"`<br>`"bicubic"`<br>`"bislerp"` | Алгоритм интерполяции, используемый на начальном этапе увеличения (по умолчанию: `"bilinear"`). |
| `width` | INT | Нет | от 0 до 16384 | Целевая ширина увеличенного скрытого представления в пикселях. Значение 0 автоматически вычисляет ширину на основе целевой высоты и исходного соотношения сторон. Итоговая ширина будет кратна 16 (по умолчанию: 1280). |
| `height` | INT | Нет | от 0 до 16384 | Целевая высота увеличенного скрытого представления в пикселях. Значение 0 автоматически вычисляет высоту на основе целевой ширины и исходного соотношения сторон. Итоговая высота будет кратна 16 (по умолчанию: 720). |
| `crop` | COMBO | Нет | `"disabled"`<br>`"center"` | Определяет, как увеличенное скрытое представление обрезается для соответствия целевым размерам. |

**Примечание о размерах:** Если и `width`, и `height` установлены в 0, узел возвращает входные `samples` без изменений. Если только один из размеров установлен в 0, другой размер вычисляется для сохранения исходного соотношения сторон. Итоговые размеры всегда корректируются так, чтобы быть не менее 64 пикселей и кратными 16.
