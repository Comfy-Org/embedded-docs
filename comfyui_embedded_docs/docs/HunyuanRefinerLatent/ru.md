Вот перевод документации на русский язык:

Узел HunyuanRefinerLatent обрабатывает входные данные conditioning и latent для операций уточнения. Он применяет шумовую аугментацию как к положительному, так и к отрицательному conditioning, одновременно интегрируя данные скрытого представления изображения, и генерирует новый выходной latent с заданными размерами для дальнейшей обработки.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `позитив` | CONDITIONING | Да | - | Входной положительный conditioning для обработки |
| `негатив` | CONDITIONING | Да | - | Входной отрицательный conditioning для обработки |
| `латент` | LATENT | Да | - | Входное скрытое представление |
| `шумовое_усиление` | FLOAT | Да | 0.0 - 1.0 | Степень применяемой шумовой аугментации (по умолчанию: 0.10) |

## Выходы

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `негатив` | CONDITIONING | Обработанный положительный conditioning с применённой шумовой аугментацией и конкатенацией скрытого изображения |
| `латент` | CONDITIONING | Обработанный отрицательный conditioning с применённой шумовой аугментацией и конкатенацией скрытого изображения |
| `латент` | LATENT | Новый выходной latent с размерами [batch_size, 32, height, width, channels] |
