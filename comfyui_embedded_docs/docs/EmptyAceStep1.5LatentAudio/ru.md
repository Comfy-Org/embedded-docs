Узел Empty Ace Step 1.5 Latent Audio создаёт пустой латентный тензор, предназначенный для обработки аудио. Он генерирует тихий аудиолатент заданной длительности и размера пакета, который можно использовать в качестве отправной точки для рабочих процессов генерации аудио в ComfyUI. Узел вычисляет длину латентного представления на основе входных секунд и фиксированной частоты дискретизации.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `seconds` | FLOAT | Да | 1.0 - 1000.0 | Длительность генерируемого аудио в секундах (по умолчанию: 120.0). |
| `batch_size` | INT | Да | 1 - 4096 | Количество латентных изображений в пакете (по умолчанию: 1). |

## Выходные данные

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `LATENT` | LATENT | Пустой латентный тензор, представляющий тихое аудио, с идентификатором типа "audio". |
