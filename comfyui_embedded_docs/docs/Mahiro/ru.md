Вот перевод документации на русский язык:

Узел Mahiro изменяет функцию управления (guidance), чтобы она больше фокусировалась на направлении позитивного промпта, а не на разнице между позитивным и негативным промптами. Он создает модифицированную модель, которая применяет пользовательский подход к масштабированию управления, используя косинусное сходство между нормализованными условными и безусловными результатами шумоподавления. Этот экспериментальный узел помогает направить генерацию более точно в сторону заданного направления позитивного промпта.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `модель` | MODEL | Да | | Модель, к которой будет применена модифицированная функция управления |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `patched_model` | MODEL | Модифицированная модель с примененной функцией управления Mahiro |
