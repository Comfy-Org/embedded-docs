گره Mahiro تابع هدایت (guidance) را تغییر می‌دهد تا بیشتر بر جهت پرامپت مثبت تمرکز کند تا تفاوت بین پرامپت مثبت و منفی. این گره یک مدل وصله‌شده (patched model) ایجاد می‌کند که با استفاده از شباهت کسینوسی (cosine similarity) بین خروجی‌های نویززدایی‌شده شرطی و غیرشرطی نرمال‌شده، رویکرد مقیاس‌گذاری هدایت سفارشی را اعمال می‌کند. این گره آزمایشی به هدایت قوی‌تر تولید به سمت جهت مورد نظر پرامپت مثبت کمک می‌کند.
