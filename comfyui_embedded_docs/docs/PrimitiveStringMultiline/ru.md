Вот перевод документации на русский язык:

Узел PrimitiveStringMultiline предоставляет поле ввода многострочного текста для ввода и передачи строковых значений через ваш рабочий процесс. Он принимает текстовый ввод с несколькими строками и выводит то же строковое значение без изменений. Этот узел полезен, когда вам нужно ввести более длинный текст или форматированный текст, занимающий несколько строк.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `значение` | STRING | Да | Н/Д | Входное текстовое значение, которое может занимать несколько строк |

## Выходы

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `output` | STRING | То же строковое значение, которое было передано на вход |
