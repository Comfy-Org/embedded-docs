Узел LaplaceScheduler генерирует последовательность значений сигма, следующих распределению Лапласа, для использования в диффузионном сэмплировании. Он создает расписание уровней шума, которые постепенно уменьшаются от максимального до минимального значения, используя параметры распределения Лапласа для управления прогрессией. Этот планировщик обычно используется в пользовательских рабочих процессах сэмплирования для определения расписания шума для диффузионных моделей.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `steps` | INT | Да | от 1 до 10000 | Количество шагов сэмплирования в расписании (по умолчанию: 20) |
| `sigma_max` | FLOAT | Да | от 0.0 до 5000.0 | Максимальное значение сигма в начале расписания (по умолчанию: 14.614642) |
| `sigma_min` | FLOAT | Да | от 0.0 до 5000.0 | Минимальное значение сигма в конце расписания (по умолчанию: 0.0291675) |
| `mu` | FLOAT | Да | от -10.0 до 10.0 | Параметр среднего для распределения Лапласа (по умолчанию: 0.0) |
| `beta` | FLOAT | Да | от 0.0 до 10.0 | Параметр масштаба для распределения Лапласа (по умолчанию: 0.5) |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `SIGMAS` | SIGMAS | Последовательность значений сигма, следующих расписанию распределения Лапласа |
