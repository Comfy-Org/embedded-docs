بخشی از یک نمونه صوتی موجود را با استفاده از دستورالعمل‌های متنی تغییر می‌دهد. این گره به شما امکان می‌دهد بخش‌های خاصی از صدا را با ارائه توضیحات توصیفی اصلاح کنید، به‌طور مؤثر بخش‌های انتخابی را «نقاشی‌سازی» یا بازتولید کنید و بقیه صدا را حفظ نمایید.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|---------|----------|--------|--------|---------|
| `model` | COMBO | بله | "stable-audio-2.5" | مدل هوش مصنوعی مورد استفاده برای نقاشی‌سازی صدا. |
| `prompt` | STRING | بله |  | توضیح متنی که نحوه تغییر صدا را هدایت می‌کند (پیش‌فرض: خالی). |
| `audio` | AUDIO | بله |  | فایل صوتی ورودی برای تغییر. طول صدا باید بین ۶ تا ۱۹۰ ثانیه باشد. |
| `duration` | INT | خیر | ۱-۱۹۰ | مدت زمان صدای تولید شده را بر حسب ثانیه کنترل می‌کند (پیش‌فرض: ۱۹۰). |
| `seed` | INT | خیر | ۰-۴۲۹۴۹۶۷۲۹۴ | دانه تصادفی مورد استفاده برای تولید (پیش‌فرض: ۰). |
| `steps` | INT | خیر | ۴-۸ | تعداد مراحل نمونه‌برداری را کنترل می‌کند (پیش‌فرض: ۸). |
| `mask_start` | INT | خیر | ۰-۱۹۰ | موقعیت شروع بر حسب ثانیه برای بخش صوتی که تغییر می‌کند (پیش‌فرض: ۳۰). |
| `mask_end` | INT | خیر | ۰-۱۹۰ | موقعیت پایان بر حسب ثانیه برای بخش صوتی که تغییر می‌کند (پیش‌فرض: ۱۹۰). |

**توجه:** مقدار `mask_end` باید بزرگ‌تر از مقدار `mask_start` باشد. طول صدای ورودی باید بین ۶ تا ۱۹۰ ثانیه باشد.
