Преобразует часть существующего аудиосэмпла с помощью текстовых инструкций. Этот узел позволяет изменять определенные участки аудио, задавая описательные подсказки, фактически выполняя "инпейнтинг" или регенерацию выбранных фрагментов, сохраняя остальную часть аудио без изменений.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `модель` | COMBO | Да | "stable-audio-2.5" | Модель ИИ, используемая для инпейнтинга аудио. |
| `промпт` | STRING | Да | | Текстовое описание, определяющее, как должно быть преобразовано аудио (по умолчанию: пусто). |
| `аудио` | AUDIO | Да | | Входной аудиофайл для преобразования. Длительность аудио должна быть от 6 до 190 секунд. |
| `длительность` | INT | Нет | 1-190 | Управляет длительностью генерируемого аудио в секундах (по умолчанию: 190). |
| `сид` | INT | Нет | 0-4294967294 | Случайное зерно, используемое для генерации (по умолчанию: 0). |
| `шаги` | INT | Нет | 4-8 | Управляет количеством шагов дискретизации (по умолчанию: 8). |
| `начало_маски` | INT | Нет | 0-190 | Начальная позиция в секундах для преобразуемого участка аудио (по умолчанию: 30). |
| `конец_маски` | INT | Нет | 0-190 | Конечная позиция в секундах для преобразуемого участка аудио (по умолчанию: 190). |

**Примечание:** Значение `mask_end` должно быть больше значения `mask_start`. Длительность входного аудио должна быть от 6 до 190 секунд.

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `аудио` | AUDIO | Преобразованный аудиовыход с измененным указанным участком в соответствии с подсказкой. |
