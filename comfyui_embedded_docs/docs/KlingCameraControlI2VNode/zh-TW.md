# Kling 圖片轉影片攝影機控制節點

Kling 圖片轉影片攝影機控制節點能將靜態圖片轉換為具有專業攝影機運鏡的電影級影片。這個專門的圖片轉影片節點讓您能夠控制虛擬攝影機動作，包括縮放、旋轉、平移、傾斜和第一人稱視角，同時保持對原始圖片的關注。攝影機控制目前僅在專業模式下支援，需使用 kling-v1-5 模型並設定為 5 秒鐘時長。

| 參數 | 資料類型 | 必要 | 範圍 | 說明 |
|-----------|-----------|----------|-------|-------------|
| `start_frame` | IMAGE | 是 | - | 參考圖片 - URL 或 Base64 編碼字串，不得超過 10MB，解析度不低於 300x300px，長寬比介於 1:2.5 至 2.5:1 之間。Base64 不應包含 data:image 前綴。 |
