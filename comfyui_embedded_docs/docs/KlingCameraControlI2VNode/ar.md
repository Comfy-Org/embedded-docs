# ترجمة وثيقة عقدة التحكم في كاميرا Kling لتحويل الصورة إلى فيديو

عقدة التحكم في كاميرا Kling لتحويل الصورة إلى فيديو تحول الصور الثابتة إلى مقاطع فيديو سينمائية مع حركات كاميرا احترافية. تتيح لك هذه العقدة المتخصصة في تحويل الصورة إلى فيديو التحكم في حركات الكاميرا الافتراضية بما في ذلك التكبير والتدوير والتحريك والميل ومنظور الشخص الأول مع الحفاظ على التركيز على صورتك الأصلية. التحكم في الكاميرا مدعوم حالياً فقط في الوضع الاحترافي مع نموذج kling-v1-5 بمدة 5 ثوانٍ.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|---------|--------------|--------|--------|-------|
| `الإطار الابتدائي` | IMAGE | نعم | - | الصورة المرجعية - رابط URL أو سلسلة مشفرة بـ Base64، لا يمكن أن يتجاوز حجمها 10 ميجابايت، دقة لا تقل عن 300×300 بكسل، نسبة أبعاد بين 1:2.5 و 2.5:1. يجب ألا تتضمن Base64 بادئة data:image. |
| `نص التوجيه الإيجابي` | STRING | نعم | - | النص الإيجابي الذي يصف محتوى الفيديو المطلوب |
| `نص التوجيه السلبي` | STRING | نعم | - | النص السلبي الذي يصف ما يجب تجنبه في الفيديو المُنشأ |
| `مقياس CFG` | FLOAT | لا | 0.0 إلى 1.0 | يتحكم في قوة توجيه النص. القيم الأعلى تجعل المخرجات تتبع النص بشكل أكثر دقة (الافتراضي: 0.75) |
| `نسبة العرض إلى الارتفاع` | COMBO | لا | `"16:9"`<br>`"9:16"`<br>`"1:1"` | نسبة أبعاد الفيديو المُنشأ (الافتراضي: "16:9") |
| `تحكم الكاميرا` | CAMERA_CONTROL | نعم | - | يمكن إنشاؤه باستخدام عقدة تحكمات كاميرا Kling. يتحكم في حركة الكاميرا والحركة أثناء إنشاء الفيديو. |

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|------------|--------------|-------|
| `معرّف الفيديو` | VIDEO | مخرج الفيديو المُنشأ |
| `المدة` | STRING | معرف فريد للفيديو المُنشأ |
| `duration` | STRING | مدة الفيديو المُنشأ |
