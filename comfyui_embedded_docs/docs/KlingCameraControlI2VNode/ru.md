Вот перевод документации на русский язык:

Узел управления камерой Kling Image to Video преобразует статичные изображения в кинематографичные видео с профессиональными движениями камеры. Этот специализированный узел преобразования изображения в видео позволяет управлять виртуальными движениями камеры, включая масштабирование, вращение, панорамирование, наклон и вид от первого лица, сохраняя при этом фокус на исходном изображении. Управление камерой в настоящее время поддерживается только в профессиональном режиме с моделью kling-v1-5 при длительности 5 секунд.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `start_frame` | IMAGE | Да | - | Исходное изображение — URL или строка в кодировке Base64, размер не более 10 МБ, разрешение не менее 300x300 пикселей, соотношение сторон от 1:2,5 до 2,5:1. Base64 не должен содержать префикс data:image. |
| `prompt` | STRING | Да | - | Положительный текстовый запрос, описывающий желаемое содержимое видео |
| `negative_prompt` | STRING | Да | - | Отрицательный текстовый запрос, описывающий, чего следует избегать в сгенерированном видео |
| `cfg_scale` | FLOAT | Нет | от 0,0 до 1,0 | Управляет силой текстового воздействия. Более высокие значения заставляют выходной результат точнее следовать запросу (по умолчанию: 0,75) |
| `aspect_ratio` | COMBO | Нет | `"16:9"`<br>`"9:16"`<br>`"1:1"` | Соотношение сторон генерируемого видео (по умолчанию: "16:9") |
| `camera_control` | CAMERA_CONTROL | Да | - | Может быть создан с помощью узла управления камерами Kling. Управляет движением камеры и перемещением во время генерации видео. |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `output` | VIDEO | Сгенерированное видео на выходе |
| `video_id` | STRING | Уникальный идентификатор сгенерированного видео |
| `duration` | STRING | Длительность сгенерированного видео |
