Узел PerpNegGuider создает систему управления для контроля генерации изображений с использованием перпендикулярного негативного кондиционирования. Он принимает положительные, отрицательные и пустые входные данные кондиционирования и применяет специализированный алгоритм управления для направления процесса генерации. Этот узел предназначен для экспериментального тестирования и обеспечивает точный контроль над силой управления и масштабированием негативного воздействия.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `модель` | MODEL | Да | - | Модель, используемая для генерации управления |
| `положительный` | CONDITIONING | Да | - | Положительное кондиционирование, направляющее генерацию к желаемому содержимому |
| `отрицательный` | CONDITIONING | Да | - | Отрицательное кондиционирование, отводящее генерацию от нежелательного содержимого |
| `пустое_условие` | CONDITIONING | Да | - | Пустое или нейтральное кондиционирование, используемое в качестве базового эталона |
| `cfg` | FLOAT | Да | 0.0 - 100.0 | Масштаб бесклассификационного управления, определяющий силу влияния кондиционирования на генерацию (по умолчанию: 8.0) |
| `масштаб_отриц` | FLOAT | Да | 0.0 - 100.0 | Коэффициент масштабирования негативного воздействия, регулирующий силу отрицательного кондиционирования (по умолчанию: 1.0) |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `guider` | GUIDER | Настроенная система управления, готовая к использованию в конвейере генерации |
