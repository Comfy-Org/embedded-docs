گره PerpNegGuider یک سیستم هدایت برای کنترل تولید تصویر با استفاده از شرط‌گذاری عمود بردار منفی ایجاد می‌کند. این گره ورودی‌های شرط‌گذاری مثبت، منفی و خالی را دریافت کرده و با استفاده از یک الگوریتم هدایت تخصصی، فرآیند تولید را هدایت می‌کند. این گره برای آزمایش‌های تجربی طراحی شده و کنترل دقیقی بر قدرت هدایت و مقیاس‌گذاری منفی ارائه می‌دهد.

## ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|---------|----------|-------|--------|---------|
| `مدل` | MODEL | بله | - | مدلی که برای تولید هدایت استفاده می‌شود |
| `مثبت` | CONDITIONING | بله | - | شرط‌گذاری مثبت که تولید را به سمت محتوای دلخواه هدایت می‌کند |
| `منفی` | CONDITIONING | بله | - | شرط‌گذاری منفی که تولید را از محتوای ناخواسته دور می‌کند |
| `شرط‌گذاری خالی` | CONDITIONING | بله | - | شرط‌گذاری خالی یا خنثی که به عنوان مرجع پایه استفاده می‌شود |
| `cfg` | FLOAT | بله | 0.0 - 100.0 | مقیاس هدایت بدون طبقه‌بند که میزان تأثیر شرط‌گذاری بر تولید را کنترل می‌کند (پیش‌فرض: 8.0) |
| `مقیاس منفی` | FLOAT | بله | 0.0 - 100.0 | ضریب مقیاس‌گذاری منفی که قدرت شرط‌گذاری منفی را تنظیم می‌کند (پیش‌فرض: 1.0) |
