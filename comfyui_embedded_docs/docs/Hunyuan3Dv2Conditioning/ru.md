Вот перевод документации на русский язык:

Узел Hunyuan3Dv2Conditioning обрабатывает выходные данные CLIP vision для генерации данных кондиционирования для 3D-моделей. Он извлекает эмбеддинги последнего скрытого состояния из выходных данных vision и создает как положительные, так и отрицательные пары кондиционирования. Положительное кондиционирование использует фактические эмбеддинги, в то время как отрицательное кондиционирование использует нулевые эмбеддинги той же формы.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `выход_clip_vision` | CLIP_VISION_OUTPUT | Да | - | Выходные данные модели CLIP vision, содержащие визуальные эмбеддинги |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `positive` | CONDITIONING | Положительные данные кондиционирования, содержащие эмбеддинги CLIP vision |
| `negative` | CONDITIONING | Отрицательные данные кондиционирования, содержащие нулевые эмбеддинги, соответствующие форме положительных эмбеддингов |
