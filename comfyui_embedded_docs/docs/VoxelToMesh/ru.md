Узел VoxelToMeshBasic преобразует трёхмерные воксельные данные в полигональную сетку путём извлечения поверхности при заданном пороговом значении. Он обрабатывает каждую воксельную решётку на входе и генерирует вершины и грани, формирующие трёхмерное представление сетки.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `voxel` | VOXEL | Да | - | Входные воксельные данные для преобразования в полигональную сетку |
| `threshold` | FLOAT | Да | -1.0 до 1.0 | Пороговое значение для извлечения поверхности (по умолчанию: 0.6) |

## Выходные данные

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `MESH` | MESH | Сгенерированная трёхмерная сетка, содержащая объединённые вершины и грани из всех входных воксельных решёток |
