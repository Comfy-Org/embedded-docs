Узел StringCompare сравнивает две текстовые строки, используя различные методы сравнения. Он может проверять, начинается ли одна строка с другой, заканчивается ли другой, или являются ли обе строки полностью равными. Сравнение может выполняться с учётом или без учёта регистра символов.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `string_a` | STRING | Да | - | Первая строка для сравнения |
| `string_b` | STRING | Да | - | Вторая строка для сравнения |
| `mode` | COMBO | Да | "Starts With"<br>"Ends With"<br>"Equal" | Метод сравнения (по умолчанию: "Starts With") |
| `case_sensitive` | BOOLEAN | Нет | - | Учитывать ли регистр символов при сравнении (по умолчанию: true) |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `output` | BOOLEAN | Возвращает true, если условие сравнения выполнено, и false в противном случае |
