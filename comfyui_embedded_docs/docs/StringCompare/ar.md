# ترجمة وثيقة عقدة StringCompare

عقدة StringCompare تقارن بين سلسلتين نصيتين باستخدام طرق مقارنة مختلفة. يمكنها التحقق مما إذا كانت إحدى السلاسل تبدأ بأخرى، أو تنتهي بأخرى، أو إذا كانت السلستان متطابقتين تمامًا. يمكن إجراء المقارنة مع مراعاة أو تجاهل حالة الأحرف.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|---------|---------------|--------|---------|-------|
| `string_a` | STRING | نعم | - | السلسلة النصية الأولى للمقارنة |
| `string_b` | STRING | نعم | - | السلسلة النصية الثانية للمقارنة معها |
| `mode` | COMBO | نعم | "Starts With"<br>"Ends With"<br>"Equal" | طريقة المقارنة المستخدمة (الافتراضي: "Starts With") |
| `case_sensitive` | BOOLEAN | لا | - | هل يتم مراعاة حالة الأحرف أثناء المقارنة (الافتراضي: true) |

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|-------------|---------------|-------|
| `output` | BOOLEAN | يُرجع true إذا تحقق شرط المقارنة، و false في الحالة المعاكسة |
