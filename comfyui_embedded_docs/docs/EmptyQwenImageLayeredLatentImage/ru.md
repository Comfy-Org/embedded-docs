Вот перевод документации на русский язык:

Узел Empty Qwen Image Layered Latent создает пустое многослойное латентное представление для использования с моделями изображений Qwen. Он генерирует тензор, заполненный нулями, структурированный с заданным количеством слоев, размером пакета и пространственными размерами. Это пустое латентное представление служит отправной точкой для последующих рабочих процессов генерации или обработки изображений.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `width` | INT | Да | 16 до MAX_RESOLUTION | Ширина создаваемого латентного изображения. Значение должно делиться на 16. (по умолчанию: 640) |
| `height` | INT | Да | 16 до MAX_RESOLUTION | Высота создаваемого латентного изображения. Значение должно делиться на 16. (по умолчанию: 640) |
| `layers` | INT | Да | 0 до MAX_RESOLUTION | Количество дополнительных слоев, добавляемых к латентной структуре. Определяет глубину латентного представления. (по умолчанию: 3) |
| `batch_size` | INT | Нет | 1 до 4096 | Количество латентных образцов, генерируемых в одном пакете. (по умолчанию: 1) |

**Примечание:** Параметры `width` и `height` внутренне делятся на 8 для определения пространственных размеров выходного латентного тензора.

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `samples` | LATENT | Латентный тензор, заполненный нулями. Его форма: `[batch_size, 16, layers + 1, height // 8, width // 8]`. |
