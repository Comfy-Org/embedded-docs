## Обзор

Генерируйте текстовые ответы с использованием моделей ByteDance Seed 2.0. Укажите текстовый запрос и при необходимости добавьте изображения или видео для мультимодального контекста.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-----------|-------------|
| `prompt` | STRING | Да | Н/Д | Текстовый ввод для модели. |
| `model` | COMBO | Да | `"Seed 2.0 Pro"`<br>`"Seed 2.0 Lite"`<br>`"Seed 2.0 Mini"` | Модель Seed, используемая для генерации ответа. |
| `seed` | INT | Да | от 0 до 2147483647 | Значение seed определяет, должен ли узел выполниться повторно; результаты недетерминированы независимо от seed. (по умолчанию: 0) |
| `system_prompt` | STRING | Нет | Н/Д | Базовые инструкции, определяющие поведение модели. (по умолчанию: "") |

**Примечание по параметру `model`:** Параметр `model` представляет собой динамическое комбинированное поле, которое также принимает изображения и видео. Вы можете подключить входные данные изображений и видео к этому параметру для обеспечения мультимодального контекста. На один запрос поддерживается не более 20 изображений и 4 видео.

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `output` | STRING | Сгенерированный текстовый ответ от модели Seed. |
