## نمای کلی

با استفاده از مدل‌های Seed 2.0 شرکت ByteDance، پاسخ‌های متنی تولید کنید. یک متن راهنما (prompt) ارائه دهید و در صورت تمایل، تصاویر یا ویدیوهایی برای زمینه چندرسانه‌ای اضافه کنید.

## ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `prompt` | STRING | بله | نامشخص | ورودی متنی به مدل. |
| `model` | COMBO | بله | `"Seed 2.0 Pro"`<br>`"Seed 2.0 Lite"`<br>`"Seed 2.0 Mini"` | مدل Seed مورد استفاده برای تولید پاسخ. |
| `seed` | INT | بله | 0 تا 2147483647 | seed مشخص می‌کند که آیا گره باید دوباره اجرا شود یا خیر؛ نتایج بدون توجه به seed غیرقطعی هستند. (پیش‌فرض: 0) |
| `system_prompt` | STRING | خیر | نامشخص | دستورالعمل‌های پایه‌ای که رفتار مدل را تعیین می‌کنند. (پیش‌فرض: "") |

**نکته درباره پارامتر `model`:** پارامتر `model` یک جعبه ترکیبی پویا است که تصاویر و ویدیوها را نیز می‌پذیرد. می‌توانید ورودی‌های تصویر و ویدیو را به این پارامتر متصل کنید تا زمینه چندرسانه‌ای فراهم شود. حداکثر ۲۰ تصویر و ۴ ویدیو در هر درخواست پشتیبانی می‌شود.
