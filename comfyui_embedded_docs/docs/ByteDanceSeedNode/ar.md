# نظرة عامة

توليد استجابات نصية باستخدام نماذج Seed 2.0 من ByteDance. قم بتوفير مطالبة نصية، ويمكنك اختياريًا تضمين الصور أو مقاطع الفيديو لتوفير سياق متعدد الوسائط.

# المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|----------|---------------|--------|---------|-------------|
| `prompt` | STRING | نعم | غير متاح | الإدخال النصي للنموذج. |
| `model` | COMBO | نعم | `"Seed 2.0 Pro"`<br>`"Seed 2.0 Lite"`<br>`"Seed 2.0 Mini"` | نموذج Seed المستخدم لتوليد الاستجابة. |
| `seed` | INT | نعم | 0 إلى 2147483647 | يتحكم البذرة في ما إذا كان يجب إعادة تشغيل العقدة؛ النتائج غير حتمية بغض النظر عن البذرة. (الافتراضي: 0) |
| `system_prompt` | STRING | لا | غير متاح | تعليمات أساسية تحدد سلوك النموذج. (الافتراضي: "") |

**ملاحظة حول معامل `model`:** معامل `model` هو قائمة ديناميكية تقبل أيضًا الصور ومقاطع الفيديو. يمكنك توصيل مدخلات الصور والفيديو بهذا المعامل لتوفير سياق متعدد الوسائط. يُدعم حد أقصى 20 صورة و4 مقاطع فيديو لكل طلب.

# المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|-------------|---------------|-------------|
| `output` | STRING | الاستجابة النصية المُولَّدة من نموذج Seed. |
