WanDancerVideo ノードは、WanDancer モデルによる動画生成のための条件付けデータと空の潜在テンソルを準備します。開始画像、マスク、CLIP ビジョン埋め込み、オーディオ特徴量などのオプション入力を使用して、生成される動画を制御するために、ポジティブ条件付けとネガティブ条件付けを組み合わせます。

## 入力

| パラメータ | データ型 | 必須 | 範囲 | 説明 |
|-----------|-----------|----------|-------|-------------|
| `positive` | CONDITIONING | はい | | 動画生成をガイドするポジティブ条件付け。 |
| `negative` | CONDITIONING | はい | | 動画生成をガイドするネガティブ条件付け。 |
| `vae` | VAE | はい | | 開始画像を潜在空間にエンコードするために使用される VAE。 |
| `width` | INT | はい | 16 ～ MAX_RESOLUTION (ステップ: 16) | 生成される動画の幅（ピクセル単位、デフォルト: 480）。 |
| `height` | INT | はい | 16 ～ MAX_RESOLUTION (ステップ: 16) | 生成される動画の高さ（ピクセル単位、デフォルト: 832）。 |
| `length` | INT | はい | 1 ～ MAX_RESOLUTION (ステップ: 4) | 生成される動画のフレーム数。WanDancer では 149 を維持する必要があります（デフォルト: 149）。 |
| `clip_vision_output` | CLIP_VISION_OUTPUT | いいえ | | 最初のフレームの CLIP ビジョン埋め込み。 |
| `clip_vision_output_ref` | CLIP_VISION_OUTPUT | いいえ | | 参照画像の CLIP ビジョン埋め込み。 |
| `start_image` | IMAGE | いいえ | | エンコードされる初期画像。指定された `length` までの任意のフレーム数にできます。 |
| `mask` | MASK | いいえ | | 開始画像の画像条件付けマスク。白い領域は保持され、黒い領域は生成されます。ローカル生成に使用されます。 |
| `audio_encoder_output` | AUDIO_ENCODER_OUTPUT | いいえ | | オーディオエンコーダからの出力。オーディオ条件付き生成のためのオーディオ特徴量、fps、注入スケールを提供します。 |

**パラメータ制約に関する注意事項:**
- `start_image` と `mask` の入力はオプションですが、一緒に使用できます。`start_image` が提供されると、それがエンコードされ、潜在テンソルと連結されます。`mask` も提供される場合、開始画像のどの部分を保持するか（白）と、どの部分を再生成するか（黒）を制御します。`mask` が提供されない場合、開始画像領域全体が条件付けガイドとして使用されます。
- `clip_vision_output` と `clip_vision_output_ref` の入力はオプションであり、最初のフレームと参照画像の視覚的コンテキストを提供するために一緒に使用できます。
- `audio_encoder_output` の入力はオプションであり、オーディオ条件付き生成のためのオーディオ特徴量を提供します。
