Вот перевод документации на русский язык:

Этот узел генерирует 3D-активы с использованием API Rodin. Он принимает входные изображения и преобразует их в 3D-модели через внешний сервис. Узел обрабатывает весь процесс от создания задачи до загрузки конечных файлов 3D-модели.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `Images` | IMAGE | Да | - | Входные изображения для преобразования в 3D-модели. Можно указать несколько изображений. |
| `Seed` | INT | Нет | 0-65535 | Значение случайного зерна для генерации (по умолчанию: 0). Установите 0 для случайного зерна. |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `3D Model Path` | STRING | Путь к файлу сгенерированной 3D-модели (только для обратной совместимости) |
| `GLB` | FILE3DGLB | Сгенерированная 3D-модель в формате GLB |
