این گره با استفاده از API رادین (Rodin) دارایی‌های سه‌بعدی تولید می‌کند. تصاویر ورودی را دریافت کرده و از طریق یک سرویس خارجی به مدل‌های سه‌بعدی تبدیل می‌کند. این گره کل فرایند را از ایجاد وظیفه تا دانلود فایل‌های نهایی مدل سه‌بعدی مدیریت می‌کند.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|---------|----------|--------|--------|---------|
| `Images` | IMAGE | بله | - | تصاویر ورودی برای تبدیل به مدل‌های سه‌بعدی. می‌توان چندین تصویر ارائه داد. |
| `Seed` | INT | خیر | 0-65535 | مقدار دانه تصادفی برای تولید (پیش‌فرض: 0). برای دانه تصادفی، مقدار 0 را تنظیم کنید. |

## خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-----------|----------|---------|
| `3D Model Path` | STRING | مسیر فایل مدل سه‌بعدی تولیدشده (فقط برای سازگاری با نسخه‌های قبلی) |
| `GLB` | FILE3DGLB | مدل سه‌بعدی تولیدشده در قالب GLB |
