Узел CLIPAdd объединяет две модели CLIP путём слияния их ключевых патчей. Он создаёт копию первой модели CLIP, а затем добавляет большинство ключевых патчей из второй модели, за исключением идентификаторов позиций и параметров шкалы логитов. Это позволяет смешивать признаки из разных моделей CLIP, сохраняя структуру первой модели.

## Входные параметры

| Параметр | Тип данных | Тип входа | По умолчанию | Диапазон | Описание |
|-----------|-----------|------------|---------|-------|-------------|
| `clip1` | CLIP | Обязательный | - | - | Основная модель CLIP, которая будет использоваться в качестве базы для слияния |
| `clip2` | CLIP | Обязательный | - | - | Вторичная модель CLIP, предоставляющая дополнительные патчи для добавления |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `CLIP` | CLIP | Возвращает объединённую модель CLIP, сочетающую признаки обеих входных моделей |
