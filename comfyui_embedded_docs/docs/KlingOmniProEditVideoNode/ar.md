# ترجمة وثيقة عقد KlingOmniProEditVideoNode

عقد تعديل الفيديو Kling Omni (المتقدم) يستخدم نموذج ذكاء اصطناعي لتعديل فيديو موجود بناءً على وصف نصي. تقوم بتوفير فيديو مصدر وموجه نصي، ويقوم العقد بإنشاء فيديو جديد بنفس المدة مع التغييرات المطلوبة. يمكنه استخدام صور مرجعية اختيارياً لتوجيه الأسلوب والاحتفاظ بالصوت الأصلي من الفيديو المصدر.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|---------|---------------|--------|---------|-------|
| `model_name` | COMBO | نعم | `"kling-v3-omni"`<br>`"kling-video-o1"` | نموذج الذكاء الاصطناعي المستخدم لتعديل الفيديو (الافتراضي: `"kling-v3-omni"`). |
| `prompt` | STRING | نعم | | موجه نصي يصف محتوى الفيديو. يمكن أن يتضمن أوصافاً إيجابية وسلبية. |
| `video` | VIDEO | نعم | | الفيديو المراد تعديله. ستكون مدة الفيديو الناتج مماثلة. |
| `keep_original_sound` | BOOLEAN | نعم | | يحدد ما إذا كان سيتم الاحتفاظ بالصوت الأصلي من الفيديو المدخل في المخرجات (الافتراضي: صحيح). |
| `reference_images` | IMAGE | لا | | ما يصل إلى 4 صور مرجعية إضافية. |
| `resolution` | COMBO | لا | `"1080p"`<br>`"720p"` | دقة الفيديو الناتج (الافتراضي: `"1080p"`). |
| `seed` | INT | لا | 0 إلى 2147483647 | البذرة تتحكم فيما إذا كان العقد سيعيد التشغيل؛ النتائج غير حتمية بغض النظر عن البذرة (الافتراضي: 0). |

**القيود والمحددات:**

* يجب أن يتراوح طول `prompt` بين حرف واحد و2500 حرف.
* يجب أن تتراوح مدة `video` المدخل بين 3.0 و10.05 ثانية.
* يجب أن تتراوح أبعاد `video` المدخل بين 720×720 و2160×2160 بكسل.
* يمكن تقديم حد أقصى 4 `reference_images` عند استخدام فيديو.
* يجب أن تكون كل `reference_image` بحجم لا يقل عن 300×300 بكسل.
* يجب أن تكون نسبة العرض إلى الارتفاع لكل `reference_image` بين 1:2.5 و2.5:1.
