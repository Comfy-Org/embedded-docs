گره ویرایش ویدیوی Kling Omni (Pro) از یک مدل هوش مصنوعی برای ویرایش ویدیوی موجود بر اساس توضیحات متنی استفاده می‌کند. شما یک ویدیوی منبع و یک prompt ارائه می‌دهید و گره یک ویدیوی جدید با همان طول و تغییرات درخواستی تولید می‌کند. این گره می‌تواند به‌صورت اختیاری از تصاویر مرجع برای هدایت سبک استفاده کرده و صدای اصلی ویدیوی منبع را حفظ کند.

## ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `نام مدل` | COMBO | بله | `"kling-v3-omni"`<br>`"kling-video-o1"` | مدل هوش مصنوعی مورد استفاده برای ویرایش ویدیو (پیش‌فرض: `"kling-v3-omni"`). |
| `پرامپت` | STRING | بله | | یک prompt متنی که محتوای ویدیو را توصیف می‌کند. این می‌تواند شامل توضیحات مثبت و منفی باشد. |
| `ویدیو` | VIDEO | بله | | ویدیو برای ویرایش. طول ویدیوی خروجی مشابه خواهد بود. |
| `حفظ صدای اصلی` | BOOLEAN | بله | | تعیین می‌کند که آیا صدای اصلی ویدیوی ورودی در خروجی حفظ شود (پیش‌فرض: True). |
| `تصاویر مرجع` | IMAGE | خیر | | حداکثر ۴ تصویر مرجع اضافی. |
| `وضوح` | COMBO | خیر | `"1080p"`<br>`"720p"` | وضوح تصویر برای ویدیوی خروجی (پیش‌فرض: `"1080p"`). |
| `seed` | INT | خیر | 0 تا 2147483647 | seed مشخص می‌کند که آیا گره باید دوباره اجرا شود؛ نتایج صرف‌نظر از seed غیرقطعی هستند (پیش‌فرض: 0). |

**محدودیت‌ها:**

* `prompt` باید بین ۱ تا ۲۵۰۰ کاراکتر باشد.
* مدت زمان `video` ورودی باید بین ۳.۰ تا ۱۰.۰۵ ثانیه باشد.
* ابعاد `video` ورودی باید بین ۷۲۰×۷۲۰ و ۲۱۶۰×۲۱۶۰ پیکسل باشد.
* حداکثر ۴ `reference_images` می‌تواند هنگام استفاده از ویدیو ارائه شود.
* هر `reference_image` باید حداقل ۳۰۰×۳۰۰ پیکسل باشد.
* هر `reference_image` باید نسبت تصویری بین ۱:۲.۵ و ۲.۵:۱ داشته باشد.
