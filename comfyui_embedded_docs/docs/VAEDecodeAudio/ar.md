# ترجمة وثيقة عقدة VAEDecodeAudio

تقوم عقدة VAEDecodeAudio بتحويل التمثيلات الكامنة (latent representations) إلى موجات صوتية باستخدام جهاز الترميز التلقائي المتغير (Variational Autoencoder). تستقبل هذه العقدة عينات صوتية مشفرة وتعالجها عبر VAE لإعادة بناء الصوت الأصلي، مع تطبيق التسوية (normalization) لضمان مستويات إخراج متسقة. يتم إرجاع الصوت الناتج بمعدل عينة قياسي يبلغ 44100 هرتز.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|---------|--------------|--------|--------|-------|
| `samples` | LATENT | نعم | - | العينات الصوتية المشفرة في الفضاء الكامن التي سيتم فك تشفيرها إلى موجة صوتية |
| `vae` | VAE | نعم | - | نموذج جهاز الترميز التلقائي المتغير المستخدم لفك تشفير العينات الكامنة إلى صوت |

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|------------|--------------|-------|
| `AUDIO` | AUDIO | الموجة الصوتية المفكوكة مع حجم مضبوط (normalized) ومعدل عينة 44100 هرتز |
