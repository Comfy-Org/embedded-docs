Узел VAEDecodeAudio преобразует скрытые представления обратно в аудиосигналы с помощью вариационного автоэнкодера. Он принимает закодированные аудиосэмплы и обрабатывает их через VAE для восстановления исходного аудио, применяя нормализацию для обеспечения стабильного уровня выходного сигнала. Полученное аудио возвращается со стандартной частотой дискретизации 44100 Гц.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `образцы` | LATENT | Да | - | Закодированные аудиосэмплы в скрытом пространстве, которые будут декодированы обратно в аудиосигнал |
| `vae` | VAE | Да | - | Модель вариационного автоэнкодера, используемая для декодирования скрытых сэмплов в аудио |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `AUDIO` | AUDIO | Декодированный аудиосигнал с нормализованной громкостью и частотой дискретизации 44100 Гц |
