Узел SamplerDPMPP_3M_SDE создает сэмплер DPM++ 3M SDE для использования в процессе сэмплирования. Этот сэмплер использует метод стохастических дифференциальных уравнений третьего порядка с несколькими шагами и настраиваемыми параметрами шума. Узел позволяет выбрать, будут ли вычисления шума выполняться на GPU или CPU.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `eta` | FLOAT | Да | 0.0 - 100.0 | Управляет стохастичностью процесса сэмплирования (по умолчанию: 1.0) |
| `s_noise` | FLOAT | Да | 0.0 - 100.0 | Управляет количеством шума, добавляемого во время сэмплирования (по умолчанию: 1.0) |
| `устройство_шума` | COMBO | Да | "gpu"<br>"cpu" | Выбирает устройство для вычислений шума: GPU или CPU (по умолчанию: "gpu") |

## Выходные данные

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `sampler` | SAMPLER | Возвращает настроенный объект сэмплера для использования в рабочих процессах сэмплирования |
