هذه ترجمة الوثيقة التقنية لعقدة ComfyUI إلى العربية، مع الالتزام بالقواعد المحددة:

عقدة **QwenImageDiffsynthControlnet** تطبق تصحيح شبكة تحكم في التوليف الانتشارى لتعديل سلوك النموذج الأساسي. تستخدم العقدة صورة إدخال وقناع اختياري لتوجيه عملية توليد النموذج بقوة قابلة للتعديل، مما يُنتج نموذجًا مُصححًا يدمج تأثير شبكة التحكم لتحقيق توليف صور أكثر تحكمًا.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|-----------|-----------|----------|-------|-------------|
| `model` | MODEL | نعم | - | النموذج الأساسي المراد تصحيحه باستخدام شبكة التحكم |
| `model_patch` | MODEL_PATCH | نعم | - | نموذج تصحيح شبكة التحكم المراد تطبيقه على النموذج الأساسي |
| `vae` | VAE | نعم | - | المشفر التلقائي المتغير (VAE) المستخدم في عملية الانتشار |
| `image` | IMAGE | نعم | - | صورة الإدخال المستخدمة لتوجيه شبكة التحكم (يتم استخدام قنوات RGB فقط) |
| `strength` | FLOAT | نعم | -10.0 إلى 10.0 | قوة تأثير شبكة التحكم (القيمة الافتراضية: 1.0) |
| `mask` | MASK | لا | - | قناع اختياري يحدد المناطق التي سيتم تطبيق شبكة التحكم فيها (يتم عكسه داخليًا) |

**ملاحظة:** عند توفير قناع، يتم عكسه تلقائيًا (1.0 - mask) وإعادة تشكيله ليتوافق مع الأبعاد المتوقعة لمعالجة شبكة التحكم. تستخدم العقدة طرق معالجة داخلية مختلفة اعتمادًا على ما إذا كان تصحيح النموذج من نوع تحكم صورة ZImage أم شبكة تحكم DiffSynth قياسية.
