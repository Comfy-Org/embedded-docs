# SaveText

گره ذخیره متن (Save Text) محتوای متنی را در یک فایل در دایرکتوری خروجی ذخیره می‌کند. این گره از ذخیره‌سازی در قالب‌های `.txt`، `.md` یا `.json` پشتیبانی کرده و در صورت ارائه JSON معتبر، به‌طور خودکار قالب‌بندی زیبا (pretty-printing) آن را انجام می‌دهد.

## ورودی‌ها

| پارامتر | توضیحات | نوع داده | ضروری | محدوده |
|---------|---------|----------|-------|--------|
| `text` | محتوای متنی که باید در فایل ذخیره شود | STRING | بله | - |
| `filename_prefix` | پیشوند نام فایل خروجی (پیش‌فرض: "ComfyUI") | STRING | خیر | - |
| `format` | قالب فایل برای ذخیره متن (پیش‌فرض: "txt") | STRING | خیر | `"txt"`<br>`"md"`<br>`"json"` |

## خروجی‌ها

| نام خروجی | توضیحات | نوع داده |
|-----------|---------|----------|
| `text` | محتوای متنی اصلی که در فایل ذخیره شد | STRING |
