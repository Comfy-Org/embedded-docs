# SaveText

گره ذخیره متن (Save Text) محتوای متنی را در یک فایل در دایرکتوری خروجی ذخیره می‌کند. این گره از ذخیره‌سازی در سه فرمت فایل مختلف پشتیبانی می‌کند: متن ساده (.txt)، مارک‌داون (.md) و JSON (.json). هنگام ذخیره به صورت JSON، گره سعی می‌کند متن ورودی را به عنوان JSON معتبر تجزیه کرده و با تورفتگی مناسب قالب‌بندی کند.

## ورودی‌ها

| پارامتر | توضیحات | نوع داده | الزامی | محدوده |
|---------|---------|----------|--------|--------|
| `text` | محتوای متنی که باید در فایل ذخیره شود | STRING | بله | |
| `filename_prefix` | پیشوند نام فایل خروجی (پیش‌فرض: "ComfyUI") | STRING | خیر | |
| `format` | فرمت فایل برای ذخیره متن (پیش‌فرض: "txt") | STRING | خیر | `"txt"`<br>`"md"`<br>`"json"` |

## خروجی‌ها

| نام خروجی | توضیحات | نوع داده |
|-----------|---------|----------|
| `text` | محتوای متنی اصلی که در فایل ذخیره شد | STRING |
