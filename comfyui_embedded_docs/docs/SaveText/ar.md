# SaveText

إليك ترجمة الوثيقة التقنية لعقدة "حفظ النص" (Save Text) في ComfyUI إلى العربية، مع الالتزام التام بقواعد الترجمة المحددة:

يقوم عقدة حفظ النص بكتابة محتوى نصي إلى ملف في دليل الإخراج. تدعم العقدة الحفظ بصيغ `.txt` أو `.md` أو `.json`، وتتعامل تلقائيًا مع تنسيق JSON الجميل عند توفير نص JSON صالح.

## المدخلات

| المعامل | الوصف | نوع البيانات | إلزامي | النطاق |
|-----------|-------------|-----------|----------|-------|
| `text` | المحتوى النصي المراد حفظه في ملف | STRING | نعم | - |
| `filename_prefix` | بادئة لاسم ملف الإخراج (الافتراضي: "ComfyUI") | STRING | لا | - |
| `format` | صيغة الملف لحفظ النص بها (الافتراضي: "txt") | STRING | لا | `"txt"`<br>`"md"`<br>`"json"` |

## المخرجات

| اسم المخرج | الوصف | نوع البيانات |
|-------------|-------------|-----------|
| `text` | المحتوى النصي الأصلي الذي تم حفظه في الملف | STRING |
