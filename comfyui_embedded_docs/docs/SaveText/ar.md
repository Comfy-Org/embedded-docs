# SaveText

عقدة حفظ النص تقوم بكتابة محتوى نصي إلى ملف في مجلد الإخراج. وهي تدعم الحفظ بثلاثة تنسيقات ملفات مختلفة: نص عادي (.txt)، وماركداون (.md)، وجيسون (.json). عند الحفظ بصيغة JSON، تحاول العقدة تحليل النص المدخل كـ JSON صالح وتنسيقه بمسافات بادئة مناسبة.

## المدخلات

| المعامل | الوصف | نوع البيانات | إلزامي | النطاق |
|-----------|-------------|-----------|----------|-------|
| `text` | المحتوى النصي المراد حفظه في ملف | STRING | نعم | |
| `filename_prefix` | بادئة لاسم ملف الإخراج (الافتراضي: "ComfyUI") | STRING | لا | |
| `format` | تنسيق الملف لحفظ النص به (الافتراضي: "txt") | STRING | لا | `"txt"`<br>`"md"`<br>`"json"` |

## المخرجات

| اسم المخرج | الوصف | نوع البيانات |
|-------------|-------------|-----------|
| `text` | المحتوى النصي الأصلي الذي تم حفظه في الملف | STRING |
