# Save3DAdvanced

يقوم هذا العقدة بحفظ نموذج ثلاثي الأبعاد في دليل الإخراج الخاص بـ ComfyUI ويوفر إمكانيات معاينة متقدمة. يتيح لك تحديد اسم ملف الإخراج وأبعاد المعاينة، ويمرر اختياريًا معلومات الكاميرا والنموذج لتجربة محسّنة في عارض ثلاثي الأبعاد.

## المدخلات

| المعامل | الوصف | نوع البيانات | إلزامي | النطاق |
|---------|--------|---------------|---------|--------|
| `model_3d` | ملف نموذج ثلاثي الأبعاد من عقدة ثلاثية الأبعاد سابقة. | FILE3D | نعم | GLB, GLTF, FBX, OBJ, STL, USDZ أو أي تنسيق ثلاثي الأبعاد |
| `filename_prefix` | بادئة لاسم الملف المحفوظ. سيتم إلحاق عداد وامتداد الملف تلقائيًا. (الافتراضي: "3d/ComfyUI") | STRING | نعم | أي سلسلة اسم ملف صالحة |
| `viewport_state` | حالة منفذ العرض الحالية، عادةً من عقدة عارض ثلاثي الأبعاد. | LOAD3D | نعم | - |
| `model_3d_info` | معلومات اختيارية للنموذج ثلاثي الأبعاد للمعاينة. | LOAD3DMODELINFO | لا | - |
| `camera_info` | معلومات اختيارية للكاميرا للمعاينة. | LOAD3DCAMERA | لا | - |
| `width` | عرض المعاينة بالبكسل. (الافتراضي: 1024) | INT | نعم | 1 إلى 4096 |
| `height` | ارتفاع المعاينة بالبكسل. (الافتراضي: 1024) | INT | نعم | 1 إلى 4096 |

## المخرجات

| اسم المخرج | الوصف | نوع البيانات |
|------------|--------|---------------|
| `model_3d` | ملف النموذج ثلاثي الأبعاد المحفوظ. | FILE3D |
| `model_3d_info` | معلومات النموذج ثلاثي الأبعاد التي تم تمريرها للعقد اللاحقة. | LOAD3DMODELINFO |
| `camera_info` | معلومات الكاميرا التي تم تمريرها للعقد اللاحقة. | LOAD3DCAMERA |
| `width` | قيمة العرض التي تم تمريرها للعقد اللاحقة. | INT |
| `height` | قيمة الارتفاع التي تم تمريرها للعقد اللاحقة. | INT |
