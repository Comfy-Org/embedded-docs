# Save3DAdvanced

Этот узел сохраняет 3D-модель в выходной каталог ComfyUI и предоставляет расширенные возможности предварительного просмотра. Он позволяет указать имя выходного файла, размеры превью и, при необходимости, передать информацию о камере и модели для улучшенного просмотра в 3D-вьювере.

## Входные параметры

| Параметр | Описание | Тип данных | Обязательный | Диапазон |
|-----------|-------------|-----------|----------|-------|
| `model_3d` | Файл 3D-модели из вышестоящего 3D-узла. | FILE3D | Да | GLB, GLTF, FBX, OBJ, STL, USDZ или любой 3D-формат |
| `filename_prefix` | Префикс для сохраняемого имени файла. Счетчик и расширение файла будут добавлены автоматически. (по умолчанию: "3d/ComfyUI") | STRING | Да | Любая допустимая строка имени файла |
| `viewport_state` | Текущее состояние области просмотра, обычно из узла 3D-вьювера. | LOAD3D | Да | - |
| `model_3d_info` | Необязательная информация о 3D-модели для предварительного просмотра. | LOAD3DMODELINFO | Нет | - |
| `camera_info` | Необязательная информация о камере для предварительного просмотра. | LOAD3DCAMERA | Нет | - |
| `width` | Ширина превью в пикселях. (по умолчанию: 1024) | INT | Да | от 1 до 4096 |
| `height` | Высота превью в пикселях. (по умолчанию: 1024) | INT | Да | от 1 до 4096 |

## Выходные параметры

| Имя выхода | Описание | Тип данных |
|-------------|-------------|-----------|
| `model_3d` | Сохраненный файл 3D-модели. | FILE3D |
| `model_3d_info` | Информация о 3D-модели, переданная для нижестоящих узлов. | LOAD3DMODELINFO |
| `camera_info` | Информация о камере, переданная для нижестоящих узлов. | LOAD3DCAMERA |
| `width` | Значение ширины, переданное для нижестоящих узлов. | INT |
| `height` | Значение высоты, переданное для нижестоящих узлов. | INT |
