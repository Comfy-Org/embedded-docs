# Save3DAdvanced

Этот узел сохраняет 3D-модель в файл в выходной директории ComfyUI с расширенным контролем над размерами вывода и настройками камеры/области просмотра. Он также передаёт 3D-модель, информацию о модели, информацию о камере и размеры для последующих узлов.

## Входы

| Параметр | Описание | Тип данных | Обязательный | Диапазон |
|-----------|-------------|-----------|----------|-------|
| `model_3d` | Файл 3D-модели из вышестоящего 3D-узла. | FILE3D | Да | GLB, GLTF, FBX, OBJ, STL, USDZ, Любой |
| `filename_prefix` | Префикс для имени сохраняемого файла (по умолчанию: "3d/ComfyUI"). | STRING | Да | Произвольный текст |
| `viewport_state` | Состояние области просмотра из узла "Загрузить 3D", содержащее информацию о камере и модели. | LOAD3D | Да | - |
| `model_3d_info` | Необязательная информация о 3D-модели для переопределения состояния области просмотра. | LOAD3DMODELINFO | Нет | - |
| `camera_info` | Необязательная информация о камере для переопределения состояния области просмотра. | LOAD3DCAMERA | Нет | - |
| `width` | Ширина предварительного просмотра вывода в пикселях (по умолчанию: 1024). | INT | Да | от 1 до 4096 |
| `height` | Высота предварительного просмотра вывода в пикселях (по умолчанию: 1024). | INT | Да | от 1 до 4096 |

## Выходы

| Имя выхода | Описание | Тип данных |
|-------------|-------------|-----------|
| `model_3d` | Файл 3D-модели, переданный из входных данных. | FILE3D |
| `model_3d_info` | Информация о модели, либо из состояния области просмотра, либо из необязательного входного параметра. | LOAD3DMODELINFO |
| `camera_info` | Информация о камере, либо из состояния области просмотра, либо из необязательного входного параметра. | LOAD3DCAMERA |
| `width` | Значение ширины, переданное из входных данных. | INT |
| `height` | Значение высоты, переданное из входных данных. | INT |
