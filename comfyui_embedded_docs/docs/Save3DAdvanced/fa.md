# Save3DAdvanced

این گره یک مدل سه‌بعدی را در دایرکتوری خروجی ComfyUI شما ذخیره می‌کند و قابلیت‌های پیش‌نمایش پیشرفته را فراهم می‌آورد. به شما امکان می‌دهد نام فایل خروجی، ابعاد پیش‌نمایش را مشخص کنید و به‌صورت اختیاری اطلاعات دوربین و مدل را برای تجربه‌ای بهتر در نمایشگر سه‌بعدی ارسال کنید.

## ورودی‌ها

| پارامتر | توضیحات | نوع داده | الزامی | محدوده |
|---------|---------|----------|--------|--------|
| `model_3d` | فایل مدل سه‌بعدی از یک گره سه‌بعدی بالادستی. | FILE3D | بله | GLB، GLTF، FBX، OBJ، STL، USDZ یا هر فرمت سه‌بعدی |
| `filename_prefix` | پیشوند نام فایل ذخیره‌شده. یک شمارنده و پسوند فایل به‌طور خودکار اضافه می‌شود. (پیش‌فرض: "3d/ComfyUI") | STRING | بله | هر رشته نام فایل معتبر |
| `viewport_state` | وضعیت نمای دید فعلی، معمولاً از یک گره نمایشگر سه‌بعدی. | LOAD3D | بله | - |
| `model_3d_info` | اطلاعات اختیاری مدل سه‌بعدی برای پیش‌نمایش. | LOAD3DMODELINFO | خیر | - |
| `camera_info` | اطلاعات اختیاری دوربین برای پیش‌نمایش. | LOAD3DCAMERA | خیر | - |
| `width` | عرض پیش‌نمایش بر حسب پیکسل. (پیش‌فرض: 1024) | INT | بله | 1 تا 4096 |
| `height` | ارتفاع پیش‌نمایش بر حسب پیکسل. (پیش‌فرض: 1024) | INT | بله | 1 تا 4096 |

## خروجی‌ها

| نام خروجی | توضیحات | نوع داده |
|-----------|---------|----------|
| `model_3d` | فایل مدل سه‌بعدی ذخیره‌شده. | FILE3D |
| `model_3d_info` | اطلاعات مدل سه‌بعدی که برای گره‌های پایین‌دستی عبور داده می‌شود. | LOAD3DMODELINFO |
| `camera_info` | اطلاعات دوربین که برای گره‌های پایین‌دستی عبور داده می‌شود. | LOAD3DCAMERA |
| `width` | مقدار عرض که برای گره‌های پایین‌دستی عبور داده می‌شود. | INT |
| `height` | مقدار ارتفاع که برای گره‌های پایین‌دستی عبور داده می‌شود. | INT |

> این مستند با هوش مصنوعی تهیه شده است. اگر خطایی دیدید یا پیشنهادی برای بهبود دارید، خوشحال می‌شویم مشارکت کنید! [ویرایش در GitHub](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/Save3DAdvanced/fa.md)
