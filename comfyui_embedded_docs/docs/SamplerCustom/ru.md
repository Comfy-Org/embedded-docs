Узел SamplerCustom предназначен для обеспечения гибкого и настраиваемого механизма семплирования в различных приложениях. Он позволяет пользователям выбирать и настраивать различные стратегии семплирования в соответствии с их конкретными потребностями, повышая адаптивность и эффективность процесса.

## Входные параметры

| Параметр | Тип данных | Описание |
|-----------|--------------|-------------|
| `model`   | `MODEL`      | Входной параметр 'model' определяет модель, используемую для семплирования, играя ключевую роль в формировании поведения и результатов процесса. |
| `add_noise` | `BOOLEAN`    | Входной параметр 'add_noise' позволяет указать, следует ли добавлять шум в процесс семплирования, влияя на разнообразие и характеристики генерируемых образцов. |
| `noise_seed` | `INT`        | Входной параметр 'noise_seed' задает зерно для генерации шума, обеспечивая воспроизводимость и согласованность процесса семплирования при добавлении шума. |
| `cfg`     | `FLOAT`      | Входной параметр 'cfg' задает конфигурацию процесса семплирования, позволяя точно настраивать параметры и поведение семплирования. |
| `positive` | `CONDITIONING` | Входной параметр 'positive' представляет положительную информацию для кондиционирования, направляя процесс семплирования на генерацию образцов, соответствующих заданным положительным атрибутам. |
| `negative` | `CONDITIONING` | Входной параметр 'negative' представляет отрицательную информацию для кондиционирования, отводя процесс семплирования от генерации образцов, демонстрирующих заданные отрицательные атрибуты. |
| `sampler` | `SAMPLER`    | Входной параметр 'sampler' выбирает конкретную стратегию семплирования, напрямую влияя на характер и качество генерируемых образцов. |
| `sigmas`  | `SIGMAS`     | Входной параметр 'sigmas' определяет уровни шума, используемые в процессе семплирования, влияя на исследование пространства образцов и разнообразие выходных данных. |
| `latent_image` | `LATENT` | Входной параметр 'latent_image' предоставляет начальное латентное изображение для процесса семплирования, служа отправной точкой для генерации образцов. |

## Выходные параметры

| Параметр | Тип данных | Описание |
|-----------|--------------|-------------|
| `output`  | `LATENT`     | Выходной параметр 'output' представляет основной результат процесса семплирования, содержащий сгенерированные образцы. |
| `denoised_output` | `LATENT` | Выходной параметр 'denoised_output' представляет образцы после применения процесса шумоподавления, потенциально повышая четкость и качество сгенерированных образцов. |
