Вот перевод документации на русский язык:

Узел PerturbedAttentionGuidance применяет возмущённое управление вниманием (perturbed attention guidance) к диффузионной модели для повышения качества генерации. Он модифицирует механизм самовнимания модели во время сэмплирования, заменяя его упрощённой версией, которая фокусируется на проекциях значений. Этот метод помогает улучшить согласованность и качество генерируемых изображений за счёт корректировки процесса условного шумоподавления.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `модель` | MODEL | Да | - | Диффузионная модель, к которой применяется возмущённое управление вниманием |
| `масштаб` | FLOAT | Нет | 0.0 - 100.0 | Сила эффекта возмущённого управления вниманием (по умолчанию: 3.0). При значении 0 узел не оказывает эффекта и возвращает исходный результат шумоподавления. |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `модель` | MODEL | Модифицированная модель с применённым возмущённым управлением вниманием |
