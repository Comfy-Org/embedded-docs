Вот перевод документации на русский язык:

Узел SamplerEulerCFGpp предоставляет метод семплирования Euler CFG++ для генерации выходных данных. Этот узел предлагает две различные версии реализации сэмплера Euler CFG++, которые можно выбирать в зависимости от предпочтений пользователя.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `версия` | STRING | Да | `"regular"`<br>`"alternative"` | Версия реализации сэмплера Euler CFG++ для использования (по умолчанию: "regular") |

## Выходные данные

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `sampler` | SAMPLER | Возвращает настроенный экземпляр сэмплера Euler CFG++ |
