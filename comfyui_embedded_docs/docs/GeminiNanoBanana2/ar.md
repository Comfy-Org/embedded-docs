بالتأكيد، إليك الترجمة العربية للوثيقة التقنية لعقدة ComfyUI مع الالتزام التام بقواعد الترجمة المحددة:

عُقدة **GeminiNanoBanana2** تقوم بإنشاء أو تعديل الصور باستخدام نموذج Gemini من Google Vertex AI. تعمل عن طريق إرسال مطالبة نصية، بالإضافة إلى صور أو ملفات مرجعية اختيارية، إلى واجهة API، ثم تقوم بإرجاع الصورة المُنشأة وأي نص مصاحب لها.

## المدخلات

| المعامل | نوع البيانات | إجباري | النطاق | الوصف |
|-----------|-----------|----------|-------|-------------|
| `prompt` | STRING | نعم | لا ينطبق | مطالبة نصية تصف الصورة المراد إنشاؤها أو التعديلات المراد تطبيقها. قم بتضمين أي قيود أو أنماط أو تفاصيل يجب أن يتبعها النموذج. |
| `model` | COMBO | نعم | `"Nano Banana 2 (Gemini 3.1 Flash Image)"` | نموذج Gemini المحدد لاستخدامه في إنشاء الصور. |
| `seed` | INT | نعم | 0 إلى 18446744073709551615 | عندما يتم تثبيت البذرة على قيمة محددة، يبذل النموذج قصارى جهده لتقديم نفس الاستجابة للطلبات المتكررة. لا يُضمن الحصول على مخرجات حتمية. أيضًا، تغيير النموذج أو إعدادات المعاملات، مثل درجة الحرارة، قد يسبب اختلافات في الاستجابة حتى عند استخدام نفس قيمة البذرة. افتراضيًا، يتم استخدام قيمة بذرة عشوائية. (الافتراضي: 42) |
| `aspect_ratio` | COMBO | نعم | `"auto"`<br>`"1:1"`<br>`"2:3"`<br>`"3:2"`<br>`"3:4"`<br>`"4:3"`<br>`"4:5"`<br>`"5:4"`<br>`"9:16"`<br>`"16:9"`<br>`"21:9"` | إذا تم تعيينه على 'auto'، فإنه يطابق نسبة العرض إلى الارتفاع لصورة الإدخال الخاصة بك؛ إذا لم يتم توفير صورة، فعادةً ما يتم إنشاء مربع بنسبة 16:9. (الافتراضي: "auto") |
| `resolution` | COMBO | نعم | `"1K"`<br>`"2K"`<br>`"4K"` | دقة الإخراج المستهدفة. بالنسبة لـ 2K/4K، يتم استخدام أداة رفع الدقة الأصلية من Gemini. |
| `response_modalities` | COMBO | نعم | `"IMAGE"`<br>`"IMAGE+TEXT"` | يحدد نوع المحتوى الذي سيعيده النموذج. (متقدم) |
| `thinking_level` | COMBO | نعم | `"MINIMAL"`<br>`"HIGH"` | يتحكم في عمق عملية التفكير للنموذج. |
| `images` | IMAGE | لا | لا ينطبق | صورة (صور) مرجعية اختيارية. لتضمين صور متعددة، استخدم عقدة Batch Images (حتى 14 صورة). |
| `files` | CUSTOM | لا | لا ينطبق | ملف (ملفات) اختيارية لاستخدامها كسياق للنموذج. يقبل المدخلات من عقدة Gemini Generate Content Input Files. |
| `system_prompt` | STRING | لا | لا ينطبق | تعليمات أساسية تملي سلوك الذكاء الاصطناعي. (متقدم) |

**ملاحظة:** يدخل `images` يدعم حدًا أقصى يبلغ 14 صورة. إذا تم توفير أكثر من ذلك، ستقوم العقدة بإظهار خطأ.

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|-------------|-----------|-------------|
| `image` | IMAGE | الصورة الأساسية التي تم إنشاؤها أو تعديلها بواسطة النموذج. |
| `thought_image` | STRING | أي محتوى نصي يعيده النموذج. |
| `thought_image` | IMAGE | الصورة الأولى من عملية تفكير النموذج. متاحة فقط مع thinking_level HIGH وطريقة IMAGE+TEXT. |
