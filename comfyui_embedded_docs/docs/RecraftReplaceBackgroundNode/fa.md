### مرور کلی

جایگزینی پس‌زمینه تصویر بر اساس دستور متنی ارائه‌شده. این گره از API Recraft برای تولید پس‌زمینه‌های جدید مطابق با توضیحات متنی شما استفاده می‌کند و به شما امکان می‌دهد بدون تغییر سوژه اصلی، پس‌زمینه را کاملاً تغییر دهید.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|---------|----------|--------|--------|---------|
| `image` | IMAGE | بله | - | تصویر ورودی برای پردازش |
| `prompt` | STRING | بله | - | دستور متنی برای تولید تصویر (پیش‌فرض: خالی) |
| `n` | INT | بله | 1-6 | تعداد تصاویر تولیدی (پیش‌فرض: 1) |
| `seed` | INT | بله | 0-18446744073709551615 | دانه (seed) برای تعیین اجرای مجدد گره؛ نتایج واقعی بدون توجه به seed غیرقطعی هستند (پیش‌فرض: 0) |
| `recraft_style` | STYLEV3 | خیر | - | انتخاب سبک اختیاری برای پس‌زمینه تولیدی. در صورت عدم ارائه، به طور پیش‌فرض از سبک "realistic_image" استفاده می‌شود |
| `negative_prompt` | STRING | خیر | - | توضیحات متنی اختیاری از عناصر نامطلوب در تصویر (پیش‌فرض: خالی) |

**توجه:** پارامتر `seed` زمان اجرای مجدد گره را کنترل می‌کند اما به دلیل ماهیت API خارجی، نتایج قطعی را تضمین نمی‌کند.
