Вот перевод документации на русский язык:

Замена фона на изображении на основе заданного текстового описания. Этот узел использует API Recraft для генерации новых фонов для ваших изображений в соответствии с текстовым описанием, позволяя полностью изменить фон, сохраняя основной объект нетронутым.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `изображение` | IMAGE | Да | - | Входное изображение для обработки |
| `промпт` | STRING | Да | - | Описание для генерации изображения (по умолчанию: пусто) |
| `n` | INT | Да | 1-6 | Количество генерируемых изображений (по умолчанию: 1) |
| `seed` | INT | Да | 0-18446744073709551615 | Зерно для определения необходимости повторного запуска узла; фактические результаты недетерминированы независимо от зерна (по умолчанию: 0) |
| `recraft_style` | STYLEV3 | Нет | - | Необязательный выбор стиля для сгенерированного фона. Если не указан, по умолчанию используется стиль "realistic_image" |
| `негативный_промпт` | STRING | Нет | - | Необязательное текстовое описание нежелательных элементов на изображении (по умолчанию: пусто) |

**Примечание:** Параметр `seed` управляет моментом повторного выполнения узла, но не гарантирует детерминированных результатов из-за особенностей внешнего API.

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `IMAGE` | IMAGE | Сгенерированное изображение(я) с заменённым фоном |
