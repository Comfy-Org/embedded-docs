گره SaveAudioOpus داده‌های صوتی را در قالب فایل Opus ذخیره می‌کند. این گره ورودی صوتی را دریافت کرده و آن را به‌عنوان یک فایل Opus فشرده با تنظیمات کیفیت قابل‌تغییر خروجی می‌دهد. گره به‌طور خودکار نام‌گذاری فایل را انجام داده و خروجی را در دایرکتوری خروجی تعیین‌شده ذخیره می‌کند.

## ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|---------|----------|-------|--------|---------|
| `audio` | AUDIO | بله | - | داده‌های صوتی که باید به‌عنوان فایل Opus ذخیره شوند |
| `filename_prefix` | STRING | خیر | - | پیشوند نام فایل خروجی (پیش‌فرض: "audio/ComfyUI") |
| `quality` | COMBO | خیر | "64k"<br>"96k"<br>"128k"<br>"192k"<br>"320k" | تنظیم کیفیت صدا برای فایل Opus (پیش‌فرض: "128k") |
