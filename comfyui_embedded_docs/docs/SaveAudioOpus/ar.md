عقدة حفظ الصوت بصيغة Opus تقوم بحفظ بيانات صوتية في ملف بتنسيق Opus. تستقبل العقدة مدخلات صوتية وتصدرها كملف Opus مضغوط مع إعدادات جودة قابلة للتخصيص. تقوم العقدة تلقائيًا بمعالجة تسمية الملف وحفظ المخرجات في دليل الإخراج المحدد.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|---------|--------------|--------|--------|-------|
| `audio` | AUDIO | نعم | - | البيانات الصوتية المراد حفظها كملف Opus |
| `filename_prefix` | STRING | لا | - | بادئة اسم ملف الإخراج (الافتراضي: "audio/ComfyUI") |
| `quality` | COMBO | لا | "64k"<br>"96k"<br>"128k"<br>"192k"<br>"320k" | إعداد جودة الصوت لملف Opus (الافتراضي: "128k") |

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|------------|--------------|-------|
| - | - | لا تُرجع هذه العقدة أي قيم مخرجة. وظيفتها الأساسية هي حفظ ملف الصوت على القرص. |
