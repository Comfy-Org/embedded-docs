Вот перевод документации для узла SaveAudioOpus на русский язык:

Узел SaveAudioOpus сохраняет аудиоданные в файл формата Opus. Он принимает аудиовходные данные и экспортирует их в сжатый файл Opus с настраиваемыми параметрами качества. Узел автоматически обрабатывает именование файлов и сохраняет результат в указанную выходную директорию.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `audio` | AUDIO | Да | - | Аудиоданные, которые необходимо сохранить в файл Opus |
| `префикс_имени_файла` | STRING | Нет | - | Префикс для имени выходного файла (по умолчанию: "audio/ComfyUI") |
| `качество` | COMBO | Нет | "64k"<br>"96k"<br>"128k"<br>"192k"<br>"320k" | Настройка качества звука для файла Opus (по умолчанию: "128k") |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| - | - | Этот узел не возвращает никаких выходных значений. Его основная функция — сохранение аудиофайла на диск. |
