گره StringContains بررسی می‌کند که آیا یک رشته مشخص حاوی زیررشته‌ای معین است یا خیر. این عملیات را می‌توان با تطبیق حساس به بزرگی/کوچکی حروف یا بدون حساسیت به آن انجام داد و نتیجه‌ای بولی برمی‌گرداند که نشان می‌دهد زیررشته درون رشته اصلی یافت شده است یا نه.

## ورودی‌ها

| پارامتر | نوع داده | اجباری | محدوده | توضیحات |
|---------|----------|--------|--------|---------|
| `رشته` | STRING | بله | - | رشته متنی اصلی که باید در آن جستجو شود |
| `زیررشته` | STRING | بله | - | متنی که باید درون رشته اصلی جستجو شود |
| `حساس به حروف بزرگ و کوچک` | BOOLEAN | خیر | - | تعیین می‌کند که جستجو به بزرگی/کوچکی حروف حساس باشد (پیش‌فرض: true) |

## خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-----------|----------|---------|
| `contains` | BOOLEAN | اگر زیررشته در رشته پیدا شود true و در غیر این صورت false برمی‌گرداند |
