Узел StringContains проверяет, содержит ли заданная строка указанную подстроку. Проверка может выполняться с учетом или без учета регистра символов, возвращая логический результат, указывающий, была ли найдена подстрока в основной строке.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `string` | STRING | Да | - | Основная текстовая строка, в которой выполняется поиск |
| `substring` | STRING | Да | - | Текст для поиска в основной строке |
| `case_sensitive` | BOOLEAN | Нет | - | Определяет, следует ли учитывать регистр символов при поиске (по умолчанию: true) |

## Выходные данные

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `contains` | BOOLEAN | Возвращает true, если подстрока найдена в строке, и false в противном случае |
