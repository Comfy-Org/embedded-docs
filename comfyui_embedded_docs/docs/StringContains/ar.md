عقدة `StringContains` تتحقق مما إذا كانت سلسلة نصية معينة تحتوي على سلسلة فرعية محددة. يمكنها إجراء هذا التحقق إما بمطابقة حساسة لحالة الأحرف أو غير حساسة لها، مع إرجاع نتيجة منطقية تشير إلى ما إذا تم العثور على السلسلة الفرعية داخل السلسلة الرئيسية.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|-----------|-----------|----------|-------|-------------|
| `سلسلة نصية` | STRING | نعم | - | السلسلة النصية الرئيسية المراد البحث داخلها |
| `جزء نصي` | STRING | نعم | - | النص المراد البحث عنه داخل السلسلة الرئيسية |
| `حساس لحالة الأحرف` | BOOLEAN | لا | - | يحدد ما إذا كان البحث حساسًا لحالة الأحرف (القيمة الافتراضية: true) |

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|-------------|-----------|-------------|
| `contains` | BOOLEAN | يُرجع true إذا تم العثور على السلسلة الفرعية في السلسلة النصية، و false في الحالة المعاكسة |
