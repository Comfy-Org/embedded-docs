Вот перевод документации на русский язык:

Этот узел автоматически обнаруживает модели, расположенные в папке LoRA (включая вложенные папки), при этом соответствующий путь к модели — `ComfyUI\models\loras`. Для получения дополнительной информации обратитесь к разделу Установка моделей LoRA.

Узел LoRA Loader в основном используется для загрузки моделей LoRA. Модели LoRA можно рассматривать как фильтры, которые могут придавать вашим изображениям определённые стили, содержание и детали:

- Применять определённые художественные стили (например, тушь)
- Добавлять характеристики определённых персонажей (например, игровых персонажей)
- Добавлять определённые детали к изображению
Всё это можно реализовать с помощью LoRA.

Если вам нужно загрузить несколько моделей LoRA, вы можете напрямую соединить несколько узлов последовательно, как показано ниже:

## Входы

| Параметр | Тип данных | Описание |
| --- | --- | --- |
| `model` | MODEL | Обычно используется для подключения к базовой модели |
| `clip` | CLIP | Обычно используется для подключения к модели CLIP |
| `lora_name` | COMBO[STRING] | Выберите имя модели LoRA для использования |
| `strength_model` | FLOAT | Диапазон значений от -100.0 до 100.0, обычно используется в диапазоне 0~1 для повседневной генерации изображений. Более высокие значения приводят к более выраженному эффекту настройки модели |
| `strength_clip` | FLOAT | Диапазон значений от -100.0 до 100.0, обычно используется в диапазоне 0~1 для повседневной генерации изображений. Более высокие значения приводят к более выраженному эффекту настройки модели |

## Выходы

| Параметр | Тип данных | Описание |
| --- | --- | --- |
| `model` | MODEL | Модель с применёнными настройками LoRA |
| `clip` | CLIP | Экземпляр CLIP с применёнными настройками LoRA |
