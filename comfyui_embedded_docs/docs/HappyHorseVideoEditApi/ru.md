## Обзор

Редактирование видео с использованием текстовых инструкций или референсных изображений с помощью модели HappyHorse. Длительность выходного видео составляет 3–15 секунд и соответствует входному видео; входные данные длиннее 15 секунд обрезаются.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `model` | DICT | Да | См. ниже | Конфигурация модели, содержащая выбор модели, промпт, разрешение, соотношение сторон и опциональные референсные изображения. |
| `video` | VIDEO | Да | - | Видео для редактирования. |
| `seed` | INT | Да | от 0 до 2147483647 | Сид для генерации (по умолчанию: 0). |
| `watermark` | BOOLEAN | Нет | True / False | Добавлять ли AI-водяной знак к результату (по умолчанию: False). |

### Детали параметра `model`

Параметр `model` представляет собой словарь со следующими полями:

| Поле | Тип данных | Обязательный | Диапазон | Описание |
|-------|-----------|----------|-------|-------------|
| `model` | STRING | Да | `"happyhorse-1.0-video-edit"` | Модель редактирования видео HappyHorse для использования. |
| `prompt` | STRING | Да | - | Инструкции по редактированию или требования к переносу стиля. Должен содержать не менее 1 символа. |
| `resolution` | STRING | Да | `"720P"`<br>`"1080P"` | Выходное разрешение. |
| `ratio` | STRING | Да | `"16:9"`<br>`"9:16"`<br>`"1:1"`<br>`"4:3"`<br>`"3:4"` | Соотношение сторон. Если не изменено, аппроксимирует соотношение сторон входного видео. |
| `reference_images` | DICT | Нет | от 0 до 5 изображений | Опциональные референсные изображения (image1, image2, image3, image4, image5) для направления редактирования. |

## Выходные данные

| Имя выходного параметра | Тип данных | Описание |
|-------------|-----------|-------------|
| `video` | VIDEO | Выходное отредактированное видео. |
