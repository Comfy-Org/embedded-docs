## نمای کلی

با استفاده از مدل HappyHorse، یک ویدیو را با دستورات متنی یا تصاویر مرجع ویرایش کنید. مدت زمان خروجی ۳ تا ۱۵ ثانیه بوده و با ویدیوی ورودی مطابقت دارد؛ ورودی‌های طولانی‌تر از ۱۵ ثانیه کوتاه می‌شوند.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `مدل` | DICT | بله | زیر را ببینید | پیکربندی مدل شامل انتخاب مدل، فرمان، وضوح، نسبت تصویر و تصاویر مرجع اختیاری. |
| `ویدیو` | VIDEO | بله | - | ویدیویی که باید ویرایش شود. |
| `seed` | INT | بله | ۰ تا ۲۱۴۷۴۸۳۶۴۷ | دانه (seed) مورد استفاده برای تولید (پیش‌فرض: ۰). |
| `واترمارک` | BOOLEAN | خیر | True / False | آیا واترمارک تولیدشده توسط هوش مصنوعی به نتیجه اضافه شود (پیش‌فرض: False). |

### جزئیات پارامتر `model`

پارامتر `model` یک دیکشنری با فیلدهای زیر است:

| فیلد | نوع داده | الزامی | محدوده | توضیحات |
|-------|-----------|----------|-------|-------------|
| `مدل` | STRING | بله | `"happyhorse-1.0-video-edit"` | مدل ویرایش ویدیوی HappyHorse مورد استفاده. |
| `prompt` | STRING | بله | - | دستورالعمل‌های ویرایش یا الزامات انتقال سبک. باید حداقل ۱ کاراکتر داشته باشد. |
| `resolution` | STRING | بله | `"720P"`<br>`"1080P"` | وضوح خروجی. |
| `ratio` | STRING | بله | `"16:9"`<br>`"9:16"`<br>`"1:1"`<br>`"4:3"`<br>`"3:4"` | نسبت تصویر. در صورت عدم تغییر، نسبت به نسبت ویدیوی ورودی تقریب می‌زند. |
| `reference_images` | DICT | خیر | ۰ تا ۵ تصویر | تصاویر مرجع اختیاری (image1, image2, image3, image4, image5) برای هدایت ویرایش. |
