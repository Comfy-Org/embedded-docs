نظرة عامة

تحرير فيديو باستخدام تعليمات نصية أو صور مرجعية من خلال نموذج HappyHorse. مدة الإخراج تتراوح بين 3 و15 ثانية وتتطابق مع مدة الفيديو المُدخل؛ حيث يتم اقتطاع المُدخلات التي تزيد مدتها عن 15 ثانية.

المُدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|-----------|-----------|----------|-------|-------------|
| `النموذج` | DICT | نعم | انظر أدناه | إعدادات النموذج التي تحتوي على اختيار النموذج، والتعليمات النصية، والدقة، ونسبة العرض إلى الارتفاع، والصور المرجعية الاختيارية. |
| `الفيديو` | VIDEO | نعم | - | الفيديو المراد تحريره. |
| `البذرة` | INT | نعم | 0 إلى 2147483647 | القيمة الأولية المستخدمة في التوليد (القيمة الافتراضية: 0). |
| `علامة مائية` | BOOLEAN | لا | True / False | ما إذا كان سيتم إضافة علامة مائية منشأة بالذكاء الاصطناعي إلى النتيجة (القيمة الافتراضية: False). |

### تفاصيل معامل `model`

معامل `model` هو قاموس يحتوي على الحقول التالية:

| الحقل | نوع البيانات | إلزامي | النطاق | الوصف |
|-------|-----------|----------|-------|-------------|
| `النموذج` | STRING | نعم | `"happyhorse-1.0-video-edit"` | نموذج تحرير الفيديو HappyHorse المراد استخدامه. |
| `prompt` | STRING | نعم | - | تعليمات التحرير أو متطلبات نقل النمط. يجب أن يكون طوله حرفًا واحدًا على الأقل. |
| `resolution` | STRING | نعم | `"720P"`<br>`"1080P"` | دقة الإخراج. |
| `ratio` | STRING | نعم | `"16:9"`<br>`"9:16"`<br>`"1:1"`<br>`"4:3"`<br>`"3:4"` | نسبة العرض إلى الارتفاع. إذا لم يتم تغييرها، يتم تقريبها من نسبة الفيديو المُدخل. |
| `reference_images` | DICT | لا | 0 إلى 5 صور | صور مرجعية اختيارية (image1, image2, image3, image4, image5) لتوجيه عملية التحرير. |
