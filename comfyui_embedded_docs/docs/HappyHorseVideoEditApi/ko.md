## 개요

HappyHorse 모델을 사용하여 텍스트 명령어나 참조 이미지를 통해 비디오를 편집합니다. 출력 길이는 입력 비디오(3~15초)와 일치하며, 15초를 초과하는 입력은 자동으로 잘립니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `model` | DICT | 예 | 아래 참조 | 모델 선택, 프롬프트, 해상도, 화면 비율 및 선택적 참조 이미지를 포함하는 모델 구성입니다. |
| `video` | VIDEO | 예 | - | 편집할 비디오입니다. |
| `seed` | INT | 예 | 0 ~ 2147483647 | 생성에 사용할 시드입니다(기본값: 0). |
| `watermark` | BOOLEAN | 아니요 | True / False | 결과물에 AI 생성 워터마크를 추가할지 여부입니다(기본값: False). |

### `model` 매개변수 상세

`model` 매개변수는 다음 필드를 포함하는 딕셔너리입니다:

| 필드 | 데이터 타입 | 필수 여부 | 범위 | 설명 |
|-------|-----------|----------|-------|-------------|
| `model` | STRING | 예 | `"happyhorse-1.0-video-edit"` | 사용할 HappyHorse 비디오 편집 모델입니다. |
| `prompt` | STRING | 예 | - | 편집 지침 또는 스타일 전환 요구사항입니다. 최소 1자 이상이어야 합니다. |
| `resolution` | STRING | 예 | `"720P"`<br>`"1080P"` | 출력 해상도입니다. |
| `ratio` | STRING | 예 | `"16:9"`<br>`"9:16"`<br>`"1:1"`<br>`"4:3"`<br>`"3:4"` | 화면 비율입니다. 변경하지 않을 경우 입력 비디오 비율에 근사합니다. |
| `reference_images` | DICT | 아니요 | 0 ~ 5개 이미지 | 편집을 안내하는 선택적 참조 이미지(image1, image2, image3, image4, image5)입니다. |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `video` | VIDEO | 편집된 비디오 출력입니다. |
