Вот перевод документации на русский язык:

Узел StringSubstring извлекает часть текста из более длинной строки. Он принимает начальную и конечную позиции, чтобы определить участок, который вы хотите извлечь, а затем возвращает текст между этими двумя позициями.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `строка` | STRING | Да | - | Входная текстовая строка, из которой производится извлечение. Поддерживает многострочный текст. |
| `начало` | INT | Да | - | Индекс начальной позиции для подстроки. Первый символ имеет индекс 0. |
| `конец` | INT | Да | - | Индекс конечной позиции для подстроки. Символ на этой позиции не включается в результат. |

## Выходные данные

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `output` | STRING | Извлечённая подстрока из входного текста, содержащая все символы от позиции `начало` до (не включая) позиции `конец`. |
