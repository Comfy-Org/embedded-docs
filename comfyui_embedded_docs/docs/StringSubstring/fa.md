گره StringSubstring بخشی از متن را از یک رشته بزرگ‌تر استخراج می‌کند. این گره با دریافت موقعیت شروع و پایان، بخش مورد نظر شما را تعریف کرده و متن بین آن دو موقعیت را بازمی‌گرداند.

## ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|---------|----------|-------|--------|---------|
| `رشته` | STRING | بله | - | رشته متنی ورودی برای استخراج. از متن چندخطی پشتیبانی می‌کند. |
| `شروع` | INT | بله | - | اندیس موقعیت شروع برای زیررشته. اولین کاراکتر در اندیس 0 قرار دارد. |
| `پایان` | INT | بله | - | اندیس موقعیت پایان برای زیررشته. کاراکتر در این اندیس در نتیجه نهایی گنجانده نمی‌شود. |

## خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-----------|----------|---------|
| `output` | STRING | زیررشته استخراج‌شده از متن ورودی، شامل تمام کاراکترهای موقعیت `شروع` تا (اما نه شامل) موقعیت `پایان`. |
