Узел «Изменение размера изображения/маски» предоставляет несколько способов изменения размеров входного изображения или маски. Он может масштабировать с помощью множителя, задавать конкретные размеры, подгонять под размер другого входного элемента или изменять на основе количества пикселей, используя различные методы интерполяции для обеспечения качества.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `input` | IMAGE или MASK | Да | Н/Д | Изображение или маска, размер которых необходимо изменить. |
| `resize_type` | COMBO | Да | `SCALE_BY`<br>`SCALE_DIMENSIONS`<br>`SCALE_LONGER_DIMENSION`<br>`SCALE_SHORTER_DIMENSION`<br>`SCALE_WIDTH`<br>`SCALE_HEIGHT`<br>`SCALE_TOTAL_PIXELS`<br>`MATCH_SIZE` | Метод, используемый для определения нового размера. Требуемые параметры меняются в зависимости от выбранного типа. |
| `multiplier` | FLOAT | Нет | от 0.01 до 8.0 | Коэффициент масштабирования. Требуется, когда `resize_type` имеет значение `SCALE_BY` (по умолчанию: 1.00). |
| `width` | INT | Нет | от 0 до 8192 | Целевая ширина в пикселях. Требуется, когда `resize_type` имеет значение `SCALE_DIMENSIONS` или `SCALE_WIDTH` (по умолчанию: 512). |
| `height` | INT | Нет | от 0 до 8192 | Целевая высота в пикселях. Требуется, когда `resize_type` имеет значение `SCALE_DIMENSIONS` или `SCALE_HEIGHT` (по умолчанию: 512). |
| `crop` | COMBO | Нет | `"disabled"`<br>`"center"` | Метод кадрирования, применяемый, когда размеры не соответствуют соотношению сторон. Доступен только когда `resize_type` имеет значение `SCALE_DIMENSIONS` или `MATCH_SIZE` (по умолчанию: "center"). |
| `longer_size` | INT | Нет | от 0 до 8192 | Целевой размер для длинной стороны изображения. Требуется, когда `resize_type` имеет значение `SCALE_LONGER_DIMENSION` (по умолчанию: 512). |
| `shorter_size` | INT | Нет | от 0 до 8192 | Целевой размер для короткой стороны изображения. Требуется, когда `resize_type` имеет значение `SCALE_SHORTER_DIMENSION` (по умолчанию: 512). |
| `megapixels` | FLOAT | Нет | от 0.01 до 16.0 | Целевое общее количество мегапикселей. Требуется, когда `resize_type` имеет значение `SCALE_TOTAL_PIXELS` (по умолчанию: 1.0). |
| `match` | IMAGE или MASK | Нет | Н/Д | Изображение или маска, под размер которых будет изменен входной элемент. Требуется, когда `resize_type` имеет значение `MATCH_SIZE`. |
| `scale_method` | COMBO | Да | `"nearest-exact"`<br>`"bilinear"`<br>`"area"`<br>`"bicubic"`<br>`"lanczos"` | Алгоритм интерполяции, используемый для масштабирования (по умолчанию: "area"). |

**Примечание:** Параметр `crop` доступен и актуален только когда для `resize_type` установлено значение `SCALE_DIMENSIONS` или `MATCH_SIZE`. При использовании `SCALE_WIDTH` или `SCALE_HEIGHT` другое измерение автоматически масштабируется для сохранения исходного соотношения сторон.

## Выходные параметры

| Имя выходного параметра | Тип данных | Описание |
|-------------|-----------|-------------|
| `resized` | IMAGE или MASK | Измененное изображение или маска, соответствующие типу данных входного элемента. |
