گره تغییر اندازه تصویر/ماسک روش‌های متعددی برای تغییر ابعاد یک تصویر یا ماسک ورودی ارائه می‌دهد. این گره می‌تواند با استفاده از ضریب مقیاس‌دهی، ابعاد مشخص، هم‌اندازه‌سازی با ورودی دیگر یا تنظیم بر اساس تعداد پیکسل‌ها، با بهره‌گیری از روش‌های درون‌یابی مختلف برای کیفیت، اندازه را تغییر دهد.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|---------|----------|--------|--------|---------|
| `input` | IMAGE یا MASK | بله | نامعتبر | تصویر یا ماسکی که باید تغییر اندازه دهد. |
| `resize_type` | COMBO | بله | `SCALE_BY`<br>`SCALE_DIMENSIONS`<br>`SCALE_LONGER_DIMENSION`<br>`SCALE_SHORTER_DIMENSION`<br>`SCALE_WIDTH`<br>`SCALE_HEIGHT`<br>`SCALE_TOTAL_PIXELS`<br>`MATCH_SIZE` | روش تعیین اندازه جدید. پارامترهای مورد نیاز بر اساس نوع انتخاب‌شده تغییر می‌کنند. |
| `multiplier` | FLOAT | خیر | 0.01 تا 8.0 | ضریب مقیاس‌دهی. هنگامی که `resize_type` برابر `SCALE_BY` باشد الزامی است (پیش‌فرض: 1.00). |
| `width` | INT | خیر | 0 تا 8192 | عرض هدف بر حسب پیکسل. هنگامی که `resize_type` برابر `SCALE_DIMENSIONS` یا `SCALE_WIDTH` باشد الزامی است (پیش‌فرض: 512). |
| `height` | INT | خیر | 0 تا 8192 | ارتفاع هدف بر حسب پیکسل. هنگامی که `resize_type` برابر `SCALE_DIMENSIONS` یا `SCALE_HEIGHT` باشد الزامی است (پیش‌فرض: 512). |
| `crop` | COMBO | خیر | `"disabled"`<br>`"center"` | روش برش‌دهی زمانی که ابعاد با نسبت تصویر هماهنگ نیستند. فقط هنگامی که `resize_type` برابر `SCALE_DIMENSIONS` یا `MATCH_SIZE` باشد در دسترس است (پیش‌فرض: "center"). |
| `longer_size` | INT | خیر | 0 تا 8192 | اندازه هدف برای ضلع بلندتر تصویر. هنگامی که `resize_type` برابر `SCALE_LONGER_DIMENSION` باشد الزامی است (پیش‌فرض: 512). |
| `shorter_size` | INT | خیر | 0 تا 8192 | اندازه هدف برای ضلع کوتاه‌تر تصویر. هنگامی که `resize_type` برابر `SCALE_SHORTER_DIMENSION` باشد الزامی است (پیش‌فرض: 512). |
| `megapixels` | FLOAT | خیر | 0.01 تا 16.0 | تعداد کل مگاپیکسل هدف. هنگامی که `resize_type` برابر `SCALE_TOTAL_PIXELS` باشد الزامی است (پیش‌فرض: 1.0). |
| `match` | IMAGE یا MASK | خیر | نامعتبر | تصویر یا ماسکی که ابعاد ورودی برای هم‌اندازه‌سازی با آن تغییر می‌کند. هنگامی که `resize_type` برابر `MATCH_SIZE` باشد الزامی است. |
| `scale_method` | COMBO | بله | `"nearest-exact"`<br>`"bilinear"`<br>`"area"`<br>`"bicubic"`<br>`"lanczos"` | الگوریتم درون‌یابی مورد استفاده برای مقیاس‌دهی (پیش‌فرض: "area"). |

**توجه:** پارامتر `crop` فقط زمانی در دسترس و مرتبط است که `resize_type` روی `SCALE_DIMENSIONS` یا `MATCH_SIZE` تنظیم شده باشد. هنگام استفاده از `SCALE_WIDTH` یا `SCALE_HEIGHT`، بعد دیگر به‌طور خودکار برای حفظ نسبت تصویر اصلی مقیاس‌دهی می‌شود.
