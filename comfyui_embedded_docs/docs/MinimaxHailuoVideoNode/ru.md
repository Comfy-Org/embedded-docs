Вот перевод документации на русский язык:

Генерирует видео из текстовых подсказок с использованием модели MiniMax Hailuo-02. Вы можете дополнительно указать начальное изображение в качестве первого кадра, чтобы создать видео, продолжающееся с этого изображения.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `текстовый_промпт` | STRING | Да | - | Текстовая подсказка для управления генерацией видео. |
| `сид` | INT | Нет | от 0 до 18446744073709551615 | Случайное зерно, используемое для создания шума (по умолчанию: 0). |
| `изображение_первого_кадра` | IMAGE | Нет | - | Необязательное изображение, используемое в качестве первого кадра для генерации видео. |
| `оптимизатор_промпта` | BOOLEAN | Нет | - | Оптимизировать подсказку для улучшения качества генерации при необходимости (по умолчанию: True). |
| `длительность` | COMBO | Нет | `6`<br>`10` | Длина выходного видео в секундах (по умолчанию: 6). |
| `разрешение` | COMBO | Нет | `"768P"`<br>`"1080P"` | Размеры отображения видео. 1080p — 1920x1080, 768p — 1366x768 (по умолчанию: "768P"). |

**Примечание:** При использовании модели MiniMax-Hailuo-02 с разрешением 1080P длительность ограничена 6 секундами.

## Выходные параметры

| Имя выходного параметра | Тип данных | Описание |
|-------------|-----------|-------------|
| `output` | VIDEO | Сгенерированный видеофайл. |
