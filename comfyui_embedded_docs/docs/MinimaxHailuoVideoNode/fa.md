### نمای کلی

با استفاده از مدل MiniMax Hailuo-02، ویدیوهایی از توضیحات متنی تولید می‌کند. می‌توانید به‌صورت اختیاری یک تصویر شروع به عنوان فریم اول ارائه دهید تا ویدیویی ایجاد کنید که از آن تصویر ادامه یابد.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `prompt_text` | STRING | بله | - | توضیح متنی برای هدایت تولید ویدیو. |
| `seed` | INT | خیر | 0 تا 18446744073709551615 | دانه تصادفی مورد استفاده برای ایجاد نویز (پیش‌فرض: 0). |
| `first_frame_image` | IMAGE | خیر | - | تصویر اختیاری برای استفاده به عنوان فریم اول جهت تولید ویدیو. |
| `prompt_optimizer` | BOOLEAN | خیر | - | بهینه‌سازی توضیح برای بهبود کیفیت تولید در صورت نیاز (پیش‌فرض: True). |
| `duration` | COMBO | خیر | `6`<br>`10` | طول ویدیوی خروجی بر حسب ثانیه (پیش‌فرض: 6). |
| `resolution` | COMBO | خیر | `"768P"`<br>`"1080P"` | ابعاد نمایش ویدیو. 1080p معادل 1920x1080 و 768p معادل 1366x768 است (پیش‌فرض: "768P"). |

**نکته:** هنگام استفاده از مدل MiniMax-Hailuo-02 با وضوح 1080P، مدت زمان ویدیو به 6 ثانیه محدود می‌شود.
