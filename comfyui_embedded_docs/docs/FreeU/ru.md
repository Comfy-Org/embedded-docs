Вот перевод документации узла FreeU на русский язык:

Узел FreeU применяет частотные модификации к выходным блокам модели для улучшения качества генерации изображений. Он работает путем масштабирования различных групп каналов и применения фильтрации Фурье к определенным картам признаков, что позволяет точно настраивать поведение модели в процессе генерации.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `model` | MODEL | Да | - | Модель, к которой применяются модификации FreeU |
| `b1` | FLOAT | Да | 0.0 - 10.0 | Коэффициент масштабирования магистрали для признаков model_channels × 4 (по умолчанию: 1.1) |
| `b2` | FLOAT | Да | 0.0 - 10.0 | Коэффициент масштабирования магистрали для признаков model_channels × 2 (по умолчанию: 1.2) |
| `s1` | FLOAT | Да | 0.0 - 10.0 | Коэффициент масштабирования пропускающего соединения для признаков model_channels × 4 (по умолчанию: 0.9) |
| `s2` | FLOAT | Да | 0.0 - 10.0 | Коэффициент масштабирования пропускающего соединения для признаков model_channels × 2 (по умолчанию: 0.2) |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `model` | MODEL | Модифицированная модель с примененными патчами FreeU |
