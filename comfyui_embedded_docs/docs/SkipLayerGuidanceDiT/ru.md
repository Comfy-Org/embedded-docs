Усиливает направление к детализированной структуре, используя дополнительный набор CFG-негативов с пропущенными слоями. Эта универсальная версия SkipLayerGuidance может применяться к любой модели DiT и вдохновлена методом Perturbed Attention Guidance. Оригинальная экспериментальная реализация была создана для SD3.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `model` | MODEL | Да | - | Модель, к которой применяется направление с пропуском слоёв |
| `double_layers` | STRING | Да | - | Номера слоёв для двойных блоков, разделённые запятыми (по умолчанию: "7, 8, 9") |
| `single_layers` | STRING | Да | - | Номера слоёв для одиночных блоков, разделённые запятыми (по умолчанию: "7, 8, 9") |
| `scale` | FLOAT | Да | 0.0 - 10.0 | Коэффициент масштабирования направления (по умолчанию: 3.0) |
| `start_percent` | FLOAT | Да | 0.0 - 1.0 | Начальный процент применения направления (по умолчанию: 0.01) |
| `end_percent` | FLOAT | Да | 0.0 - 1.0 | Конечный процент применения направления (по умолчанию: 0.15) |
| `rescaling_scale` | FLOAT | Да | 0.0 - 10.0 | Коэффициент перемасштабирования для корректировки выходной амплитуды (по умолчанию: 0.0, что означает отсутствие перемасштабирования) |

**Примечание:** Если оба параметра `double_layers` и `single_layers` пусты (не содержат номеров слоёв), узел возвращает исходную модель без применения какого-либо направления.

## Выходные данные

| Имя выходного параметра | Тип данных | Описание |
|-------------------------|------------|----------|
| `model` | MODEL | Модифицированная модель с применённым направлением с пропуском слоёв |
