# ترجمة وثيقة عقدة Rodin3D_Gen2

تقوم عقدة Rodin3D_Gen2 بإنشاء أصول ثلاثية الأبعاد باستخدام واجهة برمجة تطبيقات Rodin. تستقبل صورًا مدخلة وتحولها إلى نماذج ثلاثية الأبعاد بأنواع مواد وعدد مضلعات متنوعة. تدير العقدة عملية التوليد بأكملها بما في ذلك إنشاء المهمة، والاستعلام عن الحالة، وتنزيل الملفات تلقائيًا.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|----------|---------------|--------|---------|-------|
| `Images` | IMAGE | نعم | - | الصور المدخلة المستخدمة لتوليد النموذج ثلاثي الأبعاد |
| `Seed` | INT | لا | 0-65535 | قيمة البذرة العشوائية للتوليد (الافتراضي: 0) |
| `Material_Type` | COMBO | لا | "PBR"<br>"Shaded" | نوع المادة المراد تطبيقها على النموذج ثلاثي الأبعاد (الافتراضي: "PBR") |
| `Polygon_count` | COMBO | لا | "4K-Quad"<br>"8K-Quad"<br>"18K-Quad"<br>"50K-Quad"<br>"2K-Triangle"<br>"20K-Triangle"<br>"150K-Triangle"<br>"500K-Triangle" | عدد المضلعات المستهدف للنموذج ثلاثي الأبعاد المُنشأ (الافتراضي: "500K-Triangle") |
| `TAPose` | BOOLEAN | لا | - | ما إذا كان سيتم تطبيق معالجة TAPose (الافتراضي: False) |

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|-------------|---------------|-------|
| `3D Model Path` | STRING | مسار الملف للنموذج ثلاثي الأبعاد المُنشأ (للتوافق مع الإصدارات السابقة) |
| `GLB` | FILE3DGLB | النموذج ثلاثي الأبعاد المُنشأ بتنسيق GLB |
