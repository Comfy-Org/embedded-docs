### مرور کلی

گره Rodin3D_Gen2 با استفاده از API رادین، دارایی‌های سه‌بعدی تولید می‌کند. این گره تصاویر ورودی را دریافت کرده و با انواع متریال و تعداد چندضلعی‌های مختلف، به مدل‌های سه‌بعدی تبدیل می‌کند. این گره به‌طور خودکار کل فرآیند تولید شامل ایجاد وظیفه، بررسی وضعیت و دانلود فایل را مدیریت می‌کند.

## ورودی‌ها

| پارامتر | نوع داده | اجباری | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `Images` | IMAGE | بله | - | تصاویر ورودی برای تولید مدل سه‌بعدی |
| `Seed` | INT | خیر | 0-65535 | مقدار دانه تصادفی برای تولید (پیش‌فرض: 0) |
| `Material_Type` | COMBO | خیر | "PBR"<br>"Shaded" | نوع متریال اعمال‌شده به مدل سه‌بعدی (پیش‌فرض: "PBR") |
| `Polygon_count` | COMBO | خیر | "4K-Quad"<br>"8K-Quad"<br>"18K-Quad"<br>"50K-Quad"<br>"2K-Triangle"<br>"20K-Triangle"<br>"150K-Triangle"<br>"500K-Triangle" | تعداد چندضلعی هدف برای مدل سه‌بعدی تولیدشده (پیش‌فرض: "500K-Triangle") |
| `TAPose` | BOOLEAN | خیر | - | آیا پردازش TAPose اعمال شود (پیش‌فرض: False) |

## خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-------------|-----------|-------------|
| `3D Model Path` | STRING | مسیر فایل مدل سه‌بعدی تولیدشده (برای سازگاری با نسخه‌های قبلی) |
| `GLB` | FILE3DGLB | مدل سه‌بعدی تولیدشده در قالب GLB |
