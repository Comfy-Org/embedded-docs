Вот перевод документации на русский язык:

Узел Kling Single Image Video Effect создает видео с различными спецэффектами на основе одного референсного изображения. Он применяет различные визуальные эффекты и сцены для преобразования статичных изображений в динамический видеоконтент. Узел поддерживает разные сцены эффектов, опции моделей и длительность видео для достижения желаемого визуального результата.

## Входные данные

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `image` | IMAGE | Да | - | Референсное изображение. URL или строка в кодировке Base64 (без префикса data:image). Размер файла не должен превышать 10 МБ, разрешение не менее 300x300 пикселей, соотношение сторон от 1:2,5 до 2,5:1 |
| `effect_scene` | COMBO | Да | `"dizzydizzy"`<br>`"bloombloom"`<br>`"neon"`<br>`"cartoon"`<br>`"sketch"`<br>`"oil"`<br>`"watercolor"`<br>`"3d"` | Тип сцены спецэффекта, применяемого к генерации видео. Некоторые эффекты могут иметь разную стоимость. |
| `model_name` | COMBO | Да | `"kling-v1-5"`<br>`"kling-v1-6"` | Конкретная версия модели, используемая для генерации видеоэффекта. |
| `duration` | COMBO | Да | `"5"`<br>`"10"` | Длительность генерируемого видео в секундах. |

**Примечание:** Параметр `effect_scene` влияет на стоимость использования узла. Эффекты `dizzydizzy` и `bloombloom` стоят $0,49 USD за генерацию, в то время как все остальные эффекты стоят $0,28 USD за генерацию.

## Выходные данные

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `output` | VIDEO | Сгенерированное видео с примененными эффектами |
| `video_id` | STRING | Уникальный идентификатор сгенерированного видео |
| `duration` | STRING | Длительность сгенерированного видео |
