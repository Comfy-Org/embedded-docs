## Обзор

Генерирует текстовые ответы от модели Anthropic Claude. Этот узел отправляет текстовый запрос и опциональные изображения модели Claude и возвращает сгенерированный текстовый ответ.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `prompt` | STRING | Да | Н/Д | Текстовый ввод для модели. (по умолчанию: пустая строка) |
| `model` | COMBO | Да | `"Opus 4.7"`<br>`"Opus 4.6"`<br>`"Sonnet 4.6"`<br>`"Sonnet 4.5"`<br>`"Haiku 4.5"` | Модель Claude, используемая для генерации ответа. |
| `seed` | INT | Да | от 0 до 2147483647 | Сид управляет повторным запуском узла; результаты недетерминированы независимо от сида. (по умолчанию: 0) |
| `images` | IMAGE | Нет | от 0 до 20 изображений | Опциональные изображения для использования в качестве контекста модели. До 20 изображений. |
| `system_prompt` | STRING | Нет | Н/Д | Базовые инструкции, определяющие поведение модели. (по умолчанию: пустая строка) |

### Ограничения параметров

- **Лимит изображений**: Максимум 20 изображений на один запрос.
- **Управление температурой**: Когда включено мышление или используется модель "Opus 4.7", параметр температуры автоматически сбрасывается (по умолчанию 1.0) в соответствии с требованиями API Anthropic. Для других моделей температуру можно задать через конфигурацию модели.
- **Мышление/Рассуждение**: Конфигурация модели включает параметр `reasoning_effort`, управляющий включением мышления. При включении узел автоматически настраивает соответствующий режим мышления (адаптивный или с бюджетом) в зависимости от выбранной модели.

## Выходные данные

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `output` | STRING | Сгенерированный текстовый ответ от модели Claude. Возвращает "Пустой ответ от модели Claude.", если текст не был сгенерирован. |
