## نمای کلی

متن پاسخ را از یک مدل Anthropic Claude تولید کنید. این گره یک متن درخواست و تصاویر اختیاری را به مدل Claude ارسال کرده و متن پاسخ تولید شده را بازمی‌گرداند.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `پرامپت` | STRING | بله | نامشخص | ورودی متن به مدل. (پیش‌فرض: رشته خالی) |
| `مدل` | COMBO | بله | `"Opus 4.7"`<br>`"Opus 4.6"`<br>`"Sonnet 4.6"`<br>`"Sonnet 4.5"`<br>`"Haiku 4.5"` | مدل Claude مورد استفاده برای تولید پاسخ. |
| `seed` | INT | بله | 0 تا 2147483647 | دانه (Seed) مشخص می‌کند که آیا گره باید دوباره اجرا شود؛ نتایج بدون توجه به دانه، قطعی نیستند. (پیش‌فرض: 0) |
| `تصاویر` | IMAGE | خیر | 0 تا 20 تصویر | تصویر(های) اختیاری برای استفاده به عنوان زمینه مدل. حداکثر 20 تصویر. |
| `system_prompt` | STRING | خیر | نامشخص | دستورالعمل‌های بنیادین که رفتار مدل را تعیین می‌کنند. (پیش‌فرض: رشته خالی) |

### محدودیت‌های پارامتر

- **محدودیت تصویر**: حداکثر 20 تصویر می‌تواند در هر درخواست ارائه شود.
- **مدیریت دما**: هنگامی که تفکر فعال است یا هنگام استفاده از مدل "Opus 4.7"، پارامتر دما به طور خودکار تنظیم نشده (پیش‌فرض 1.0) مطابق با الزامات API Anthropic باقی می‌ماند. برای سایر مدل‌ها، دما از طریق پیکربندی مدل قابل تنظیم است.
- **تفکر/استدلال**: پیکربندی مدل شامل یک تنظیم `reasoning_effort` است که تعیین می‌کند آیا تفکر فعال است یا خیر. هنگامی که فعال باشد، گره به طور خودکار حالت تفکر مناسب (تطبیقی یا مبتنی بر بودجه) را بسته به مدل انتخاب شده پیکربندی می‌کند.
