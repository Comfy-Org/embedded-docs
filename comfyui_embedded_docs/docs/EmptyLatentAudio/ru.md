Вот перевод документации на русский язык:

Узел EmptyLatentAudio создает пустой латентный тензор для обработки аудио. Он генерирует пустое аудио-латентное представление с заданной длительностью и размером пакета, которое может использоваться в качестве отправной точки для рабочих процессов генерации или обработки аудио. Узел автоматически вычисляет соответствующие размерности латентного пространства на основе длительности аудио и частоты дискретизации.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `seconds` | FLOAT | Да | 1.0 - 1000.0 | Длительность аудио в секундах (по умолчанию: 47.6) |
| `batch_size` | INT | Да | 1 - 4096 | Количество латентных изображений в пакете (по умолчанию: 1) |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `LATENT` | LATENT | Возвращает пустой латентный тензор для обработки аудио с указанной длительностью и размером пакета. Тензор имеет форму [batch_size, 64, length], где length вычисляется на основе длительности аудио и частоты дискретизации. |
