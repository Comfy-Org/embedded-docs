# ByteDanceSeedAudio

Генерируйте речь, музыку, звуковые эффекты и диалоги нескольких говорящих из одного запроса с помощью ByteDance Seed Audio 1.0. Опишите голос(а), эмоции, атмосферу, фоновую музыку и звуковые эффекты в запросе, а также включите реплики для озвучивания. При желании выберите встроенный голос из набора, клонируйте голоса из 1–3 эталонных клипов (помеченных как @Audio1-3 в запросе) или создайте голос на основе изображения персонажа. До 2 минут аудио за один запуск.

## Входные данные

| Параметр | Описание | Тип данных | Обязательный | Диапазон |
|----------|----------|------------|--------------|----------|
| `text_prompt` | Опишите голос(а), эмоции, темп, атмосферу, фоновую музыку и звуковые эффекты, а также включите реплики для озвучивания (для диалогов указывайте имена персонажей в тексте). В режиме «аудиореференс» ссылайтесь на подключенные клики по порядку как @Audio1, @Audio2, @Audio3. Максимум 3000 символов. | STRING | Да | от 1 до 3000 символов |
| `reference_mode` | Способ задания голоса: «только текст» (всё описание в запросе), «аудиореференс» (клонирование до 3 голосов, помеченных @Audio1-3), «референс по изображению» (создание голоса по изображению одного персонажа) или «голос из набора» (выбор встроенного именованного голоса, который озвучит запрос). | COMBO | Да | `"text only"`<br>`"audio reference"`<br>`"image reference"`<br>`"preset voice"` |
| `reference_audio_1` | Эталонный клип для клонирования голоса, помеченный @Audio1 в запросе. До 30 с. Доступен только при `reference_mode` = "audio reference". | AUDIO | Нет | до 30 секунд |
| `reference_audio_2` | Эталонный клип, помеченный @Audio2 в запросе. До 30 с. Доступен только при `reference_mode` = "audio reference". | AUDIO | Нет | до 30 секунд |
| `reference_audio_3` | Эталонный клип, помеченный @Audio3 в запросе. До 30 с. Доступен только при `reference_mode` = "audio reference". | AUDIO | Нет | до 30 секунд |
| `reference_image` | Изображение одного персонажа; модель создаёт голос на его основе. Нельзя комбинировать с эталонным аудио. Доступен только при `reference_mode` = "image reference". | IMAGE | Нет | - |
| `preset_voice` | Встроенный голос TTS 2.0, который озвучивает запрос. Эталонный клип не требуется, теги @AudioN в этом режиме не используются. Доступен только при `reference_mode` = "preset voice". | COMBO | Нет | Доступно несколько вариантов (см. описание) |
| `sample_rate` | Частота дискретизации выходного аудио в Гц. (по умолчанию: "24000") | COMBO | Да | `"8000"`<br>`"16000"`<br>`"24000"`<br>`"32000"`<br>`"44100"`<br>`"48000"` |
| `speech_rate` | Скорость речи. 0 = нормально, 100 = 2.0x, -50 = 0.5x. (по умолчанию: 0) | INT | Да | от -50 до 100 |
| `loudness_rate` | Громкость. 0 = нормально, 100 = 2.0x, -50 = 0.5x. (по умолчанию: 0) | INT | Да | от -50 до 100 |
| `pitch_rate` | Сдвиг высоты тона в полутонах (от -12 до 12). (по умолчанию: 0) | INT | Да | от -12 до 12 |
| `seed` | Seed управляет тем, будет ли узел запускаться заново; результаты недетерминированы независимо от seed. (по умолчанию: 42) | INT | Да | от 0 до 2147483647 |

### Ограничения параметров

- **Зависимости от режима референса**: Параметр `reference_mode` определяет, какие дополнительные входные данные обязательны:
  - **"text only"**: Дополнительные входные данные не требуются. Запрос не должен содержать теги @AudioN.
  - **"audio reference"**: Требуется подключение хотя бы одного из `reference_audio_1`, `reference_audio_2` или `reference_audio_3`. Эталонные клипы должны подключаться по порядку без пропусков (например, _1, затем _2, затем _3). Длительность каждого клипа ограничена 30 секундами. Запрос должен ссылаться на подключенные клипы с помощью тегов @Audio1, @Audio2, @Audio3.
  - **"image reference"**: Требуется подключение `reference_image`. Запрос не должен содержать теги @AudioN.
  - **"preset voice"**: Требуется выбор `preset_voice`. Запрос не должен содержать теги @AudioN (весь запрос озвучивается выбранным голосом).

- **Порядок аудиореференсов**: При использовании режима "audio reference" входные аудиоданные референсов должны подключаться последовательно, начиная с `reference_audio_1`, без пропусков. Например, можно подключить _1 и _2, но нельзя подключить _1 и _3 без _2.

- **Максимальное количество аудиотегов**: В режиме "audio reference" запрос может ссылаться на 1–3 аудиоклипа (@Audio1, @Audio2, @Audio3). Номер самого старшего тега не должен превышать количество подключенных эталонных аудиовходов.

## Выходные данные

| Имя выхода | Описание | Тип данных |
|------------|----------|------------|
