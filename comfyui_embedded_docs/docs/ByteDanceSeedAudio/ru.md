# ByteDance Seed Audio 1.0

Генерируйте речь, музыку, звуковые эффекты и диалоги нескольких говорящих по одному запросу с помощью ByteDance Seed Audio 1.0. Опишите голос(а), эмоции, атмосферу, фоновую музыку и звуковые эффекты в запросе, а также укажите реплики для озвучивания. При необходимости выберите встроенный голос из набора, клонируйте голоса из до 3 эталонных аудиофрагментов (помеченных @Audio1-3 в запросе) или создайте голос на основе изображения персонажа. До 2 минут аудио за один запуск.

## Входные параметры

| Параметр | Описание | Тип данных | Обязательный | Диапазон |
|-----------|-------------|-----------|----------|-------|
| `text_prompt` | Опишите голос(а), эмоции, темп, атмосферу, фоновую музыку и звуковые эффекты, а также укажите реплики для озвучивания (для диалогов называйте персонажей по ходу текста). В режиме 'audio reference' обращайтесь к подключенным аудиофрагментам по порядку как @Audio1, @Audio2, @Audio3. Максимум 3000 символов. | STRING | Да | от 1 до 3000 символов |
| `reference_mode` | Способ задания голоса: 'text only' (все описывается в запросе), 'audio reference' (клонирование до 3 голосов, помеченных @Audio1-3), 'image reference' (создание голоса на основе изображения одного персонажа) или 'preset voice' (выбор встроенного именованного голоса, который читает запрос). | COMBO | Да | `"text only"`<br>`"audio reference"`<br>`"image reference"`<br>`"preset voice"` |
| `reference_audio_1` | Эталонный аудиофрагмент для клонирования голоса, помеченный @Audio1 в запросе. До 30 секунд. | AUDIO | Нет | До 30 секунд |
| `reference_audio_2` | Эталонный аудиофрагмент, помеченный @Audio2 в запросе. До 30 секунд. | AUDIO | Нет | До 30 секунд |
| `reference_audio_3` | Эталонный аудиофрагмент, помеченный @Audio3 в запросе. До 30 секунд. | AUDIO | Нет | До 30 секунд |
| `reference_image` | Изображение одного персонажа; модель создает голос на его основе. Нельзя комбинировать с эталонными аудиофрагментами. | IMAGE | Нет | Одно изображение |
| `preset_voice` | Встроенный голос TTS 2.0, который читает запрос. Эталонные аудиофрагменты не требуются, теги @AudioN в этом режиме не используются. | COMBO | Нет | Доступно несколько вариантов встроенных голосов |
| `sample_rate` | Частота дискретизации выходного аудио в Гц. (по умолчанию: "24000") | COMBO | Да | `"8000"`<br>`"16000"`<br>`"24000"`<br>`"32000"`<br>`"44100"`<br>`"48000"` |
| `speech_rate` | Скорость речи. 0 = нормальная, 100 = 2.0x, -50 = 0.5x. (по умолчанию: 0) | INT | Да | от -50 до 100 |
| `loudness_rate` | Громкость. 0 = нормальная, 100 = 2.0x, -50 = 0.5x. (по умолчанию: 0) | INT | Да | от -50 до 100 |
| `pitch_rate` | Сдвиг высоты тона в полутонах (от -12 до 12). (по умолчанию: 0) | INT | Да | от -12 до 12 |
| `seed` | Значение seed определяет, должен ли узел перезапускаться; результаты недетерминированы независимо от seed. (по умолчанию: 42) | INT | Да | от 0 до 2147483647 |

**Ограничения и зависимости параметров:**

- Выбор `reference_mode` определяет, какие дополнительные параметры являются обязательными:
  - **"text only"**: Эталонные входные данные не требуются. Запрос не должен содержать теги @AudioN.
  - **"audio reference"**: Требуется подключение хотя бы одного из параметров `reference_audio_1`, `reference_audio_2` или `reference_audio_3`. Эталонные аудиофрагменты должны подключаться по порядку без пропусков (например, при использовании двух фрагментов подключите `reference_audio_1` и `reference_audio_2`). Длительность каждого эталонного аудиофрагмента ограничена 30 секундами. В запросе необходимо ссылаться на подключенные аудиофрагменты, используя теги @Audio1, @Audio2, @Audio3 по порядку.
  - **"image reference"**: Требуется подключение `reference_image`. Нельзя комбинировать с любыми эталонными аудиофрагментами. Запрос не должен содержать теги @AudioN.
  - **"preset voice"**: Требуется выбор `preset_voice`. Эталонные аудиофрагменты не нужны. Запрос не должен содержать теги @AudioN, кроме @Audio1.

## Выходные параметры

| Имя выхода | Описание | Тип данных |
|-------------|-------------|-----------|
| `audio` | Сгенерированный аудиовыход в виде аудиосигнала. | AUDIO |
