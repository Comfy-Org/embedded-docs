# ByteDanceSeedAudio

透過單一提示詞生成語音、音樂、音效及多說話者對話，使用位元組跳動 Seed Audio 1.0 模型。在提示詞中描述聲音、情緒、氛圍、背景音樂和音效，並包含要說出的台詞。可選擇內建預設聲音、從最多 3 個參考片段（在提示詞中標記為 @Audio1-3）複製聲音，或從角色圖片衍生聲音。每次執行最多可生成 2 分鐘音訊。

### 參數限制條件

- **參考模式依賴關係**：`reference_mode` 參數決定哪些其他輸入為必要：
  - **「僅文字」**：無需額外輸入。提示詞不得包含 @AudioN 標籤。
  - **「音訊參考」**：需要至少連接一個 `reference_audio_1`、`reference_audio_2` 或 `reference_audio_3`。參考片段必須按順序連接，中間不可中斷（例如 _1，然後 _2，然後 _3）。每個片段最長 30 秒。提示詞必須使用 @Audio1、@Audio2、@Audio3 標籤引用已連接的片段。
  - **「圖片參考」**：需要連接 `reference_image`。提示詞不得包含 @AudioN 標籤。
  - **「預設聲音」**：需要選擇 `preset_voice`。提示詞不得包含 @AudioN 標籤（整個提示詞將以所選聲音朗讀）。

- **音訊參考順序**：使用「音訊參考」模式時，參考音訊輸入必須從 `reference_audio_1` 開始依序連接，中間不可中斷。例如，可以連接 _1 和 _2，但不能在沒有 _2 的情況下連接 _1 和 _3。

- **最大音訊標籤數量**：在「音訊參考」模式下，提示詞最多可引用 3 個音訊片段（@Audio1、@Audio2、@Audio3）。最高編號的標籤不得超過已連接的參考音訊輸入數量。
