Вот перевод документации на русский язык:

Узел FluxKontextMultiReferenceLatentMethod изменяет данные кондиционирования, устанавливая конкретный метод обработки референсных скрытых представлений (reference latents). Он добавляет выбранный метод к входным данным кондиционирования, что влияет на то, как референсные скрытые представления обрабатываются на последующих этапах генерации. Данный узел помечен как экспериментальный и является частью системы кондиционирования Flux.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `кондиционирование` | CONDITIONING | Да | - | Данные кондиционирования, которые будут изменены с помощью метода референсных скрытых представлений |
| `метод_референтных_латентов` | STRING | Да | `"offset"`<br>`"index"`<br>`"uxo/uno"`<br>`"index_timestep_zero"` | Метод обработки референсных скрытых представлений. Если выбрано "uxo" или "uso", значение будет преобразовано в "uxo". Данный параметр помечен как расширенный. |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `кондиционирование` | CONDITIONING | Измененные данные кондиционирования с примененным методом референсных скрытых представлений |
