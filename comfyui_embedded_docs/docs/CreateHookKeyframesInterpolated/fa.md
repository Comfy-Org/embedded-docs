یک دنباله از کلیدفریم‌های هوک با مقادیر قدرت درون‌یابی شده بین نقطه شروع و پایان ایجاد می‌کند. این گره چندین کلیدفریم تولید می‌کند که پارامتر قدرت را به‌صورت هموار در یک بازه درصدی مشخص از فرآیند تولید تغییر می‌دهد و از روش‌های مختلف درون‌یابی برای کنترل منحنی انتقال استفاده می‌کند.

## ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|---------|----------|-------|--------|---------|
| `قدرت شروع` | FLOAT | بله | 0.0 - 10.0 | مقدار قدرت شروع برای دنباله درون‌یابی (پیش‌فرض: 1.0) |
| `قدرت پایان` | FLOAT | بله | 0.0 - 10.0 | مقدار قدرت پایان برای دنباله درون‌یابی (پیش‌فرض: 1.0) |
| `میان‌یابی` | COMBO | بله | `LINEAR`<br>`EASE_IN`<br>`EASE_OUT`<br>`EASE_IN_OUT`<br>`EASE_OUT_IN`<br>`SINE`<br>`CUBIC`<br>`QUARTIC`<br>`QUINTIC`<br>`EXPO`<br>`CIRC`<br>`BACK`<br>`BOUNCE`<br>`ELASTIC` | روش درون‌یابی مورد استفاده برای انتقال بین مقادیر قدرت (پیش‌فرض: LINEAR) |
| `درصد شروع` | FLOAT | بله | 0.0 - 1.0 | موقعیت درصد شروع در فرآیند تولید (پیش‌فرض: 0.0) |
| `درصد پایان` | FLOAT | بله | 0.0 - 1.0 | موقعیت درصد پایان در فرآیند تولید (پیش‌فرض: 1.0) |
| `تعداد keyframe` | INT | بله | 2 - 100 | تعداد کلیدفریم‌هایی که در دنباله درون‌یابی تولید می‌شود (پیش‌فرض: 5) |
| `نمایش keyframeها` | BOOLEAN | بله | True/False | آیا اطلاعات کلیدفریم‌های تولید شده در لاگ چاپ شود (پیش‌فرض: False) |
| `keyframe قبلی hook` | HOOK_KEYFRAMES | خیر | - | گروه کلیدفریم‌های هوک قبلی اختیاری برای الحاق به آن |
