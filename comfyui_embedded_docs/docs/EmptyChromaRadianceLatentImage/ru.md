Узел EmptyChromaRadianceLatentImage создает пустое латентное изображение с заданными размерами для использования в рабочих процессах хроматического свечения. Он генерирует тензор, заполненный нулями, который служит отправной точкой для операций в латентном пространстве. Узел позволяет задать ширину, высоту и размер пакета пустого латентного изображения.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `width` | INT | Да | от 16 до MAX_RESOLUTION | Ширина латентного изображения в пикселях (по умолчанию: 1024, должна быть кратна 16) |
| `height` | INT | Да | от 16 до MAX_RESOLUTION | Высота латентного изображения в пикселях (по умолчанию: 1024, должна быть кратна 16) |
| `batch_size` | INT | Нет | от 1 до 4096 | Количество латентных изображений для генерации в одном пакете (по умолчанию: 1) |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `samples` | LATENT | Сгенерированный тензор пустого латентного изображения с заданными размерами |
