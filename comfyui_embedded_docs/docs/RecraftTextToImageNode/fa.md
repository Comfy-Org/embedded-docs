## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `prompt` | STRING | بله | - | متن راهنما برای تولید تصویر. (پیش‌فرض: "") |
| `size` | COMBO | بله | "1024x1024"<br>"1152x896"<br>"896x1152"<br>"1216x832"<br>"832x1216"<br>"1344x768"<br>"768x1344"<br>"1536x640"<br>"640x1536" | اندازه تصویر تولیدشده. (پیش‌فرض: "1024x1024") |
| `n` | INT | بله | 1-6 | تعداد تصاویر برای تولید. (پیش‌فرض: 1) |
| `seed` | INT | بله | 0-18446744073709551615 | دانه‌ای برای تعیین اینکه گره دوباره اجرا شود؛ نتایج واقعی بدون توجه به دانه، غیرقطعی هستند. (پیش‌فرض: 0) |
| `recraft_style` | RECRAFT_STYLE | خیر | گزینه‌های متعدد در دسترس | انتخاب سبک اختیاری برای تولید تصویر. در صورت عدم ارائه، به طور پیش‌فرض از سبک "realistic_image" استفاده می‌شود. |
| `negative_prompt` | STRING | خیر | - | توضیح متنی اختیاری از عناصر نامطلوب در تصویر. (پیش‌فرض: "") |
| `recraft_controls` | RECRAFT_CONTROLS | خیر | گزینه‌های متعدد در دسترس | کنترل‌های اضافی اختیاری بر روی تولید از طریق گره Recraft Controls. |

**توجه:** پارامتر `seed` فقط زمان اجرای مجدد گره را کنترل می‌کند و تولید تصویر را قطعی نمی‌سازد. تصاویر خروجی واقعی حتی با مقدار دانه یکسان نیز متفاوت خواهند بود.

**توجه:** پارامتر `prompt` باید بین 1 تا 1000 کاراکتر طول داشته باشد.
