# Beeble SwitchX Video Edit

Редактирование видео с помощью Beeble SwitchX. Этот узел может заменить любой элемент сцены (фон, освещение, костюм), сохраняя при этом пиксели и движение исходного объекта. Предоставьте референсное изображение и/или текстовый запрос для описания нового вида.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `видео` | VIDEO | Да | Н/Д | Входное видео для редактирования. Максимум 240 кадров, максимум ~2,77 мегапикселя на кадр. |
| `промпт` | STRING | Да | Н/Д | Текстовое описание желаемого нового вида сцены. |
| `alpha_mode` | COMBO | Да | `"fill"`<br>`"select"`<br>`"custom"` | Режим альфа-маски. Режим "fill" не использует отдельную маску и заполняет весь кадр. Режим "select" использует одно ключевое изображение для определения области редактирования. Режим "custom" использует полное альфа-видео для покадрового определения области редактирования. |
| `максимальное разрешение` | COMBO | Да | `"720p"`<br>`"1080p"` | Максимальное разрешение выходного видео (по умолчанию: "1080p"). |
| `seed` | INT | Да | от 0 до 2147483647 | Значение зерна для воспроизводимости результатов. Использование одинакового зерна с одинаковыми входными данными даст одинаковый результат. |
| `референс-изображение` | IMAGE | Нет | Н/Д | Необязательное референсное изображение, описывающее желаемый новый вид сцены. |

### Подробности о режимах альфа-маски

Параметр `alpha_mode` определяет, какие части видео будут редактироваться:

- **fill**: Редактируется весь видеокадр. Отдельная альфа-маска не создается.
- **select**: Вы предоставляете одно ключевое изображение, определяющее область редактирования. Узел использует его для определения частей видео, которые нужно изменить.
- **custom**: Вы предоставляете полное альфа-видео, определяющее область редактирования покадрово. Это дает точный контроль над тем, какие части каждого кадра будут отредактированы.

При использовании режима `select` необходимо предоставить изображение `alpha_keyframe`. При использовании режима `custom` необходимо предоставить видео `alpha_mask`.

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `видео` | VIDEO | Отредактированное видео с примененными изменениями сцены. |
| `alpha` | VIDEO | Альфа-маска, используемая Beeble. Пустая для режима "fill", который не использует отдельную маску. |
