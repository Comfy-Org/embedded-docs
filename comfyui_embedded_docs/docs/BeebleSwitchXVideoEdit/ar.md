# Beeble SwitchX تحرير الفيديو

قم بتحرير فيديو باستخدام Beeble SwitchX. يمكن لهذه العقدة تبديل أي شيء في المشهد (الخلفية، الإضاءة، الزي) مع الحفاظ على وحدات البكسل والحركة الأصلية للكائن الرئيسي. قم بتوفير صورة مرجعية و/أو نص وصفي لوصف المظهر الجديد.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|-----------|-----------|----------|-------|-------------|
| `فيديو` | VIDEO | نعم | غير متاح | الفيديو المدخل للتحرير. الحد الأقصى 240 إطارًا، والحد الأقصى ~2.77 ميجابكسل لكل إطار. |
| `وصف نصي` | STRING | نعم | غير متاح | وصف نصي للمظهر الجديد المطلوب للمشهد. |
| `وضع ألفا` | COMBO | نعم | `"fill"`<br>`"select"`<br>`"custom"` | وضع قناة الشفافية. وضع "fill" لا يحتوي على قناة شفافية منفصلة ويملأ الإطار بالكامل. وضع "select" يستخدم صورة إطار مفتاحي واحدة لتحديد المنطقة المراد تحريرها. وضع "custom" يستخدم فيديو شفافية كامل لتحديد المنطقة المراد تحريرها إطارًا بإطار. |
| `أقصى دقة` | COMBO | نعم | `"720p"`<br>`"1080p"` | الدقة القصوى لفيديو الإخراج (الافتراضي: "1080p"). |
| `البذرة` | INT | نعم | 0 إلى 2147483647 | قيمة بذرة لإمكانية إعادة الإنتاج. استخدام نفس البذرة مع نفس المدخلات سينتج نفس النتيجة. |
| `الصورة المرجعية` | IMAGE | لا | غير متاح | صورة مرجعية اختيارية تصف المظهر الجديد المطلوب للمشهد. |

### تفاصيل وضع قناة الشفافية

يتحكم المعامل `alpha_mode` في أجزاء الفيديو التي سيتم تحريرها:

- **fill**: يتم تحرير إطار الفيديو بالكامل. لا يتم إنتاج قناة شفافية منفصلة.
- **select**: تقوم بتوفير صورة إطار مفتاحي واحدة تحدد المنطقة المراد تحريرها. ستستخدم العقدة هذه الصورة لتحديد أجزاء الفيديو التي سيتم تغييرها.
- **custom**: تقوم بتوفير فيديو شفافية كامل يحدد المنطقة المراد تحريرها إطارًا بإطار. يمنحك هذا تحكمًا دقيقًا في أجزاء كل إطار التي سيتم تحريرها.

عند استخدام وضع `select`، يجب توفير صورة `alpha_keyframe`. عند استخدام وضع `custom`، يجب توفير فيديو `alpha_mask`.
