Вот перевод документации узла SplitAudioChannels на русский язык:

Узел SplitAudioChannels разделяет стереофонический звук на отдельные левый и правый каналы. Он принимает стереофонический аудиовход с двумя каналами и выводит два отдельных аудиопотока: один для левого канала и один для правого канала.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `audio` | AUDIO | Да | - | Стереофонический аудиовход, который необходимо разделить на каналы |

**Примечание:** Входной аудиосигнал должен содержать ровно два канала (стерео). Узел выдаст ошибку, если входной аудиосигнал содержит только один канал.

## Выходы

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `left` | AUDIO | Разделенный аудиосигнал левого канала |
| `right` | AUDIO | Разделенный аудиосигнал правого канала |
