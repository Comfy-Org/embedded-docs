Узел `LatentInterpolate` предназначен для выполнения интерполяции между двумя наборами латентных (скрытых) выборок на основе заданного коэффициента, смешивая характеристики обоих наборов для получения нового, промежуточного набора латентных выборок.

## Входные параметры

| Параметр      | Тип данных | Описание |
|---------------|------------|----------|
| `образцы1`    | `LATENT`   | Первый набор латентных выборок для интерполяции. Служит начальной точкой процесса интерполяции. |
| `образцы2`    | `LATENT`   | Второй набор латентных выборок для интерполяции. Служит конечной точкой процесса интерполяции. |
| `соотношение`       | `FLOAT`    | Число с плавающей точкой, определяющее вес каждого набора выборок в интерполированном результате. Коэффициент 0 создает копию первого набора, а коэффициент 1 — копию второго набора. |

## Выходные параметры

| Параметр  | Тип данных | Описание |
|-----------|------------|----------|
| `latent`  | `LATENT`   | Результатом является новый набор латентных выборок, представляющий интерполированное состояние между двумя входными наборами на основе заданного коэффициента. |
