Синхронно генерирует видео на основе текстового запроса и дополнительных параметров с помощью API MiniMax. Этот узел создает видеоконтент из текстовых описаний, подключаясь к сервису преобразования текста в видео MiniMax.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `prompt_text` | STRING | Да | - | Текстовый запрос для управления генерацией видео |
| `model` | COMBO | Нет | "T2V-01"<br>"T2V-01-Director" | Модель для генерации видео (по умолчанию: "T2V-01") |
| `seed` | INT | Нет | от 0 до 18446744073709551615 | Случайное зерно, используемое для создания шума (по умолчанию: 0) |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `output` | VIDEO | Сгенерированное видео на основе входного запроса |
