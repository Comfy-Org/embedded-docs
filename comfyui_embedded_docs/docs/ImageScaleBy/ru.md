Узел ImageScaleBy предназначен для увеличения изображений с заданным коэффициентом масштабирования с использованием различных методов интерполяции. Он позволяет гибко изменять размер изображения, удовлетворяя различные потребности в увеличении.

## Входные параметры

| Параметр         | Тип данных   | Описание                                                                                     |
|------------------|--------------|----------------------------------------------------------------------------------------------|
| `изображение`          | `IMAGE`      | Входное изображение для увеличения. Этот параметр является ключевым, так как предоставляет исходное изображение, которое будет подвергнуто процессу масштабирования. |
| `метод_апскейла` | COMBO[STRING]| Определяет метод интерполяции, используемый для увеличения. Выбор метода может повлиять на качество и характеристики увеличенного изображения. |
| `увеличить_на`       | `FLOAT`      | Коэффициент, на который будет увеличено изображение. Определяет увеличение размера выходного изображения относительно входного. |

## Выходные параметры

| Параметр | Тип данных | Описание                                                                                   |
|----------|------------|--------------------------------------------------------------------------------------------|
| `изображение`  | `IMAGE`    | Увеличенное изображение, которое больше входного в соответствии с заданным коэффициентом масштабирования и методом интерполяции. |
