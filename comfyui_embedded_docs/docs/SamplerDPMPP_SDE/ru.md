Вот перевод документации на русский язык:

Узел SamplerDPMPP_SDE создаёт сэмплер DPM++ SDE (стохастическое дифференциальное уравнение) для использования в процессе сэмплирования. Этот сэмплер предоставляет стохастический метод сэмплирования с настраиваемыми параметрами шума и выбором устройства. Возвращает объект сэмплера, который можно использовать в конвейере сэмплирования.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `eta` | FLOAT | Да | 0.0 - 100.0 | Управляет стохастичностью процесса сэмплирования (по умолчанию: 1.0) |
| `s_noise` | FLOAT | Да | 0.0 - 100.0 | Управляет количеством шума, добавляемого во время сэмплирования (по умолчанию: 1.0) |
| `r` | FLOAT | Да | 0.0 - 100.0 | Параметр, влияющий на поведение сэмплирования (по умолчанию: 0.5) |
| `устройство_шума` | COMBO | Да | "gpu"<br>"cpu" | Выбирает устройство, на котором выполняются вычисления шума (по умолчанию: "gpu") |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `sampler` | SAMPLER | Возвращает настроенный объект сэмплера DPM++ SDE для использования в конвейерах сэмплирования |
