Вот перевод документации на русский язык:

Узел PrimitiveFloat создает значение числа с плавающей точкой, которое можно использовать в вашем рабочем процессе. Он принимает одно числовое значение на вход и выводит то же самое значение, позволяя вам определять и передавать значения с плавающей точкой между различными узлами в вашем конвейере ComfyUI.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `value` | FLOAT | Да | от -sys.maxsize до sys.maxsize (шаг: 0.1) | Значение числа с плавающей точкой для вывода (по умолчанию: 0.0) |

## Выходы

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `output` | FLOAT | Входное значение числа с плавающей точкой |
