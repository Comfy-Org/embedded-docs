Узел DualCLIPLoader предназначен для одновременной загрузки двух моделей CLIP, что упрощает выполнение операций, требующих интеграции или сравнения признаков обеих моделей.

Этот узел обнаружит модели, расположенные в папке `ComfyUI/models/text_encoders`.

## Входы

| Параметр      | Comfy dtype     | Описание                                                                                                                                                                 |
| ------------- | --------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| `clip_name1`  | COMBO[STRING] | Указывает имя первой загружаемой модели CLIP. Этот параметр важен для идентификации и извлечения правильной модели из предопределенного списка доступных моделей CLIP. |
| `clip_name2`  | COMBO[STRING] | Указывает имя второй загружаемой модели CLIP. Этот параметр позволяет загрузить вторую отдельную модель CLIP для сравнительного или интегративного анализа вместе с первой моделью. |
| `тип`        | `option`        | Выберите из "sdxl", "sd3", "flux" для адаптации к различным моделям.                                                                                                    |

* Порядок загрузки не влияет на выходной эффект

## Выходы

| Параметр | Тип данных | Описание                                                                                                                      |
| -------- | ---------- | ----------------------------------------------------------------------------------------------------------------------------- |
| `clip`   | CLIP       | На выходе получается комбинированная модель CLIP, которая объединяет признаки или функциональные возможности двух указанных моделей CLIP. |
