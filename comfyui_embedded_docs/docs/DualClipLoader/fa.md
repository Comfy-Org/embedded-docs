گره DualCLIPLoader برای بارگذاری همزمان دو مدل CLIP طراحی شده است و عملیاتی را که نیاز به ادغام یا مقایسه ویژگی‌های هر دو مدل دارند، تسهیل می‌کند.

این گره مدل‌های موجود در پوشه `ComfyUI/models/text_encoders` را شناسایی می‌کند.

## ورودی‌ها

| پارامتر       | نوع داده کامفی     | توضیحات                                                                                                                                                            |
| ------------- | ------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `clip_name1` | COMBO[STRING]      | نام اولین مدل CLIP که باید بارگذاری شود را مشخص می‌کند. این پارامتر برای شناسایی و بازیابی مدل صحیح از لیست از پیش تعریف‌شده مدل‌های CLIP ضروری است.                |
| `clip_name2` | COMBO[STRING]      | نام دومین مدل CLIP که باید بارگذاری شود را مشخص می‌کند. این پارامتر امکان بارگذاری یک مدل CLIP دوم و متمایز را برای تحلیل مقایسه‌ای یا یکپارچه در کنار مدل اول فراهم می‌کند. |
| `type`       | `option`           | از میان گزینه‌های "sdxl"، "sd3" و "flux" برای تطبیق با مدل‌های مختلف انتخاب کنید.                                                                                  |

* ترتیب بارگذاری تأثیری بر خروجی نهایی ندارد.

## خروجی‌ها

| پارامتر | نوع داده | توضیحات                                                                                                                          |
| ------- | ----------- | --------------------------------------------------------------------------------------------------------------------------------- |
| `clip`  | CLIP        | خروجی یک مدل CLIP ترکیبی است که ویژگی‌ها یا قابلیت‌های دو مدل CLIP مشخص‌شده را یکپارچه می‌کند. |
