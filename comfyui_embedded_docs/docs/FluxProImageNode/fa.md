### مرور کلی

تصاویر را به‌صورت همزمان بر اساس prompt و وضوح تصویر تولید می‌کند. این گره با استفاده از مدل Flux 1.1 Pro و با ارسال درخواست به یک API و انتظار برای دریافت پاسخ کامل، تصاویر را تولید کرده و تصویر نهایی را بازمی‌گرداند.

## ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|---------|----------|-------|--------|---------|
| `prompt` | STRING | بله | - | prompt برای تولید تصویر (پیش‌فرض: رشته خالی) |
| `prompt_upsampling` | BOOLEAN | بله | - | آیا بر روی prompt افزایش وضوح (upsampling) انجام شود یا خیر. در صورت فعال بودن، به‌طور خودکار prompt را برای تولید خلاقانه‌تر تغییر می‌دهد، اما نتایج غیرقطعی هستند (seed یکسان نتیجه دقیقاً یکسانی تولید نخواهد کرد). (پیش‌فرض: False) |
| `width` | INT | بله | 256-1440 | عرض تصویر بر حسب پیکسل (پیش‌فرض: 1024، گام: 32) |
| `height` | INT | بله | 256-1440 | ارتفاع تصویر بر حسب پیکسل (پیش‌فرض: 768، گام: 32) |
| `seed` | INT | بله | 0-18446744073709551615 | seed تصادفی مورد استفاده برای ایجاد نویز. (پیش‌فرض: 0) |
| `image_prompt` | IMAGE | خیر | - | تصویر مرجع اختیاری برای هدایت فرآیند تولید |

## خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-----------|----------|---------|
| `output` | IMAGE | تصویر تولید شده که از API بازگردانده شده است |
