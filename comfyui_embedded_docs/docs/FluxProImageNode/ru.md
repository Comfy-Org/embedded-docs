Генерирует изображения синхронно на основе запроса и разрешения. Этот узел создает изображения с использованием модели Flux 1.1 Pro, отправляя запросы к конечной точке API и ожидая полного ответа перед возвратом сгенерированного изображения.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `prompt` | STRING | Да | - | Запрос для генерации изображения (по умолчанию: пустая строка) |
| `prompt_upsampling` | BOOLEAN | Да | - | Выполнять ли апсемплинг запроса. Если активно, автоматически изменяет запрос для более креативной генерации, но результаты недетерминированы (одинаковый seed не даст точно такого же результата). (по умолчанию: False) |
| `width` | INT | Да | 256-1440 | Ширина изображения в пикселях (по умолчанию: 1024, шаг: 32) |
| `height` | INT | Да | 256-1440 | Высота изображения в пикселях (по умолчанию: 768, шаг: 32) |
| `seed` | INT | Да | 0-18446744073709551615 | Случайное зерно, используемое для создания шума. (по умолчанию: 0) |
| `image_prompt` | IMAGE | Нет | - | Дополнительное референсное изображение для направления генерации |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `output` | IMAGE | Сгенерированное изображение, полученное от API |
