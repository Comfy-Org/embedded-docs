Вот перевод документации на русский язык:

Узел ByteDance Seedream 4.5 и 5.0 обеспечивает унифицированную генерацию изображений по текстовому описанию и точное редактирование отдельных предложений с разрешением до 4K. Он может создавать новые изображения из текстовых подсказок или редактировать существующие изображения с помощью текстовых инструкций. Узел поддерживает как генерацию одного изображения, так и последовательную генерацию нескольких связанных изображений.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `model` | STRING | Да | См. описание | Модель Seedream для использования при генерации. Доступные модели включают варианты seedream-4-0, seedream-4-5 и seedream-5-0. |
| `prompt` | STRING | Да | - | Текстовый запрос для создания или редактирования изображения. Должен содержать не менее 1 символа. |
| `image` | IMAGE | Нет | - | Входное изображение(я) для генерации "изображение-в-изображение". Референтное изображение(я) для одно- или многореферентной генерации. Максимум 10 референтных изображений для большинства моделей или 14 для seedream-5-0-260128. |
| `size_preset` | STRING | Нет | Доступно несколько вариантов | Выберите рекомендуемый размер. Выберите Custom, чтобы использовать ширину и высоту, указанные ниже. По умолчанию: первый пресет из RECOMMENDED_PRESETS_SEEDREAM_4. |
| `width` | INT | Нет | от 1024 до 6240 (шаг 2) | Пользовательская ширина изображения. Значение работает только если `size_preset` установлен в `Custom`. По умолчанию: 2048. |
| `height` | INT | Нет | от 1024 до 4992 (шаг 2) | Пользовательская высота изображения. Значение работает только если `size_preset` установлен в `Custom`. По умолчанию: 2048. |
| `sequential_image_generation` | STRING | Нет | "disabled"<br>"auto" | Режим групповой генерации изображений. "disabled" генерирует одно изображение. "auto" позволяет модели решать, генерировать ли несколько связанных изображений (например, сцены истории, вариации персонажей). По умолчанию: "disabled". |
| `max_images` | INT | Нет | от 1 до 15 (шаг 1) | Максимальное количество изображений для генерации, когда sequential_image_generation='auto'. Общее количество изображений (входные + сгенерированные) не может превышать 15. По умолчанию: 1. |
| `seed` | INT | Нет | от 0 до 2147483647 (шаг 1) | Сид для использования при генерации. По умолчанию: 0. |
| `watermark` | BOOLEAN | Нет | - | Добавлять ли водяной знак "AI generated" на изображение. По умолчанию: False. |
| `fail_on_partial` | BOOLEAN | Нет | - | Если включено, прервать выполнение, если какие-либо запрошенные изображения отсутствуют, или вернуть ошибку. По умолчанию: True. |

**Примечания по ограничениям параметров:**
- Минимальное разрешение изображения зависит от выбранной модели: 3,68 МП для моделей seedream-4-5 и seedream-5-0, 0,92 МП для моделей seedream-4-0.
- Максимальное разрешение изображения составляет 10,4 МП для моделей seedream-5-0 и 16,78 МП для других моделей.
- Референтные изображения должны иметь соотношение сторон от 1:3 до 3:1.
- Когда `sequential_image_generation` установлен в "auto", общее количество входных изображений плюс `max_images` не может превышать 15.
- Параметры `width` и `height` используются только когда `size_preset` установлен в "Custom".

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `IMAGE` | IMAGE | Сгенерированное изображение(я) на основе входных параметров и запроса. Возвращает один тензор изображения или пакет тензоров изображений, если генерируется несколько изображений. |
