# ترجمة وثيقة عقدة ByteDance Seedream

توفر عقدة ByteDance Seedream 4.5 و 5.0 إمكانيات موحدة لتوليد الصور من النصوص وتحرير دقيق لجملة واحدة بدقة تصل إلى 4K. يمكنها إنشاء صور جديدة من النصوص الوصفية أو تحرير الصور الموجودة باستخدام تعليمات نصية. تدعم العقدة توليد صورة واحدة وكذلك التوليد المتسلسل لصور متعددة مترابطة.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|---------|--------------|--------|--------|-------|
| `model` | STRING | نعم | انظر الوصف | نموذج Seedream المستخدم في التوليد. تشمل النماذج المتاحة seedream-4-0 و seedream-4-5 و seedream-5-0 بمتغيراتها. |
| `prompt` | STRING | نعم | - | النص الوصفي لإنشاء أو تحرير صورة. يجب أن يكون طوله حرفًا واحدًا على الأقل. |
| `image` | IMAGE | لا | - | الصورة(الصور) المدخلة لتوليد صورة-إلى-صورة. صورة(صور) مرجعية للتوليد الفردي أو المتعدد. حد أقصى 10 صور مرجعية لمعظم النماذج، أو 14 لنموذج seedream-5-0-260128. |
| `size_preset` | STRING | لا | خيارات متعددة متاحة | اختر حجمًا موصى به. اختر "مخصص" لاستخدام قيم العرض والارتفاع أدناه. الافتراضي: أول إعداد مسبق من RECOMMENDED_PRESETS_SEEDREAM_4. |
| `width` | INT | لا | 1024 إلى 6240 (خطوة 2) | العرض المخصص للصورة. تعمل القيمة فقط إذا تم تعيين `size_preset` إلى "مخصص". الافتراضي: 2048. |
| `height` | INT | لا | 1024 إلى 4992 (خطوة 2) | الارتفاع المخصص للصورة. تعمل القيمة فقط إذا تم تعيين `size_preset` إلى "مخصص". الافتراضي: 2048. |
| `sequential_image_generation` | STRING | لا | "معطل"<br>"تلقائي" | وضع توليد الصور المجمعة. "معطل" يولد صورة واحدة. "تلقائي" يسمح للنموذج بتحديد ما إذا كان سيولد صورًا متعددة مترابطة (مثل مشاهد القصة، تنويعات الشخصيات). الافتراضي: "معطل". |
| `max_images` | INT | لا | 1 إلى 15 (خطوة 1) | الحد الأقصى لعدد الصور المراد توليدها عند ضبط sequential_image_generation='تلقائي'. إجمالي الصور (المدخلة + المولدة) لا يمكن أن يتجاوز 15. الافتراضي: 1. |
| `seed` | INT | لا | 0 إلى 2147483647 (خطوة 1) | البذرة المستخدمة في التوليد. الافتراضي: 0. |
| `watermark` | BOOLEAN | لا | - | ما إذا كان سيتم إضافة علامة مائية "مُولَّد بالذكاء الاصطناعي" إلى الصورة. الافتراضي: خطأ. |
| `fail_on_partial` | BOOLEAN | لا | - | إذا تم التفعيل، قم بإحباط التنفيذ إذا كانت أي صور مطلوبة مفقودة أو إرجاع خطأ. الافتراضي: صحيح. |

**ملاحظات حول قيود المعاملات:**
- الحد الأدنى لدقة الصورة يعتمد على النموذج المحدد: 3.68 ميجابكسل لنماذج seedream-4-5 و seedream-5-0، و 0.92 ميجابكسل لنماذج seedream-4-0.
- الحد الأقصى لدقة الصورة هو 10.4 ميجابكسل لنماذج seedream-5-0 و 16.78 ميجابكسل للنماذج الأخرى.
- يجب أن تكون نسبة العرض إلى الارتفاع للصور المرجعية بين 1:3 و 3:1.
- عند ضبط `sequential_image_generation` على "تلقائي"، لا يمكن أن يتجاوز إجمالي عدد الصور المدخلة بالإضافة إلى `max_images` 15.
- يتم استخدام معاملَي `width` و `height` فقط عندما يتم تعيين `size_preset` إلى "مخصص".
