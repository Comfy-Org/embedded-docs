### نمای کلی

گره «فریم اول-آخر به ویدیو» (Runway First-Last-Frame to Video) با آپلود فریم‌های کلیدی اول و آخر به همراه یک متن راهنما، ویدیو تولید می‌کند. این گره با استفاده از مدل Gen-3 شرکت Runway، انتقال‌های روانی بین فریم‌های شروع و پایان ایجاد می‌کند. این قابلیت به‌ویژه برای انتقال‌های پیچیده‌ای که فریم پایان تفاوت زیادی با فریم شروع دارد، مفید است.

### ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|---------|----------|--------|--------|---------|
| `prompt` | STRING | بله | نامشخص | متن راهنما برای تولید (پیش‌فرض: رشته خالی) |
| `start_frame` | IMAGE | بله | نامشخص | فریم شروع برای استفاده در ویدیو |
| `end_frame` | IMAGE | بله | نامشخص | فریم پایان برای استفاده در ویدیو. فقط برای gen3a_turbo پشتیبانی می‌شود. |
| `duration` | COMBO | بله | `"5"`<br>`"10"` | مدت زمان ویدیو بر حسب ثانیه (پیش‌فرض: "5") |
| `ratio` | COMBO | بله | `"16:9"`<br>`"9:16"`<br>`"1:1"` | نسبت ابعاد ویدیوی تولید شده (پیش‌فرض: "16:9") |
| `seed` | INT | خیر | 0 تا 4294967295 | دانه تصادفی برای تولید. برای دانه تصادفی، مقدار 0 را تنظیم کنید (پیش‌فرض: 0). |

**محدودیت‌های پارامتر:**

- `prompt` باید حداقل شامل 1 کاراکتر باشد
- هر دو `start_frame` و `end_frame` باید حداکثر ابعاد 7999×7999 پیکسل داشته باشند
- هر دو `start_frame` و `end_frame` باید نسبت ابعادی بین 0.5 و 2.0 داشته باشند
- پارامتر `end_frame` فقط هنگام استفاده از مدل gen3a_turbo پشتیبانی می‌شود
