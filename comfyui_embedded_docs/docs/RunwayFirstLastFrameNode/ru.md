Вот перевод документации на русский язык:

Узел Runway «Первый-Последний кадр в видео» создает видео путем загрузки первого и последнего ключевых кадров вместе с текстовым запросом. Он создает плавные переходы между предоставленными начальным и конечным кадрами с использованием модели Gen-3 от Runway. Это особенно полезно для сложных переходов, где конечный кадр существенно отличается от начального.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `prompt` | STRING | Да | Н/Д | Текстовый запрос для генерации (по умолчанию: пустая строка) |
| `start_frame` | IMAGE | Да | Н/Д | Начальный кадр, используемый для видео |
| `end_frame` | IMAGE | Да | Н/Д | Конечный кадр, используемый для видео. Поддерживается только для gen3a_turbo. |
| `duration` | COMBO | Да | `"5"`<br>`"10"` | Длительность видео в секундах (по умолчанию: "5") |
| `ratio` | COMBO | Да | `"16:9"`<br>`"9:16"`<br>`"1:1"` | Соотношение сторон для генерируемого видео (по умолчанию: "16:9") |
| `seed` | INT | Нет | от 0 до 4294967295 | Случайное зерно для генерации. Установите 0 для случайного зерна (по умолчанию: 0). |

**Ограничения параметров:**

- `prompt` должен содержать не менее 1 символа
- Как `start_frame`, так и `end_frame` должны иметь максимальные размеры 7999x7999 пикселей
- Как `start_frame`, так и `end_frame` должны иметь соотношение сторон от 0,5 до 2,0
- Параметр `end_frame` поддерживается только при использовании модели gen3a_turbo

## Выходные данные

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `output` | VIDEO | Сгенерированное видео с переходом между начальным и конечным кадрами |
