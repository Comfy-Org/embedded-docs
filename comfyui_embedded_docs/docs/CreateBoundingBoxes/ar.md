# إنشاء صناديق محيطة

هذه الواجهة توفر مساحة رسم تفاعلية لرسم مربعات إحاطة حول الكائنات أو مناطق النصوص داخل الصورة. تُخرج إحداثيات مربعات الإحاطة بوحدات البكسل، وبيانات عناصر منظمة متوافقة مع تنسيق مطالبات Ideogram، بالإضافة إلى صورة معاينة تُظهر المربعات المرسومة مع التسميات ولوحات الألوان.

## المدخلات

| المعامل | الوصف | نوع البيانات | إلزامي | النطاق |
|---------|-------|---------------|--------|--------|
| `الخلفية` | صورة اختيارية تُستخدم كخلفية في مساحة الرسم والمعاينة. | IMAGE | لا | - |
| `bboxes` | مربعات إحاطة أو عناصر أو سلسلة JSON لتهيئة مساحة الرسم. قيمة جديدة من المنبع تُهيئ مساحة الرسم؛ التعديلات التي تُجرى على مساحة الرسم لها الأولوية وتُحتفظ بها حتى تتغير القيمة القادمة من المنبع مرة أخرى. | BOUNDING_BOX, ARRAY, STRING | لا | - |
| `العرض` | عرض مساحة الرسم وشبكة البكسل لمربعات الإحاطة (الافتراضي: 1024). | INT | نعم | 64 إلى 16384 (الخطوة: 16) |
| `الارتفاع` | ارتفاع مساحة الرسم وشبكة البكسل لمربعات الإحاطة (الافتراضي: 1024). | INT | نعم | 64 إلى 16384 (الخطوة: 16) |
| `حالة المحرر` | ارسم مربعات إحاطة واضبط نوع كل مربع ونصه ووصفه ولوحة ألوانه. ابدأ بعنصر الخلفية أولاً وانتهي بالمقدمة أخيرًا. | BOUNDING_BOXES | نعم | - |
| `last_incoming` | حالة داخلية تُدار بواسطة مساحة الرسم: قيمة مربعات الإحاطة القادمة من المنبع التي قامت بتهيئتها آخر مرة. اتركه فارغًا لإعادة تهيئة مساحة الرسم من مدخل مربعات الإحاطة في التشغيل التالي. | BOUNDING_BOXES | لا | - |

## المخرجات

| اسم المخرج | الوصف | نوع البيانات |
|------------|-------|--------------|
| `معاينة` | صورة RGB تُظهر مساحة الرسم مع جميع مربعات الإحاطة المعروضة، بما في ذلك التسميات وعينات لوحة الألوان والنصوص الوصفية. | IMAGE |
| `صناديق محيطة` | قائمة بمربعات الإحاطة بإحداثيات البكسل، حيث يحتوي كل مربع على قيم x و y والعرض والارتفاع. | BOUNDING_BOX |
| `عناصر` | مصفوفة منظمة من كائنات العناصر، يحتوي كل منها على النوع وإحداثيات مربع الإحاطة (موحدة 0-1000) والنص (للنوع النصي) والوصف ولوحة الألوان. | ARRAY |

> تم إنشاء هذه الوثيقة بواسطة الذكاء الاصطناعي. إذا وجدت أي أخطاء أو لديك اقتراحات للتحسين، فلا تتردد في المساهمة! [تحرير على GitHub](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/CreateBoundingBoxes/ar.md)
