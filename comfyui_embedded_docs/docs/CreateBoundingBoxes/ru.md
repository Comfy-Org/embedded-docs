# Создать ограничивающие рамки

Этот узел предоставляет интерфейс холста для рисования ограничивающих рамок вокруг объектов или текстовых областей на изображении. Он выводит координаты ограничивающих рамок в пиксельном пространстве, структурированные данные элементов, совместимые с форматированием подсказок Ideogram, а также изображение для предварительного просмотра с нарисованными рамками, метками и цветовыми палитрами.

## Входы

| Параметр | Описание | Тип данных | Обязательный | Диапазон |
|----------|----------|------------|--------------|----------|
| `фон` | Необязательное изображение, используемое в качестве фона на холсте и в предварительном просмотре. | IMAGE | Нет | - |
| `bboxes` | Ограничивающие рамки, элементы или строка JSON для инициализации холста. Новое значение от вышестоящего узла инициализирует холст; изменения, внесенные на холсте, имеют приоритет и сохраняются до тех пор, пока значение вышестоящего узла снова не изменится. | BOUNDING_BOX, ARRAY, STRING | Нет | - |
| `ширина` | Ширина холста и пиксельной сетки для ограничивающих рамок (по умолчанию: 1024). | INT | Да | от 64 до 16384 (шаг: 16) |
| `высота` | Высота холста и пиксельной сетки для ограничивающих рамок (по умолчанию: 1024). | INT | Да | от 64 до 16384 (шаг: 16) |
| `состояние_редактора` | Рисуйте ограничивающие рамки и задавайте для каждой рамки тип, текст, описание, цветовую палитру. Начинайте с фонового элемента, заканчивайте передним планом. | BOUNDING_BOXES | Да | - |
| `last_incoming` | Внутреннее состояние, управляемое холстом: значение bboxes от вышестоящего узла, которое последним инициализировало холст. Оставьте пустым, чтобы повторно инициализировать холст из входа bboxes при следующем запуске. | BOUNDING_BOXES | Нет | - |

## Выходы

| Имя выхода | Описание | Тип данных |
|------------|----------|------------|
| `предпросмотр` | RGB-изображение, показывающее холст со всеми отрисованными ограничивающими рамками, включая метки, образцы цветовой палитры и описательный текст. | IMAGE |
| `рамки` | Список ограничивающих рамок в пиксельных координатах, где каждая рамка содержит значения x, y, ширины и высоты. | BOUNDING_BOX |
| `элементы` | Структурированный массив объектов элементов, каждый из которых содержит тип, координаты ограничивающей рамки (нормализованные 0-1000), текст (для текстового типа), описание и цветовую палитру. | ARRAY |
