# 建立邊界框

此節點提供一個畫布介面，用於在影像中的物體或文字區域周圍繪製邊界框。它輸出像素空間中的邊界框座標、與 Ideogram 提示格式相容的結構化元素資料，以及顯示繪製框（附帶標籤和色板）的預覽影像。

## 輸入
| 參數 | 說明 | 資料類型 | 必要 | 範圍 |
|-----------|-------------|-----------|----------|-------|
| `背景` | 可選影像，用作畫布和預覽中的背景。 | IMAGE | 否 | - |
| `bboxes` | 邊界框、元素或 JSON 字串，用於初始化畫布。新的上游值會初始化畫布；在畫布上進行的編輯具有優先權，並會保留，直到上游值再次變更。 | BOUNDING_BOX, ARRAY, STRING | 否 | - |
| `寬度` | 畫布及邊界框像素網格的寬度（預設值：1024）。 | INT | 是 | 64 至 16384（步長：16） |
| `高度` | 畫布及邊界框像素網格的高度（預設值：1024）。 | INT | 是 | 64 至 16384（步長：16） |
| `編輯器狀態` | 繪製邊界框，並設定每個框的類型、文字、描述、色板。從背景元素開始，前景元素最後。 | BOUNDING_BOXES | 是 | - |
| `last_incoming` | 由畫布管理的內部狀態：上次初始化畫布的上游 bboxes 值。留空以在下次執行時從 bboxes 輸入重新初始化畫布。 | BOUNDING_BOXES | 否 | - |

## 輸出
| 輸出名稱 | 說明 | 資料類型 |
|-------------|-------------|-----------|
| `預覽` | 一個 RGB 影像，顯示渲染了所有邊界框的畫布，包含標籤、色板樣本和描述性文字。 | IMAGE |
| `邊界框` | 像素座標中的邊界框列表，每個框包含 x、y、寬度和高度值。 | BOUNDING_BOX |
| `元素` | 元素物件的結構化陣列，每個元素包含類型、邊界框座標（標準化為 0-1000）、文字（針對文字類型）、描述和色板。 | ARRAY |
