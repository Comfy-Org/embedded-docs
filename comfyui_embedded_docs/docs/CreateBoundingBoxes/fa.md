# ایجاد جعبه‌های مرزی

این گره یک رابط کاربری بوم‌مانند برای رسم کادرهای محدودکننده (bounding boxes) پیرامون اشیاء یا نواحی متنی در یک تصویر فراهم می‌کند. خروجی آن مختصات کادرهای محدودکننده در فضای پیکسلی، داده‌های ساختاریافته عنصر سازگار با قالب‌بندی پرامپت Ideogram، و یک تصویر پیش‌نمایش شامل کادرهای رسم‌شده به همراه برچسب‌ها و پالت‌های رنگی است.

## ورودی‌ها

| پارامتر | توضیحات | نوع داده | ضروری | محدوده |
|---------|---------|----------|-------|--------|
| `background` | تصویر اختیاری که به‌عنوان پس‌زمینه در بوم و پیش‌نمایش استفاده می‌شود. | IMAGE | خیر | - |
| `bboxes` | کادرهای محدودکننده، عناصر یا یک رشته JSON برای مقداردهی اولیه بوم. یک مقدار بالادستی جدید، بوم را مقداردهی اولیه می‌کند؛ ویرایش‌های انجام‌شده روی بوم اولویت دارند و تا زمانی که مقدار بالادستی دوباره تغییر کند، حفظ می‌شوند. | BOUNDING_BOX, ARRAY, STRING | خیر | - |
| `width` | عرض بوم و شبکه پیکسلی برای کادرهای محدودکننده (پیش‌فرض: 1024). | INT | بله | 64 تا 16384 (گام: 16) |
| `height` | ارتفاع بوم و شبکه پیکسلی برای کادرهای محدودکننده (پیش‌فرض: 1024). | INT | بله | 64 تا 16384 (گام: 16) |
| `editor_state` | کادرهای محدودکننده را رسم کرده و نوع، متن، توضیحات و پالت رنگی هر کادر را تنظیم کنید. با عنصر پس‌زمینه شروع کرده و با عنصر پیش‌زمینه پایان دهید. | BOUNDING_BOXES | بله | - |
| `last_incoming` | وضعیت داخلی مدیریت‌شده توسط بوم: مقدار بالادستی bboxes که آخرین بار بوم را مقداردهی اولیه کرد. برای مقداردهی اولیه مجدد بوم از ورودی bboxes در اجرای بعدی، این فیلد را خالی بگذارید. | BOUNDING_BOXES | خیر | - |

## خروجی‌ها

| نام خروجی | توضیحات | نوع داده |
|-----------|---------|----------|
| `preview` | یک تصویر RGB که بوم را با تمام کادرهای محدودکننده رسم‌شده شامل برچسب‌ها، نمونه‌های پالت رنگی و متن توصیفی نشان می‌دهد. | IMAGE |
| `bboxes` | فهرستی از کادرهای محدودکننده در مختصات پیکسلی که هر کادر شامل مقادیر x، y، عرض و ارتفاع است. | BOUNDING_BOX |
| `elements` | یک آرایه ساختاریافته از اشیاء عنصر که هر کدام شامل نوع، مختصات کادر محدودکننده (نرمال‌شده 0-1000)، متن (برای نوع متنی)، توضیحات و پالت رنگی است. | ARRAY |

> این مستند با هوش مصنوعی تهیه شده است. اگر خطایی دیدید یا پیشنهادی برای بهبود دارید، خوشحال می‌شویم مشارکت کنید! [ویرایش در GitHub](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/CreateBoundingBoxes/fa.md)
