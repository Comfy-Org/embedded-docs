Узел EmptyMochiLatentVideo создает пустой тензор скрытого видео с заданными размерами. Он генерирует заполненное нулями скрытое представление, которое можно использовать в качестве отправной точки для рабочих процессов генерации видео. Узел позволяет задать ширину, высоту, длину и размер пакета для тензора скрытого видео.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `ширина` | INT | Да | от 16 до MAX_RESOLUTION | Ширина скрытого видео в пикселях (по умолчанию: 848, должна делиться на 16) |
| `высота` | INT | Да | от 16 до MAX_RESOLUTION | Высота скрытого видео в пикселях (по умолчанию: 480, должна делиться на 16) |
| `длина` | INT | Да | от 7 до MAX_RESOLUTION | Количество кадров в скрытом видео (по умолчанию: 25, должно делиться на 6 после вычитания 1) |
| `размер_пакета` | INT | Нет | от 1 до 4096 | Количество скрытых видео для генерации в одном пакете (по умолчанию: 1) |

**Примечание:** Фактические размеры скрытого пространства вычисляются как ширина/8 и высота/8, а временное измерение вычисляется как ((длина - 1) // 6) + 1. Параметр `length` должен удовлетворять условию, что `(length - 1)` делится на 6, то есть допустимые значения: 7, 13, 19, 25 и т.д.

## Выходные данные

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `samples` | LATENT | Пустой тензор скрытого видео с заданными размерами, содержащий все нули |
