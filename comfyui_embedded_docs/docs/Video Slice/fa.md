گره Video Slice به شما امکان می‌دهد یک بخش مشخص از یک ویدیو را استخراج کنید. می‌توانید زمان شروع و مدت‌زمان را برای برش ویدیو تعریف کنید، یا صرفاً فریم‌های ابتدایی را نادیده بگیرید. اگر مدت‌زمان درخواستی بیشتر از ویدیوی باقی‌مانده باشد، این گره می‌تواند یا بخش موجود را برگرداند یا یک خطا ایجاد کند.

## ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `ویدیو` | VIDEO | بله | - | ویدیوی ورودی که قرار است برش داده شود. |
| `زمان شروع` | FLOAT | خیر | 1e5- تا 1e5 | زمان شروع بر حسب ثانیه (پیش‌فرض: 0.0). |
| `مدت زمان` | FLOAT | خیر | 0.0 و بالاتر | مدت‌زمان بر حسب ثانیه، یا 0 برای مدت‌زمان نامحدود (پیش‌فرض: 0.0). |
| `مدت زمان دقیق` | BOOLEAN | خیر | - | اگر True باشد، زمانی که مدت‌زمان مشخص‌شده امکان‌پذیر نباشد، یک خطا ایجاد می‌شود (پیش‌فرض: False). |
