Узел LTXVLatentUpsampler увеличивает пространственное разрешение скрытого представления видео в два раза. Для обработки скрытых данных используется специализированная модель апскейла, которая сначала выполняет денормализацию, а затем повторную нормализацию с использованием статистики каналов указанного VAE. Данный узел предназначен для работы с видео в скрытом пространстве.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `образцы` | LATENT | Да | | Входное скрытое представление видео для увеличения разрешения. |
| `модель увеличения` | LATENT_UPSCALE_MODEL | Да | | Загруженная модель, используемая для двукратного увеличения разрешения скрытых данных. |
| `vae` | VAE | Да | | Модель VAE, используемая для денормализации входных скрытых данных перед апскейлом и для нормализации выходных скрытых данных после него. |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `LATENT` | LATENT | Увеличенное скрытое представление с удвоенными пространственными размерами по сравнению с входными данными. Выходное скрытое представление имеет тот же размер пакета, количество каналов и временную длину, что и входное. `noise_mask` из входных данных, если он присутствует, удаляется из выходных данных. |
