# HeyGenVideoTranslateNode

### مرور کلی

یک ویدیوی گفتاری را به زبانی دیگر با شبیه‌سازی صدا و هماهنگی لب ترجمه می‌کند. این گره صدای گوینده اصلی را شبیه‌سازی کرده و حرکت لب را برای مطابقت با گفتار ترجمه‌شده بازپویایی می‌کند و نتیجه‌ای طبیعی به نظر می‌رساند.

### ورودی‌ها

| پارامتر | توضیحات | نوع داده | الزامی | محدوده |
|----------|----------|-----------|----------|-------|
| `video` | ویدیوی حاوی گفتار برای ترجمه. | VIDEO | بله | - |
| `output_language` | زبان هدف برای ویدیوی ترجمه‌شده. | STRING | بله | "عربی"<br>"بنگالی"<br>"چینی"<br>"دانمارکی"<br>"هلندی"<br>"انگلیسی"<br>"فرانسوی"<br>"آلمانی"<br>"یونانی"<br>"هندی"<br>"اندونزیایی"<br>"ایتالیایی"<br>"ژاپنی"<br>"کره‌ای"<br>"مالایی"<br>"لهستانی"<br>"پرتغالی"<br>"روسی"<br>"اسپانیایی"<br>"سوئدی"<br>"تامیلی"<br>"تلوگویی"<br>"تایلندی"<br>"ترکی"<br>"اوکراینی"<br>"ویتنامی" |
| `mode` | حالت 'speed' سریع‌تر است؛ حالت 'precision' هماهنگی لب با کیفیت بالاتر را با دو برابر هزینه ارائه می‌دهد. | STRING | بله | "speed"<br>"precision" |
| `translate_audio_only` | فقط آهنگ صوتی را تعویض می‌کند و حرکات اصلی لب را حفظ می‌کند (بدون هماهنگی لب). (پیش‌فرض: False) | BOOLEAN | خیر | True<br>False |
| `speaker_count` | تعداد گویندگان در ویدیو. 0 = تشخیص خودکار. (پیش‌فرض: 0) | INT | خیر | 0 تا 10 |
| `seed` | به HeyGen ارسال نمی‌شود؛ آن را تغییر دهید تا اجرای مجدد انجام شود. (پیش‌فرض: 42) | INT | خیر | 0 تا 2147483647 |

### خروجی‌ها

| نام خروجی | توضیحات | نوع داده |
|------------|----------|-----------|
| `video` | ویدیوی ترجمه‌شده با شبیه‌سازی صدا و هماهنگی لب اعمال‌شده. | VIDEO |
