Узел LTXVConditioning добавляет информацию о частоте кадров к позитивному и негативному условиям для моделей генерации видео. Он принимает существующие данные условий и применяет указанное значение частоты кадров к обоим наборам условий, делая их пригодными для обработки видео моделями.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `positive` | CONDITIONING | Да | - | Позитивное условие, которое получит информацию о частоте кадров |
| `negative` | CONDITIONING | Да | - | Негативное условие, которое получит информацию о частоте кадров |
| `frame_rate` | FLOAT | Нет | 0.0 - 1000.0 | Значение частоты кадров, применяемое к обоим наборам условий (по умолчанию: 25.0) |

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `positive` | CONDITIONING | Позитивное условие с примененной информацией о частоте кадров |
| `negative` | CONDITIONING | Негативное условие с примененной информацией о частоте кадров |
