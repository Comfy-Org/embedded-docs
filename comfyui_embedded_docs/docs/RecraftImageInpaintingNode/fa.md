این گره نواحی خاصی از یک تصویر را بر اساس یک متن راهنما (prompt) و یک ماسک (mask) تغییر می‌دهد. از API شرکت Recraft برای ویرایش هوشمندانه فقط نواحی ماسک‌شده استفاده می‌کند، در حالی که بقیه تصویر بدون تغییر باقی می‌ماند.

## ورودی‌ها

| پارامتر | نوع داده | اجباری | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `تصویر` | IMAGE | بله | - | تصویر ورودی که قرار است تغییر کند |
| `ماسک` | MASK | بله | - | ماسکی که نواحی مورد نظر برای تغییر در تصویر را مشخص می‌کند |
| `پرامپت` | STRING | بله | - | راهنمای تولید تصویر (پیش‌فرض: رشته خالی، حداکثر طول: ۱۰۰۰ کاراکتر) |
| `تعداد` | INT | بله | ۱-۶ | تعداد تصاویر تولیدی (پیش‌فرض: ۱، حداقل: ۱، حداکثر: ۶) |
| `seed` | INT | بله | ۰-۱۸۴۴۶۷۴۴۰۷۳۷۰۹۵۵۱۶۱۵ | دانه (seed) برای تعیین اینکه آیا گره دوباره اجرا شود؛ نتایج واقعی بدون توجه به دانه، غیرقطعی هستند (پیش‌فرض: ۰) |
| `سبک Recraft` | STYLEV3 | خیر | - | پارامتر سبک اختیاری برای API Recraft. در صورت عدم ارائه، به طور پیش‌فرض از سبک "realistic_image" استفاده می‌شود |
| `پرامپت منفی` | STRING | خیر | - | توضیح متنی اختیاری از عناصر ناخواسته در تصویر (پیش‌فرض: رشته خالی) |

*توجه: `image` و `mask` باید با هم ارائه شوند تا عملیات inpainting (پر کردن هوشمند) کار کند. ماسک به طور خودکار برای تطبیق با ابعاد تصویر تغییر اندازه می‌دهد. `prompt` اعتبارسنجی می‌شود و حداکثر طول آن ۱۰۰۰ کاراکتر است.*

## خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-------------|-----------|-------------|
| `تصویر` | IMAGE | تصویر(های) تغییر یافته بر اساس راهنما و ماسک تولید شده است. به ازای هر تصویر ورودی ضرب‌در پارامتر `تعداد`، یک تصویر برمی‌گرداند |
