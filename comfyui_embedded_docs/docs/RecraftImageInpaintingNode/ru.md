Этот узел изменяет определённые области изображения на основе текстового запроса и маски. Он использует API Recraft для интеллектуального редактирования только замаскированных областей, оставляя остальную часть изображения без изменений.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `image` | IMAGE | Да | - | Входное изображение для модификации |
| `mask` | MASK | Да | - | Маска, определяющая области изображения, которые следует изменить |
| `prompt` | STRING | Да | - | Запрос для генерации изображения (по умолчанию: пустая строка, максимальная длина: 1000 символов) |
| `n` | INT | Да | 1-6 | Количество генерируемых изображений (по умолчанию: 1, минимум: 1, максимум: 6) |
| `seed` | INT | Да | 0-18446744073709551615 | Значение для определения необходимости повторного запуска узла; фактические результаты недетерминированы независимо от seed (по умолчанию: 0) |
| `recraft_style` | STYLEV3 | Нет | - | Необязательный параметр стиля для API Recraft. Если не указан, по умолчанию используется стиль "realistic_image" |
| `negative_prompt` | STRING | Нет | - | Необязательное текстовое описание нежелательных элементов на изображении (по умолчанию: пустая строка) |

*Примечание: Параметры `image` и `mask` должны быть указаны вместе для работы операции инпейнтинга. Маска будет автоматически изменена в размере для соответствия размерам изображения. Параметр `prompt` проверяется и имеет максимальную длину в 1000 символов.*

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `image` | IMAGE | Модифицированное изображение(я), сгенерированное на основе запроса и маски. Возвращает одно изображение на каждое входное изображение, умноженное на параметр `n` |
