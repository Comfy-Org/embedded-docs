هذه العقدة تقوم بتعديل مناطق محددة من الصورة بناءً على نص موجه وقناع. تستخدم واجهة برمجة تطبيقات Recraft لتحرير المناطق المقنعة فقط بذكاء مع الحفاظ على باقي الصورة دون تغيير.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|-----------|-----------|----------|-------|-------------|
| `صورة` | IMAGE | نعم | - | الصورة المدخلة المراد تعديلها |
| `قناع` | MASK | نعم | - | القناع الذي يحدد مناطق الصورة التي يجب تعديلها |
| `الوصف` | STRING | نعم | - | النص الموجّه لتوليد الصورة (القيمة الافتراضية: سلسلة فارغة، الحد الأقصى للطول: 1000 حرف) |
| `عدد` | INT | نعم | 1-6 | عدد الصور المراد توليدها (القيمة الافتراضية: 1، الحد الأدنى: 1، الحد الأقصى: 6) |
| `بذرة` | INT | نعم | 0-18446744073709551615 | البذرة لتحديد ما إذا كانت العقدة ستعيد التشغيل؛ النتائج الفعلية غير حتمية بغض النظر عن البذرة (القيمة الافتراضية: 0) |
| `نمط إعادة الصياغة` | STYLEV3 | لا | - | معامل النمط الاختياري لواجهة برمجة تطبيقات Recraft. إذا لم يتم توفيره، فسيتم استخدام النمط الافتراضي "realistic_image" |
| `الوصف السلبي` | STRING | لا | - | وصف نصي اختياري للعناصر غير المرغوب فيها في الصورة (القيمة الافتراضية: سلسلة فارغة) |

*ملاحظة: يجب توفير `image` و `mask` معًا لكي تعمل عملية الترميم. سيتم تغيير حجم القناع تلقائيًا ليتطابق مع أبعاد الصورة. يتم التحقق من صحة `prompt` ويبلغ الحد الأقصى لطوله 1000 حرف.*
