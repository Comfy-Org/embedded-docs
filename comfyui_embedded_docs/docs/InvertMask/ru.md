Узел InvertMask предназначен для инвертирования значений заданной маски, что позволяет поменять местами замаскированные и незамаскированные области. Эта операция является основополагающей в задачах обработки изображений, когда необходимо переключить фокус внимания между передним и задним планом.

## Входные параметры

| Параметр | Тип данных | Описание |
|-----------|--------------|-------------|
| `маска`    | MASK         | Параметр `маска` представляет собой входную маску, подлежащую инвертированию. Он имеет решающее значение для определения областей, которые будут изменены в процессе инверсии. |

## Выходные параметры

| Параметр | Тип данных | Описание |
|-----------|--------------|-------------|
| `маска`    | MASK         | На выходе получается инвертированная версия входной маски, где ранее замаскированные области становятся незамаскированными и наоборот. |
