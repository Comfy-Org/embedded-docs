# ترجمة وثيقة عقدة MiniMaxImageToVideoNode

## نظرة عامة

توليد فيديوهات متزامنة بناءً على صورة ونص وصفي، مع معاملات اختيارية باستخدام واجهة برمجة تطبيقات MiniMax. تأخذ هذه العقدة صورة إدخال ووصفًا نصيًا لإنشاء سلسلة فيديو، مع توفر خيارات نماذج متنوعة وإعدادات تكوين مختلفة.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|---------|---------------|--------|---------|-------|
| `صورة` | IMAGE | نعم | - | الصورة المستخدمة كإطار أول لتوليد الفيديو |
| `نص النص المطلوب` | STRING | نعم | - | النص الوصفي لتوجيه عملية توليد الفيديو (الافتراضي: سلسلة فارغة) |
| `نموذج` | COMBO | نعم | "I2V-01-Director"<br>"I2V-01"<br>"I2V-01-live" | النموذج المستخدم لتوليد الفيديو (الافتراضي: "I2V-01") |
| `بذرة` | INT | لا | 0 إلى 18446744073709551615 | البذرة العشوائية المستخدمة لإنشاء الضوضاء (الافتراضي: 0) |

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|-------------|---------------|---------|
| `output` | VIDEO | مخرج الفيديو المُولَّد |
