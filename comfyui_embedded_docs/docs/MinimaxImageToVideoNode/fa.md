### مرور کلی
این گره با استفاده از API مینی‌مکس (MiniMax)، بر اساس یک تصویر و متن راهنما (prompt) و پارامترهای اختیاری، ویدیو تولید می‌کند. این گره یک تصویر ورودی و یک توضیح متنی دریافت کرده و یک دنباله ویدیویی ایجاد می‌کند. گزینه‌های مختلف مدل و تنظیمات پیکربندی نیز در دسترس هستند.

### ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `image` | IMAGE | بله | - | تصویری که به عنوان فریم اول تولید ویدیو استفاده می‌شود |
| `prompt_text` | STRING | بله | - | متن راهنما برای هدایت تولید ویدیو (پیش‌فرض: رشته خالی) |
| `model` | COMBO | بله | "I2V-01-Director"<br>"I2V-01"<br>"I2V-01-live" | مدل مورد استفاده برای تولید ویدیو (پیش‌فرض: "I2V-01") |
| `seed` | INT | خیر | 0 تا 18446744073709551615 | دانه تصادفی (seed) مورد استفاده برای ایجاد نویز (پیش‌فرض: 0) |

### خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-------------|-----------|-------------|
| `output` | VIDEO | خروجی ویدیوی تولید شده |
