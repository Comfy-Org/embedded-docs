Вот перевод документации на русский язык:

Генерирует видео синхронно на основе изображения, текстовой подсказки и дополнительных параметров с использованием API MiniMax. Этот узел принимает входное изображение и текстовое описание для создания видеопоследовательности с различными доступными опциями модели и настройками конфигурации.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `изображение` | IMAGE | Да | - | Изображение, используемое в качестве первого кадра для генерации видео |
| `текстовый промпт` | STRING | Да | - | Текстовая подсказка для управления генерацией видео (по умолчанию: пустая строка) |
| `модель` | COMBO | Да | "I2V-01-Director"<br>"I2V-01"<br>"I2V-01-live" | Модель, используемая для генерации видео (по умолчанию: "I2V-01") |
| `seed` | INT | Нет | от 0 до 18446744073709551615 | Случайное зерно, используемое для создания шума (по умолчанию: 0) |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `output` | VIDEO | Сгенерированный видеовыход |
