Этот узел сокращает текст, обрезая его до заданной максимальной длины. Он принимает любой входной текст и возвращает только первую часть, содержащую указанное количество символов. Это простой способ гарантировать, что текст не превышает определенный размер.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `text` | STRING | Да | Н/Д | Текстовая строка, подлежащая сокращению. |
| `max_length` | INT | Да | от 1 до 10000 | Максимальная длина текста. Текст будет обрезан после указанного количества символов (по умолчанию: 77). |

## Выходные данные

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `string` | STRING | Сокращенный текст, содержащий только первые `max_length` символов из входных данных. |
