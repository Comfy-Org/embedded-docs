Узел `GLIGENTextBoxApply` предназначен для интеграции текстового кондиционирования во входные данные генеративной модели, а именно для применения параметров текстового поля и их кодирования с помощью модели CLIP. Этот процесс обогащает кондиционирование пространственной и текстовой информацией, обеспечивая более точную и контекстно-зависимую генерацию.

## Входные данные

| Параметр            | Comfy dtype        | Описание |
|----------------------|--------------------|-------------|
| `conditioning_to`     | `CONDITIONING`     | Определяет исходные данные кондиционирования, к которым будут добавлены параметры текстового поля и закодированная текстовая информация. Играет ключевую роль в формировании конечного результата за счет интеграции новых данных кондиционирования. |
| `clip`               | `CLIP`             | Модель CLIP, используемая для кодирования предоставленного текста в формат, пригодный для использования генеративной моделью. Необходима для преобразования текстовой информации в совместимый формат кондиционирования. |
| `gligen_textbox_model` | `GLIGEN`         | Представляет конкретную конфигурацию модели GLIGEN, используемую для генерации текстового поля. Важна для обеспечения генерации текстового поля в соответствии с заданными параметрами. |
| `text`               | `STRING`           | Текстовое содержимое, которое будет закодировано и интегрировано в кондиционирование. Предоставляет семантическую информацию, направляющую работу генеративной модели. |
| `width`              | `INT`              | Ширина текстового поля в пикселях. Определяет пространственный размер текстового поля в сгенерированном изображении. |
| `height`             | `INT`              | Высота текстового поля в пикселях. Аналогично ширине, определяет пространственный размер текстового поля в сгенерированном изображении. |
| `x`                  | `INT`              | Координата X левого верхнего угла текстового поля в сгенерированном изображении. Задает горизонтальное положение текстового поля. |
| `y`                  | `INT`              | Координата Y левого верхнего угла текстового поля в сгенерированном изображении. Задает вертикальное положение текстового поля. |

## Выходные данные

| Параметр            | Comfy dtype        | Описание |
|----------------------|--------------------|-------------|
| `conditioning`        | `CONDITIONING`     | Обогащенные данные кондиционирования, которые включают исходные данные кондиционирования вместе с новыми параметрами текстового поля и закодированной текстовой информацией. Используются для направления генеративной модели при создании контекстно-зависимых результатов. |
