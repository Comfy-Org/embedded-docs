Узел SamplerLMS создаёт сэмплер на основе метода наименьших средних квадратов (LMS) для использования в диффузионных моделях. Он генерирует объект сэмплера, который можно применять в процессе сэмплирования, позволяя управлять порядком алгоритма LMS для обеспечения численной стабильности и точности.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `order` | INT | Да | от 1 до 100 | Параметр порядка для алгоритма сэмплера LMS, который управляет точностью и стабильностью численного метода (по умолчанию: 4) |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `sampler` | SAMPLER | Настроенный объект сэмплера LMS, который можно использовать в конвейере сэмплирования |
