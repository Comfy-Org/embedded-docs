Узел `LTXVSeparateAVLatent` принимает комбинированное аудиовизуальное латентное представление и разделяет его на две отдельные части: одна для видео, другая для аудио. Он разделяет сэмплы и, при наличии, маски шума из входного латента, создавая два новых латентных объекта.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `av_latent` | LATENT | Да | Н/Д | Комбинированное аудиовизуальное латентное представление, которое необходимо разделить. |

**Примечание:** Тензор `samples` входного латента должен содержать как минимум два элемента по первому измерению (измерению пакета). Первый элемент используется для видео-латента, а второй — для аудио-латента. Если присутствует `noise_mask`, он разделяется аналогичным образом.

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `video_latent` | LATENT | Латентное представление, содержащее разделенные видеоданные. |
| `audio_latent` | LATENT | Латентное представление, содержащее разделенные аудиоданные. |
