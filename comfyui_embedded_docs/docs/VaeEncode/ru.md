Этот узел предназначен для кодирования изображений в представление скрытого пространства с использованием указанной модели VAE. Он абстрагирует сложность процесса кодирования, предоставляя простой способ преобразования изображений в их скрытые представления.

## Входные параметры

| Параметр | Тип данных | Описание |
|-----------|-------------|-------------|
| `pixels`  | `IMAGE`     | Параметр `pixels` представляет данные изображения, которые необходимо закодировать в скрытое пространство. Он играет ключевую роль в определении выходного скрытого представления, выступая в качестве прямого входа для процесса кодирования. |
| `vae`     | VAE       | Параметр `vae` определяет модель вариационного автоэнкодера, используемую для кодирования данных изображения в скрытое пространство. Он необходим для определения механизма кодирования и характеристик создаваемого скрытого представления. |

## Выходные параметры

| Параметр | Тип данных | Описание |
|-----------|-------------|-------------|
| `latent`  | `LATENT`    | Выходные данные представляют собой представление входного изображения в скрытом пространстве, инкапсулирующее его основные характеристики в сжатой форме. |
