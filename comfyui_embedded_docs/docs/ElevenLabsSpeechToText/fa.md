### گره تبدیل گفتار به متن ElevenLabs

گره تبدیل گفتار به متن ElevenLabs، فایل‌های صوتی را به متن تبدیل می‌کند. این گره از API شرکت ElevenLabs برای تبدیل کلمات گفتاری به متن نوشتاری استفاده می‌کند و از ویژگی‌هایی مانند تشخیص خودکار زبان، شناسایی گویندگان مختلف و برچسب‌گذاری صداهای غیرگفتاری مانند موسیقی یا خنده پشتیبانی می‌کند.

## ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|---------|----------|-------|--------|---------|
| `audio` | AUDIO | بله | - | فایل صوتی برای تبدیل به متن. |
| `model` | COMBO | بله | `"scribe_v2"` | مدل مورد استفاده برای تبدیل گفتار به متن. انتخاب این مدل، پارامترهای اضافی را نمایش می‌دهد. |
| `tag_audio_events` | BOOLEAN | خیر | - | برچسب‌گذاری صداهایی مانند (خنده)، (موسیقی) و غیره در متن خروجی. این پارامتر زمانی نمایش داده می‌شود که مدل `"scribe_v2"` انتخاب شده باشد. (پیش‌فرض: False) |
| `diarize` | BOOLEAN | خیر | - | مشخص کردن اینکه کدام گوینده در حال صحبت است. این پارامتر زمانی نمایش داده می‌شود که مدل `"scribe_v2"` انتخاب شده باشد. (پیش‌فرض: False) |
| `diarization_threshold` | FLOAT | خیر | 0.1 - 0.4 | حساسیت تفکیک گویندگان. مقادیر پایین‌تر به تغییرات گوینده حساس‌تر هستند. این پارامتر زمانی نمایش داده می‌شود که مدل `"scribe_v2"` انتخاب شده و گزینه `diarize` فعال باشد. (پیش‌فرض: 0.22) |
| `temperature` | FLOAT | خیر | 0.0 - 2.0 | کنترل تصادفی بودن خروجی. مقدار 0.0 از پیش‌فرض مدل استفاده می‌کند. مقادیر بالاتر تصادفی بودن را افزایش می‌دهد. این پارامتر زمانی نمایش داده می‌شود که مدل `"scribe_v2"` انتخاب شده باشد. (پیش‌فرض: 0.0) |
| `timestamps_granularity` | COMBO | خیر | `"word"`<br>`"character"`<br>`"none"` | دقت زمان‌بندی برای کلمات متن خروجی. این پارامتر زمانی نمایش داده می‌شود که مدل `"scribe_v2"` انتخاب شده باشد. (پیش‌فرض: "word") |
| `language_code` | STRING | خیر | - | کد زبان بر اساس استاندارد ISO-639-1 یا ISO-639-3 (مانند 'en'، 'es'، 'fra'). برای تشخیص خودکار خالی بگذارید. (پیش‌فرض: "") |
| `num_speakers` | INT | خیر | 0 - 32 | حداکثر تعداد گویندگان برای پیش‌بینی. برای تشخیص خودکار مقدار 0 را تنظیم کنید. (پیش‌فرض: 0) |
| `seed` | INT | خیر | 0 - 2147483647 | دانه (seed) برای تکرارپذیری (تکرارپذیری تضمین نمی‌شود). (پیش‌فرض: 1) |

**توجه:** پارامتر `num_speakers` زمانی که گزینه `diarize` فعال است، نمی‌تواند مقداری بیشتر از 0 داشته باشد. باید یا گزینه `diarize` را غیرفعال کنید یا `num_speakers` را روی 0 تنظیم نمایید.

## خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
