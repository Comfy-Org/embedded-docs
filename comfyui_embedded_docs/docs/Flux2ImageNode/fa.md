### مرور کلی

با استفاده از مدل Flux.2 [pro] یا Flux.2 [max] از روی یک متن راهنما (prompt) و تصاویر مرجع اختیاری، تصویر تولید کنید. این گره درخواست شما را به API سرویس BFL ارسال می‌کند، نتیجه را دریافت می‌کند و تصویر تولیدشده را به صورت یک تنسور بازمی‌گرداند.

### ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `prompt` | STRING | بله | نامشخص | متن راهنما برای تولید یا ویرایش تصویر (پیش‌فرض: رشته خالی). |
| `model` | COMBO | بله | `"Flux.2 [pro]"`<br>`"Flux.2 [max]"` | نسخه مدل Flux.2 مورد استفاده. انتخاب یک مدل، پارامترهای اضافی برای عرض، ارتفاع و تصاویر مرجع اختیاری را فعال می‌کند. |
| `seed` | INT | بله | 0 تا 18446744073709551615 | دانه تصادفی (seed) مورد استفاده برای ایجاد نویز. می‌توان آن را برای تصادفی‌سازی پس از هر تولید تنظیم کرد (پیش‌فرض: 0). |

**پارامترهای اضافی (با انتخاب `model` فعال می‌شوند):**

هنگامی که یک مدل را انتخاب می‌کنید، پارامترهای زیر در دسترس قرار می‌گیرند:

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `model.width` | INT | بله | 256 تا 1440 | عرض تصویر تولیدشده بر حسب پیکسل. |
| `model.height` | INT | بله | 256 تا 1440 | ارتفاع تصویر تولیدشده بر حسب پیکسل. |
| `model.images` | IMAGE | خیر | 0 تا 8 تصویر | تصاویر مرجع اختیاری برای هدایت تولید. حداکثر 8 تصویر پشتیبانی می‌شود. |

**محدودیت‌ها:**
- حداکثر تعداد تصاویر مرجع 8 عدد است. اگر بیش از 8 تصویر ارائه شود، خطا ایجاد می‌شود.
- مقادیر `model.width` و `model.height` بر هزینه تولید تأثیر می‌گذارند (به منطق نشان‌بهای قیمت در کد منبع مراجعه کنید).

### خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-------------|-----------|-------------|
| `image` | IMAGE | تصویر تولیدشده به صورت یک تنسور که از نتیجه API سرویس BFL دریافت شده است. |
