# SeedVR2TemporalChunk

Этот узел разделяет видео-латент SeedVR2 на меньшие перекрывающиеся временные фрагменты, которые могут обрабатываться по одному в рамках доступной видеопамяти. Фрагменты предназначены для передачи как в узел Apply SeedVR2 Conditioning, так и на вход латента сэмплера, а затем повторно объединяются с помощью узла Merge SeedVR2 Latents.

## Входы

| Параметр | Описание | Тип данных | Обязательный | Диапазон |
|----------|----------|------------|--------------|----------|
| `latent` | Закодированный VAE латент SeedVR2 для разделения | LATENT | Да | - |
| `temporal_overlap` | Кадры латента, общие для соседних фрагментов и перекрестно смешиваемые при слиянии; 0 означает отсутствие перекрытия (по умолчанию: 0) | INT | Нет | от 0 до 16384 |
| `chunking_mode` | manual = использовать frames_per_chunk точно; auto = предсказать наибольший фрагмент, помещающийся в свободную видеопамять | COMBO | Да | "auto"<br>"manual" |

Если для `chunking_mode` установлено значение "manual", становится доступен следующий параметр:

| Параметр | Описание | Тип данных | Обязательный | Диапазон |
|----------|----------|------------|--------------|----------|
| `frames_per_chunk` | Пиксельные кадры на временной фрагмент (должно быть 4n+1: 1, 5, 9, 13, ...) (по умолчанию: 21) | INT | Да | от 1 до 16384, шаг 4 |

**Примечания к ограничениям параметров:**
- Входной `latent` должен быть 5-мерным видео-латентным представлением с формой (B, C, T, H, W) и ровно 4 каналами латента
- При использовании режима "manual" значение `frames_per_chunk` должно быть 4n+1 (1, 5, 9, 13, 17, 21, ...)
- `temporal_overlap` автоматически ограничивается значением, меньшим размера фрагмента
- В режиме "auto" узел вычисляет оптимальный размер фрагмента на основе доступной свободной видеопамяти

## Выходы

| Имя выхода | Описание | Тип данных |
|------------|----------|------------|
| `latents` | Временные фрагменты в порядке следования | LATENT |
| `temporal_overlap` | Эффективное перекрытие кадров латента между соседними фрагментами для узла Merge SeedVR2 Latents | INT |
