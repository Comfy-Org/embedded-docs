### مرور کلی
موسیقی و افکت‌های صوتی با کیفیت بالا را از توضیحات متنی تولید می‌کند. این گره از فناوری تولید صوتی Stability AI برای ایجاد محتوای صوتی بر اساس متن ورودی شما استفاده می‌کند.

## ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `مدل` | COMBO | بله | `"stable-audio-2.5"` | مدل تولید صوتی مورد استفاده (پیش‌فرض: "stable-audio-2.5") |
| `پرامپت` | STRING | بله | - | توضیحات متنی برای تولید محتوای صوتی (پیش‌فرض: رشته خالی) |
| `مدت زمان` | INT | خیر | 1-190 | مدت زمان خروجی صوتی را بر حسب ثانیه کنترل می‌کند (پیش‌فرض: 190) |
| `seed` | INT | خیر | 0-4294967294 | دانه تصادفی (seed) مورد استفاده برای تولید (پیش‌فرض: 0) |
| `گام‌ها` | INT | خیر | 4-8 | تعداد مراحل نمونه‌برداری را کنترل می‌کند (پیش‌فرض: 8) |
