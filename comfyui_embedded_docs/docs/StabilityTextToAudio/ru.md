Генерирует высококачественную музыку и звуковые эффекты на основе текстовых описаний. Этот узел использует технологию генерации аудио от Stability AI для создания аудиоконтента на основе ваших текстовых подсказок.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `модель` | COMBO | Да | `"stable-audio-2.5"` | Модель генерации аудио для использования (по умолчанию: "stable-audio-2.5") |
| `промпт` | STRING | Да | - | Текстовое описание для генерации аудиоконтента (по умолчанию: пустая строка) |
| `длительность` | INT | Нет | 1-190 | Управляет длительностью генерируемого аудио в секундах (по умолчанию: 190) |
| `сид` | INT | Нет | 0-4294967294 | Случайное зерно, используемое для генерации (по умолчанию: 0) |
| `шаги` | INT | Нет | 4-8 | Управляет количеством шагов дискретизации (по умолчанию: 8) |

## Выходные данные

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `audio` | AUDIO | Сгенерированный аудиофайл на основе текстовой подсказки |
