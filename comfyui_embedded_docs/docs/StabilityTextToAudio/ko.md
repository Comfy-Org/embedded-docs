# 개요

텍스트 설명으로부터 고품질의 음악과 음향 효과를 생성합니다. 이 노드는 Stability AI의 오디오 생성 기술을 사용하여 텍스트 프롬프트를 기반으로 오디오 콘텐츠를 만듭니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `모델` | COMBO | 예 | `"stable-audio-2.5"` | 사용할 오디오 생성 모델입니다 (기본값: "stable-audio-2.5") |
| `프롬프트` | STRING | 예 | - | 오디오 콘텐츠를 생성하는 데 사용되는 텍스트 설명입니다 (기본값: 빈 문자열) |
| `지속 시간` | INT | 아니요 | 1-190 | 생성된 오디오의 길이를 초 단위로 제어합니다 (기본값: 190) |
| `시드` | INT | 아니요 | 0-4294967294 | 생성에 사용되는 무작위 시드입니다 (기본값: 0) |
| `단계` | INT | 아니요 | 4-8 | 샘플링 단계 수를 제어합니다 (기본값: 8) |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `audio` | AUDIO | 텍스트 프롬프트를 기반으로 생성된 오디오 파일입니다 |
