هذه ترجمة الوثيقة التقنية لعقدة ComfyUI وفقًا للقواعد المحددة:

عقدة إضافة بادئة نصية (Add Text Prefix) تقوم بتعديل النص عن طريق إضافة سلسلة نصية محددة إلى بداية كل نص مدخل. تستقبل العقدة النص والبادئة كمدخلات، ثم تُرجع النتيجة المجمعة.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|-----------|-----------|----------|-------|-------------|
| `text` | STRING | نعم | | النص الذي ستُضاف إليه البادئة. |
| `البادئة` | STRING | لا | | السلسلة النصية المراد إضافتها إلى بداية النص (القيمة الافتراضية: ""). |

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|-------------|-----------|-------------|
| `text` | STRING | النص الناتج مع إضافة البادئة إلى المقدمة. |
