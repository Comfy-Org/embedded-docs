Вот перевод документации на русский язык:

Узел ReplaceVideoLatentFrames вставляет кадры из исходного латентного видео в целевое латентное видео, начиная с указанного индекса кадра. Если исходный латент не предоставлен, целевой латент возвращается без изменений. Узел обрабатывает отрицательные индексы и выдает предупреждение, если исходные кадры не помещаются в целевое видео.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `destination` | LATENT | Да | - | Целевой латент, в котором будут заменяться кадры. |
| `source` | LATENT | Нет | - | Исходный латент, предоставляющий кадры для вставки в целевой латент. Если не указан, целевой латент возвращается без изменений. |
| `index` | INT | Да | от -MAX_RESOLUTION до MAX_RESOLUTION | Начальный индекс латентного кадра в целевом латенте, куда будут помещены кадры из исходного латента. Отрицательные значения отсчитываются от конца (по умолчанию: 0). |

**Ограничения:**

* Значение `index` должно находиться в пределах количества кадров целевого латента. Если это не так, выводится предупреждение, и целевой латент возвращается без изменений.
* Кадры исходного латента должны помещаться в кадры целевого латента, начиная с указанного `index`. Если это не так, выводится предупреждение, и целевой латент возвращается без изменений.

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `output` | LATENT | Результирующее латентное видео после операции замены кадров. |
