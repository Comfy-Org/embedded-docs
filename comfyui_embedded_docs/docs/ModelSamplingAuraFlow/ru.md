Вот перевод документации на русский язык:

Узел ModelSamplingAuraFlow применяет специализированную конфигурацию сэмплирования к диффузионным моделям, специально разработанную для архитектур моделей AuraFlow. Он изменяет поведение сэмплирования модели, применяя параметр сдвига, который корректирует распределение сэмплирования. Этот узел наследуется от фреймворка сэмплирования модели SD3 и обеспечивает точный контроль над процессом сэмплирования.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `model` | MODEL | Да | - | Диффузионная модель, к которой применяется конфигурация сэмплирования AuraFlow |
| `shift` | FLOAT | Да | 0.0 - 100.0 | Значение сдвига, применяемое к распределению сэмплирования (по умолчанию: 1.73) |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `model` | MODEL | Модифицированная модель с примененной конфигурацией сэмплирования AuraFlow |
