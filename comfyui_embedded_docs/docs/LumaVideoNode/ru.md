Вот перевод документации на русский язык:

Генерирует видео синхронно на основе текстового запроса и настроек вывода. Этот узел создает видеоконтент, используя текстовые описания и различные параметры генерации, и выдает готовый видеофайл после завершения процесса генерации.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `prompt` | STRING | Да | - | Запрос для генерации видео (по умолчанию: пустая строка). Должен содержать не менее 3 символов. |
| `model` | COMBO | Да | `"ray_1_6"`<br>`"ray_2"` | Модель генерации видео для использования. |
| `aspect_ratio` | COMBO | Да | `"16:9"`<br>`"9:16"`<br>`"1:1"`<br>`"4:3"`<br>`"3:4"`<br>`"21:9"`<br>`"9:21"` | Соотношение сторон для генерируемого видео (по умолчанию: "16:9"). |
| `resolution` | COMBO | Да | `"540p"`<br>`"720p"`<br>`"1080p"` | Разрешение выходного видео (по умолчанию: "540p"). Этот параметр игнорируется при использовании модели `ray_1_6`. |
| `duration` | COMBO | Да | `"5s"`<br>`"9s"` | Длительность генерируемого видео. Этот параметр игнорируется при использовании модели `ray_1_6`. |
| `loop` | BOOLEAN | Да | - | Должно ли видео зацикливаться (по умолчанию: False). |
| `seed` | INT | Да | от 0 до 18446744073709551615 | Сид для определения необходимости повторного запуска узла; фактические результаты недетерминированы независимо от сида (по умолчанию: 0). |
| `luma_concepts` | CUSTOM | Нет | - | Необязательные концепции камеры для управления движением камеры через узел Luma Concepts. |

**Примечание:** При использовании модели `ray_1_6` параметры `duration` и `resolution` автоматически игнорируются и не влияют на генерацию.

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `output` | VIDEO | Сгенерированный видеофайл. |
