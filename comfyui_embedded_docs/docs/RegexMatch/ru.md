Узел RegexMatch проверяет, содержит ли текстовая строка совпадение с заданным регулярным выражением. Он ищет во входной строке и возвращает простой результат «да/нет», указывающий, был ли найден шаблон в тексте. Вы можете настроить поиск, включив такие опции, как игнорирование регистра или многострочный режим.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `строка` | STRING | Да | - | Текстовая строка для поиска совпадений |
| `регулярное_выражение` | STRING | Да | - | Шаблон регулярного выражения для сопоставления со строкой |
| `без_учета_регистра` | BOOLEAN | Нет | - | Игнорировать ли регистр при сопоставлении (по умолчанию: True) |
| `многострочный` | BOOLEAN | Нет | - | Включать ли многострочный режим для сопоставления регулярных выражений (по умолчанию: False) |
| `dotall` | BOOLEAN | Нет | - | Включать ли режим dotall для сопоставления регулярных выражений (по умолчанию: False) |

## Выходные параметры

| Имя выходного параметра | Тип данных | Описание |
|-------------------------|------------|----------|
| `matches` | BOOLEAN | Возвращает True, если шаблон регулярного выражения совпадает с любой частью входной строки, иначе False |
