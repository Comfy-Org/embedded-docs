گره RegexMatch بررسی می‌کند که آیا یک رشته متنی شامل تطابقی با الگوی عبارت باقاعده (regex) داده شده است یا خیر. این گره رشته ورودی را جستجو کرده و یک نتیجه ساده بله/خیر برمی‌گرداند که نشان می‌دهد آیا الگو در هر جای متن پیدا شده است یا نه. می‌توانید نحوه عملکرد جستجو را با فعال‌سازی گزینه‌هایی مانند تطبیق بدون حساسیت به بزرگی/کوچکی حروف یا حالت چندخطی تنظیم کنید.

## ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|---------|----------|-------|--------|---------|
| `رشته` | STRING | بله | - | رشته متنی که باید برای یافتن تطابق جستجو شود |
| `الگوی regex` | STRING | بله | - | الگوی عبارت باقاعده برای تطبیق با رشته |
| `بدون حساسیت به حروف` | BOOLEAN | خیر | - | آیا هنگام تطبیق، بزرگی/کوچکی حروف نادیده گرفته شود (پیش‌فرض: True) |
| `چندخطی` | BOOLEAN | خیر | - | آیا حالت چندخطی برای تطبیق عبارت باقاعده فعال شود (پیش‌فرض: False) |
| `dotall` | BOOLEAN | خیر | - | آیا حالت dotall برای تطبیق عبارت باقاعده فعال شود (پیش‌فرض: False) |

## خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-----------|----------|---------|
| `matches` | BOOLEAN | در صورت تطبیق الگوی عبارت باقاعده با هر بخشی از رشته ورودی، True و در غیر این صورت False برمی‌گرداند |
