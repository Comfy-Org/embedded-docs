# ترجمة وثيقة عقدة RegexMatch

تتحقق عقدة RegexMatch مما إذا كان النص يحتوي على تطابق مع نمط تعبير نمطي (Regex) معين. تبحث العقدة في السلسلة النصية المدخلة وتعيد نتيجة بسيطة بنعم/لا تشير إلى ما إذا تم العثور على النمط في أي مكان داخل النص. يمكنك تعديل طريقة عمل البحث من خلال تفعيل خيارات مثل المطابقة غير حساسة لحالة الأحرف أو وضع الأسطر المتعددة.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|---------|---------------|--------|---------|-------------|
| `السلسلة النصية` | STRING | نعم | - | السلسلة النصية المراد البحث فيها عن التطابقات |
| `نمط التعبير النمطي` | STRING | نعم | - | نمط التعبير النمطي للمطابقة مع السلسلة النصية |
| `غير حساس لحالة الأحرف` | BOOLEAN | لا | - | تجاهل حالة الأحرف عند المطابقة (الافتراضي: True) |
| `متعدد الأسطر` | BOOLEAN | لا | - | تفعيل وضع الأسطر المتعددة لمطابقة التعبير النمطي (الافتراضي: False) |
| `dotall` | BOOLEAN | لا | - | تفعيل وضع النقطة للكل لمطابقة التعبير النمطي (الافتراضي: False) |

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|-------------|---------------|-------------|
| `matches` | BOOLEAN | تُرجع True إذا تطابق نمط التعبير النمطي مع أي جزء من السلسلة النصية المدخلة، وترجع False في الحالة المعاكسة |
