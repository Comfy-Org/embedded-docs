RegexMatch 節點會檢查文字字串是否包含符合指定正規表示式模式的內容。它會搜尋輸入字串，並回傳簡單的是/否結果，指示該模式是否存在於文字中的任何位置。您可以透過啟用不區分大小寫比對或多行模式等選項來調整搜尋方式。

## 輸入

| 參數 | 資料類型 | 必要 | 範圍 | 說明 |
|-----------|-----------|----------|-------|-------------|
| `字串` | STRING | 是 | - | 要搜尋比對的文字字串 |
| `正則表達式模式` | STRING | 是 | - | 用於比對字串的正規表示式模式 |
| `忽略大小寫` | BOOLEAN | 否 | - | 比對時是否忽略大小寫（預設：True） |
| `多行模式` | BOOLEAN | 否 | - | 是否啟用正規表示式比對的多行模式（預設：False） |
| `點號匹配所有` | BOOLEAN | 否 | - | 是否啟用正規表示式比對的 dotall 模式（預設：False） |

## 輸出

| 輸出名稱 | 資料類型 | 說明 |
|-------------|-----------|-------------|
| `matches` | BOOLEAN | 如果正規表示式模式比對到輸入字串的任何部分，則回傳 True，否則回傳 False |
