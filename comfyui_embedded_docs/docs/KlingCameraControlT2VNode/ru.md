Вот перевод документации на русский язык:

Kling Text to Video Camera Control Node преобразует текст в кинематографичные видео с профессиональными движениями камеры, имитирующими реальную киносъемку. Этот узел поддерживает управление виртуальными движениями камеры, включая зуммирование, вращение, панорамирование, наклон и вид от первого лица, сохраняя фокус на вашем исходном тексте. Продолжительность, режим и название модели жестко заданы, поскольку управление камерой поддерживается только в профессиональном режиме с моделью kling-v1-5 при длительности 5 секунд.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `prompt` | STRING | Да | - | Положительный текстовый запрос |
| `negative_prompt` | STRING | Да | - | Отрицательный текстовый запрос |
| `cfg_scale` | FLOAT | Нет | 0.0-1.0 | Определяет, насколько точно результат следует запросу (по умолчанию: 0.75) |
| `aspect_ratio` | COMBO | Нет | "16:9"<br>"9:16"<br>"1:1"<br>"21:9"<br>"3:4"<br>"4:3" | Соотношение сторон для создаваемого видео (по умолчанию: "16:9") |
| `camera_control` | CAMERA_CONTROL | Нет | - | Может быть создан с помощью узла Kling Camera Controls. Управляет движением камеры и перемещением во время генерации видео. |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `output` | VIDEO | Сгенерированное видео с эффектами управления камерой |
| `video_id` | STRING | Уникальный идентификатор сгенерированного видео |
| `duration` | STRING | Продолжительность сгенерированного видео |
