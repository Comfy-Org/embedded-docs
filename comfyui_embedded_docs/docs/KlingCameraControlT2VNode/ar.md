# ترجمة وثيقة KlingCameraControlT2VNode

## نظرة عامة

تقوم عقدة التحكم في كاميرا تحويل النص إلى فيديو من Kling بتحويل النصوص إلى فيديوهات سينمائية مع حركات كاميرا احترافية تحاكي التصوير السينمائي الواقعي. تدعم هذه العقدة التحكم في حركات الكاميرا الافتراضية بما في ذلك التكبير، والتدوير، والتحريك الأفقي، والتحريك العمودي، ومنظور الرؤية الأولى، مع الحفاظ على التركيز على النص الأصلي. المدة والوضع واسم النموذج مُحدَّدة مسبقًا لأن التحكم في الكاميرا مدعوم فقط في الوضع الاحترافي مع نموذج kling-v1-5 بمدة 5 ثوانٍ.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|---------|--------------|--------|--------|-------|
| `prompt` | STRING | نعم | - | موجه النص الإيجابي |
| `negative_prompt` | STRING | نعم | - | موجه النص السلبي |
| `cfg_scale` | FLOAT | لا | 0.0-1.0 | يتحكم في مدى التزام المخرجات بالموجه (الافتراضي: 0.75) |
| `aspect_ratio` | COMBO | لا | "16:9"<br>"9:16"<br>"1:1"<br>"21:9"<br>"3:4"<br>"4:3" | نسبة العرض إلى الارتفاع للفيديو المُنشأ (الافتراضي: "16:9") |
| `camera_control` | CAMERA_CONTROL | لا | - | يمكن إنشاؤه باستخدام عقدة عناصر تحكم كاميرا Kling. يتحكم في حركة الكاميرا والحركة أثناء إنشاء الفيديو. |

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|------------|--------------|--------|
| `output` | VIDEO | الفيديو المُنشأ مع تأثيرات التحكم في الكاميرا |
| `video_id` | STRING | المعرف الفريد للفيديو المُنشأ |
| `duration` | STRING | مدة الفيديو المُنشأ |
