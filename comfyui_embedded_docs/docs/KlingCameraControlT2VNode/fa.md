### نمای کلی

گره **کنترل دوربین Kling متن به ویدیو**، متن را با حرکات حرفه‌ای دوربین که سینماتوگرافی واقعی را شبیه‌سازی می‌کند، به ویدیوهای سینمایی تبدیل می‌کند. این گره از کنترل حرکات دوربین مجازی شامل زوم، چرخش، پن، تیلت و نمای اول شخص پشتیبانی می‌کند و در عین حال تمرکز بر متن اصلی شما را حفظ می‌کند. مدت زمان، حالت و نام مدل به صورت ثابت کدنویسی شده‌اند زیرا کنترل دوربین فقط در حالت حرفه‌ای با مدل kling-v1-5 و مدت زمان ۵ ثانیه پشتیبانی می‌شود.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|---------|----------|--------|--------|---------|
| `prompt` | STRING | بله | - | پرامپت متنی مثبت |
| `negative_prompt` | STRING | بله | - | پرامپت متنی منفی |
| `cfg_scale` | FLOAT | خیر | 0.0-1.0 | میزان تطابق خروجی با پرامپت را کنترل می‌کند (پیش‌فرض: 0.75) |
| `aspect_ratio` | COMBO | خیر | "16:9"<br>"9:16"<br>"1:1"<br>"21:9"<br>"3:4"<br>"4:3" | نسبت ابعاد ویدیوی تولید شده (پیش‌فرض: "16:9") |
| `camera_control` | CAMERA_CONTROL | خیر | - | می‌تواند با استفاده از گره کنترل‌های دوربین Kling ایجاد شود. حرکت و جابجایی دوربین را در طول تولید ویدیو کنترل می‌کند. |

## خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-----------|----------|---------|
| `output` | VIDEO | ویدیوی تولید شده با افکت‌های کنترل دوربین |
| `video_id` | STRING | شناسه یکتای ویدیوی تولید شده |
| `duration` | STRING | مدت زمان ویدیوی تولید شده |
