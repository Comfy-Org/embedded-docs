Узел StringLength вычисляет количество символов в текстовой строке. Он принимает любой текстовый ввод и возвращает общее количество символов, включая пробелы и знаки препинания. Это полезно для измерения длины текста или проверки требований к размеру строки.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `string` | STRING | Да | Н/Д | Текстовая строка, длину которой необходимо измерить. Поддерживает многострочный ввод. |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `length` | INT | Общее количество символов во входной строке, включая пробелы и специальные символы. |
