Вот перевод документации на русский язык:

Узел Runway Image to Video (Gen3a Turbo) генерирует видео из одного начального кадра с использованием модели Gen3a Turbo от Runway. Он принимает текстовый запрос и начальное изображение, затем создает видеопоследовательность на основе заданной длительности и соотношения сторон. Этот узел подключается к API Runway для удаленной обработки генерации.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `промпт` | STRING | Да | Н/Д | Текстовый запрос для генерации (по умолчанию: "") |
| `начальный_кадр` | IMAGE | Да | Н/Д | Начальный кадр для создания видео |
| `продолжительность` | COMBO | Да | `"5"`<br>`"10"` | Длительность видео в секундах (по умолчанию: "5") |
| `соотношение` | COMBO | Да | `"1280x720"`<br>`"720x1280"`<br>`"1920x1080"`<br>`"1080x1920"`<br>`"1080x1080"` | Соотношение сторон генерируемого видео (по умолчанию: "1280x720") |
| `сид` | INT | Нет | от 0 до 4294967295 | Случайное зерно для генерации (по умолчанию: 0) |

**Ограничения параметров:**

- Размеры `start_frame` не должны превышать 7999x7999 пикселей.
- Соотношение сторон `start_frame` должно быть в диапазоне от 0,5 до 2,0.
- `prompt` должен содержать как минимум один символ (не может быть пустым).

## Выходные данные

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `output` | VIDEO | Сгенерированная видеопоследовательность |
