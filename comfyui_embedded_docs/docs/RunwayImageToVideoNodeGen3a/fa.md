گره «تبدیل تصویر به ویدیو Runway (Gen3a Turbo)» با استفاده از مدل Gen3a Turbo شرکت Runway، یک ویدیو از یک فریم شروع تولید می‌کند. این گره یک متن راهنما (prompt) و یک تصویر اولیه دریافت کرده و بر اساس مدت‌زمان و نسبت تصویر مشخص‌شده، یک دنباله ویدیویی ایجاد می‌کند. این گره برای پردازش از راه دور به API شرکت Runway متصل می‌شود.

## ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|---------|----------|-------|--------|---------|
| `پرامپت` | STRING | بله | نامشخص | متن راهنما برای تولید (پیش‌فرض: "") |
| `فریم شروع` | IMAGE | بله | نامشخص | فریم شروع برای استفاده در ویدیو |
| `مدت زمان` | COMBO | بله | `"5"`<br>`"10"` | مدت‌زمان ویدیو بر حسب ثانیه (پیش‌فرض: "5") |
| `نسبت تصویر` | COMBO | بله | `"1280x720"`<br>`"720x1280"`<br>`"1920x1080"`<br>`"1080x1920"`<br>`"1080x1080"` | نسبت تصویر ویدیوی تولیدشده (پیش‌فرض: "1280x720") |
| `seed` | INT | خیر | 0 تا 4294967295 | دانه تصادفی برای تولید (پیش‌فرض: 0) |

**محدودیت‌های پارامتر:**

- ابعاد `start_frame` نباید از 7999x7999 پیکسل تجاوز کند.
- نسبت تصویر `start_frame` باید بین 0.5 و 2.0 باشد.
- `prompt` باید حداقل شامل یک نویسه باشد (نمی‌تواند خالی باشد).
