Вот перевод документации на русский язык:

Узел LoadLatent загружает ранее сохранённые латентные представления из файлов .latent в папке input. Он считывает тензорные данные латентного представления из файла и применяет необходимые корректировки масштабирования перед возвратом данных для использования в других узлах.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `latent` | STRING | Да | Все файлы .latent в папке input | Выбирает, какой файл .latent загрузить из доступных файлов в папке input |

## Выходные данные

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `LATENT` | LATENT | Возвращает загруженные данные латентного представления из выбранного файла |
