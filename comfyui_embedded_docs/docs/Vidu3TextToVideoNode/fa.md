### نمای کلی

گره **تولید ویدیو از متن Vidu Q3** یک ویدیو بر اساس توضیحات متنی ایجاد می‌کند. این گره از مدل Vidu Q3 Pro یا Q3 Turbo برای تولید محتوای ویدیویی بر اساس دستور متنی شما استفاده می‌کند و به شما امکان کنترل طول، وضوح، نسبت تصویر و وجود یا عدم وجود صدا در ویدیو را می‌دهد.

### ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `model` | COMBO | بله | `"viduq3-pro"`<br>`"viduq3-turbo"` | مدل مورد استفاده برای تولید ویدیو. انتخاب یک مدل، پارامترهای پیکربندی اضافی برای نسبت تصویر، وضوح، مدت زمان و صدا را نمایش می‌دهد. |
| `model.aspect_ratio` | COMBO | بله* | `"16:9"`<br>`"9:16"`<br>`"3:4"`<br>`"4:3"`<br>`"1:1"` | نسبت تصویر ویدیوی خروجی. این پارامتر پس از انتخاب یک `model` نمایش داده می‌شود. |
| `model.resolution` | COMBO | بله* | `"720p"`<br>`"1080p"` | وضوح ویدیوی خروجی. این پارامتر پس از انتخاب یک `model` نمایش داده می‌شود. |
| `model.duration` | INT | بله* | 1 تا 16 | مدت زمان ویدیوی خروجی بر حسب ثانیه (پیش‌فرض: 5). این پارامتر پس از انتخاب یک `model` نمایش داده می‌شود. |
| `model.audio` | BOOLEAN | بله* | True/False | در صورت فعال بودن، ویدیو با صدا (شامل دیالوگ و افکت‌های صوتی) خروجی می‌دهد (پیش‌فرض: False). این پارامتر پس از انتخاب یک `model` نمایش داده می‌شود. |
| `prompt` | STRING | بله | نامشخص | توضیحات متنی برای تولید ویدیو، با حداکثر طول ۲۰۰۰ کاراکتر. |
| `seed` | INT | خیر | 0 تا 2147483647 | یک مقدار seed برای کنترل تصادفی بودن تولید (پیش‌فرض: 1). |

*توجه: پارامترهای `aspect_ratio`، `resolution`، `duration` و `audio` پس از انتخاب یک `model` الزامی می‌شوند، زیرا بخشی از پیکربندی آن هستند.
