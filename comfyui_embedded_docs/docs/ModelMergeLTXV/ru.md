Узел `ModelMergeLTXV` выполняет операции продвинутого слияния моделей, специально разработанные для архитектур моделей LTXV. Он позволяет смешивать две разные модели путем настройки весов интерполяции для различных компонентов модели, включая блоки трансформера, проекционные слои и другие специализированные модули.
