Узел LTXVPreprocess применяет предварительную обработку сжатия к изображениям. Он принимает входные изображения и обрабатывает их с заданным уровнем сжатия, выводя обработанные изображения с применёнными настройками сжатия.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `изображение` | IMAGE | Да | - | Входное изображение для обработки |
| `сжатие изображения` | INT | Нет | 0-100 | Степень сжатия, применяемая к изображению (по умолчанию: 35) |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `output_image` | IMAGE | Обработанное выходное изображение с применённым сжатием |
