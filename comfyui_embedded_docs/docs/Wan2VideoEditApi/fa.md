گره Wan2VideoEditApi از مدل Wan 2.7 برای ویرایش ویدئو بر اساس دستورات متنی، تصاویر مرجع یا انتقال سبک استفاده می‌کند. این گره ویدئوی ورودی را پردازش کرده و با توجه به پارامترهای مشخص‌شده مانند رزولوشن، مدت زمان و نسبت تصویر، ویدئوی جدیدی تولید می‌کند.

## ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `مدل` | COMBO | بله | `"wan2.7-videoedit"` | مدل مورد استفاده برای ویرایش ویدئو. |
| `model.prompt` | STRING | بله | - | دستورالعمل‌های ویرایش یا الزامات انتقال سبک. (پیش‌فرض: رشته خالی) |
| `model.resolution` | COMBO | بله | `"720P"`<br>`"1080P"` | رزولوشن ویدئوی خروجی. |
| `model.ratio` | COMBO | بله | `"16:9"`<br>`"9:16"`<br>`"1:1"`<br>`"4:3"`<br>`"3:4"` | نسبت تصویر ویدئوی خروجی. در صورت عدم تغییر، به نسبت تصویر ویدئوی ورودی نزدیک می‌شود. |
| `model.duration` | COMBO | بله | `"auto"`<br>`"2"`<br>`"3"`<br>`"4"`<br>`"5"`<br>`"6"`<br>`"7"`<br>`"8"`<br>`"9"`<br>`"10"` | مدت زمان خروجی بر حسب ثانیه. گزینه 'auto' با مدت زمان ویدئوی ورودی مطابقت دارد. مقدار مشخص، ویدئو را از ابتدا کوتاه می‌کند. (پیش‌فرض: "auto") |
| `model.reference_images` | IMAGE | خیر | - | فهرستی حداکثر از ۴ تصویر مرجع برای هدایت ویرایش. |
| `ویدیو` | VIDEO | بله | - | ویدئوی مورد نظر برای ویرایش. |
| `بذر` | INT | خیر | 0 تا 2147483647 | دانه (seed) مورد استفاده برای تولید. (پیش‌فرض: 0) |
| `تنظیمات صدا` | COMBO | خیر | `"auto"`<br>`"origin"` | 'auto': مدل تصمیم می‌گیرد که آیا بر اساس prompt صدا را بازتولید کند یا خیر. 'origin': صدای اصلی ویدئوی ورودی حفظ می‌شود. (پیش‌فرض: "auto") |
| `واترمارک` | BOOLEAN | خیر | - | آیا به نتیجه، واترمارک تولیدشده توسط هوش مصنوعی اضافه شود یا خیر. (پیش‌فرض: False) |

**محدودیت‌ها:**
*   `model.prompt` باید حداقل ۱ کاراکتر داشته باشد.
*   ویدئوی ورودی `video` باید بین ۲ تا ۱۰ ثانیه مدت زمان داشته باشد.
*   ورودی `model.reference_images` حداکثر می‌تواند ۴ تصویر را بپذیرد.
