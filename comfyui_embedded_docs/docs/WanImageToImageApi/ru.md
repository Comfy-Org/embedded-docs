Вот перевод документации узла WanImageToImageApi на русский язык:

Узел Wan Image to Image генерирует изображение на основе одного или двух входных изображений и текстового запроса. Он преобразует ваши входные изображения в соответствии с заданным описанием, создавая новое изображение, сохраняющее соотношение сторон исходного входа. Выходное изображение имеет фиксированное разрешение 1,6 мегапикселя независимо от размера входа.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `модель` | COMBO | Да | "wan2.5-i2i-preview" | Используемая модель (по умолчанию: "wan2.5-i2i-preview"). |
| `изображение` | IMAGE | Да | - | Редактирование одного изображения или слияние нескольких изображений, максимум 2 изображения. |
| `промпт` | STRING | Да | - | Запрос, описывающий элементы и визуальные особенности. Поддерживает английский и китайский языки (по умолчанию: пусто). |
| `негативный_промпт` | STRING | Нет | - | Негативный запрос, описывающий, чего следует избегать (по умолчанию: пусто). |
| `сид` | INT | Нет | от 0 до 2147483647 | Сид, используемый для генерации (по умолчанию: 0). |
| `водяной_знак` | BOOLEAN | Нет | - | Добавлять ли AI-водяной знак к результату (по умолчанию: false). |

**Примечание:** Этот узел принимает ровно 1 или 2 входных изображения. Если вы передадите более 2 изображений или не передадите ни одного, узел вернет ошибку.

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `изображение` | IMAGE | Сгенерированное изображение на основе входных изображений и текстовых запросов. |
