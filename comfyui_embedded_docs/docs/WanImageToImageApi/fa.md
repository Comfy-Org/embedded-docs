گره **Wan Image to Image** یک تصویر را بر اساس یک یا دو تصویر ورودی و یک توضیح متنی تولید می‌کند. این گره تصاویر ورودی شما را با توجه به توضیح ارائه‌شده تغییر می‌دهد و تصویری جدید ایجاد می‌کند که نسبت ابعاد تصویر اصلی را حفظ می‌کند. تصویر خروجی بدون توجه به اندازه ورودی، ۱.۶ مگاپیکسل ثابت است.

## ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|---------|----------|-------|--------|---------|
| `مدل` | COMBO | بله | "wan2.5-i2i-preview" | مدل مورد استفاده (پیش‌فرض: "wan2.5-i2i-preview"). |
| `تصویر` | IMAGE | بله | - | ویرایش تک‌تصویر یا ترکیب چند تصویر، حداکثر ۲ تصویر. |
| `پرامپت` | STRING | بله | - | توضیحی درباره عناصر و ویژگی‌های بصری. از زبان انگلیسی و چینی پشتیبانی می‌کند (پیش‌فرض: خالی). |
| `پرامپت منفی` | STRING | خیر | - | توضیح منفی درباره مواردی که باید اجتناب شود (پیش‌فرض: خالی). |
| `بذر` | INT | خیر | ۰ تا ۲۱۴۷۴۸۳۶۴۷ | دانه (seed) مورد استفاده برای تولید (پیش‌فرض: ۰). |
| `واترمارک` | BOOLEAN | خیر | - | آیا به نتیجه، واترمارک تولیدشده با هوش مصنوعی اضافه شود (پیش‌فرض: false). |

**توجه:** این گره دقیقاً ۱ یا ۲ تصویر ورودی می‌پذیرد. اگر بیش از ۲ تصویر یا هیچ تصویری ارائه دهید، گره خطا برمی‌گرداند.
