تقوم عقدة "تحويل الصورة إلى صورة Wan" بإنشاء صورة من صورة إدخال واحدة أو اثنتين مع نص وصفي. تقوم بتحويل صور الإدخال بناءً على الوصف الذي تقدمه، مما يُنتج صورة جديدة تحافظ على نسبة العرض إلى الارتفاع للصورة الأصلية. تكون الصورة الناتجة ثابتة الدقة عند 1.6 ميغابكسل بغض النظر عن حجم الإدخال.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|---------|---------------|--------|---------|-------|
| `model` | COMBO | نعم | "wan2.5-i2i-preview" | النموذج المستخدم (الافتراضي: "wan2.5-i2i-preview"). |
| `image` | IMAGE | نعم | - | تحرير صورة واحدة أو دمج صور متعددة، بحد أقصى صورتين. |
| `prompt` | STRING | نعم | - | النص الوصفي الذي يصف العناصر والميزات البصرية. يدعم اللغتين الإنجليزية والصينية (الافتراضي: فارغ). |
| `negative_prompt` | STRING | لا | - | النص الوصفي السلبي الذي يصف ما يجب تجنبه (الافتراضي: فارغ). |
| `seed` | INT | لا | 0 إلى 2147483647 | البذرة المستخدمة للتوليد (الافتراضي: 0). |
| `watermark` | BOOLEAN | لا | - | ما إذا كان سيتم إضافة علامة مائية من إنشاء الذكاء الاصطناعي إلى النتيجة (الافتراضي: false). |

**ملاحظة:** تقبل هذه العقدة صورتين إدخال بالضبط (1 أو 2). إذا قمت بتوفير أكثر من صورتين أو لم تقدم أي صور على الإطلاق، فستُرجع العقدة خطأً.
