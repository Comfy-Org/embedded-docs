Вот перевод документации на русский язык:

Узел ByteDance Seedance 2.0 Reference to Video использует модель ИИ Seedance 2.0 для создания, редактирования или расширения видео на основе вашего текстового запроса и предоставленных референсных материалов. Он может использовать изображения, видео и аудио в качестве референсов для управления процессом генерации, поддерживая такие задачи, как редактирование и расширение видео.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `модель` | COMBO | Да | `"Seedance 2.0"`<br>`"Seedance 2.0 Fast"` | Используемая модель ИИ. Seedance 2.0 предназначена для максимального качества, в то время как Seedance 2.0 Fast оптимизирована для скорости. Выбор модели открывает дополнительные обязательные входные параметры: `prompt`, `resolution`, `duration`, `ratio`, `generate_audio`, а также необязательные: `reference_images`, `reference_videos`, `reference_audios`, `reference_assets` и `auto_downscale`. |
| `seed` | INT | Нет | от 0 до 2147483647 | Число, используемое для управления повторным запуском узла. Результаты не являются детерминированными независимо от значения seed (по умолчанию: 0). |
| `водяной знак` | BOOLEAN | Нет | `True` / `False` | Добавлять ли водяной знак к сгенерированному видео (по умолчанию: False). |

**Важные ограничения:**
*   Для работы узла требуется хотя бы одно референсное изображение или видео (предоставленное через входные параметры `reference_images`, `reference_videos` или `reference_assets`).
*   Максимально можно использовать 9 референсных изображений (включая те, что переданы через `reference_images` и `reference_assets`).
*   Максимально можно использовать 3 референсных видео (включая те, что переданы через `reference_videos` и `reference_assets`).
*   Максимально можно использовать 3 референсных аудиоклипа (включая те, что переданы через `reference_audios` и `reference_assets`).
*   Каждое референсное видео должно быть длиной не менее 1,8 секунды. Суммарная длительность всех референсных видео не может превышать 15,1 секунды.
*   Каждый референсный аудиоклип должен быть длиной не менее 1,8 секунды. Суммарная длительность всех референсных аудиоклипов не может превышать 15,1 секунды.

## Выходные параметры

| Имя выходного параметра | Тип данных | Описание |
|-------------|-----------|-------------|
| `video` | VIDEO | Сгенерированный видеофайл. |
