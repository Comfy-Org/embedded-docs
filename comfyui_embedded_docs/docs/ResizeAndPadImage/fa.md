گره ResizeAndPadImage تصویر را به‌گونه‌ای تغییر اندازه می‌دهد که درون ابعاد مشخص‌شده قرار گیرد، در حالی که نسبت ابعاد اصلی آن حفظ می‌شود. این گره تصویر را به‌طور متناسب کوچک می‌کند تا درون عرض و ارتفاع هدف جای گیرد، سپس در اطراف لبه‌ها حاشیه (padding) اضافه می‌کند تا فضای باقی‌مانده پر شود. رنگ حاشیه و روش درون‌یابی (interpolation) قابل شخصی‌سازی است تا ظاهر نواحی حاشیه‌دار و کیفیت تغییر اندازه کنترل شود.

|---------|----------|--------|--------|---------|

| `عرض هدف` | INT | بله | 1 تا MAX_RESOLUTION | عرض مورد نظر تصویر خروجی (پیش‌فرض: 512) |

| `درون‌یابی` | COMBO | بله | "area"<br>"bicubic"<br>"nearest-exact"<br>"bilinear"<br>"lanczos" | روش درون‌یابی مورد استفاده برای تغییر اندازه تصویر (پیش‌فرض: "area") |
