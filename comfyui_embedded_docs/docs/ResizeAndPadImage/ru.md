Узел ResizeAndPadImage изменяет размер изображения, чтобы оно поместилось в заданные размеры с сохранением исходного соотношения сторон. Изображение пропорционально уменьшается, чтобы вписаться в целевую ширину и высоту, после чего по краям добавляются поля для заполнения оставшегося пространства. Цвет полей и метод интерполяции можно настроить для управления внешним видом заполненных областей и качеством изменения размера.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `image` | IMAGE | Да | - | Входное изображение для изменения размера и добавления полей |
| `target_width` | INT | Да | от 1 до MAX_RESOLUTION | Желаемая ширина выходного изображения (по умолчанию: 512) |
| `target_height` | INT | Да | от 1 до MAX_RESOLUTION | Желаемая высота выходного изображения (по умолчанию: 512) |
| `padding_color` | COMBO | Да | "white"<br>"black" | Цвет, используемый для полей вокруг изменённого изображения (по умолчанию: "white") |
| `interpolation` | COMBO | Да | "area"<br>"bicubic"<br>"nearest-exact"<br>"bilinear"<br>"lanczos" | Метод интерполяции, используемый при изменении размера изображения (по умолчанию: "area") |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `image` | IMAGE | Выходное изображение после изменения размера и добавления полей |
