Вот перевод документации на русский язык:

Узел VideoTriangleCFGGuidance применяет треугольный паттерн масштабирования бесклассового направляющего воздействия (CFG) к видеомоделям. Он изменяет масштаб кондиционирования во времени с помощью треугольной волновой функции, которая колеблется между минимальным значением CFG и исходным масштабом кондиционирования. Это создает динамический паттерн направляющего воздействия, который может помочь улучшить согласованность и качество генерации видео.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `модель` | MODEL | Да | - | Видеомодель, к которой применяется треугольное CFG-направляющее воздействие |
| `мин_cfg` | FLOAT | Да | 0.0 - 100.0 | Минимальное значение масштаба CFG для треугольного паттерна (по умолчанию: 1.0) |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `модель` | MODEL | Модифицированная модель с примененным треугольным CFG-направляющим воздействием |
