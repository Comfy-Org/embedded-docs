Вот перевод документации на русский язык:

Узел Magnific Image Upscale (Precise V2) выполняет высокоточное масштабирование изображений с точным контролем резкости, зернистости и детализации. Он обрабатывает изображения через внешний API, поддерживая максимальное выходное разрешение до 10060×10060 пикселей. Узел предлагает различные стили обработки и может автоматически уменьшать масштаб входного изображения, если запрошенный выходной размер превышает максимально допустимый.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `image` | IMAGE | Да | - | Входное изображение для масштабирования. Требуется ровно одно изображение. Минимальные размеры: 160x160 пикселей. Соотношение сторон должно быть от 1:3 до 3:1. |
| `scale_factor` | STRING | Да | `"2x"`<br>`"4x"`<br>`"8x"`<br>`"16x"` | Желаемый коэффициент увеличения. |
| `flavor` | STRING | Да | `"sublime"`<br>`"photo"`<br>`"photo_denoiser"` | Стиль обработки. "sublime" — для общего использования, "photo" — оптимизирован для фотографий, "photo_denoiser" — для зашумленных фото. |
| `sharpen` | INT | Нет | от 0 до 100 | Управляет интенсивностью повышения резкости для улучшения четкости краев и контуров. Более высокие значения дают более резкий результат. По умолчанию: 7. |
| `smart_grain` | INT | Нет | от 0 до 100 | Добавляет интеллектуальное зерно или улучшение текстуры, чтобы предотвратить излишнюю гладкость или искусственность масштабированного изображения. По умолчанию: 7. |
| `ultra_detail` | INT | Нет | от 0 до 100 | Управляет количеством мелких деталей, текстур и микродеталей, добавляемых в процессе масштабирования. По умолчанию: 30. |
| `auto_downscale` | BOOLEAN | Нет | - | При включении узел автоматически уменьшает масштаб входного изображения, если расчетные выходные размеры превышают максимально допустимое разрешение 10060x10060 пикселей. Это помогает избежать ошибок, но может повлиять на качество. По умолчанию: False. |

**Примечание:** Если `auto_downscale` отключен и запрошенный выходной размер (размеры входного изображения × `scale_factor`) превышает 10060x10060 пикселей, узел выдаст ошибку.

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `image` | IMAGE | Результирующее масштабированное изображение. |
