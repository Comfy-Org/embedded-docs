Вот перевод документации узла CLIPSubtract на русский язык:

Узел CLIPSubtract выполняет операцию вычитания между двумя моделями CLIP. Он использует первую модель CLIP в качестве базовой и вычитает ключевые патчи из второй модели CLIP, с возможностью использования множителя для управления силой вычитания. Это позволяет выполнять тонкую настройку смешивания моделей путем удаления определенных характеристик из одной модели с помощью другой.

## Входные параметры

| Параметр | Тип данных | Тип входа | По умолчанию | Диапазон | Описание |
|-----------|-----------|------------|---------|-------|-------------|
| `clip1` | CLIP | Обязательный | - | - | Базовая модель CLIP, которая будет изменена |
| `clip2` | CLIP | Обязательный | - | - | Модель CLIP, ключевые патчи которой будут вычтены из базовой модели |
| `multiplier` | FLOAT | Обязательный | 1.0 | от -10.0 до 10.0, шаг 0.01 | Управляет силой операции вычитания. Положительные значения вычитают признаки из clip2, отрицательные значения, наоборот, добавляют признаки. |

## Выходные данные

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `CLIP` | CLIP | Результирующая модель CLIP после операции вычитания |
