## 概要
CLIPSubtractノードは、2つのCLIPモデル間で減算操作を実行します。最初のCLIPモデルをベースとして、2番目のCLIPモデルからキーパッチを減算します。オプションの乗数を使用して減算の強度を制御できます。これにより、一方のモデルを使用して特定の特性を除去することで、微調整されたモデルのブレンドが可能になります。

| `clip2` | CLIP | 必須 | - | - | ベースモデルからキーパッチが減算されるCLIPモデル |
| `multiplier` | FLOAT | 必須 | 1.0 | -10.0 ～ 10.0、ステップ 0.01 | 減算操作の強度を制御します。正の値はclip2の特徴を減算し、負の値は代わりに特徴を加算します。 |
