گره CLIPSubtract عملیات تفریق بین دو مدل CLIP را انجام می‌دهد. این گره از اولین مدل CLIP به عنوان پایه استفاده کرده و وصله‌های کلیدی (key patches) مدل CLIP دوم را از آن کم می‌کند. یک ضریب اختیاری نیز برای کنترل شدت تفریق وجود دارد. این امکان ترکیب دقیق مدل‌ها را با حذف ویژگی‌های خاص از یک مدل با استفاده از مدل دیگر فراهم می‌کند.

## ورودی‌ها

| پارامتر | نوع داده | نوع ورودی | پیش‌فرض | محدوده | توضیحات |
|---------|----------|-----------|---------|--------|---------|
| `clip1` | CLIP | اجباری | - | - | مدل CLIP پایه که تغییر داده خواهد شد |
| `clip2` | CLIP | اجباری | - | - | مدل CLIP که وصله‌های کلیدی آن از مدل پایه کم می‌شود |
| `multiplier` | FLOAT | اجباری | 1.0 | 10.0- تا 10.0، گام 0.01 | شدت عملیات تفریق را کنترل می‌کند. مقادیر مثبت ویژگی‌های clip2 را کم می‌کنند، مقادیر منفی به جای آن ویژگی‌ها را اضافه می‌کنند. |
