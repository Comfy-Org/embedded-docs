## Обзор

Узел Remove Background использует модель удаления фона для создания маски, отделяющей объект переднего плана от фона входного изображения. Он принимает изображение и модель удаления фона, после чего создает маску, выделяющую основной объект.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `image` | IMAGE | Да | Н/Д | Входное изображение, из которого требуется удалить фон |
| `bg_removal_model` | BACKGROUND_REMOVAL_MODEL | Да | Н/Д | Модель удаления фона, используемая для создания маски |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `mask` | MASK | Созданная маска переднего плана, выделяющая основной объект на входном изображении |
