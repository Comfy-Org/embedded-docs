Этот узел предназначен для пакетной обработки данных кондиционирования, специально адаптированной для модели StableZero123. Он ориентирован на эффективную параллельную обработку нескольких наборов данных кондиционирования, оптимизируя рабочий процесс в сценариях, где критически важна пакетная обработка.

## Входные параметры

| Параметр | Тип данных | Описание |
|---|---|---|
| `clip_vision` | `CLIP_VISION` | Визуальные эмбеддинги CLIP, обеспечивающие визуальный контекст для процесса кондиционирования. |
| `init_image` | `IMAGE` | Исходное изображение, служащее отправной точкой для процесса генерации. |
| `vae` | `VAE` | Вариационный автоэнкодер, используемый для кодирования и декодирования изображений в процессе кондиционирования. |
| `width` | `INT` | Ширина выходного изображения. |
| `height` | `INT` | Высота выходного изображения. |
| `batch_size` | `INT` | Количество наборов кондиционирования, обрабатываемых за один пакет. |
| `elevation` | `FLOAT` | Угол возвышения для кондиционирования 3D-модели, влияющий на перспективу генерируемого изображения. |
| `azimuth` | `FLOAT` | Азимутальный угол для кондиционирования 3D-модели, влияющий на ориентацию генерируемого изображения. |
| `elevation_batch_increment` | `FLOAT` | Инкрементальное изменение угла возвышения в пределах пакета, позволяющее получать различные перспективы. |
| `azimuth_batch_increment` | `FLOAT` | Инкрементальное изменение азимутального угла в пределах пакета, позволяющее получать различные ориентации. |

## Выходные параметры

| Параметр | Тип данных | Описание |
|---|---|---|
| `positive` | `CONDITIONING` | Положительный выход кондиционирования, предназначенный для усиления определенных признаков или аспектов в генерируемом контенте. |
| `negative` | `CONDITIONING` | Отрицательный выход кондиционирования, предназначенный для ослабления определенных признаков или аспектов в генерируемом контенте. |
| `latent` | `LATENT` | Скрытое представление, полученное в результате процесса кондиционирования, готовое к дальнейшей обработке или этапам генерации. |
