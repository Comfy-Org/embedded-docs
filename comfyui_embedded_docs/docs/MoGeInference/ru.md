## Обзор

Запуск MoGe на одном изображении для оценки глубины и геометрии. Этот узел обрабатывает входное изображение через модель MoGe для создания трехмерного облака точек, карты глубины, внутренних параметров камеры, маски и нормалей поверхности.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `moge_model` | MOGE_MODEL | Да | Н/Д | Модель MoGe для выполнения инференса. |
| `image` | IMAGE | Да | Н/Д | Входное изображение для оценки глубины и геометрии. |
| `resolution_level` | INT | Да | от 0 до 9 | Управляет разрешением обработки. 0 — самый быстрый, 9 обеспечивает максимальную детализацию. (по умолчанию: 9) |
| `fov_x_degrees` | FLOAT | Да | от 0.0 до 170.0 | Горизонтальное поле зрения исходной камеры в градусах. Задает фокусное расстояние, используемое для проецирования карты глубины в 3D. Установите значение 0.0 для автоматического восстановления поля зрения из предсказанных точек. (по умолчанию: 0.0) |
| `batch_size` | INT | Да | от 1 до 64 | Количество изображений, обрабатываемых за один вызов инференса. Уменьшите это значение при нехватке памяти при обработке длинных видео или больших наборов изображений. (по умолчанию: 4) |
| `force_projection` | BOOLEAN | Да | True/False | (Расширенный) Принудительное проецирование предсказанных точек. (по умолчанию: True) |
| `apply_mask` | BOOLEAN | Да | True/False | При включении устанавливает для маскированных (небо или недопустимые) пикселей значение бесконечности в выходах точек и глубины. Это помогает инструментам создания сетки игнорировать эти области. Отключите, чтобы сохранить исходную предсказанную геометрию везде; маска по-прежнему возвращается отдельно. (по умолчанию: True) |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `moge_geometry` | MOGE_GEOMETRY | Словарь, содержащий оцененную геометрию. Включает исходное `image` и может содержать `points` (трехмерное облако точек), `depth` (карта глубины), `intrinsics` (матрица внутренних параметров камеры), `mask` (маска, определяющая допустимые пиксели) и `normal` (нормали поверхности). |
