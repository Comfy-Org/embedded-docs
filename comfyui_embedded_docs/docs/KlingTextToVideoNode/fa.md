### گره متن به ویدیوی Kling

گره متن به ویدیوی Kling، توضیحات متنی را به محتوای ویدیویی تبدیل می‌کند. این گره با دریافت提示‌های متنی، دنباله‌های ویدیویی متناظر را بر اساس تنظیمات پیکربندی مشخص‌شده تولید می‌کند. این گره از نسبت‌های ابعادی و حالت‌های تولید مختلف برای ساخت ویدیوهایی با مدت‌زمان و کیفیت متفاوت پشتیبانی می‌کند.

## ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `prompt` | STRING | بله | - |提示 متنی مثبت |
| `negative_prompt` | STRING | بله | - |提示 متنی منفی |
| `cfg_scale` | FLOAT | خیر | 0.0 تا 1.0 | مقدار مقیاس پیکربندی (پیش‌فرض: 1.0) |
| `aspect_ratio` | COMBO | خیر | گزینه‌های KlingVideoGenAspectRatio | تنظیم نسبت ابعاد ویدیو (پیش‌فرض: "16:9") |
| `mode` | COMBO | خیر | چندین گزینه موجود | پیکربندی مورد استفاده برای تولید ویدیو با فرمت: حالت / مدت‌زمان / نام_مدل. (پیش‌فرض: modes[8]) |

## خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-------------|-----------|-------------|
| `output` | VIDEO | خروجی ویدیوی تولیدشده |
| `video_id` | STRING | شناسه یکتای ویدیوی تولیدشده |
| `duration` | STRING | اطلاعات مدت‌زمان ویدیوی تولیدشده |
