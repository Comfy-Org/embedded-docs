Вот перевод документации на русский язык:

Узел "Текст в видео Kling" преобразует текстовые описания в видеоконтент. Он принимает текстовые подсказки и генерирует соответствующие видеопоследовательности на основе указанных параметров конфигурации. Узел поддерживает различные соотношения сторон и режимы генерации для создания видео разной продолжительности и качества.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `prompt` | STRING | Да | - | Положительная текстовая подсказка |
| `negative_prompt` | STRING | Да | - | Отрицательная текстовая подсказка |
| `cfg_scale` | FLOAT | Нет | 0.0 до 1.0 | Значение масштаба конфигурации (по умолчанию: 1.0) |
| `aspect_ratio` | COMBO | Нет | Опции из KlingVideoGenAspectRatio | Настройка соотношения сторон видео (по умолчанию: "16:9") |
| `mode` | COMBO | Нет | Доступно несколько опций | Конфигурация для генерации видео в формате: режим / длительность / имя_модели. (по умолчанию: modes[8]) |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `output` | VIDEO | Сгенерированный видеофайл |
| `video_id` | STRING | Уникальный идентификатор сгенерированного видео |
| `duration` | STRING | Информация о длительности сгенерированного видео |
