# ترجمة وثيقة عقدة Kling Text to Video

عقدة تحويل النص إلى فيديو من Kling تقوم بتحويل الأوصاف النصية إلى محتوى فيديو. تأخذ العقدة مطالبات نصية وتُنتج تسلسلات فيديو مقابلة بناءً على إعدادات التهيئة المحددة. تدعم العقدة نسب أبعاد مختلفة وأوضاع توليد لإنتاج فيديوهات بأطوال وجودة متفاوتة.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|---------|---------------|--------|---------|-------|
| `prompt` | STRING | نعم | - | المطالبة النصية الإيجابية |
| `negative_prompt` | STRING | نعم | - | المطالبة النصية السلبية |
| `cfg_scale` | FLOAT | لا | 0.0 إلى 1.0 | قيمة مقياس التهيئة (القيمة الافتراضية: 1.0) |
| `aspect_ratio` | COMBO | لا | خيارات من KlingVideoGenAspectRatio | إعداد نسبة أبعاد الفيديو (القيمة الافتراضية: "16:9") |
| `mode` | COMBO | لا | خيارات متعددة متاحة | التهيئة المستخدمة لتوليد الفيديو وفقًا للتنسيق: الوضع / المدة / اسم_النموذج. (القيمة الافتراضية: modes[8]) |

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|-------------|---------------|---------|
| `output` | VIDEO | مخرج الفيديو المُولَّد |
| `video_id` | STRING | معرف فريد للفيديو المُولَّد |
| `duration` | STRING | معلومات مدة الفيديو المُولَّد |
