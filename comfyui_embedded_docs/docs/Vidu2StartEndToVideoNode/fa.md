### نمای کلی

این گره با درون‌یابی بین یک فریم شروع و یک فریم پایان، تحت راهنمایی یک متن راهنما (prompt)، ویدیو تولید می‌کند. از یک مدل مشخص Vidu برای ایجاد انتقال روان بین دو تصویر در مدت زمان تعیین‌شده استفاده می‌کند.

### ورودی‌ها

| پارامتر | نوع داده | اجباری | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `model` | COMBO | بله | `"viduq2-pro-fast"`<br>`"viduq2-pro"`<br>`"viduq2-turbo"` | مدل Vidu مورد استفاده برای تولید ویدیو. |
| `first_frame` | IMAGE | بله | - | تصویر شروع برای دنباله ویدیو. فقط یک تصویر مجاز است. |
| `end_frame` | IMAGE | بله | - | تصویر پایان برای دنباله ویدیو. فقط یک تصویر مجاز است. |
| `prompt` | STRING | بله | - | توضیح متنی که تولید ویدیو را هدایت می‌کند (حداکثر ۲۰۰۰ کاراکتر). |
| `duration` | INT | خیر | ۲ تا ۸ | طول ویدیوی تولید شده بر حسب ثانیه (پیش‌فرض: ۵). |
| `seed` | INT | خیر | ۰ تا ۲۱۴۷۴۸۳۶۴۷ | عددی برای مقداردهی اولیه تولید تصادفی جهت نتایج قابل تکرار (پیش‌فرض: ۱). |
| `resolution` | COMBO | خیر | `"720p"`<br>`"1080p"` | وضوح خروجی ویدیوی تولید شده. |
| `movement_amplitude` | COMBO | خیر | `"auto"`<br>`"small"`<br>`"medium"`<br>`"large"` | دامنه حرکت اشیاء در قاب. |

**نکته:** تصاویر `first_frame` و `end_frame` باید نسبت ابعاد مشابهی داشته باشند. گره بررسی می‌کند که نسبت ابعاد آنها در محدوده نسبی ۰.۸ تا ۱.۲۵ باشد.

|-------------|-----------|-------------|
| `output` | VIDEO | فایل ویدیوی تولید شده. |
