Узел ModelSamplingStableCascade применяет устойчивое каскадное сэмплирование к модели, корректируя параметры сэмплирования с помощью значения сдвига. Он создает модифицированную версию входной модели с пользовательской конфигурацией сэмплирования для генерации по методу устойчивого каскада.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `model` | MODEL | Да | - | Входная модель, к которой применяется устойчивое каскадное сэмплирование |
| `shift` | FLOAT | Да | 0.0 - 100.0 | Значение сдвига, применяемое к параметрам сэмплирования (по умолчанию: 2.0) |

## Выходные данные

| Выходной параметр | Тип данных | Описание |
|-------------|-----------|-------------|
| `model` | MODEL | Модифицированная модель с примененным устойчивым каскадным сэмплированием |
