Вот перевод документации на русский язык:

Узел ModelSamplingStableCascade применяет стабильный каскадный сэмплинг к модели путем настройки параметров сэмплирования со значением сдвига. Он создает модифицированную версию входной модели с пользовательской конфигурацией сэмплирования для стабильной каскадной генерации.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `model` | MODEL | Да | - | Входная модель, к которой применяется стабильный каскадный сэмплинг |
| `shift` | FLOAT | Да | 0.0 - 100.0 | Значение сдвига, применяемое к параметрам сэмплирования (по умолчанию: 2.0) |

## Выходы

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `model` | MODEL | Модифицированная модель с примененным стабильным каскадным сэмплингом |
