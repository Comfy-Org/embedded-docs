### نمای کلی

با استفاده از مدل هوش مصنوعی Sonilo از ویدیو موسیقی تولید کنید. این گره محتوای یک ویدیوی ورودی را تحلیل کرده و یک قطعه موسیقی متناسب با آن ایجاد می‌کند. برای پردازش ویدیو و تولید صدا از یک سرویس هوش مصنوعی خارجی استفاده می‌کند.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `video` | VIDEO | بله | - | ویدیوی ورودی برای تولید موسیقی از آن. حداکثر مدت زمان: ۶ دقیقه. |
| `prompt` | STRING | خیر | - | توضیح متنی اختیاری برای هدایت تولید موسیقی. برای بهترین کیفیت خالی بگذارید - مدل محتوای ویدیو را به طور کامل تحلیل خواهد کرد. (پیش‌فرض: رشته خالی) |
| `seed` | INT | خیر | ۰ تا ۱۸۴۴۶۷۴۴۰۷۳۷۰۹۵۵۱۶۱۵ | دانه (seed) برای تکرارپذیری. در حال حاضر توسط سرویس Sonilo نادیده گرفته می‌شود اما برای سازگاری گراف حفظ شده است. (پیش‌فرض: ۰) |

## خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-------------|-----------|-------------|
| `audio` | AUDIO | موسیقی تولید شده به صورت یک فایل صوتی. |
