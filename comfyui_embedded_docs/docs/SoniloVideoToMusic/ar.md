# ترجمة وثيقة عقدة SoniloVideoToMusic

## نظرة عامة

توليد موسيقى من الفيديو باستخدام نموذج الذكاء الاصطناعي من Sonilo. تقوم هذه العقدة بتحليل محتوى فيديو الإدخال وإنشاء مقطوعة موسيقية متوافقة معه. تستخدم خدمة ذكاء اصطناعي خارجية لمعالجة الفيديو وتوليد الصوت.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|---------|---------------|--------|---------|-------------|
| `الفيديو` | VIDEO | نعم | - | فيديو الإدخال لتوليد الموسيقى منه. الحد الأقصى للمدة: 6 دقائق. |
| `الوصف` | STRING | لا | - | نص توجيهي اختياري لتوجيه عملية توليد الموسيقى. يُترك فارغًا للحصول على أفضل جودة - سيقوم النموذج بتحليل محتوى الفيديو بالكامل. (الافتراضي: سلسلة فارغة) |
| `البذرة` | INT | لا | 0 إلى 18446744073709551615 | بذرة لإمكانية إعادة إنتاج النتائج. يتم تجاهلها حاليًا بواسطة خدمة Sonilo ولكنها محفوظة لاتساق الرسم البياني. (الافتراضي: 0) |
