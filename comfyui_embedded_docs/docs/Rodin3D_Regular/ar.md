عناوين الأقسام: ## نظرة عامة، ## المدخلات، ## المخرجات

تولّد عقدة Rodin 3D العادية أصولًا ثلاثية الأبعاد باستخدام واجهة برمجة تطبيقات Rodin. تستقبل صورًا كمدخلات وتعالجها عبر خدمة Rodin لإنشاء نماذج ثلاثية الأبعاد. تدير العقدة سير العمل بالك�ל من إنشاء المهمة إلى تنزيل ملفات النموذج ثلاثي الأبعاد النهائي.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|-----------|-----------|----------|-------|-------------|
| `Images` | IMAGE | نعم | - | الصور المدخلة المستخدمة في توليد النموذج ثلاثي الأبعاد. يمكن توفير صور متعددة. |
| `Seed` | INT | نعم | - | قيمة البذرة العشوائية للحصول على نتائج قابلة للتكرار. |
| `Material_Type` | STRING | نعم | - | نوع المادة المراد تطبيقها على النموذج ثلاثي الأبعاد. |
| `Polygon_count` | STRING | نعم | - | عدد المضلعات المستهدف للنموذج ثلاثي الأبعاد المُنشأ. يحدد هذا المعامل وضع الجودة وتعقيد الشبكة. |

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|-------------|-----------|-------------|
| `3D Model Path` | STRING | مسار الملف إلى النموذج ثلاثي الأبعاد المُنشأ (مُحتفظ به للتوافق مع الإصدارات السابقة). |
| `GLB` | FILE3DGLB | النموذج ثلاثي الأبعاد المُنشأ بتنسيق GLB. |
