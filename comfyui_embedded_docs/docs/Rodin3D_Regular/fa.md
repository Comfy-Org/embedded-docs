گره Rodin 3D Regular با استفاده از API رودین، دارایی‌های سه‌بعدی تولید می‌کند. این گره تصاویر ورودی را دریافت کرده و از طریق سرویس رودین پردازش می‌کند تا مدل‌های سه‌بعدی ایجاد کند. این گره کل گردش کار را از ایجاد وظیفه تا دانلود فایل‌های نهایی مدل سه‌بعدی مدیریت می‌کند.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|---------|----------|--------|--------|---------|
| `Images` | IMAGE | بله | - | تصاویر ورودی برای تولید مدل سه‌بعدی. می‌توان چندین تصویر ارائه داد. |
| `Seed` | INT | بله | - | مقدار دانه تصادفی برای نتایج قابل تکرار. |
| `Material_Type` | STRING | بله | - | نوع ماده‌ای که روی مدل سه‌بعدی اعمال می‌شود. |
| `Polygon_count` | STRING | بله | - | تعداد چندضلعی هدف برای مدل سه‌بعدی تولیدشده. این پارامتر حالت کیفیت و پیچیدگی مش را تعیین می‌کند. |

## خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-----------|----------|---------|
| `مسیر مدل سه‌بعدی` | STRING | مسیر فایل مدل سه‌بعدی تولیدشده (برای سازگاری با نسخه‌های قبلی حفظ شده است). |
| `GLB` | FILE3DGLB | مدل سه‌بعدی تولیدشده در قالب GLB. |
