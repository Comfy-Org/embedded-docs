Преобразует существующие аудиосэмплы в новые высококачественные композиции с помощью текстовых инструкций. Этот узел принимает входной аудиофайл и изменяет его на основе вашего текстового запроса для создания нового аудиоконтента.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `модель` | COMBO | Да | "stable-audio-2.5"<br> | Модель ИИ, используемая для преобразования аудио |
| `промпт` | STRING | Да |  | Текстовые инструкции, описывающие способ преобразования аудио (по умолчанию: пусто) |
| `аудио` | AUDIO | Да |  | Аудио должно быть длительностью от 6 до 190 секунд |
| `длительность` | INT | Нет | 1-190 | Управляет длительностью генерируемого аудио в секундах (по умолчанию: 190) |
| `сид` | INT | Нет | 0-4294967294 | Случайное зерно, используемое для генерации (по умолчанию: 0) |
| `шаги` | INT | Нет | 4-8 | Управляет количеством шагов дискретизации (по умолчанию: 8) |
| `интенсивность` | FLOAT | Нет | 0.01-1.0 | Параметр, управляющий степенью влияния входного аудио на генерируемый результат (по умолчанию: 1.0) |

**Примечание:** Входное аудио должно быть длительностью от 6 до 190 секунд.

## Выходы

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `аудио` | AUDIO | Преобразованное аудио, сгенерированное на основе входного аудио и текстового запроса |
