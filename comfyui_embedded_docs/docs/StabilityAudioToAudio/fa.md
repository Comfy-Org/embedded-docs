نمونه‌های صوتی موجود را با استفاده از دستورالعمل‌های متنی به ترکیب‌های جدید با کیفیت بالا تبدیل می‌کند. این گره یک فایل صوتی ورودی دریافت کرده و بر اساس متن راهنما (prompt) آن را تغییر می‌دهد تا محتوای صوتی جدیدی ایجاد کند.

## ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `model` | COMBO | بله | "stable-audio-2.5"<br> | مدل هوش مصنوعی مورد استفاده برای تبدیل صدا |
| `prompt` | STRING | بله |  | دستورالعمل‌های متنی توصیف‌کننده نحوه تبدیل صدا (پیش‌فرض: خالی) |
| `audio` | AUDIO | بله |  | صدا باید بین ۶ تا ۱۹۰ ثانیه طول داشته باشد |
| `duration` | INT | خیر | 1-190 | مدت زمان صدای تولید شده را بر حسب ثانیه کنترل می‌کند (پیش‌فرض: ۱۹۰) |
| `seed` | INT | خیر | 0-4294967294 | دانه تصادفی (seed) مورد استفاده برای تولید (پیش‌فرض: ۰) |
| `steps` | INT | خیر | 4-8 | تعداد مراحل نمونه‌برداری را کنترل می‌کند (پیش‌فرض: ۸) |
| `strength` | FLOAT | خیر | 0.01-1.0 | میزان تأثیر پارامتر صوتی بر صدای تولید شده را کنترل می‌کند (پیش‌فرض: ۱.۰) |

**توجه:** صدای ورودی باید بین ۶ تا ۱۹۰ ثانیه طول داشته باشد.

## خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-------------|-----------|-------------|
| `audio` | AUDIO | صدای تبدیل‌شده بر اساس صدای ورودی و متن راهنما (prompt) تولید شده است |
