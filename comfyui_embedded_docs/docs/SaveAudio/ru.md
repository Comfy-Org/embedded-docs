Узел SaveAudio сохраняет аудиоданные в файл формата FLAC. Он принимает аудиовход и записывает его в указанную выходную директорию с заданным префиксом имени файла. Узел автоматически управляет именованием файлов и обеспечивает корректное сохранение аудио для последующего использования.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `audio` | AUDIO | Да | - | Аудиоданные для сохранения |
| `filename_prefix` | STRING | Нет | - | Префикс для имени выходного файла (по умолчанию: "audio/ComfyUI") |

*Примечание: Параметры `prompt` и `extra_pnginfo` скрыты и автоматически обрабатываются системой.*

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| *Нет* | - | Этот узел не возвращает выходных данных, но сохраняет аудиофайл в выходную директорию |
