Узел SaveAudio сохраняет аудиоданные в файл формата FLAC. Он принимает аудиовходные данные и записывает их в указанную выходную директорию с заданным префиксом имени файла. Узел автоматически обрабатывает именование файлов и обеспечивает корректное сохранение аудио для последующего использования.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `аудио` | AUDIO | Да | - | Аудиоданные для сохранения |
| `префикс_названия_файла` | STRING | Нет | - | Префикс для имени выходного файла (по умолчанию: "audio/ComfyUI") |

*Примечание: Параметры `prompt` и `extra_pnginfo` являются скрытыми и автоматически обрабатываются системой.*

## Выходные данные

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| *Нет* | - | Этот узел не возвращает выходных данных, но сохраняет аудиофайл в выходную директорию |
