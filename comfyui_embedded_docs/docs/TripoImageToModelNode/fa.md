### نمای کلی

این گره با استفاده از API سرویس Tripo، مدل‌های سه‌بعدی را به‌صورت همزمان بر اساس یک تصویر واحد تولید می‌کند. این گره یک تصویر ورودی دریافت کرده و آن را با گزینه‌های سفارشی‌سازی متنوع برای بافت، کیفیت و ویژگی‌های مدل به یک مدل سه‌بعدی تبدیل می‌کند.

### ورودی‌ها

| پارامتر | نوع داده | اجباری | محدوده | توضیحات |
|---------|----------|--------|--------|---------|
| `image` | IMAGE | بله | - | تصویر ورودی برای تولید مدل سه‌بعدی |
| `model_version` | COMBO | خیر | گزینه‌های متعدد موجود | نسخه مدل Tripo مورد استفاده برای تولید |
| `style` | COMBO | خیر | گزینه‌های متعدد موجود | تنظیم سبک برای مدل تولیدشده (پیش‌فرض: "None") |
| `texture` | BOOLEAN | خیر | - | آیا بافت برای مدل تولید شود (پیش‌فرض: True) |
| `pbr` | BOOLEAN | خیر | - | آیا از رندرینگ مبتنی بر فیزیک استفاده شود (پیش‌فرض: True) |
| `model_seed` | INT | خیر | - | دانه تصادفی برای تولید مدل (پیش‌فرض: 42) |
| `orientation` | COMBO | خیر | گزینه‌های متعدد موجود | تنظیم جهت‌گیری برای مدل تولیدشده |
| `texture_seed` | INT | خیر | - | دانه تصادفی برای تولید بافت (پیش‌فرض: 42) |
| `texture_quality` | COMBO | خیر | "standard"<br>"detailed" | سطح کیفیت برای تولید بافت (پیش‌فرض: "standard") |
| `texture_alignment` | COMBO | خیر | "original_image"<br>"geometry" | روش تراز برای نگاشت بافت (پیش‌فرض: "original_image") |
| `face_limit` | INT | خیر | 1- تا 500000 | حداکثر تعداد وجه‌ها در مدل تولیدشده، 1- به معنی بدون محدودیت (پیش‌فرض: 1-) |
| `quad` | BOOLEAN | خیر | - | آیا از وجه‌های چهارضلعی به جای مثلث استفاده شود (پیش‌فرض: False) |
| `geometry_quality` | COMBO | خیر | "standard"<br>"detailed" | سطح کیفیت برای تولید هندسه (پیش‌فرض: "standard") |

**توجه:** پارامتر `image` اجباری است و باید برای عملکرد گره ارائه شود. اگر تصویری ارائه نشود، گره یک خطای RuntimeError ایجاد می‌کند.

### خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-----------|----------|---------|
| `model_file` | STRING | فایل مدل سه‌بعدی تولیدشده (فقط برای سازگاری با نسخه‌های قبلی) |
| `model task_id` | MODEL_TASK_ID | شناسه وظیفه برای پیگیری فرآیند تولید مدل |
| `GLB` | FILE3DGLB | مدل سه‌بعدی تولیدشده در قالب GLB |
