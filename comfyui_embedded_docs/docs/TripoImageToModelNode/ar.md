أنت خبير في الترجمة التقنية متخصص في توثيق عُقد ComfyUI من الإنجليزية إلى العربية.

## قواعد الترجمة

1. **المحتوى الذي يجب عدم ترجمته:**
   - أسماء المعاملات بين علامات الاقتباس الخلفية: `image`, `seed`, `model`
   - أنواع البيانات بالأحرف الكبيرة: IMAGE, STRING, INT, FLOAT, MODEL, CONDITIONING, إلخ
   - القيم في عمود Range: الأرقام، "auto"، أسماء الخيارات
   - الكود، مسارات الملفات

2. **المحتوى الذي يجب ترجمته:**
   - عناوين الأقسام: ## نظرة عامة, ## المدخلات, ## المخرجات
   - جميع النصوص الوصفية والتوضيحية
   - أوصاف المعاملات

3. **جودة الترجمة:**
   - استخدام اللغة العربية الفصحى المعاصرة
   - الحفاظ على نبرة احترافية ولكن سهلة الفهم
   - ضمان الدقة التقنية
   - استخدام المصطلحات التقنية العربية القياسية

4. **التنسيق:**
   - الحفاظ على جميع تنسيقات Markdown
   - الحفاظ على بنية الجداول
   - عدم إضافة أي ملاحظة أو رابط في بداية الوثيقة (سيتم إضافتها تلقائيًا)

يقوم هذا العقدة بتوليد نماذج ثلاثية الأبعاد بشكل متزامن بناءً على صورة واحدة باستخدام واجهة برمجة تطبيقات Tripo. تأخذ هذه العقدة صورة إدخال وتحولها إلى نموذج ثلاثي الأبعاد مع خيارات تخصيص متنوعة للنسيج والجودة وخصائص النموذج.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|-----------|-----------|----------|-------|-------------|
| `image` | IMAGE | نعم | - | الصورة المدخلة المستخدمة لتوليد النموذج ثلاثي الأبعاد |
| `model_version` | COMBO | لا | تتوفر خيارات متعددة | إصدار نموذج Tripo المراد استخدامه للتوليد |
| `style` | COMBO | لا | تتوفر خيارات متعددة | إعداد النمط للنموذج المُولَّد (الافتراضي: "None") |
| `texture` | BOOLEAN | لا | - | ما إذا كان سيتم توليد أنسجة للنموذج (الافتراضي: True) |
| `pbr` | BOOLEAN | لا | - | ما إذا كان سيتم استخدام العرض الفيزيائي القائم على الواقع (الافتراضي: True) |
| `model_seed` | INT | لا | - | البذرة العشوائية لتوليد النموذج (الافتراضي: 42) |
| `orientation` | COMBO | لا | تتوفر خيارات متعددة | إعداد الاتجاه للنموذج المُولَّد |
| `texture_seed` | INT | لا | - | البذرة العشوائية لتوليد النسيج (الافتراضي: 42) |
| `texture_quality` | COMBO | لا | "standard"<br>"detailed" | مستوى جودة توليد النسيج (الافتراضي: "standard") |
| `texture_alignment` | COMBO | لا | "original_image"<br>"geometry" | طريقة المحاذاة لتعيين النسيج (الافتراضي: "original_image") |
| `face_limit` | INT | لا | -1 إلى 500000 | الحد الأقصى لعدد الوجوه في النموذج المُولَّد، -1 يعني بدون حد (الافتراضي: -1) |
| `quad` | BOOLEAN | لا | - | ما إذا كان سيتم استخدام وجوه رباعية الأضلاع بدلاً من المثلثات (الافتراضي: False) |
| `geometry_quality` | COMBO | لا | "standard"<br>"detailed" | مستوى جودة توليد الهندسة (الافتراضي: "standard") |

**ملاحظة:** معامل `image` إلزامي ويجب توفيره لكي تعمل العقدة. إذا لم يتم توفير صورة، ستقوم العقدة بإثارة خطأ RuntimeError.

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|-------------|-----------|-------------|
| `model_file` | STRING | ملف النموذج ثلاثي الأبعاد المُولَّد (للتوافق مع الإصدارات السابقة فقط) |
| `model task_id` | MODEL_TASK_ID | معرف المهمة لتتبع عملية توليد النموذج |
| `GLB` | FILE3DGLB | النموذج ثلاثي الأبعاد المُولَّد بتنسيق GLB |
