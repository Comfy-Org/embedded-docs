Вот перевод документации на русский язык:

Генерирует 3D-модели синхронно на основе одного изображения с использованием API Tripo. Этот узел принимает входное изображение и преобразует его в 3D-модель с различными опциями настройки текстуры, качества и свойств модели.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `image` | IMAGE | Да | - | Входное изображение, используемое для генерации 3D-модели |
| `model_version` | COMBO | Нет | Доступно несколько опций | Версия модели Tripo для использования при генерации |
| `style` | COMBO | Нет | Доступно несколько опций | Стилевая настройка для сгенерированной модели (по умолчанию: "None") |
| `texture` | BOOLEAN | Нет | - | Создавать ли текстуры для модели (по умолчанию: True) |
| `pbr` | BOOLEAN | Нет | - | Использовать ли физически корректный рендеринг (по умолчанию: True) |
| `model_seed` | INT | Нет | - | Случайное зерно для генерации модели (по умолчанию: 42) |
| `orientation` | COMBO | Нет | Доступно несколько опций | Настройка ориентации для сгенерированной модели |
| `texture_seed` | INT | Нет | - | Случайное зерно для генерации текстуры (по умолчанию: 42) |
| `texture_quality` | COMBO | Нет | "standard"<br>"detailed" | Уровень качества генерации текстуры (по умолчанию: "standard") |
| `texture_alignment` | COMBO | Нет | "original_image"<br>"geometry" | Метод выравнивания для наложения текстуры (по умолчанию: "original_image") |
| `face_limit` | INT | Нет | -1 до 500000 | Максимальное количество полигонов в сгенерированной модели, -1 без ограничения (по умолчанию: -1) |
| `quad` | BOOLEAN | Нет | - | Использовать ли четырёхугольные полигоны вместо треугольных (по умолчанию: False) |
| `geometry_quality` | COMBO | Нет | "standard"<br>"detailed" | Уровень качества генерации геометрии (по умолчанию: "standard") |

**Примечание:** Параметр `image` является обязательным и должен быть предоставлен для работы узла. Если изображение не предоставлено, узел вызовет RuntimeError.

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `model_file` | STRING | Файл сгенерированной 3D-модели (только для обратной совместимости) |
| `model task_id` | MODEL_TASK_ID | Идентификатор задачи для отслеживания процесса генерации модели |
| `GLB` | FILE3DGLB | Сгенерированная 3D-модель в формате GLB |
