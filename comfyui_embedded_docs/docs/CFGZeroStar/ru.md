Вот перевод документации на русский язык:

Узел CFGZeroStar применяет специализированную технику масштабирования направляющего сигнала к диффузионным моделям. Он модифицирует процесс бесклассового направляющего сигнала путем вычисления оптимизированного коэффициента масштабирования на основе разницы между условными и безусловными предсказаниями. Данный подход корректирует итоговый вывод, обеспечивая расширенный контроль над процессом генерации при сохранении стабильности модели.

## Входные параметры

| Параметр | Тип данных | Тип входа | По умолчанию | Диапазон | Описание |
|----------|------------|-----------|--------------|----------|----------|
| `model` | MODEL | обязательный | - | - | Диффузионная модель, к которой применяется техника масштабирования направляющего сигнала CFGZeroStar |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `patched_model` | MODEL | Модифицированная модель с примененным масштабированием направляющего сигнала CFGZeroStar |
