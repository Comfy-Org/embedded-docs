### نظرة عامة

تم تصميم هذه العقدة لتشفير إدخال النص باستخدام نموذج CLIP مُخصص خصيصًا لبنية SDXL. تستخدم نظام تشفير مزدوج (CLIP-L و CLIP-G) لمعالجة الأوصاف النصية، مما يؤدي إلى توليد صور أكثر دقة.

### المدخلات

| المعامل | نوع البيانات | الوصف |
|-----------|-----------|-------------|
| `clip` | CLIP | مثيل نموذج CLIP المستخدم لتشفير النص. |
| `width` | INT | يحدد عرض الصورة بالبكسل، القيمة الافتراضية 1024. |
| `height` | INT | يحدد ارتفاع الصورة بالبكسل، القيمة الافتراضية 1024. |
| `crop_w` | INT | عرض منطقة الاقتصاص بالبكسل، القيمة الافتراضية 0. |
| `crop_h` | INT | ارتفاع منطقة الاقتصاص بالبكسل، القيمة الافتراضية 0. |
| `target_width` | INT | العرض المستهدف للصورة الناتجة، القيمة الافتراضية 1024. |
| `target_height` | INT | الارتفاع المستهدف للصورة الناتجة، القيمة الافتراضية 1024. |
| `text_g` | STRING | الوصف النصي العام لوصف المشهد الكلي. |
| `text_l` | STRING | الوصف النصي المحلي لوصف التفاصيل. |

### المخرجات

| المعامل | نوع البيانات | الوصف |
|-----------|-----------|-------------|
| `CONDITIONING` | CONDITIONING | يحتوي على النص المُشفر والمعلومات الشرطية اللازمة لتوليد الصورة. |
