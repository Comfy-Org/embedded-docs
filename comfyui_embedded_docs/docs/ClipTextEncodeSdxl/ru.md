Этот узел предназначен для кодирования текстового ввода с помощью модели CLIP, специально адаптированной для архитектуры SDXL. Он использует двойную систему кодировщиков (CLIP-L и CLIP-G) для обработки текстовых описаний, что обеспечивает более точную генерацию изображений.

## Входные параметры

| Параметр | Тип данных | Описание |
|-----------|-----------|-------------|
| `clip` | CLIP | Экземпляр модели CLIP, используемый для кодирования текста. |
| `width` | INT | Задает ширину изображения в пикселях, по умолчанию 1024. |
| `height` | INT | Задает высоту изображения в пикселях, по умолчанию 1024. |
| `crop_w` | INT | Ширина области кадрирования в пикселях, по умолчанию 0. |
| `crop_h` | INT | Высота области кадрирования в пикселях, по умолчанию 0. |
| `target_width` | INT | Целевая ширина выходного изображения, по умолчанию 1024. |
| `target_height` | INT | Целевая высота выходного изображения, по умолчанию 1024. |
| `text_g` | STRING | Глобальное текстовое описание для общего описания сцены. |
| `text_l` | STRING | Локальное текстовое описание для детального описания. |

## Выходные параметры

| Параметр | Тип данных | Описание |
|-----------|-----------|-------------|
| `CONDITIONING` | CONDITIONING | Содержит закодированный текст и условную информацию, необходимую для генерации изображения. |
