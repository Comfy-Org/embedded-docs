این گره برای رمزگذاری ورودی متنی با استفاده از یک مدل CLIP که به‌طور خاص برای معماری SDXL سفارشی‌سازی شده است، طراحی شده است. این گره از یک سیستم رمزگذار دوگانه (CLIP-L و CLIP-G) برای پردازش توصیف‌های متنی استفاده می‌کند که منجر به تولید تصاویر دقیق‌تری می‌شود.

## ورودی‌ها

| پارامتر | نوع داده | توضیحات |
|-----------|-----------|-------------|
| `clip` | CLIP | نمونه مدل CLIP مورد استفاده برای رمزگذاری متن. |
| `width` | INT | عرض تصویر را بر حسب پیکسل مشخص می‌کند، پیش‌فرض 1024. |
| `height` | INT | ارتفاع تصویر را بر حسب پیکسل مشخص می‌کند، پیش‌فرض 1024. |
| `crop_w` | INT | عرض ناحیه برش بر حسب پیکسل، پیش‌فرض 0. |
| `crop_h` | INT | ارتفاع ناحیه برش بر حسب پیکسل، پیش‌فرض 0. |
| `target_width` | INT | عرض هدف برای تصویر خروجی، پیش‌فرض 1024. |
| `target_height` | INT | ارتفاع هدف برای تصویر خروجی، پیش‌فرض 1024. |
| `text_g` | STRING | توصیف متنی سراسری برای توصیف کلی صحنه. |
| `text_l` | STRING | توصیف متنی محلی برای توصیف جزئیات. |

## خروجی‌ها

| پارامتر | نوع داده | توضیحات |
|-----------|-----------|-------------|
| `CONDITIONING` | CONDITIONING | شامل متن رمزگذاری‌شده و اطلاعات شرطی مورد نیاز برای تولید تصویر است. |
