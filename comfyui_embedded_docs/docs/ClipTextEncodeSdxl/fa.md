این گره برای رمزگذاری ورودی متنی با استفاده از یک مدل CLIP که به‌طور خاص برای معماری SDXL سفارشی‌سازی شده است، طراحی شده است. این گره از یک سیستم رمزگذار دوگانه (CLIP-L و CLIP-G) برای پردازش توصیف‌های متنی استفاده می‌کند که منجر به تولید تصاویر دقیق‌تری می‌شود.

## ورودی‌ها

| پارامتر | نوع داده | توضیحات |
|-----------|-----------|-------------|
| `clip` | CLIP | نمونه مدل CLIP مورد استفاده برای رمزگذاری متن. |
| `عرض` | INT | عرض تصویر را بر حسب پیکسل مشخص می‌کند، پیش‌فرض 1024. |
| `ارتفاع` | INT | ارتفاع تصویر را بر حسب پیکسل مشخص می‌کند، پیش‌فرض 1024. |
| `برش_عرض` | INT | عرض ناحیه برش بر حسب پیکسل، پیش‌فرض 0. |
| `برش_ارتفاع` | INT | ارتفاع ناحیه برش بر حسب پیکسل، پیش‌فرض 0. |
| `عرض هدف` | INT | عرض هدف برای تصویر خروجی، پیش‌فرض 1024. |
| `ارتفاع هدف` | INT | ارتفاع هدف برای تصویر خروجی، پیش‌فرض 1024. |
| `متن_g` | STRING | توصیف متنی سراسری برای توصیف کلی صحنه. |
| `متن_l` | STRING | توصیف متنی محلی برای توصیف جزئیات. |

## خروجی‌ها

| پارامتر | نوع داده | توضیحات |
|-----------|-----------|-------------|
| `CONDITIONING` | CONDITIONING | شامل متن رمزگذاری‌شده و اطلاعات شرطی مورد نیاز برای تولید تصویر است. |
