Вот перевод документации на русский язык:

Узел `GenerateTracks` создает несколько параллельных траекторий движения для генерации видео. Он определяет основной путь от начальной точки до конечной точки, затем генерирует набор дорожек, которые проходят параллельно этому пути, равномерно распределяясь по ширине. Вы можете управлять формой пути (прямая линия или кривая Безье), скоростью движения вдоль него и кадрами, в которых дорожки видны.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `width` | INT | Да | 16 - 4096 | Ширина видеокадра в пикселях. Значение по умолчанию: 832. |
| `height` | INT | Да | 16 - 4096 | Высота видеокадра в пикселях. Значение по умолчанию: 480. |
| `start_x` | FLOAT | Да | 0.0 - 1.0 | Нормализованная координата X (0-1) для начальной позиции. Значение по умолчанию: 0.0. |
| `start_y` | FLOAT | Да | 0.0 - 1.0 | Нормализованная координата Y (0-1) для начальной позиции. Значение по умолчанию: 0.0. |
| `end_x` | FLOAT | Да | 0.0 - 1.0 | Нормализованная координата X (0-1) для конечной позиции. Значение по умолчанию: 1.0. |
| `end_y` | FLOAT | Да | 0.0 - 1.0 | Нормализованная координата Y (0-1) для конечной позиции. Значение по умолчанию: 1.0. |
| `num_frames` | INT | Да | 1 - 1024 | Общее количество кадров, для которых генерируются позиции дорожек. Значение по умолчанию: 81. |
| `num_tracks` | INT | Да | 1 - 100 | Количество параллельных дорожек для генерации. Значение по умолчанию: 5. |
| `track_spread` | FLOAT | Да | 0.0 - 1.0 | Нормализованное расстояние между дорожками. Дорожки распределяются перпендикулярно направлению движения. Значение по умолчанию: 0.025. |
| `bezier` | BOOLEAN | Да | True / False | Включить путь по кривой Безье, используя среднюю точку в качестве контрольной точки. Значение по умолчанию: False. |
| `mid_x` | FLOAT | Да | 0.0 - 1.0 | Нормализованная контрольная точка X для кривой Безье. Используется только когда параметр 'bezier' включен. Значение по умолчанию: 0.5. |
| `mid_y` | FLOAT | Да | 0.0 - 1.0 | Нормализованная контрольная точка Y для кривой Безье. Используется только когда параметр 'bezier' включен. Значение по умолчанию: 0.5. |
| `interpolation` | COMBO | Да | `"linear"`<br>`"ease_in"`<br>`"ease_out"`<br>`"ease_in_out"`<br>`"constant"` | Управляет временем/скоростью движения вдоль пути. Значение по умолчанию: "linear". |
| `track_mask` | MASK | Нет | - | Необязательная маска для указания видимых кадров. |

**Примечание:** Параметры `mid_x` и `mid_y` используются только когда параметр `bezier` установлен в `True`. Когда `bezier` равен `False`, путь представляет собой прямую линию от начальной до конечной точки.

## Выходные данные

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `TRACKS` | TRACKS | Объект дорожек, содержащий сгенерированные координаты пути и информацию о видимости для всех дорожек во всех кадрах. |
| `track_length` | INT | Количество кадров, для которых были сгенерированы дорожки, соответствует входному параметру `num_frames`. |
