عُقدة `GenerateTracks` تنشئ مسارات حركة متوازية متعددة لتوليد الفيديو. تُعرّف مسارًا رئيسيًا من نقطة بداية إلى نقطة نهاية، ثم تُنشئ مجموعة من المسارات المتوازية لهذا المسار، موزعة على مسافات متساوية. يمكنك التحكم في شكل المسار (خط مستقيم أو منحنى بيزير)، وسرعة الحركة على طوله، والإطارات التي تظهر فيها المسارات.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|-----------|-----------|----------|-------|-------------|
| `العرض` | INT | نعم | 16 - 4096 | عرض إطار الفيديو بالبكسل. القيمة الافتراضية هي 832. |
| `الارتفاع` | INT | نعم | 16 - 4096 | ارتفاع إطار الفيديو بالبكسل. القيمة الافتراضية هي 480. |
| `إحداثي X للبداية` | FLOAT | نعم | 0.0 - 1.0 | الإحداثي X المُطبّع (0-1) لموضع البداية. القيمة الافتراضية هي 0.0. |
| `إحداثي Y للبداية` | FLOAT | نعم | 0.0 - 1.0 | الإحداثي Y المُطبّع (0-1) لموضع البداية. القيمة الافتراضية هي 0.0. |
| `إحداثي X للنهاية` | FLOAT | نعم | 0.0 - 1.0 | الإحداثي X المُطبّع (0-1) لموضع النهاية. القيمة الافتراضية هي 1.0. |
| `إحداثي Y للنهاية` | FLOAT | نعم | 0.0 - 1.0 | الإحداثي Y المُطبّع (0-1) لموضع النهاية. القيمة الافتراضية هي 1.0. |
| `عدد الإطارات` | INT | نعم | 1 - 1024 | العدد الإجمالي للإطارات التي سيتم توليد مواقع المسارات لها. القيمة الافتراضية هي 81. |
| `عدد المسارات` | INT | نعم | 1 - 100 | عدد المسارات المتوازية المراد توليدها. القيمة الافتراضية هي 5. |
| `توزيع المسارات` | FLOAT | نعم | 0.0 - 1.0 | المسافة المُطبّعة بين المسارات. تنتشر المسارات بشكل عمودي على اتجاه الحركة. القيمة الافتراضية هي 0.025. |
| `منحنى بيزيه` | BOOLEAN | نعم | True / False | تفعيل مسار منحنى بيزير باستخدام نقطة المنتصف كنقطة تحكم. القيمة الافتراضية هي False. |
| `إحداثي X للمنتصف` | FLOAT | نعم | 0.0 - 1.0 | نقطة التحكم X المُطبّعة لمنحنى بيزير. تُستخدم فقط عند تفعيل 'bezier'. القيمة الافتراضية هي 0.5. |
| `إحداثي Y للمنتصف` | FLOAT | نعم | 0.0 - 1.0 | نقطة التحكم Y المُطبّعة لمنحنى بيزير. تُستخدم فقط عند تفعيل 'bezier'. القيمة الافتراضية هي 0.5. |
| `الاستيفاء` | COMBO | نعم | `"linear"`<br>`"ease_in"`<br>`"ease_out"`<br>`"ease_in_out"`<br>`"constant"` | يتحكم في توقيت/سرعة الحركة على طول المسار. القيمة الافتراضية هي "linear". |
| `قناع المسار` | MASK | لا | - | قناع اختياري للإشارة إلى الإطارات المرئية. |

**ملاحظة:** يتم استخدام المعاملين `mid_x` و `mid_y` فقط عندما يتم تعيين المعامل `bezier` على `True`. عندما يكون `bezier` على `False`، يكون المسار خطًا مستقيمًا من نقطة البداية إلى نقطة النهاية.
