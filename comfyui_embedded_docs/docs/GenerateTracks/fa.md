گره `GenerateTracks` چندین مسیر حرکتی موازی برای تولید ویدیو ایجاد می‌کند. این گره یک مسیر اصلی از نقطه شروع تا نقطه پایان تعریف می‌کند، سپس مجموعه‌ای از مسیرهای موازی با این مسیر را با فاصله‌های مساوی از یکدیگر تولید می‌کند. می‌توانید شکل مسیر (خط مستقیم یا منحنی بزیه)، سرعت حرکت در طول آن و فریم‌هایی که مسیرها در آنها قابل مشاهده هستند را کنترل کنید.

## ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|---------|----------|-------|---------|----------|
| `width` | INT | بله | 16 - 4096 | عرض فریم ویدیو بر حسب پیکسل. مقدار پیش‌فرض 832 است. |
| `height` | INT | بله | 16 - 4096 | ارتفاع فریم ویدیو بر حسب پیکسل. مقدار پیش‌فرض 480 است. |
| `start_x` | FLOAT | بله | 0.0 - 1.0 | مختصات X نرمال‌شده (0-1) برای موقعیت شروع. مقدار پیش‌فرض 0.0 است. |
| `start_y` | FLOAT | بله | 0.0 - 1.0 | مختصات Y نرمال‌شده (0-1) برای موقعیت شروع. مقدار پیش‌فرض 0.0 است. |
| `end_x` | FLOAT | بله | 0.0 - 1.0 | مختصات X نرمال‌شده (0-1) برای موقعیت پایان. مقدار پیش‌فرض 1.0 است. |
| `end_y` | FLOAT | بله | 0.0 - 1.0 | مختصات Y نرمال‌شده (0-1) برای موقعیت پایان. مقدار پیش‌فرض 1.0 است. |
| `num_frames` | INT | بله | 1 - 1024 | تعداد کل فریم‌هایی که موقعیت‌های مسیر برای آنها تولید می‌شود. مقدار پیش‌فرض 81 است. |
| `num_tracks` | INT | بله | 1 - 100 | تعداد مسیرهای موازی برای تولید. مقدار پیش‌فرض 5 است. |
| `track_spread` | FLOAT | بله | 0.0 - 1.0 | فاصله نرمال‌شده بین مسیرها. مسیرها عمود بر جهت حرکت پخش می‌شوند. مقدار پیش‌فرض 0.025 است. |
| `bezier` | BOOLEAN | بله | True / False | فعال‌سازی مسیر منحنی بزیه با استفاده از نقطه میانی به عنوان نقطه کنترل. مقدار پیش‌فرض False است. |
| `mid_x` | FLOAT | بله | 0.0 - 1.0 | نقطه کنترل X نرمال‌شده برای منحنی بزیه. فقط زمانی استفاده می‌شود که 'bezier' فعال باشد. مقدار پیش‌فرض 0.5 است. |
| `mid_y` | FLOAT | بله | 0.0 - 1.0 | نقطه کنترل Y نرمال‌شده برای منحنی بزیه. فقط زمانی استفاده می‌شود که 'bezier' فعال باشد. مقدار پیش‌فرض 0.5 است. |
| `interpolation` | COMBO | بله | `"linear"`<br>`"ease_in"`<br>`"ease_out"`<br>`"ease_in_out"`<br>`"constant"` | زمان‌بندی/سرعت حرکت در طول مسیر را کنترل می‌کند. مقدار پیش‌فرض "linear" است. |
| `track_mask` | MASK | خیر | - | ماسک اختیاری برای نشان دادن فریم‌های قابل مشاهده. |

**نکته:** پارامترهای `mid_x` و `mid_y` فقط زمانی استفاده می‌شوند که پارامتر `bezier` روی `True` تنظیم شده باشد. وقتی `bezier` روی `False` است، مسیر یک خط مستقیم از نقطه شروع تا نقطه پایان خواهد بود.

## خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-----------|----------|----------|
| `TRACKS` | TRACKS | یک شیء مسیر شامل مختصات مسیر تولیدشده و اطلاعات قابلیت مشاهده برای تمام مسیرها در تمام فریم‌ها. |
| `track_length` | INT | تعداد فریم‌هایی که مسیرها برای آنها تولید شده‌اند، مطابق با `num_frames` ورودی. |
