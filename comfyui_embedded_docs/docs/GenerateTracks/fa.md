گره `GenerateTracks` چندین مسیر حرکتی موازی برای تولید ویدیو ایجاد می‌کند. این گره یک مسیر اصلی از نقطه شروع تا نقطه پایان تعریف می‌کند، سپس مجموعه‌ای از مسیرهای موازی با این مسیر را با فاصله‌های مساوی از یکدیگر تولید می‌کند. می‌توانید شکل مسیر (خط مستقیم یا منحنی بزیه)، سرعت حرکت در طول آن و فریم‌هایی که مسیرها در آنها قابل مشاهده هستند را کنترل کنید.

## ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|---------|----------|-------|---------|----------|
| `عرض` | INT | بله | 16 - 4096 | عرض فریم ویدیو بر حسب پیکسل. مقدار پیش‌فرض 832 است. |
| `ارتفاع` | INT | بله | 16 - 4096 | ارتفاع فریم ویدیو بر حسب پیکسل. مقدار پیش‌فرض 480 است. |
| `مختصات X شروع` | FLOAT | بله | 0.0 - 1.0 | مختصات X نرمال‌شده (0-1) برای موقعیت شروع. مقدار پیش‌فرض 0.0 است. |
| `مختصات Y شروع` | FLOAT | بله | 0.0 - 1.0 | مختصات Y نرمال‌شده (0-1) برای موقعیت شروع. مقدار پیش‌فرض 0.0 است. |
| `مختصات X پایان` | FLOAT | بله | 0.0 - 1.0 | مختصات X نرمال‌شده (0-1) برای موقعیت پایان. مقدار پیش‌فرض 1.0 است. |
| `مختصات Y پایان` | FLOAT | بله | 0.0 - 1.0 | مختصات Y نرمال‌شده (0-1) برای موقعیت پایان. مقدار پیش‌فرض 1.0 است. |
| `تعداد فریم‌ها` | INT | بله | 1 - 1024 | تعداد کل فریم‌هایی که موقعیت‌های مسیر برای آنها تولید می‌شود. مقدار پیش‌فرض 81 است. |
| `تعداد مسیرها` | INT | بله | 1 - 100 | تعداد مسیرهای موازی برای تولید. مقدار پیش‌فرض 5 است. |
| `فاصله مسیرها` | FLOAT | بله | 0.0 - 1.0 | فاصله نرمال‌شده بین مسیرها. مسیرها عمود بر جهت حرکت پخش می‌شوند. مقدار پیش‌فرض 0.025 است. |
| `Bezier` | BOOLEAN | بله | True / False | فعال‌سازی مسیر منحنی بزیه با استفاده از نقطه میانی به عنوان نقطه کنترل. مقدار پیش‌فرض False است. |
| `کنترل X میانی` | FLOAT | بله | 0.0 - 1.0 | نقطه کنترل X نرمال‌شده برای منحنی بزیه. فقط زمانی استفاده می‌شود که 'bezier' فعال باشد. مقدار پیش‌فرض 0.5 است. |
| `کنترل Y میانی` | FLOAT | بله | 0.0 - 1.0 | نقطه کنترل Y نرمال‌شده برای منحنی بزیه. فقط زمانی استفاده می‌شود که 'bezier' فعال باشد. مقدار پیش‌فرض 0.5 است. |
| `درون‌یابی` | COMBO | بله | `"linear"`<br>`"ease_in"`<br>`"ease_out"`<br>`"ease_in_out"`<br>`"constant"` | زمان‌بندی/سرعت حرکت در طول مسیر را کنترل می‌کند. مقدار پیش‌فرض "linear" است. |
| `ماسک مسیر` | MASK | خیر | - | ماسک اختیاری برای نشان دادن فریم‌های قابل مشاهده. |

**نکته:** پارامترهای `mid_x` و `mid_y` فقط زمانی استفاده می‌شوند که پارامتر `bezier` روی `True` تنظیم شده باشد. وقتی `bezier` روی `False` است، مسیر یک خط مستقیم از نقطه شروع تا نقطه پایان خواهد بود.
