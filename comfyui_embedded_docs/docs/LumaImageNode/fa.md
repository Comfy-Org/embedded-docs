### نمای کلی

تصاویر را به‌صورت همزمان بر اساس یک متن راهنما (prompt) و نسبت ابعاد تولید می‌کند. این گره با استفاده از توضیحات متنی، تصاویر ایجاد کرده و به شما امکان می‌دهد ابعاد و سبک تصویر را از طریق ورودی‌های مرجع مختلف، از جمله تصاویر شخصیت و سبک، کنترل کنید.

### ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `prompt` | STRING | بله | - | راهنمای تولید تصویر (پیش‌فرض: رشته خالی). باید حداقل ۳ کاراکتر داشته باشد. |
| `model` | COMBO | بله | `photon-flash-1`<br>`photon-1`<br>`photon` | انتخاب مدل برای تولید تصویر. مدل‌های مختلف هزینه‌های متفاوتی دارند. |
| `aspect_ratio` | COMBO | بله | `16:9`<br>`1:1`<br>`4:3`<br>`3:2`<br>`21:9`<br>`9:16`<br>`3:4`<br>`2:3`<br>`9:21` | نسبت ابعاد تصویر تولیدشده (پیش‌فرض: `16:9`) |
| `seed` | INT | بله | 0 تا 18446744073709551615 | دانه (seed) برای تعیین اینکه گره دوباره اجرا شود؛ نتایج واقعی بدون توجه به دانه، غیرقطعی هستند (پیش‌فرض: 0) |
| `style_image_weight` | FLOAT | خیر | 0.0 تا 1.0 | وزن تصویر سبک. اگر `style_image` ارائه نشود، نادیده گرفته می‌شود (پیش‌فرض: 1.0) |
| `image_luma_ref` | LUMA_REF | خیر | - | اتصال گره مرجع Luma برای تأثیرگذاری بر تولید با تصاویر ورودی؛ حداکثر ۴ تصویر در نظر گرفته می‌شود. |
| `style_image` | IMAGE | خیر | - | تصویر مرجع سبک؛ فقط ۱ تصویر استفاده خواهد شد. |
| `character_image` | IMAGE | خیر | - | تصاویر مرجع شخصیت؛ می‌تواند یک دسته از چندین تصویر باشد و حداکثر ۴ تصویر در نظر گرفته می‌شود. |

**محدودیت‌های پارامتر:**

- `prompt` باید پس از حذف فاصله‌های خالی، حداقل ۳ کاراکتر داشته باشد.
- پارامتر `image_luma_ref` می‌تواند حداکثر ۴ تصویر مرجع را بپذیرد.
- پارامتر `character_image` می‌تواند حداکثر ۴ تصویر مرجع شخصیت را بپذیرد.
- پارامتر `style_image` فقط ۱ تصویر مرجع سبک را می‌پذیرد.
- پارامتر `style_image_weight` تنها زمانی استفاده می‌شود که `style_image` ارائه شده باشد.
