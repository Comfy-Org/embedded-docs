Этот узел создаёт ключевые кадры перехвата из списка значений силы с плавающей точкой, равномерно распределяя их между заданными начальным и конечным процентами. Он генерирует последовательность ключевых кадров, где каждое значение силы назначается определённой процентной позиции на временной шкале анимации. Узел может либо создать новую группу ключевых кадров, либо добавить их к существующей, с возможностью вывода сгенерированных ключевых кадров в консоль для отладки.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `сила_в_виде_чисел` | FLOATS | Да | от -1 до ∞ | Одно значение с плавающей точкой или список значений с плавающей точкой, представляющих значения силы для ключевых кадров (по умолчанию: -1) |
| `процент_начала` | FLOAT | Да | от 0.0 до 1.0 | Начальная процентная позиция для первого ключевого кадра на временной шкале (по умолчанию: 0.0) |
| `процент_конца` | FLOAT | Да | от 0.0 до 1.0 | Конечная процентная позиция для последнего ключевого кадра на временной шкале (по умолчанию: 1.0) |
| `печать_ключевых_кадров` | BOOLEAN | Да | True/False | При включении выводит информацию о сгенерированных ключевых кадрах в консоль (по умолчанию: False) |
| `предыдущий_hook_kf` | HOOK_KEYFRAMES | Нет | - | Существующая группа ключевых кадров перехвата, к которой будут добавлены новые ключевые кадры, или создаётся новая группа, если не указана |

**Примечание:** Параметр `floats_strength` принимает как одно значение с плавающей точкой, так и итерируемый список чисел с плавающей точкой. Ключевые кадры распределяются линейно между `start_percent` и `end_percent` на основе количества предоставленных значений силы. Гарантируется, что первый ключевой кадр будет иметь как минимум один шаг, чтобы он был применён.

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `HOOK_KF` | HOOK_KEYFRAMES | Группа ключевых кадров перехвата, содержащая вновь созданные ключевые кадры, либо как новая группа, либо добавленная к входной группе ключевых кадров |
