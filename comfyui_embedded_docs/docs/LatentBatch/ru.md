Узел LatentBatch предназначен для объединения двух наборов скрытых (латентных) выборок в один пакет с возможностью изменения размера одного набора для соответствия размерам другого перед объединением. Эта операция упрощает комбинирование различных скрытых представлений для дальнейшей обработки или задач генерации.

## Входные данные

| Параметр     | Тип данных | Описание |
|--------------|-------------|-------------|
| `samples1`   | `LATENT`    | Первый набор скрытых выборок для объединения. Играет ключевую роль в определении окончательной формы объединённого пакета. |
| `samples2`   | `LATENT`    | Второй набор скрытых выборок для объединения. Если его размеры отличаются от первого набора, он изменяется для обеспечения совместимости перед объединением. |

## Выходные данные

| Параметр | Тип данных | Описание |
|-----------|-------------|-------------|
| `latent`  | `LATENT`    | Объединённый набор скрытых выборок, теперь скомбинированный в один пакет для дальнейшей обработки. |
