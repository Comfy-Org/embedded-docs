Узел Grok Image Edit изменяет существующее изображение на основе текстового запроса. Он использует API Grok для создания одного или нескольких новых изображений, являющихся вариациями исходного, с учётом вашего описания.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `model` | COMBO | Да | `"grok-imagine-image-quality"`<br>`"grok-imagine-image-pro"`<br>`"grok-imagine-image"`<br>`"grok-imagine-image-beta"` | Конкретная модель ИИ для редактирования изображений. |
| `image` | IMAGE | Да | | Входное изображение(я) для редактирования. Поддерживается до 3 входных изображений, кроме модели "pro", которая поддерживает только 1. |
| `prompt` | STRING | Да | | Текстовый запрос для создания отредактированного изображения. Должен содержать не менее 1 символа после удаления пробелов. |
| `resolution` | COMBO | Да | `"1K"`<br>`"2K"` | Разрешение выходного изображения. |
| `number_of_images` | INT | Нет | от 1 до 10 | Количество создаваемых отредактированных изображений (по умолчанию: 1). |
| `seed` | INT | Нет | от 0 до 2147483647 | Сид для определения необходимости повторного запуска узла; фактические результаты недетерминированы независимо от сида (по умолчанию: 0). |
| `соотношение сторон` | COMBO | Нет | `"auto"`<br>`"1:1"`<br>`"2:3"`<br>`"3:2"`<br>`"3:4"`<br>`"4:3"`<br>`"9:16"`<br>`"16:9"`<br>`"9:19.5"`<br>`"19.5:9"`<br>`"9:20"`<br>`"20:9"`<br>`"1:2"`<br>`"2:1"` | Соотношение сторон выходного изображения. Допускается только при подключении нескольких изображений к входу `image`. Если установлено значение "auto", соотношение сторон определяется автоматически (по умолчанию: "auto"). |

**Важные ограничения:**
- Вход `image` поддерживает до 3 изображений, за исключением использования модели `grok-imagine-image-pro`, которая поддерживает только 1 входное изображение.
- Параметр `aspect_ratio` можно установить в пользовательское значение (не "auto") только при подключении нескольких изображений к входу `image`. Установка пользовательского соотношения сторон при одном входном изображении вызовет ошибку.

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `output` | IMAGE | Отредактированное(ые) изображение(я), созданное(ые) узлом. Если `number_of_images` больше 1, выходные данные объединяются в пакет. |
