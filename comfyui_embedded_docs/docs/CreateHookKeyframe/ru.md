Узел Create Hook Keyframe позволяет задавать конкретные точки в процессе генерации, в которых изменяется поведение хуков. Он создаёт ключевые кадры, изменяющие силу хуков на определённых процентах выполнения генерации. Эти ключевые кадры можно объединять в цепочки для создания сложных схем управления.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `множитель_силы` | FLOAT | Да | -20.0 до 20.0 | Множитель силы хука в этом ключевом кадре (по умолчанию: 1.0) |
| `процент_начала` | FLOAT | Да | 0.0 до 1.0 | Процентная точка в процессе генерации, в которой вступает в силу этот ключевой кадр (по умолчанию: 0.0) |
| `предыдущий_hook_kf` | HOOK_KEYFRAMES | Нет | - | Необязательная предыдущая группа ключевых кадров хука, к которой нужно добавить этот ключевой кадр |

## Выходные параметры

| Имя выходного параметра | Тип данных | Описание |
|-------------|-----------|-------------|
| `HOOK_KF` | HOOK_KEYFRAMES | Группа ключевых кадров хука, включающая только что созданный ключевой кадр |
