## نمای کلی

نویز همبسته زمانی را برای پاس دوم فرآیند بهبود ویدیوی VOID تولید می‌کند. این گره، ویدیوی خروجی از پاس ۱ را گرفته و نویز گاوسی را در امتداد بردارهای جریان نوری تغییر شکل می‌دهد و نویزی ایجاد می‌کند که به‌طور هماهنگ با محتوای ویدیو حرکت می‌کند. این نویز تغییرشکل‌یافته به‌عنوان لِیتنت اولیه برای پاس ۲ استفاده می‌شود که سازگاری زمانی را در خروجی نهایی بهبود می‌بخشد.

## ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `optical_flow` | MODEL | بله | - | مدل جریان نوری از OpticalFlowLoader (RAFT-large). |
| `video` | IMAGE | بله | - | فریم‌های ویدیوی خروجی پاس ۱ [T, H, W, 3]. |
| `width` | INT | بله | ۱۶ تا MAX_RESOLUTION (گام ۸) | عرض لِیتنت خروجی (پیش‌فرض: ۶۷۲). |
| `height` | INT | بله | ۱۶ تا MAX_RESOLUTION (گام ۸) | ارتفاع لِیتنت خروجی (پیش‌فرض: ۳۸۴). |
| `length` | INT | بله | ۱ تا MAX_RESOLUTION (گام ۱) | تعداد فریم‌های پیکسلی. به پایین گرد می‌شود تا latent_t زوج شود (نیازمندی patch_size_t=2)، مثلاً ۴۹ → ۴۵ (پیش‌فرض: ۴۵). |
| `batch_size` | INT | بله | ۱ تا ۶۴ | تعداد دنباله‌های نویز یکسان برای تولید (پیش‌فرض: ۱). |

**نکته درباره پارامتر `length`:** مقدار `length` به‌طور خودکار به نزدیک‌ترین مقدار معتبری که باعث زوج شدن بعد `latent_t` می‌شود، به پایین گرد می‌شود. این امر به دلیل محدودیت `patch_size_t=2` مدل CogVideoX-Fun-V1.5 ضروری است. در صورت گرد شدن، یک هشدار ثبت می‌شود.
