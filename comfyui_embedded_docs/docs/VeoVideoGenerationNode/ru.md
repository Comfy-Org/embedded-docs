Вот перевод документации на русский язык:

Генерирует видео из текстовых запросов с использованием API Google Veo 2. Этот узел может создавать видео на основе текстовых описаний и опциональных изображений, с контролем таких параметров, как соотношение сторон, длительность и другое.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `prompt` | STRING | Да | - | Текстовое описание видео (по умолчанию: пусто) |
| `aspect_ratio` | COMBO | Да | "16:9"<br>"9:16" | Соотношение сторон выходного видео (по умолчанию: "16:9") |
| `negative_prompt` | STRING | Нет | - | Отрицательный текстовый запрос, указывающий, чего следует избегать в видео (по умолчанию: пусто) |
| `duration_seconds` | INT | Нет | 5-8 | Длительность выходного видео в секундах (по умолчанию: 5) |
| `enhance_prompt` | BOOLEAN | Нет | - | Улучшать ли запрос с помощью ИИ (по умолчанию: True). Это расширенный параметр. |
| `person_generation` | COMBO | Нет | "ALLOW"<br>"BLOCK" | Разрешить ли генерацию людей в видео (по умолчанию: "ALLOW"). Это расширенный параметр. |
| `seed` | INT | Нет | 0-4294967295 | Сид для генерации видео (0 для случайного) (по умолчанию: 0). Это расширенный параметр. |
| `image` | IMAGE | Нет | - | Опциональное референсное изображение для направления генерации видео |
| `model` | COMBO | Нет | "veo-2.0-generate-001" | Модель Veo 2 для генерации видео (по умолчанию: "veo-2.0-generate-001") |

**Примечание:** Параметр `generate_audio` доступен только для моделей Veo 3.0 и автоматически обрабатывается узлом на основе выбранной модели. При использовании моделей Veo 3.0 параметр `enhance_prompt` принудительно устанавливается в True.

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `output` | VIDEO | Сгенерированный видеофайл |
