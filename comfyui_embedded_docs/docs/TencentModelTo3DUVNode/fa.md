این گره از API Tencent Hunyuan3D برای انجام بازکردن UV بر روی یک مدل سه‌بعدی استفاده می‌کند. یک فایل مدل سه‌بعدی را به عنوان ورودی دریافت کرده، آن را برای پردازش به API ارسال می‌کند و مدل پردازش‌شده را در قالب‌های OBJ و FBX به همراه یک تصویر بافت UV تولیدشده بازمی‌گرداند. مدل ورودی باید کمتر از ۳۰,۰۰۰ وجه داشته باشد.

## ورودی‌ها

| پارامتر | نوع داده | اجباری | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `model_3d` | FILE3D | بله | GLB<br>OBJ<br>FBX | مدل سه‌بعدی ورودی (GLB، OBJ یا FBX). مدل باید کمتر از ۳۰۰۰۰ وجه داشته باشد. |
| `seed` | INT | خیر | ۰ تا ۲۱۴۷۴۸۳۶۴۷ | یک مقدار دانه (پیش‌فرض: ۱). این مقدار مشخص می‌کند که آیا گره باید دوباره اجرا شود، اما نتایج صرف‌نظر از مقدار دانه، غیرقطعی هستند. |

## خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-------------|-----------|-------------|
| `OBJ` | FILE3D | فایل مدل سه‌بعدی پردازش‌شده در قالب OBJ. |
| `FBX` | FILE3D | فایل مدل سه‌بعدی پردازش‌شده در قالب FBX. |
| `uv_image` | IMAGE | تصویر بافت UV تولیدشده. |
