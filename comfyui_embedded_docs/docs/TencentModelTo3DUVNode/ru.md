Этот узел использует API Tencent Hunyuan3D для выполнения UV-развертки 3D-модели. Он принимает файл 3D-модели на вход, отправляет его в API для обработки и возвращает обработанную модель в форматах OBJ и FBX вместе с сгенерированным UV-текстурным изображением. Входная модель должна содержать менее 30 000 полигонов.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `3D-модель` | FILE3D | Да | GLB<br>OBJ<br>FBX | Входная 3D-модель (GLB, OBJ или FBX). Модель должна содержать менее 30 000 полигонов. |
| `seed` | INT | Нет | от 0 до 2147483647 | Значение зерна (по умолчанию: 1). Управляет необходимостью повторного запуска узла, однако результаты остаются недетерминированными независимо от значения зерна. |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `OBJ` | FILE3D | Обработанный файл 3D-модели в формате OBJ. |
| `FBX` | FILE3D | Обработанный файл 3D-модели в формате FBX. |
| `uv_image` | IMAGE | Сгенерированное UV-текстурное изображение. |
