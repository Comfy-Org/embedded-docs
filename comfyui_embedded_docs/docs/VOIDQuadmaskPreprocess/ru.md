## Обзор

Узел VOIDQuadmaskPreprocess подготавливает маску для инpainting-а VOID, преобразуя её в специальную четырёхуровневую "квадромаску". Он принимает входную маску, опционально расширяет основную область, затем квантует значения маски на четыре различных уровня, представляющих разные семантические области (основной объект, перекрытие, затронутая область и фон). В завершение маска инвертируется и нормализуется, так что выходные значения находятся в диапазоне [0, 1], где 1.0 указывает область для удаления, а 0.0 — область для сохранения.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `mask` | MASK | Да | Н/Д | Входная маска для предварительной обработки. |
| `dilate_width` | INT | Нет | от 0 до 50 (шаг: 1) | Радиус расширения для основной области маски. Значение 0 означает отсутствие расширения. (по умолчанию: 0) |

## Выходные данные

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `quadmask` | MASK | Предварительно обработанная квадромаска со значениями в диапазоне [0, 1], представляющая четыре дискретных уровня: 1.0 (основной объект для удаления), ~0.75 (перекрытие основного и затронутого), ~0.50 (затронутая область) и 0.0 (фон для сохранения). |
