## ## 概述

VOIDQuadmaskPreprocess 节点通过将输入掩码转换为特殊的四层级“quadmask”，为 VOID 图像修复任务准备掩码。该节点接收输入掩码，可选择对主要区域进行膨胀处理，然后将掩码值量化为四个不同层级，分别代表不同的语义区域（主要物体、重叠区域、受影响区域和背景）。最后，节点会对掩码进行反转和归一化，使输出值范围在 [0, 1] 之间，其中 1.0 表示需要移除的区域，0.0 表示需要保留的区域。

## ## 输入

| 参数 | 数据类型 | 是否必填 | 范围 | 描述 |
|-----------|-----------|----------|-------|-------------|
| `mask` | MASK | 是 | 不适用 | 需要预处理的输入掩码。 |
| `dilate_width` | INT | 否 | 0 到 50（步长：1） | 主要掩码区域的膨胀半径。值为 0 表示不进行膨胀处理。（默认值：0） |

## ## 输出

| 输出名称 | 数据类型 | 描述 |
|-------------|-----------|-------------|
| `quadmask` | MASK | 预处理后的 quadmask，值范围在 [0, 1] 之间，代表四个离散层级：1.0（需要移除的主要物体）、约 0.75（主要物体与受影响区域的重叠部分）、约 0.50（受影响区域）和 0.0（需要保留的背景）。 |
