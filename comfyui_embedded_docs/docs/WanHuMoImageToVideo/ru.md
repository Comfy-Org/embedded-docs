Вот перевод документации узла WanHuMoImageToVideo на русский язык:

Узел WanHuMoImageToVideo преобразует изображения в видеопоследовательности, генерируя скрытые представления для видеокадров. Он обрабатывает входные данные кондиционирования и может использовать референсные изображения и аудиовложения для влияния на генерацию видео. Узел выводит модифицированные данные кондиционирования и скрытые представления, подходящие для синтеза видео.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `позитивный` | CONDITIONING | Да | - | Положительное кондиционирование, направляющее генерацию видео к желаемому содержимому |
| `негативный` | CONDITIONING | Да | - | Отрицательное кондиционирование, отводящее генерацию видео от нежелательного содержимого |
| `VAE` | VAE | Да | - | Модель VAE, используемая для кодирования референсных изображений в скрытое пространство |
| `ширина` | INT | Да | от 16 до MAX_RESOLUTION | Ширина выходных видеокадров в пикселях (по умолчанию: 832, должна делиться на 16) |
| `высота` | INT | Да | от 16 до MAX_RESOLUTION | Высота выходных видеокадров в пикселях (по умолчанию: 480, должна делиться на 16) |
| `длина` | INT | Да | от 1 до MAX_RESOLUTION | Количество кадров в сгенерированной видеопоследовательности (по умолчанию: 97, должно быть таким, чтобы (length - 1) делилось на 4) |
| `размер_пакета` | INT | Да | от 1 до 4096 | Количество видеопоследовательностей, генерируемых одновременно (по умолчанию: 1) |
| `выход_аудиокодера` | AUDIOENCODEROUTPUT | Нет | - | Необязательные данные аудиокодирования, которые могут влиять на генерацию видео на основе аудиосодержимого |
| `опорное_изображение` | IMAGE | Нет | - | Необязательное референсное изображение, используемое для направления стиля и содержимого генерации видео |

**Примечание:** Если указано референсное изображение, оно кодируется и добавляется как к положительному, так и к отрицательному кондиционированию. Если указан выход аудиокодера, он обрабатывается и включается в данные кондиционирования. Если ни то, ни другое не указано, для референсных скрытых представлений и аудиовложений используются заполненные нулями тензоры-заполнители.

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `позитивный` | CONDITIONING | Модифицированное положительное кондиционирование с включенными референсным изображением и/или аудиовложениями |
| `негативный` | CONDITIONING | Модифицированное отрицательное кондиционирование с включенными референсным изображением и/или аудиовложениями |
| `latent` | LATENT | Сгенерированное скрытое представление, содержащее данные видеопоследовательности |
