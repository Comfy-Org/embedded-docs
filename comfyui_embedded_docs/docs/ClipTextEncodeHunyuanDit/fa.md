گره `CLIPTextEncodeHunyuanDiT` توضیحات متنی را به قالبی تبدیل می‌کند که مدل HunyuanDiT قادر به درک آن باشد. این یک گره شرط‌گذاری پیشرفته است که برای معماری رمزگذار دوگانه متنی HunyuanDiT طراحی شده و دو ورودی متنی مجزا را از طریق توکنایزرهای مختلف پردازش می‌کند.

## ورودی‌ها

| پارامتر | نوع داده | اجباری | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `clip` | CLIP | بله | - | یک نمونه مدل CLIP که برای توکن‌سازی و رمزگذاری متن استفاده می‌شود و هسته اصلی تولید شرایط است. |
| `bert` | STRING | بله | - | ورودی متن برای رمزگذاری از طریق توکنایزر BERT. عبارات و کلمات کلیدی را ترجیح می‌دهد. از متن چندخطی و پرامپت‌های پویا پشتیبانی می‌کند. |
| `mt5xl` | STRING | بله | - | ورودی متن برای رمزگذاری از طریق توکنایزر mT5-XL. از متن چندخطی و پرامپت‌های پویا (چندزبانه) پشتیبانی می‌کند. می‌توان از جملات کامل و توضیحات پیچیده استفاده کرد. |
