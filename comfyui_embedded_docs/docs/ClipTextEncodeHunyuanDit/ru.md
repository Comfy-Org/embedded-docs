Вот перевод документации на русский язык:

Узел `CLIPTextEncodeHunyuanDiT` преобразует текстовые описания в формат, понятный модели HunyuanDiT. Это продвинутый узел кондиционирования, разработанный для архитектуры с двумя текстовыми энкодерами HunyuanDiT, который обрабатывает два отдельных текстовых входа через разные токенизаторы.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `clip` | CLIP | Да | - | Экземпляр модели CLIP, используемый для токенизации и кодирования текста, являющийся основой для создания условий. |
| `bert` | STRING | Да | - | Текстовый ввод для кодирования через токенизатор BERT. Предпочтительны фразы и ключевые слова. Поддерживает многострочный и динамический ввод. |
| `mt5xl` | STRING | Да | - | Текстовый ввод для кодирования через токенизатор mT5-XL. Поддерживает многострочный и динамический ввод (многоязычный). Можно использовать полные предложения и сложные описания. |

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `CONDITIONING` | CONDITIONING | Закодированный результат кондиционирования, объединяющий текст, токенизированный как BERT, так и mT5-XL, используемый для дальнейшей обработки в задачах генерации. |
