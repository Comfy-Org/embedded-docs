# ترجمة وثيقة عقد ComfyUI

هذه العقدة تقوم بتعديل صورة موجودة بناءً على نص موجه ومعامل قوة. تستخدم واجهة برمجة تطبيقات Recraft لتحويل الصورة المدخلة وفقًا للوصف المقدم مع الحفاظ على بعض التشابه مع الصورة الأصلية بناءً على إعداد القوة.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|---------|--------------|--------|--------|-------|
| `صورة` | IMAGE | نعم | - | الصورة المدخلة المراد تعديلها |
| `الوصف` | STRING | نعم | - | النص الموجّه لتوليد الصورة (الافتراضي: ""، الحد الأقصى للطول: 1000 حرف) |
| `عدد` | INT | نعم | 1-6 | عدد الصور المراد توليدها (الافتراضي: 1) |
| `القوة` | FLOAT | نعم | 0.0-1.0 | يحدد درجة الاختلاف عن الصورة الأصلية، يجب أن يكون في النطاق [0, 1]، حيث 0 يعني متطابقة تقريبًا، و1 يعني تشابه ضئيل جدًا (الافتراضي: 0.5) |
| `بذرة` | INT | نعم | 0-18446744073709551615 | البذرة لتحديد ما إذا كانت العقدة ستعيد التشغيل؛ النتائج الفعلية غير حتمية بغض النظر عن البذرة (الافتراضي: 0) |
| `نمط إعادة الصياغة` | STYLEV3 | لا | - | اختيار نمط اختياري لتوليد الصورة. إذا لم يتم توفيره، يكون الافتراضي هو `realistic_image` |
| `الوصف السلبي` | STRING | لا | - | وصف نصي اختياري للعناصر غير المرغوب فيها في الصورة (الافتراضي: "") |
| `عناصر تحكم إعادة الصياغة` | CONTROLS | لا | - | عناصر تحكم إضافية اختيارية على عملية التوليد عبر عقدة عناصر تحكم Recraft |

**ملاحظة:** معامل `seed` يؤدي فقط إلى إعادة تنفيذ العقدة ولكنه لا يضمن نتائج حتمية. يتم تقريب معامل القوة إلى منزلتين عشريتين داخليًا. يتم التحقق من صحة النص الموجّه ويجب ألا يتجاوز 1000 حرف. إذا لم يتم توفير `recraft_style`، فإن العقدة تستخدم النمط `realistic_image` كإعداد افتراضي.
