این گره یک تصویر موجود را بر اساس یک متن راهنما (prompt) و پارامتر شدت (strength) تغییر می‌دهد. از API Recraft برای تبدیل تصویر ورودی مطابق با توضیحات ارائه‌شده استفاده می‌کند، در حالی که بر اساس تنظیم شدت، شباهتی به تصویر اصلی حفظ می‌کند.

## ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|---------|----------|-------|--------|---------|
| `تصویر` | IMAGE | بله | - | تصویر ورودی که باید تغییر کند |
| `پرامپت` | STRING | بله | - | راهنمای تولید تصویر (پیش‌فرض: ""، حداکثر طول: ۱۰۰۰ کاراکتر) |
| `تعداد` | INT | بله | ۱-۶ | تعداد تصاویری که باید تولید شود (پیش‌فرض: ۱) |
| `قدرت` | FLOAT | بله | ۰.۰-۱.۰ | میزان تفاوت با تصویر اصلی را تعیین می‌کند؛ باید در بازه [0, 1] باشد، که ۰ به معنی تقریباً یکسان و ۱ به معنی شباهت بسیار کم است (پیش‌فرض: ۰.۵) |
| `seed` | INT | بله | ۰-۱۸۴۴۶۷۴۴۰۷۳۷۰۹۵۵۱۶۱۵ | دانه (seed) برای تعیین اینکه آیا گره دوباره اجرا شود؛ نتایج واقعی بدون توجه به دانه، قطعی نیستند (پیش‌فرض: ۰) |
| `سبک Recraft` | STYLEV3 | خیر | - | انتخاب سبک اختیاری برای تولید تصویر. در صورت عدم ارائه، به طور پیش‌فرض `realistic_image` استفاده می‌شود |
| `پرامپت منفی` | STRING | خیر | - | توضیح متنی اختیاری از عناصر نامطلوب در تصویر (پیش‌فرض: "") |
| `کنترل‌های Recraft` | CONTROLS | خیر | - | کنترل‌های اضافی اختیاری بر روی تولید از طریق گره Recraft Controls |

**توجه:** پارامتر `seed` فقط باعث اجرای مجدد گره می‌شود اما نتایج قطعی را تضمین نمی‌کند. پارامتر strength به صورت داخلی به ۲ رقم اعشار گرد می‌شود. راهنما (prompt) اعتبارسنجی می‌شود و نباید از ۱۰۰۰ کاراکتر تجاوز کند. اگر `recraft_style` ارائه نشود، گره به طور پیش‌فرض از سبک `realistic_image` استفاده می‌کند.
