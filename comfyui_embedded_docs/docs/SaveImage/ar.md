# ترجمة وثيقة عقدة SaveImage

يقوم عقدة **حفظ الصورة** (SaveImage) بحفظ الصور التي يستقبلها في دليل `ComfyUI/output` الخاص بك. يحفظ كل صورة كملف PNG، ويمكنه تضمين بيانات وصفية لسير العمل، مثل الاستعلام (prompt)، في الملف المحفوظ للرجوع إليها مستقبلاً.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|---------|--------------|--------|--------|-------|
| `الصور` | IMAGE | نعم | - | الصور المراد حفظها. |
| `بادئة اسم الملف` | STRING | نعم | - | البادئة لاسم الملف المراد حفظه. قد تتضمن معلومات تنسيق مثل `%date:yyyy-MM-dd%` أو `%Empty Latent Image.width%` لتضمين قيم من العقد (الافتراضي: "ComfyUI"). |

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|------------|--------------|-------|
| `ui` | UI_RESULT | يُخرج هذا العقد نتيجة واجهة مستخدم تحتوي على قائمة بالصور المحفوظة مع أسماء ملفاتها ومجلداتها الفرعية. لا يُخرج بيانات للتوصيل بعقد أخرى. |
