گره SaveImage تصاویر دریافتی را در دایرکتوری `ComfyUI/output` ذخیره می‌کند. هر تصویر را به‌عنوان یک فایل PNG ذخیره کرده و می‌تواند فراداده‌های گردش کار، مانند پرامپت، را برای مراجعات بعدی در فایل ذخیره‌شده جاسازی کند.

## ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|---------|----------|-------|--------|---------|
| `images` | IMAGE | بله | - | تصاویری که باید ذخیره شوند. |
| `filename_prefix` | STRING | بله | - | پیشوند نام فایل برای ذخیره‌سازی. این می‌تواند شامل اطلاعات قالب‌بندی مانند `%date:yyyy-MM-dd%` یا `%Empty Latent Image.width%` برای درج مقادیر از گره‌ها باشد (پیش‌فرض: "ComfyUI"). |

## خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-----------|----------|---------|
| `ui` | UI_RESULT | این گره یک نتیجه رابط کاربری شامل لیستی از تصاویر ذخیره‌شده به‌همراه نام فایل‌ها و زیرپوشه‌های آن‌ها را خروجی می‌دهد. این خروجی داده‌ای برای اتصال به سایر گره‌ها ارائه نمی‌دهد. |
