گره Hunyuan3Dv2ConditioningMultiView، جاسازی‌های دید چندنمایی CLIP را برای تولید ویدیوی سه‌بعدی پردازش می‌کند. این گره جاسازی‌های اختیاری نماهای جلو، چپ، عقب و راست را دریافت کرده و با رمزگذاری موقعیتی ترکیب می‌کند تا داده‌های شرطی‌سازی برای مدل‌های ویدیویی ایجاد کند. خروجی گره شامل شرطی‌سازی مثبت از جاسازی‌های ترکیبی و شرطی‌سازی منفی با مقادیر صفر است.

## ورودی‌ها

| پارامتر | نوع داده | اجباری | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `front` | CLIP_VISION_OUTPUT | خیر | - | خروجی دید CLIP برای نمای جلو |
| `left` | CLIP_VISION_OUTPUT | خیر | - | خروجی دید CLIP برای نمای چپ |
| `back` | CLIP_VISION_OUTPUT | خیر | - | خروجی دید CLIP برای نمای عقب |
| `right` | CLIP_VISION_OUTPUT | خیر | - | خروجی دید CLIP برای نمای راست |

**نکته:** برای عملکرد گره، حداقل یک ورودی نما باید ارائه شود. گره تنها نماهایی را پردازش می‌کند که حاوی داده‌های معتبر خروجی دید CLIP باشند.

## خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-------------|-----------|-------------|
| `positive` | CONDITIONING | شرطی‌سازی مثبت حاوی جاسازی‌های ترکیبی چندنمایی با رمزگذاری موقعیتی |
| `negative` | CONDITIONING | شرطی‌سازی منفی با مقادیر صفر برای یادگیری تقابلی |
