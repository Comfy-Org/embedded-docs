# ترجمة وثيقة عقد Hunyuan3Dv2ConditioningMultiView

يقوم عقد Hunyuan3Dv2ConditioningMultiView بمعالجة تضمينات الرؤية متعددة الزوايا من CLIP لتوليد الفيديو ثلاثي الأبعاد. يستقبل العقد تضمينات اختيارية للزوايا الأمامية واليسرى والخلفية واليمنى، ويدمجها مع الترميز الموضعي لإنشاء بيانات تهيئة لنماذج الفيديو. ينتج العقد تهيئة إيجابية من التضمينات المدمجة وتهيئة سلبية بقيم صفرية.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|---------|---------------|--------|--------|-------|
| `front` | CLIP_VISION_OUTPUT | لا | - | مخرجات رؤية CLIP للزاوية الأمامية |
| `left` | CLIP_VISION_OUTPUT | لا | - | مخرجات رؤية CLIP للزاوية اليسرى |
| `back` | CLIP_VISION_OUTPUT | لا | - | مخرجات رؤية CLIP للزاوية الخلفية |
| `right` | CLIP_VISION_OUTPUT | لا | - | مخرجات رؤية CLIP للزاوية اليمنى |

**ملاحظة:** يجب توفير مدخل زاوية واحد على الأقل لكي يعمل العقد. سيقوم العقد بمعالجة الزوايا التي تحتوي على بيانات مخرجات رؤية CLIP صالحة فقط.

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|------------|---------------|--------|
| `positive` | CONDITIONING | تهيئة إيجابية تحتوي على التضمينات متعددة الزوايا المدمجة مع الترميز الموضعي |
| `negative` | CONDITIONING | تهيئة سلبية بقيم صفرية للتعلم التبايني |
