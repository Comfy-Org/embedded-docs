Узел Hunyuan3Dv2ConditioningMultiView обрабатывает многовидовые CLIP-эмбеддинги для генерации 3D-видео. Он принимает опциональные эмбеддинги переднего, левого, заднего и правого видов и объединяет их с позиционным кодированием для создания данных conditioning для видеомоделей. Узел выводит как положительное conditioning на основе объединённых эмбеддингов, так и отрицательное conditioning с нулевыми значениями.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `front` | CLIP_VISION_OUTPUT | Нет | - | Выходные данные CLIP vision для переднего вида |
| `left` | CLIP_VISION_OUTPUT | Нет | - | Выходные данные CLIP vision для левого вида |
| `back` | CLIP_VISION_OUTPUT | Нет | - | Выходные данные CLIP vision для заднего вида |
| `right` | CLIP_VISION_OUTPUT | Нет | - | Выходные данные CLIP vision для правого вида |

**Примечание:** Для работы узла необходимо указать хотя бы один входной вид. Узел обрабатывает только те виды, которые содержат корректные выходные данные CLIP vision.

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `positive` | CONDITIONING | Положительное conditioning, содержащее объединённые многовидовые эмбеддинги с позиционным кодированием |
| `negative` | CONDITIONING | Отрицательное conditioning с нулевыми значениями для контрастного обучения |
