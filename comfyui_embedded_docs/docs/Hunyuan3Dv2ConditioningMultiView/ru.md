Узел Hunyuan3Dv2ConditioningMultiView обрабатывает мульти-видовые эмбеддинги CLIP vision для генерации 3D-видео. Он принимает опциональные эмбеддинги для фронтального, левого, заднего и правого видов и объединяет их с позиционным кодированием для создания кондиционирующих данных для видео-моделей. Узел выводит как позитивное кондиционирование из объединённых эмбеддингов, так и негативное кондиционирование с нулевыми значениями.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `front` | CLIP_VISION_OUTPUT | Нет | - | Выход CLIP vision для фронтального вида |
| `left` | CLIP_VISION_OUTPUT | Нет | - | Выход CLIP vision для левого вида |
| `back` | CLIP_VISION_OUTPUT | Нет | - | Выход CLIP vision для заднего вида |
| `right` | CLIP_VISION_OUTPUT | Нет | - | Выход CLIP vision для правого вида |

**Примечание:** Для работы узла должен быть предоставлен как минимум один вход вида. Узел будет обрабатывать только те виды, которые содержат валидные данные выхода CLIP vision.

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `positive` | CONDITIONING | Позитивное кондиционирование, содержащее объединённые мульти-видовые эмбеддинги с позиционным кодированием |
| `negative` | CONDITIONING | Негативное кондиционирование с нулевыми значениями для контрастного обучения |
