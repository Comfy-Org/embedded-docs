Этот узел обнаруживает модели, расположенные в папке `ComfyUI/models/gligen`, а также считывает модели из дополнительных путей, указанных в файле extra_model_paths.yaml. Иногда может потребоваться **обновить интерфейс ComfyUI**, чтобы он смог прочитать файлы моделей из соответствующей папки.

Узел `GLIGENLoader` предназначен для загрузки моделей GLIGEN, которые являются специализированными генеративными моделями. Он упрощает процесс поиска и инициализации этих моделей по указанным путям, подготавливая их к дальнейшим генеративным задачам.

## Входы

| Поле          | Тип Comfy        | Описание                                                                                     |
|---------------|------------------|----------------------------------------------------------------------------------------------|
| `название_gligen` | `COMBO[STRING]`  | Имя загружаемой модели GLIGEN, указывающее, какой файл модели необходимо найти и загрузить. Имеет решающее значение для инициализации модели GLIGEN. |

## Выходы

| Поле     | Тип данных | Описание                                                                                              |
|----------|------------|-------------------------------------------------------------------------------------------------------|
| `gligen` | `GLIGEN`   | Загруженная модель GLIGEN, готовая к использованию в генеративных задачах. Представляет собой полностью инициализированную модель, загруженную из указанного пути. |
