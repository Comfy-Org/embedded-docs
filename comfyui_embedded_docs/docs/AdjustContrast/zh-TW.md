# 調整對比度節點

調整對比度節點用於修改輸入影像的對比度等級。其運作原理是調整影像中亮部與暗部之間的差異。係數為 1.0 時保持影像不變，低於 1.0 的值會降低對比度，高於 1.0 的值則會增加對比度。

| 參數 | 資料類型 | 必要 | 範圍 | 說明 |
|-----------|-----------|----------|-------|-------------|
| `image` | IMAGE | 是 | - | 需要調整對比度的輸入影像。 |
| `對比度係數` | FLOAT | 否 | 0.0 - 2.0 | 對比度係數。1.0 = 無變化，<1.0 = 降低對比度，>1.0 = 增加對比度。（預設值：1.0） |
