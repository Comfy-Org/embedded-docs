Узел LatentAdd предназначен для сложения двух латентных представлений. Он обеспечивает объединение признаков или характеристик, закодированных в этих представлениях, путём поэлементного сложения.

## Входы

| Параметр     | Тип данных | Описание |
|--------------|-------------|-------------|
| `образцы1`   | `LATENT`    | Первый набор латентных сэмплов для сложения. Представляет собой один из входов, признаки которого объединяются с другим набором латентных сэмплов. |
| `образцы2`   | `LATENT`    | Второй набор латентных сэмплов для сложения. Служит другим входом, признаки которого объединяются с первым набором латентных сэмплов путём поэлементного сложения. |

## Выходы

| Параметр | Тип данных | Описание |
|-----------|-------------|-------------|
| `latent`  | `LATENT`    | Результат поэлементного сложения двух латентных сэмплов, представляющий собой новый набор латентных сэмплов, объединяющий признаки обоих входов. |
