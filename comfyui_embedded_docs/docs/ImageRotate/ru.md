Вот перевод документации на русский язык:

Узел ImageRotate поворачивает входное изображение на заданные углы. Он поддерживает четыре варианта поворота: без поворота, на 90 градусов по часовой стрелке, на 180 градусов и на 270 градусов по часовой стрелке. Поворот выполняется с помощью эффективных операций с тензорами, которые сохраняют целостность данных изображения.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `image` | IMAGE | Да | - | Входное изображение для поворота |
| `rotation` | STRING | Да | "none"<br>"90 degrees"<br>"180 degrees"<br>"270 degrees" | Угол поворота, применяемый к изображению (по умолчанию: "none") |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `image` | IMAGE | Повернутое выходное изображение |
