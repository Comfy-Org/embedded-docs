# ترجمة وثيقة عقدة ImageRotate

عقدة **ImageRotate** تقوم بتدوير الصورة المدخلة بزوايا محددة. تدعم أربعة خيارات للتدوير: بدون تدوير، 90 درجة باتجاه عقارب الساعة، 180 درجة، و270 درجة باتجاه عقارب الساعة. يتم إجراء التدوير باستخدام عمليات موتر فعالة تحافظ على سلامة بيانات الصورة.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|---------|---------------|--------|---------|-------|
| `image` | IMAGE | نعم | - | الصورة المدخلة المراد تدويرها |
| `rotation` | STRING | نعم | "none"<br>"90 degrees"<br>"180 degrees"<br>"270 degrees" | زاوية التدوير المطبقة على الصورة (الافتراضي: "none") |

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|------------|---------------|-------|
| `image` | IMAGE | الصورة الناتجة بعد التدوير |
