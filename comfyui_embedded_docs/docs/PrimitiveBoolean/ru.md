Вот перевод документации на русский язык:

Узел Boolean предоставляет простой способ передачи логических значений (true/false) через ваш рабочий процесс. Он принимает логическое значение на входе и выводит то же самое значение без изменений, позволяя управлять логическими параметрами в других узлах.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `value` | BOOLEAN | Да | true<br>false | Логическое значение для передачи через узел |

## Выходы

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `output` | BOOLEAN | То же логическое значение, которое было подано на вход |
