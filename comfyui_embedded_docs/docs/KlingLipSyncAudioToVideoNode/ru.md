Вот перевод документации на русский язык:

Узел Kling Lip Sync Audio to Video синхронизирует движения губ в видеофайле с аудиосодержимым звукового файла. Этот узел анализирует голосовые паттерны в аудио и корректирует движения лица в видео для создания реалистичной синхронизации губ. Для работы требуется как видео с четко различимым лицом, так и аудиофайл с хорошо различимым вокалом.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `video` | VIDEO | Да | - | Видеофайл, содержащий лицо для синхронизации губ |
| `audio` | AUDIO | Да | - | Аудиофайл с вокалом для синхронизации с видео |
| `voice_language` | COMBO | Да | `"en"`<br>`"zh"`<br>`"es"`<br>`"fr"`<br>`"de"`<br>`"it"`<br>`"pt"`<br>`"pl"`<br>`"tr"`<br>`"ru"`<br>`"nl"`<br>`"cs"`<br>`"ar"`<br>`"ja"`<br>`"hu"`<br>`"ko"` | Язык голоса в аудиофайле (по умолчанию: "en") |

**Важные ограничения:**

- Размер аудиофайла не должен превышать 5 МБ
- Размер видеофайла не должен превышать 100 МБ
- Размеры видео должны быть от 720 до 1920 пикселей по высоте/ширине
- Длительность видео должна составлять от 2 до 10 секунд
- Аудио должно содержать четко различимый вокал
- Видео должно содержать отчетливое лицо

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `output` | VIDEO | Обработанное видео с синхронизированными движениями губ |
| `video_id` | STRING | Уникальный идентификатор обработанного видео |
| `duration` | STRING | Длительность обработанного видео |
