Вот перевод документации на русский язык:

Этот узел присоединяет пользовательские хуки к данным conditioning, позволяя перехватывать и изменять процесс conditioning во время выполнения модели. Он принимает набор хуков и применяет их к предоставленным данным conditioning, обеспечивая расширенную настройку рабочего процесса генерации изображений по текстовому описанию. Изменённые данные conditioning с присоединёнными хуками затем возвращаются для использования на последующих этапах обработки.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `conditioning` | CONDITIONING | Да | - | Данные conditioning, к которым будут присоединены хуки |
| `hooks` | HOOKS | Да | - | Определения хуков, которые будут применены к данным conditioning |

## Выходы

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `conditioning` | CONDITIONING | Изменённые данные conditioning с присоединёнными хуками |
