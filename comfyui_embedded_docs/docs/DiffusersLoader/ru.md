Узел DiffusersLoader загружает предварительно обученные модели в формате diffusers. Он выполняет поиск каталогов моделей diffusers, содержащих файл `model_index.json`, и загружает их как компоненты MODEL, CLIP и VAE для использования в конвейере. Этот узел относится к категории устаревших загрузчиков и обеспечивает совместимость с моделями diffusers от Hugging Face.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `model_path` | STRING | Да | Доступно несколько вариантов<br>(автозаполнение из папок diffusers) | Путь к каталогу модели diffusers для загрузки. Узел автоматически сканирует настроенные папки diffusers на наличие валидных моделей и выводит список доступных вариантов. |

## Выходные данные

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `MODEL` | MODEL | Загруженный компонент модели из формата diffusers |
| `CLIP` | CLIP | Загруженный компонент модели CLIP из формата diffusers |
| `VAE` | VAE | Загруженный компонент VAE (вариационного автоэнкодера) из формата diffusers |
