## Обзор

Этот узел генерирует короткое видео из одного исходного изображения с использованием модели HappyHorse. Вы предоставляете изображение первого кадра и текстовый запрос, описывающий желаемое движение и сцену, а узел создает видео, которое продолжается от этого изображения.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-----------|-------------|
| `model` | COMBO | Да | `"happyhorse-1.0-i2v"` | Модель HappyHorse, используемая для генерации видео. |
| `model.prompt` | STRING | Нет | Н/Д | Запрос, описывающий элементы и визуальные особенности. Поддерживает английский и китайский языки. (по умолчанию: "") |
| `model.resolution` | COMBO | Да | `"720P"`<br>`"1080P"` | Разрешение выходного видео. (по умолчанию: "720P") |
| `model.duration` | INT | Да | от 3 до 15 | Длительность генерируемого видео в секундах. (по умолчанию: 5) |
| `first_frame` | IMAGE | Да | Н/Д | Изображение первого кадра. Соотношение сторон выходного видео определяется из этого изображения. |
| `seed` | INT | Нет | от 0 до 2147483647 | Зерно для генерации. (по умолчанию: 0) |
| `watermark` | BOOLEAN | Нет | True / False | Добавлять ли AI-водяной знак к результату. (по умолчанию: False) |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `video` | VIDEO | Сгенерированный видеофайл. |
