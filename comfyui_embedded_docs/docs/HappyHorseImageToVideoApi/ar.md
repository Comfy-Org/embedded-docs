# نظرة عامة

تنتج هذه العقدة مقطع فيديو قصير من صورة بداية واحدة باستخدام نموذج HappyHorse. يمكنك تقديم صورة الإطار الأول ووصف نصي يصف الحركة والمشهد المطلوبين، وتقوم العقدة بإنشاء فيديو يستمر من تلك الصورة.

# المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|-----------|-----------|----------|-------|-------------|
| `النموذج` | COMBO | نعم | `"happyhorse-1.0-i2v"` | نموذج HappyHorse المستخدم لتوليد الفيديو. |
| `model.prompt` | STRING | لا | غير متاح | وصف يصف العناصر والميزات البصرية. يدعم اللغتين الإنجليزية والصينية. (الافتراضي: "") |
| `model.resolution` | COMBO | نعم | `"720P"`<br>`"1080P"` | دقة الفيديو الناتج. (الافتراضي: "720P") |
| `model.duration` | INT | نعم | 3 إلى 15 | مدة الفيديو المُنشأ بالثواني. (الافتراضي: 5) |
| `الإطار الأول` | IMAGE | نعم | غير متاح | صورة الإطار الأول. يُشتق نسبة العرض إلى الارتفاع للمخرجات من هذه الصورة. |
| `البذرة` | INT | لا | 0 إلى 2147483647 | البذرة المستخدمة في التوليد. (الافتراضي: 0) |
| `علامة مائية` | BOOLEAN | لا | صحيح / خطأ | ما إذا كان سيتم إضافة علامة مائية تشير إلى أن المحتوى منشأ بالذكاء الاصطناعي إلى النتيجة. (الافتراضي: خطأ) |

# المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|-------------|-----------|-------------|
| `video` | VIDEO | ملف الفيديو المُنشأ. |
