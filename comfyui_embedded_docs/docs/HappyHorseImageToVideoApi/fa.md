## نمای کلی

این گره با استفاده از مدل HappyHorse، یک ویدئوی کوتاه از یک تصویر شروع واحد تولید می‌کند. شما یک تصویر فریم اول و یک提示 متنی که حرکت و صحنه مورد نظر را توصیف می‌کند ارائه می‌دهید و گره ویدئویی را ایجاد می‌کند که از آن تصویر ادامه می‌یابد.

## ورودی‌ها

| پارامتر | نوع داده | اجباری | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `model` | COMBO | بله | `"happyhorse-1.0-i2v"` | مدل HappyHorse مورد استفاده برای تولید ویدئو. |
| `model.prompt` | STRING | خیر | نامشخص |提示 توصیف‌کننده عناصر و ویژگی‌های بصری. از زبان انگلیسی و چینی پشتیبانی می‌کند. (پیش‌فرض: "") |
| `model.resolution` | COMBO | بله | `"720P"`<br>`"1080P"` | وضوح ویدئوی خروجی. (پیش‌فرض: "720P") |
| `model.duration` | INT | بله | 3 تا 15 | مدت زمان ویدئوی تولید شده بر حسب ثانیه. (پیش‌فرض: 5) |
| `first_frame` | IMAGE | بله | نامشخص | تصویر فریم اول. نسبت ابعاد خروجی از این تصویر استخراج می‌شود. |
| `seed` | INT | خیر | 0 تا 2147483647 | دانه (seed) مورد استفاده برای تولید. (پیش‌فرض: 0) |
| `watermark` | BOOLEAN | خیر | True / False | آیا به نتیجه واترمارک تولید شده با هوش مصنوعی اضافه شود. (پیش‌فرض: False) |
