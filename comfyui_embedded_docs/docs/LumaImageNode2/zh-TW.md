## 概述

此節點使用 Luma UNI-1 模型從文字描述生成圖像。它接收文字提示以及可選的設定（如長寬比和風格），然後將請求發送到 Luma API 以創建圖像。

## 輸入

| 參數 | 資料類型 | 必要 | 範圍 | 說明 |
|-----------|-----------|----------|-------|-------------|
| `prompt` | STRING | 是 | 1–6000 個字元 | 所需圖像的文字描述。 |
| `model` | COMBO | 是 | `"uni-1"`<br>`"uni-1-max"` | 用於生成的模型。選擇模型會顯示該模型的額外設定。 |
| `seed` | INT | 是 | 0 到 2147483647 | 種子碼控制節點是否應重新執行；無論種子碼為何，結果都是非確定性的。（預設值：0） |

### 模型特定輸入

當為 `model` 參數選擇 `"uni-1"` 或 `"uni-1-max"` 時，以下輸入將可用：

| 參數 | 資料類型 | 必要 | 範圍 | 說明 |
|-----------|-----------|----------|-------|-------------|
| `aspect_ratio` | COMBO | 是 | `"auto"`<br>`"3:1"`<br>`"2:1"`<br>`"16:9"`<br>`"3:2"`<br>`"1:1"`<br>`"2:3"`<br>`"9:16"`<br>`"1:2"`<br>`"1:3"` | 輸出圖像的長寬比。`"auto"` 讓模型根據提示自行選擇。（預設值：`"auto"`） |
| `style` | COMBO | 是 | `"auto"`<br>`"manga"` | 生成圖像的視覺風格。（預設值：`"auto"`） |
| `web_search` | BOOLEAN | 是 | True / False | 是否允許模型搜尋網路以獲取更多上下文。（預設值：False） |
| `image_ref` | IMAGE | 否 | 最多 9 張圖像 | 用於引導生成的參考圖像。 |

**關於 `style` 和 `aspect_ratio` 限制的說明：** 如果將 `style` 設定為 `"manga"`，則 `aspect_ratio` 必須是 `"auto"` 或以下直向比例之一：`"2:3"`、`"9:16"`、`"1:2"`、`"1:3"`。在 `"manga"` 風格下使用橫向或正方形比例將會導致錯誤。
