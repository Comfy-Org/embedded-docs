## Обзор

Этот узел генерирует изображения на основе текстовых описаний с использованием модели Luma UNI-1. Он принимает текстовый запрос и необязательные настройки, такие как соотношение сторон и стиль, после чего отправляет запрос к API Luma для создания изображения.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `prompt` | STRING | Да | 1–6000 символов | Текстовое описание желаемого изображения. |
| `model` | COMBO | Да | `"uni-1"`<br>`"uni-1-max"` | Модель для генерации. Выбор модели открывает дополнительные настройки для этой модели. |
| `seed` | INT | Да | от 0 до 2147483647 | Сид определяет, должен ли узел перезапускаться; результаты недетерминированы независимо от сида. (по умолчанию: 0) |

### Входные параметры для конкретной модели

При выборе `"uni-1"` или `"uni-1-max"` для параметра `model` становятся доступны следующие входные параметры:

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `aspect_ratio` | COMBO | Да | `"auto"`<br>`"3:1"`<br>`"2:1"`<br>`"16:9"`<br>`"3:2"`<br>`"1:1"`<br>`"2:3"`<br>`"9:16"`<br>`"1:2"`<br>`"1:3"` | Соотношение сторон выходного изображения. Значение `"auto"` позволяет модели выбрать соотношение на основе запроса. (по умолчанию: `"auto"`) |
| `style` | COMBO | Да | `"auto"`<br>`"manga"` | Визуальный стиль генерируемого изображения. (по умолчанию: `"auto"`) |
| `web_search` | BOOLEAN | Да | True / False | Разрешить модели выполнять поиск в интернете для получения дополнительного контекста. (по умолчанию: False) |
| `image_ref` | IMAGE | Нет | До 9 изображений | Референтные изображения для направления генерации. |

**Примечание об ограничениях `style` и `aspect_ratio`:** Если для параметра `style` установлено значение `"manga"`, то `aspect_ratio` должен быть либо `"auto"`, либо одним из следующих портретных соотношений: `"2:3"`, `"9:16"`, `"1:2"`, `"1:3"`. Использование альбомного или квадратного соотношения сторон со стилем `"manga"` приведет к ошибке.

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `image` | IMAGE | Сгенерированное изображение в виде тензора. |
