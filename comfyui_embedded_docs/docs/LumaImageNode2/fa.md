### مرور کلی

این گره با استفاده از مدل Luma UNI-1، تصاویری را از توضیحات متنی تولید می‌کند. یک متن راهنما و تنظیمات اختیاری مانند نسبت تصویر و سبک دریافت کرده، سپس درخواست را به API Luma ارسال می‌کند تا تصویر ساخته شود.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `prompt` | STRING | بله | ۱ تا ۶۰۰۰ کاراکتر | توضیحات متنی تصویر مورد نظر. |
| `model` | COMBO | بله | `"uni-1"`<br>`"uni-1-max"` | مدل مورد استفاده برای تولید. انتخاب یک مدل، تنظیمات اضافی مربوط به آن مدل را نمایش می‌دهد. |
| `seed` | INT | بله | ۰ تا ۲۱۴۷۴۸۳۶۴۷ | دانه (seed) مشخص می‌کند که آیا گره باید دوباره اجرا شود؛ نتایج صرف‌نظر از دانه، قطعی نیستند. (پیش‌فرض: ۰) |

### ورودی‌های مختص مدل

هنگامی که `"uni-1"` یا `"uni-1-max"` برای پارامتر `model` انتخاب شود، ورودی‌های زیر در دسترس قرار می‌گیرند:

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `aspect_ratio` | COMBO | بله | `"auto"`<br>`"3:1"`<br>`"2:1"`<br>`"16:9"`<br>`"3:2"`<br>`"1:1"`<br>`"2:3"`<br>`"9:16"`<br>`"1:2"`<br>`"1:3"` | نسبت تصویر خروجی. گزینه `"auto"` به مدل اجازه می‌دهد بر اساس متن راهنما انتخاب کند. (پیش‌فرض: `"auto"`) |
| `style` | COMBO | بله | `"auto"`<br>`"manga"` | سبک بصری تصویر تولید شده. (پیش‌فرض: `"auto"`) |
| `web_search` | BOOLEAN | بله | True / False | آیا مدل مجاز به جستجوی وب برای زمینه‌های اضافی است. (پیش‌فرض: False) |
| `image_ref` | IMAGE | خیر | حداکثر ۹ تصویر | تصاویر مرجع برای هدایت تولید. |

**توجه به محدودیت‌های `style` و `aspect_ratio`:** اگر `style` روی `"manga"` تنظیم شود، `aspect_ratio` باید یا `"auto"` باشد یا یکی از نسبت‌های پرتره زیر: `"2:3"`, `"9:16"`, `"1:2"`, `"1:3"`. استفاده از نسبت منظره یا مربع با سبک `"manga"` باعث بروز خطا می‌شود.
