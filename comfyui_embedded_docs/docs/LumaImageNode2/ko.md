# 개요

이 노드는 Luma UNI-1 모델을 사용하여 텍스트 설명으로부터 이미지를 생성합니다. 텍스트 프롬프트와 종횡비, 스타일 등의 선택적 설정을 받아 Luma API에 요청을 전송하여 이미지를 생성합니다.

# 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `prompt` | STRING | 예 | 1–6000자 | 원하는 이미지에 대한 텍스트 설명입니다. |
| `model` | COMBO | 예 | `"uni-1"`<br>`"uni-1-max"` | 생성에 사용할 모델입니다. 모델을 선택하면 해당 모델에 대한 추가 설정이 표시됩니다. |
| `seed` | INT | 예 | 0 ~ 2147483647 | 시드는 노드 재실행 여부를 제어하며, 시드와 관계없이 결과는 비결정적입니다. (기본값: 0) |

## 모델별 입력

`model` 매개변수에서 `"uni-1"` 또는 `"uni-1-max"`를 선택하면 다음 입력을 사용할 수 있습니다:

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `aspect_ratio` | COMBO | 예 | `"auto"`<br>`"3:1"`<br>`"2:1"`<br>`"16:9"`<br>`"3:2"`<br>`"1:1"`<br>`"2:3"`<br>`"9:16"`<br>`"1:2"`<br>`"1:3"` | 출력 이미지의 종횡비입니다. `"auto"`는 프롬프트에 따라 모델이 자동으로 선택합니다. (기본값: `"auto"`) |
| `style` | COMBO | 예 | `"auto"`<br>`"manga"` | 생성된 이미지의 시각적 스타일입니다. (기본값: `"auto"`) |
| `web_search` | BOOLEAN | 예 | True / False | 모델이 추가 컨텍스트를 위해 웹 검색을 허용할지 여부입니다. (기본값: False) |
| `image_ref` | IMAGE | 아니요 | 최대 9개 이미지 | 생성을 안내하는 참조 이미지입니다. |

**`style`과 `aspect_ratio` 제약사항 참고:** `style`을 `"manga"`로 설정한 경우, `aspect_ratio`는 반드시 `"auto"`이거나 다음 세로형 종횡비 중 하나여야 합니다: `"2:3"`, `"9:16"`, `"1:2"`, `"1:3"`. `"manga"` 스타일과 함께 가로형이나 정사각형 종횡비를 사용하면 오류가 발생합니다.

# 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `image` | IMAGE | 텐서 형태로 생성된 이미지입니다. |
