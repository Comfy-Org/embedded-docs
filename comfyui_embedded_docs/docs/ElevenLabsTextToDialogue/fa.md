گره ElevenLabs Text to Dialogue، یک دیالوگ صوتی چند‌گوینده از متن تولید می‌کند. این گره به شما امکان می‌دهد با مشخص کردن خطوط مختلف متن و صداهای مجزا برای هر شرکت‌کننده، یک مکالمه ایجاد کنید. این گره درخواست دیالوگ را به API ElevenLabs ارسال کرده و صدای تولید شده را بازمی‌گرداند.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `stability` | FLOAT | خیر | 0.0 - 1.0 | پایداری صدا. مقادیر پایین‌تر دامنه احساسی گسترده‌تری می‌دهند، مقادیر بالاتر گفتاری یکنواخت‌تر اما بالقوه خسته‌کننده تولید می‌کنند. (پیش‌فرض: 0.5) |
| `apply_text_normalization` | COMBO | خیر | `"auto"`<br>`"on"`<br>`"off"` | حالت نرمال‌سازی متن. 'auto' به سیستم اجازه تصمیم‌گیری می‌دهد، 'on' همیشه نرمال‌سازی را اعمال می‌کند، 'off' از آن صرف‌نظر می‌کند. |
| `model` | COMBO | خیر | `"eleven_v3"` | مدل مورد استفاده برای تولید دیالوگ. |
| `inputs` | DYNAMICCOMBO | بله | `"1"`<br>`"2"`<br>`"3"`<br>`"4"`<br>`"5"`<br>`"6"`<br>`"7"`<br>`"8"`<br>`"9"`<br>`"10"` | تعداد ورودی‌های دیالوگ. انتخاب یک عدد، به همان تعداد فیلد ورودی متن و صدا ایجاد می‌کند. |
| `language_code` | STRING | خیر | - | کد زبان ISO-639-1 یا ISO-639-3 (مثلاً 'en'، 'es'، 'fra'). برای تشخیص خودکار خالی بگذارید. (پیش‌فرض: خالی) |
| `seed` | INT | خیر | 0 - 4294967295 | دانه (seed) برای تکرارپذیری. (پیش‌فرض: 1) |
| `output_format` | COMBO | خیر | `"mp3_44100_192"`<br>`"opus_48000_192"` | فرمت خروجی صوتی. |

**توجه:** پارامتر `inputs` پویا است. وقتی عددی را انتخاب می‌کنید (مثلاً "3")، گره سه فیلد ورودی `text` و `voice` متناظر نمایش می‌دهد (مثلاً `text1`، `voice1`، `text2`، `voice2`، `text3`، `voice3`). هر فیلد `text` باید حداقل یک کاراکتر داشته باشد.
