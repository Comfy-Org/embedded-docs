این گره از مدل Kling AI برای تولید ویدیوی جدید بر اساس یک ویدیوی ورودی و تصاویر مرجع اختیاری استفاده می‌کند. شما یک توضیح متنی از محتوای مورد نظر خود ارائه می‌دهید و گره ویدیوی مرجع را مطابق آن تغییر می‌دهد. همچنین می‌تواند تا چهار تصویر مرجع اضافی برای هدایت سبک و محتوای خروجی دریافت کند.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|----------|----------|--------|--------|----------|
| `model_name` | COMBO | بله | `"kling-v3-omni"`<br>`"kling-video-o1"` | مدل خاص Kling برای تولید ویدیو (پیش‌فرض: "kling-v3-omni"). |
| `prompt` | STRING | بله | نامشخص | توضیح متنی محتوای ویدیو. می‌تواند شامل توصیفات مثبت و منفی باشد. |
| `aspect_ratio` | COMBO | بله | `"16:9"`<br>`"9:16"`<br>`"1:1"` | نسبت ابعاد مورد نظر برای ویدیوی تولیدشده. |
| `duration` | INT | بله | ۳ تا ۱۰ | طول ویدیوی تولیدشده بر حسب ثانیه (پیش‌فرض: ۳). |
| `reference_video` | VIDEO | بله | نامشخص | ویدیویی که به عنوان مرجع استفاده می‌شود. |
| `keep_original_sound` | BOOLEAN | بله | نامشخص | تعیین می‌کند که آیا صدای ویدیوی مرجع در خروجی حفظ شود (پیش‌فرض: True). |
| `reference_images` | IMAGE | خیر | نامشخص | حداکثر ۴ تصویر مرجع اضافی. |
| `resolution` | COMBO | خیر | `"1080p"`<br>`"720p"` | وضوح تصویر برای ویدیوی تولیدشده (پیش‌فرض: "1080p"). |
| `seed` | INT | خیر | ۰ تا ۲۱۴۷۴۸۳۶۴۷ | دانه تصادفی تعیین می‌کند که آیا گره دوباره اجرا شود؛ نتایج صرف‌نظر از دانه تصادفی غیرقطعی هستند (پیش‌فرض: ۰). |

**محدودیت‌های پارامتر:**

* `prompt` باید بین ۱ تا ۲۵۰۰ کاراکتر باشد.
* `reference_video` باید بین ۳.۰ تا ۱۰.۰۵ ثانیه طول داشته باشد.
* ابعاد `reference_video` باید بین ۷۲۰×۷۲۰ تا ۲۱۶۰×۲۱۶۰ پیکسل باشد.
* حداکثر ۴ `reference_images` قابل ارائه است. هر تصویر باید حداقل ۳۰۰×۳۰۰ پیکسل باشد و نسبت ابعاد آن بین ۱:۲.۵ تا ۲.۵:۱ باشد.
