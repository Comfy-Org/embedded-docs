# ترجمة وثيقة عقدة WanMoveTrackToVideo

تقوم عقدة WanMoveTrackToVideo بإعداد بيانات التكييف (conditioning) والفضاء الكامن (latent space) لتوليد الفيديو، مع دمج معلومات تتبع الحركة الاختيارية. تقوم بتشفير تسلسل الصورة الأولية إلى تمثيل كامن، ويمكنها دمج البيانات الموضعية من مسارات الأجسام لتوجيه الحركة في الفيديو المُنشأ. تُخرج العقدة تكييفًا إيجابيًا وسلبيًا معدّلًا بالإضافة إلى موتر كامن فارغ جاهز لنموذج الفيديو.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|---------|--------------|--------|--------|-------|
| `إيجابي` | CONDITIONING | نعم | - | مدخل التكييف الإيجابي المراد تعديله. |
| `سلبي` | CONDITIONING | نعم | - | مدخل التكييف السلبي المراد تعديله. |
| `vae` | VAE | نعم | - | نموذج VAE المستخدم لتشفير الصورة الأولية إلى الفضاء الكامن. |
| `المسارات` | TRACKS | لا | - | بيانات تتبع الحركة الاختيارية التي تحتوي على مسارات الأجسام. |
| `القوة` | FLOAT | لا | 0.0 - 100.0 | قوة تكييف المسار. (القيمة الافتراضية: 1.0) |
| `العرض` | INT | لا | 16 - MAX_RESOLUTION | عرض الفيديو الناتج. يجب أن يكون قابلاً للقسمة على 16. (القيمة الافتراضية: 832) |
| `الارتفاع` | INT | لا | 16 - MAX_RESOLUTION | ارتفاع الفيديو الناتج. يجب أن يكون قابلاً للقسمة على 16. (القيمة الافتراضية: 480) |
| `الطول` | INT | لا | 1 - MAX_RESOLUTION | عدد الإطارات في تسلسل الفيديو. (القيمة الافتراضية: 81) |
| `حجم الدفعة` | INT | لا | 1 - 4096 | حجم الدفعة للمخرجات الكامنة. (القيمة الافتراضية: 1) |
| `صورة البداية` | IMAGE | نعم | - | الصورة الأولية أو تسلسل الصور المراد تشفيرها. |
| `clip_vision_output` | CLIPVISIONOUTPUT | لا | - | مخرجات نموذج CLIP البصري الاختيارية لإضافتها إلى التكييف. |

**ملاحظة:** معامل `strength` له تأثير فقط عند توفير `tracks`. إذا لم يتم توفير `tracks` أو كانت قيمة `strength` تساوي 0.0، فلن يتم تطبيق تكييف المسار. تُستخدم `start_image` لإنشاء صورة كامنة وقناع للتكييف؛ إذا لم يتم توفيرها، تقوم العقدة فقط بتمرير التكييف وإخراج كامن فارغ.

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|------------|--------------|--------|
| `سلبي` | CONDITIONING | التكييف الإيجابي المعدّل، والذي قد يحتوي على `concat_latent_image` و`concat_mask` و`clip_vision_output`. |
| `كامنة` | CONDITIONING | التكييف السلبي المعدّل، والذي قد يحتوي على `concat_latent_image` و`concat_mask` و`clip_vision_output`. |
| `latent` | LATENT | موتر كامن فارغ بأبعاد محددة بواسطة مدخلات `حجم الدفعة` و`الطول` و`الارتفاع` و`العرض`. |
