گره WanMoveTrackToVideo داده‌های شرطی (conditioning) و فضای نهفته (latent space) را برای تولید ویدئو آماده می‌کند و اطلاعات ردیابی حرکت اختیاری را نیز در خود جای می‌دهد. این گره یک دنباله تصویر شروع را به یک نمایش نهفته رمزگذاری می‌کند و می‌تواند داده‌های موقعیتی حاصل از ردیابی اشیا را برای هدایت حرکت در ویدئوی تولیدشده ترکیب کند. خروجی این گره شامل شرطی‌سازی مثبت و منفی اصلاح‌شده و همچنین یک تنسور نهفته خالی است که برای یک مدل ویدئو آماده شده است.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `مثبت` | CONDITIONING | بله | - | ورودی شرطی‌سازی مثبت که باید اصلاح شود. |
| `منفی` | CONDITIONING | بله | - | ورودی شرطی‌سازی منفی که باید اصلاح شود. |
| `vae` | VAE | بله | - | مدل VAE مورد استفاده برای رمزگذاری تصویر شروع به فضای نهفته. |
| `ترک‌ها` | TRACKS | خیر | - | داده‌های اختیاری ردیابی حرکت شامل مسیرهای اشیا. |
| `شدت` | FLOAT | خیر | 0.0 - 100.0 | شدت شرطی‌سازی ردیابی. (پیش‌فرض: 1.0) |
| `عرض` | INT | خیر | 16 - MAX_RESOLUTION | عرض ویدئوی خروجی. باید بر 16 بخش‌پذیر باشد. (پیش‌فرض: 832) |
| `ارتفاع` | INT | خیر | 16 - MAX_RESOLUTION | ارتفاع ویدئوی خروجی. باید بر 16 بخش‌پذیر باشد. (پیش‌فرض: 480) |
| `طول` | INT | خیر | 1 - MAX_RESOLUTION | تعداد فریم‌های دنباله ویدئو. (پیش‌فرض: 81) |
| `اندازه بچ` | INT | خیر | 1 - 4096 | اندازه دسته (batch) برای خروجی نهفته. (پیش‌فرض: 1) |
| `تصویر شروع` | IMAGE | بله | - | تصویر یا دنباله تصویر شروع برای رمزگذاری. |
| `خروجی clip vision` | CLIPVISIONOUTPUT | خیر | - | خروجی اختیاری مدل vision CLIP برای افزودن به شرطی‌سازی. |

**نکته:** پارامتر `strength` تنها زمانی تأثیر دارد که `tracks` ارائه شده باشد. اگر `tracks` ارائه نشود یا `strength` برابر 0.0 باشد، شرطی‌سازی ردیابی اعمال نمی‌شود. از `start_image` برای ایجاد یک تصویر نهفته و ماسک برای شرطی‌سازی استفاده می‌شود؛ اگر ارائه نشود، گره تنها شرطی‌سازی را عبور داده و یک خروجی نهفته خالی تولید می‌کند.

## خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-------------|-----------|-------------|
| `مثبت` | CONDITIONING | شرطی‌سازی مثبت اصلاح‌شده که احتمالاً شامل `concat_latent_image`، `concat_mask` و `خروجی clip vision` است. |
| `منفی` | CONDITIONING | شرطی‌سازی منفی اصلاح‌شده که احتمالاً شامل `concat_latent_image`، `concat_mask` و `خروجی clip vision` است. |
| `latent` | LATENT | یک تنسور نهفته خالی با ابعادی که توسط ورودی‌های `اندازه بچ`، `طول`، `ارتفاع` و `عرض` تعیین می‌شود. |
