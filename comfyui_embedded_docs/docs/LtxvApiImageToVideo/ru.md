Вот перевод документации на русский язык:

Узел LTXV Image To Video создает профессиональное видео из одного исходного изображения. Он использует внешний API для создания видеопоследовательности на основе вашего текстового запроса, позволяя настроить длительность, разрешение и частоту кадров.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `image` | IMAGE | Да | - | Первый кадр, используемый для создания видео. |
| `model` | COMBO | Да | `"LTX-2 (Fast)"`<br>`"LTX-2 (Quality)"` | Модель ИИ для генерации видео. Модель "Fast" оптимизирована для скорости, а модель "Quality" отдает приоритет визуальному качеству. |
| `prompt` | STRING | Да | - | Текстовое описание, которое определяет содержание и движение создаваемого видео. |
| `duration` | COMBO | Да | `6`<br>`8`<br>`10`<br>`12`<br>`14`<br>`16`<br>`18`<br>`20` | Длительность видео в секундах (по умолчанию: 8). |
| `resolution` | COMBO | Да | `"1920x1080"`<br>`"2560x1440"`<br>`"3840x2160"` | Разрешение выходного видео. |
| `fps` | COMBO | Да | `25`<br>`50` | Количество кадров в секунду для видео (по умолчанию: 25). |
| `generate_audio` | BOOLEAN | Нет | - | Если true, созданное видео будет включать сгенерированный ИИ звук, соответствующий сцене (по умолчанию: False). |

**Важные ограничения:**

* Входной параметр `image` должен содержать ровно одно изображение.
* Длина `prompt` должна составлять от 1 до 10 000 символов.
* Если вы выбираете `duration` более 10 секунд, необходимо использовать модель **"LTX-2 (Fast)"**, разрешение **"1920x1080"** и **25** FPS. Эта комбинация обязательна для более длинных видео.

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `video` | VIDEO | Созданный видеофайл. |
