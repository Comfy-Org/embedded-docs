Вот перевод документации на русский язык:

Модифицирует изображения синхронно на основе текстового запроса и соотношения сторон исходного изображения. Этот узел принимает входное изображение и преобразует его в соответствии с заданным запросом, используя настраиваемый вес изображения для контроля степени изменения оригинала.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `image` | IMAGE | Да | - | Входное изображение для модификации |
| `prompt` | STRING | Да | - | Запрос для генерации изображения (по умолчанию: "") |
| `image_weight` | FLOAT | Нет | 0.0-0.98 | Вес изображения; чем ближе к 1.0, тем меньше изображение будет изменено (по умолчанию: 0.1). Внутренне это значение инвертируется (1.0 - image_weight) и ограничивается диапазоном от 0.0 до 0.98. |
| `model` | STRING | Да | `"photon-flash-1"`<br>`"photon-1"`<br>`"photon"` | Модель Luma для модификации изображения. Разные модели имеют разную стоимость. |
| `seed` | INT | Нет | 0-18446744073709551615 | Сид для определения необходимости повторного запуска узла; фактические результаты недетерминированы независимо от сида (по умолчанию: 0) |

## Выходные данные

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `image` | IMAGE | Модифицированное изображение, сгенерированное моделью Luma |
