# ترجمة وثيقة عقدة ComfyUI

## نظرة عامة

تعديل الصور بشكل متزامن بناءً على نص موجه ونسبة العرض إلى الارتفاع للصورة الأصلية. تأخذ هذه العقدة صورة إدخال وتحولها وفقًا للنص الموجه المقدم، باستخدام وزن صورة قابل للتعديل للتحكم في مدى تغيير الصورة الأصلية.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|---------|---------------|--------|---------|-------|
| `صورة` | IMAGE | نعم | - | صورة الإدخال المراد تعديلها |
| `النص المطلوب` | STRING | نعم | - | النص الموجه لتوليد الصورة (الافتراضي: "") |
| `وزن الصورة` | FLOAT | لا | 0.0-0.98 | وزن الصورة؛ كلما اقتربت القيمة من 1.0، قل تعديل الصورة (الافتراضي: 0.1). داخليًا، يتم عكس هذه القيمة (1.0 - image_weight) وتقييدها بين 0.0 و 0.98. |
| `نموذج` | STRING | نعم | `"photon-flash-1"`<br>`"photon-1"`<br>`"photon"` | نموذج Luma المستخدم لتعديل الصورة. النماذج المختلفة لها تكاليف مختلفة. |
| `البذرة` | INT | لا | 0-18446744073709551615 | البذرة لتحديد ما إذا كانت العقدة ستعيد التشغيل؛ النتائج الفعلية غير حتمية بغض النظر عن البذرة (الافتراضي: 0) |
