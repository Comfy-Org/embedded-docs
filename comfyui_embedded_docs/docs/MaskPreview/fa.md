گره MaskPreview داده‌های ماسک را به‌عنوان یک تصویر پیش‌نمایش در دایرکتوری خروجی ComfyUI شما ذخیره می‌کند و به شما امکان می‌دهد تا در حین اجرای گردش کار، داده‌های ماسک را به‌صورت بصری بررسی کنید. این گره، ماسک ورودی را به قالبی مناسب برای نمایش تصویر تبدیل کرده و با یک پیشوند نام فایل قابل تنظیم ذخیره می‌کند.

## ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|---------|----------|-------|--------|---------|
| `ماسک` | MASK | بله | - | داده‌های ماسک که باید پیش‌نمایش شده و به‌عنوان تصویر ذخیره شوند |
| `filename_prefix` | STRING | خیر | - | پیشوند نام فایل خروجی (پیش‌فرض: "ComfyUI") |
| `prompt` | PROMPT | خیر | - | اطلاعات prompt برای فراداده (به‌صورت خودکار ارائه می‌شود) |
| `extra_pnginfo` | EXTRA_PNGINFO | خیر | - | اطلاعات اضافی PNG برای فراداده (به‌صورت خودکار ارائه می‌شود) |

## خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-----------|----------|---------|
| `ui` | DICT | شامل اطلاعات تصویر پیش‌نمایش و فراداده برای نمایش در رابط کاربری است |
