Вот перевод документации на русский язык:

Узел MaskPreview сохраняет данные маски в виде изображения для предпросмотра в выходной каталог ComfyUI, позволяя визуально проверять данные маски во время выполнения рабочего процесса. Он преобразует входную маску в формат, подходящий для отображения изображения, и сохраняет её с настраиваемым префиксом имени файла.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `mask` | MASK | Да | - | Данные маски для предпросмотра и сохранения в виде изображения |
| `filename_prefix` | STRING | Нет | - | Префикс для имени выходного файла (по умолчанию: "ComfyUI") |
| `prompt` | PROMPT | Нет | - | Информация о запросе для метаданных (предоставляется автоматически) |
| `extra_pnginfo` | EXTRA_PNGINFO | Нет | - | Дополнительная информация PNG для метаданных (предоставляется автоматически) |

## Выходные параметры

| Имя выходного параметра | Тип данных | Описание |
|-------------|-----------|-------------|
| `ui` | DICT | Содержит информацию об изображении для предпросмотра и метаданные для отображения в интерфейсе |
