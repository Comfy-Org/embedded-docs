تم إنشاء هذه الوثائق بواسطة الذكاء الاصطناعي. إذا وجدت أي أخطاء أو لديك اقتراحات للتحسين، فلا تتردد في المساهمة! [عدّل على GitHub](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/MaskPreview/en.md)

تقوم عقدة MaskPreview بحفظ بيانات القناع كصورة معاينة إلى دليل الإخراج الخاص بـ ComfyUI، مما يتيح لك فحص بيانات القناع بصريًا أثناء تنفيذ سير العمل. تقوم بتحويل القناع المدخل إلى تنسيق مناسب لعرض الصورة وحفظه مع بادئة اسم ملف قابلة للتخصيص.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|-----------|-----------|----------|-------|-------------|
| `قناع` | MASK | نعم | - | بيانات القناع المراد معاينتها وحفظها كصورة |
| `filename_prefix` | STRING | لا | - | بادئة اسم ملف الإخراج (الافتراضي: "ComfyUI") |
| `prompt` | PROMPT | لا | - | معلومات الموجه للبيانات الوصفية (يتم توفيرها تلقائيًا) |
| `extra_pnginfo` | EXTRA_PNGINFO | لا | - | معلومات PNG إضافية للبيانات الوصفية (يتم توفيرها تلقائيًا) |

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|-------------|-----------|-------------|
| `ui` | DICT | يحتوي على معلومات صورة المعاينة والبيانات الوصفية لعرضها في واجهة المستخدم |
