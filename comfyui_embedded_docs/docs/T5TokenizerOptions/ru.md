Узел T5TokenizerOptions позволяет настроить параметры токенизатора для различных типов моделей T5. Он задает минимальное дополнение (padding) и минимальную длину для нескольких вариантов моделей T5, включая t5xxl, pile_t5xl, t5base, mt5xl и umt5xxl. Узел принимает на вход CLIP и возвращает модифицированный CLIP с примененными указанными параметрами токенизатора.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `clip` | CLIP | Да | - | Модель CLIP, для которой настраиваются параметры токенизатора |
| `минимальное_дополнение` | INT | Нет | от 0 до 10000 | Минимальное значение дополнения для всех типов моделей T5 (по умолчанию: 0) |
| `минимальная_длина` | INT | Нет | от 0 до 10000 | Минимальное значение длины для всех типов моделей T5 (по умолчанию: 0) |

## Выходные данные

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `output` | CLIP | Модифицированная модель CLIP с обновленными параметрами токенизатора, примененными ко всем вариантам T5 |
