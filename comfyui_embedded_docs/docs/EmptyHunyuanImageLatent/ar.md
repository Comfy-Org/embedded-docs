# ترجمة وثيقة عقدة EmptyHunyuanImageLatent

تقوم عقدة EmptyHunyuanImageLatent بإنشاء موتر كامن فارغ بأبعاد محددة لاستخدامها مع نماذج توليد الصور Hunyuan. تُنشئ هذه العقدة نقطة بداية فارغة يمكن معالجتها عبر العقد اللاحقة في سير العمل. تتيح لك العقدة تحديد العرض والارتفاع وحجم الدفعة للفضاء الكامن.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|---------|---------------|--------|---------|-------|
| `العرض` | INT | نعم | 64 إلى MAX_RESOLUTION | عرض الصورة الكامنة المُنشأة بالبكسل (الافتراضي: 2048، الخطوة: 32) |
| `الارتفاع` | INT | نعم | 64 إلى MAX_RESOLUTION | ارتفاع الصورة الكامنة المُنشأة بالبكسل (الافتراضي: 2048، الخطوة: 32) |
| `حجم_الدفعة` | INT | نعم | 1 إلى 4096 | عدد العينات الكامنة المطلوب توليدها في الدفعة الواحدة (الافتراضي: 1) |
