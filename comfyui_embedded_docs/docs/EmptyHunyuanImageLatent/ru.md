Узел EmptyHunyuanImageLatent создает пустой латентный тензор с заданными размерами для использования с моделями генерации изображений Hunyuan. Он генерирует пустую начальную точку, которая может быть обработана последующими узлами в рабочем процессе. Узел позволяет указать ширину, высоту и размер пакета латентного пространства.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `width` | INT | Да | от 64 до MAX_RESOLUTION | Ширина генерируемого латентного изображения в пикселях (по умолчанию: 2048, шаг: 32) |
| `height` | INT | Да | от 64 до MAX_RESOLUTION | Высота генерируемого латентного изображения в пикселях (по умолчанию: 2048, шаг: 32) |
| `batch_size` | INT | Да | от 1 до 4096 | Количество латентных образцов для генерации в одном пакете (по умолчанию: 1) |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `LATENT` | LATENT | Пустой латентный тензор с заданными размерами для обработки изображений Hunyuan |
