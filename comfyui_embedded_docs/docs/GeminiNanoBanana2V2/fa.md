## نمای کلی

این گره با ارسال یک درخواست متنی به API Vertex AI گوگل، تصاویر را تولید یا ویرایش می‌کند. از یک مدل خاص Gemini برای ایجاد تصاویر جدید یا تغییر تصاویر موجود بر اساس دستورالعمل‌های شما استفاده می‌کند.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `prompt` | STRING | بله | N/A | درخواست متنی که تصویر مورد نظر برای تولید یا ویرایش‌های مورد اعمال را توصیف می‌کند. هرگونه محدودیت، سبک یا جزئیاتی که مدل باید رعایت کند را شامل شود. |
| `model` | COMBO | بله | `"Nano Banana 2 (Gemini 3.1 Flash Image)"` | مدل Gemini مورد استفاده برای تولید تصویر را انتخاب می‌کند. در حال حاضر فقط یک گزینه موجود است. |
| `seed` | INT | بله | 0 تا 18446744073709551615 | هنگامی که seed روی یک مقدار مشخص ثابت می‌شود، مدل نهایت تلاش خود را برای ارائه پاسخ یکسان در درخواست‌های تکراری به کار می‌گیرد. خروجی قطعی تضمین نمی‌شود. همچنین تغییر مدل یا تنظیمات پارامترهایی مانند دما (temperature) می‌تواند باعث تغییر در پاسخ حتی با استفاده از مقدار seed یکسان شود. به‌طور پیش‌فرض از یک مقدار seed تصادفی استفاده می‌شود. (پیش‌فرض: 42) |
| `response_modalities` | COMBO | بله | `"IMAGE"`<br>`"IMAGE+TEXT"` | قالب پاسخ را تعیین می‌کند. گزینه "IMAGE" را برای دریافت فقط تصویر، یا "IMAGE+TEXT" را برای دریافت هم تصویر و هم توضیح متنی انتخاب کنید. (پیش‌فرض: "IMAGE") |
| `system_prompt` | STRING | خیر | N/A | دستورالعمل‌های بنیادینی که رفتار هوش مصنوعی را دیکته می‌کنند. این یک پارامتر پیشرفته است. |

**نکته درباره پارامتر `model`:** پارامتر `model` یک گزینه ترکیبی پویا است که شامل زیرپارامترهای اضافی برای وضوح، نسبت تصویر و سطح تفکر می‌شود. این زیرپارامترها درون انتخاب مدل تعریف شده‌اند و به‌عنوان ورودی‌های جداگانه در این جدول فهرست نشده‌اند.

**نکته درباره ورودی تصویر:** می‌توانید حداکثر ۱۴ تصویر را به‌عنوان ورودی به مدل ارائه دهید. این تصاویر از طریق زیرفیلد تصویر پارامتر `model` منتقل می‌شوند و برای ویرایش یا به‌عنوان زمینه بصری برای تولید استفاده می‌شوند.
