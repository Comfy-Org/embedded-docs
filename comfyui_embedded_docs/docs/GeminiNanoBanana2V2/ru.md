## Обзор

Этот узел создаёт или редактирует изображения, отправляя текстовый запрос в API Google Vertex AI. Он использует определённую модель Gemini для создания новых изображений или изменения существующих на основе ваших инструкций.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `prompt` | STRING | Да | Н/Д | Текстовый запрос, описывающий изображение для создания или изменения. Включает любые ограничения, стили или детали, которым должна следовать модель. |
| `model` | COMBO | Да | `"Nano Banana 2 (Gemini 3.1 Flash Image)"` | Выбирает модель Gemini для генерации изображений. В настоящее время доступен только один вариант. |
| `seed` | INT | Да | от 0 до 18446744073709551615 | Когда значение seed фиксировано, модель прилагает максимальные усилия для предоставления одинакового ответа на повторные запросы. Детерминированный вывод не гарантируется. Кроме того, изменение модели или настроек параметров, таких как температура, может вызвать изменения в ответе даже при использовании одного и того же значения seed. По умолчанию используется случайное значение seed. (по умолчанию: 42) |
| `response_modalities` | COMBO | Да | `"IMAGE"`<br>`"IMAGE+TEXT"` | Определяет формат ответа. Выберите "IMAGE", чтобы получить только изображение, или "IMAGE+TEXT", чтобы получить как изображение, так и текстовое описание. (по умолчанию: "IMAGE") |
| `system_prompt` | STRING | Нет | Н/Д | Базовые инструкции, определяющие поведение ИИ. Это расширенный параметр. |

**Примечание о параметре `model`:** Параметр `model` представляет собой динамический комбинированный список, который включает дополнительные подпараметры для разрешения, соотношения сторон и уровня мышления. Эти подпараметры определены в рамках выбора модели и не перечислены как отдельные входные данные в этой таблице.

## Выходные данные

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `IMAGE` | IMAGE | Сгенерированное или отредактированное изображение. |
| `STRING` | STRING | Текстовое описание или подпись, сгенерированные моделью. |
| `thought_image` | IMAGE | Первое изображение из процесса мышления модели. Доступно только при уровне мышления HIGH и модальности IMAGE+TEXT. |
