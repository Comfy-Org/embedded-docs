Вот перевод документации на русский язык:

Узел Topaz Image Enhance обеспечивает апскейлинг и улучшение изображения на профессиональном уровне. Он обрабатывает одно входное изображение с помощью облачной модели ИИ для повышения качества, детализации и разрешения. Узел предоставляет тонкую настройку процесса улучшения, включая опции для творческого управления, фокусировки на объекте и сохранения лиц.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `модель` | COMBO | Да | `"Reimagine"` | Модель ИИ, используемая для улучшения изображения. |
| `изображение` | IMAGE | Да | - | Входное изображение для обработки. Поддерживается только одно изображение. |
| `промпт` | STRING | Нет | - | Необязательный текстовый запрос для творческого управления апскейлингом (по умолчанию: пусто). |
| `обнаружение объекта` | COMBO | Нет | `"All"`<br>`"Foreground"`<br>`"Background"` | Определяет, какая часть изображения будет в фокусе улучшения (по умолчанию: "All"). |
| `улучшение лица` | BOOLEAN | Нет | - | Включить улучшение лиц, если они присутствуют на изображении (по умолчанию: True). |
| `креативность улучшения лица` | FLOAT | Нет | 0.0 - 1.0 | Задает уровень творчества при улучшении лиц (по умолчанию: 0.0). |
| `степень улучшения лица` | FLOAT | Нет | 0.0 - 1.0 | Управляет резкостью улучшенных лиц относительно фона (по умолчанию: 1.0). |
| `обрезать до заполнения` | BOOLEAN | Нет | - | По умолчанию изображение дополняется полями, если соотношение сторон вывода отличается. Включите, чтобы вместо этого обрезать изображение для заполнения выходных размеров (по умолчанию: False). |
| `выходная ширина` | INT | Нет | 0 - 32000 | Желаемая ширина выходного изображения. Значение 0 означает автоматический расчет, обычно на основе исходного размера или `выходная высота`, если указано (по умолчанию: 0). |
| `выходная высота` | INT | Нет | 0 - 32000 | Желаемая высота выходного изображения. Значение 0 означает автоматический расчет, обычно на основе исходного размера или `выходная ширина`, если указано (по умолчанию: 0). |
| `креативность` | INT | Нет | 1 - 9 | Управляет общим уровнем творчества при улучшении (по умолчанию: 3). |
| `сохранение лица` | BOOLEAN | Нет | - | Сохранять идентичность лиц объектов на изображении (по умолчанию: True). |
| `сохранение цвета` | BOOLEAN | Нет | - | Сохранять исходные цвета входного изображения (по умолчанию: True). |

**Примечание:** Этот узел может обрабатывать только одно входное изображение. Передача пакета из нескольких изображений приведет к ошибке.

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `изображение` | IMAGE | Улучшенное выходное изображение. |
