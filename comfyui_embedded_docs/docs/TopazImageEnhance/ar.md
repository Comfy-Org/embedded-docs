عقدة تحسين الصور من Topaz توفر تحسينًا احترافيًا للصور ورفع دقتها. تقوم بمعالجة صورة إدخال واحدة باستخدام نموذج ذكاء اصطناعي قائم على السحابة لتحسين الجودة والتفاصيل والدقة. تمنح العقدة تحكمًا دقيقًا في عملية التحسين، بما في ذلك خيارات التوجيه الإبداعي والتركيز على الموضوع والحفاظ على الوجوه.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|---------|--------------|--------|--------|-------|
| `model` | COMBO | نعم | `"Reimagine"` | نموذج الذكاء الاصطناعي المستخدم لتحسين الصورة. |
| `image` | IMAGE | نعم | - | صورة الإدخال المراد تحسينها. يتم دعم صورة واحدة فقط. |
| `prompt` | STRING | لا | - | نص وصفي اختياري لتوجيه تحسين الصورة بشكل إبداعي (الافتراضي: فارغ). |
| `subject_detection` | COMBO | لا | `"All"`<br>`"Foreground"`<br>`"Background"` | يتحكم في أي جزء من الصورة يركز عليه التحسين (الافتراضي: "All"). |
| `face_enhancement` | BOOLEAN | لا | - | تفعيل لتحسين الوجوه إذا كانت موجودة في الصورة (الافتراضي: True). |
| `face_enhancement_creativity` | FLOAT | لا | 0.0 - 1.0 | يحدد مستوى الإبداع لتحسين الوجوه (الافتراضي: 0.0). |
| `face_enhancement_strength` | FLOAT | لا | 0.0 - 1.0 | يتحكم في مدى وضوح الوجوه المحسنة مقارنة بالخلفية (الافتراضي: 1.0). |
| `crop_to_fill` | BOOLEAN | لا | - | افتراضيًا، يتم إضافة أشرطة سوداء جانبية عند اختلاف نسبة أبعاد الإخراج. قم بالتفعيل لقص الصورة لملء أبعاد الإخراج بدلاً من ذلك (الافتراضي: False). |
| `output_width` | INT | لا | 0 - 32000 | العرض المطلوب لصورة الإخراج. القيمة 0 تعني الحساب التلقائي، عادةً بناءً على الحجم الأصلي أو `output_height` إذا تم تحديده (الافتراضي: 0). |
| `output_height` | INT | لا | 0 - 32000 | الارتفاع المطلوب لصورة الإخراج. القيمة 0 تعني الحساب التلقائي، عادةً بناءً على الحجم الأصلي أو `output_width` إذا تم تحديده (الافتراضي: 0). |
| `creativity` | INT | لا | 1 - 9 | يتحكم في المستوى العام للإبداع في التحسين (الافتراضي: 3). |
| `face_preservation` | BOOLEAN | لا | - | الحفاظ على هوية الوجوه للأشخاص في الصورة (الافتراضي: True). |
| `color_preservation` | BOOLEAN | لا | - | الحفاظ على الألوان الأصلية لصورة الإدخال (الافتراضي: True). |

**ملاحظة:** يمكن لهذه العقدة معالجة صورة إدخال واحدة فقط. سيؤدي توفير مجموعة من عدة صور إلى حدوث خطأ.
