عقدة تحسين الصور من Topaz توفر تحسينًا احترافيًا للصور ورفع دقتها. تقوم بمعالجة صورة إدخال واحدة باستخدام نموذج ذكاء اصطناعي قائم على السحابة لتحسين الجودة والتفاصيل والدقة. تمنح العقدة تحكمًا دقيقًا في عملية التحسين، بما في ذلك خيارات التوجيه الإبداعي والتركيز على الموضوع والحفاظ على الوجوه.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|---------|--------------|--------|--------|-------|
| `النموذج` | COMBO | نعم | `"Reimagine"` | نموذج الذكاء الاصطناعي المستخدم لتحسين الصورة. |
| `الصورة` | IMAGE | نعم | - | صورة الإدخال المراد تحسينها. يتم دعم صورة واحدة فقط. |
| `الموجه` | STRING | لا | - | نص وصفي اختياري لتوجيه تحسين الصورة بشكل إبداعي (الافتراضي: فارغ). |
| `اكتشاف الموضوع` | COMBO | لا | `"All"`<br>`"Foreground"`<br>`"Background"` | يتحكم في أي جزء من الصورة يركز عليه التحسين (الافتراضي: "All"). |
| `تحسين الوجه` | BOOLEAN | لا | - | تفعيل لتحسين الوجوه إذا كانت موجودة في الصورة (الافتراضي: True). |
| `إبداعية تحسين الوجه` | FLOAT | لا | 0.0 - 1.0 | يحدد مستوى الإبداع لتحسين الوجوه (الافتراضي: 0.0). |
| `قوة تحسين الوجه` | FLOAT | لا | 0.0 - 1.0 | يتحكم في مدى وضوح الوجوه المحسنة مقارنة بالخلفية (الافتراضي: 1.0). |
| `قص لملء` | BOOLEAN | لا | - | افتراضيًا، يتم إضافة أشرطة سوداء جانبية عند اختلاف نسبة أبعاد الإخراج. قم بالتفعيل لقص الصورة لملء أبعاد الإخراج بدلاً من ذلك (الافتراضي: False). |
| `عرض الإخراج` | INT | لا | 0 - 32000 | العرض المطلوب لصورة الإخراج. القيمة 0 تعني الحساب التلقائي، عادةً بناءً على الحجم الأصلي أو `ارتفاع الإخراج` إذا تم تحديده (الافتراضي: 0). |
| `ارتفاع الإخراج` | INT | لا | 0 - 32000 | الارتفاع المطلوب لصورة الإخراج. القيمة 0 تعني الحساب التلقائي، عادةً بناءً على الحجم الأصلي أو `عرض الإخراج` إذا تم تحديده (الافتراضي: 0). |
| `الإبداع` | INT | لا | 1 - 9 | يتحكم في المستوى العام للإبداع في التحسين (الافتراضي: 3). |
| `حفظ الوجه` | BOOLEAN | لا | - | الحفاظ على هوية الوجوه للأشخاص في الصورة (الافتراضي: True). |
| `حفظ الألوان` | BOOLEAN | لا | - | الحفاظ على الألوان الأصلية لصورة الإدخال (الافتراضي: True). |

**ملاحظة:** يمكن لهذه العقدة معالجة صورة إدخال واحدة فقط. سيؤدي توفير مجموعة من عدة صور إلى حدوث خطأ.
