گره ارتقای تصویر Topaz، قابلیت افزایش مقیاس و بهبود تصویر در سطح صنعتی را فراهم می‌کند. این گره با استفاده از یک مدل هوش مصنوعی مبتنی بر ابر، یک تصویر ورودی واحد را پردازش کرده و کیفیت، جزئیات و وضوح آن را بهبود می‌بخشد. این گره کنترل دقیقی بر فرآیند بهبود ارائه می‌دهد که شامل گزینه‌هایی برای هدایت خلاقانه، تمرکز بر سوژه و حفظ چهره است.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|---------|----------|--------|--------|---------|
| `مدل` | COMBO | بله | `"Reimagine"` | مدل هوش مصنوعی مورد استفاده برای بهبود تصویر. |
| `تصویر` | IMAGE | بله | - | تصویر ورودی که قرار است بهبود یابد. تنها یک تصویر پشتیبانی می‌شود. |
| `پرامپت` | STRING | خیر | - | یک متن راهنمای اختیاری برای هدایت خلاقانه افزایش مقیاس (پیش‌فرض: خالی). |
| `شناسایی سوژه` | COMBO | خیر | `"All"`<br>`"Foreground"`<br>`"Background"` | مشخص می‌کد که بهبود روی کدام بخش از تصویر متمرکز شود (پیش‌فرض: "All"). |
| `بهبود چهره` | BOOLEAN | خیر | - | در صورت وجود چهره در تصویر، آن را فعال کنید تا چهره‌ها بهبود یابند (پیش‌فرض: True). |
| `خلاقیت در بهبود چهره` | FLOAT | خیر | 0.0 - 1.0 | سطح خلاقیت را برای بهبود چهره تنظیم می‌کند (پیش‌فرض: 0.0). |
| `قدرت بهبود چهره` | FLOAT | خیر | 0.0 - 1.0 | میزان وضوح چهره‌های بهبودیافته را نسبت به پس‌زمینه کنترل می‌کند (پیش‌فرض: 1.0). |
| `برش برای پر کردن` | BOOLEAN | خیر | - | به‌طور پیش‌فرض، زمانی که نسبت ابعاد خروجی متفاوت باشد، تصویر به‌صورت جعبه‌ای (letterbox) نمایش داده می‌شود. برای برش تصویر به‌منظور پر کردن ابعاد خروجی، این گزینه را فعال کنید (پیش‌فرض: False). |
| `عرض خروجی` | INT | خیر | 0 - 32000 | عرض مورد نظر تصویر خروجی. مقدار 0 به معنای محاسبه خودکار است که معمولاً بر اساس اندازه اصلی یا `ارتفاع خروجی` (در صورت مشخص بودن) انجام می‌شود (پیش‌فرض: 0). |
| `ارتفاع خروجی` | INT | خیر | 0 - 32000 | ارتفاع مورد نظر تصویر خروجی. مقدار 0 به معنای محاسبه خودکار است که معمولاً بر اساس اندازه اصلی یا `عرض خروجی` (در صورت مشخص بودن) انجام می‌شود (پیش‌فرض: 0). |
| `خلاقیت` | INT | خیر | 1 - 9 | سطح کلی خلاقیت در فرآیند بهبود را کنترل می‌کند (پیش‌فرض: 3). |
| `حفظ چهره` | BOOLEAN | خیر | - | هویت چهره سوژه‌ها را در تصویر حفظ می‌کند (پیش‌فرض: True). |
| `حفظ رنگ` | BOOLEAN | خیر | - | رنگ‌های اصلی تصویر ورودی را حفظ می‌کند (پیش‌فرض: True). |

**توجه:** این گره تنها می‌تواند یک تصویر ورودی واحد را پردازش کند. ارائه یک دسته از چندین تصویر منجر به خطا خواهد شد.
