### GeminiImage2Node

گره GeminiImage2Node با استفاده از مدل Vertex AI Gemini گوگل، تصاویر را تولید یا ویرایش می‌کند. این گره یک متن راهنما (prompt) و تصاویر یا فایل‌های مرجع اختیاری را به API ارسال کرده و تصویر تولیدشده و/یا توضیح متنی را بازمی‌گرداند.

## ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|---------|----------|-------|--------|---------|
| `prompt` | STRING | بله | نامشخص | متن راهنما که تصویر مورد نظر برای تولید یا ویرایش‌های اعمال‌شده را توصیف می‌کند. هرگونه محدودیت، سبک یا جزئیاتی که مدل باید رعایت کند را شامل شود. |
| `model` | COMBO | بله | `"gemini-3-pro-image-preview"`<br>`"Nano Banana 2 (Gemini 3.1 Flash Image)"` | مدل Gemini خاص برای تولید. گزینه "Nano Banana 2" به صورت داخلی به مدل `gemini-3.1-flash-image-preview` نگاشت می‌شود. |
| `seed` | INT | بله | 0 تا 18446744073709551615 | هنگامی که روی یک مقدار مشخص ثابت شود، مدل نهایت تلاش خود را برای ارائه پاسخ یکسان در درخواست‌های تکراری انجام می‌دهد. خروجی قطعی تضمین نمی‌شود. تغییر مدل یا تنظیمات دیگر حتی با seed یکسان می‌تواند باعث تغییرات شود. پیش‌فرض: 42. |
| `aspect_ratio` | COMBO | بله | `"auto"`<br>`"1:1"`<br>`"2:3"`<br>`"3:2"`<br>`"3:4"`<br>`"4:3"`<br>`"4:5"`<br>`"5:4"`<br>`"9:16"`<br>`"16:9"`<br>`"21:9"` | نسبت ابعاد مورد نظر برای تصویر خروجی. اگر روی 'auto' تنظیم شود، با نسبت ابعاد تصویر ورودی شما مطابقت دارد؛ اگر تصویری ارائه نشود، معمولاً یک مربع 16:9 تولید می‌شود. پیش‌فرض: "auto". |
| `resolution` | COMBO | بله | `"1K"`<br>`"2K"`<br>`"4K"` | وضوح هدف خروجی. برای 2K/4K از مقیاس‌کننده بومی Gemini استفاده می‌شود. |
| `response_modalities` | COMBO | بله | `"IMAGE+TEXT"`<br>`"IMAGE"` | گزینه 'IMAGE' را برای خروجی فقط تصویر، یا 'IMAGE+TEXT' را برای بازگرداندن هم تصویر تولیدشده و هم پاسخ متنی انتخاب کنید. |
| `images` | IMAGE | خیر | نامشخص | تصویر(های) مرجع اختیاری. برای گنجاندن چندین تصویر، از گره Batch Images (حداکثر 14 تصویر) استفاده کنید. |
| `files` | CUSTOM | خیر | نامشخص | فایل(های) اختیاری برای استفاده به عنوان زمینه برای مدل. ورودی‌های گره Gemini Generate Content Input Files را می‌پذیرد. |
| `system_prompt` | STRING | خیر | نامشخص | دستورالعمل‌های بنیادی که رفتار هوش مصنوعی را تعیین می‌کند. پیش‌فرض: یک system prompt از پیش تعریف‌شده برای تولید تصویر. |

**محدودیت‌ها:**

* ورودی `images` حداکثر از 14 تصویر پشتیبانی می‌کند. اگر تعداد بیشتری ارائه شود، خطا ایجاد می‌شود.
* ورودی `files` باید به گره‌ای متصل شود که نوع داده `GEMINI_INPUT_FILES` را خروجی می‌دهد.

## خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-----------|----------|---------|
| `image` | IMAGE | تصویر تولید یا ویرایش‌شده توسط مدل Gemini. |
| `string` | STRING | پاسخ متنی از مدل. اگر `response_modalities` روی "IMAGE" تنظیم شده باشد، این خروجی خالی خواهد بود. |
