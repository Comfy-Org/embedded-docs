بالتأكيد، إليك الترجمة العربية للوثيقة التقنية الخاصة بالعقدة `GeminiImage2Node` مع الالتزام التام بقواعد الترجمة المحددة:

يقوم عقدة GeminiImage2Node بإنشاء أو تحرير الصور باستخدام نموذج Gemini من Google عبر Vertex AI. يرسل العقدة مطالبة نصية وصورًا أو ملفات مرجعية اختيارية إلى واجهة برمجة التطبيقات (API) ويعيد الصورة المُنشأة و/أو وصفًا نصيًا.

## المدخلات

| المعامل | نوع البيانات | إجباري | النطاق | الوصف |
|-----------|-----------|----------|-------|-------------|
| `prompt` | STRING | نعم | غير متاح | مطالبة نصية تصف الصورة المراد إنشاؤها أو التعديلات المراد تطبيقها. قم بتضمين أي قيود أو أنماط أو تفاصيل يجب أن يتبعها النموذج. |
| `model` | COMBO | نعم | `"gemini-3-pro-image-preview"`<br>`"Nano Banana 2 (Gemini 3.1 Flash Image)"` | نموذج Gemini المحدد لاستخدامه في عملية الإنشاء. خيار "Nano Banana 2" يوجه داخليًا إلى النموذج `gemini-3.1-flash-image-preview`. |
| `seed` | INT | نعم | 0 إلى 18446744073709551615 | عند تثبيته على قيمة محددة، يبذل النموذج قصارى جهده لتقديم نفس الاستجابة للطلبات المتكررة. لا يُضمن إخراج حتمي. قد يؤدي تغيير النموذج أو الإعدادات الأخرى إلى حدوث اختلافات حتى مع استخدام نفس البذرة. القيمة الافتراضية: 42. |
| `aspect_ratio` | COMBO | نعم | `"auto"`<br>`"1:1"`<br>`"2:3"`<br>`"3:2"`<br>`"3:4"`<br>`"4:3"`<br>`"4:5"`<br>`"5:4"`<br>`"9:16"`<br>`"16:9"`<br>`"21:9"` | نسبة العرض إلى الارتفاع المطلوبة للصورة الناتجة. إذا تم تعيينها على 'auto'، فإنها تطابق نسبة العرض إلى الارتفاع لصورتك المدخلة؛ إذا لم يتم توفير صورة، فعادةً ما يتم إنشاء مربع بنسبة 16:9. القيمة الافتراضية: "auto". |
| `resolution` | COMBO | نعم | `"1K"`<br>`"2K"`<br>`"4K"` | دقة الإخراج المستهدفة. بالنسبة للدقة 2K/4K، يتم استخدام أداة تحسين الدقة الأصلية من Gemini. |
| `response_modalities` | COMBO | نعم | `"IMAGE+TEXT"`<br>`"IMAGE"` | اختر 'IMAGE' لإخراج صورة فقط، أو 'IMAGE+TEXT' لإرجاع كل من الصورة المُنشأة واستجابة نصية. |
| `images` | IMAGE | لا | غير متاح | صورة (صور) مرجعية اختيارية. لتضمين صور متعددة، استخدم عقدة Batch Images (حتى 14 صورة). |
| `files` | CUSTOM | لا | غير متاح | ملف (ملفات) اختيارية لاستخدامها كسياق للنموذج. يقبل المدخلات من عقدة Gemini Generate Content Input Files. |
| `system_prompt` | STRING | لا | غير متاح | تعليمات أساسية تملي سلوك الذكاء الاصطناعي. القيمة الافتراضية: موجه نظام محدد مسبقًا لإنشاء الصور. |

**القيود:**

* يدخل `images` يدعم حدًا أقصى يبلغ 14 صورة. إذا تم توفير أكثر من ذلك، فسيتم إظهار خطأ.
* يجب أن يكون مدخل `files` متصلاً بعقدة تُخرج نوع البيانات `GEMINI_INPUT_FILES`.

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|-------------|-----------|-------------|
| `image` | IMAGE | الصورة التي تم إنشاؤها أو تحريرها بواسطة نموذج Gemini. |
| `string` | STRING | الاستجابة النصية من النموذج. سيكون هذا المخرج فارغًا إذا تم تعيين `response_modalities` على "IMAGE". |
