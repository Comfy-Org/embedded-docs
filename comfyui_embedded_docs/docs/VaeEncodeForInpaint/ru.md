Этот узел предназначен для кодирования изображений в латентное представление, подходящее для задач инпейнтинга (дорисовки изображений), и включает дополнительные этапы предварительной обработки для настройки входного изображения и маски с целью оптимального кодирования моделью VAE.

## Входные параметры

| Параметр | Тип данных | Описание |
|-----------|-------------|-------------|
| `пиксели`  | `IMAGE`     | Входное изображение для кодирования. Перед кодированием это изображение проходит предварительную обработку и масштабирование для соответствия ожидаемым размерам входных данных модели VAE. |
| `vae`     | VAE       | Модель VAE, используемая для кодирования изображения в его латентное представление. Она играет ключевую роль в процессе преобразования, определяя качество и характеристики выходного латентного пространства. |
| `маска`    | `MASK`      | Маска, указывающая области входного изображения, подлежащие инпейнтингу. Она используется для изменения изображения перед кодированием, чтобы VAE фокусировалась на соответствующих областях. |
| `увеличить_маску_на` | `INT` | Определяет, насколько расширить маску инпейнтинга для обеспечения плавных переходов в латентном пространстве. Большее значение увеличивает область, затрагиваемую инпейнтингом. |

## Выходные параметры

| Параметр | Тип данных | Описание |
|-----------|-------------|-------------|
| `latent`  | `LATENT`    | Выходные данные включают закодированное латентное представление изображения и маску шума, которые необходимы для последующих задач инпейнтинга. |
