Вот перевод документации на русский язык:

Узел CosmosPredict2ImageToVideoLatent создает скрытые представления видео из изображений для генерации видео. Он может генерировать пустое скрытое представление видео или включать начальные и конечные изображения для создания видеопоследовательностей с заданными размерами и длительностью. Узел выполняет кодирование изображений в соответствующий формат скрытого пространства для обработки видео.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `vae` | VAE | Да | - | Модель VAE, используемая для кодирования изображений в скрытое пространство |
| `width` | INT | Нет | 16 до MAX_RESOLUTION | Ширина выходного видео в пикселях (по умолчанию: 848, должна делиться на 16) |
| `height` | INT | Нет | 16 до MAX_RESOLUTION | Высота выходного видео в пикселях (по умолчанию: 480, должна делиться на 16) |
| `length` | INT | Нет | 1 до MAX_RESOLUTION | Количество кадров в видеопоследовательности (по умолчанию: 93, шаг: 4) |
| `batch_size` | INT | Нет | 1 до 4096 | Количество видеопоследовательностей для генерации (по умолчанию: 1) |
| `start_image` | IMAGE | Нет | - | Необязательное начальное изображение для видеопоследовательности |
| `end_image` | IMAGE | Нет | - | Необязательное конечное изображение для видеопоследовательности |

**Примечание:** Если не указаны ни `start_image`, ни `end_image`, узел генерирует пустое скрытое представление видео. Если изображения предоставлены, они кодируются и размещаются в начале и/или конце видеопоследовательности с соответствующим маскированием.

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `samples` | LATENT | Сгенерированное скрытое представление видео, содержащее закодированную видеопоследовательность |
| `noise_mask` | LATENT | Маска, указывающая, какие части скрытого представления следует сохранить во время генерации |
