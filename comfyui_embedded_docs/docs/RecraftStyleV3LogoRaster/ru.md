Этот узел выбирает растровый стиль логотипа и необязательный подстиль для генерации изображений логотипов. Он специализируется на создании дизайнов логотипов с растровыми визуальными эффектами.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `substyle` | STRING | Да | Доступно несколько опций | Конкретный растровый подстиль логотипа для применения при генерации логотипа |

## Выходные параметры

| Имя выходного параметра | Тип данных | Описание |
|-------------|-----------|-------------|
| `recraft_style` | CUSTOM | Выбранная конфигурация стиля Recraft, включая растровый стиль логотипа и выбранный подстиль |
