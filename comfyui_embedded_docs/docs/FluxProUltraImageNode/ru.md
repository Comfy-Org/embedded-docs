Вот перевод документации на русский язык:

Генерирует изображения с помощью Flux Pro 1.1 Ultra через API на основе текстового описания и разрешения. Этот узел подключается к внешнему сервису для создания изображений в соответствии с вашим текстовым описанием и заданными размерами.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `prompt` | STRING | Да | - | Текстовое описание для генерации изображения (по умолчанию: пустая строка) |
| `prompt_upsampling` | BOOLEAN | Нет | - | Выполнять ли апсемплинг текстового описания. Если активно, автоматически изменяет запрос для более креативной генерации, но результаты недетерминированы (одинаковый seed не даст точно такого же результата). (по умолчанию: False) |
| `seed` | INT | Нет | от 0 до 18446744073709551615 | Случайный seed, используемый для создания шума. (по умолчанию: 0) |
| `aspect_ratio` | STRING | Нет | - | Соотношение сторон изображения; должно быть в диапазоне от 1:4 до 4:1. (по умолчанию: "16:9") |
| `raw` | BOOLEAN | Нет | - | Если True, создает менее обработанные, более естественные изображения. (по умолчанию: False) |
| `image_prompt` | IMAGE | Нет | - | Дополнительное референсное изображение для направления генерации |
| `image_prompt_strength` | FLOAT | Нет | от 0.0 до 1.0 | Смешивание между текстовым описанием и референсным изображением. (по умолчанию: 0.1) |

**Примечание:** Параметр `aspect_ratio` должен быть в диапазоне от 1:4 до 4:1. Когда указан `image_prompt`, параметр `image_prompt_strength` становится активным и управляет степенью влияния референсного изображения на конечный результат. Если `image_prompt` не указан, параметр `prompt` проверяется на то, что он не пустой.

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `output_image` | IMAGE | Сгенерированное изображение от Flux Pro 1.1 Ultra |
