أنت خبير في الترجمة التقنية متخصص في توثيق عُقد ComfyUI من الإنجليزية إلى العربية.

## قواعد الترجمة

1. **المحتوى الذي يجب عدم ترجمته:**
   - أسماء المعاملات بين علامات الاقتباس الخلفية: `image`, `seed`, `model`
   - أنواع البيانات بالأحرف الكبيرة: IMAGE, STRING, INT, FLOAT, MODEL, CONDITIONING, إلخ
   - القيم في عمود Range: الأرقام، "auto"، أسماء الخيارات
   - الكود، مسارات الملفات

2. **المحتوى الذي يجب ترجمته:**
   - عناوين الأقسام: ## نظرة عامة, ## المدخلات, ## المخرجات
   - جميع النصوص الوصفية والتوضيحية
   - أوصاف المعاملات

3. **جودة الترجمة:**
   - استخدام اللغة العربية الفصحى المعاصرة
   - الحفاظ على نبرة احترافية ولكن سهلة الفهم
   - ضمان الدقة التقنية
   - استخدام المصطلحات التقنية العربية القياسية

4. **التنسيق:**
   - الحفاظ على جميع تنسيقات Markdown
   - الحفاظ على بنية الجداول
   - عدم إضافة أي ملاحظة أو رابط في بداية الوثيقة (سيتم إضافتها تلقائيًا)

يقوم هذا العقدة بإنشاء الصور باستخدام Flux Pro 1.1 Ultra عبر API بناءً على المطالبة (prompt) والدقة. تتصل هذه العقدة بخدمة خارجية لإنشاء الصور وفقًا لوصفك النصي والأبعاد المحددة.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|-----------|-----------|----------|-------|-------------|
| `prompt` | STRING | نعم | - | المطالبة (prompt) لإنشاء الصورة (الافتراضي: سلسلة فارغة) |
| `prompt_upsampling` | BOOLEAN | لا | - | ما إذا كان سيتم إجراء رفع العينة (upsampling) على المطالبة. إذا كان نشطًا، يقوم تلقائيًا بتعديل المطالبة لتوليد أكثر إبداعًا، لكن النتائج غير حتمية (نفس البذرة لن تنتج نفس النتيجة تمامًا). (الافتراضي: False) |
| `seed` | INT | لا | 0 إلى 18446744073709551615 | البذرة العشوائية المستخدمة لإنشاء الضوضاء. (الافتراضي: 0) |
| `aspect_ratio` | STRING | لا | - | نسبة العرض إلى الارتفاع للصورة؛ يجب أن تكون بين 1:4 و 4:1. (الافتراضي: "16:9") |
| `raw` | BOOLEAN | لا | - | عندما يكون True، يتم إنشاء صور أقل معالجة وذات مظهر طبيعي أكثر. (الافتراضي: False) |
| `image_prompt` | IMAGE | لا | - | صورة مرجعية اختيارية لتوجيه عملية التوليد |
| `image_prompt_strength` | FLOAT | لا | 0.0 إلى 1.0 | المزج بين المطالبة وصورة المطالبة. (الافتراضي: 0.1) |

**ملاحظة:** يجب أن تكون قيمة المعامل `aspect_ratio` بين 1:4 و 4:1. عند توفير `image_prompt`، يصبح `image_prompt_strength` نشطًا ويتحكم في مدى تأثير الصورة المرجعية على المخرجات النهائية. إذا لم يتم توفير `image_prompt`، يتم التحقق من صحة المعامل `prompt` للتأكد من أنه ليس فارغًا.
