### نمای کلی
تصاویری را با استفاده از Flux Pro 1.1 Ultra از طریق API بر اساس prompt و وضوح تصویر تولید می‌کند. این گره به یک سرویس خارجی متصل می‌شود تا تصاویری مطابق با توضیحات متنی و ابعاد مشخص شده شما ایجاد کند.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `پرامپت` | STRING | بله | - | prompt برای تولید تصویر (پیش‌فرض: رشته خالی) |
| `افزایش وضوح پرامپت` | BOOLEAN | خیر | - | آیا بر روی prompt افزایش نمونه (upsampling) انجام شود. در صورت فعال بودن، به‌طور خودکار prompt را برای تولید خلاقانه‌تر تغییر می‌دهد، اما نتایج غیرقطعی هستند (همان seed نتیجه دقیقاً یکسانی تولید نخواهد کرد). (پیش‌فرض: False) |
| `seed` | INT | خیر | 0 تا 18446744073709551615 | seed تصادفی مورد استفاده برای ایجاد نویز. (پیش‌فرض: 0) |
| `نسبت تصویر` | STRING | خیر | - | نسبت ابعاد تصویر؛ باید بین 1:4 و 4:1 باشد. (پیش‌فرض: "16:9") |
| `خام` | BOOLEAN | خیر | - | در صورت True، تصاویری با پردازش کمتر و طبیعی‌تر تولید می‌کند. (پیش‌فرض: False) |
| `پرامپت تصویری` | IMAGE | خیر | - | تصویر مرجع اختیاری برای هدایت تولید |
| `قدرت پرامپت تصویری` | FLOAT | خیر | 0.0 تا 1.0 | میزان ترکیب بین prompt و تصویر prompt. (پیش‌فرض: 0.1) |

**توجه:** پارامتر `aspect_ratio` باید بین 1:4 و 4:1 باشد. هنگامی که `image_prompt` ارائه می‌شود، `image_prompt_strength` فعال می‌شود و میزان تأثیر تصویر مرجع بر خروجی نهایی را کنترل می‌کند. اگر `image_prompt` ارائه نشود، پارامتر `prompt` اعتبارسنجی می‌شود تا اطمینان حاصل شود که خالی نیست.
