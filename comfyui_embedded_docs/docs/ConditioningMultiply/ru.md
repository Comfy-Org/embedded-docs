# ConditioningMultiply

Вот перевод документации на русский язык:

Этот узел умножает значения conditioning на заданный коэффициент, позволяя масштабировать влияние conditioning на процесс генерации. Он работает путем применения множителя как к основному тензору conditioning, так и к объединенным выходным значениям.

## Входы

| Параметр | Описание | Тип данных | Обязательный | Диапазон |
|----------|----------|------------|--------------|----------|
| `conditioning` | Данные conditioning, которые необходимо масштабировать | CONDITIONING | Да | - |
| `multiplier` | Коэффициент, на который умножаются значения conditioning (по умолчанию: 1.0) | FLOAT | Да | от -100.0 до 100.0 (шаг: 0.01) |

## Выходы

| Имя выхода | Описание | Тип данных |
|------------|----------|------------|
| `CONDITIONING` | Масштабированные данные conditioning с умноженными значениями | CONDITIONING |
