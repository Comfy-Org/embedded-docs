Узел LTXV Audio VAE Encode принимает аудиовход и сжимает его в компактное латентное представление с помощью указанной модели Audio VAE. Этот процесс необходим для генерации или обработки аудио в рамках рабочего процесса с латентным пространством, так как он преобразует необработанные аудиоданные в формат, который могут понимать и обрабатывать другие узлы конвейера.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `audio` | AUDIO | Да | - | Аудио для кодирования. |
| `audio_vae` | VAE | Да | - | Модель Audio VAE, используемая для кодирования. |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| Audio Latent | LATENT | Сжатое латентное представление входного аудио. Выходные данные включают латентные сэмплы, частоту дискретизации модели VAE и идентификатор типа. |
