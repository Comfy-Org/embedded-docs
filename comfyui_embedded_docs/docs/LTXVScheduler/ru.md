Узел LTXVScheduler генерирует значения сигм для пользовательских процессов сэмплирования. Он вычисляет параметры шумового расписания на основе количества токенов во входном латентном представлении и применяет сигмоидное преобразование для создания расписания сэмплирования. Узел может дополнительно растягивать результирующие сигмы для соответствия заданному конечному значению.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `steps` | INT | Да | 1-10000 | Количество шагов сэмплирования (по умолчанию: 20) |
| `max_shift` | FLOAT | Да | 0.0-100.0 | Максимальное значение смещения для вычисления сигм (по умолчанию: 2.05) |
| `base_shift` | FLOAT | Да | 0.0-100.0 | Базовое значение смещения для вычисления сигм (по умолчанию: 0.95) |
| `stretch` | BOOLEAN | Да | True/False | Растягивать сигмы в диапазон [terminal, 1] (по умолчанию: True) |
| `terminal` | FLOAT | Да | 0.0-0.99 | Конечное значение сигм после растяжения (по умолчанию: 0.1) |
| `latent` | LATENT | Нет | - | Опциональный латентный вход, используемый для вычисления количества токенов для корректировки сигм |

**Примечание:** Параметр `latent` является опциональным. Когда он не предоставлен, узел использует количество токенов по умолчанию 4096 для вычислений.

## Выходные данные

| Выходной параметр | Тип данных | Описание |
|-------------|-----------|-------------|
| `sigmas` | SIGMAS | Сгенерированные значения сигм для процесса сэмплирования |
