### نمای کلی

با تعیین مقادیر مجزای قرمز، سبز و آبی، یک رنگ Recraft ایجاد کنید. این گره مقادیر صحیح RGB (0-255) را دریافت کرده و آن‌ها را به فرمت رنگ Recraft تبدیل می‌کند که می‌توان در سایر عملیات‌های Recraft استفاده کرد. همچنین می‌توانید به‌صورت اختیاری یک زنجیره رنگ Recraft موجود را برای گسترش آن با رنگ جدید ارائه دهید.

### ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|----------|----------|--------|--------|----------|
| `r` | INT | بله | 0-255 | مقدار قرمز رنگ (پیش‌فرض: 0) |
| `g` | INT | بله | 0-255 | مقدار سبز رنگ (پیش‌فرض: 0) |
| `b` | INT | بله | 0-255 | مقدار آبی رنگ (پیش‌فرض: 0) |
| `رنگ Recraft` | COLOR | خیر | - | زنجیره رنگ Recraft موجود برای گسترش با رنگ RGB جدید (اختیاری) |

### خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|------------|----------|----------|
| `رنگ Recraft` | COLOR | شیء رنگ Recraft ایجاد شده حاوی مقادیر RGB مشخص‌شده، یا زنجیره رنگ گسترش‌یافته در صورت ارائه یک زنجیره موجود |
