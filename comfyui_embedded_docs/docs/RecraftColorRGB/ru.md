Создаёт цвет Recraft, задавая отдельные значения красного, зелёного и синего. Этот узел принимает целочисленные значения RGB (0–255) и преобразует их в формат цвета Recraft, который можно использовать в других операциях Recraft. Вы также можете опционально предоставить существующую цепочку цветов Recraft, чтобы расширить её новым цветом.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `r` | INT | Да | 0–255 | Значение красного компонента цвета (по умолчанию: 0) |
| `g` | INT | Да | 0–255 | Значение зелёного компонента цвета (по умолчанию: 0) |
| `b` | INT | Да | 0–255 | Значение синего компонента цвета (по умолчанию: 0) |
| `recraft_color` | COLOR | Нет | — | Опциональная существующая цепочка цветов Recraft для расширения новым цветом RGB |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `recraft_color` | COLOR | Созданный объект цвета Recraft, содержащий указанные значения RGB, или расширенная цепочка цветов, если была предоставлена существующая |
