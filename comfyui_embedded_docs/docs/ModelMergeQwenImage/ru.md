Вот перевод документации на русский язык:

Узел ModelMergeQwenImage объединяет две модели ИИ, комбинируя их компоненты с настраиваемыми весами. Он позволяет смешивать определенные части моделей Qwen image, включая блоки трансформера, позиционные внедрения и компоненты обработки текста. Вы можете контролировать степень влияния каждой модели на различные участки объединенного результата.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `модель1` | MODEL | Да | - | Первая модель для объединения (по умолчанию: нет) |
| `модель2` | MODEL | Да | - | Вторая модель для объединения (по умолчанию: нет) |
| `pos_embeds.` | FLOAT | Да | 0.0 до 1.0 | Вес смешивания позиционных внедрений (по умолчанию: 1.0) |
| `img_in.` | FLOAT | Да | 0.0 до 1.0 | Вес смешивания обработки входного изображения (по умолчанию: 1.0) |
| `txt_norm.` | FLOAT | Да | 0.0 до 1.0 | Вес смешивания нормализации текста (по умолчанию: 1.0) |
| `txt_in.` | FLOAT | Да | 0.0 до 1.0 | Вес смешивания обработки входного текста (по умолчанию: 1.0) |
| `time_text_embed.` | FLOAT | Да | 0.0 до 1.0 | Вес смешивания внедрений времени и текста (по умолчанию: 1.0) |
| `transformer_blocks.0.` до `transformer_blocks.59.` | FLOAT | Да | 0.0 до 1.0 | Вес смешивания для каждого блока трансформера (по умолчанию: 1.0) |
| `proj_out.` | FLOAT | Да | 0.0 до 1.0 | Вес смешивания выходной проекции (по умолчанию: 1.0) |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `model` | MODEL | Объединенная модель, сочетающая компоненты обеих входных моделей с указанными весами |
