گره Kandinsky5ImageToVideo داده‌های شرطی و فضای نهان را برای تولید ویدئو با استفاده از مدل Kandinsky آماده می‌کند. این گره یک تنسور ویدئوی نهان خالی ایجاد می‌کند و می‌تواند به‌صورت اختیاری یک تصویر شروع را رمزگذاری کند تا فریم‌های اولیه ویدئوی تولیدشده را هدایت نماید و شرطی‌سازی مثبت و منفی را متناسباً تغییر دهد.

## ورودی‌ها

| پارامتر | نوع داده | اجباری | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `positive` | CONDITIONING | بله | نامعتبر | پرامپت‌های شرطی‌سازی مثبت برای هدایت تولید ویدئو. |
| `negative` | CONDITIONING | بله | نامعتبر | پرامپت‌های شرطی‌سازی منفی برای دور کردن تولید ویدئو از مفاهیم خاص. |
| `vae` | VAE | بله | نامعتبر | مدل VAE مورد استفاده برای رمزگذاری تصویر شروع اختیاری به فضای نهان. |
| `width` | INT | خیر | ۱۶ تا ۸۱۹۲ (گام ۱۶) | عرض ویدئوی خروجی بر حسب پیکسل (پیش‌فرض: ۷۶۸). |
| `height` | INT | خیر | ۱۶ تا ۸۱۹۲ (گام ۱۶) | ارتفاع ویدئوی خروجی بر حسب پیکسل (پیش‌فرض: ۵۱۲). |
| `length` | INT | خیر | ۱ تا ۸۱۹۲ (گام ۴) | تعداد فریم‌های ویدئو (پیش‌فرض: ۱۲۱). |
| `batch_size` | INT | خیر | ۱ تا ۴۰۹۶ | تعداد دنباله‌های ویدئویی که به‌طور همزمان تولید می‌شوند (پیش‌فرض: ۱). |
| `start_image` | IMAGE | خیر | نامعتبر | یک تصویر شروع اختیاری. در صورت ارائه، رمزگذاری شده و برای جایگزینی شروع نویزی خروجی‌های نهان مدل استفاده می‌شود. |

**نکته:** هنگامی که `start_image` ارائه می‌شود، به‌طور خودکار با استفاده از درون‌یابی دوخطی به اندازه `width` و `height` مشخص‌شده تغییر اندازه می‌یابد. اولین `length` فریم از دسته تصویر برای رمزگذاری استفاده می‌شود. سپس نهان رمزگذاری‌شده به هر دو شرطی‌سازی `positive` و `negative` تزریق می‌شود تا ظاهر اولیه ویدئو را هدایت کند.

## خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-------------|-----------|-------------|
| `positive` | CONDITIONING | شرطی‌سازی مثبت اصلاح‌شده، که احتمالاً با داده‌های تصویر شروع رمزگذاری‌شده به‌روزرسانی شده است. |
| `negative` | CONDITIONING | شرطی‌سازی منفی اصلاح‌شده، که احتمالاً با داده‌های تصویر شروع رمزگذاری‌شده به‌روزرسانی شده است. |
| `latent` | LATENT | یک تنسور ویدئوی نهان خالی با مقادیر صفر، شکل‌دهی‌شده برای ابعاد مشخص‌شده. |
| `cond_latent` | LATENT | نمایش نهان تمیز و رمزگذاری‌شده از تصاویر شروع ارائه‌شده. این به‌صورت داخلی برای جایگزینی شروع نویزی نهان‌های ویدئوی تولیدشده استفاده می‌شود. |
