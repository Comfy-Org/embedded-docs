Этот узел обнуляет определённые элементы внутри структуры данных conditioning, эффективно нейтрализуя их влияние на последующие этапы обработки. Он предназначен для продвинутых операций с conditioning, где требуется прямое манипулирование внутренним представлением conditioning.

## Входные параметры

| Параметр | Comfy dtype | Описание |
|-----------|----------------------------|-------------|
| `CONDITIONING` | CONDITIONING | Структура данных conditioning, подлежащая изменению. Этот узел обнуляет элементы 'pooled_output' в каждой записи conditioning, если они присутствуют. |

## Выходные параметры

| Параметр | Comfy dtype | Описание |
|-----------|----------------------------|-------------|
| `CONDITIONING` | CONDITIONING | Изменённая структура данных conditioning, в которой элементы 'pooled_output' обнулены, где это применимо. |
