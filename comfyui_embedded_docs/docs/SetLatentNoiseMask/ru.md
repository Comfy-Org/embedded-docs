Этот узел предназначен для применения шумовой маски к набору скрытых (latent) семплов. Он изменяет входные семплы, интегрируя заданную маску, тем самым изменяя их шумовые характеристики.

## Входы

| Параметр | Тип данных | Описание |
|-----------|-------------|-------------|
| `samples` | `LATENT` | Скрытые семплы, к которым будет применена шумовая маска. Этот параметр необходим для определения базового содержимого, которое будет изменено. |
| `mask` | `MASK` | Маска, применяемая к скрытым семплам. Она определяет области и интенсивность изменения шума внутри семплов. |

## Выходы

| Параметр | Тип данных | Описание |
|-----------|-------------|-------------|
| `latent` | `LATENT` | Измененные скрытые семплы с примененной шумовой маской. |
