## ورودی‌ها

| پارامتر | نوع داده | اجباری | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `start_frame` | IMAGE | بله | - | تصویر مرجع - رشته URL یا رمزگذاری‌شده با Base64، حداکثر ۱۰ مگابایت، وضوح تصویر کمتر از ۳۰۰×۳۰۰ پیکسل نباشد، نسبت ابعاد بین ۱:۲.۵ تا ۲.۵:۱. در رمزگذاری Base64 نباید پیشوند data:image گنجانده شود. |
| `end_frame` | IMAGE | بله | - | تصویر مرجع - کنترل فریم پایانی. رشته URL یا رمزگذاری‌شده با Base64، حداکثر ۱۰ مگابایت، وضوح تصویر کمتر از ۳۰۰×۳۰۰ پیکسل نباشد. در رمزگذاری Base64 نباید پیشوند data:image گنجانده شود. |
| `prompt` | STRING | بله | - | متن راهنمای مثبت |
| `negative_prompt` | STRING | بله | - | متن راهنمای منفی |
| `cfg_scale` | FLOAT | خیر | ۰.۰-۱.۰ | قدرت تأثیر راهنمای متن را کنترل می‌کند (پیش‌فرض: ۰.۵) |
| `aspect_ratio` | COMBO | خیر | "16:9"<br>"9:16"<br>"1:1" | نسبت ابعاد ویدیوی تولیدشده (پیش‌فرض: "16:9") |
| `mode` | COMBO | خیر | گزینه‌های متعدد موجود | پیکربندی مورد استفاده برای تولید ویدیو با فرمت: حالت / مدت‌زمان / نام_مدل. (پیش‌فرض: هفتمین گزینه از حالت‌های موجود) |

**محدودیت‌های تصویر:**

- هر دو `start_frame` و `end_frame` باید ارائه شوند و حجم فایل آنها از ۱۰ مگابایت بیشتر نشود
- حداقل وضوح تصویر: ۳۰۰×۳۰۰ پیکسل برای هر دو تصویر
- نسبت ابعاد `start_frame` باید بین ۱:۲.۵ تا ۲.۵:۱ باشد
- تصاویر رمزگذاری‌شده با Base64 نباید پیشوند "data:image" را شامل شوند
