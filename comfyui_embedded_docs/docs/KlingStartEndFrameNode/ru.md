Вот перевод документации на русский язык:

Узел Kling Start-End Frame to Video создает видеопоследовательность, которая плавно переходит между предоставленными начальным и конечным изображениями. Он генерирует все промежуточные кадры, чтобы обеспечить плавную трансформацию от первого кадра к последнему. Этот узел вызывает API преобразования изображения в видео, но поддерживает только те входные опции, которые работают с полем запроса `image_tail`.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `start_frame` | IMAGE | Да | - | Опорное изображение — URL или строка в кодировке Base64, размер не более 10 МБ, разрешение не менее 300×300 пикселей, соотношение сторон от 1:2,5 до 2,5:1. Base64 не должен содержать префикс data:image. |
| `end_frame` | IMAGE | Да | - | Опорное изображение — управление конечным кадром. URL или строка в кодировке Base64, размер не более 10 МБ, разрешение не менее 300×300 пикселей. Base64 не должен содержать префикс data:image. |
| `prompt` | STRING | Да | - | Положительный текстовый запрос |
| `negative_prompt` | STRING | Да | - | Отрицательный текстовый запрос |
| `cfg_scale` | FLOAT | Нет | 0.0–1.0 | Управляет силой влияния запроса (по умолчанию: 0.5) |
| `aspect_ratio` | COMBO | Нет | "16:9"<br>"9:16"<br>"1:1" | Соотношение сторон для генерируемого видео (по умолчанию: "16:9") |
| `mode` | COMBO | Нет | Доступно несколько опций | Конфигурация для генерации видео в формате: режим / длительность / имя_модели. (по умолчанию: седьмая опция из доступных режимов) |

**Ограничения для изображений:**

- Оба изображения `start_frame` и `end_frame` должны быть предоставлены, и их размер не должен превышать 10 МБ
- Минимальное разрешение: 300×300 пикселей для обоих изображений
- Соотношение сторон `start_frame` должно быть в диапазоне от 1:2,5 до 2,5:1
- Изображения в кодировке Base64 не должны содержать префикс "data:image"

## Выходные данные

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `output` | VIDEO | Сгенерированная видеопоследовательность |
| `video_id` | STRING | Уникальный идентификатор сгенерированного видео |
| `duration` | STRING | Длительность сгенерированного видео |
