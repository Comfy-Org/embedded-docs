Вот перевод документации на русский язык:

Узел Shuffle Dataset принимает список изображений и случайным образом изменяет их порядок. Для управления случайностью используется значение seed, что позволяет воспроизвести тот же порядок перемешивания. Это полезно для рандомизации последовательности изображений в наборе данных перед обработкой.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `images` | IMAGE | Да | - | Список изображений для перемешивания. |
| `seed` | INT | Нет | от 0 до 18446744073709551615 | Случайное зерно. Значение 0 будет каждый раз создавать новый порядок перемешивания. (по умолчанию: 0) |

## Выходы

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `images` | IMAGE | Тот же список изображений, но в новом, случайно перемешанном порядке. |
