### نمای کلی

این گره از API مدل Seedance 2.0 شرکت ByteDance برای تولید ویدیو بر اساس توضیحات متنی استفاده می‌کند. درخواست شما را به مدل انتخاب‌شده ارسال می‌کند، منتظر پردازش ویدیو می‌ماند و نتیجه نهایی را بازمی‌گرداند.

### ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|---------|----------|-------|--------|---------|
| `مدل` | COMBO | بله | `"Seedance 2.0"`<br>`"Seedance 2.0 Fast"` | مدل مورد استفاده برای تولید ویدیو. با انتخاب یک مدل، ورودی‌های اضافی مورد نیاز برای متن راهنما، وضوح تصویر، نسبت ابعاد، مدت زمان و تولید صدا نمایش داده می‌شوند. "Seedance 2.0" برای حداکثر کیفیت و "Seedance 2.0 Fast" برای بهینه‌سازی سرعت است. |
| `seed` | INT | خیر | 0 تا 2147483647 | مقدار دانه تصادفی (پیش‌فرض: 0). اگر این مقدار تغییر کند، گره دوباره اجرا می‌شود، اما نتایج صرف‌نظر از دانه تصادفی غیرقطعی هستند. |
| `واترمارک` | BOOLEAN | خیر | True / False | آیا به ویدیو واترمارک اضافه شود (پیش‌فرض: False). این یک تنظیم پیشرفته است. |

**توجه:** پارامتر `model` یک گزینه پویا است. وقتی مدلی را انتخاب می‌کنید، چندین زیرپارامتر ضروری برای تکمیل نمایش داده می‌شود که شامل متن راهنما، وضوح تصویر، نسبت ابعاد، مدت زمان و گزینه تولید صدا می‌شود. متن راهنما باید حداقل ۱ کاراکتر (پس از حذف فاصله‌های خالی) داشته باشد.
