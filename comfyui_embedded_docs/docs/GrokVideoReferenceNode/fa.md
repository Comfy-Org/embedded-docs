گره «مرجع به ویدیوی Grok» یک ویدیو بر اساس یک توضیح متنی تولید می‌کند و از حداکثر هفت تصویر مرجع برای هدایت سبک و محتوای خروجی استفاده می‌کند. این گره برای ایجاد ویدیو به یک API خارجی متصل می‌شود، سپس ویدیو دانلود شده و بازگردانده می‌شود.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|---------|----------|--------|--------|---------|
| `راهنمای متنی` | STRING | بله | نامشخص | توضیح متنی ویدیوی مورد نظر. |
| `مدل` | COMBO | بله | `"grok-imagine-video"` | مدل مورد استفاده برای تولید ویدیو. |
| `model.reference_images` | IMAGE | بله | ۱ تا ۷ تصویر | حداکثر ۷ تصویر مرجع برای هدایت تولید ویدیو. |
| `model.resolution` | COMBO | بله | `"480p"`<br>`"720p"` | وضوح تصویر ویدیوی خروجی. |
| `model.aspect_ratio` | COMBO | بله | `"16:9"`<br>`"4:3"`<br>`"3:2"`<br>`"1:1"`<br>`"2:3"`<br>`"3:4"`<br>`"9:16"` | نسبت ابعاد ویدیوی خروجی. |
| `model.duration` | INT | بله | ۲ تا ۱۰ | مدت زمان ویدیوی خروجی بر حسب ثانیه (پیش‌فرض: ۶). |
| `seed` | INT | خیر | ۰ تا ۲۱۴۷۴۸۳۶۴۷ | دانه (seed) برای تعیین اینکه گره دوباره اجرا شود؛ نتایج واقعی بدون توجه به دانه، غیرقطعی هستند (پیش‌فرض: ۰). |

**توجه:** پارامتر `model` یک گروه شامل `reference_images`، `resolution`، `aspect_ratio` و `duration` است. باید حداقل یک تصویر مرجع ارائه دهید و می‌توانید تا هفت تصویر مرجع داشته باشید.

| `video` | VIDEO | فایل ویدیوی تولیدشده. |
