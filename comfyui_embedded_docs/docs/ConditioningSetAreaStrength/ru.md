Этот узел предназначен для изменения атрибута силы заданного набора кондиционирования, что позволяет регулировать влияние или интенсивность кондиционирования на процесс генерации.

## Входные параметры

| Параметр | Тип данных | Описание |
|-----------|-------------|-------------|
| `CONDITIONING` | CONDITIONING | Набор кондиционирования, подлежащий изменению, представляющий текущее состояние кондиционирования, влияющее на процесс генерации. |
| `сила` | `FLOAT` | Значение силы, применяемое к набору кондиционирования, определяющее интенсивность его влияния. |

## Выходные параметры

| Параметр | Тип данных | Описание |
|-----------|-------------|-------------|
| `CONDITIONING` | CONDITIONING | Измененный набор кондиционирования с обновленными значениями силы для каждого элемента. |
