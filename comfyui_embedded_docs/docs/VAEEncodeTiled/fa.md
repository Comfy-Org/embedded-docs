### نمای کلی

گره VAEEncodeTiled با تقسیم تصاویر به کاشی‌های کوچک‌تر و رمزگذاری آن‌ها با استفاده از یک رمزگذار خودکار متغیر (Variational Autoencoder)، آن‌ها را پردازش می‌کند. این رویکرد کاشی‌بندی شده امکان پردازش تصاویر بزرگ را فراهم می‌کند که در غیر این صورت ممکن است از محدودیت حافظه فراتر روند. این گره از VAEهای تصویری و ویدیویی پشتیبانی می‌کند و کنترل‌های جداگانه‌ای برای کاشی‌بندی ابعاد مکانی و زمانی دارد.

## ورودی‌ها

| پارامتر | نوع داده | اجباری | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `پیکسل‌ها` | IMAGE | بله | - | داده‌های تصویر ورودی برای رمزگذاری |
| `vae` | VAE | بله | - | مدل رمزگذار خودکار متغیر مورد استفاده برای رمزگذاری |
| `اندازه کاشی` | INT | بله | 64-4096 (گام: 64) | اندازه هر کاشی برای پردازش مکانی (پیش‌فرض: 512) |
| `همپوشانی` | INT | بله | 0-4096 (گام: 32) | میزان هم‌پوشانی بین کاشی‌های مجاور (پیش‌فرض: 64) |
| `اندازه زمانی` | INT | بله | 8-4096 (گام: 4) | فقط برای VAEهای ویدیویی استفاده می‌شود: تعداد فریم‌هایی که در یک زمان رمزگذاری می‌شوند (پیش‌فرض: 64) |
| `همپوشانی زمانی` | INT | بله | 4-4096 (گام: 4) | فقط برای VAEهای ویدیویی استفاده می‌شود: تعداد فریم‌های هم‌پوشانی (پیش‌فرض: 8) |

**توجه:** پارامترهای `temporal_size` و `temporal_overlap` فقط هنگام استفاده از VAEهای ویدیویی مرتبط هستند و بر VAEهای استاندارد تصویری تأثیری ندارند.
