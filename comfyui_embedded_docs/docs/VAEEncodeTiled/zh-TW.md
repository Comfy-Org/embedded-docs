## 概述

VAEEncodeTiled 節點透過將影像分割成較小的區塊，並使用變分自編碼器進行編碼來處理影像。這種區塊化方法可以處理可能超出記憶體限制的大型影像。該節點支援影像和影片 VAE，並為空間和時間維度提供獨立的區塊控制。
