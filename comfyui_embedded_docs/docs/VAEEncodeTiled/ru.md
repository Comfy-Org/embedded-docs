Узел VAEEncodeTiled обрабатывает изображения, разбивая их на меньшие тайлы и кодируя с помощью вариационного автоэнкодера. Такой подход с разбиением на тайлы позволяет работать с большими изображениями, которые в противном случае могли бы превысить ограничения памяти. Узел поддерживает как обычные VAE для изображений, так и VAE для видео, с отдельными параметрами управления тайлами для пространственных и временных измерений.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `pixels` | IMAGE | Да | - | Входные данные изображения для кодирования |
| `vae` | VAE | Да | - | Модель вариационного автоэнкодера, используемая для кодирования |
| `tile_size` | INT | Да | 64-4096 (шаг: 64) | Размер каждого тайла для пространственной обработки (по умолчанию: 512) |
| `overlap` | INT | Да | 0-4096 (шаг: 32) | Величина перекрытия между соседними тайлами (по умолчанию: 64) |
| `temporal_size` | INT | Да | 8-4096 (шаг: 4) | Используется только для видео VAE: количество кадров, кодируемых за раз (по умолчанию: 64) |
| `temporal_overlap` | INT | Да | 4-4096 (шаг: 4) | Используется только для видео VAE: количество перекрывающихся кадров (по умолчанию: 8) |

**Примечание:** Параметры `temporal_size` и `temporal_overlap` актуальны только при использовании видео VAE и не влияют на работу стандартных VAE для изображений.

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `LATENT` | LATENT | Закодированное латентное представление входного изображения |
