Вот перевод документации на русский язык:

Выполняет инпейнтинг изображения на основе маски и текстового описания. Этот узел использует модель Flux.1 для заполнения замаскированных областей изображения в соответствии с предоставленным текстовым описанием, генерируя новое содержимое, которое соответствует окружающему изображению.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `image` | IMAGE | Да | - | Входное изображение, подлежащее инпейнтингу |
| `mask` | MASK | Да | - | Маска, определяющая области изображения, которые необходимо заполнить |
| `prompt` | STRING | Нет | - | Текстовое описание для генерации изображения (по умолчанию: пустая строка) |
| `повышение разрешения промпта` | BOOLEAN | Нет | - | Выполнять ли апсемплинг текстового описания. Если активно, автоматически изменяет описание для более креативной генерации, но результаты становятся недетерминированными (одинаковый seed не даст точно такого же результата). (по умолчанию: false) |
| `направление` | FLOAT | Нет | 1.5-100 | Сила управления процессом генерации изображения (по умолчанию: 60) |
| `шаги` | INT | Нет | 15-50 | Количество шагов для процесса генерации изображения (по умолчанию: 50) |
| `seed` | INT | Нет | 0-18446744073709551615 | Случайное зерно, используемое для создания шума. (по умолчанию: 0) |

## Выходные параметры

| Имя выходного параметра | Тип данных | Описание |
|-------------|-----------|-------------|
| `output_image` | IMAGE | Сгенерированное изображение с заполненными в соответствии с текстовым описанием замаскированными областями |
