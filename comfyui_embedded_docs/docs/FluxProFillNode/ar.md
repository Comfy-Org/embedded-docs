# ترجمة وثيقة عقدة ComfyUI: FluxProFillNode

يقوم هذا العقدة بملء الصورة بناءً على القناع والنص الوصفي. تستخدم العقدة نموذج Flux.1 لملء المناطق المقنعة في الصورة وفقًا للوصف النصي المقدم، مما يُنتج محتوى جديدًا يتطابق مع الصورة المحيطة.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|---------|--------------|--------|--------|-------|
| `image` | IMAGE | نعم | - | الصورة المدخلة المراد ملؤها |
| `mask` | MASK | نعم | - | القناع الذي يحدد مناطق الصورة التي يجب ملؤها |
| `prompt` | STRING | لا | - | النص الوصفي لتوليد الصورة (القيمة الافتراضية: سلسلة فارغة) |
| `prompt_upsampling` | BOOLEAN | لا | - | هل يتم إجراء رفع دقة النص الوصفي. إذا كان نشطًا، يقوم تلقائيًا بتعديل النص الوصفي لتوليد أكثر إبداعًا، لكن النتائج غير حتمية (نفس البذرة لن تُنتج نفس النتيجة تمامًا). (القيمة الافتراضية: false) |
| `guidance` | FLOAT | لا | 1.5-100 | قوة التوجيه لعملية توليد الصورة (القيمة الافتراضية: 60) |
| `steps` | INT | لا | 15-50 | عدد خطوات عملية توليد الصورة (القيمة الافتراضية: 50) |
| `seed` | INT | لا | 0-18446744073709551615 | البذرة العشوائية المستخدمة لإنشاء الضوضاء. (القيمة الافتراضية: 0) |

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|------------|--------------|--------|
| `output_image` | IMAGE | الصورة المُنشأة مع المناطق المقنعة المملوءة وفقًا للنص الوصفي |
