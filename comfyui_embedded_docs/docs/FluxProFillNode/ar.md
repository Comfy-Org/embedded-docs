# ترجمة وثيقة عقدة ComfyUI: FluxProFillNode

يقوم هذا العقدة بملء الصورة بناءً على القناع والنص الوصفي. تستخدم العقدة نموذج Flux.1 لملء المناطق المقنعة في الصورة وفقًا للوصف النصي المقدم، مما يُنتج محتوى جديدًا يتطابق مع الصورة المحيطة.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|---------|--------------|--------|--------|-------|
| `الصورة` | IMAGE | نعم | - | الصورة المدخلة المراد ملؤها |
| `القناع` | MASK | نعم | - | القناع الذي يحدد مناطق الصورة التي يجب ملؤها |
| `الوصف` | STRING | لا | - | النص الوصفي لتوليد الصورة (القيمة الافتراضية: سلسلة فارغة) |
| `تحسين_الوصف` | BOOLEAN | لا | - | هل يتم إجراء رفع دقة النص الوصفي. إذا كان نشطًا، يقوم تلقائيًا بتعديل النص الوصفي لتوليد أكثر إبداعًا، لكن النتائج غير حتمية (نفس البذرة لن تُنتج نفس النتيجة تمامًا). (القيمة الافتراضية: false) |
| `الإرشاد` | FLOAT | لا | 1.5-100 | قوة التوجيه لعملية توليد الصورة (القيمة الافتراضية: 60) |
| `الخطوات` | INT | لا | 15-50 | عدد خطوات عملية توليد الصورة (القيمة الافتراضية: 50) |
| `البذرة` | INT | لا | 0-18446744073709551615 | البذرة العشوائية المستخدمة لإنشاء الضوضاء. (القيمة الافتراضية: 0) |

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|------------|--------------|--------|
| `output_image` | IMAGE | الصورة المُنشأة مع المناطق المقنعة المملوءة وفقًا للنص الوصفي |
