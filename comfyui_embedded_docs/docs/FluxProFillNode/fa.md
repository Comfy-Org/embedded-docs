### نمای کلی

این گره بر اساس ماسک و راهنما (prompt) بخش‌هایی از تصویر را بازسازی (inpaint) می‌کند. با استفاده از مدل Flux.1، نواحی مشخص‌شده توسط ماسک را مطابق توضیحات متنی ارائه‌شده پر کرده و محتوای جدیدی تولید می‌کند که با تصویر اطراف هماهنگ باشد.

### ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `تصویر` | IMAGE | بله | - | تصویر ورودی که قرار است بازسازی شود |
| `ماسک` | MASK | بله | - | ماسکی که نواحی قابل‌پر شدن تصویر را مشخص می‌کند |
| `پرامپت` | STRING | خیر | - | راهنمای تولید تصویر (پیش‌فرض: رشته خالی) |
| `افزایش وضوح پرامپت` | BOOLEAN | خیر | - | آیا بر روی راهنما (prompt) افزایش وضوح (upsampling) انجام شود؟ در صورت فعال بودن، به‌طور خودکار راهنما را برای تولید خلاقانه‌تر تغییر می‌دهد، اما نتایج غیرقطعی هستند (همان seed نتیجه دقیقاً یکسانی تولید نمی‌کند). (پیش‌فرض: false) |
| `قدرت راهنمایی` | FLOAT | خیر | 1.5-100 | قدرت هدایت فرآیند تولید تصویر (پیش‌فرض: 60) |
| `تعداد مراحل` | INT | خیر | 15-50 | تعداد گام‌های فرآیند تولید تصویر (پیش‌فرض: 50) |
| `بذر` | INT | خیر | 0-18446744073709551615 | دانه تصادفی (random seed) برای ایجاد نویز. (پیش‌فرض: 0) |
