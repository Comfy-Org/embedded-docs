### نمای کلی

گره تولید ویدئو از تصویر Vidu2، یک دنباله ویدئویی را با شروع از یک تصویر ورودی واحد ایجاد می‌کند. این گره از یک مدل مشخص Vidu2 برای پویانمایی صحنه بر اساس یک متن راهنمای اختیاری استفاده کرده و طول، وضوح و شدت حرکت ویدئو را کنترل می‌کند.

## ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `model` | COMBO | بله | `"viduq2-pro-fast"`<br>`"viduq2-pro"`<br>`"viduq2-turbo"` | مدل Vidu2 مورد استفاده برای تولید ویدئو. مدل‌های مختلف، سرعت و کیفیت متفاوتی ارائه می‌دهند. |
| `image` | IMAGE | بله | - | تصویری که به عنوان فریم شروع ویدئوی تولید شده استفاده می‌شود. تنها یک تصویر مجاز است. |
| `prompt` | STRING | خیر | - | یک متن راهنمای اختیاری برای تولید ویدئو (حداکثر ۲۰۰۰ کاراکتر). پیش‌فرض رشته خالی است. |
| `duration` | INT | بله | ۱ تا ۱۰ | طول ویدئوی تولید شده بر حسب ثانیه. پیش‌فرض ۵ است. |
| `seed` | INT | خیر | ۰ تا ۲۱۴۷۴۸۳۶۴۷ | مقدار دانه (seed) برای تولید اعداد تصادفی جهت اطمینان از نتایج قابل تکرار. پیش‌فرض ۱ است. |
| `resolution` | COMBO | بله | `"720p"`<br>`"1080p"` | وضوح خروجی ویدئوی تولید شده. این پارامتر پیشرفته است. |
| `movement_amplitude` | COMBO | بله | `"auto"`<br>`"small"`<br>`"medium"`<br>`"large"` | دامنه حرکت اشیاء در قاب. این پارامتر پیشرفته است. |

**محدودیت‌ها:**

* ورودی `image` باید دقیقاً شامل یک تصویر باشد.
* نسبت ابعاد تصویر ورودی باید بین ۱:۴ و ۴:۱ باشد.
* متن `prompt` حداکثر به ۲۰۰۰ کاراکتر محدود شده است.
