Узел `NAGuidance` применяет к модели технику нормализованного управления вниманием (Normalized Attention Guidance). Этот метод позволяет использовать негативные промпты с дистиллированными или быстрыми (schnell) моделями, изменяя механизм внимания модели в процессе сэмплирования, чтобы направить генерацию в сторону от нежелательных концепций.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `model` | MODEL | Да | - | Модель, к которой применяется нормализованное управление вниманием. |
| `nag_scale` | FLOAT | Да | 0.0 - 50.0 | Коэффициент масштабирования управления. Более высокие значения сильнее отталкивают генерацию от негативного промпта. (по умолчанию: 5.0) |
| `nag_alpha` | FLOAT | Да | 0.0 - 1.0 | Коэффициент смешивания для нормализованного внимания. Значение 1.0 полностью заменяет исходное внимание, значение 0.0 не оказывает эффекта. (по умолчанию: 0.5) |
| `nag_tau` | FLOAT | Да | 1.0 - 10.0 | Масштабирующий коэффициент, используемый для ограничения коэффициента нормализации. (по умолчанию: 1.5) |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `model` | MODEL | Модифицированная модель с включенным нормализованным управлением вниманием. |
