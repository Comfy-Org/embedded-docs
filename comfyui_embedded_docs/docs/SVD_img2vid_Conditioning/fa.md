گره SVD_img2vid_Conditioning داده‌های شرطی‌سازی را برای تولید ویدیو با استفاده از Stable Video Diffusion آماده می‌کند. این گره یک تصویر اولیه را دریافت کرده و آن را از طریق رمزگذارهای CLIP vision و VAE پردازش می‌کند تا جفت‌های شرطی‌سازی مثبت و منفی، همراه با یک فضای نهان خالی برای تولید ویدیو ایجاد کند. این گره پارامترهای لازم برای کنترل حرکت، نرخ فریم و سطوح افزایش داده را در ویدیوی تولیدشده تنظیم می‌کند.

## ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `clip_vision` | CLIP_VISION | بله | - | مدل بینایی CLIP برای رمزگذاری تصویر ورودی |
| `init_image` | IMAGE | بله | - | تصویر اولیه به عنوان نقطه شروع برای تولید ویدیو |
| `vae` | VAE | بله | - | مدل VAE برای رمزگذاری تصویر به فضای نهان |
| `width` | INT | بله | 16 تا MAX_RESOLUTION | عرض ویدیوی خروجی (پیش‌فرض: 1024، گام: 8) |
| `height` | INT | بله | 16 تا MAX_RESOLUTION | ارتفاع ویدیوی خروجی (پیش‌فرض: 576، گام: 8) |
| `video_frames` | INT | بله | 1 تا 4096 | تعداد فریم‌های تولیدشده در ویدیو (پیش‌فرض: 14) |
| `motion_bucket_id` | INT | بله | 1 تا 1023 | میزان حرکت در ویدیوی تولیدشده را کنترل می‌کند (پیش‌فرض: 127) |
| `fps` | INT | بله | 1 تا 1024 | فریم بر ثانیه برای ویدیوی تولیدشده (پیش‌فرض: 6) |
| `augmentation_level` | FLOAT | بله | 0.0 تا 10.0 | سطح نویز افزایش داده برای اعمال به تصویر ورودی (پیش‌فرض: 0.0، گام: 0.01) |

## خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-------------|-----------|-------------|
| `positive` | CONDITIONING | داده‌های شرطی‌سازی مثبت شامل جاسازی‌های تصویر و پارامترهای ویدیو |
| `negative` | CONDITIONING | داده‌های شرطی‌سازی منفی با جاسازی‌های صفرشده و پارامترهای ویدیو |
| `latent` | LATENT | تنسور فضای نهان خالی آماده برای تولید ویدیو |
