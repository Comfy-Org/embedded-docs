Узел `SVD_img2vid_Conditioning` подготавливает данные кондиционирования для генерации видео с использованием Stable Video Diffusion. Он принимает исходное изображение и обрабатывает его через кодировщики CLIP vision и VAE, создавая пары положительного и отрицательного кондиционирования, а также пустое латентное пространство для генерации видео. Этот узел задает необходимые параметры для управления движением, частотой кадров и уровнем аугментации в генерируемом видео.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `clip_vision` | CLIP_VISION | Да | - | Модель CLIP vision для кодирования входного изображения |
| `init_image` | IMAGE | Да | - | Исходное изображение, используемое в качестве отправной точки для генерации видео |
| `vae` | VAE | Да | - | Модель VAE для кодирования изображения в латентное пространство |
| `ширина` | INT | Да | от 16 до MAX_RESOLUTION | Ширина выходного видео (по умолчанию: 1024, шаг: 8) |
| `высота` | INT | Да | от 16 до MAX_RESOLUTION | Высота выходного видео (по умолчанию: 576, шаг: 8) |
| `кадры_видео` | INT | Да | от 1 до 4096 | Количество кадров для генерации в видео (по умолчанию: 14) |
| `id_корзины_движения` | INT | Да | от 1 до 1023 | Управляет интенсивностью движения в генерируемом видео (по умолчанию: 127) |
| `fps` | INT | Да | от 1 до 1024 | Количество кадров в секунду для генерируемого видео (по умолчанию: 6) |
| `уровень_аугментации` | FLOAT | Да | от 0.0 до 10.0 | Уровень шумовой аугментации, применяемой к входному изображению (по умолчанию: 0.0, шаг: 0.01) |

## Выходные параметры

| Имя выходного параметра | Тип данных | Описание |
|-------------------------|------------|----------|
| `positive` | CONDITIONING | Данные положительного кондиционирования, содержащие эмбеддинги изображения и параметры видео |
| `negative` | CONDITIONING | Данные отрицательного кондиционирования с обнуленными эмбеддингами и параметрами видео |
| `latent` | LATENT | Тензор пустого латентного пространства, готовый к генерации видео |
