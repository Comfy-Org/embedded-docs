Этот узел изменяет размер изображений таким образом, чтобы более короткая сторона соответствовала заданной длине, сохраняя исходное соотношение сторон. Он вычисляет новые размеры на основе целевой длины для более короткой стороны и возвращает изменённое изображение.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `image` | IMAGE | Да | - | Входное изображение для изменения размера. |
| `shorter_edge` | INT | Нет | от 1 до 8192 | Целевая длина для более короткой стороны. (по умолчанию: 512) |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `image` | IMAGE | Изменённое изображение, у которого более короткая сторона соответствует заданной целевой длине. |
