گره SaveLoRA مدل‌های LoRA (Low-Rank Adaptation) را در دایرکتوری خروجی شما ذخیره می‌کند. این گره یک مدل LoRA را به عنوان ورودی دریافت کرده و یک فایل safetensors با نامی که به‌طور خودکار تولید شده است ایجاد می‌کند. می‌توانید پیشوند نام فایل را سفارشی کنید و به‌صورت اختیاری تعداد مراحل آموزش را برای سازماندهی بهتر در نام فایل بگنجانید.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|---------|----------|--------|--------|---------|
| `lora` | LORA_MODEL | بله | - | مدل LoRA که باید ذخیره شود. از مدل دارای لایه‌های LoRA استفاده نکنید. |
| `prefix` | STRING | بله | - | پیشوندی که برای فایل LoRA ذخیره‌شده استفاده می‌شود (پیش‌فرض: "loras/ComfyUI_trained_lora"). |
| `steps` | INT | خیر | - | اختیاری: تعداد مراحل آموزش LoRA که برای نام‌گذاری فایل ذخیره‌شده استفاده می‌شود. |
