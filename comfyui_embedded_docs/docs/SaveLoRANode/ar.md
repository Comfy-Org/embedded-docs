# ترجمة وثيقة عقدة SaveLoRA

عقدة حفظ لورا (SaveLoRA) تقوم بحفظ نماذج لورا (Low-Rank Adaptation) في دليل المخرجات الخاص بك. تستقبل نموذج لورا كمدخل وتُنشئ ملفًا بصيغة safetensors مع اسم ملف يتم إنشاؤه تلقائيًا. يمكنك تخصيص بادئة اسم الملف، وإدراج عدد خطوات التدريب اختياريًا في اسم الملف لتحسين التنظيم.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|---------|---------------|--------|---------|-------|
| `lora` | LORA_MODEL | نعم | - | نموذج لورا المراد حفظه. لا تستخدم النموذج مع طبقات لورا. |
| `prefix` | STRING | نعم | - | البادئة المستخدمة لملف لورا المحفوظ (الافتراضي: "loras/ComfyUI_trained_lora"). |
| `steps` | INT | لا | - | اختياري: عدد خطوات تدريب لورا، يُستخدم لتسمية الملف المحفوظ. |

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|------------|---------------|---------|
| *لا يوجد* | - | لا تُرجع هذه العقدة أي مخرجات، ولكنها تحفظ نموذج لورا في دليل المخرجات. |
