Вот перевод документации узла SaveLoRA на русский язык:

Узел SaveLoRA сохраняет модели LoRA (Low-Rank Adaptation) в ваш выходной каталог. Он принимает модель LoRA на вход и создает файл safetensors с автоматически сгенерированным именем. Вы можете настроить префикс имени файла и при необходимости включить количество шагов обучения в имя файла для лучшей организации.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `lora` | LORA_MODEL | Да | - | Модель LoRA для сохранения. Не используйте модель со слоями LoRA. |
| `prefix` | STRING | Да | - | Префикс для сохраняемого файла LoRA (по умолчанию: "loras/ComfyUI_trained_lora"). |
| `steps` | INT | Нет | - | Необязательно: количество шагов, на которых обучалась LoRA, используется для именования сохраняемого файла. |

## Выходы

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| *Нет* | - | Этот узел не возвращает никаких выходных данных, но сохраняет модель LoRA в выходной каталог. |
