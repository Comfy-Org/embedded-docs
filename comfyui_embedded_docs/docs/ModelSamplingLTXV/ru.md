Вот перевод документации на русский язык:

Узел ModelSamplingLTXV применяет расширенные параметры сэмплирования к модели на основе количества токенов. Он вычисляет значение сдвига с помощью линейной интерполяции между базовым и максимальным значениями сдвига, причем расчет зависит от количества токенов во входном латентном представлении. Затем узел создает специализированную конфигурацию сэмплирования модели и применяет ее к входной модели.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `модель` | MODEL | Да | - | Входная модель, к которой применяются параметры сэмплирования |
| `макс_сдвиг` | FLOAT | Да | от 0.0 до 100.0 | Максимальное значение сдвига, используемое в расчете линейной интерполяции (по умолчанию: 2.05) |
| `базовый_сдвиг` | FLOAT | Да | от 0.0 до 100.0 | Базовое значение сдвига, используемое в расчете линейной интерполяции (по умолчанию: 0.95) |
| `скрытый` | LATENT | Нет | - | Необязательное входное латентное представление, используемое для определения количества токенов при расчете сдвига. Если не указано, используется количество токенов по умолчанию — 4096 |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `модель` | MODEL | Модифицированная модель с примененными параметрами сэмплирования |
