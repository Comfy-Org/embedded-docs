گره ModelSamplingLTXV پارامترهای نمونه‌برداری پیشرفته را بر اساس تعداد توکن‌ها به یک مدل اعمال می‌کند. این گره یک مقدار شیفت را با استفاده از درون‌یابی خطی بین مقادیر شیفت پایه و حداکثر محاسبه می‌کند که این محاسبه به تعداد توکن‌های موجود در ورودی نهفته (latent) بستگی دارد. سپس گره یک پیکربندی نمونه‌برداری مدل تخصصی ایجاد کرده و آن را به مدل ورودی اعمال می‌کند.

## ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `مدل` | MODEL | بله | - | مدل ورودی که پارامترهای نمونه‌برداری به آن اعمال می‌شود |
| `بیشینه شیفت` | FLOAT | بله | 0.0 تا 100.0 | حداکثر مقدار شیفت مورد استفاده در محاسبه درون‌یابی خطی (پیش‌فرض: 2.05) |
| `شیفت پایه` | FLOAT | بله | 0.0 تا 100.0 | مقدار شیفت پایه مورد استفاده در محاسبه درون‌یابی خطی (پیش‌فرض: 0.95) |
| `latent` | LATENT | خیر | - | ورودی نهفته اختیاری که برای تعیین تعداد توکن‌ها در محاسبه شیفت استفاده می‌شود. در صورت عدم ارائه، تعداد توکن پیش‌فرض 4096 استفاده می‌شود |
