## Обзор

Этот узел генерирует или редактирует изображения с использованием моделей Seedream от ByteDance (версии 4.0, 4.5 и 5.0). Он может создавать новые изображения по текстовому запросу или редактировать существующие изображения, предоставляя референсные изображения, с поддержкой разрешения до 4K.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `prompt` | STRING | Да | Н/Д | Текстовый запрос для создания или редактирования изображения. |
| `model` | COMBO | Да | `"seedream 5.0 lite"`<br>`"seedream-4-5-251128"`<br>`"seedream-4-0-250828"` | Версия модели Seedream для генерации. Каждая модель имеет разные возможности и стоимость. |
| `seed` | INT | Нет | от 0 до 2147483647 | Сид для генерации (по умолчанию: 0). |
| `watermark` | BOOLEAN | Нет | True / False | Добавлять ли водяной знак "AI generated" на изображение (по умолчанию: False). |

### Параметры, зависящие от модели

При выборе модели становятся доступны дополнительные параметры:

- **Size Preset**: Выпадающий список для выбора предустановленного разрешения изображения (например, "2048x2048", "1024x1024"). Доступные пресеты зависят от выбранной модели.
- **Width**: Ширина генерируемого изображения в пикселях (по умолчанию: 2048).
- **Height**: Высота генерируемого изображения в пикселях (по умолчанию: 2048).
- **Max Images**: Максимальное количество изображений для генерации (по умолчанию: 1). При значении 1 последовательная генерация изображений отключается.
- **Reference Images**: До 10 (для Seedream 4.0 и 4.5) или 14 (для Seedream 5.0 Lite) референсных изображений для редактирования. Изображения должны иметь соотношение сторон от 1:3 до 3:1.
- **Fail on Partial**: Если включено, узел выдаст ошибку, если не все запрошенные изображения будут успешно сгенерированы (по умолчанию: False).

### Ограничения по разрешению

- **Seedream 5.0 Lite и 4.5**: Минимальное разрешение — 3,68 мегапикселя (например, 1920x1920).
- **Seedream 4.0**: Минимальное разрешение — 0,92 мегапикселя (например, 960x960).
- **Все модели**: Максимальное разрешение — 16,78 мегапикселя (например, 4096x4096).

### Ограничения по количеству изображений

- Общее количество референсных и сгенерированных изображений не может превышать 15.
- Для Seedream 5.0 Lite поддерживается максимум 14 референсных изображений.
- Для Seedream 4.0 и 4.5 поддерживается максимум 10 референсных изображений.

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `image` | IMAGE | Сгенерированное или отредактированное изображение в виде тензора. Если было запрошено несколько изображений, они объединяются в один пакет. |
