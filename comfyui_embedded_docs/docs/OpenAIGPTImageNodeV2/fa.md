## نمای کلی

این گره با استفاده از API تصویر GPT شرکت OpenAI، تصاویر را تولید می‌کند. از چندین مدل پشتیبانی می‌کند، امکان ارائه تصاویر ورودی برای ویرایش را فراهم می‌کند و می‌تواند از یک ماسک برای مشخص کردن بخش‌هایی از تصویر که باید تغییر کنند استفاده کند.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|---------|----------|--------|--------|----------|
| `prompt` | STRING | بله | نامشخص | متن راهنمای (پرامپت) برای GPT Image (پیش‌فرض: ""). |
| `model` | COMBO | بله | `"gpt-image-2"`<br>`"gpt-image-1.5"`<br>`"gpt-image-1"` | مدل GPT Image OpenAI که استفاده می‌شود. انتخاب یک مدل، پارامترهای اضافی مخصوص آن مدل را نمایش می‌دهد. |
| `model.size` | COMBO | بله | `"auto"`<br>`"1024x1024"`<br>`"1024x1536"`<br>`"1536x1024"`<br>`"2048x2048"`<br>`"2048x1152"`<br>`"1152x2048"`<br>`"3840x2160"`<br>`"2160x3840"`<br>`"Custom"` | اندازه تصویر. گزینه 'Custom' را برای استفاده از عرض و ارتفاع سفارشی انتخاب کنید (پیش‌فرض: "auto"). فقط برای `gpt-image-2` در دسترس است. |
| `model.custom_width` | INT | خیر | 1024 تا 3840 | فقط زمانی استفاده می‌شود که `size` برابر 'Custom' باشد. باید مضربی از 16 باشد (پیش‌فرض: 1024). فقط برای `gpt-image-2` در دسترس است. |
| `model.custom_height` | INT | خیر | 1024 تا 3840 | فقط زمانی استفاده می‌شود که `size` برابر 'Custom' باشد. باید مضربی از 16 باشد (پیش‌فرض: 1024). فقط برای `gpt-image-2` در دسترس است. |
| `model.background` | COMBO | بله | `"auto"`<br>`"opaque"` | بازگرداندن تصویر با یا بدون پس‌زمینه (پیش‌فرض: "auto"). فقط برای `gpt-image-2` در دسترس است. |
| `model.quality` | COMBO | بله | `"standard"`<br>`"hd"` | کیفیت تصویر تولیدشده. فقط برای `gpt-image-2` در دسترس است. |
| `model.images` | IMAGE | خیر | نامشخص | تصاویر ورودی برای ویرایش. فقط برای `gpt-image-2` در دسترس است. |
| `model.mask` | MASK | خیر | نامشخص | ماسکی برای مشخص کردن بخش‌هایی از تصویر ورودی که باید ویرایش شوند. فقط برای `gpt-image-2` در دسترس است. |
| `n` | INT | بله | 1 تا 8 | تعداد تصاویر تولیدی (پیش‌فرض: 1). |
| `seed` | INT | بله | 0 تا 2147483647 | دانه (seed) برای تکرارپذیری (پیش‌فرض: 0). توجه: هنوز در بک‌اند پیاده‌سازی نشده است. |

**محدودیت‌ها و قیود پارامترها:**

- هنگام استفاده از `gpt-image-2` با `model.size` برابر "Custom"، `custom_width` و `custom_height` باید مضرب 16 باشند، حداکثر ضلع باید ≤ 3840 باشد، نسبت ابعاد نباید از 3:1 تجاوز کند، و تعداد کل پیکسل‌ها باید بین 655,360 و 8,294,400 باشد.
- اگر `mask` ارائه شود، یک تصویر ورودی (`model.images`) الزامی است. ماسک بدون تصویر ورودی قابل استفاده نیست.
- ماسک با چندین تصویر ورودی قابل استفاده نیست.
- هنگامی که ماسک ارائه می‌شود، ابعاد ماسک باید با ابعاد تصویر ورودی مطابقت داشته باشد.
- پارامتر `seed` در حال حاضر عملیاتی نیست.
