## Обзор

Этот узел генерирует изображения с использованием OpenAI GPT Image API. Он поддерживает несколько моделей, позволяет предоставлять входные изображения для редактирования и может использовать маску для указания частей изображения, которые необходимо изменить.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `prompt` | STRING | Да | Н/Д | Текстовый запрос для GPT Image (по умолчанию: ""). |
| `model` | COMBO | Да | `"gpt-image-2"`<br>`"gpt-image-1.5"`<br>`"gpt-image-1"` | Используемая модель OpenAI GPT Image. Выбор модели открывает дополнительные параметры, специфичные для данной модели. |
| `model.size` | COMBO | Да | `"auto"`<br>`"1024x1024"`<br>`"1024x1536"`<br>`"1536x1024"`<br>`"2048x2048"`<br>`"2048x1152"`<br>`"1152x2048"`<br>`"3840x2160"`<br>`"2160x3840"`<br>`"Custom"` | Размер изображения. Выберите 'Custom', чтобы использовать пользовательские ширину и высоту (по умолчанию: "auto"). Доступно только для `gpt-image-2`. |
| `model.custom_width` | INT | Нет | от 1024 до 3840 | Используется только когда `size` имеет значение 'Custom'. Должно быть кратно 16 (по умолчанию: 1024). Доступно только для `gpt-image-2`. |
| `model.custom_height` | INT | Нет | от 1024 до 3840 | Используется только когда `size` имеет значение 'Custom'. Должно быть кратно 16 (по умолчанию: 1024). Доступно только для `gpt-image-2`. |
| `model.background` | COMBO | Да | `"auto"`<br>`"opaque"` | Возврат изображения с фоном или без него (по умолчанию: "auto"). Доступно только для `gpt-image-2`. |
| `model.quality` | COMBO | Да | `"standard"`<br>`"hd"` | Качество сгенерированного изображения. Доступно только для `gpt-image-2`. |
| `model.images` | IMAGE | Нет | Н/Д | Входные изображения для редактирования. Доступно только для `gpt-image-2`. |
| `model.mask` | MASK | Нет | Н/Д | Маска для указания частей входного изображения, которые необходимо изменить. Доступно только для `gpt-image-2`. |
| `n` | INT | Да | от 1 до 8 | Количество генерируемых изображений (по умолчанию: 1). |
| `seed` | INT | Да | от 0 до 2147483647 | Зерно для воспроизводимости результатов (по умолчанию: 0). Примечание: в настоящее время не реализовано в бэкенде. |

**Ограничения параметров:**

- При использовании `gpt-image-2` с `model.size` равным "Custom", значения `custom_width` и `custom_height` должны быть кратны 16, максимальная сторона не должна превышать 3840, соотношение сторон не должно превышать 3:1, а общее количество пикселей должно быть от 655 360 до 8 294 400.
- Если указана `mask`, требуется входное изображение (`model.images`). Маска не может использоваться без входного изображения.
- Маска не может использоваться с несколькими входными изображениями.
- При использовании маски её размеры должны совпадать с размерами входного изображения.
- Параметр `seed` в настоящее время не работает.

| Имя выхода | Тип данных | Описание |

| `image` | IMAGE | Сгенерированное изображение или изображения. |
