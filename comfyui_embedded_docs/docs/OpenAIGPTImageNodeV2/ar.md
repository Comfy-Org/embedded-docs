## نظرة عامة

تقوم هذه العقدة بإنشاء الصور باستخدام واجهة برمجة تطبيقات GPT Image من OpenAI. وهي تدعم نماذج متعددة، وتتيح لك توفير صور إدخال للتحرير، ويمكنها استخدام قناع لتحديد أجزاء الصورة المراد تعديلها.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|-----------|-----------|----------|-------|-------------|
| `الموجه` | STRING | نعم | غير متاح | النص التوجيهي لـ GPT Image (الافتراضي: ""). |
| `النموذج` | COMBO | نعم | `"gpt-image-2"`<br>`"gpt-image-1.5"`<br>`"gpt-image-1"` | نموذج GPT Image من OpenAI المراد استخدامه. يؤدي اختيار نموذج إلى إظهار معاملات إضافية خاصة بذلك النموذج. |
| `model.size` | COMBO | نعم | `"auto"`<br>`"1024x1024"`<br>`"1024x1536"`<br>`"1536x1024"`<br>`"2048x2048"`<br>`"2048x1152"`<br>`"1152x2048"`<br>`"3840x2160"`<br>`"2160x3840"`<br>`"Custom"` | حجم الصورة. اختر 'Custom' لاستخدام العرض والارتفاع المخصصين (الافتراضي: "auto"). متاح فقط لـ `gpt-image-2`. |
| `model.custom_width` | INT | لا | 1024 إلى 3840 | يُستخدم فقط عندما يكون `size` هو 'Custom'. يجب أن يكون مضاعفًا للرقم 16 (الافتراضي: 1024). متاح فقط لـ `gpt-image-2`. |
| `model.custom_height` | INT | لا | 1024 إلى 3840 | يُستخدم فقط عندما يكون `size` هو 'Custom'. يجب أن يكون مضاعفًا للرقم 16 (الافتراضي: 1024). متاح فقط لـ `gpt-image-2`. |
| `model.background` | COMBO | نعم | `"auto"`<br>`"opaque"` | إرجاع الصورة مع أو بدون خلفية (الافتراضي: "auto"). متاح فقط لـ `gpt-image-2`. |
| `model.quality` | COMBO | نعم | `"standard"`<br>`"hd"` | جودة الصورة المُنشأة. متاح فقط لـ `gpt-image-2`. |
| `model.images` | IMAGE | لا | غير متاح | صور الإدخال للتحرير. متاح فقط لـ `gpt-image-2`. |
| `model.mask` | MASK | لا | غير متاح | قناع لتحديد أجزاء صورة الإدخال المراد تحريرها. متاح فقط لـ `gpt-image-2`. |
| `عدد الصور` | INT | نعم | 1 إلى 8 | عدد الصور المراد إنشاؤها (الافتراضي: 1). |
| `البذرة` | INT | نعم | 0 إلى 2147483647 | بذرة لإمكانية إعادة الإنتاج (الافتراضي: 0). ملاحظة: لم يتم تفعيلها بعد في الخلفية. |

**قيود المعاملات والحدود:**

- عند استخدام `gpt-image-2` مع `model.size` بقيمة "Custom"، يجب أن يكون `custom_width` و `custom_height` مضاعفين للرقم 16، ويجب ألا تتجاوز الحافة القصوى 3840، ويجب ألا تتجاوز نسبة العرض إلى الارتفاع 3:1، ويجب أن يتراوح إجمالي عدد البكسلات بين 655,360 و 8,294,400.
- إذا تم توفير `mask`، فإن صورة الإدخال (`model.images`) مطلوبة. لا يمكن استخدام القناع بدون صورة إدخال.
- لا يمكن استخدام القناع مع صور إدخال متعددة.
- عند توفير قناع، يجب أن تتطابق أبعاد القناع مع أبعاد صورة الإدخال.
- معامل `seed` غير فعال حاليًا.
