`ImageCrop` 節點專為裁剪圖像而設計，可從指定的 x 和 y 座標開始，將圖像裁剪至指定的寬度和高度。此功能對於聚焦圖像的特定區域，或調整圖像尺寸以滿足特定需求至關重要。

## 輸入

| 欄位 | 資料類型 | 說明 |
|-------|-------------|-----------------------------------------------------------------------------------------------|
| `image` | `IMAGE` | 要裁剪的輸入圖像。此參數至關重要，它定義了將根據指定的尺寸和座標從中提取區域的來源圖像。 |
| `width` | `INT` | 指定裁剪後圖像的寬度。此參數決定最終裁剪圖像的寬度。 |
| `height` | `INT` | 指定裁剪後圖像的高度。此參數決定最終裁剪圖像的高度。 |
| `x` | `INT` | 裁剪區域左上角的 x 座標。此參數設定裁剪寬度維度的起始點。 |
| `y` | `INT` | 裁剪區域左上角的 y 座標。此參數設定裁剪高度維度的起始點。 |

## 輸出

| 欄位 | 資料類型 | 說明 |
|-------|-------------|-----------------------------------------------------------------------------------------------|
| `image` | `IMAGE` | 裁剪操作產生的裁剪後圖像。此輸出對於指定圖像區域的進一步處理或分析非常重要。 |
