Узел `ImageCrop` предназначен для обрезки изображений до заданной ширины и высоты, начиная с указанных координат x и y. Эта функция необходима для выделения определённых областей изображения или для приведения размера изображения в соответствие с определёнными требованиями.

## Входные параметры

| Поле | Тип данных | Описание                                                                                   |
|-------|-------------|-----------------------------------------------------------------------------------------------|
| `image` | `IMAGE` | Входное изображение для обрезки. Этот параметр является ключевым, так как определяет исходное изображение, из которого будет извлечена область на основе заданных размеров и координат. |
| `width` | `INT` | Задаёт ширину обрезанного изображения. Этот параметр определяет, насколько широким будет результирующее обрезанное изображение. |
| `height` | `INT` | Задаёт высоту обрезанного изображения. Этот параметр определяет высоту результирующего обрезанного изображения. |
| `x` | `INT` | Координата x верхнего левого угла области обрезки. Этот параметр задаёт начальную точку для размера обрезки по ширине. |
| `y` | `INT` | Координата y верхнего левого угла области обрезки. Этот параметр задаёт начальную точку для размера обрезки по высоте. |

## Выходные параметры

| Поле | Тип данных | Описание                                                                   |
|-------|-------------|-------------------------------------------------------------------------------|
| `image` | `IMAGE` | Обрезанное изображение как результат операции обрезки. Этот выходной параметр важен для дальнейшей обработки или анализа указанной области изображения. |
