گره `EmptyHunyuanLatentVideo` مشابه گره `EmptyLatentImage` است. می‌توانید آن را به‌عنوان یک بوم خالی برای تولید ویدیو در نظر بگیرید، جایی که عرض، ارتفاع و طول، ویژگی‌های بوم را تعریف می‌کنند و اندازه دسته (batch size) تعداد بوم‌های ایجاد شده را مشخص می‌کند. این گره بوم‌های خالی را برای وظایف بعدی تولید ویدیو آماده می‌سازد.

## ورودی‌ها

| پارامتر       | نوع Comfy | توضیحات                                                                                |
| ----------- | ---------- | ------------------------------------------------------------------------------------------ |
| `عرض`     | `INT`      | عرض ویدیو، پیش‌فرض 848، حداقل 16، حداکثر `nodes.MAX_RESOLUTION`، اندازه گام 16.        |
| `ارتفاع`    | `INT`      | ارتفاع ویدیو، پیش‌فرض 480، حداقل 16، حداکثر `nodes.MAX_RESOLUTION`، اندازه گام 16.       |
| `طول`    | `INT`      | طول ویدیو، پیش‌فرض 25، حداقل 1، حداکثر `nodes.MAX_RESOLUTION`، اندازه گام 4.          |
| `اندازه دسته`| `INT`      | اندازه دسته، پیش‌فرض 1، حداقل 1، حداکثر 4096.                                           |

## خروجی‌ها

| پارامتر    | نوع Comfy | توضیحات                                                                               |
| --------- | ---------- | ----------------------------------------------------------------------------------------- |
| `samples` | `LATENT`   | نمونه‌های ویدیویی نهفته (latent) تولید شده شامل تنسورهای صفر، آماده برای وظایف پردازش و تولید. |
