عقدة `EmptyHunyuanLatentVideo` تشبه عقدة `EmptyLatentImage`. يمكنك اعتبارها لوحة فارغة لتوليد الفيديو، حيث يُحدد كل من العرض والارتفاع والطول خصائص اللوحة، بينما يحدد حجم الدفعة عدد اللوحات المراد إنشاؤها. تقوم هذه العقدة بإنشاء لوحات فارغة جاهزة لمهام توليد الفيديو اللاحقة.

## المدخلات

| المعامل | نوع Comfy | الوصف |
| ----------- | ---------- | ------------------------------------------------------------------------------------------ |
| `العرض` | `INT` | عرض الفيديو، القيمة الافتراضية 848، الحد الأدنى 16، الحد الأقصى `nodes.MAX_RESOLUTION`، حجم الخطوة 16. |
| `الارتفاع` | `INT` | ارتفاع الفيديو، القيمة الافتراضية 480، الحد الأدنى 16، الحد الأقصى `nodes.MAX_RESOLUTION`، حجم الخطوة 16. |
| `المدة` | `INT` | طول الفيديو، القيمة الافتراضية 25، الحد الأدنى 1، الحد الأقصى `nodes.MAX_RESOLUTION`، حجم الخطوة 4. |
| `حجم_الدُفعة` | `INT` | حجم الدفعة، القيمة الافتراضية 1، الحد الأدنى 1، الحد الأقصى 4096. |

## المخرجات

| المعامل | نوع Comfy | الوصف |
