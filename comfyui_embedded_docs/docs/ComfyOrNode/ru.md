# ComfyOrNode

Узел ComfyOrNode выполняет логическую операцию ИЛИ над набором входных значений. Он возвращает `true`, если хотя бы одно из предоставленных значений считается истинным в соответствии со стандартными правилами истинности Python.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `value` | ANY | Да | Принимается несколько значений | Значение для проверки на истинность. Вы можете предоставить несколько значений, добавляя дополнительные входы. Узел возвращает `true`, если хотя бы одно из этих значений является истинным. |

**Примечание:** Узел принимает минимум 1 входное значение. При необходимости вы можете добавить больше входов с помощью функции авторасширения.

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `BOOLEAN` | BOOLEAN | Возвращает `true`, если хотя бы одно из входных значений является истинным; возвращает `false`, если все входные значения являются ложными. |
