# ترجمة وثيقة عقد WanTrackToVideo

## نظرة عامة

يقوم عقد WanTrackToVideo بتحويل بيانات تتبع الحركة إلى تسلسلات فيديو من خلال معالجة نقاط التتبع وإنشاء إطارات الفيديو المقابلة. يستقبل إحداثيات التتبع كمدخلات وينتج تمثيلات تكييف الفيديو وتمثيلات كامنة يمكن استخدامها لتوليد الفيديو. عندما لا يتم توفير مسارات تتبع، يتحول العقد إلى التحويل القياسي من صورة إلى فيديو.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|----------|---------------|--------|---------|-------|
| `positive` | CONDITIONING | نعم | - | التكييف الإيجابي لتوليد الفيديو |
| `negative` | CONDITIONING | نعم | - | التكييف السلبي لتوليد الفيديو |
| `vae` | VAE | نعم | - | نموذج VAE للتشفير وفك التشفير |
| `tracks` | STRING | نعم | - | بيانات التتبع بتنسيق JSON كنص متعدد الأسطر (القيمة الافتراضية: "[]") |
| `width` | INT | نعم | 16 إلى MAX_RESOLUTION | عرض الفيديو الناتج بالبكسل (القيمة الافتراضية: 832، الخطوة: 16) |
| `height` | INT | نعم | 16 إلى MAX_RESOLUTION | ارتفاع الفيديو الناتج بالبكسل (القيمة الافتراضية: 480، الخطوة: 16) |
| `length` | INT | نعم | 1 إلى MAX_RESOLUTION | عدد الإطارات في الفيديو الناتج (القيمة الافتراضية: 81، الخطوة: 4) |
| `batch_size` | INT | نعم | 1 إلى 4096 | عدد مقاطع الفيديو المولدة في وقت واحد (القيمة الافتراضية: 1) |
| `temperature` | FLOAT | نعم | 1.0 إلى 1000.0 | معامل درجة الحرارة لترقيع الحركة (القيمة الافتراضية: 220.0، الخطوة: 0.1) |
| `topk` | INT | نعم | 1 إلى 10 | قيمة Top-k لترقيع الحركة (القيمة الافتراضية: 2) |
| `start_image` | IMAGE | لا | - | الصورة الابتدائية لتوليد الفيديو |
| `clip_vision_output` | CLIPVISIONOUTPUT | لا | - | مخرجات رؤية CLIP لتكييف إضافي |

**ملاحظة:** عندما يحتوي `tracks` على بيانات تتبع صالحة، يقوم العقد بمعالجة مسارات الحركة لتوليد الفيديو. عندما يكون `tracks` فارغًا، يتحول العقد إلى وضع الصورة إلى فيديو القياسي. إذا تم توفير `start_image`، فإنه يهيئ الإطار الأول من تسلسل الفيديو.

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|-------------|---------------|-------|
| `positive` | CONDITIONING | التكييف الإيجابي مع تطبيق معلومات مسار الحركة |
| `negative` | CONDITIONING | التكييف السلبي مع تطبيق معلومات مسار الحركة |
| `latent` | LATENT | التمثيل الكامن للفيديو المولد |
