# ترجمة وثيقة عقدة ComfyUI: OpenAI Chat Node

## نظرة عامة

تولد هذه العقدة استجابات نصية من نموذج OpenAI. تقوم بإرسال النص المُدخل (وبشكل اختياري الصور أو الملفات) إلى نموذج OpenAI وتعيد الاستجابة النصية المُولَّدة.

## المُدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|---------|---------------|--------|---------|-------|
| `prompt` | STRING | نعم | - | النصوص المُدخلة للنموذج، تُستخدم لتوليد استجابة (الافتراضي: فارغ) |
| `persist_context` | BOOLEAN | نعم | - | هذه المعلمة مُهمَلة وليس لها أي تأثير (الافتراضي: False) |
| `model` | COMBO | نعم | نماذج OpenAI متعددة متاحة | النموذج المستخدم لتوليد الاستجابة |
| `images` | IMAGE | لا | - | صورة (صور) اختيارية لاستخدامها كسياق للنموذج. لتضمين صور متعددة، يمكنك استخدام عقدة Batch Images |
| `files` | OPENAI_INPUT_FILES | لا | - | ملف (ملفات) اختيارية لاستخدامها كسياق للنموذج. يقبل المُدخلات من عقدة OpenAI Chat Input Files |
| `advanced_options` | OPENAI_CHAT_CONFIG | لا | - | إعدادات اختيارية للنموذج. يقبل المُدخلات من عقدة OpenAI Chat Advanced Options |

## المُخرجات

| اسم المُخرج | نوع البيانات | الوصف |
|--------------|---------------|---------|
| `output_text` | STRING | الاستجابة النصية المُولَّدة بواسطة نموذج OpenAI |
