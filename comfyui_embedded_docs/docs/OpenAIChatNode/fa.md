> این مستند با هوش مصنوعی تهیه شده است. اگر خطایی دیدید یا پیشنهادی برای بهبود دارید، خوشحال می‌شویم مشارکت کنید! [ویرایش در GitHub](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/OpenAIChatNode/fa.md)

این گره متن پاسخ را از یک مدل OpenAI تولید می‌کند. این گره متن prompt شما (و به‌صورت اختیاری تصاویر یا فایل‌ها) را به یک مدل OpenAI ارسال کرده و متن پاسخ تولیدشده را بازمی‌گرداند.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|----------|----------|--------|--------|----------|
| `prompt` | STRING | بله | - | ورودی‌های متنی به مدل، که برای تولید پاسخ استفاده می‌شود (پیش‌فرض: خالی) |
| `persist_context` | BOOLEAN | بله | - | این پارامتر منسوخ شده است و تأثیری ندارد (پیش‌فرض: False) |
| `model` | COMBO | بله | چندین مدل OpenAI در دسترس | مدلی که برای تولید پاسخ استفاده می‌شود |
| `images` | IMAGE | خیر | - | تصویر(های) اختیاری برای استفاده به‌عنوان زمینه مدل. برای گنجاندن چندین تصویر، می‌توانید از گره Batch Images استفاده کنید |
| `files` | OPENAI_INPUT_FILES | خیر | - | فایل(های) اختیاری برای استفاده به‌عنوان زمینه مدل. ورودی‌های گره OpenAI Chat Input Files را می‌پذیرد |
| `advanced_options` | OPENAI_CHAT_CONFIG | خیر | - | پیکربندی اختیاری برای مدل. ورودی‌های گره OpenAI Chat Advanced Options را می‌پذیرد |

## خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|------------|----------|----------|
| `output_text` | STRING | متن پاسخ تولیدشده توسط مدل OpenAI |
