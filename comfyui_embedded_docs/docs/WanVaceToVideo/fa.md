گره WanVaceToVideo داده‌های شرطی‌سازی ویدئو را برای مدل‌های تولید ویدئو پردازش می‌کند. این گره ورودی‌های شرطی‌سازی مثبت و منفی را به همراه داده‌های کنترل ویدئو دریافت کرده و نمایش‌های نهفته (latent) را برای تولید ویدئو آماده می‌سازد. این گره وظایف افزایش مقیاس ویدئو، ماسک‌گذاری و رمزگذاری VAE را برای ایجاد ساختار شرطی‌سازی مناسب برای مدل‌های ویدئویی انجام می‌دهد.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `مثبت` | CONDITIONING | بله | - | ورودی شرطی‌سازی مثبت برای هدایت فرآیند تولید |
| `منفی` | CONDITIONING | بله | - | ورودی شرطی‌سازی منفی برای هدایت فرآیند تولید |
| `vae` | VAE | بله | - | مدل VAE مورد استفاده برای رمزگذاری تصاویر و فریم‌های ویدئو |
| `عرض` | INT | بله | 16 تا MAX_RESOLUTION | عرض ویدئوی خروجی بر حسب پیکسل (پیش‌فرض: 832، گام: 16) |
| `ارتفاع` | INT | بله | 16 تا MAX_RESOLUTION | ارتفاع ویدئوی خروجی بر حسب پیکسل (پیش‌فرض: 480، گام: 16) |
| `طول` | INT | بله | 1 تا MAX_RESOLUTION | تعداد فریم‌های ویدئو (پیش‌فرض: 81، گام: 4) |
| `اندازه بچ` | INT | بله | 1 تا 4096 | تعداد ویدئوهایی که به‌طور همزمان تولید می‌شوند (پیش‌فرض: 1) |
| `شدت` | FLOAT | بله | 0.0 تا 1000.0 | قدرت کنترل برای شرطی‌سازی ویدئو (پیش‌فرض: 1.0، گام: 0.01) |
| `ویدئوی کنترل` | IMAGE | خیر | - | ویدئوی ورودی اختیاری برای شرطی‌سازی کنترل |
| `ماسک‌های کنترل` | MASK | خیر | - | ماسک‌های اختیاری برای کنترل بخش‌هایی از ویدئو که باید تغییر کنند |
| `تصویر مرجع` | IMAGE | خیر | - | تصویر مرجع اختیاری برای شرطی‌سازی اضافی |

**نکته:** هنگامی که `control_video` ارائه می‌شود، برای مطابقت با عرض و ارتفاع مشخص‌شده، افزایش مقیاس می‌یابد. اگر `control_masks` ارائه شوند، باید با ابعاد ویدئوی کنترل مطابقت داشته باشند. `reference_image` از طریق VAE رمزگذاری شده و در صورت ارائه، به ابتدای دنباله نهفته اضافه می‌شود.

## خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-------------|-----------|-------------|
| `مثبت` | CONDITIONING | شرطی‌سازی مثبت با داده‌های کنترل ویدئوی اعمال‌شده |
| `منفی` | CONDITIONING | شرطی‌سازی منفی با داده‌های کنترل ویدئوی اعمال‌شده |
| `latent` | LATENT | تنسور نهفته خالی آماده برای تولید ویدئو |
| `trim_latent` | INT | تعداد فریم‌های نهفته‌ای که هنگام استفاده از تصویر مرجع باید حذف شوند |
