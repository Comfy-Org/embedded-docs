Узел WanVaceToVideo обрабатывает данные кондиционирования видео для моделей генерации видео. Он принимает положительные и отрицательные входные данные кондиционирования вместе с данными управления видео и подготавливает латентные представления для генерации видео. Узел выполняет масштабирование видео, создание масок и VAE-кодирование для формирования соответствующей структуры кондиционирования для видеомоделей.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `позитивный` | CONDITIONING | Да | - | Положительное кондиционирование для направления генерации |
| `негативный` | CONDITIONING | Да | - | Отрицательное кондиционирование для направления генерации |
| `vae` | VAE | Да | - | Модель VAE, используемая для кодирования изображений и видеокадров |
| `ширина` | INT | Да | 16 до MAX_RESOLUTION | Ширина выходного видео в пикселях (по умолчанию: 832, шаг: 16) |
| `высота` | INT | Да | 16 до MAX_RESOLUTION | Высота выходного видео в пикселях (по умолчанию: 480, шаг: 16) |
| `длина` | INT | Да | 1 до MAX_RESOLUTION | Количество кадров в видео (по умолчанию: 81, шаг: 4) |
| `размер_пакета` | INT | Да | 1 до 4096 | Количество видео для одновременной генерации (по умолчанию: 1) |
| `интенсивность` | FLOAT | Да | 0.0 до 1000.0 | Сила управления для кондиционирования видео (по умолчанию: 1.0, шаг: 0.01) |
| `control_video` | IMAGE | Нет | - | Необязательное входное видео для управляющего кондиционирования |
| `control_masks` | MASK | Нет | - | Необязательные маски для управления изменяемыми частями видео |
| `референсное_изображение` | IMAGE | Нет | - | Необязательное эталонное изображение для дополнительного кондиционирования |

**Примечание:** Если указан `control_video`, он будет масштабирован до заданных значений ширины и высоты. Если указаны `control_masks`, они должны соответствовать размерам управляющего видео. При наличии `reference_image` он кодируется через VAE и добавляется в начало последовательности латентных представлений.

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `позитивный` | CONDITIONING | Положительное кондиционирование с применёнными данными управления видео |
| `негативный` | CONDITIONING | Отрицательное кондиционирование с применёнными данными управления видео |
| `latent` | LATENT | Пустой латентный тензор, готовый для генерации видео |
| `trim_latent` | INT | Количество латентных кадров для обрезки при использовании эталонного изображения |
