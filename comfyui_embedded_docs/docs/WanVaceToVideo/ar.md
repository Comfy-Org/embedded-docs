بالتأكيد، إليك الترجمة العربية للوثيقة التقنية الخاصة بالعقدة `WanVaceToVideo` مع الالتزام التام بقواعد الترجمة المحددة:

تقوم العقدة WanVaceToVideo بمعالجة بيانات تهيئة الفيديو لنماذج توليد الفيديو. تستقبل هذه العقدة مدخلات تهيئة إيجابية وسلبية إلى جانب بيانات التحكم في الفيديو، وتُحضّر التمثيلات الكامنة (latent representations) لتوليد الفيديو. تتعامل العقدة مع عمليات رفع دقة الفيديو، والقص (masking)، وترميز VAE لإنشاء بنية التهيئة المناسبة لنماذج الفيديو.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|-----------|-----------|----------|-------|-------------|
| `إيجابي` | CONDITIONING | نعم | - | مدخل التهيئة الإيجابي لتوجيه عملية التوليد |
| `سلبي` | CONDITIONING | نعم | - | مدخل التهيئة السلبي لتوجيه عملية التوليد |
| `vae` | VAE | نعم | - | نموذج VAE المستخدم لترميز الصور وإطارات الفيديو |
| `العرض` | INT | نعم | 16 إلى MAX_RESOLUTION | عرض فيديو الإخراج بالبكسل (الافتراضي: 832، الخطوة: 16) |
| `الارتفاع` | INT | نعم | 16 إلى MAX_RESOLUTION | ارتفاع فيديو الإخراج بالبكسل (الافتراضي: 480، الخطوة: 16) |
| `الطول` | INT | نعم | 1 إلى MAX_RESOLUTION | عدد الإطارات في الفيديو (الافتراضي: 81، الخطوة: 4) |
| `حجم الدفعة` | INT | نعم | 1 إلى 4096 | عدد مقاطع الفيديو المراد توليدها في وقت واحد (الافتراضي: 1) |
| `القوة` | FLOAT | نعم | 0.0 إلى 1000.0 | قوة التحكم في تهيئة الفيديو (الافتراضي: 1.0، الخطوة: 0.01) |
| `فيديو التحكم` | IMAGE | لا | - | فيديو إدخال اختياري لتهيئة التحكم |
| `أقنعة التحكم` | MASK | لا | - | أقنعة اختيارية للتحكم في أجزاء الفيديو المراد تعديلها |
| `صورة مرجعية` | IMAGE | لا | - | صورة مرجعية اختيارية لتهيئة إضافية |

**ملاحظة:** عند توفير `control_video`، سيتم رفع دقته لمطابقة العرض والارتفاع المحددين. إذا تم توفير `control_masks`، فيجب أن تتطابق أبعادها مع أبعاد فيديو التحكم. يتم ترميز `reference_image` عبر VAE وإضافتها إلى بداية التسلسل الكامن (latent sequence) عند توفيرها.

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|-------------|-----------|-------------|
| `سلبي` | CONDITIONING | التهيئة الإيجابية مع تطبيق بيانات التحكم في الفيديو |
| `مضمر` | CONDITIONING | التهيئة السلبية مع تطبيق بيانات التحكم في الفيديو |
| `اقتطاع المضمر` | LATENT | موتر كامن فارغ جاهز لتوليد الفيديو |
