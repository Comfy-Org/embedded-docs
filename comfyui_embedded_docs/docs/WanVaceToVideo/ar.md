بالتأكيد، إليك الترجمة العربية للوثيقة التقنية الخاصة بالعقدة `WanVaceToVideo` مع الالتزام التام بقواعد الترجمة المحددة:

تقوم العقدة WanVaceToVideo بمعالجة بيانات تهيئة الفيديو لنماذج توليد الفيديو. تستقبل هذه العقدة مدخلات تهيئة إيجابية وسلبية إلى جانب بيانات التحكم في الفيديو، وتُحضّر التمثيلات الكامنة (latent representations) لتوليد الفيديو. تتعامل العقدة مع عمليات رفع دقة الفيديو، والقص (masking)، وترميز VAE لإنشاء بنية التهيئة المناسبة لنماذج الفيديو.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|-----------|-----------|----------|-------|-------------|
| `positive` | CONDITIONING | نعم | - | مدخل التهيئة الإيجابي لتوجيه عملية التوليد |
| `negative` | CONDITIONING | نعم | - | مدخل التهيئة السلبي لتوجيه عملية التوليد |
| `vae` | VAE | نعم | - | نموذج VAE المستخدم لترميز الصور وإطارات الفيديو |
| `width` | INT | نعم | 16 إلى MAX_RESOLUTION | عرض فيديو الإخراج بالبكسل (الافتراضي: 832، الخطوة: 16) |
| `height` | INT | نعم | 16 إلى MAX_RESOLUTION | ارتفاع فيديو الإخراج بالبكسل (الافتراضي: 480، الخطوة: 16) |
| `length` | INT | نعم | 1 إلى MAX_RESOLUTION | عدد الإطارات في الفيديو (الافتراضي: 81، الخطوة: 4) |
| `batch_size` | INT | نعم | 1 إلى 4096 | عدد مقاطع الفيديو المراد توليدها في وقت واحد (الافتراضي: 1) |
| `strength` | FLOAT | نعم | 0.0 إلى 1000.0 | قوة التحكم في تهيئة الفيديو (الافتراضي: 1.0، الخطوة: 0.01) |
| `control_video` | IMAGE | لا | - | فيديو إدخال اختياري لتهيئة التحكم |
| `control_masks` | MASK | لا | - | أقنعة اختيارية للتحكم في أجزاء الفيديو المراد تعديلها |
| `reference_image` | IMAGE | لا | - | صورة مرجعية اختيارية لتهيئة إضافية |

**ملاحظة:** عند توفير `control_video`، سيتم رفع دقته لمطابقة العرض والارتفاع المحددين. إذا تم توفير `control_masks`، فيجب أن تتطابق أبعادها مع أبعاد فيديو التحكم. يتم ترميز `reference_image` عبر VAE وإضافتها إلى بداية التسلسل الكامن (latent sequence) عند توفيرها.

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|-------------|-----------|-------------|
| `positive` | CONDITIONING | التهيئة الإيجابية مع تطبيق بيانات التحكم في الفيديو |
| `negative` | CONDITIONING | التهيئة السلبية مع تطبيق بيانات التحكم في الفيديو |
| `latent` | LATENT | موتر كامن فارغ جاهز لتوليد الفيديو |
| `trim_latent` | INT | عدد الإطارات الكامنة المراد قصها عند استخدام الصورة المرجعية |
