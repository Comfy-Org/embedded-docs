گره Pikadditions به شما امکان می‌دهد هر شیء یا تصویری را به ویدیوی خود اضافه کنید. شما یک ویدیو آپلود کرده و مشخص می‌کنید چه چیزی می‌خواهید اضافه کنید تا نتیجه‌ای یکپارچه ایجاد شود. این گره از Pika API برای درج تصاویر در ویدیوها با ادغامی طبیعی استفاده می‌کند.

## ورودی‌ها

| پارامتر | نوع داده | اجباری | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `video` | VIDEO | بله | - | ویدیویی که تصویر به آن اضافه می‌شود. |
| `image` | IMAGE | بله | - | تصویری که به ویدیو اضافه می‌شود. |
| `prompt_text` | STRING | بله | - | توضیح متنی درباره آنچه به ویدیو اضافه می‌شود. |
| `negative_prompt` | STRING | بله | - | توضیح متنی درباره آنچه باید در ویدیو اجتناب شود. |
| `seed` | INT | بله | 0 تا 4294967295 | مقدار seed تصادفی برای نتایج قابل تکرار. |

## خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-------------|-----------|-------------|
| `output` | VIDEO | ویدیوی پردازش‌شده با تصویر درج‌شده. |
