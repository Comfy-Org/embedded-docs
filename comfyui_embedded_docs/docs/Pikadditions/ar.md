# ترجمة وثيقة عقدة Pikadditions

عقدة Pikadditions تتيح لك إضافة أي كائن أو صورة إلى الفيديو الخاص بك. يمكنك رفع فيديو وتحديد ما ترغب في إضافته لإنشاء نتيجة متكاملة بسلاسة. تستخدم هذه العقدة واجهة برمجة تطبيقات Pika لإدراج الصور في مقاطع الفيديو مع تكامل طبيعي المظهر.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|---------|---------------|--------|---------|-------|
| `video` | VIDEO | نعم | - | الفيديو المراد إضافة صورة إليه. |
| `image` | IMAGE | نعم | - | الصورة المراد إضافتها إلى الفيديو. |
| `prompt_text` | STRING | نعم | - | وصف نصي لما يجب إضافته إلى الفيديو. |
| `negative_prompt` | STRING | نعم | - | وصف نصي لما يجب تجنبه في الفيديو. |
| `seed` | INT | نعم | 0 إلى 4294967295 | قيمة البذرة العشوائية للحصول على نتائج قابلة للتكرار. |

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|-------------|---------------|---------|
| `output` | VIDEO | الفيديو المعالج مع الصورة المدرجة. |
