Вот перевод документации на русский язык:

Узел Pikadditions позволяет добавить любой объект или изображение в ваше видео. Вы загружаете видео и указываете, что хотите добавить, чтобы получить бесшовно интегрированный результат. Этот узел использует API Pika для вставки изображений в видео с естественным внешним видом.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `video` | VIDEO | Да | - | Видео, в которое нужно добавить изображение. |
| `image` | IMAGE | Да | - | Изображение, которое нужно добавить в видео. |
| `prompt_text` | STRING | Да | - | Текстовое описание того, что нужно добавить в видео. |
| `negative_prompt` | STRING | Да | - | Текстовое описание того, чего следует избегать в видео. |
| `seed` | INT | Да | от 0 до 4294967295 | Значение случайного зерна для воспроизводимых результатов. |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `output` | VIDEO | Обработанное видео со вставленным изображением. |
