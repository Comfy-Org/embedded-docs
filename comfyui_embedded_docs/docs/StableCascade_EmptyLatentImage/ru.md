Узел `StableCascade_EmptyLatentImage` создаёт пустые тензоры скрытого пространства для моделей Stable Cascade. Он генерирует два отдельных представления скрытого пространства — одно для этапа C и другое для этапа B — с соответствующими размерами, основанными на входном разрешении и настройках сжатия. Этот узел является отправной точкой для конвейера генерации Stable Cascade.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `width` | INT | Да | 256 до MAX_RESOLUTION | Ширина выходного изображения в пикселях (по умолчанию: 1024, шаг: 8) |
| `height` | INT | Да | 256 до MAX_RESOLUTION | Высота выходного изображения в пикселях (по умолчанию: 1024, шаг: 8) |
| `compression` | INT | Да | 4 до 128 | Коэффициент сжатия, определяющий размеры скрытого пространства для этапа C (по умолчанию: 42, шаг: 1) |
| `batch_size` | INT | Нет | 1 до 4096 | Количество образцов скрытого пространства для генерации в одном пакете (по умолчанию: 1) |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `stage_c` | LATENT | Тензор скрытого пространства этапа C с размерами [batch_size, 16, height//compression, width//compression] |
| `stage_b` | LATENT | Тензор скрытого пространства этапа B с размерами [batch_size, 4, height//4, width//4] |
