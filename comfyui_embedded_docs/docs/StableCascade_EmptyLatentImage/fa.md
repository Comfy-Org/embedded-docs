گره `StableCascade_EmptyLatentImage` تنسورهای نهفته (latent) خالی برای مدل‌های Stable Cascade ایجاد می‌کند. این گره دو نمایش نهفته مجزا - یکی برای مرحله C و دیگری برای مرحله B - با ابعاد مناسب بر اساس وضوح ورودی و تنظیمات فشرده‌سازی تولید می‌کند. این گره نقطه شروع خط لوله تولید Stable Cascade را فراهم می‌کند.

## ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `width` | INT | بله | 256 تا MAX_RESOLUTION | عرض تصویر خروجی بر حسب پیکسل (پیش‌فرض: 1024، گام: 8) |
| `height` | INT | بله | 256 تا MAX_RESOLUTION | ارتفاع تصویر خروجی بر حسب پیکسل (پیش‌فرض: 1024، گام: 8) |
| `compression` | INT | بله | 4 تا 128 | ضریب فشرده‌سازی که ابعاد نهفته را برای مرحله C تعیین می‌کند (پیش‌فرض: 42، گام: 1) |
| `batch_size` | INT | خیر | 1 تا 4096 | تعداد نمونه‌های نهفته برای تولید در یک دسته (پیش‌فرض: 1) |

## خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-------------|-----------|-------------|
| `stage_c` | LATENT | تنسور نهفته مرحله C با ابعاد [batch_size, 16, height//compression, width//compression] |
| `stage_b` | LATENT | تنسور نهفته مرحله B با ابعاد [batch_size, 4, height//4, width//4] |
