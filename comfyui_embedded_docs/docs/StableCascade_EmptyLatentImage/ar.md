عقدة `StableCascade_EmptyLatentImage` تنشئ موترات كامنة فارغة لنماذج Stable Cascade. تُنتج تمثيلين كامنين منفصلين - أحدهما للمرحلة C والآخر للمرحلة B - بأبعاد مناسبة بناءً على دقة الإدخال وإعدادات الضغط. توفر هذه العقدة نقطة البداية لخط أنابيب توليد Stable Cascade.

## المدخلات

| المعامل | نوع البيانات | إجباري | النطاق | الوصف |
|-----------|-----------|----------|-------|-------------|
| `العرض` | INT | نعم | 256 إلى MAX_RESOLUTION | عرض الصورة الناتجة بالبكسل (الافتراضي: 1024، الخطوة: 8) |
| `الارتفاع` | INT | نعم | 256 إلى MAX_RESOLUTION | ارتفاع الصورة الناتجة بالبكسل (الافتراضي: 1024، الخطوة: 8) |
| `ضغط` | INT | نعم | 4 إلى 128 | عامل الضغط الذي يحدد الأبعاد الكامنة للمرحلة C (الافتراضي: 42، الخطوة: 1) |
| `حجم الدفعة` | INT | لا | 1 إلى 4096 | عدد العينات الكامنة المراد توليدها في الدفعة الواحدة (الافتراضي: 1) |
