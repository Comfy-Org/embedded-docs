Этот узел применяет Temporal Score Rescaling (TSR) к диффузионной модели. Он изменяет поведение сэмплирования модели путем перемасштабирования предсказанного шума или скора в процессе шумоподавления, что позволяет управлять разнообразием генерируемого результата. Реализован как пост-CFG (Classifier-Free Guidance) функция.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `модель` | MODEL | Да | - | Диффузионная модель, к которой применяется патч с функцией TSR. |
| `tsr_k` | FLOAT | Нет | 0.01 - 100.0 | Управляет силой перемасштабирования. Меньшие значения k дают более детализированные результаты; большие значения k дают более гладкие результаты при генерации изображений. Установка k = 1 отключает перемасштабирование. (по умолчанию: 0.95) |
| `tsr_sigma` | FLOAT | Нет | 0.01 - 100.0 | Управляет тем, насколько рано начинает действовать перемасштабирование. Большие значения начинают действовать раньше. (по умолчанию: 1.0) |

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `patched_model` | MODEL | Входная модель, к процессу сэмплирования которой теперь применен патч с функцией Temporal Score Rescaling. |
