# ترجمة وثيقة عقدة Painter في ComfyUI

عقدة Painter توفر لوحة رسم تفاعلية لإنشاء أو تعديل الصور والقناعات مباشرة داخل ComfyUI. تتيح لك البدء بلوحة فارغة أو صورة موجودة، والرسم عليها باستخدام أداة فرشاة، وتُخرج كلاً من الصورة الناتجة وقناع ألفا المطابق. يحدد القناع المناطق المرسومة، والتي يتم تركيبها بعد ذلك فوق الصورة الأساسية أو لون الخلفية.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|---------|---------------|--------|---------|-------|
| `الصورة` | IMAGE | لا | - | صورة أساسية اختيارية للرسم عليها. إذا لم يتم توفيرها، يتم إنشاء لوحة فارغة باستخدام لون الخلفية والعرض والارتفاع المحددين. |
| `mask` | STRING | نعم | - | بيانات الرسم، التي يتم إنشاؤها عادةً بواسطة الأداة التفاعلية المدمجة في العقدة. تتم إدارة هذا المعامل بواسطة أداة الرسم في واجهة المستخدم وليس المقصود توصيله بمقبس قياسي. |
| `العرض` | INT | نعم | 64 إلى 4096 | عرض اللوحة بالبكسل، يُستخدم عند عدم توفير صورة أساسية `الصورة`. يجب أن تكون القيمة من مضاعفات 64. القيمة الافتراضية هي 512. |
| `الارتفاع` | INT | نعم | 64 إلى 4096 | ارتفاع اللوحة بالبكسل، يُستخدم عند عدم توفير صورة أساسية `الصورة`. يجب أن تكون القيمة من مضاعفات 64. القيمة الافتراضية هي 512. |
| `لون الخلفية` | COLOR | نعم | - | لون خلفية اللوحة، محدد كرمز سداسي عشري (مثل #000000). يُستخدم هذا فقط عند عدم توفير صورة أساسية `الصورة`. اللون الافتراضي هو الأسود (#000000). |

**ملاحظة:** تم تصميم مدخل `mask` للعمل مع أداة واجهة المستخدم المتخصصة للعقدة. عند الرسم على اللوحة، تقوم الأداة تلقائياً بتعبئة هذه القيمة. يتم إخفاء مدخلي `width` و `height` في واجهة المستخدم القياسية ولكنهما يحددان أبعاد اللوحة عند إنشاء صورة جديدة.

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|------------|---------------|-------|
| `IMAGE` | IMAGE | الصورة النهائية المركبة. هذه هي نتيجة دمج المناطق المرسومة (من `mask`) فوق الصورة الأساسية `الصورة` المقدمة أو الخلفية الملونة. |
| `MASK` | MASK | قناع قناة ألفا (الشفافية) المستخرج من الرسم. تمثل المناطق البيضاء المناطق المرسومة، وتمثل المناطق السوداء الخلفية غير المعالجة. |
