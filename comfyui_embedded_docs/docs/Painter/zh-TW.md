# Painter 節點

Painter 節點提供了一個互動式畫布，讓您可以直接在 ComfyUI 中建立或編輯影像和遮罩。它允許您從空白畫布或現有影像開始，使用畫筆工具進行繪製，並輸出最終影像和相應的 Alpha 遮罩。該遮罩定義了繪製區域，然後這些區域會合成到基礎影像或背景顏色之上。

## 輸入

| 參數 | 資料類型 | 必要 | 範圍 | 說明 |
|-----------|-----------|----------|-------|-------------|
| `image` | IMAGE | 否 | - | 可選的基礎影像，用於在其上進行繪製。如果未提供，則使用指定的背景顏色、寬度和高度建立空白畫布。 |
| `mask` | STRING | 是 | - | 繪製資料，通常由節點的內建互動式小工具產生。此參數由 UI 的繪畫工具管理，不應連接到標準插座。 |
| `width` | INT | 是 | 64 至 4096 | 畫布的寬度（以像素為單位），在未提供基礎 `image` 時使用。該值必須是 64 的倍數。預設值為 512。 |
| `height` | INT | 是 | 64 至 4096 | 畫布的高度（以像素為單位），在未提供基礎 `image` 時使用。該值必須是 64 的倍數。預設值為 512。 |
| `bg_color` | COLOR | 是 | - | 畫布的背景顏色，以十六進位代碼指定（例如 #000000）。僅在未提供基礎 `image` 時使用。預設為黑色（#000000）。 |

**注意：** `mask` 輸入專為與節點的專用 UI 小工具配合使用而設計。當您在畫布上繪製時，小工具會自動填入此值。`width` 和 `height` 輸入在標準 UI 中隱藏，但在建立新影像時定義畫布尺寸。

## 輸出

| 輸出名稱 | 資料類型 | 說明 |
|-------------|-----------|-------------|
| `IMAGE` | IMAGE | 最終合成影像。這是將繪製區域（來自 `mask`）與提供的基礎 `image` 或彩色背景混合的結果。 |
| `MASK` | MASK | 從繪製中提取的 Alpha 通道（透明度）遮罩。白色區域代表繪製區域，黑色區域代表未觸及的背景。 |
