### مرور کلی

گره CropMask برای برش یک ناحیه مشخص از یک ماسک ورودی طراحی شده است. این گره به کاربران امکان می‌دهد ناحیه مورد نظر را با تعیین مختصات و ابعاد مشخص کنند و بخشی از ماسک را برای پردازش یا تحلیل بیشتر استخراج نمایند.

## ورودی‌ها

| پارامتر | نوع داده | توضیحات |
|-----------|-------------|-------------|
| `mask` | MASK | ورودی ماسک، تصویر ماسکی را نشان می‌دهد که قرار است برش داده شود. این ورودی برای تعیین ناحیه‌ای که بر اساس مختصات و ابعاد مشخص استخراج می‌شود، ضروری است. |
| `x` | INT | مختصات x نقطه شروع برش را روی محور افقی مشخص می‌کند. |
| `y` | INT | مختصات y نقطه شروع عملیات برش را روی محور عمودی تعیین می‌کند. |
| `width` | INT | عرض، گستردگی افقی ناحیه برش را از نقطه شروع مشخص می‌کند. |
| `height` | INT | ارتفاع، گستردگی عمودی ناحیه برش را از نقطه شروع تعیین می‌کند. |

## خروجی‌ها

| پارامتر | نوع داده | توضیحات |
|-----------|-------------|-------------|
| `mask` | MASK | خروجی یک ماسک برش‌خورده است که بخشی از ماسک اصلی بوده و توسط مختصات و ابعاد مشخص تعریف می‌شود. |
