Вот перевод документации на русский язык:

Узел ConditioningSetAreaPercentageVideo изменяет данные кондиционирования, определяя конкретную пространственную и временную область для генерации видео. Он позволяет задать положение, размер и длительность области, в которой будет применяться кондиционирование, используя процентные значения относительно общих размеров. Это полезно для фокусировки генерации на определенных частях видеопоследовательности.

## Входные параметры

| Параметр | Тип данных | Тип входа | По умолчанию | Диапазон | Описание |
|----------|------------|-----------|--------------|----------|----------|
| `conditioning` | CONDITIONING | Обязательный | - | - | Данные кондиционирования, которые необходимо изменить |
| `width` | FLOAT | Обязательный | 1.0 | 0.0 - 1.0 | Ширина области в процентах от общей ширины |
| `height` | FLOAT | Обязательный | 1.0 | 0.0 - 1.0 | Высота области в процентах от общей высоты |
| `temporal` | FLOAT | Обязательный | 1.0 | 0.0 - 1.0 | Временная длительность области в процентах от общей длины видео |
| `x` | FLOAT | Обязательный | 0.0 | 0.0 - 1.0 | Горизонтальная начальная позиция области в процентах |
| `y` | FLOAT | Обязательный | 0.0 | 0.0 - 1.0 | Вертикальная начальная позиция области в процентах |
| `z` | FLOAT | Обязательный | 0.0 | 0.0 - 1.0 | Временная начальная позиция области в процентах от временной шкалы видео |
| `strength` | FLOAT | Обязательный | 1.0 | 0.0 - 10.0 | Множитель силы, применяемый к кондиционированию в пределах заданной области |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `conditioning` | CONDITIONING | Измененные данные кондиционирования с примененными настройками области и силы |
