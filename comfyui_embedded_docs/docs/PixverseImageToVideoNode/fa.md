بر اساس تصویر ورودی و متن راهنما، ویدیو تولید می‌کند. این گره یک تصویر دریافت کرده و با اعمال تنظیمات حرکت و کیفیت مشخص‌شده، آن را به یک دنباله متحرک تبدیل می‌کند.

## ورودی‌ها

| پارامتر | نوع داده | اجباری | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `تصویر` | IMAGE | بله | - | تصویر ورودی برای تبدیل به ویدیو |
| `پرامپت` | STRING | بله | - | راهنمای متن برای تولید ویدیو |
| `کیفیت` | COMBO | بله | `res_540p`<br>`res_1080p` | تنظیم کیفیت ویدیو (پیش‌فرض: res_540p) |
| `مدت (ثانیه)` | COMBO | بله | `dur_2`<br>`dur_5`<br>`dur_10` | مدت زمان ویدیوی تولید شده بر حسب ثانیه |
| `حالت حرکت` | COMBO | بله | `normal`<br>`fast`<br>`slow`<br>`zoom_in`<br>`zoom_out`<br>`pan_left`<br>`pan_right`<br>`pan_up`<br>`pan_down`<br>`tilt_up`<br>`tilt_down`<br>`roll_clockwise`<br>`roll_counterclockwise` | سبک حرکتی اعمال شده بر تولید ویدیو |
| `بذر` | INT | بله | 0-2147483647 | دانه (seed) برای تولید ویدیو (پیش‌فرض: 0) |
| `پرامپت منفی` | STRING | خیر | - | توضیح متنی اختیاری از عناصر نامطلوب در تصویر |
| `قالب PixVerse` | CUSTOM | خیر | - | یک الگوی اختیاری برای تأثیرگذاری بر سبک تولید، ایجاد شده توسط گره PixVerse Template |

**توجه:** هنگام استفاده از کیفیت 1080p، حالت حرکت به طور خودکار روی normal تنظیم می‌شود و مدت زمان به ۵ ثانیه محدود می‌گردد. برای مدت‌های زمانی غیر از ۵ ثانیه نیز حالت حرکت به طور خودکار روی normal تنظیم می‌شود.
