Узел EmptyLTXVLatentVideo создает пустой латентный тензор для обработки видео. Он генерирует пустую начальную точку с заданными размерами, которая может использоваться в качестве входных данных для рабочих процессов генерации видео. Узел создает заполненное нулями латентное представление с заданной шириной, высотой, длиной и размером пакета.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `width` | INT | Да | от 64 до MAX_RESOLUTION | Ширина латентного видео-тензора (по умолчанию: 768, шаг: 32) |
| `height` | INT | Да | от 64 до MAX_RESOLUTION | Высота латентного видео-тензора (по умолчанию: 512, шаг: 32) |
| `length` | INT | Да | от 1 до MAX_RESOLUTION | Количество кадров в латентном видео (по умолчанию: 97, шаг: 8) |
| `batch_size` | INT | Нет | от 1 до 4096 | Количество латентных видео для генерации в одном пакете (по умолчанию: 1) |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `samples` | LATENT | Сгенерированный пустой латентный тензор с нулевыми значениями в заданных размерах |
