Узел LatentRotate предназначен для поворота скрытых представлений изображений на заданные углы. Он абстрагирует сложность манипулирования скрытым пространством для достижения эффектов поворота, позволяя пользователям легко трансформировать изображения в скрытом пространстве генеративной модели.

## Входные параметры

| Параметр | Тип данных | Описание |
|-----------|-------------|-------------|
| `образцы` | `LATENT` | Параметр 'samples' представляет собой скрытые представления изображений, подлежащих повороту. Он имеет решающее значение для определения начальной точки операции поворота. |
| `вращение` | COMBO[STRING] | Параметр 'rotation' задает угол, на который должны быть повернуты скрытые изображения. Он напрямую влияет на ориентацию результирующих изображений. |

## Выходные параметры

| Параметр | Тип данных | Описание |
|-----------|-------------|-------------|
| `latent` | `LATENT` | Результатом является модифицированная версия входных скрытых представлений, повернутая на заданный угол. |
