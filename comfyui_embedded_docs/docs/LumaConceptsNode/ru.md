Содержит один или несколько концептов камеры для использования с узлами Luma Text to Video и Luma Image to Video. Этот узел позволяет выбрать до четырех концептов камеры и, при необходимости, объединить их с существующими цепочками концептов.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `concept1` | STRING | Да | Доступно несколько опций<br>Включает опцию "None" | Выбор первого концепта камеры из доступных концептов Luma |
| `concept2` | STRING | Да | Доступно несколько опций<br>Включает опцию "None" | Выбор второго концепта камеры из доступных концептов Luma |
| `concept3` | STRING | Да | Доступно несколько опций<br>Включает опцию "None" | Выбор третьего концепта камеры из доступных концептов Luma |
| `concept4` | STRING | Да | Доступно несколько опций<br>Включает опцию "None" | Выбор четвертого концепта камеры из доступных концептов Luma |
| `luma_concepts` | LUMA_CONCEPTS | Нет | Н/Д | Дополнительные концепты камеры для добавления к выбранным здесь |

**Примечание:** Все параметры концептов (`concept1` – `concept4`) могут быть установлены в значение "None", если вы не хотите использовать все четыре слота для концептов. Узел объединит любые переданные `luma_concepts` с выбранными концептами для создания комбинированной цепочки концептов.

## Выходные данные

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `luma_concepts` | LUMA_CONCEPTS | Комбинированная цепочка концептов камеры, содержащая все выбранные концепты |
