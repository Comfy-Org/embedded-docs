### نمای کلی

یک یا چند مفهوم دوربین را برای استفاده با گره‌های Luma Text to Video و Luma Image to Video نگهداری می‌کند. این گره به شما امکان می‌دهد تا حداکثر چهار مفهوم دوربین را انتخاب کرده و در صورت تمایل آن‌ها را با زنجیره‌های مفهومی موجود ترکیب کنید.

### ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|---------|----------|--------|--------|---------|
| `concept1` | STRING | بله | گزینه‌های متعدد موجود<br>شامل گزینه "None" | انتخاب اولین مفهوم دوربین از میان مفاهیم موجود Luma |
| `concept2` | STRING | بله | گزینه‌های متعدد موجود<br>شامل گزینه "None" | انتخاب دومین مفهوم دوربین از میان مفاهیم موجود Luma |
| `concept3` | STRING | بله | گزینه‌های متعدد موجود<br>شامل گزینه "None" | انتخاب سومین مفهوم دوربین از میان مفاهیم موجود Luma |
| `concept4` | STRING | بله | گزینه‌های متعدد موجود<br>شامل گزینه "None" | انتخاب چهارمین مفهوم دوربین از میان مفاهیم موجود Luma |
| `luma_concepts` | LUMA_CONCEPTS | خیر | نامشخص | مفاهیم دوربین اختیاری برای افزودن به مفاهیم انتخاب‌شده در اینجا |

**نکته:** تمام پارامترهای مفهوم (`concept1` تا `concept4`) در صورت عدم نیاز به استفاده از هر چهار جایگاه مفهوم، می‌توانند روی "None" تنظیم شوند. این گره هر `luma_concepts` ارائه‌شده را با مفاهیم انتخاب‌شده ادغام کرده و یک زنجیره مفهومی ترکیبی ایجاد می‌کند.
