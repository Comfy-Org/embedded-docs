Этот узел специализируется на объединении двух входных масок с помощью различных операций, таких как сложение, вычитание и логические операции, для создания новой, изменённой маски. Он абстрактно управляет манипуляциями с данными маски для достижения сложных эффектов маскирования, выступая в качестве ключевого компонента в рабочих процессах редактирования и обработки изображений на основе масок.

## Входы

| Параметр      | Тип данных | Описание                                                                                                                                      |
| ------------- | ------------ | ------------------------------------------------------------------------------------------------------------------------------------------------ |
| `destination` | MASK        | Основная маска, которая будет изменена в результате операции с исходной маской. Она играет центральную роль в составной операции, выступая в качестве основы для изменений. |
| `source`      | MASK        | Вторичная маска, которая будет использоваться вместе с целевой маской для выполнения указанной операции, влияя на итоговую выходную маску. |
| `x`           | INT         | Горизонтальное смещение, с которым исходная маска будет наложена на целевую маску, влияющее на позиционирование составного результата.       |
| `y`           | INT         | Вертикальное смещение, с которым исходная маска будет наложена на целевую маску, влияющее на позиционирование составного результата.         |
| `operation`   | COMBO[STRING]| Определяет тип операции, применяемой между целевой и исходной масками, например 'add' (сложение), 'subtract' (вычитание) или логические операции, определяя характер составного эффекта. |

## Выходы

| Параметр | Тип данных | Описание                                                                 |
| -------- | ------------ | ---------------------------------------------------------------------------- |
| `mask`   | MASK        | Результирующая маска после применения указанной операции между целевой и исходной масками, представляющая собой составной результат. |
