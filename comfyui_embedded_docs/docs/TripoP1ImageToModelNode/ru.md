## Обзор

Этот узел преобразует одно 2D-изображение в 3D-модель с помощью API Tripo P1. Он оптимизирован для создания низкополигональных сеток, готовых к использованию в играх.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `image` | IMAGE | Да | - | Входное изображение для преобразования в 3D-модель. |
| `output_mode` | DICT | Да | См. описание | Словарь, определяющий режим вывода и настройки качества. Этот параметр управляет типом создаваемой модели и качеством её текстуры. Доступные опции задаются вспомогательной функцией `_build_p1_output_mode` и включают настройки `texture_quality` (например, "standard", "high", "ultra") и `image_alignment`. |
| `enable_image_autofix` | BOOLEAN | Нет | True<br>False | Предварительная обработка входного изображения для улучшения качества генерации. (по умолчанию: False) |
| `face_limit` | INT | Нет | - | Ограничивает количество полигонов в сгенерированной сетке. Значение -1 означает отсутствие ограничений. (по умолчанию: -1) |
| `model_seed` | INT | Нет | - | Значение seed для воспроизводимой генерации модели. Если не указано, используется случайный seed. (по умолчанию: None) |
| `auto_size` | BOOLEAN | Нет | True<br>False | Автоматически определить оптимальный размер для сгенерированной модели. (по умолчанию: False) |
| `export_uv` | BOOLEAN | Нет | True<br>False | Экспортировать UV-координаты вместе с моделью. (по умолчанию: True) |
| `compress_geometry` | BOOLEAN | Нет | True<br>False | Сжать данные геометрии для уменьшения размера файла. (по умолчанию: False) |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `model_file` | STRING | Путь к файлу сгенерированной 3D-модели. Этот выход предоставляется только для обратной совместимости. |
| `model task_id` | MODEL_TASK_ID | Уникальный идентификатор задачи для запроса генерации модели. |
| `GLB` | FILE3DGLB | Сгенерированная 3D-модель в формате GLB. |
