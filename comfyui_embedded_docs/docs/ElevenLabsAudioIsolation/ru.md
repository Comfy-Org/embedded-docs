Вот перевод документации на русский язык:

Узел ElevenLabs Voice Isolation удаляет фоновый шум из аудиофайла, выделяя вокал или речь. Он отправляет аудио в API ElevenLabs для обработки и возвращает очищенный аудиофайл.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `audio` | AUDIO | Да | | Аудио для обработки с целью удаления фонового шума. |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `audio` | AUDIO | Обработанное аудио с удаленным фоновым шумом. |
