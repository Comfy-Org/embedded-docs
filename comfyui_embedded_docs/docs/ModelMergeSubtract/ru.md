Этот узел предназначен для продвинутых операций слияния моделей, в частности для вычитания параметров одной модели из другой на основе заданного множителя. Он позволяет настраивать поведение моделей, регулируя влияние параметров одной модели на другую, что облегчает создание новых гибридных моделей.

## Входы

| Параметр      | Тип данных | Описание |
|---------------|------------|-------------|
| `model1`      | `MODEL`    | Базовая модель, из которой будут вычитаться параметры. |
| `model2`      | `MODEL`    | Модель, параметры которой будут вычитаться из базовой модели. |
| `multiplier`  | `FLOAT`    | Значение с плавающей точкой, которое масштабирует эффект вычитания для параметров базовой модели. |

## Выходы

| Параметр | Тип данных | Описание |
|-----------|------------|-------------|
| `model`   | MODEL      | Результирующая модель после вычитания параметров одной модели из другой, масштабированных множителем. |
