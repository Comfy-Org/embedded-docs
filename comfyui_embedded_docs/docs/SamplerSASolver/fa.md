گره SamplerSASolver یک الگوریتم نمونه‌گیری سفارشی برای مدل‌های انتشار (diffusion models) پیاده‌سازی می‌کند. این گره از رویکرد پیش‌بینی‌کننده-تصحیح‌کننده با تنظیمات قابل‌پیکربندی مرتبه و پارامترهای معادله دیفرانسیل تصادفی (SDE) برای تولید نمونه‌ها از مدل ورودی استفاده می‌کند.

## ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|---------|----------|-------|--------|---------|
| `model` | MODEL | بله | - | مدل انتشار مورد استفاده برای نمونه‌گیری |
| `eta` | FLOAT | خیر | 0.0 - 10.0 | ضریب مقیاس‌دهی اندازه گام را کنترل می‌کند (پیش‌فرض: 1.0) |
| `sde_start_percent` | FLOAT | خیر | 0.0 - 1.0 | درصد شروع برای نمونه‌گیری SDE (پیش‌فرض: 0.2) |
| `sde_end_percent` | FLOAT | خیر | 0.0 - 1.0 | درصد پایان برای نمونه‌گیری SDE (پیش‌فرض: 0.8) |
| `s_noise` | FLOAT | خیر | 0.0 - 100.0 | میزان نویز اضافه‌شده در طول نمونه‌گیری را کنترل می‌کند (پیش‌فرض: 1.0) |
| `predictor_order` | INT | خیر | 1 - 6 | مرتبه مؤلفه پیش‌بینی‌کننده در حل‌کننده (پیش‌فرض: 3) |
| `corrector_order` | INT | خیر | 0 - 6 | مرتبه مؤلفه تصحیح‌کننده در حل‌کننده (پیش‌فرض: 4) |
| `use_pece` | BOOLEAN | خیر | - | روش PECE (پیش‌بینی-ارزیابی-تصحیح-ارزیابی) را فعال یا غیرفعال می‌کند |
| `simple_order_2` | BOOLEAN | خیر | - | محاسبات ساده‌شده مرتبه دوم را فعال یا غیرفعال می‌کند |

## خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-----------|----------|---------|
| `sampler` | SAMPLER | یک شیء نمونه‌گیر پیکربندی‌شده که می‌تواند با مدل‌های انتشار استفاده شود |
