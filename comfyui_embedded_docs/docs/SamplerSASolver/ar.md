عقدة SamplerSASolver تنفذ خوارزمية أخذ عينات مخصصة لنماذج الانتشار. تستخدم نهجًا قائمًا على المتنبئ-المصحح مع إعدادات ترتيب قابلة للتكوين ومعاملات المعادلة التفاضلية العشوائية (SDE) لتوليد عينات من النموذج المُدخل.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|-----------|-----------|----------|-------|-------------|
| `النموذج` | MODEL | نعم | - | نموذج الانتشار المستخدم في أخذ العينات |
| `إيتا` | FLOAT | لا | 0.0 - 10.0 | يتحكم في عامل تحجيم حجم الخطوة (القيمة الافتراضية: 1.0) |
| `نسبة بدء SDE` | FLOAT | لا | 0.0 - 1.0 | النسبة المئوية لبدء أخذ العينات باستخدام المعادلة التفاضلية العشوائية (القيمة الافتراضية: 0.2) |
| `نسبة انتهاء SDE` | FLOAT | لا | 0.0 - 1.0 | النسبة المئوية لإنهاء أخذ العينات باستخدام المعادلة التفاضلية العشوائية (القيمة الافتراضية: 0.8) |
| `الضوضاء s` | FLOAT | لا | 0.0 - 100.0 | يتحكم في مقدار الضوضاء المضافة أثناء أخذ العينات (القيمة الافتراضية: 1.0) |
| `ترتيب المتنبئ` | INT | لا | 1 - 6 | ترتيب مكون المتنبئ في الحلال (القيمة الافتراضية: 3) |
| `ترتيب المصحح` | INT | لا | 0 - 6 | ترتيب مكون المصحح في الحلال (القيمة الافتراضية: 4) |
| `استخدام PECE` | BOOLEAN | لا | - | يُفعّل أو يُعطّل طريقة PECE (تنبؤ-تقييم-تصحيح-تقييم) |
| `الترتيب البسيط 2` | BOOLEAN | لا | - | يُفعّل أو يُعطّل حسابات الترتيب الثاني المبسّطة |

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|-------------|-----------|-------------|
| `sampler` | SAMPLER | كائن أخذ عينات مُهيأ يمكن استخدامه مع نماذج الانتشار |
