Вот перевод документации на русский язык:

Узел SamplerSASolver реализует пользовательский алгоритм сэмплирования для диффузионных моделей. Он использует подход предиктор-корректор с настраиваемыми параметрами порядка и параметрами стохастического дифференциального уравнения (SDE) для генерации сэмплов из входной модели.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `model` | MODEL | Да | - | Диффузионная модель, используемая для сэмплирования |
| `eta` | FLOAT | Нет | 0.0 - 10.0 | Управляет коэффициентом масштабирования шага (по умолчанию: 1.0) |
| `sde_start_percent` | FLOAT | Нет | 0.0 - 1.0 | Начальный процент для SDE-сэмплирования (по умолчанию: 0.2) |
| `sde_end_percent` | FLOAT | Нет | 0.0 - 1.0 | Конечный процент для SDE-сэмплирования (по умолчанию: 0.8) |
| `s_noise` | FLOAT | Нет | 0.0 - 100.0 | Управляет количеством шума, добавляемого во время сэмплирования (по умолчанию: 1.0) |
| `predictor_order` | INT | Нет | 1 - 6 | Порядок компонента предиктора в решателе (по умолчанию: 3) |
| `corrector_order` | INT | Нет | 0 - 6 | Порядок компонента корректора в решателе (по умолчанию: 4) |
| `use_pece` | BOOLEAN | Нет | - | Включает или отключает метод PECE (Predict-Evaluate-Correct-Evaluate) |
| `simple_order_2` | BOOLEAN | Нет | - | Включает или отключает упрощенные вычисления второго порядка |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `sampler` | SAMPLER | Настроенный объект сэмплера, который можно использовать с диффузионными моделями |
