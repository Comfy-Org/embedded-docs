گره WanPhantomSubjectToVideo با پردازش ورودی‌های شرطی‌سازی (conditioning) و تصاویر مرجع اختیاری، محتوای ویدئویی تولید می‌کند. این گره بازنمایی‌های نهفته (latent) برای تولید ویدئو ایجاد کرده و در صورت ارائه تصاویر ورودی، می‌تواند از راهنمایی بصری آن‌ها استفاده کند. این گره داده‌های شرطی‌سازی را با الحاق زمانی (time-dimensional concatenation) برای مدل‌های ویدئویی آماده کرده و خروجی آن شامل شرطی‌سازی اصلاح‌شده و داده‌های ویدئویی نهفته تولیدشده است.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|---------|----------|--------|--------|---------|
| `positive` | CONDITIONING | بله | - | ورودی شرطی‌سازی مثبت برای هدایت تولید ویدئو |
| `negative` | CONDITIONING | بله | - | ورودی شرطی‌سازی منفی برای جلوگیری از ویژگی‌های خاص |
| `vae` | VAE | بله | - | مدل VAE برای رمزگذاری تصاویر در صورت ارائه |
| `width` | INT | بله | 16 تا MAX_RESOLUTION | عرض ویدئوی خروجی بر حسب پیکسل (پیش‌فرض: 832، باید بر 16 بخش‌پذیر باشد) |
| `height` | INT | بله | 16 تا MAX_RESOLUTION | ارتفاع ویدئوی خروجی بر حسب پیکسل (پیش‌فرض: 480، باید بر 16 بخش‌پذیر باشد) |
| `length` | INT | بله | 1 تا MAX_RESOLUTION | تعداد فریم‌های ویدئوی تولیدشده (پیش‌فرض: 81، باید بر 4 بخش‌پذیر باشد) |
| `batch_size` | INT | بله | 1 تا 4096 | تعداد ویدئوهایی که هم‌زمان تولید می‌شوند (پیش‌فرض: 1) |
| `images` | IMAGE | خیر | - | تصاویر مرجع اختیاری برای شرطی‌سازی زمانی |

**نکته:** در صورت ارائه `images`، آن‌ها به‌طور خودکار به اندازه `width` و `height` مشخص‌شده بزرگ‌نمایی (upscale) می‌شوند و تنها `length` فریم اول برای پردازش استفاده می‌گردد.

## خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-----------|----------|---------|
