Вот перевод документации на русский язык:

Узел WanPhantomSubjectToVideo генерирует видеоконтент путем обработки входных данных conditioning и опциональных эталонных изображений. Он создает латентные представления для генерации видео и может использовать визуальное руководство от входных изображений, если они предоставлены. Узел подготавливает данные conditioning с временной конкатенацией для видеомоделей и выводит модифицированный conditioning вместе с сгенерированными латентными видеоданными.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `positive` | CONDITIONING | Да | - | Положительный conditioning для направления генерации видео |
| `negative` | CONDITIONING | Да | - | Отрицательный conditioning для избегания определенных характеристик |
| `vae` | VAE | Да | - | Модель VAE для кодирования изображений, если они предоставлены |
| `width` | INT | Да | от 16 до MAX_RESOLUTION | Ширина выходного видео в пикселях (по умолчанию: 832, должна делиться на 16) |
| `height` | INT | Да | от 16 до MAX_RESOLUTION | Высота выходного видео в пикселях (по умолчанию: 480, должна делиться на 16) |
| `length` | INT | Да | от 1 до MAX_RESOLUTION | Количество кадров в сгенерированном видео (по умолчанию: 81, должно делиться на 4) |
| `batch_size` | INT | Да | от 1 до 4096 | Количество видео для одновременной генерации (по умолчанию: 1) |
| `images` | IMAGE | Нет | - | Опциональные эталонные изображения для conditioning с временной размерностью |

**Примечание:** Если `images` предоставлены, они автоматически масштабируются до указанных `width` и `height`, и для обработки используются только первые `length` кадров.

## Выходные данные

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `positive` | CONDITIONING | Модифицированный положительный conditioning с временной конкатенацией, если предоставлены изображения |
| `negative_text` | CONDITIONING | Модифицированный отрицательный conditioning с временной конкатенацией, если предоставлены изображения |
| `negative_img_text` | CONDITIONING | Отрицательный conditioning с обнуленной временной конкатенацией, если предоставлены изображения |
| `latent` | LATENT | Сгенерированное латентное представление видео с указанными размерами и длиной |
