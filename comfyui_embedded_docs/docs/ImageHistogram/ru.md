Вот перевод документации на русский язык:

Узел ImageHistogram анализирует цветовое распределение входного изображения. Он вычисляет и выводит несколько гистограмм — графиков, показывающих, сколько пикселей изображения имеют каждое возможное значение интенсивности. Узел генерирует отдельные гистограммы для красного, зеленого и синего цветовых каналов, составную RGB-гистограмму, а также гистограмму яркости, основанную на стандартной формуле яркости.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `изображение` | IMAGE | Да | Н/Д | Входное изображение для анализа. Узел обрабатывает первое изображение в пакете. |

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `яркость` | HISTOGRAM | Составная гистограмма, представляющая среднюю интенсивность пикселей по красному, зеленому и синему каналам. |
| `красный` | HISTOGRAM | Гистограмма воспринимаемой яркости изображения, рассчитанная с использованием стандартной формулы яркости ITU-R BT.709. |
| `зелёный` | HISTOGRAM | Гистограмма, показывающая распределение интенсивностей пикселей в красном цветовом канале. |
| `синий` | HISTOGRAM | Гистограмма, показывающая распределение интенсивностей пикселей в зеленом цветовом канале. |
| `blue` | HISTOGRAM | Гистограмма, показывающая распределение интенсивностей пикселей в синем цветовом канале. |
