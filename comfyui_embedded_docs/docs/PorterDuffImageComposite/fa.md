گره PorterDuffImageComposite برای ترکیب تصاویر با استفاده از عملگرهای ترکیب Porter-Duff طراحی شده است. این گره امکان ترکیب تصاویر مبدأ و مقصد را با توجه به حالت‌های مختلف ترکیب فراهم می‌کند و با دستکاری شفافیت تصاویر و روی هم قرار دادن آن‌ها به روش‌های خلاقانه، ایجاد جلوه‌های بصری پیچیده را ممکن می‌سازد.

## ورودی‌ها

| پارامتر | نوع داده | توضیحات |
| --------- | ------------ | ----------- |
| `source`  | `IMAGE`     | تنسور تصویر مبدأ که قرار است روی تصویر مقصد ترکیب شود. این پارامتر نقش مهمی در تعیین نتیجه نهایی بصری بر اساس حالت ترکیب انتخاب‌شده دارد. |
| `source_alpha` | `MASK` | کانال آلفای تصویر مبدأ که شفافیت هر پیکسل در تصویر مبدأ را مشخص می‌کند. این پارامتر بر نحوه ترکیب تصویر مبدأ با تصویر مقصد تأثیر می‌گذارد. |
| `destination` | `IMAGE` | تنسور تصویر مقصد که به‌عنوان پس‌زمینه‌ای عمل می‌کند که تصویر مبدأ روی آن ترکیب می‌شود. این پارامتر بر اساس حالت ترکیب به تصویر نهایی ترکیب‌شده کمک می‌کند. |
| `destination_alpha` | `MASK` | کانال آلفای تصویر مقصد که شفافیت پیکسل‌های تصویر مقصد را تعریف می‌کند. این پارامتر بر ترکیب تصاویر مبدأ و مقصد تأثیر می‌گذارد. |
| `mode` | COMBO[STRING] | حالت ترکیب Porter-Duff که اعمال می‌شود و نحوه ترکیب تصاویر مبدأ و مقصد را تعیین می‌کند. هر حالت جلوه‌های بصری متفاوتی ایجاد می‌کند. |

## خروجی‌ها

| پارامتر | نوع داده | توضیحات |
| --------- | ------------ | ----------- |
| `image`   | `IMAGE`     | تصویر ترکیب‌شده حاصل از اعمال حالت Porter-Duff مشخص‌شده. |
| `mask`    | `MASK`      | کانال آلفای تصویر ترکیب‌شده که شفافیت هر پیکسل را نشان می‌دهد. |
