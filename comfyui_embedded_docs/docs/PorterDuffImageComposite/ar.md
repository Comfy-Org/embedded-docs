عُقدة PorterDuffImageComposite مصممة لأداء تركيب الصور باستخدام عوامل تركيب Porter-Duff. تتيح هذه العقدة دمج صور المصدر والوجهة وفقًا لأنماط مزج متنوعة، مما يتيح إنشاء تأثيرات بصرية معقدة من خلال التحكم في شفافية الصور وتراكبها بطرق إبداعية.

## المدخلات

| المعامل | نوع البيانات | الوصف |
| --------- | ------------ | ----------- |
| `المصدر`  | `IMAGE`     | موتر الصورة المصدرية المراد تركيبها فوق الصورة الوجهة. يلعب دورًا حاسمًا في تحديد النتيجة البصرية النهائية بناءً على نمط التركيب المحدد. |
| `ألفا المصدر` | `MASK` | قناة ألفا للصورة المصدرية، والتي تحدد شفافية كل بكسل في الصورة المصدرية. تؤثر على كيفية مزج الصورة المصدرية مع الصورة الوجهة. |
| `الوجهة` | `IMAGE` | موتر الصورة الوجهة التي تعمل كخلفية يتم تركيب الصورة المصدرية فوقها. تساهم في الصورة النهائية المركبة بناءً على نمط المزج. |
| `ألفا الوجهة` | `MASK` | قناة ألفا للصورة الوجهة، والتي تحدد شفافية بكسلات الصورة الوجهة. تؤثر على مزج الصورتين المصدرية والوجهة. |
| `الوضع` | COMBO[STRING] | نمط تركيب Porter-Duff المطلوب تطبيقه، والذي يحدد كيفية مزج الصورتين المصدرية والوجهة معًا. ينتج كل نمط تأثيرات بصرية مختلفة. |

## المخرجات

| المعامل | نوع البيانات | الوصف |
| --------- | ------------ | ----------- |
| `image`   | `IMAGE`     | الصورة المركبة الناتجة عن تطبيق نمط Porter-Duff المحدد. |
| `mask`    | `MASK`      | قناة ألفا للصورة المركبة، والتي تشير إلى شفافية كل بكسل. |
