Этот узел создает персональный ассет изображения для сервиса Seedance 2.0 от ByteDance. Он загружает входное изображение и регистрирует его в указанной группе ассетов. Если идентификатор группы не указан, в вашем браузере будет запущен процесс проверки реального человека для создания новой группы перед добавлением ассета.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `image` | IMAGE | Да | | Изображение, которое будет зарегистрировано как персональный ассет. |
| `group_id` | STRING | Нет | | Повторное использование существующего идентификатора группы ассетов Seedance для пропуска повторной проверки человека для одного и того же лица. Оставьте пустым для запуска проверки реального человека в браузере и создания новой группы (по умолчанию: пусто). |

**Ограничения изображения:**
*   Ширина изображения должна быть от 300 до 6000 пикселей.
*   Высота изображения должна быть от 300 до 6000 пикселей.
*   Соотношение сторон изображения должно быть от 0,4:1 до 2,5:1.

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `asset_id` | STRING | Уникальный идентификатор созданного ассета изображения. |
| `group_id` | STRING | Идентификатор группы ассетов. Это будет указанный `group_id` или вновь созданный. |
