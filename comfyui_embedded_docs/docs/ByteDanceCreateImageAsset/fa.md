این گره یک دارایی تصویر شخصی برای سرویس Seedance 2.0 شرکت ByteDance ایجاد می‌کند. یک تصویر ورودی را بارگذاری کرده و آن را در یک گروه دارایی مشخص ثبت می‌کند. اگر شناسه گروهی ارائه نشود، فرایند تأیید هویت شخص واقعی را در مرورگر شما آغاز کرده و قبل از افزودن دارایی، یک گروه جدید ایجاد می‌کند.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|---------|----------|--------|--------|---------|
| `image` | IMAGE | بله | | تصویری که به عنوان دارایی شخصی ثبت می‌شود. |
| `group_id` | STRING | خیر | | استفاده مجدد از شناسه گروه دارایی Seedance موجود برای جلوگیری از تأیید هویت تکراری برای همان شخص. برای اجرای تأیید هویت واقعی در مرورگر و ایجاد گروه جدید، این فیلد را خالی بگذارید (پیش‌فرض: خالی). |

**محدودیت‌های تصویر:**
*   عرض تصویر باید بین ۳۰۰ تا ۶۰۰۰ پیکسل باشد.
*   ارتفاع تصویر باید بین ۳۰۰ تا ۶۰۰۰ پیکسل باشد.
*   نسبت ابعاد تصویر باید بین ۰.۴:۱ تا ۲.۵:۱ باشد.

## خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-----------|----------|---------|
| `asset_id` | STRING | شناسه یکتای دارایی تصویر تازه ایجاد شده. |
| `group_id` | STRING | شناسه گروه دارایی. این مقدار همان `group_id` ارائه شده یا یک شناسه تازه ایجاد شده خواهد بود. |
