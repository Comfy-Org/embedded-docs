# ترجمة وثيقة عقدة Meshy: Refine Draft Model

عقدة **Meshy: Refine Draft Model** تأخذ نموذجًا ثلاثي الأبعاد أوليًا تم إنشاؤه مسبقًا وتحسّن جودته، مع إمكانية إضافة القوام (النسيج). تقوم بإرسال مهمة تحسين إلى واجهة Meshy API وتعيد ملفات النموذج ثلاثي الأبعاد النهائية بمجرد اكتمال المعالجة.

## المدخلات

| المعامل | نوع البيانات | إجباري | النطاق | الوصف |
|---------|--------------|--------|--------|-------|
| `model` | COMBO | نعم | `"latest"` | يحدد نموذج الذكاء الاصطناعي المستخدم للتحسين. حاليًا، يتوفر نموذج "latest" فقط. |
| `meshy_task_id` | MESHY_TASK_ID | نعم | - | معرف المهمة الفريد للنموذج الأولي الذي ترغب في تحسينه. |
| `enable_pbr` | BOOLEAN | لا | - | إنشاء خرائط PBR (معدنية، خشونة، عادية) بالإضافة إلى اللون الأساسي. ملاحظة: يجب ضبط هذا على false عند استخدام نمط النحت (Sculpture)، حيث يولد نمط النحت مجموعته الخاصة من خرائط PBR. (الافتراضي: `False`) |
| `texture_prompt` | STRING | لا | - | تقديم موجه نصي لتوجيه عملية إضافة القوام. الحد الأقصى 600 حرف. لا يمكن استخدامه في نفس الوقت مع `texture_image`. (الافتراضي: سلسلة فارغة) |
| `texture_image` | IMAGE | لا | - | يمكن استخدام واحد فقط من `texture_image` أو `texture_prompt` في نفس الوقت. |

**ملاحظة:** مدخلات `texture_prompt` و `texture_image` متنافية. لا يمكنك تقديم كل من موجه نصي وصورة لإضافة القوام في نفس العملية.

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|------------|--------------|--------|
| `meshy_task_id` | STRING | اسم ملف النموذج المُنشأ بصيغة GLB. (للتوصيل العكسي فقط) |
| `GLB` | MESHY_TASK_ID | معرف المهمة الفريد لمهمة التحسين المُرسلة. |
| `FBX` | FILE3DGLB | النموذج ثلاثي الأبعاد النهائي المُحسّن بصيغة GLB. |
| `FBX` | FILE3DFBX | النموذج ثلاثي الأبعاد النهائي المُحسّن بصيغة FBX. |
