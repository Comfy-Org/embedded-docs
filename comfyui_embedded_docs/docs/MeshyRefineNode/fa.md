### گره Meshy: Refine Draft Model

گره «Meshy: Refine Draft Model» یک مدل سه‌بعدی پیش‌نویس که پیش‌تر تولید شده را دریافت کرده و کیفیت آن را بهبود می‌بخشد و به‌صورت اختیاری بافت‌سازی را نیز انجام می‌دهد. این گره یک وظیفه بهینه‌سازی را به API مسهی ارسال کرده و پس از اتمام پردازش، فایل‌های نهایی مدل سه‌بعدی را بازمی‌گرداند.

## ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `model` | COMBO | بله | `"latest"` | مدل هوش مصنوعی مورد استفاده برای بهینه‌سازی را مشخص می‌کند. در حال حاضر، تنها مدل "latest" در دسترس است. |
| `meshy_task_id` | MESHY_TASK_ID | بله | - | شناسه یکتای وظیفه مدل پیش‌نویسی که می‌خواهید بهینه کنید. |
| `enable_pbr` | BOOLEAN | خیر | - | تولید نقشه‌های PBR (فلزی، زبری، نرمال) به‌علاوه رنگ پایه. توجه: هنگام استفاده از سبک Sculpture باید روی false تنظیم شود، زیرا سبک Sculpture مجموعه نقشه‌های PBR خود را تولید می‌کند. (پیش‌فرض: `False`) |
| `texture_prompt` | STRING | خیر | - | یک提示 متنی برای هدایت فرآیند بافت‌سازی ارائه دهید. حداکثر ۶۰۰ کاراکتر. نمی‌توان همزمان با `texture_image` استفاده کرد. (پیش‌فرض: رشته خالی) |
| `texture_image` | IMAGE | خیر | - | تنها یکی از `texture_image` یا `texture_prompt` می‌تواند همزمان استفاده شود. |

**نکته:** ورودی‌های `texture_prompt` و `texture_image` متقابلاً انحصاری هستند. نمی‌توانید همزمان یک提示 متنی و یک تصویر برای بافت‌سازی در یک عملیات ارائه دهید.

## خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-------------|-----------|-------------|
| `model_file` | STRING | نام فایل مدل GLB تولید شده. (فقط برای سازگاری با نسخه‌های قبلی) |
| `meshy_task_id` | MESHY_TASK_ID | شناسه یکتای وظیفه برای کار بهینه‌سازی ارسال شده. |
| `GLB` | FILE3DGLB | مدل سه‌بعدی نهایی بهینه‌سازی شده در قالب GLB. |
| `FBX` | FILE3DFBX | مدل سه‌بعدی نهایی بهینه‌سازی شده در قالب FBX. |
