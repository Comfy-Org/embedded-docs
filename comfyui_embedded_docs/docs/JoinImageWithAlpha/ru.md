Этот узел предназначен для операций композитинга, а именно для объединения изображения с соответствующим альфа-каналом (маской прозрачности) для получения единого выходного изображения. Он эффективно объединяет визуальный контент с информацией о прозрачности, позволяя создавать изображения, в которых определенные области являются прозрачными или полупрозрачными.

## Входные параметры

| Параметр | Тип данных | Описание |
|-----------|-------------|-------------|
| `изображение`   | `IMAGE`     | Основное визуальное содержимое, которое будет объединено с альфа-маской. Представляет собой изображение без информации о прозрачности. |
| `альфа`   | `MASK`      | Альфа-маска, определяющая прозрачность соответствующего изображения. Используется для определения того, какие части изображения должны быть прозрачными или полупрозрачными. |

## Выходные параметры

| Параметр | Тип данных | Описание |
|-----------|-------------|-------------|
| `изображение`   | `IMAGE`     | Выходные данные представляют собой единое изображение, которое объединяет входное изображение с альфа-маской, добавляя информацию о прозрачности к визуальному содержимому. |
