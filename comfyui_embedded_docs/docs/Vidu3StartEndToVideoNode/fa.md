این گره با درون‌یابی (interpolation) بین یک فریم شروع و یک فریم پایان، تحت راهنمایی یک توضیح متنی (prompt)، یک ویدیو تولید می‌کند. این گره از مدل Vidu Q3 برای ایجاد یک انتقال بی‌درز بین دو تصویر استفاده کرده و ویدیویی با مدت‌زمان و وضوح مشخص تولید می‌کند.

## ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|---------|----------|-------|--------|---------|
| `model` | COMBO | بله | `"viduq3-pro"`<br>`"viduq3-turbo"` | مدل مورد استفاده برای تولید ویدیو. انتخاب یک گزینه، پارامترهای پیکربندی اضافی برای `resolution`، `duration` و `audio` را آشکار می‌کند. |
| `model.resolution` | COMBO | بله | `"720p"`<br>`"1080p"` | وضوح ویدیوی خروجی. این پارامتر پس از انتخاب یک `model` آشکار می‌شود. |
| `model.duration` | INT | بله | 1 تا 16 | مدت‌زمان ویدیوی خروجی بر حسب ثانیه (پیش‌فرض: 5). این پارامتر پس از انتخاب یک `model` آشکار می‌شود. |
| `model.audio` | BOOLEAN | بله | `True` / `False` | در صورت فعال بودن، ویدیو با صدا (شامل دیالوگ و افکت‌های صوتی) خروجی می‌دهد (پیش‌فرض: False). این پارامتر پس از انتخاب یک `model` آشکار می‌شود. |
| `first_frame` | IMAGE | بله | - | تصویر شروع برای دنباله ویدیو. |
| `end_frame` | IMAGE | بله | - | تصویر پایان برای دنباله ویدیو. |
| `prompt` | STRING | بله | - | یک توضیح متنی که تولید ویدیو را هدایت می‌کند (حداکثر 2000 کاراکتر). |
| `seed` | INT | خیر | 0 تا 2147483647 | یک مقدار دانه (seed) برای کنترل تصادفی بودن تولید (پیش‌فرض: 1). |

**نکته:** تصاویر `first_frame` و `end_frame` باید نسبت ابعاد مشابهی داشته باشند تا بهترین نتیجه حاصل شود. نسبت ابعاد دو تصویر باید بین 80% تا 125% از یکدیگر باشد (نزدیکی نسبی بین 0.8 و 1.25).
