Вот перевод документации на русский язык:

Узел Get Video Components извлекает все основные элементы из видеофайла. Он разделяет видео на отдельные кадры, извлекает аудиодорожку и предоставляет информацию о частоте кадров видео. Это позволяет работать с каждым компонентом независимо для дальнейшей обработки или анализа.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `видео` | VIDEO | Да | - | Видео, из которого необходимо извлечь компоненты. |

## Выходы

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `аудио` | IMAGE | Отдельные кадры, извлеченные из видео в виде отдельных изображений. |
| `fps` | AUDIO | Аудиодорожка, извлеченная из видео. |
| `fps` | FLOAT | Частота кадров видео в кадрах в секунду. |
