تقوم عقدة StringConcatenate بدمج سلسلتين نصيتين في سلسلة واحدة عن طريق ضمهما باستخدام فاصل محدد. تأخذ العقدة سلسلتين نصيتين للإدخال وحرفًا أو سلسلة فاصلة، ثم تُخرج سلسلة واحدة حيث يتم ربط المدخلين مع وضع الفاصل بينهما.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|-----------|-----------|----------|-------|-------------|
| `السلسلة_أ` | STRING | نعم | - | السلسلة النصية الأولى المراد دمجها |
| `السلسلة_ب` | STRING | نعم | - | السلسلة النصية الثانية المراد دمجها |
| `الفاصل` | STRING | لا | - | الحرف أو السلسلة النصية المراد إدراجها بين السلسلتين النصيتين للإدخال (الافتراضي: سلسلة فارغة) |

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|-------------|-----------|-------------|
| `output` | STRING | السلسلة النصية المدمجة مع إدراج الفاصل بين string_a و string_b |
