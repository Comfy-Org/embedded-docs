گره `StringConcatenate` دو رشته متنی را با استفاده از یک جداکننده مشخص به یکدیگر متصل می‌کند. این گره دو رشته ورودی و یک کاراکتر یا رشته جداکننده دریافت کرده و یک رشته خروجی تولید می‌کند که در آن دو ورودی با قرار گرفتن جداکننده بین آن‌ها به هم متصل شده‌اند.

## ورودی‌ها

| پارامتر | نوع داده | اجباری | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `رشته الف` | STRING | بله | - | اولین رشته متنی برای الحاق |
| `رشته ب` | STRING | بله | - | دومین رشته متنی برای الحاق |
| `جداکننده` | STRING | خیر | - | کاراکتر یا رشته‌ای که بین دو رشته ورودی قرار می‌گیرد (پیش‌فرض: رشته خالی) |

## خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-------------|-----------|-------------|
| `output` | STRING | رشته ترکیبی که در آن جداکننده بین string_a و string_b قرار گرفته است |
