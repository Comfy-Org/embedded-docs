Вот перевод документации на русский язык:

Узел ImageScaleToMaxDimension изменяет размер изображений, чтобы они помещались в заданный максимальный размер, сохраняя исходное соотношение сторон. Он определяет, является ли изображение портретным или альбомным, затем масштабирует большую сторону до целевого размера, пропорционально изменяя меньшую сторону. Узел поддерживает несколько методов масштабирования для различных требований к качеству и производительности.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `изображение` | IMAGE | Да | - | Входное изображение для масштабирования |
| `метод увеличения` | STRING | Да | "area"<br>"lanczos"<br>"bilinear"<br>"nearest-exact"<br>"bilinear"<br>"bicubic" | Метод интерполяции, используемый для масштабирования изображения (по умолчанию: "area") |
| `максимальный размер` | INT | Да | от 0 до 16384 | Максимальный размер масштабированного изображения (по умолчанию: 512) |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `изображение` | IMAGE | Масштабированное изображение, наибольшая сторона которого соответствует заданному размеру |
