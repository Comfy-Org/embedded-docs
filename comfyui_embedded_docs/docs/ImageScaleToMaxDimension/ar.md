# ترجمة وثيقة ImageScaleToMaxDimension

عقدة ImageScaleToMaxDimension تقوم بتغيير حجم الصور لتناسب بُعدًا أقصى محدد مع الحفاظ على نسبة العرض إلى الارتفاع الأصلية. تقوم بحساب ما إذا كانت الصورة باتجاه عمودي (بورتريه) أو أفقي (لاندسكيب)، ثم تقوم بقياس البُعد الأكبر ليتطابق مع الحجم المستهدف مع تعديل البُعد الأصغر بشكل متناسب. تدعم العقدة طرقًا متعددة للرفع لدقة الصورة لتلبية متطلبات الجودة والأداء المختلفة.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|-----------|-----------|----------|-------|-------------|
| `image` | IMAGE | نعم | - | الصورة المدخلة المراد تغيير حجمها |
| `upscale_method` | STRING | نعم | "area"<br>"lanczos"<br>"bilinear"<br>"nearest-exact"<br>"bilinear"<br>"bicubic" | طريقة الاستيفاء المستخدمة لقياس الصورة (الافتراضي: "area") |
| `largest_size` | INT | نعم | 0 إلى 16384 | البُعد الأقصى للصورة بعد القياس (الافتراضي: 512) |

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|-------------|-----------|-------------|
| `image` | IMAGE | الصورة بعد القياس مع تطابق البُعد الأكبر مع الحجم المحدد |
