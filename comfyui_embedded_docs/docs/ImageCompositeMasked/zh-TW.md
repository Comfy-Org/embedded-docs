`ImageCompositeMasked` 節點專為影像合成而設計，允許將來源影像疊加到目標影像的指定座標上，並可選擇調整大小及使用遮罩。

## 輸入

| 參數 | 資料類型 | 說明 |
|-----------|-------------|-------------|
| `目標圖像` | `IMAGE` | 目標影像，來源影像將合成至此影像上。此影像作為合成操作的背景。 |
| `來源圖像` | `IMAGE` | 要合成到目標影像上的來源影像。此影像可選擇調整大小以符合目標影像的尺寸。 |
| `X 座標` | `INT` | 來源影像左上角在目標影像中放置的 x 座標。 |
| `Y 座標` | `INT` | 來源影像左上角在目標影像中放置的 y 座標。 |
| `調整來源大小` | `BOOLEAN` | 一個布林值旗標，指示是否應將來源影像調整大小以符合目標影像的尺寸。 |
| `遮罩` | `MASK` | 一個可選的遮罩，用於指定應將來源影像的哪些部分合成到目標影像上。這允許進行更複雜的合成操作，例如混合或部分疊加。 |

## 輸出

| 參數 | 資料類型 | 說明 |
|-----------|-------------|-------------|
| `image` | `IMAGE` | 合成操作後產生的結果影像，結合了來源影像和目標影像的元素。 |
