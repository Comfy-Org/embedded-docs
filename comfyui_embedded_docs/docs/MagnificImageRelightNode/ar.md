بالتأكيد، إليك الترجمة العربية للوثيقة مع الالتزام التام بقواعد الترجمة المحددة:

عقدة إضاءة الصورة المحسّنة (Magnific Image Relight) تقوم بضبط إضاءة صورة مُدخلة. يمكنها تطبيق إضاءة أسلوبية بناءً على نص وصفي أو نقل خصائص الإضاءة من صورة مرجعية اختيارية. توفر العقدة عناصر تحكم متنوعة لضبط السطوع والتباين والمزاج العام للصورة النهائية بدقة.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|-----------|-----------|----------|-------|-------------|
| `image` | IMAGE | نعم | غير متاح | الصورة المراد إعادة إضاءتها. مطلوب صورة واحدة بالضبط. الحد الأدنى للأبعاد هو 160×160 بكسل. يجب أن تكون نسبة العرض إلى الارتفاع بين 1:3 و 3:1. |
| `prompt` | STRING | لا | غير متاح | توجيه وصفي للإضاءة. يدعم تدوين التأكيد (1-1.4). القيمة الافتراضية هي سلسلة نصية فارغة. |
| `light_transfer_strength` | INT | نعم | 0 إلى 100 | شدة تطبيق نقل الإضاءة. القيمة الافتراضية: 100. |
| `style` | COMBO | نعم | `"standard"`<br>`"darker_but_realistic"`<br>`"clean"`<br>`"smooth"`<br>`"brighter"`<br>`"contrasted_n_hdr"`<br>`"just_composition"` | تفضيل النمط الناتج. |
| `interpolate_from_original` | BOOLEAN | نعم | غير متاح | يقيد حرية التوليد ليتطابق بشكل أكبر مع الصورة الأصلية. القيمة الافتراضية: خطأ. |
| `change_background` | BOOLEAN | نعم | غير متاح | يعدل الخلفية بناءً على النص الوصفي/الصورة المرجعية. القيمة الافتراضية: صحيح. |
| `preserve_details` | BOOLEAN | نعم | غير متاح | يحافظ على الملمس والتفاصيل الدقيقة من الصورة الأصلية. القيمة الافتراضية: صحيح. |
| `advanced_settings` | DYNAMICCOMBO | نعم | `"disabled"`<br>`"enabled"` | خيارات الضبط الدقيق للتحكم المتقدم في الإضاءة. عند تعيينها على `"enabled"`، تصبح المعاملات الإضافية متاحة. |
| `reference_image` | IMAGE | لا | غير متاح | صورة مرجعية اختيارية لنقل الإضاءة منها. إذا تم توفيرها، يلزم صورة واحدة بالضبط. الحد الأدنى للأبعاد هو 160×160 بكسل. يجب أن تكون نسبة العرض إلى الارتفاع بين 1:3 و 3:1. |

**ملاحظة حول الإعدادات المتقدمة:** عندما يتم تعيين `advanced_settings` على `"enabled"`، تصبح المعاملات المتداخلة التالية نشطة:

* `whites`: يضبط أفتح درجات الألوان في الصورة. النطاق: 0 إلى 100. القيمة الافتراضية: 50.
* `blacks`: يضبط أغمق درجات الألوان في الصورة. النطاق: 0 إلى 100. القيمة الافتراضية: 50.
* `brightness`: ضبط السطوع الكلي. النطاق: 0 إلى 100. القيمة الافتراضية: 50.
* `contrast`: ضبط التباين. النطاق: 0 إلى 100. القيمة الافتراضية: 50.
* `saturation`: ضبط تشبع اللون. النطاق: 0 إلى 100. القيمة الافتراضية: 50.
* `engine`: اختيار محرك المعالجة. الخيارات: `"automatic"`, `"balanced"`, `"cool"`, `"real"`, `"illusio"`, `"fairy"`, `"colorful_anime"`, `"hard_transform"`, `"softy"`.
* `transfer_light_a`: شدة نقل الإضاءة. الخيارات: `"automatic"`, `"low"`, `"medium"`, `"normal"`, `"high"`, `"high_on_faces"`.
* `transfer_light_b`: يعدل أيضًا شدة نقل الإضاءة. يمكن دمجه مع عنصر التحكم السابق لتأثيرات متنوعة. الخيارات: `"automatic"`, `"composition"`, `"straight"`, `"smooth_in"`, `"smooth_out"`, `"smooth_both"`, `"reverse_both"`, `"soft_in"`, `"soft_out"`, `"soft_mid"`, `"style_shift"`, `"strong_shift"`.
* `fixed_generation`: يضمن مخرجات متسقة مع نفس الإعدادات. القيمة الافتراضية: صحيح.
