### نمای کلی

گره **Magnific Image Relight** نوردهی یک تصویر ورودی را تنظیم می‌کند. این گره می‌تواند نورپردازی سبک‌مند را بر اساس یک توضیح متنی اعمال کند یا ویژگی‌های نوردهی را از یک تصویر مرجع اختیاری انتقال دهد. این گره کنترل‌های متنوعی برای تنظیم دقیق روشنایی، کنتراست و حال‌وهوای کلی خروجی نهایی ارائه می‌دهد.

## ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|---------|----------|-------|--------|---------|
| `تصویر` | IMAGE | بله | نامشخص | تصویری که باید نوردهی آن تنظیم شود. دقیقاً یک تصویر مورد نیاز است. حداقل ابعاد 160×160 پیکسل. نسبت تصویر باید بین 1:3 و 3:1 باشد. |
| `پرامپت` | STRING | خیر | نامشخص | راهنمای توصیفی برای نورپردازی. از نماد تأکید (1-1.4) پشتیبانی می‌کند. پیش‌فرض: رشته خالی. |
| `شدت_انتقال_نور` | INT | بله | 0 تا 100 | شدت اعمال انتقال نور. پیش‌فرض: 100. |
| `سبک` | COMBO | بله | `"standard"`<br>`"darker_but_realistic"`<br>`"clean"`<br>`"smooth"`<br>`"brighter"`<br>`"contrasted_n_hdr"`<br>`"just_composition"` | اولویت سبک خروجی. |
| `درون‌یابی_از_مبدأ` | BOOLEAN | بله | نامشخص | آزادی تولید را برای تطابق بیشتر با تصویر اصلی محدود می‌کند. پیش‌فرض: False. |
| `تغییر_پس‌زمینه` | BOOLEAN | بله | نامشخص | پس‌زمینه را بر اساس توضیح/تصویر مرجع تغییر می‌دهد. پیش‌فرض: True. |
| `حفظ_جزئیات` | BOOLEAN | بله | نامشخص | بافت و جزئیات دقیق تصویر اصلی را حفظ می‌کند. پیش‌فرض: True. |
| `تنظیمات_پیشرفته` | DYNAMICCOMBO | بله | `"disabled"`<br>`"enabled"` | گزینه‌های تنظیم دقیق برای کنترل پیشرفته نورپردازی. وقتی روی `"enabled"` تنظیم شود، پارامترهای اضافی در دسترس قرار می‌گیرند. |
| `تصویر_مرجع` | IMAGE | خیر | نامشخص | تصویر مرجع اختیاری برای انتقال نور از آن. در صورت ارائه، دقیقاً یک تصویر مورد نیاز است. حداقل ابعاد 160×160 پیکسل. نسبت تصویر باید بین 1:3 و 3:1 باشد. |

**نکته درباره تنظیمات پیشرفته:** وقتی `advanced_settings` روی `"enabled"` تنظیم شود، پارامترهای تو در توی زیر فعال می‌شوند:

* `whites`: تنظیم روشن‌ترین تن‌های تصویر. محدوده: 0 تا 100. پیش‌فرض: 50.
* `blacks`: تنظیم تاریک‌ترین تن‌های تصویر. محدوده: 0 تا 100. پیش‌فرض: 50.
* `brightness`: تنظیم روشنایی کلی. محدوده: 0 تا 100. پیش‌فرض: 50.
* `contrast`: تنظیم کنتراست. محدوده: 0 تا 100. پیش‌فرض: 50.
* `saturation`: تنظیم اشباع رنگ. محدوده: 0 تا 100. پیش‌فرض: 50.
* `engine`: انتخاب موتور پردازش. گزینه‌ها: `"automatic"`, `"balanced"`, `"cool"`, `"real"`, `"illusio"`, `"fairy"`, `"colorful_anime"`, `"hard_transform"`, `"softy"`.
* `transfer_light_a`: شدت انتقال نور. گزینه‌ها: `"automatic"`, `"low"`, `"medium"`, `"normal"`, `"high"`, `"high_on_faces"`.
* `transfer_light_b`: همچنین شدت انتقال نور را تغییر می‌دهد. می‌تواند با کنترل قبلی برای جلوه‌های متنوع ترکیب شود. گزینه‌ها: `"automatic"`, `"composition"`, `"straight"`, `"smooth_in"`, `"smooth_out"`, `"smooth_both"`, `"reverse_both"`, `"soft_in"`, `"soft_out"`, `"soft_mid"`, `"style_shift"`, `"strong_shift"`.
* `fixed_generation`: خروجی یکسان با تنظیمات مشابه را تضمین می‌کند. پیش‌فرض: True.
