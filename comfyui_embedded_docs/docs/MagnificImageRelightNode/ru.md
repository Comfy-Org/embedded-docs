Вот перевод документации на русский язык:

Узел Magnific Image Relight изменяет освещение входного изображения. Он может применять стилистическое освещение на основе текстовой подсказки или переносить характеристики освещения из дополнительного эталонного изображения. Узел предлагает различные элементы управления для точной настройки яркости, контрастности и общего настроения конечного результата.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `image` | IMAGE | Да | Н/Д | Изображение для изменения освещения. Требуется ровно одно изображение. Минимальные размеры: 160x160 пикселей. Соотношение сторон должно быть от 1:3 до 3:1. |
| `prompt` | STRING | Нет | Н/Д | Описательное указание для освещения. Поддерживает нотацию выделения (1-1.4). По умолчанию — пустая строка. |
| `light_transfer_strength` | INT | Да | от 0 до 100 | Интенсивность применения переноса освещения. По умолчанию: 100. |
| `style` | COMBO | Да | `"standard"`<br>`"darker_but_realistic"`<br>`"clean"`<br>`"smooth"`<br>`"brighter"`<br>`"contrasted_n_hdr"`<br>`"just_composition"` | Предпочтение стилистического результата. |
| `interpolate_from_original` | BOOLEAN | Да | Н/Д | Ограничивает свободу генерации для более точного соответствия оригиналу. По умолчанию: False. |
| `change_background` | BOOLEAN | Да | Н/Д | Изменяет фон в соответствии с подсказкой/эталоном. По умолчанию: True. |
| `preserve_details` | BOOLEAN | Да | Н/Д | Сохраняет текстуру и мелкие детали оригинала. По умолчанию: True. |
| `advanced_settings` | DYNAMICCOMBO | Да | `"disabled"`<br>`"enabled"` | Опции тонкой настройки для расширенного управления освещением. При установке значения `"enabled"` становятся доступны дополнительные параметры. |
| `reference_image` | IMAGE | Нет | Н/Д | Дополнительное эталонное изображение для переноса освещения. Если указано, требуется ровно одно изображение. Минимальные размеры: 160x160 пикселей. Соотношение сторон должно быть от 1:3 до 3:1. |

**Примечание о расширенных настройках:** Когда `advanced_settings` установлен в `"enabled"`, становятся активными следующие вложенные параметры:

* `whites`: Регулирует самые светлые тона изображения. Диапазон: от 0 до 100. По умолчанию: 50.
* `blacks`: Регулирует самые темные тона изображения. Диапазон: от 0 до 100. По умолчанию: 50.
* `brightness`: Общая регулировка яркости. Диапазон: от 0 до 100. По умолчанию: 50.
* `contrast`: Регулировка контрастности. Диапазон: от 0 до 100. По умолчанию: 50.
* `saturation`: Регулировка насыщенности цвета. Диапазон: от 0 до 100. По умолчанию: 50.
* `engine`: Выбор обрабатывающего движка. Варианты: `"automatic"`, `"balanced"`, `"cool"`, `"real"`, `"illusio"`, `"fairy"`, `"colorful_anime"`, `"hard_transform"`, `"softy"`.
* `transfer_light_a`: Интенсивность переноса света. Варианты: `"automatic"`, `"low"`, `"medium"`, `"normal"`, `"high"`, `"high_on_faces"`.
* `transfer_light_b`: Также изменяет интенсивность переноса света. Может комбинироваться с предыдущим элементом управления для различных эффектов. Варианты: `"automatic"`, `"composition"`, `"straight"`, `"smooth_in"`, `"smooth_out"`, `"smooth_both"`, `"reverse_both"`, `"soft_in"`, `"soft_out"`, `"soft_mid"`, `"style_shift"`, `"strong_shift"`.
* `fixed_generation`: Обеспечивает согласованный результат при одинаковых настройках. По умолчанию: True.

| Имя выхода | Тип данных | Описание |

| `image` | IMAGE | Изображение с измененным освещением. |
