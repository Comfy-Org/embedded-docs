Узел LatentUpscaleModelLoader загружает специализированную модель, предназначенную для увеличения разрешения латентных представлений. Он считывает файл модели из назначенной системой папки и автоматически определяет её тип (720p, 1080p или другой), чтобы создать и настроить правильную внутреннюю архитектуру модели. Загруженная модель затем готова к использованию другими узлами для задач супер-разрешения в латентном пространстве.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `model_name` | STRING | Да | *Все файлы в папке `latent_upscale_models`* | Имя файла модели латентного увеличения разрешения для загрузки. Доступные варианты динамически формируются из файлов, находящихся в вашей директории `latent_upscale_models` ComfyUI. |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `model` | LATENT_UPSCALE_MODEL | Загруженная модель латентного увеличения разрешения, настроенная и готовая к использованию. |
