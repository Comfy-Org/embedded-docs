## عقدة تحويل النص إلى فيديو من Wan

تقوم عقدة "تحويل النص إلى فيديو من Wan" بإنشاء محتوى فيديو بناءً على أوصاف نصية. تستخدم هذه العقدة نماذج ذكاء اصطناعي لإنشاء مقاطع فيديو من التعليمات النصية، وتدعم أحجام فيديو متنوعة، ومدد زمنية مختلفة، وإدخال صوتي اختياري. يمكن للعقدة إنشاء صوت تلقائياً عند الحاجة، وتوفر خيارات لتحسين التعليمات النصية وإضافة علامة مائية.

## المدخلات

| المعامل | نوع البيانات | إجباري | النطاق | الوصف |
|-----------|-----------|----------|-------|-------------|
| `model` | COMBO | نعم | "wan2.5-t2v-preview"<br>"wan2.6-t2v" | النموذج المراد استخدامه (الافتراضي: "wan2.6-t2v") |
| `prompt` | STRING | نعم | - | تعليمات نصية تصف العناصر والخصائص البصرية. يدعم اللغتين الإنجليزية والصينية (الافتراضي: "") |
| `negative_prompt` | STRING | لا | - | تعليمات نصية سلبية تصف ما يجب تجنبه (الافتراضي: "") |
| `size` | COMBO | لا | "480p: 1:1 (624x624)"<br>"480p: 16:9 (832x480)"<br>"480p: 9:16 (480x832)"<br>"720p: 1:1 (960x960)"<br>"720p: 16:9 (1280x720)"<br>"720p: 9:16 (720x1280)"<br>"720p: 4:3 (1088x832)"<br>"720p: 3:4 (832x1088)"<br>"1080p: 1:1 (1440x1440)"<br>"1080p: 16:9 (1920x1080)"<br>"1080p: 9:16 (1080x1920)"<br>"1080p: 4:3 (1632x1248)"<br>"1080p: 3:4 (1248x1632)" | دقة الفيديو ونسبة العرض إلى الارتفاع (الافتراضي: "720p: 1:1 (960x960)") |
| `duration` | INT | لا | 5-15 (بخطوات مقدارها 5) | مدة الفيديو بالثواني. المدة 15 ثانية متاحة فقط لنموذج Wan 2.6 (الافتراضي: 5) |
| `audio` | AUDIO | لا | - | يجب أن يحتوي الصوت على صوت واضح وقوي، دون ضوضاء غريبة أو موسيقى خلفية |
| `seed` | INT | لا | 0-2147483647 | البذرة المستخدمة في التوليد (الافتراضي: 0) |
| `generate_audio` | BOOLEAN | لا | - | إذا لم يتم توفير إدخال صوتي، فسيتم إنشاء الصوت تلقائياً (الافتراضي: False) |
| `prompt_extend` | BOOLEAN | لا | - | ما إذا كان سيتم تحسين التعليمات النصية بمساعدة الذكاء الاصطناعي (الافتراضي: True) |
| `watermark` | BOOLEAN | لا | - | ما إذا كان سيتم إضافة علامة مائية من إنشاء الذكاء الاصطناعي إلى النتيجة (الافتراضي: False) |
| `shot_type` | COMBO | لا | "single"<br>"multi" | يحدد نوع اللقطة للفيديو المُنشأ، أي ما إذا كان الفيديو لقطة واحدة متصلة أم لقطات متعددة مع فواصل. هذا المعامل يؤثر فقط عندما يكون prompt_extend مساوياً لـ True (الافتراضي: "single") |

**ملاحظة:** نموذج Wan 2.6 لا يدعم دقة 480p. المدة 15 ثانية مدعومة فقط بواسطة نموذج Wan 2.6. عند تقديم إدخال صوتي، يجب أن تتراوح مدته بين 3.0 و 29.0 ثانية وأن يحتوي على صوت واضح دون ضوضاء خلفية أو موسيقى.
