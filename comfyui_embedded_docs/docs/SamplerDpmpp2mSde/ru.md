Этот узел предназначен для создания сэмплера для модели DPMPP_2M_SDE, позволяя генерировать образцы на основе заданных типов решателя, уровней шума и предпочтений по вычислительному устройству. Он абстрагирует сложности настройки сэмплера, предоставляя упрощённый интерфейс для генерации образцов с пользовательскими параметрами.

## Входные данные

| Параметр        | Тип данных    | Описание                                                                 |
|-----------------|---------------|--------------------------------------------------------------------------|
| `solver_type`   | COMBO[STRING] | Определяет тип решателя, используемого в процессе сэмплирования, предлагая варианты между 'midpoint' и 'heun'. Этот выбор влияет на метод численного интегрирования, применяемый во время сэмплирования. |
| `eta`           | `FLOAT`       | Определяет размер шага в численном интегрировании, влияя на детализацию процесса сэмплирования. Более высокое значение указывает на больший размер шага. |
| `s_noise`       | `FLOAT`       | Управляет уровнем шума, вносимого в процессе сэмплирования, влияя на вариативность генерируемых образцов. |
| `noise_device`  | COMBO[STRING] | Указывает вычислительное устройство ('gpu' или 'cpu'), на котором выполняется процесс генерации шума, влияя на производительность и эффективность. |

## Выходные данные

| Параметр        | Тип данных    | Описание                                                                 |
|-----------------|---------------|--------------------------------------------------------------------------|
| `sampler`       | `SAMPLER`     | На выходе получается сэмплер, настроенный в соответствии с указанными параметрами и готовый к генерации образцов. |
