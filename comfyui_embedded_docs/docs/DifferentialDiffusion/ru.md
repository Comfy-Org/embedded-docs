Вот перевод документации на русский язык:

Узел Differential Diffusion изменяет процесс шумоподавления, применяя бинарную маску на основе пороговых значений временных шагов. Он создает маску, которая смешивает исходную маску шумоподавления с бинарной маской на основе порога, что позволяет контролируемо регулировать силу процесса диффузии.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `модель` | MODEL | Да | - | Модель диффузии для модификации |
| `интенсивность` | FLOAT | Нет | 0.0 - 1.0 | Управляет силой смешивания между исходной маской шумоподавления и бинарной пороговой маской (по умолчанию: 1.0) |

## Выходные данные

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `модель` | MODEL | Модифицированная модель диффузии с обновленной функцией маски шумоподавления |
