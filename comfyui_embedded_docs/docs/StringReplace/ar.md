بالتأكيد، إليك الترجمة العربية للوثيقة التقنية الخاصة بعقدة ComfyUI مع الالتزام التام بقواعد الترجمة المحددة:

عقدة استبدال النصوص (StringReplace) تقوم بعمليات استبدال نصية على سلاسل الإدخال النصية. تبحث عن سلسلة فرعية محددة داخل النص المُدخل وتستبدل جميع تكراراتها بسلسلة فرعية مختلفة. تُرجع هذه العقدة السلسلة النصية المعدلة مع تطبيق جميع عمليات الاستبدال.

## المدخلات

| المعامل | نوع البيانات | إجباري | النطاق | الوصف |
|-----------|-----------|----------|-------|-------------|
| `سلسلة نصية` | STRING | نعم | - | السلسلة النصية المُدخلة التي سيتم إجراء عمليات الاستبدال عليها |
| `بحث` | STRING | نعم | - | السلسلة الفرعية المراد البحث عنها داخل النص المُدخل |
| `استبدال` | STRING | نعم | - | النص البديل الذي سيحل محل جميع التكرارات التي تم العثور عليها |

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|-------------|-----------|-------------|
| `output` | STRING | السلسلة النصية المعدلة مع استبدال جميع تكرارات النص المطلوب البحث عنه بالنص البديل |
