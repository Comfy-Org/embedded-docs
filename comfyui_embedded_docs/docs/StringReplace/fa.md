گره StringReplace عملیات جایگزینی متن را روی رشته‌های ورودی انجام می‌دهد. این گره یک زیررشته مشخص را درون متن ورودی جستجو کرده و تمام موارد یافت‌شده را با یک زیررشته دیگر جایگزین می‌کند. این گره رشته اصلاح‌شده را با تمام جایگزینی‌های اعمال‌شده بازمی‌گرداند.

## ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|---------|----------|-------|--------|---------|
| `رشته` | STRING | بله | - | رشته متنی ورودی که عملیات جایگزینی روی آن انجام می‌شود |
| `یافتن` | STRING | بله | - | زیررشته‌ای که باید درون متن ورودی جستجو شود |
| `جایگزین` | STRING | بله | - | متن جایگزین که جای تمام موارد یافت‌شده را می‌گیرد |

## خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-----------|----------|---------|
| `output` | STRING | رشته اصلاح‌شده که در آن تمام موارد متن جستجو با متن جایگزین تعویض شده است |
