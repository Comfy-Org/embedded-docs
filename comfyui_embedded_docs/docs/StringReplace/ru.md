Узел StringReplace выполняет операции замены текста во входных строках. Он ищет указанную подстроку во входном тексте и заменяет все её вхождения на другую подстроку. Этот узел возвращает изменённую строку со всеми выполненными заменами.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `строка` | STRING | Да | - | Входная текстовая строка, в которой будет выполняться замена |
| `найти` | STRING | Да | - | Подстрока для поиска во входном тексте |
| `заменить` | STRING | Да | - | Текст замены, который будет подставлен вместо всех найденных вхождений |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `output` | STRING | Изменённая строка, в которой все вхождения искомого текста заменены на текст замены |
