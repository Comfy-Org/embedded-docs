# ترجمة وثيقة عقدة SD_4XUpscale_Conditioning

تقوم عقدة SD_4XUpscale_Conditioning بإعداد بيانات التكييف لرفع دقة الصور باستخدام نماذج الانتشار. تأخذ هذه العقدة الصور المدخلة وبيانات التكييف، ثم تطبق عمليات القياس وإضافة الضوضاء لإنشاء تكييف معدل يوجه عملية رفع الدقة. تُخرج العقدة كلاً من التكييف الإيجابي والسلبي بالإضافة إلى التمثيلات الكامنة للأبعاد المرفوعة الدقة.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|---------|---------------|--------|--------|-------|
| `الصور` | IMAGE | نعم | - | الصور المدخلة المراد رفع دقتها |
| `إيجابي` | CONDITIONING | نعم | - | بيانات التكييف الإيجابي التي توجه التوليد نحو المحتوى المرغوب |
| `سلبي` | CONDITIONING | نعم | - | بيانات التكييف السلبي التي تبعد التوليد عن المحتوى غير المرغوب |
| `نسبة المقياس` | FLOAT | لا | 0.0 - 10.0 | عامل القياس المطبق على الصور المدخلة (القيمة الافتراضية: 4.0) |
| `تعزيز الضجيج` | FLOAT | لا | 0.0 - 1.0 | مقدار الضوضاء المراد إضافتها أثناء عملية رفع الدقة (القيمة الافتراضية: 0.0) |

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|-------------|---------------|-------|
| `سلبي` | CONDITIONING | التكييف الإيجابي المعدل مع تطبيق معلومات رفع الدقة |
| `كامن` | CONDITIONING | التكييف السلبي المعدل مع تطبيق معلومات رفع الدقة |
| `latent` | LATENT | تمثيل كامن فارغ يطابق الأبعاد المرفوعة الدقة |
