گره «تصویر به ویدیوی Runway (Gen4 Turbo)» با استفاده از مدل Gen4 Turbo شرکت Runway، یک ویدیو از یک فریم شروع واحد تولید می‌کند. این گره یک متن راهنما (prompt) و یک فریم تصویر اولیه دریافت کرده و سپس بر اساس مدت‌زمان و نسبت ابعاد تعیین‌شده، یک دنباله ویدیویی ایجاد می‌کند. این گره وظیفه آپلود فریم شروع در API Runway را بر عهده داشته و ویدیوی تولیدشده را بازمی‌گرداند.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `پرامپت` | STRING | بله | - | متن راهنما برای تولید (پیش‌فرض: رشته خالی) |
| `فریم شروع` | IMAGE | بله | - | فریم شروع برای استفاده در ویدیو |
| `مدت زمان` | COMBO | بله | `"5"`<br>`"10"` | مدت‌زمان ویدیو بر حسب ثانیه (پیش‌فرض: "5") |
| `نسبت تصویر` | COMBO | بله | `"1024:1024"`<br>`"1280:720"`<br>`"720:1280"`<br>`"1920:1080"`<br>`"1080:1920"`<br>`"2048:1080"`<br>`"1080:2048"` | نسبت ابعاد برای ویدیوی تولیدشده (پیش‌فرض: "1024:1024") |
| `seed` | INT | خیر | 0 تا 4294967295 | دانه تصادفی برای تولید (پیش‌فرض: 0) |

**محدودیت‌های پارامتر:**

- تصویر `start_frame` باید ابعادی بیشتر از 7999x7999 پیکسل نداشته باشد
- تصویر `start_frame` باید نسبت ابعادی بین 0.5 و 2.0 داشته باشد
- `prompt` باید حداقل شامل یک کاراکتر باشد
