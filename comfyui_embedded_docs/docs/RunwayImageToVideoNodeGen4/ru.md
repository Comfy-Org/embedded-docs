Узел "Runway Image to Video (Gen4 Turbo)" генерирует видео из одного начального кадра с помощью модели Gen4 Turbo от Runway. Он принимает текстовый запрос и начальное изображение, затем создает видеопоследовательность на основе заданных параметров длительности и соотношения сторон. Узел обрабатывает загрузку начального кадра в API Runway и возвращает сгенерированное видео.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `prompt` | STRING | Да | - | Текстовый запрос для генерации (по умолчанию: пустая строка) |
| `start_frame` | IMAGE | Да | - | Начальный кадр, используемый для видео |
| `duration` | COMBO | Да | `"5"`<br>`"10"` | Длительность видео в секундах (по умолчанию: "5") |
| `ratio` | COMBO | Да | `"1024:1024"`<br>`"1280:720"`<br>`"720:1280"`<br>`"1920:1080"`<br>`"1080:1920"`<br>`"2048:1080"`<br>`"1080:2048"` | Соотношение сторон для генерируемого видео (по умолчанию: "1024:1024") |
| `seed` | INT | Нет | от 0 до 4294967295 | Случайное зерно для генерации (по умолчанию: 0) |

**Ограничения параметров:**

- Изображение `start_frame` должно иметь размеры, не превышающие 7999x7999 пикселей
- Изображение `start_frame` должно иметь соотношение сторон от 0.5 до 2.0
- Параметр `prompt` должен содержать хотя бы один символ

## Выходные данные

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `output` | VIDEO | Сгенерированное видео на основе входного кадра и запроса |
