TextEncodeQwenImageEdit 節點處理文字提示和可選的圖像，以生成用於圖像生成或編輯的調校資料。它使用 CLIP 模型對輸入進行標記化，並可選擇性地使用 VAE 對參考圖像進行編碼以創建參考潛在特徵。當提供圖像時，它會自動調整圖像大小以保持一致的處理尺寸。

## 輸入參數

| 參數名稱 | 資料類型 | 必填 | 數值範圍 | 參數說明 |
|-----------|-----------|----------|-------|-------------|
| `clip` | CLIP | 是 | - | 用於文字和圖像標記化的 CLIP 模型 |
| `prompt` | STRING | 是 | - | 用於調校生成的文字提示，支援多行輸入和動態提示 |
| `vae` | VAE | 否 | - | 用於將參考圖像編碼為潛在特徵的可選 VAE 模型 |
| `image` | IMAGE | 否 | - | 用於參考或編輯目的的可選輸入圖像 |

**注意：** 當同時提供 `image` 和 `vae` 時，節點會將圖像編碼為參考潛在特徵並將其附加到調校輸出中。圖像會自動調整大小以維持約 1024x1024 像素的一致處理尺度。
