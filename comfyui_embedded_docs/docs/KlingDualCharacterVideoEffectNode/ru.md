Вот перевод документации на русский язык:

Узел Kling Dual Character Video Effect создает видео со спецэффектами на основе выбранной сцены. Он принимает два изображения и размещает первое изображение в левой части, а второе — в правой части итогового видео. В зависимости от выбранной сцены эффекта применяются различные визуальные эффекты.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `image_left` | IMAGE | Да | - | Изображение для левой стороны |
| `image_right` | IMAGE | Да | - | Изображение для правой стороны |
| `effect_scene` | COMBO | Да | `"chat"`<br>`"dance"`<br>`"hug"`<br>`"kill"`<br>`"kiss"`<br>`"pat"`<br>`"punch"`<br>`"shrug"`<br>`"slap"`<br>`"tickle"` | Тип сцены спецэффекта, применяемого при генерации видео |
| `model_name` | COMBO | Нет | `"kling-v1"`<br>`"kling-v1-5"`<br>`"kling-v1-6"` | Модель, используемая для эффектов персонажей (по умолчанию: "kling-v1") |
| `mode` | COMBO | Нет | `"std"`<br>`"pro"` | Режим генерации видео (по умолчанию: "std") |
| `duration` | COMBO | Да | `"5"`<br>`"10"` | Длительность генерируемого видео в секундах |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `output` | VIDEO | Сгенерированное видео с эффектами двух персонажей |
| `duration` | STRING | Информация о длительности сгенерированного видео |
