Этот узел использует модель Kling AI для генерации видео на основе текстового запроса и до семи референсных изображений. Он позволяет управлять соотношением сторон, длительностью, разрешением видео, а также опционально использовать раскадровки или генерировать аудио. Узел отправляет запрос во внешний API и возвращает сгенерированное видео.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `model_name` | COMBO | Да | `"kling-v3-omni"`<br>`"kling-video-o1"` | Конкретная модель Kling для генерации видео (по умолчанию: "kling-v3-omni"). |
| `prompt` | STRING | Да | - | Текстовый запрос, описывающий содержимое видео. Может включать как положительные, так и отрицательные описания. Текст автоматически нормализуется и должен содержать от 1 до 2500 символов. Игнорируется, если включены раскадровки. |
| `aspect_ratio` | COMBO | Да | `"16:9"`<br>`"9:16"`<br>`"1:1"` | Желаемое соотношение сторон для генерируемого видео. |
| `duration` | INT | Да | от 3 до 15 | Длительность видео в секундах. Значение можно регулировать ползунком (по умолчанию: 5). |
| `reference_images` | IMAGE | Да | - | До 7 референсных изображений. Каждое изображение должно быть не менее 300x300 пикселей и иметь соотношение сторон от 1:2.5 до 2.5:1. |
| `resolution` | COMBO | Нет | `"4k"`<br>`"1080p"`<br>`"720p"` | Выходное разрешение видео. Этот параметр необязателен (по умолчанию: "1080p"). |
| `storyboards` | DYNAMIC_COMBO | Нет | `"disabled"`<br>`"1 storyboard"`<br>`"2 storyboards"`<br>`"3 storyboards"`<br>`"4 storyboards"`<br>`"5 storyboards"`<br>`"6 storyboards"` | Генерация серии видеосегментов с индивидуальными запросами и длительностями. Поддерживается только для `kling-v3-omni`. При включении глобальный `prompt` игнорируется, а общая длительность всех сегментов раскадровки должна равняться глобальной `duration`. |
| `generate_audio` | BOOLEAN | Нет | `true`<br>`false` | Генерация аудио для видео. Поддерживается только для `kling-v3-omni` (по умолчанию: false). |
| `seed` | INT | Нет | от 0 до 2147483647 | Seed управляет повторным запуском узла; результаты недетерминированы независимо от seed (по умолчанию: 0). |

**Примечание:** Вход `reference_images` принимает максимум 7 изображений. Если передано больше, узел выдаст ошибку. Каждое изображение проверяется на минимальные размеры и соотношение сторон.

**Ограничения, связанные с моделью:**
- `kling-video-o1` не поддерживает длительность более 10 секунд.
- `kling-video-o1` не поддерживает генерацию аудио.
- `kling-video-o1` не поддерживает разрешение 4k.
- `kling-video-o1` не поддерживает раскадровки.

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `output` | VIDEO | Сгенерированный видеофайл. |
