# 繪製文字覆蓋

此節點可在單張影像或一批影像上繪製文字。它會建立一個可自訂字型大小、顏色、位置、對齊方式及可選外框的文字疊加層，然後將文字合成到原始影像上。

## 輸入

| 參數 | 說明 | 資料類型 | 必要 | 範圍 |
|-----------|-------------|-----------|----------|-------|
| `圖片` | 要繪製文字的輸入影像或一批影像 | IMAGE | 是 | |
| `文字` | 要疊加在影像上的文字（預設：""） | STRING | 是 | |
| `字型大小` | 字型大小，以影像高度的百分比表示（預設：5.0） | FLOAT | 是 | 0.5 至 50.0 |
| `顏色` | 文字的顏色（預設："#ffffff"） | STRING | 是 | |
| `位置` | 文字在影像上的垂直位置（預設："top"） | COMBO | 是 | `"top"`<br>`"bottom"` |
| `對齊` | 文字的水平對齊方式（預設："left"） | COMBO | 是 | `"left"`<br>`"center"`<br>`"right"` |
| `外框` | 在文字周圍繪製黑色外框（預設：True） | BOOLEAN | 是 | |

## 輸出

| 輸出名稱 | 說明 | 資料類型 |
|-------------|-------------|-----------|
| `圖片` | 已合成文字疊加層的輸入影像 | IMAGE |
