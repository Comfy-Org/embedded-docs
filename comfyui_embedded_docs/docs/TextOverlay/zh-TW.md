# 繪製文字覆蓋

此節點在單張圖片或一批圖片上繪製文字。它支援自訂文字、字型大小、顏色、位置、對齊方式以及可選的外框，並將文字合成到原始圖片之上。

## 輸入
| 參數 | 說明 | 資料類型 | 必要 | 範圍 |
|------|------|---------|------|------|
| `圖片` | 要繪製文字的輸入圖片或一批圖片。 | IMAGE | 是 | |
| `文字` | 要疊加的文字字串。支援換行符（`\n`）和定位字元（`\t`）。預設值："" | STRING | 是 | |
| `字型大小` | 字型大小，以圖片高度的百分比表示。預設值：5.0 | FLOAT | 是 | 0.5 至 50.0 |
| `顏色` | 文字的顏色。預設值："#ffffff"（白色） | STRING | 是 | |
| `位置` | 文字在圖片上的垂直位置。預設值："top" | COMBO | 是 | "top"<br>"bottom" |
| `對齊` | 文字的水平對齊方式。預設值："left" | COMBO | 是 | "left"<br>"center"<br>"right" |
| `外框` | 在文字周圍繪製黑色外框。預設值：True | BOOLEAN | 是 | |

## 輸出
| 輸出名稱 | 說明 | 資料類型 |
|---------|------|---------|
