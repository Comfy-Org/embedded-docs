# Наложение текста

Этот узел наносит текст поверх одного изображения или пакета изображений. Он поддерживает произвольный текст, размер шрифта, цвет, положение, выравнивание и опциональную обводку, накладывая текст поверх исходного изображения.

## Входы

| Параметр | Описание | Тип данных | Обязательный | Диапазон |
|----------|----------|------------|--------------|----------|
| `изображения` | Входное изображение или пакет изображений, на которые будет нанесен текст. | IMAGE | Да | |
| `текст` | Текстовая строка для наложения. Поддерживает символы новой строки (`\n`) и табуляции (`\t`). По умолчанию: "" | STRING | Да | |
| `размер_шрифта` | Размер шрифта в процентах от высоты изображения. По умолчанию: 5.0 | FLOAT | Да | от 0.5 до 50.0 |
| `цвет` | Цвет текста. По умолчанию: "#ffffff" (белый) | STRING | Да | |
| `позиция` | Вертикальное положение текста на изображении. По умолчанию: "top" | COMBO | Да | "top"<br>"bottom" |
| `выравнивание` | Горизонтальное выравнивание текста. По умолчанию: "left" | COMBO | Да | "left"<br>"center"<br>"right" |
| `контур` | Рисовать черную обводку вокруг текста. По умолчанию: True | BOOLEAN | Да | |

## Выходы

| Имя выхода | Описание | Тип данных |
|------------|----------|------------|
| `изображения` | Изображение или пакет изображений с наложенным поверх текстом. | IMAGE |
