# افزودن متن به تصویر

این گره متن را روی یک تصویر یا یک دسته از تصاویر رسم می‌کند. یک لایه متنی با اندازه قلم، رنگ، موقعیت، تراز و خط‌نمای اختیاری قابل تنظیم ایجاد کرده و سپس متن را روی تصاویر اصلی ترکیب می‌کند.

## ورودی‌ها

| پارامتر | توضیحات | نوع داده | اجباری | محدوده |
|---------|---------|----------|--------|--------|
| `تصاویر` | تصویر ورودی یا دسته تصاویری که قرار است متن روی آن‌ها رسم شود | IMAGE | بله | |
| `متن` | متنی که روی تصویر قرار می‌گیرد (پیش‌فرض: "") | STRING | بله | |
| `اندازه فونت` | اندازه قلم به‌صورت درصدی از ارتفاع تصویر (پیش‌فرض: 5.0) | FLOAT | بله | 0.5 تا 50.0 |
| `رنگ` | رنگ متن (پیش‌فرض: "#ffffff") | STRING | بله | |
| `موقعیت` | موقعیت عمودی متن روی تصویر (پیش‌فرض: "top") | COMBO | بله | `"top"`<br>`"bottom"` |
| `تراز` | تراز افقی متن (پیش‌فرض: "left") | COMBO | بله | `"left"`<br>`"center"`<br>`"right"` |
| `حاشیه` | رسم خط‌نمای مشکی دور متن (پیش‌فرض: True) | BOOLEAN | بله | |

## خروجی‌ها

| نام خروجی | توضیحات | نوع داده |
|-----------|---------|----------|
| `تصاویر` | تصاویر ورودی با لایه متنی ترکیب‌شده روی آن‌ها | IMAGE |
