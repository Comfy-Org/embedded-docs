# افزودن متن به تصویر

این گره متن را روی یک تصویر یا یک دسته از تصاویر رسم می‌کند. از متن سفارشی، اندازه قلم، رنگ، موقعیت، تراز و یک حاشیه اختیاری پشتیبانی می‌کند و متن را روی تصویر اصلی ترکیب می‌کند.

## ورودی‌ها

| پارامتر | توضیحات | نوع داده | ضروری | محدوده |
|---------|---------|----------|-------|--------|
| `تصاویر` | تصویر ورودی یا دسته تصاویری که قرار است متن روی آن‌ها رسم شود. | IMAGE | بله | |
| `متن` | رشته متنی که قرار است روی تصویر قرار گیرد. از کاراکترهای خط جدید (`\n`) و تب (`\t`) پشتیبانی می‌کند. پیش‌فرض: "" | STRING | بله | |
| `اندازه فونت` | اندازه قلم به صورت درصدی از ارتفاع تصویر. پیش‌فرض: 5.0 | FLOAT | بله | 0.5 تا 50.0 |
| `رنگ` | رنگ متن. پیش‌فرض: "#ffffff" (سفید) | STRING | بله | |
| `موقعیت` | موقعیت عمودی متن روی تصویر. پیش‌فرض: "top" | COMBO | بله | "top"<br>"bottom" |
| `تراز` | تراز افقی متن. پیش‌فرض: "left" | COMBO | بله | "left"<br>"center"<br>"right" |
| `حاشیه` | رسم یک حاشیه سیاه در اطراف متن. پیش‌فرض: True | BOOLEAN | بله | |

## خروجی‌ها

| نام خروجی | توضیحات | نوع داده |
|-----------|---------|----------|
| `تصاویر` | تصویر یا دسته تصاویر با متن ترکیب‌شده روی آن. | IMAGE |
