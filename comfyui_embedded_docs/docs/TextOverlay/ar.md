# إضافة نص على الصورة

هذه العقدة ترسم نصًا فوق صورة أو مجموعة من الصور. تقوم بإنشاء طبقة نصية مع خيارات تخصيص لحجم الخط واللون والموضع والمحاذاة، مع إمكانية إضافة إطار خارجي اختياري، ثم تدمج النص مع الصور الأصلية.

## المدخلات

| المعامل | الوصف | نوع البيانات | إجباري | النطاق |
|---------|-------|---------------|---------|--------|
| `الصور` | الصورة المدخلة أو مجموعة الصور لرسم النص عليها | IMAGE | نعم | |
| `النص` | النص المراد تركيبه على الصورة (الافتراضي: "") | STRING | نعم | |
| `حجم الخط` | حجم الخط كنسبة مئوية من ارتفاع الصورة (الافتراضي: 5.0) | FLOAT | نعم | 0.5 إلى 50.0 |
| `اللون` | لون النص (الافتراضي: "#ffffff") | STRING | نعم | |
| `الموضع` | الموضع الرأسي للنص على الصورة (الافتراضي: "أعلى") | COMBO | نعم | `"أعلى"`<br>`"أسفل"` |
| `محاذاة` | المحاذاة الأفقية للنص (الافتراضي: "يسار") | COMBO | نعم | `"يسار"`<br>`"وسط"`<br>`"يمين"` |
| `حد خارجي` | رسم إطار خارجي أسود حول النص (الافتراضي: صحيح) | BOOLEAN | نعم | |

## المخرجات

| اسم المخرج | الوصف | نوع البيانات |
|------------|-------|--------------|
| `الصور` | الصور المدخلة مع طبقة النص المدمجة في الأعلى | IMAGE |
