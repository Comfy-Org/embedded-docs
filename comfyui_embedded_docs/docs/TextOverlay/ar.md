# إضافة نص على الصورة

هذه العقدة ترسم نصًا فوق صورة أو مجموعة من الصور. تدعم النص المخصص، وحجم الخط، واللون، والموضع، والمحاذاة، بالإضافة إلى إطار اختياري، مع دمج النص فوق الصورة الأصلية.

## المدخلات

| المعامل | الوصف | نوع البيانات | إلزامي | النطاق |
|-----------|-------------|-----------|----------|-------|
| `الصور` | الصورة المدخلة أو مجموعة الصور التي سيتم رسم النص عليها. | IMAGE | نعم | |
| `النص` | سلسلة النص المراد تركيبه. يدعم أحرف السطر الجديد (`\n`) والتبويب (`\t`). القيمة الافتراضية: "" | STRING | نعم | |
| `حجم الخط` | حجم الخط كنسبة مئوية من ارتفاع الصورة. القيمة الافتراضية: 5.0 | FLOAT | نعم | 0.5 إلى 50.0 |
| `اللون` | لون النص. القيمة الافتراضية: "#ffffff" (أبيض) | STRING | نعم | |
| `الموضع` | الموضع الرأسي للنص على الصورة. القيمة الافتراضية: "top" | COMBO | نعم | "top"<br>"bottom" |
| `محاذاة` | المحاذاة الأفقية للنص. القيمة الافتراضية: "left" | COMBO | نعم | "left"<br>"center"<br>"right" |
| `حد خارجي` | رسم إطار أسود حول النص. القيمة الافتراضية: True | BOOLEAN | نعم | |

## المخرجات

| اسم المخرج | الوصف | نوع البيانات |
|-------------|-------------|-----------|
| `الصور` | الصورة أو مجموعة الصور مع النص المدمج في الأعلى. | IMAGE |
