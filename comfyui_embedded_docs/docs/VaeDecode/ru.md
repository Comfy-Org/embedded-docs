Узел VAEDecode предназначен для декодирования латентных представлений в изображения с помощью заданного вариационного автоэнкодера (VAE). Его цель — создание изображений из сжатых представлений данных, что позволяет восстанавливать изображения из их кодировок в латентном пространстве.

## Входные параметры

| Параметр | Тип данных | Описание |
|-----------|-------------|-------------|
| `samples` | `LATENT`    | Параметр 'samples' представляет собой латентные представления, подлежащие декодированию в изображения. Он критически важен для процесса декодирования, так как содержит сжатые данные, из которых восстанавливаются изображения. |
| `vae`     | VAE       | Параметр 'vae' задает модель вариационного автоэнкодера, используемую для декодирования латентных представлений в изображения. Он определяет механизм декодирования и качество восстанавливаемых изображений. |

## Выходные параметры

| Параметр | Тип данных | Описание |
|-----------|-------------|-------------|
| `image`   | `IMAGE`     | На выходе получается изображение, восстановленное из предоставленного латентного представления с помощью указанной модели VAE. |
