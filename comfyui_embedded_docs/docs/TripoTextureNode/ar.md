تقوم عقدة TripoTextureNode بإنشاء نماذج ثلاثية الأبعاد منسوجة باستخدام واجهة برمجة تطبيقات Tripo. تأخذ هذه العقدة معرف مهمة النموذج وتطبق عملية توليد النسيج مع خيارات متنوعة تشمل مواد PBR وإعدادات جودة النسيج وطرق المحاذاة. تتواصل العقدة مع واجهة برمجة تطبيقات Tripo لمعالجة طلب توليد النسيج وإرجاع ملف النموذج الناتج ومعرف المهمة.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|-----------|-----------|----------|-------|-------------|
| `معرف_مهمة_النموذج` | MODEL_TASK_ID | نعم | - | معرف مهمة النموذج المراد تطبيق النسيج عليه |
| `الملمس` | BOOLEAN | لا | - | هل سيتم توليد النسيج (الافتراضي: True) |
| `PBR` | BOOLEAN | لا | - | هل سيتم توليد مواد PBR (المواد القائمة على الفيزياء) (الافتراضي: True) |
| `بذرة_الملمس` | INT | لا | - | البذرة العشوائية لتوليد النسيج (الافتراضي: 42) |
| `جودة_الملمس` | COMBO | لا | "standard"<br>"detailed" | مستوى جودة توليد النسيج (الافتراضي: "standard"). خيار "detailed" يكلف 0.20 دولار أمريكي، بينما يكلف "standard" 0.10 دولار أمريكي. |
| `محاذاة_الملمس` | COMBO | لا | "original_image"<br>"geometry" | طريقة محاذاة النسيج (الافتراضي: "original_image"). تقوم "original_image" بمحاذاة النسيج مع صورة الإدخال الأصلية، بينما تقوم "geometry" بمحاذاته مع الهندسة ثلاثية الأبعاد. |

*ملاحظة: تتطلب هذه العقدة رموز المصادقة ومفاتيح واجهة برمجة التطبيقات التي يتم التعامل معها تلقائيًا بواسطة النظام.*

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|-------------|-----------|-------------|
| `معرّف مهمة النموذج` | STRING | ملف النموذج المُنشأ مع النسيج المطبق (لأغراض التوافق مع الإصدارات السابقة فقط) |
| `GLB` | MODEL_TASK_ID | معرف المهمة لتتبع عملية توليد النسيج |
| `GLB` | FILE3DGLB | النموذج ثلاثي الأبعاد المُنشأ بتنسيق GLB مع النسيج المطبق |
