Этот узел использует модель Kling 3.0 для генерации видео. Он создает видео на основе текстового запроса, указанной длительности и двух предоставленных изображений: начального и конечного кадра. Узел также может генерировать сопутствующую аудиодорожку для видео.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `prompt` | STRING | Да | Н/Д | Текстовое описание, которое направляет генерацию видео. Длина должна быть от 1 до 2500 символов. |
| `длительность` | INT | Нет | от 3 до 15 | Длительность видео в секундах (по умолчанию: 5). |
| `первый кадр` | IMAGE | Да | Н/Д | Начальное изображение для видео. Должно быть не менее 300x300 пикселей и иметь соотношение сторон от 1:2.5 до 2.5:1. |
| `последний кадр` | IMAGE | Да | Н/Д | Конечное изображение для видео. Должно быть не менее 300x300 пикселей и иметь соотношение сторон от 1:2.5 до 2.5:1. |
| `создать аудио` | BOOLEAN | Нет | Н/Д | Управляет генерацией аудиодорожки для видео (по умолчанию: True). |
| `модель` | COMBO | Нет | `"kling-v3"` | Модель и настройки генерации. Выбор этой опции открывает вложенный параметр `resolution`. |
| `model.resolution` | COMBO | Нет | `"4k"`<br>`"1080p"`<br>`"720p"` | Разрешение генерируемого видео. Этот параметр доступен только при установке `модель` в значение `"kling-v3"` (по умолчанию: `"1080p"`). |
| `seed` | INT | Нет | от 0 до 2147483647 | Число, используемое для управления повторным запуском узла. Результаты недетерминированы независимо от значения seed (по умолчанию: 0). |

**Примечание:** Изображения `first_frame` и `end_frame` должны соответствовать указанным требованиям к минимальному размеру и соотношению сторон для корректной работы узла.

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `output` | VIDEO | Сгенерированный видеофайл. |
