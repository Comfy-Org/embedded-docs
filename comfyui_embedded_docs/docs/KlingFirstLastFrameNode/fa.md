این گره از مدل Kling 3.0 برای تولید ویدیو استفاده می‌کند. ویدیو را بر اساس یک توضیح متنی، مدت زمان مشخص و دو تصویر ارائه‌شده (یک فریم شروع و یک فریم پایان) ایجاد می‌کند. این گره همچنین می‌تواند صدای همراه ویدیو را نیز تولید کند.

## ورودی‌ها

| پارامتر | نوع داده | اجباری | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `prompt` | STRING | بله | نامشخص | توضیح متنی که تولید ویدیو را هدایت می‌کند. باید بین ۱ تا ۲۵۰۰ کاراکتر باشد. |
| `duration` | INT | خیر | ۳ تا ۱۵ | طول ویدیو بر حسب ثانیه (پیش‌فرض: ۵). |
| `first_frame` | IMAGE | بله | نامشخص | تصویر شروع ویدیو. باید حداقل ۳۰۰×۳۰۰ پیکسل باشد و نسبت تصویر آن بین ۱:۲٫۵ تا ۲٫۵:۱ باشد. |
| `end_frame` | IMAGE | بله | نامشخص | تصویر پایان ویدیو. باید حداقل ۳۰۰×۳۰۰ پیکسل باشد و نسبت تصویر آن بین ۱:۲٫۵ تا ۲٫۵:۱ باشد. |
| `generate_audio` | BOOLEAN | خیر | نامشخص | کنترل می‌کند که آیا برای ویدیو صدا تولید شود یا خیر (پیش‌فرض: True). |
| `model` | COMBO | خیر | `"kling-v3"` | تنظیمات مدل و تولید. انتخاب این گزینه یک پارامتر تو در تو به نام `resolution` را آشکار می‌کند. |
| `model.resolution` | COMBO | خیر | `"4k"`<br>`"1080p"`<br>`"720p"` | وضوح تصویر برای ویدیوی تولیدشده. این پارامتر تنها زمانی در دسترس است که `model` روی `"kling-v3"` تنظیم شده باشد (پیش‌فرض: `"1080p"`). |
| `seed` | INT | خیر | ۰ تا ۲۱۴۷۴۸۳۶۴۷ | عددی که برای کنترل اجرای مجدد گره استفاده می‌شود. نتایج بدون توجه به مقدار seed غیرقطعی هستند (پیش‌فرض: ۰). |

**توجه:** تصاویر `first_frame` و `end_frame` باید حداقل اندازه و نسبت تصویر مشخص‌شده را داشته باشند تا گره به درستی کار کند.
