هذه ترجمة الوثيقة التقنية لعقدة ComfyUI الخاصة بمرجع الصور من ByteDance، مع الالتزام بقواعد الترجمة المحددة:

---

تقوم عقدة مرجع الصور من ByteDance بإنشاء فيديوهات باستخدام نص وصفي (برومبت) ومن صورة إلى أربع صور مرجعية. ترسل العقد الصور والنص الوصفي إلى خدمة API خارجية تقوم بإنشاء فيديو يطابق وصفك مع دمج النمط البصري والمحتوى من صورك المرجعية. توفر العقد عناصر تحكم متنوعة لدقة الفيديو، ونسبة العرض إلى الارتفاع، والمدة، ومعلمات التوليد الأخرى.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|-----------|-----------|----------|-------|-------------|
| `model` | STRING | نعم | `"seedance-1-0-pro-250528"`<br>`"seedance-1-0-lite-i2v-250428"` | نموذج الذكاء الاصطناعي المستخدم لتوليد الفيديو (الافتراضي: `"seedance-1-0-lite-i2v-250428"`). |
| `prompt` | STRING | نعم | - | النص الوصفي المستخدم لتوليد الفيديو. |
| `images` | IMAGE | نعم | - | من صورة إلى أربع صور. يجب أن يتراوح حجم كل صورة بين 300×300 و6000×6000 بكسل، مع نسبة عرض إلى ارتفاع بين 0.4 و2.5. |
| `resolution` | STRING | نعم | `"480p"`<br>`"720p"` | دقة الفيديو الناتج. |
| `aspect_ratio` | STRING | نعم | `"adaptive"`<br>`"16:9"`<br>`"4:3"`<br>`"1:1"`<br>`"3:4"`<br>`"9:16"`<br>`"21:9"` | نسبة العرض إلى الارتفاع للفيديو الناتج (الافتراضي: `"adaptive"`). |
| `duration` | INT | نعم | 3 - 12 | مدة الفيديو الناتج بالثواني (الافتراضي: 5). |
| `seed` | INT | لا | 0 - 2147483647 | البذرة المستخدمة في التوليد (الافتراضي: 0). |
| `watermark` | BOOLEAN | لا | - | ما إذا كان سيتم إضافة علامة مائية "مُولد بالذكاء الاصطناعي" إلى الفيديو (الافتراضي: False). |

**ملاحظة:** يجب ألا يحتوي النص الوصفي على سلاسل المعاملات التالية: `--resolution`، `--ratio`، `--duration`، `--seed`، أو `--watermark`. يتم التحكم في هذه القيم حصريًا من خلال أدوات الإدخال المخصصة لها.
