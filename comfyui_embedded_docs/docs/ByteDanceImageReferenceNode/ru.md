Вот перевод документации на русский язык:

Узел ByteDance Image Reference Node генерирует видео на основе текстового запроса и одного-четырех референсных изображений. Он отправляет изображения и запрос во внешний API-сервис, который создает видео, соответствующее вашему описанию, сохраняя визуальный стиль и содержание референсных изображений. Узел предоставляет различные настройки для разрешения видео, соотношения сторон, длительности и других параметров генерации.

## Входные данные

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `модель` | STRING | Да | `"seedance-1-0-pro-250528"`<br>`"seedance-1-0-lite-i2v-250428"` | Модель ИИ для генерации видео (по умолчанию: `"seedance-1-0-lite-i2v-250428"`). |
| `промпт` | STRING | Да | - | Текстовый запрос для генерации видео. |
| `изображения` | IMAGE | Да | - | От одного до четырех изображений. Каждое изображение должно быть размером от 300x300 до 6000x6000 пикселей, с соотношением сторон от 0,4 до 2,5. |
| `разрешение` | STRING | Да | `"480p"`<br>`"720p"` | Разрешение выходного видео. |
| `соотношение_сторон` | STRING | Да | `"adaptive"`<br>`"16:9"`<br>`"4:3"`<br>`"1:1"`<br>`"3:4"`<br>`"9:16"`<br>`"21:9"` | Соотношение сторон выходного видео (по умолчанию: `"adaptive"`). |
| `длительность` | INT | Да | 3 - 12 | Длительность выходного видео в секундах (по умолчанию: 5). |
| `сид` | INT | Нет | 0 - 2147483647 | Сид для генерации (по умолчанию: 0). |
| `водяной_знак` | BOOLEAN | Нет | - | Добавлять ли водяной знак "Создано ИИ" на видео (по умолчанию: False). |

**Примечание:** Текст запроса не должен содержать следующие строки параметров: `--resolution`, `--ratio`, `--duration`, `--seed` или `--watermark`. Эти значения управляются исключительно через соответствующие выделенные виджеты ввода.

## Выходные данные

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `output` | VIDEO | Сгенерированный видеофайл на основе входного запроса и референсных изображений. |
