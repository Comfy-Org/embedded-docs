### نمای کلی

گره مرجع تصویر ByteDance با استفاده از یک متن راهنما (prompt) و یک تا چهار تصویر مرجع، ویدیو تولید می‌کند. این گره تصاویر و متن راهنما را به یک سرویس API خارجی ارسال می‌کند که ویدیویی مطابق با توضیحات شما و با الهام از سبک بصری و محتوای تصاویر مرجع می‌سازد. این گره کنترل‌های متنوعی برای وضوح ویدیو، نسبت تصویر، مدت زمان و سایر پارامترهای تولید ارائه می‌دهد.

## ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `model` | STRING | بله | `"seedance-1-0-pro-250528"`<br>`"seedance-1-0-lite-i2v-250428"` | مدل هوش مصنوعی مورد استفاده برای تولید ویدیو (پیش‌فرض: `"seedance-1-0-lite-i2v-250428"`). |
| `prompt` | STRING | بله | - | متن راهنمای (prompt) مورد استفاده برای تولید ویدیو. |
| `images` | IMAGE | بله | - | یک تا چهار تصویر. هر تصویر باید بین 300×300 و 6000×6000 پیکسل باشد، با نسبت تصویری بین 0.4 و 2.5. |
| `resolution` | STRING | بله | `"480p"`<br>`"720p"` | وضوح ویدیوی خروجی. |
| `aspect_ratio` | STRING | بله | `"adaptive"`<br>`"16:9"`<br>`"4:3"`<br>`"1:1"`<br>`"3:4"`<br>`"9:16"`<br>`"21:9"` | نسبت تصویر ویدیوی خروجی (پیش‌فرض: `"adaptive"`). |
| `duration` | INT | بله | 3 - 12 | مدت زمان ویدیوی خروجی بر حسب ثانیه (پیش‌فرض: 5). |
| `seed` | INT | خیر | 0 - 2147483647 | دانه (seed) مورد استفاده برای تولید (پیش‌فرض: 0). |
| `watermark` | BOOLEAN | خیر | - | آیا به ویدیو واترمارک "AI generated" اضافه شود (پیش‌فرض: False). |

**نکته:** متن راهنما (prompt) نباید شامل رشته‌های پارامتری زیر باشد: `--resolution`، `--ratio`، `--duration`، `--seed` یا `--watermark`. این مقادیر منحصراً از طریق ویجت‌های ورودی اختصاصی خود کنترل می‌شوند.
