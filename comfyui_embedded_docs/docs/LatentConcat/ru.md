Вот перевод документации узла LatentConcat на русский язык:

Узел LatentConcat объединяет два латентных сэмпла путем их соединения вдоль выбранного измерения. Он принимает два латентных входа и конкатенирует их вдоль оси x, y или t, с возможностью управления порядком следования сэмплов. Узел автоматически подстраивает размер пакета второго входа под размер пакета первого перед выполнением конкатенации.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `samples1` | LATENT | Да | - | Первый латентный сэмпл для конкатенации |
| `samples2` | LATENT | Да | - | Второй латентный сэмпл для конкатенации |
| `dim` | COMBO | Да | `"x"`<br>`"-x"`<br>`"y"`<br>`"-y"`<br>`"t"`<br>`"-t"` | Измерение, вдоль которого выполняется конкатенация латентных сэмплов. Положительные значения (x, y, t) помещают samples1 перед samples2 в результате. Отрицательные значения (-x, -y, -t) помещают samples2 перед samples1. Соответствие измерений: x = ширина, y = высота, t = время/кадры |

**Примечание:** Второй латентный сэмпл (`samples2`) автоматически подстраивается под размер пакета первого латентного сэмпла (`samples1`) перед конкатенацией.

## Выходы

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `output` | LATENT | Конкатенированные латентные сэмплы, полученные в результате объединения двух входных сэмплов вдоль указанного измерения |
