Вот перевод документации на русский язык:

Узел DualCFGGuider создает систему управления для двойной бесклассовой направляющей выборки. Он объединяет два положительных входа conditioning с одним отрицательным входом conditioning, применяя разные масштабы управления к каждой паре conditioning для контроля влияния каждого промпта на генерируемый результат.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `модель` | MODEL | Да | - | Модель, используемая для управления |
| `cond1` | CONDITIONING | Да | - | Первый положительный вход conditioning |
| `cond2` | CONDITIONING | Да | - | Второй положительный вход conditioning |
| `отрицательный` | CONDITIONING | Да | - | Отрицательный вход conditioning |
| `cfg_conds` | FLOAT | Да | 0.0 - 100.0 | Масштаб управления для первого положительного conditioning (по умолчанию: 8.0) |
| `cfg_cond2_negative` | FLOAT | Да | 0.0 - 100.0 | Масштаб управления для второго положительного и отрицательного conditioning (по умолчанию: 8.0) |
| `стиль` | COMBO | Да | "regular"<br>"nested" | Стиль управления (по умолчанию: "regular"). При значении "nested" управление применяется вложенным образом |

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `GUIDER` | GUIDER | Настроенная система управления, готовая к использованию с выборкой |
