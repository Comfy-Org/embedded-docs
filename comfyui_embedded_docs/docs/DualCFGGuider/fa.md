گره DualCFGGuider یک سیستم هدایت برای نمونه‌برداری دوگانه بدون طبقه‌بندی ایجاد می‌کند. این گره دو ورودی شرطی‌سازی مثبت را با یک ورودی شرطی‌سازی منفی ترکیب می‌کند و مقیاس‌های هدایت متفاوتی را برای هر جفت شرطی‌سازی اعمال می‌کند تا میزان تأثیر هر prompt بر خروجی تولیدشده را کنترل کند.

## ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|---------|----------|-------|--------|---------|
| `مدل` | MODEL | بله | - | مدل مورد استفاده برای هدایت |
| `شرط ۱` | CONDITIONING | بله | - | اولین ورودی شرطی‌سازی مثبت |
| `شرط ۲` | CONDITIONING | بله | - | دومین ورودی شرطی‌سازی مثبت |
| `منفی` | CONDITIONING | بله | - | ورودی شرطی‌سازی منفی |
| `شرایط cfg` | FLOAT | بله | 0.0 - 100.0 | مقیاس هدایت برای اولین شرطی‌سازی مثبت (پیش‌فرض: 8.0) |
| `cfg_cond2 منفی` | FLOAT | بله | 0.0 - 100.0 | مقیاس هدایت برای دومین شرطی‌سازی مثبت و شرطی‌سازی منفی (پیش‌فرض: 8.0) |
| `استایل` | COMBO | بله | "regular"<br>"nested" | سبک هدایت اعمال‌شونده (پیش‌فرض: "regular"). وقتی روی "nested" تنظیم شود، هدایت به صورت تو در تو اعمال می‌شود |

## خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-----------|----------|---------|
| `GUIDER` | GUIDER | یک سیستم هدایت پیکربندی‌شده آماده برای استفاده با نمونه‌برداری |
