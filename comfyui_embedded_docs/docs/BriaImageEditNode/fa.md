### گره ویرایش تصویر Bria FIBO

گره ویرایش تصویر Bria FIBO به شما امکان می‌دهد با استفاده از یک دستور متنی، تصویری موجود را تغییر دهید. این گره تصویر و prompt شما را به API سرویس Bria ارسال می‌کند که از مدل FIBO برای تولید نسخه‌ای جدید و ویرایش‌شده از تصویر بر اساس درخواست شما استفاده می‌کند. همچنین می‌توانید یک ماسک برای محدود کردن ویرایش‌ها به ناحیه‌ای خاص ارائه دهید.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|---------|----------|--------|--------|---------|
| `model` | COMBO | بله | `"FIBO"` | نسخه مدل مورد استفاده برای ویرایش تصویر. |
| `image` | IMAGE | بله | - | تصویر ورودی که می‌خواهید ویرایش کنید. |
| `prompt` | STRING | خیر | - | دستور متنی که نحوه ویرایش تصویر را توصیف می‌کند (پیش‌فرض: خالی). |
| `negative_prompt` | STRING | خیر | - | متنی که آنچه را نمی‌خواهید در تصویر ویرایش‌شده ظاهر شود، توصیف می‌کند (پیش‌فرض: خالی). |
| `structured_prompt` | STRING | خیر | - | رشته‌ای حاوی prompt ویرایش ساختاریافته در قالب JSON. برای کنترل دقیق و برنامه‌نویسی‌شده به جای prompt معمولی از این استفاده کنید (پیش‌فرض: خالی). |
| `seed` | INT | بله | 1 تا 2147483647 | عددی برای مقداردهی اولیه تولید تصادفی، که نتایج قابل تکرار را تضمین می‌کند (پیش‌فرض: 1). |
| `guidance_scale` | FLOAT | بله | 3.0 تا 5.0 | میزان تطابق تصویر تولیدشده با prompt را کنترل می‌کند. مقدار بالاتر به معنای پیروی قوی‌تر است (پیش‌فرض: 3.0). |
| `steps` | INT | بله | 20 تا 50 | تعداد مراحل نویززدایی که مدل انجام می‌دهد (پیش‌فرض: 50). |
| `moderation` | DYNAMICCOMBO | بله | `"false"`<br>`"true"` | نظارت بر محتوا را فعال یا غیرفعال می‌کند. انتخاب `"true"` گزینه‌های نظارتی اضافی برای محتوای prompt، ورودی بصری و خروجی بصری را نمایش می‌دهد. |
| `mask` | MASK | خیر | - | یک تصویر ماسک اختیاری. در صورت ارائه، ویرایش‌ها فقط به نواحی ماسک‌شده تصویر اعمال می‌شوند. |

**محدودیت‌های مهم:**

* حداقل باید یکی از ورودی‌های `prompt` یا `structured_prompt` را ارائه دهید. هر دو نمی‌توانند همزمان خالی باشند.
* دقیقاً یک `image` ورودی الزامی است.
* هنگامی که پارامتر `moderation` روی `"true"` تنظیم شود، سه ورودی بولی اضافی در دسترس قرار می‌گیرند: `prompt_content_moderation` (پیش‌فرض: false)، `visual_input_moderation` (پیش‌فرض: false) و `visual_output_moderation` (پیش‌فرض: true).

## خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-----------|----------|---------|
| `IMAGE` | IMAGE | تصویر ویرایش‌شده برگردانده‌شده توسط API سرویس Bria. |
