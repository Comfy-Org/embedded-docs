# ترجمة وثيقة عقدة APG (التوجيه الإسقاطي التكيفي)

## نظرة عامة

تقوم عقدة APG (التوجيه الإسقاطي التكيفي) بتعديل عملية أخذ العينات عن طريق ضبط كيفية تطبيق التوجيه أثناء الانتشار. تقوم بفصل متجه التوجيه إلى مكونات متوازية ومتعامدة بالنسبة للمخرجات المشروطة، مما يتيح توليد صور أكثر تحكمًا. توفر العقدة معاملات لقياس التوجيه، وتطبيع حجمه، وتطبيق الزخم لتحقيق انتقالات أكثر سلاسة بين خطوات الانتشار.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|---------|--------------|--------|--------|-------|
| `النموذج` | MODEL | نعم | - | نموذج الانتشار المراد تطبيق التوجيه الإسقاطي التكيفي عليه |
| `إيتا` | FLOAT | نعم | -10.0 إلى 10.0 | يتحكم في مقياس متجه التوجيه الموازي. سلوك CFG الافتراضي عند القيمة 1 (الافتراضي: 1.0). |
| `عتبة التطبيع` | FLOAT | نعم | 0.0 إلى 50.0 | تطبيع متجه التوجيه إلى هذه القيمة، يتم تعطيل التطبيع عند القيمة 0 (الافتراضي: 5.0). |
| `الزخم` | FLOAT | نعم | -5.0 إلى 1.0 | يتحكم في المتوسط المتحرك للتوجيه أثناء الانتشار، يتم تعطيله عند القيمة 0 (الافتراضي: 0.0). |
