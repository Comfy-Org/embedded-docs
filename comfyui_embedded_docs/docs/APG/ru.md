Вот перевод документации узла APG (Adaptive Projected Guidance) на русский язык:

Узел APG (Adaptive Projected Guidance) изменяет процесс сэмплирования, регулируя применение направляющего сигнала во время диффузии. Он разделяет вектор направляющего сигнала на параллельную и ортогональную составляющие относительно условного выхода, что позволяет более контролируемо генерировать изображения. Узел предоставляет параметры для масштабирования направляющего сигнала, нормализации его величины и применения импульса для более плавных переходов между шагами диффузии.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `model` | MODEL | Да | - | Модель диффузии, к которой применяется адаптивное проекционное направление |
| `eta` | FLOAT | Да | от -10.0 до 10.0 | Управляет масштабом параллельного вектора направляющего сигнала. Поведение по умолчанию CFG при значении 1 (по умолчанию: 1.0). |
| `norm_threshold` | FLOAT | Да | от 0.0 до 50.0 | Нормализует вектор направляющего сигнала до этого значения; нормализация отключена при значении 0 (по умолчанию: 5.0). |
| `momentum` | FLOAT | Да | от -5.0 до 1.0 | Управляет скользящим средним направляющего сигнала во время диффузии; отключено при значении 0 (по умолчанию: 0.0). |

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `model` | MODEL | Возвращает модифицированную модель с примененным адаптивным проекционным направлением к ее процессу сэмплирования |
