## 概述

APG（自適應投影引導）節點透過調整擴散過程中引導的應用方式來修改取樣流程。它將引導向量相對於條件輸出分離為平行分量和正交分量，從而實現更可控的圖像生成。此節點提供參數來縮放引導強度、正規化其幅度，並應用動量以實現擴散步驟之間更平滑的過渡。
