Узел LoraSave извлекает и сохраняет файлы LoRA (Low-Rank Adaptation) из различий моделей. Он может обрабатывать различия диффузионных моделей, различия текстовых энкодеров или оба типа, преобразуя их в формат LoRA с указанным рангом и типом. Полученный файл LoRA сохраняется в выходной директории для последующего использования.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `filename_prefix` | STRING | Да | - | Префикс для имени выходного файла (по умолчанию: "loras/ComfyUI_extracted_lora") |
| `rank` | INT | Да | 1-4096 | Значение ранга для LoRA, определяющее размер и сложность (по умолчанию: 8) |
| `lora_type` | COMBO | Да | Доступно несколько опций | Тип создаваемого LoRA с различными доступными вариантами |
| `bias_diff` | BOOLEAN | Да | - | Включать ли различия смещения в расчёт LoRA (по умолчанию: True) |
| `model_diff` | MODEL | Нет | - | Результат ModelSubtract для преобразования в lora |
| `text_encoder_diff` | CLIP | Нет | - | Результат CLIPSubtract для преобразования в lora |

**Примечание:** Для работы узла должен быть предоставлен хотя бы один из параметров `model_diff` или `text_encoder_diff`. Если оба опущены, узел не выдаст результата.

## Выходные данные

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| - | - | Этот узел сохраняет файл LoRA в выходную директорию, но не возвращает никаких данных через рабочий процесс |
