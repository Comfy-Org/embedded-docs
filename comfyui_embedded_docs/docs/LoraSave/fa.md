گره LoraSave تفاوت‌های مدل را استخراج کرده و به فایل‌های LoRA (Low-Rank Adaptation) ذخیره می‌کند. این گره می‌تواند تفاوت‌های مدل انتشار، تفاوت‌های رمزگذار متن یا هر دو را پردازش کرده و با رتبه و نوع مشخص‌شده به فرمت LoRA تبدیل کند. فایل LoRA حاصل در دایرکتوری خروجی برای استفاده بعدی ذخیره می‌شود.

## ورودی‌ها

| پارامتر | نوع داده | اجباری | محدوده | توضیحات |
|---------|----------|--------|--------|---------|
| `پیشوند نام فایل` | STRING | بله | - | پیشوند نام فایل خروجی (پیش‌فرض: "loras/ComfyUI_extracted_lora") |
| `رتبه` | INT | بله | 1-4096 | مقدار رتبه برای LoRA که اندازه و پیچیدگی را کنترل می‌کند (پیش‌فرض: 8) |
| `نوع lora` | COMBO | بله | `"standard"`<br>`"locon"`<br>`"loha"`<br>`"lokr"`<br>`"dylora"` | نوع LoRA مورد نظر برای ایجاد (پیش‌فرض: "standard") |
| `تفاوت بایاس` | BOOLEAN | بله | - | آیا تفاوت‌های بایاس در محاسبه LoRA گنجانده شود (پیش‌فرض: True) |
| `تفاوت مدل` | MODEL | خیر | - | خروجی ModelSubtract که باید به LoRA تبدیل شود |
| `تفاوت رمزگذار متنی` | CLIP | خیر | - | خروجی CLIPSubtract که باید به LoRA تبدیل شود |

**توجه:** برای عملکرد گره حداقل یکی از `model_diff` یا `text_encoder_diff` باید ارائه شود. اگر هر دو حذف شوند، گره هیچ خروجی تولید نخواهد کرد.
