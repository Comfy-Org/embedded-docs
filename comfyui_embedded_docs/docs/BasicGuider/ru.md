Вот перевод документации на русский язык:

Узел BasicGuider создаёт простой механизм управления для процесса семплирования. Он принимает модель и данные кондиционирования в качестве входных данных и возвращает объект guider, который можно использовать для управления процессом генерации во время семплирования. Этот узел обеспечивает базовую функциональность управления, необходимую для контролируемой генерации.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `model` | MODEL | Да | - | Модель, используемая для управления |
| `conditioning` | CONDITIONING | Да | - | Данные кондиционирования, управляющие процессом генерации |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `GUIDER` | GUIDER | Объект guider, который можно использовать в процессе семплирования для управления генерацией |
