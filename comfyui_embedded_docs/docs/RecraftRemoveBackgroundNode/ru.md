Вот перевод документации на русский язык:

Этот узел удаляет фон из изображений с помощью API-сервиса Recraft. Он обрабатывает каждое изображение во входном пакете и возвращает как обработанные изображения с прозрачным фоном, так и соответствующие альфа-маски, указывающие на удаленные области фона.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `image` | IMAGE | Да | - | Входное изображение(я) для обработки удаления фона |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `image` | IMAGE | Обработанные изображения с прозрачным фоном |
| `mask` | MASK | Альфа-маски, указывающие на удаленные области фона |
