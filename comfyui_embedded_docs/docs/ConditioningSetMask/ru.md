Этот узел предназначен для модификации conditioning генеративной модели путем применения маски с заданной силой воздействия к определенным областям. Он позволяет выполнять целенаправленные корректировки внутри conditioning, обеспечивая более точный контроль над процессом генерации.

## Входы

### Обязательные

| Параметр       | Тип данных    | Описание |
|----------------|---------------|----------|
| `CONDITIONING` | CONDITIONING  | Данные conditioning, подлежащие изменению. Служат основой для применения маски и корректировки силы воздействия. |
| `маска`         | `MASK`        | Тензор маски, определяющий области внутри conditioning, которые необходимо изменить. |
| `сила`     | `FLOAT`       | Сила воздействия маски на conditioning, позволяющая точно настроить применяемые изменения. |
| `установить_область_условия`| COMBO[STRING] | Определяет, применяется ли эффект маски к области по умолчанию или ограничивается самой маской, предоставляя гибкость при выборе целевых областей. |

## Выходы

| Параметр       | Тип данных    | Описание |
|----------------|---------------|----------|
| `CONDITIONING` | CONDITIONING  | Измененные данные conditioning с примененными корректировками маски и силы воздействия. |
