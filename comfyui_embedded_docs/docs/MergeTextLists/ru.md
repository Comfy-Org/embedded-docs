Вот перевод документации на русский язык:

Этот узел объединяет несколько текстовых списков в один комбинированный список. Он предназначен для получения текстовых входных данных в виде списков и объединяет их в один. Узел регистрирует общее количество текстов в объединённом списке.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `тексты` | STRING | Да | Н/Д | Текстовые списки для объединения. К входу можно подключить несколько списков, и они будут объединены в один. |

**Примечание:** Этот узел настроен как групповой процесс (`is_group_process = True`), что означает, что он автоматически обрабатывает несколько входных списков, объединяя их перед выполнением основной функции обработки.

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `тексты` | STRING | Единый объединённый список, содержащий все входные тексты. |
