Узел FreeU_V2 повышает качество генерации изображений за счёт применения частотных модификаций к U-Net архитектуре диффузионной модели. Он использует настраиваемые коэффициенты масштабирования для регулировки каналов признаков в различных блоках, улучшая результат без необходимости дополнительного обучения.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `model` | MODEL | Да | - | Диффузионная модель, к которой применяется улучшение FreeU |
| `b1` | FLOAT | Да | 0.0 - 10.0 | Коэффициент масштабирования признаков магистрали для первого блока (по умолчанию: 1.3) |
| `b2` | FLOAT | Да | 0.0 - 10.0 | Коэффициент масштабирования признаков магистрали для второго блока (по умолчанию: 1.4) |
| `s1` | FLOAT | Да | 0.0 - 10.0 | Коэффициент масштабирования пропускаемых признаков для первого блока (по умолчанию: 0.9) |
| `s2` | FLOAT | Да | 0.0 - 10.0 | Коэффициент масштабирования пропускаемых признаков для второго блока (по умолчанию: 0.2) |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `model` | MODEL | Улучшенная диффузионная модель с применёнными модификациями FreeU |
