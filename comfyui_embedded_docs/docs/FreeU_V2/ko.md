FreeU_V2 노드는 확산 모델의 U-Net 아키텍처에 주파수 기반 변형을 적용하여 이미지 생성 품질을 향상시킵니다. 구성 가능한 스케일링 팩터를 사용하여 서로 다른 블록의 특징 채널을 조정하며, 추가 학습 없이 출력 품질을 개선합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `모델` | MODEL | 예 | - | FreeU 향상을 적용할 확산 모델 |
| `b1` | FLOAT | 예 | 0.0 - 10.0 | 첫 번째 블록의 백본 특징 스케일링 팩터 (기본값: 1.3) |
| `b2` | FLOAT | 예 | 0.0 - 10.0 | 두 번째 블록의 백본 특징 스케일링 팩터 (기본값: 1.4) |
| `s1` | FLOAT | 예 | 0.0 - 10.0 | 첫 번째 블록의 스킵 특징 스케일링 팩터 (기본값: 0.9) |
| `s2` | FLOAT | 예 | 0.0 - 10.0 | 두 번째 블록의 스킵 특징 스케일링 팩터 (기본값: 0.2) |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `모델` | MODEL | FreeU 변형이 적용된 향상된 확산 모델 |
