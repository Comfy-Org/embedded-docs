Вот перевод документации на русский язык:

Узел ByteDance Image to Video генерирует видео с использованием моделей ByteDance через API на основе входного изображения и текстового запроса. Он принимает начальный кадр изображения и создает видеопоследовательность, соответствующую предоставленному описанию. Узел предлагает различные параметры настройки разрешения видео, соотношения сторон, длительности и других параметров генерации.

## Входные данные

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `model` | STRING | Да | `"seedance-1-5-pro-251215"`<br>`"seedance-1-0-pro-250528"`<br>`"seedance-1-0-lite-i2v-250428"`<br>`"seedance-1-0-pro-fast-251015"` | Модель ByteDance для генерации видео (по умолчанию: `"seedance-1-0-pro-fast-251015"`). |
| `prompt` | STRING | Да | - | Текстовый запрос для генерации видео. Должен содержать не менее 1 символа после удаления пробелов. |
| `image` | IMAGE | Да | - | Первый кадр для видео. Размер должен быть от 300x300 до 6000x6000 пикселей, соотношение сторон — от 0,4 до 2,5. |
| `resolution` | STRING | Да | `"480p"`<br>`"720p"`<br>`"1080p"` | Разрешение выходного видео. |
| `aspect_ratio` | STRING | Да | `"adaptive"`<br>`"16:9"`<br>`"4:3"`<br>`"1:1"`<br>`"3:4"`<br>`"9:16"`<br>`"21:9"` | Соотношение сторон выходного видео. |
| `duration` | INT | Да | 3 - 12 | Длительность выходного видео в секундах (по умолчанию: 5). Для модели `seedance-1-5-pro-251215` минимальная поддерживаемая длительность — 4 секунды. |
| `seed` | INT | Нет | 0 - 2147483647 | Сид для генерации (по умолчанию: 0). |
| `camera_fixed` | BOOLEAN | Нет | - | Указывает, следует ли фиксировать камеру. Платформа добавляет инструкцию по фиксации камеры к вашему запросу, но не гарантирует фактический эффект (по умолчанию: False). |
| `watermark` | BOOLEAN | Нет | - | Добавлять ли водяной знак "AI generated" к видео (по умолчанию: False). |
| `generate_audio` | BOOLEAN | Нет | - | Этот параметр игнорируется для всех моделей, кроме `seedance-1-5-pro-251215` (по умолчанию: False). |

**Примечание:** Запрос не должен содержать следующие слова (без учета регистра): `resolution`, `ratio`, `duration`, `seed`, `camerafixed`, `watermark`. Эти параметры задаются через соответствующие выделенные входы.

## Выходные данные

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `output` | VIDEO | Сгенерированный видеофайл на основе входного изображения и параметров запроса. |
