# ترجمة وثيقة عقدة ComfyUI

رفع دقة الصورة مع الحد الأدنى من التعديلات إلى دقة 4K. تستخدم هذه العقدة تقنية الرفع الإبداعي من Stability AI لتحسين دقة الصورة مع الحفاظ على المحتوى الأصلي وإضافة تفاصيل إبداعية دقيقة.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|---------|---------------|--------|---------|-------|
| `image` | IMAGE | نعم | - | الصورة المدخلة المراد رفع دقتها |
| `prompt` | STRING | نعم | - | ما ترغب في رؤيته في الصورة الناتجة. سيؤدي الوصف القوي والتفصيلي الذي يحدد بوضوح العناصر والألوان والموضوعات إلى نتائج أفضل. (الافتراضي: سلسلة فارغة) |
| `creativity` | FLOAT | نعم | 0.1-0.5 | يتحكم في احتمالية إنشاء تفاصيل إضافية غير معتمدة بشكل كبير على الصورة الأولية. (الافتراضي: 0.3) |
| `style_preset` | STRING | نعم | `"3d-model"`<br>`"analog-film"`<br>`"anime"`<br>`"cinematic"`<br>`"comic-book"`<br>`"digital-art"`<br>`"enhance"`<br>`"fantasy-art"`<br>`"isometric"`<br>`"line-art"`<br>`"low-poly"`<br>`"modeling-compound"`<br>`"neon-punk"`<br>`"origami"`<br>`"photographic"`<br>`"pixel-art"`<br>`"tile-texture"` | النمط المطلوب اختياريًا للصورة المُنشأة. (الافتراضي: "لا شيء") |
| `seed` | INT | نعم | 0-4294967294 | البذرة العشوائية المستخدمة لإنشاء الضوضاء. (الافتراضي: 0) |
| `negative_prompt` | STRING | لا | - | كلمات مفتاحية لما لا ترغب في رؤيته في الصورة الناتجة. هذه ميزة متقدمة. (الافتراضي: سلسلة فارغة) |
