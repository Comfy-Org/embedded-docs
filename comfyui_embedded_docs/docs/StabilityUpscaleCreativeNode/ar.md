# ترجمة وثيقة عقدة ComfyUI

رفع دقة الصورة مع الحد الأدنى من التعديلات إلى دقة 4K. تستخدم هذه العقدة تقنية الرفع الإبداعي من Stability AI لتحسين دقة الصورة مع الحفاظ على المحتوى الأصلي وإضافة تفاصيل إبداعية دقيقة.

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |

| `صورة` | IMAGE | نعم | - | الصورة المدخلة المراد رفع دقتها |
| `النص الوصفي` | STRING | نعم | - | ما ترغب في رؤيته في الصورة الناتجة. سيؤدي الوصف القوي والتفصيلي الذي يحدد بوضوح العناصر والألوان والموضوعات إلى نتائج أفضل. (الافتراضي: سلسلة فارغة) |
| `الإبداع` | FLOAT | نعم | 0.1-0.5 | يتحكم في احتمالية إنشاء تفاصيل إضافية غير معتمدة بشكل كبير على الصورة الأولية. (الافتراضي: 0.3) |
