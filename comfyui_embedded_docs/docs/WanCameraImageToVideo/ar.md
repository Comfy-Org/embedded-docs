# ترجمة وثيقة WanCameraImageToVideo

يقوم عقدة WanCameraImageToVideo بتحويل الصور إلى تسلسلات فيديو عن طريق توليد تمثيلات كامنة لتوليد الفيديو. تعالج هذه العقدة مدخلات التكييف والصور الاختيارية الأولية لإنشاء تمثيلات فيديو كامنة يمكن استخدامها مع نماذج الفيديو. تدعم العقدة شروط الكاميرا ومخرجات رؤية القصاصات (CLIP) لتحسين التحكم في توليد الفيديو.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|---------|---------------|--------|---------|-------|
| `positive` | CONDITIONING | نعم | - | مطالبات التكييف الإيجابية لتوليد الفيديو |
| `negative` | CONDITIONING | نعم | - | مطالبات التكييف السلبية لتجنبها في توليد الفيديو |
| `vae` | VAE | نعم | - | نموذج VAE لتشفير الصور إلى الفضاء الكامن |
| `width` | INT | نعم | 16 إلى MAX_RESOLUTION | عرض الفيديو الناتج بالبكسل (الافتراضي: 832، الخطوة: 16) |
| `height` | INT | نعم | 16 إلى MAX_RESOLUTION | ارتفاع الفيديو الناتج بالبكسل (الافتراضي: 480، الخطوة: 16) |
| `length` | INT | نعم | 1 إلى MAX_RESOLUTION | عدد الإطارات في تسلسل الفيديو (الافتراضي: 81، الخطوة: 4) |
| `batch_size` | INT | نعم | 1 إلى 4096 | عدد مقاطع الفيديو المراد توليدها في وقت واحد (الافتراضي: 1) |
| `clip_vision_output` | CLIP_VISION_OUTPUT | لا | - | مخرجات رؤية القصاصات الاختيارية لتكييف إضافي |
| `start_image` | IMAGE | لا | - | صورة أولية اختيارية لتهيئة تسلسل الفيديو. عند توفيرها، ستستند الإطارات الأولى من الفيديو إلى هذه الصورة، مع تطبيق قناع لمزج الإطارات الأولية مع المحتوى المولد. يتم تغيير حجم الصورة لتتناسب مع العرض والارتفاع المحددين. |
| `camera_conditions` | WAN_CAMERA_EMBEDDING | لا | - | شروط تضمين الكاميرا الاختيارية لتوليد الفيديو. عند توفيرها، يتم تطبيق هذه الشروط على كل من التكييف الإيجابي والسلبي. |

**ملاحظة:** عند توفير `start_image`، تستخدم العقدة هذه الصورة لتهيئة تسلسل الفيديو وتطبق قناعًا لمزج الإطارات الأولية مع المحتوى المولد. معاملات `camera_conditions` و `clip_vision_output` اختيارية، ولكن عند توفيرها، فإنها تعدل التكييف لكل من المطالبات الإيجابية والسلبية.

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|-------------|---------------|---------|
| `positive` | CONDITIONING | تكييف إيجابي معدل مع تطبيق شروط الكاميرا ومخرجات رؤية القصاصات |
| `negative` | CONDITIONING | تكييف سلبي معدل مع تطبيق شروط الكاميرا ومخرجات رؤية القصاصات |
| `latent` | LATENT | تمثيل فيديو كامن مولد للاستخدام مع نماذج الفيديو. أبعاد الموتر الكامن هي [batch_size, 16, frames, height/8, width/8] حيث يتم حساب الإطارات كـ ((length - 1) // 4) + 1. |
