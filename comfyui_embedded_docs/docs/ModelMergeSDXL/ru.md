Узел ModelMergeSDXL позволяет смешивать две модели SDXL, регулируя влияние каждой модели на различные части архитектуры. Вы можете контролировать, насколько каждая модель вносит вклад во временные эмбеддинги, эмбеддинги меток и различные блоки внутри структуры модели. Это создает гибридную модель, объединяющую характеристики обеих входных моделей.
