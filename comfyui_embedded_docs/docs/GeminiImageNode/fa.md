> این مستند با هوش مصنوعی تهیه شده است. اگر خطایی دیدید یا پیشنهادی برای بهبود دارید، خوشحال می‌شویم مشارکت کنید! [ویرایش در GitHub](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/GeminiImageNode/fa.md)

گره GeminiImage پاسخ‌های متنی و تصویری را از مدل‌های هوش مصنوعی Gemini گوگل تولید می‌کند. این گره به شما امکان می‌دهد ورودی‌های چندوجهی شامل پرامپت‌های متنی، تصاویر و فایل‌ها را برای ایجاد خروجی‌های متنی و تصویری منسجم ارائه دهید. این گره تمام ارتباطات API و تجزیه پاسخ‌ها را با جدیدترین مدل‌های Gemini مدیریت می‌کند.

## ورودی‌ها

| پارامتر | نوع داده | نوع ورودی | پیش‌فرض | محدوده | توضیحات |
|---------|----------|-----------|---------|--------|---------|
| `prompt` | STRING | اجباری | "" | - | پرامپت متنی برای تولید |
| `model` | COMBO | اجباری | gemini_2_5_flash_image_preview | مدل‌های موجود Gemini<br>گزینه‌های استخراج‌شده از شمارش GeminiImageModel | مدل Gemini مورد استفاده برای تولید پاسخ‌ها |
| `seed` | INT | اجباری | 42 | 0 تا 18446744073709551615 | هنگامی که seed روی یک مقدار مشخص ثابت می‌شود، مدل بهترین تلاش خود را برای ارائه پاسخ یکسان به درخواست‌های تکراری انجام می‌دهد. خروجی قطعی تضمین نمی‌شود. همچنین، تغییر مدل یا تنظیمات پارامترها، مانند دما، می‌تواند باعث تغییر در پاسخ حتی با استفاده از مقدار seed یکسان شود. به‌طور پیش‌فرض، از یک مقدار seed تصادفی استفاده می‌شود |
| `images` | IMAGE | اختیاری | None | - | تصویر(های) اختیاری برای استفاده به‌عنوان زمینه برای مدل. برای گنجاندن چندین تصویر، می‌توانید از گره Batch Images استفاده کنید |
| `files` | GEMINI_INPUT_FILES | اختیاری | None | - | فایل(های) اختیاری برای استفاده به‌عنوان زمینه برای مدل. ورودی‌های گره Gemini Generate Content Input Files را می‌پذیرد |

*توجه: این گره شامل پارامترهای پنهان (`auth_token`، `comfy_api_key`، `unique_id`) است که به‌طور خودکار توسط سیستم مدیریت می‌شوند و نیازی به ورودی کاربر ندارند.*

## خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-----------|----------|---------|
| `IMAGE` | IMAGE | پاسخ تصویری تولیدشده از مدل Gemini |
| `STRING` | STRING | پاسخ متنی تولیدشده از مدل Gemini |
