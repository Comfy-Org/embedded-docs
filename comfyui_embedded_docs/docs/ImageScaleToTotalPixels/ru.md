Вот перевод документации на русский язык:

Узел ImageScaleToTotalPixels предназначен для изменения размера изображений до указанного общего количества пикселей с сохранением соотношения сторон. Он предоставляет различные методы для апскейла изображения для достижения желаемого количества пикселей.

## Входы

| Параметр         | Тип данных   | Описание                                                                 |
|------------------|--------------|--------------------------------------------------------------------------|
| `изображение`          | `IMAGE`      | Входное изображение, которое будет увеличено до указанного общего количества пикселей. |
| `метод_апскейла` | COMBO[STRING] | Метод, используемый для увеличения изображения. Влияет на качество и характеристики увеличенного изображения. |
| `мегапиксели`     | `FLOAT`      | Целевой размер изображения в мегапикселях. Определяет общее количество пикселей в увеличенном изображении. |

## Выходы

| Параметр | Тип данных | Описание                                                              |
|----------|------------|-----------------------------------------------------------------------|
| `изображение`  | `IMAGE`    | Увеличенное изображение с указанным общим количеством пикселей, сохраняющее исходное соотношение сторон. |
