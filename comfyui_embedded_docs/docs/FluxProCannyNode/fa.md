### نمای کلی

تصویری را با استفاده از یک تصویر کنترلی (canny) تولید کنید. این گره یک تصویر کنترلی دریافت کرده و بر اساس prompt ارائه‌شده، تصویر جدیدی تولید می‌کند که از ساختار لبه‌های شناسایی‌شده در تصویر کنترلی پیروی می‌کند.

## ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `control_image` | IMAGE | بله | - | تصویر ورودی که برای کنترل تشخیص لبه Canny استفاده می‌شود |
| `prompt` | STRING | خیر | - | prompt برای تولید تصویر (پیش‌فرض: رشته خالی) |
| `prompt_upsampling` | BOOLEAN | خیر | - | آیا بر روی prompt افزایش نمونه (upsampling) انجام شود یا خیر. در صورت فعال بودن، به‌طور خودکار prompt را برای تولید خلاقانه‌تر تغییر می‌دهد، اما نتایج غیرقطعی هستند (همان seed نتیجه دقیقاً یکسانی تولید نمی‌کند). (پیش‌فرض: False) |
| `canny_low_threshold` | FLOAT | خیر | 0.01 - 0.99 | آستانه پایین برای تشخیص لبه Canny؛ در صورت True بودن `skip_preprocessing` نادیده گرفته می‌شود (پیش‌فرض: 0.1) |
| `canny_high_threshold` | FLOAT | خیر | 0.01 - 0.99 | آستانه بالا برای تشخیص لبه Canny؛ در صورت True بودن `skip_preprocessing` نادیده گرفته می‌شود (پیش‌فرض: 0.4) |
| `skip_preprocessing` | BOOLEAN | خیر | - | آیا پیش‌پردازش نادیده گرفته شود یا خیر؛ اگر `control_image` از قبل به تصویر Canny تبدیل شده است، True و اگر تصویر خام است، False تنظیم کنید. (پیش‌فرض: False) |
| `guidance` | FLOAT | خیر | 1 - 100 | قدرت هدایت برای فرآیند تولید تصویر (پیش‌فرض: 30) |
| `steps` | INT | خیر | 15 - 50 | تعداد مراحل برای فرآیند تولید تصویر (پیش‌فرض: 50) |
| `seed` | INT | خیر | 0 - 18446744073709551615 | seed تصادفی مورد استفاده برای ایجاد نویز. (پیش‌فرض: 0) |

**نکته:** هنگامی که `skip_preprocessing` روی True تنظیم شده است، پارامترهای `canny_low_threshold` و `canny_high_threshold` نادیده گرفته می‌شوند، زیرا فرض بر این است که تصویر کنترلی از قبل به عنوان یک تصویر لبه Canny پردازش شده است. سپس `control_image` مستقیماً به عنوان تصویر پیش‌پردازش‌شده استفاده می‌شود.
