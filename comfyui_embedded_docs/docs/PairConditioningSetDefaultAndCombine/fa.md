گره **PairConditioningSetDefaultAndCombine** مقادیر شرطی پیش‌فرض را تنظیم کرده و آن‌ها را با داده‌های شرطی ورودی ترکیب می‌کند. این گره ورودی‌های شرطی مثبت و منفی را به همراه همتایان پیش‌فرض آن‌ها دریافت کرده، سپس از طریق سیستم هوک ComfyUI پردازش می‌کند تا خروجی‌های شرطی نهایی را تولید کند که مقادیر پیش‌فرض را در خود جای داده‌اند.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|---------|----------|--------|--------|---------|
| `positive` | CONDITIONING | بله | - | ورودی شرطی مثبت اصلی که باید پردازش شود |
| `negative` | CONDITIONING | بله | - | ورودی شرطی منفی اصلی که باید پردازش شود |
| `positive_DEFAULT` | CONDITIONING | بله | - | مقادیر شرطی مثبت پیش‌فرض که به‌عنوان جایگزین استفاده می‌شوند |
| `negative_DEFAULT` | CONDITIONING | بله | - | مقادیر شرطی منفی پیش‌فرض که به‌عنوان جایگزین استفاده می‌شوند |
| `hooks` | HOOKS | خیر | - | گروه هوک اختیاری برای منطق پردازش سفارشی |
