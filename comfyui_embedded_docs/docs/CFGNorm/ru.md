Вот перевод документации на русский язык:

Узел CFGNorm применяет технику нормализации к процессу бесклассового направляющего воздействия (CFG) в диффузионных моделях. Он корректирует масштаб предсказания с удалением шума путем сравнения норм условных и безусловных выходных данных, а затем применяет множитель силы для контроля эффекта. Это помогает стабилизировать процесс генерации, предотвращая экстремальные значения в масштабировании направляющего воздействия.

## Входные данные

| Параметр | Тип данных | Тип ввода | По умолчанию | Диапазон | Описание |
|----------|------------|-----------|--------------|----------|----------|
| `модель` | MODEL | обязательный | - | - | Диффузионная модель, к которой применяется нормализация CFG |
| `интенсивность` | FLOAT | обязательный | 1.0 | 0.0 - 100.0 | Управляет интенсивностью эффекта нормализации, применяемого к масштабированию CFG |

## Выходные данные

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `patched_model` | MODEL | Возвращает модифицированную модель с примененной нормализацией CFG к ее процессу семплирования |
