Узел AudioMerge объединяет две аудиодорожки путем наложения их波形. Он автоматически согласовывает частоты дискретизации обоих аудиовходов и выравнивает их длину перед объединением. Узел предоставляет несколько математических методов для комбинирования аудиосигналов и гарантирует, что выходной сигнал остается в пределах допустимого уровня громкости.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `audio1` | AUDIO | Да | - | Первый аудиовход для объединения |
| `audio2` | AUDIO | Да | - | Второй аудиовход для объединения |
| `merge_method` | COMBO | Да | `"add"`<br>`"mean"`<br>`"subtract"`<br>`"multiply"` | Метод, используемый для комбинирования аудиоволн. |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `AUDIO` | AUDIO | Выходной объединенный аудиосигнал, содержащий комбинированную форму волны и частоту дискретизации |
