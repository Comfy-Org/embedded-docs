### **ورودی‌ها**

| پارامتر | نوع داده | اجباری | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `prompt` | STRING | بله | - | توضیحات متنی برای تولید مدل سه‌بعدی (ورودی چندخطی) |
| `negative_prompt` | STRING | خیر | - | توضیحات متنی درباره مواردی که باید در مدل تولیدشده اجتناب شود (ورودی چندخطی) |
| `model_version` | COMBO | خیر | چندین گزینه موجود | نسخه مدل Tripo برای تولید (پیش‌فرض: v2.5-20250123) |
| `style` | COMBO | خیر | چندین گزینه موجود | تنظیم سبک برای مدل تولیدشده (پیش‌فرض: "هیچ‌کدام") |
| `texture` | BOOLEAN | خیر | - | آیا بافت برای مدل تولید شود (پیش‌فرض: True) |
| `pbr` | BOOLEAN | خیر | - | آیا متریال‌های PBR (رندرینگ مبتنی بر فیزیک) تولید شود (پیش‌فرض: True) |
| `image_seed` | INT | خیر | - | دانه تصادفی برای تولید تصویر (پیش‌فرض: 42) |
| `model_seed` | INT | خیر | - | دانه تصادفی برای تولید مدل (پیش‌فرض: 42) |
| `texture_seed` | INT | خیر | - | دانه تصادفی برای تولید بافت (پیش‌فرض: 42) |
| `texture_quality` | COMBO | خیر | "standard"<br>"detailed" | سطح کیفیت برای تولید بافت (پیش‌فرض: "standard") |
| `face_limit` | INT | خیر | 1- تا 2000000 | حداکثر تعداد وجه‌ها در مدل تولیدشده، 1- به معنای بدون محدودیت (پیش‌فرض: 1-) |
| `quad` | BOOLEAN | خیر | - | آیا هندسه مبتنی بر چهارضلعی به جای مثلث تولید شود (پیش‌فرض: False) |
| `geometry_quality` | COMBO | خیر | "standard"<br>"detailed" | سطح کیفیت برای تولید هندسه (پیش‌فرض: "standard") |

**توجه:** پارامتر `prompt` اجباری است و نمی‌تواند خالی باشد. اگر هیچ متنی ارائه نشود، گره خطا ایجاد می‌کند.

### **خروجی‌ها**

| نام خروجی | نوع داده | توضیحات |
|-------------|-----------|-------------|
| `model_file` | STRING | فایل مدل سه‌بعدی تولیدشده (فقط برای سازگاری با نسخه‌های قبلی) |
| `model task_id` | MODEL_TASK_ID | شناسه یکتای وظیفه برای فرآیند تولید مدل |
| `GLB` | FILE3DGLB | مدل سه‌بعدی تولیدشده در قالب GLB |
