# Tripo 텍스트-3D 모델 노드

Tripo의 API를 사용하여 텍스트 프롬프트를 기반으로 3D 모델을 동기식으로 생성합니다. 이 노드는 텍스트 설명을 받아 선택적 텍스처 및 재질 속성과 함께 3D 모델을 생성합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `프롬프트` | STRING | 예 | - | 3D 모델 생성을 위한 텍스트 설명 (여러 줄 입력 가능) |
| `네거티브 프롬프트` | STRING | 아니요 | - | 생성된 모델에서 피해야 할 내용에 대한 텍스트 설명 (여러 줄 입력 가능) |
| `모델 버전` | COMBO | 아니요 | 여러 옵션 사용 가능 | 생성에 사용할 Tripo 모델 버전 (기본값: v2.5-20250123) |
| `스타일` | COMBO | 아니요 | 여러 옵션 사용 가능 | 생성된 모델의 스타일 설정 (기본값: "None") |
| `텍스처` | BOOLEAN | 아니요 | - | 모델에 텍스처를 생성할지 여부 (기본값: True) |
| `PBR` | BOOLEAN | 아니요 | - | PBR(물리 기반 렌더링) 재질을 생성할지 여부 (기본값: True) |
| `이미지 시드` | INT | 아니요 | - | 이미지 생성을 위한 무작위 시드 (기본값: 42) |
| `모델 시드` | INT | 아니요 | - | 모델 생성을 위한 무작위 시드 (기본값: 42) |
| `텍스처 시드` | INT | 아니요 | - | 텍스처 생성을 위한 무작위 시드 (기본값: 42) |
| `텍스처 품질` | COMBO | 아니요 | "standard"<br>"detailed" | 텍스처 생성 품질 수준 (기본값: "standard") |
| `얼굴 제한` | INT | 아니요 | -1 ~ 2000000 | 생성된 모델의 최대 면 수, -1은 제한 없음 (기본값: -1) |
| `쿼드` | BOOLEAN | 아니요 | - | 삼각형 대신 사각형 기반 지오메트리를 생성할지 여부 (기본값: False) |
| `geometry_quality` | COMBO | 아니요 | "standard"<br>"detailed" | 지오메트리 생성 품질 수준 (기본값: "standard") |

**참고:** `prompt` 매개변수는 필수이며 비워둘 수 없습니다. 프롬프트가 제공되지 않으면 노드에서 오류가 발생합니다.

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `모델 task_id` | STRING | 생성된 3D 모델 파일 (하위 호환성 전용) |
| `GLB` | MODEL_TASK_ID | 모델 생성 프로세스의 고유 작업 식별자 |
| `GLB` | FILE3DGLB | GLB 형식으로 생성된 3D 모델 |
