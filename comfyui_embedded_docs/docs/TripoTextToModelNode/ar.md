بناءً على دورك كخبير في الترجمة التقنية لتوثيق عُقد ComfyUI، إليك الترجمة العربية للوثيقة المطلوبة مع الالتزام التام بقواعد الترجمة المحددة:

---

أنشئ نماذج ثلاثية الأبعاد بشكل متزامن بناءً على وصف نصي باستخدام واجهة برمجة تطبيقات Tripo. تأخذ هذه العقدة وصفًا نصيًا وتُنشئ نموذجًا ثلاثي الأبعاد مع خصائص نسيج ومواد اختيارية.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|-----------|-----------|----------|-------|-------------|
| `المطالبة` | STRING | نعم | - | وصف نصي لإنشاء النموذج ثلاثي الأبعاد (إدخال متعدد الأسطر) |
| `النص_السلبي` | STRING | لا | - | وصف نصي لما يجب تجنبه في النموذج المُنشأ (إدخال متعدد الأسطر) |
| `إصدار_النموذج` | COMBO | لا | تتوفر خيارات متعددة | إصدار نموذج Tripo المستخدم في الإنشاء (الافتراضي: v2.5-20250123) |
| `النمط` | COMBO | لا | تتوفر خيارات متعددة | إعداد النمط للنموذج المُنشأ (الافتراضي: "لا شيء") |
| `الملمس` | BOOLEAN | لا | - | هل سيتم إنشاء نسيج للنموذج (الافتراضي: صحيح) |
| `PBR` | BOOLEAN | لا | - | هل سيتم إنشاء مواد عرض فيزيائي (PBR) (الافتراضي: صحيح) |
| `بذرة_الصورة` | INT | لا | - | بذرة عشوائية لإنشاء الصورة (الافتراضي: 42) |
| `بذرة_النموذج` | INT | لا | - | بذرة عشوائية لإنشاء النموذج (الافتراضي: 42) |
| `بذرة_الملمس` | INT | لا | - | بذرة عشوائية لإنشاء النسيج (الافتراضي: 42) |
| `جودة_الملمس` | COMBO | لا | "قياسي"<br>"مفصل" | مستوى جودة إنشاء النسيج (الافتراضي: "قياسي") |
| `حد_الوجه` | INT | لا | -1 إلى 2000000 | الحد الأقصى لعدد الأسطح في النموذج المُنشأ، -1 يعني بدون حد (الافتراضي: -1) |
| `رباعي` | BOOLEAN | لا | - | هل سيتم إنشاء هندسة رباعية الأضلاع بدلاً من المثلثات (الافتراضي: خطأ) |
| `جودة الهندسة` | COMBO | لا | "قياسي"<br>"مفصل" | مستوى جودة إنشاء الهندسة (الافتراضي: "قياسي") |

**ملاحظة:** معامل `prompt` إلزامي ولا يمكن أن يكون فارغًا. إذا لم يتم توفير وصف نصي، ستُظهر العقدة خطأً.

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|-------------|-----------|-------------|
| `معرّف مهمة النموذج` | STRING | ملف النموذج ثلاثي الأبعاد المُنشأ (للتّوافق مع الإصدارات السابقة فقط) |
| `GLB` | MODEL_TASK_ID | المعرف الفريد للمهمة الخاص بعملية إنشاء النموذج |
| `GLB` | FILE3DGLB | النموذج ثلاثي الأبعاد المُنشأ بتنسيق GLB |
