# Tripo 文字轉模型節點

根據文字提示使用 Tripo API 同步生成 3D 模型。此節點接收文字描述，並建立具有可選紋理和材質屬性的 3D 模型。

## 輸入

| 參數 | 資料類型 | 必要 | 範圍 | 說明 |
|-----------|-----------|----------|-------|-------------|
| `提示詞` | STRING | 是 | - | 用於生成 3D 模型的文字描述（多行輸入） |
| `負向提示詞` | STRING | 否 | - | 描述生成模型中應避免內容的文字（多行輸入） |
| `模型版本` | COMBO | 否 | 提供多個選項 | 用於生成的 Tripo 模型版本（預設：v2.5-20250123） |
| `風格` | COMBO | 否 | 提供多個選項 | 生成模型的樣式設定（預設："None"） |
| `紋理` | BOOLEAN | 否 | - | 是否為模型生成紋理（預設：True） |
| `PBR材質` | BOOLEAN | 否 | - | 是否生成 PBR（基於物理的渲染）材質（預設：True） |
| `圖片種子` | INT | 否 | - | 影像生成的隨機種子（預設：42） |
| `模型種子` | INT | 否 | - | 模型生成的隨機種子（預設：42） |
| `紋理種子` | INT | 否 | - | 紋理生成的隨機種子（預設：42） |
| `紋理品質` | COMBO | 否 | "standard"<br>"detailed" | 紋理生成的品質等級（預設："standard"） |
| `面數限制` | INT | 否 | -1 至 2000000 | 生成模型中的最大面數，-1 表示無限制（預設：-1） |
| `四邊形` | BOOLEAN | 否 | - | 是否生成四邊形幾何體而非三角形（預設：False） |
| `幾何品質` | COMBO | 否 | "standard"<br>"detailed" | 幾何體生成的品質等級（預設："standard"） |
