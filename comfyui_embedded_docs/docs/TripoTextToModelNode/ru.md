Вот перевод документации на русский язык:

Генерирует 3D-модели синхронно на основе текстового запроса с использованием API Tripo. Этот узел принимает текстовое описание и создает 3D-модель с опциональными свойствами текстуры и материалов.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `промпт` | STRING | Да | - | Текстовое описание для генерации 3D-модели (многострочный ввод) |
| `негативный_промпт` | STRING | Нет | - | Текстовое описание того, чего следует избегать в сгенерированной модели (многострочный ввод) |
| `версия_модели` | COMBO | Нет | Доступно несколько вариантов | Версия модели Tripo для генерации (по умолчанию: v2.5-20250123) |
| `стиль` | COMBO | Нет | Доступно несколько вариантов | Настройка стиля для сгенерированной модели (по умолчанию: "None") |
| `текстура` | BOOLEAN | Нет | - | Нужно ли генерировать текстуры для модели (по умолчанию: True) |
| `PBR` | BOOLEAN | Нет | - | Нужно ли генерировать PBR-материалы (физически корректный рендеринг) (по умолчанию: True) |
| `image_seed` | INT | Нет | - | Случайное зерно для генерации изображения (по умолчанию: 42) |
| `model_seed` | INT | Нет | - | Случайное зерно для генерации модели (по умолчанию: 42) |
| `texture_seed` | INT | Нет | - | Случайное зерно для генерации текстуры (по умолчанию: 42) |
| `texture_quality` | COMBO | Нет | "standard"<br>"detailed" | Уровень качества генерации текстуры (по умолчанию: "standard") |
| `face_limit` | INT | Нет | от -1 до 2000000 | Максимальное количество полигонов в сгенерированной модели, -1 без ограничений (по умолчанию: -1) |
| `quad` | BOOLEAN | Нет | - | Нужно ли генерировать геометрию на основе четырехугольников вместо треугольников (по умолчанию: False) |
| `качество геометрии` | COMBO | Нет | "standard"<br>"detailed" | Уровень качества генерации геометрии (по умолчанию: "standard") |

**Примечание:** Параметр `prompt` является обязательным и не может быть пустым. Если запрос не указан, узел выдаст ошибку.

## Выходные данные

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `model_file` | STRING | Файл сгенерированной 3D-модели (только для обратной совместимости) |
| `model task_id` | MODEL_TASK_ID | Уникальный идентификатор задачи для процесса генерации модели |
| `GLB` | FILE3DGLB | Сгенерированная 3D-модель в формате GLB |
