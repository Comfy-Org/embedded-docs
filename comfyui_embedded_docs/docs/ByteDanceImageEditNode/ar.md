عقدة تحرير الصور من ByteDance تتيح لك تعديل الصور باستخدام نماذج الذكاء الاصطناعي من ByteDance عبر واجهة برمجة تطبيقات (API). يمكنك تقديم صورة إدخال ونص وصفي يصف التغييرات المطلوبة، وتقوم العقدة بمعالجة الصورة وفقًا لتعليماتك. تدير العقدة الاتصال بواجهة API تلقائيًا وتعيد الصورة المعدلة.

## المدخلات

| المعامل | نوع البيانات | نوع الإدخال | القيمة الافتراضية | النطاق | الوصف |
|---------|--------------|--------------|-------------------|--------|-------|
| `model` | MODEL | COMBO | seededit_3 | خيارات Image2ImageModelName | اسم النموذج |
| `image` | IMAGE | IMAGE | - | - | الصورة الأساسية المراد تعديلها |
| `prompt` | STRING | STRING | "" | - | تعليمات تعديل الصورة |
| `seed` | INT | INT | 0 | 0-2147483647 | البذرة المستخدمة في التوليد |
| `guidance_scale` | FLOAT | FLOAT | 5.5 | 1.0-10.0 | القيمة الأعلى تجعل الصورة تتبع النص الوصفي بشكل أدق |
| `watermark` | BOOLEAN | BOOLEAN | True | - | ما إذا كان سيتم إضافة علامة مائية "مُولّد بالذكاء الاصطناعي" إلى الصورة |

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|------------|--------------|-------|
| `IMAGE` | IMAGE | الصورة المعدلة التي تم إرجاعها من واجهة ByteDance API |
