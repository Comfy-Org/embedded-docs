> این مستند با هوش مصنوعی تهیه شده است. اگر خطایی دیدید یا پیشنهادی برای بهبود دارید، خوشحال می‌شویم مشارکت کنید! [ویرایش در GitHub](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/ByteDanceImageEditNode/fa.md)

گره ویرایش تصویر ByteDance به شما امکان می‌دهد با استفاده از مدل‌های هوش مصنوعی ByteDance از طریق یک API، تصاویر را ویرایش کنید. شما یک تصویر ورودی و یک متن راهنما (prompt) که تغییرات مورد نظر را توصیف می‌کند، ارائه می‌دهید و گره تصویر را مطابق دستورات شما پردازش می‌کند. این گره به‌طور خودکار ارتباط API را مدیریت کرده و تصویر ویرایش‌شده را بازمی‌گرداند.

## ورودی‌ها

| پارامتر | نوع داده | نوع ورودی | پیش‌فرض | محدوده | توضیحات |
|---------|----------|-----------|---------|--------|---------|
| `model` | MODEL | COMBO | seededit_3 | گزینه‌های Image2ImageModelName | نام مدل |
| `image` | IMAGE | IMAGE | - | - | تصویر پایه برای ویرایش |
| `prompt` | STRING | STRING | "" | - | دستورالعمل ویرایش تصویر |
| `seed` | INT | INT | 0 | 0-2147483647 | دانه (seed) مورد استفاده برای تولید |
| `guidance_scale` | FLOAT | FLOAT | 5.5 | 1.0-10.0 | مقدار بالاتر باعث می‌شود تصویر بیشتر از راهنما پیروی کند |
| `watermark` | BOOLEAN | BOOLEAN | True | - | آیا واترمارک "تولید شده با هوش مصنوعی" به تصویر اضافه شود |
