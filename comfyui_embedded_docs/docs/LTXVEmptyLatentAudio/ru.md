Узел LTXV Empty Latent Audio создаёт пакет пустых (заполненных нулями) тензоров скрытого аудиопространства. Он использует конфигурацию из предоставленной модели Audio VAE для определения правильных размерностей скрытого пространства, таких как количество каналов и частотных бинов. Этот пустой латент служит отправной точкой для рабочих процессов генерации или обработки аудио в ComfyUI.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `frames_number` | INT | Да | от 1 до 1000 | Количество кадров. Значение по умолчанию: 97. |
| `frame_rate` | INT | Да | от 1 до 1000 | Количество кадров в секунду. Значение по умолчанию: 25. |
| `batch_size` | INT | Да | от 1 до 4096 | Количество образцов скрытого аудио в пакете. Значение по умолчанию: 1. |
| `audio_vae` | VAE | Да | Н/Д | Модель Audio VAE для получения конфигурации. Этот параметр обязателен. |

**Примечание:** Входной параметр `audio_vae` является обязательным. Узел выдаст ошибку, если он не предоставлен.

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `Latent` | LATENT | Пустой тензор скрытого аудио со структурой (batch_size, z_channels, num_audio_latents, audio_freq), настроенной в соответствии с входной моделью Audio VAE. Выходные данные также содержат поле `type` со значением "audio". |
