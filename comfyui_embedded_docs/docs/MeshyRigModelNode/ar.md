# ترجمة وثيقة عقدة Meshy: Rig Model

## نظرة عامة

تأخذ عقدة Meshy: Rig Model نموذجًا ثلاثي الأبعاد من مهمة Meshy سابقة وتنشئ هيكلًا عظميًا له تلقائيًا، مما ينتج شخصية مجهزة بالهيكل العظمي يمكن تحريكها وتوجيهها. تُخرج العقدة النموذج المجهز بالهيكل العظمي بصيغتي GLB وFBX.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|---------|---------------|--------|--------|-------|
| `meshy_task_id` | STRING | نعم | غير متاح | معرف المهمة الفريد من عملية Meshy سابقة (مثل تحويل النص إلى ثلاثي الأبعاد أو الصورة إلى ثلاثي الأبعاد) التي أنتجت النموذج المراد تجهيزه بالهيكل العظمي. |
| `height_meters` | FLOAT | نعم | 0.1 إلى 15.0 | الارتفاع التقريبي لنموذج الشخصية بالأمتار. يساعد في دقة القياس والتجهيز بالهيكل العظمي (القيمة الافتراضية: 1.7). |
| `texture_image` | IMAGE | لا | غير متاح | صورة نسيج اللون الأساسي للنموذج بعد فك التغليف بالأشعة فوق البنفسجية. |

**ملاحظة:** عملية التجهيز التلقائي بالهيكل العظمي غير مناسبة حاليًا للشبكات غير المزخرفة، أو الأصول غير البشرية، أو الأصول البشرية ذات بنية الأطراف والجسم غير الواضحة.

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|-------------|---------------|--------|
| `rig_task_id` | STRING | مخرج قديم للتوافق مع الإصدارات السابقة، يحتوي على اسم ملف نموذج GLB. |
| `GLB` | STRING | معرف المهمة الفريد لعملية التجهيز بالهيكل العظمي هذه، والذي يمكن استخدامه للإشارة إلى النتيجة. |
| `FBX` | FILE3DGLB | نموذج الشخصية ثلاثي الأبعاد المجهز بالهيكل العظمي والمحفوظ بصيغة ملف GLB. |
| `FBX` | FILE3DFBX | نموذج الشخصية ثلاثي الأبعاد المجهز بالهيكل العظمي والمحفوظ بصيغة ملف FBX. |
