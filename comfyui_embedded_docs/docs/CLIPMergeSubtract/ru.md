Узел CLIPMergeSubtract выполняет слияние моделей путем вычитания весов одной модели CLIP из другой. Он создает новую модель CLIP, клонируя первую модель, а затем вычитая ключевые патчи из второй модели с регулируемым множителем для управления силой вычитания. Это позволяет выполнять тонкую настройку смешивания моделей путем удаления определенных характеристик из базовой модели.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `clip1` | CLIP | Да | - | Базовая модель CLIP, которая будет клонирована и изменена |
| `clip2` | CLIP | Да | - | Модель CLIP, ключевые патчи которой будут вычтены из базовой модели |
| `множитель` | FLOAT | Да | от -10.0 до 10.0 | Управляет силой операции вычитания (по умолчанию: 1.0) |

**Примечание:** Узел исключает параметры `.position_ids` и `.logit_scale` из операции вычитания независимо от значения множителя.

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `clip` | CLIP | Результирующая модель CLIP после вычитания весов второй модели из первой |
