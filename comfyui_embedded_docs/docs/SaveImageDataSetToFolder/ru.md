Вот перевод документации на русский язык:

Этот узел сохраняет список изображений в указанную папку в выходном каталоге ComfyUI. Он принимает несколько изображений на вход и записывает их на диск с настраиваемым префиксом имени файла.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `images` | IMAGE | Да | Н/Д | Список изображений для сохранения. |
| `folder_name` | STRING | Нет | Н/Д | Имя папки для сохранения изображений (внутри выходного каталога). Значение по умолчанию — "dataset". |
| `filename_prefix` | STRING | Нет | Н/Д | Префикс для имен сохраняемых файлов изображений. Значение по умолчанию — "image". |

**Примечание:** Вход `images` является списком, то есть может принимать и обрабатывать несколько изображений одновременно. Параметры `folder_name` и `filename_prefix` являются скалярными значениями; если к ним подключен список, будет использовано только первое значение из этого списка.

## Выходы

Этот узел не имеет выходов. Он является выходным узлом, который выполняет операцию сохранения в файловую систему.
