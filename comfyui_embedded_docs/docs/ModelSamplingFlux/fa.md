گره ModelSamplingFlux با محاسبه یک پارامتر شیفت بر اساس ابعاد تصویر، نمونه‌گیری مدل Flux را روی یک مدل مشخص اعمال می‌کند. این گره یک پیکربندی نمونه‌گیری تخصصی ایجاد می‌کند که رفتار مدل را مطابق پارامترهای عرض، ارتفاع و شیفت تنظیم می‌کند، سپس مدل اصلاح‌شده را با تنظیمات نمونه‌گیری جدید بازمی‌گرداند.

## ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|---------|----------|-------|--------|---------|
| `مدل` | MODEL | بله | - | مدلی که نمونه‌گیری Flux روی آن اعمال می‌شود |
| `بیشینه شیفت` | FLOAT | بله | 0.0 - 100.0 | حداکثر مقدار شیفت برای محاسبه نمونه‌گیری (پیش‌فرض: 1.15) |
| `شیفت پایه` | FLOAT | بله | 0.0 - 100.0 | مقدار پایه شیفت برای محاسبه نمونه‌گیری (پیش‌فرض: 0.5) |
| `عرض` | INT | بله | 16 - MAX_RESOLUTION | عرض تصویر هدف بر حسب پیکسل (پیش‌فرض: 1024) |
| `ارتفاع` | INT | بله | 16 - MAX_RESOLUTION | ارتفاع تصویر هدف بر حسب پیکسل (پیش‌فرض: 1024) |
