Узел ModelSamplingFlux применяет сэмплинг модели Flux к заданной модели, вычисляя параметр сдвига на основе размеров изображения. Он создает специализированную конфигурацию сэмплинга, которая корректирует поведение модели в соответствии с указанными параметрами ширины, высоты и сдвига, затем возвращает модифицированную модель с примененными новыми настройками сэмплинга.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `model` | MODEL | Да | - | Модель, к которой применяется сэмплинг Flux |
| `max_shift` | FLOAT | Да | 0.0 - 100.0 | Максимальное значение сдвига для расчета сэмплинга (по умолчанию: 1.15) |
| `base_shift` | FLOAT | Да | 0.0 - 100.0 | Базовое значение сдвига для расчета сэмплинга (по умолчанию: 0.5) |
| `width` | INT | Да | 16 - MAX_RESOLUTION | Ширина целевого изображения в пикселях (по умолчанию: 1024) |
| `height` | INT | Да | 16 - MAX_RESOLUTION | Высота целевого изображения в пикселях (по умолчанию: 1024) |

## Выходные параметры

| Выходной параметр | Тип данных | Описание |
|-------------|-----------|-------------|
| `model` | MODEL | Модифицированная модель с примененной конфигурацией сэмплинга Flux |
