Вот перевод документации на русский язык:

Узел ModelSamplingFlux применяет сэмплирование модели Flux к заданной модели путем вычисления параметра сдвига на основе размеров изображения. Он создает специализированную конфигурацию сэмплирования, которая корректирует поведение модели в соответствии с указанными шириной, высотой и параметрами сдвига, после чего возвращает модифицированную модель с примененными новыми настройками сэмплирования.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `модель` | MODEL | Да | - | Модель, к которой применяется сэмплирование Flux |
| `макс_сдвиг` | FLOAT | Да | 0.0 - 100.0 | Максимальное значение сдвига для расчета сэмплирования (по умолчанию: 1.15) |
| `базовый_сдвиг` | FLOAT | Да | 0.0 - 100.0 | Базовое значение сдвига для расчета сэмплирования (по умолчанию: 0.5) |
| `ширина` | INT | Да | 16 - MAX_RESOLUTION | Ширина целевого изображения в пикселях (по умолчанию: 1024) |
| `высота` | INT | Да | 16 - MAX_RESOLUTION | Высота целевого изображения в пикселях (по умолчанию: 1024) |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `модель` | MODEL | Модифицированная модель с примененной конфигурацией сэмплирования Flux |
