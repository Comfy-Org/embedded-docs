Узел ModelMergeFlux1 объединяет две диффузионные модели путем смешивания их компонентов с использованием взвешенной интерполяции. Он обеспечивает тонкий контроль над тем, как различные части моделей комбинируются, включая блоки обработки изображений, слои внедрения времени, механизмы управления, векторные входы, текстовые кодировщики и различные блоки трансформера. Это позволяет создавать гибридные модели с настраиваемыми характеристиками из двух исходных моделей.
