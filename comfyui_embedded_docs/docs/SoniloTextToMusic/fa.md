### مرور کلی

گره Sonilo Text to Music با استفاده از مدل هوش مصنوعی Sonilo، از روی توضیحات متنی، موسیقی تولید می‌کند. شما یک prompt (دستور متنی) شامل توضیح موسیقی مورد نظر خود ارائه می‌دهید و گره درخواستی را به سرویس Sonilo ارسال می‌کند تا یک فایل صوتی ایجاد شود. می‌توانید مدت زمان هدف را مشخص کنید یا اجازه دهید مدل آن را از روی prompt شما استنباط کند.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|---------|----------|--------|--------|---------|
| `prompt` | STRING | بله | نامشخص | دستور متنی توضیح‌دهنده موسیقی مورد نظر برای تولید. این فیلد الزامی است. |
| `duration` | INT | خیر | 0 تا 360 | مدت زمان هدف بر حسب ثانیه. مقدار 0 به مدل اجازه می‌دهد مدت زمان را از روی prompt استنباط کند. حداکثر: 6 دقیقه (360 ثانیه). پیش‌فرض: 0. |
| `seed` | INT | خیر | 0 تا 18446744073709551615 | دانه (seed) برای قابلیت تکرارپذیری. در حال حاضر توسط سرویس Sonilo نادیده گرفته می‌شود، اما برای سازگاری گراف نگه داشته شده است. پیش‌فرض: 0. |

**توجه:** ورودی `seed` برای سازگاری گردش کار ارائه شده است، اما در حال حاضر بر خروجی سرویس Sonilo تأثیری ندارد.
