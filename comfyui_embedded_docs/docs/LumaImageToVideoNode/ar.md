# ترجمة وثيقة عقدة LumaImageToVideoNode

## نظرة عامة

تُنشئ هذه العقدة فيديوهات بشكل متزامن بناءً على نص وصفي وصور اختيارية للبداية/النهاية. تستخدم هذه العقدة واجهة برمجة تطبيقات Luma لإنشاء الفيديوهات، مما يتيح لك تحديد محتوى الفيديو من خلال نص وصفي، وتحديد الإطار الأول و/أو الأخير اختياريًا للتحكم في بنية الفيديو.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|---------|--------------|--------|--------|-------|
| `prompt` | STRING | نعم | - | النص الوصفي لتوليد الفيديو (الافتراضي: "") |
| `model` | COMBO | نعم | خيارات متعددة متاحة | يختار نموذج توليد الفيديو من نماذج Luma المتاحة |
| `resolution` | COMBO | نعم | `"540p"`<br>`"720p"`<br>`"1080p"`<br>`"4k"` | دقة الفيديو الناتج (الافتراضي: "540p"). يتم تجاهل هذا المعامل عند استخدام نموذج `ray-1-6`. |
| `duration` | COMBO | نعم | `"5s"`<br>`"9s"` | مدة الفيديو المُنشأ. يتم تجاهل هذا المعامل عند استخدام نموذج `ray-1-6`. |
| `loop` | BOOLEAN | نعم | - | ما إذا كان الفيديو المُنشأ يجب أن يكون متكررًا (الافتراضي: خطأ) |
| `seed` | INT | نعم | 0 إلى 18446744073709551615 | البذرة لتحديد ما إذا كانت العقدة ستعيد التشغيل؛ النتائج الفعلية غير حتمية بغض النظر عن البذرة. (الافتراضي: 0) |
| `first_image` | IMAGE | لا | - | الإطار الأول للفيديو المُنشأ. (اختياري) |
| `last_image` | IMAGE | لا | - | الإطار الأخير للفيديو المُنشأ. (اختياري) |
| `luma_concepts` | CUSTOM | لا | - | مفاهيم الكاميرا الاختيارية لتوجيه حركة الكاميرا عبر عقدة Luma Concepts. (اختياري) |

**ملاحظة:** يجب توفير واحد على الأقل من `first_image` أو `last_image`. ستقوم العقدة برفع استثناء في حال عدم وجود كليهما. يتم تجاهل معاملي `resolution` و `duration` عند تعيين النموذج على `ray-1-6`.
