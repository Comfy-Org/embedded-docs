### نمای کلی

بر اساس یک توضیح متنی و تصاویر اختیاری شروع/پایان، ویدیوها را به صورت همزمان تولید می‌کند. این گره از API لومیا برای ساخت ویدیو استفاده می‌کند و به شما امکان می‌دهد محتوای ویدیو را از طریق یک توضیح تعریف کرده و به صورت اختیاری فریم اول و/یا آخر را برای کنترل ساختار ویدیو مشخص کنید.

### ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `prompt` | STRING | بله | - | توضیح برای تولید ویدیو (پیش‌فرض: "") |
| `model` | COMBO | بله | چندین گزینه موجود | مدل تولید ویدیو را از میان مدل‌های موجود لومیا انتخاب می‌کند |
| `resolution` | COMBO | بله | `"540p"`<br>`"720p"`<br>`"1080p"`<br>`"4k"` | وضوح خروجی برای ویدیوی تولید شده (پیش‌فرض: "540p"). این پارامتر هنگام استفاده از مدل `ray-1-6` نادیده گرفته می‌شود. |
| `duration` | COMBO | بله | `"5s"`<br>`"9s"` | مدت زمان ویدیوی تولید شده. این پارامتر هنگام استفاده از مدل `ray-1-6` نادیده گرفته می‌شود. |
| `loop` | BOOLEAN | بله | - | آیا ویدیوی تولید شده باید حلقه شود (پیش‌فرض: False) |
| `seed` | INT | بله | 0 تا 18446744073709551615 | دانه (seed) برای تعیین اینکه آیا گره باید دوباره اجرا شود؛ نتایج واقعی بدون توجه به seed غیرقطعی هستند. (پیش‌فرض: 0) |
| `first_image` | IMAGE | خیر | - | فریم اول ویدیوی تولید شده. (اختیاری) |
| `last_image` | IMAGE | خیر | - | فریم آخر ویدیوی تولید شده. (اختیاری) |
| `luma_concepts` | CUSTOM | خیر | - | مفاهیم دوربین اختیاری برای تعیین حرکت دوربین از طریق گره Luma Concepts. (اختیاری) |

**توجه:** حداقل یکی از `first_image` یا `last_image` باید ارائه شود. در صورت отсутствие هر دو، گره یک استثنا ایجاد می‌کند. پارامترهای `resolution` و `duration` زمانی که `model` روی `ray-1-6` تنظیم شده باشد، نادیده گرفته می‌شوند.
