Вот перевод документации на русский язык:

Узел ColorToRGBInt преобразует цвет, заданный в шестнадцатеричном формате, в целочисленное значение. Он принимает строку цвета, например `#FF5733`, и вычисляет соответствующее целое число RGB путем объединения красного, зеленого и синего компонентов.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `цвет` | STRING | Да | Н/Д | Значение цвета в шестнадцатеричном формате `#RRGGBB`. |

**Примечание:** Входная строка `color` должна содержать ровно 7 символов и начинаться с символа `#`, за которым следуют шесть шестнадцатеричных цифр (например, `#FF0000` для красного цвета). Узел выдаст ошибку, если формат будет некорректным.

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `rgb_int` | INT | Вычисленное целочисленное значение RGB. Оно получается по формуле: `(Red * 65536) + (Green * 256) + Blue`. |
