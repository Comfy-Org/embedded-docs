Вот перевод документации на русский язык:

LazyCache — это домашняя версия EasyCache, которая обеспечивает ещё более простую реализацию. Она работает с любой моделью в ComfyUI и добавляет функцию кэширования для уменьшения вычислений во время сэмплирования. Хотя в целом она работает хуже, чем EasyCache, в некоторых редких случаях она может быть более эффективной и обеспечивает универсальную совместимость.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `модель` | MODEL | Да | - | Модель, к которой добавляется LazyCache. |
| `порог повторного использования` | FLOAT | Нет | 0.0 - 3.0 | Порог повторного использования кэшированных шагов (по умолчанию: 0.2). |
| `start_percent` | FLOAT | Нет | 0.0 - 1.0 | Относительный шаг сэмплирования для начала использования LazyCache (по умолчанию: 0.15). |
| `end_percent` | FLOAT | Нет | 0.0 - 1.0 | Относительный шаг сэмплирования для завершения использования LazyCache (по умолчанию: 0.95). |
| `verbose` | BOOLEAN | Нет | - | Включает подробное логирование (по умолчанию: False). |

## Выходные параметры

| Имя выходного параметра | Тип данных | Описание |
|-------------|-----------|-------------|
| `модель` | MODEL | Модель с добавленной функциональностью LazyCache. |
