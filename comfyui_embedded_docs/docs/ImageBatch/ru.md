Узел `ImageBatch` предназначен для объединения двух изображений в один пакет. Если размеры изображений не совпадают, он автоматически масштабирует второе изображение до размеров первого перед их объединением.

## Входные параметры

| Параметр | Тип данных | Описание |
|-----------|-------------|-------------|
| `image1`  | `IMAGE`     | Первое изображение, объединяемое в пакет. Служит эталоном для размеров, к которым будет приведено второе изображение при необходимости. |
| `image2`  | `IMAGE`     | Второе изображение, объединяемое в пакет. Автоматически масштабируется до размеров первого изображения, если они различаются. |

## Выходные параметры

| Параметр | Тип данных | Описание |
|-----------|-------------|-------------|
| `image`   | `IMAGE`     | Объединённый пакет изображений, где второе изображение при необходимости масштабировано до размеров первого. |
