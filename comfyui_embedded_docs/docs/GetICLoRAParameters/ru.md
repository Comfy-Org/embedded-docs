## Обзор

Этот узел извлекает параметры IC-LoRA из метаданных модели, загруженной с помощью LoRA. Он считывает метаданные safetensors для поиска таких значений, как коэффициент понижения разрешения эталонного изображения, и выводит их в виде структурированного объекта параметров, который можно подключить к узлу LTXVAddGuide для специальной обработки направляющих.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `iclora_model` | MODEL | Да | Н/Д | Прямой выход из загрузчика LoRA для конкретной IC-LoRA, из которой необходимо извлечь метаданные. |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `iclora_parameters` | IC_LORA_PARAMETERS | Параметры IC-LoRA, извлеченные из метаданных LoRA (например, коэффициент понижения разрешения эталонного изображения). Подключите к LTXVAddGuide, если LoRA требует специальной обработки направляющих. |
