ImageScale 節點專為將影像調整至特定尺寸而設計，提供多種放大方法選擇以及裁剪調整後影像的能力。它抽象化了影像放大和裁剪的複雜性，為使用者提供根據自訂參數修改影像尺寸的直觀介面。

## 輸入參數

| 參數名稱         | 資料類型       | 描述說明                                                                           |
|-----------------|-------------|---------------------------------------------------------------------------------------|
| `影像`         | `IMAGE`     | 要進行放大的輸入影像。此參數是節點運作的核心，作為套用尺寸調整轉換的主要資料。輸出影像的品質和尺寸直接受到原始影像屬性的影響。 |
| `放大方法`| COMBO[STRING] | 指定用於影像放大的方法。方法的選擇會影響放大後影像的品質和特性，進而影響調整尺寸後輸出的視覺保真度和可能產生的瑕疵。 |
| `寬度`         | `INT`       | 放大後影像的目標寬度。此參數直接影響輸出影像的尺寸，決定尺寸調整操作的水平縮放比例。 |
| `高度`        | `INT`       | 放大後影像的目標高度。此參數直接影響輸出影像的尺寸，決定尺寸調整操作的垂直縮放比例。 |
| `裁切`          | COMBO[STRING] | 決定是否以及如何裁剪放大後的影像，提供停用裁剪或居中裁剪的選項。這會透過可能移除邊緣以符合指定尺寸的方式，影響影像的最終構圖。 |
