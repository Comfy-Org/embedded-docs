### گره تبدیل گفتار به گفتار ElevenLabs

گره تبدیل گفتار به گفتار ElevenLabs یک فایل صوتی ورودی را از یک صدا به صدای دیگر تبدیل می‌کند. این گره از API ElevenLabs برای تبدیل گفتار استفاده می‌کند و محتوای اصلی و لحن احساسی صدا را حفظ می‌نماید.

## ورودی‌ها

| پارامتر | نوع داده | اجباری | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `voice` | CUSTOM | بله | - | صدای هدف برای تبدیل. از انتخاب‌گر صدا یا کلون صوتی فوری متصل کنید. |
| `audio` | AUDIO | بله | - | صدای منبع برای تبدیل. |
| `stability` | FLOAT | خیر | 0.0 - 1.0 | پایداری صدا. مقادیر پایین‌تر دامنه احساسی گسترده‌تری می‌دهند، مقادیر بالاتر گفتاری یکنواخت‌تر اما احتمالاً خسته‌کننده تولید می‌کنند (پیش‌فرض: 0.5). |
| `model` | DYNAMICCOMBO | خیر | `eleven_multilingual_sts_v2`<br>`eleven_english_sts_v2` | مدل مورد استفاده برای تبدیل گفتار به گفتار. هر گزینه مجموعه‌ای خاص از تنظیمات صدا (افزایش شباهت، سبک، استفاده از تقویت صدا، سرعت) را ارائه می‌دهد. |
| `output_format` | COMBO | خیر | `"mp3_44100_192"`<br>`"opus_48000_192"` | فرمت خروجی صدا (پیش‌فرض: "mp3_44100_192"). |
| `seed` | INT | خیر | 0 - 4294967295 | دانه (seed) برای تکرارپذیری (پیش‌فرض: 0). |
| `remove_background_noise` | BOOLEAN | خیر | - | حذف نویز پس‌زمینه از صدای ورودی با استفاده از جداسازی صدا (پیش‌فرض: False). |
