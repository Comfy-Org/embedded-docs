Вот перевод документации узла ComfyUI **ElevenLabs Speech to Speech** на русский язык:

Узел ElevenLabs Speech to Speech преобразует входной аудиофайл из одного голоса в другой. Он использует API ElevenLabs для преобразования речи, сохраняя исходное содержание и эмоциональный тон аудио.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `голос` | CUSTOM | Да | - | Целевой голос для преобразования. Подключите от Voice Selector или Instant Voice Clone. |
| `аудио` | AUDIO | Да | - | Исходное аудио для преобразования. |
| `стабильность` | FLOAT | Нет | 0.0 - 1.0 | Стабильность голоса. Более низкие значения дают более широкий эмоциональный диапазон, более высокие значения обеспечивают более последовательную, но потенциально монотонную речь (по умолчанию: 0.5). |
| `модель` | DYNAMICCOMBO | Нет | `eleven_multilingual_sts_v2`<br>`eleven_english_sts_v2` | Модель для преобразования речи в речь. Каждый вариант предоставляет определенный набор голосовых настроек (similarity_boost, style, use_speaker_boost, speed). |
| `формат_выхода` | COMBO | Нет | `"mp3_44100_192"`<br>`"opus_48000_192"` | Формат выходного аудио (по умолчанию: "mp3_44100_192"). |
| `seed` | INT | Нет | 0 - 4294967295 | Зерно для воспроизводимости результатов (по умолчанию: 0). |
| `удалить_фоновый_шум` | BOOLEAN | Нет | - | Удалить фоновый шум из входного аудио с помощью изоляции звука (по умолчанию: False). |

## Выходы

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `аудио` | AUDIO | Преобразованный аудиофайл в указанном формате вывода. |
