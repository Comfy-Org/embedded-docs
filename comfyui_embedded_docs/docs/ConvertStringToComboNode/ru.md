Вот перевод документации на русский язык:

Узел Convert String to Combo принимает текстовую строку на входе и преобразует её в тип данных Combo. Это позволяет использовать текстовое значение в качестве варианта выбора для других узлов, которым требуется входной параметр типа Combo. Узел просто передаёт строковое значение без изменений, но меняет его тип данных.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `string` | STRING | Да | Н/Д | Текстовая строка, преобразуемая в тип Combo. |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `output` | COMBO | Входная строка, теперь отформатированная как тип данных Combo. |
