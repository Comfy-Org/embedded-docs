Узел **LatentUpscale** предназначен для масштабирования латентных представлений изображений. Он позволяет регулировать размеры выходного изображения и метод масштабирования, обеспечивая гибкость при повышении разрешения латентных изображений.

## Входные параметры

| Параметр | Тип данных | Описание |
|-----------|-------------|-------------|
| `образцы` | `LATENT` | Латентное представление изображения, которое необходимо масштабировать. Этот параметр является ключевым для определения начальной точки процесса масштабирования. |
| `метод_апскейла` | COMBO[STRING] | Определяет метод, используемый для масштабирования латентного изображения. Разные методы могут влиять на качество и характеристики увеличенного изображения. |
| `ширина` | `INT` | Желаемая ширина увеличенного изображения. Если установлено значение 0, ширина будет рассчитана на основе высоты для сохранения соотношения сторон. |
| `высота` | `INT` | Желаемая высота увеличенного изображения. Если установлено значение 0, высота будет рассчитана на основе ширины для сохранения соотношения сторон. |
| `обрезка` | COMBO[STRING] | Определяет способ кадрирования увеличенного изображения, влияя на окончательный внешний вид и размеры выходного результата. |

## Выходные параметры

| Параметр | Тип данных | Описание |
|-----------|-------------|-------------|
| `latent` | `LATENT` | Увеличенное латентное представление изображения, готовое к дальнейшей обработке или генерации. |
