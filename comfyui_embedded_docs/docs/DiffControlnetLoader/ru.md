Этот узел обнаруживает модели, расположенные в папке `ComfyUI/models/controlnet`, а также считывает модели из дополнительных путей, указанных в файле extra_model_paths.yaml. Иногда может потребоваться **обновить интерфейс ComfyUI**, чтобы он смог прочитать файлы моделей из соответствующей папки.

Узел DiffControlNetLoader предназначен для загрузки дифференциальных управляющих сетей (control networks) — специализированных моделей, которые могут изменять поведение другой модели на основе спецификаций управляющей сети. Этот узел позволяет динамически настраивать поведение моделей путем применения дифференциальных управляющих сетей, что упрощает создание настраиваемых выходных данных моделей.

## Входные данные

| Поле                | Comfy dtype       | Описание                                                                                 |
|---------------------|-------------------|------------------------------------------------------------------------------------------|
| `модель`             | `MODEL`           | Базовая модель, к которой будет применена дифференциальная управляющая сеть, что позволяет настраивать поведение модели. |
| `название_control_net`  | `COMBO[STRING]`    | Определяет конкретную дифференциальную управляющую сеть, которая будет загружена и применена к базовой модели для изменения ее поведения. |

## Выходные данные

| Поле          | Comfy dtype   | Описание                                                                   |
|---------------|---------------|----------------------------------------------------------------------------|
| `control_net` | `CONTROL_NET` | Дифференциальная управляющая сеть, которая была загружена и готова к применению к базовой модели для изменения поведения. |
