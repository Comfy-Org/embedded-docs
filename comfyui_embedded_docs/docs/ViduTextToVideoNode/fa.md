### مرور کلی

گره تولید ویدیو از متن Vidu، ویدیوهایی را بر اساس توضیحات متنی ایجاد می‌کند. این گره از مدل تولید ویدیوی Vidu برای تبدیل متن شما به محتوای ویدیویی با تنظیمات قابل‌شخصی‌سازی برای مدت زمان، نسبت تصویر و سبک بصری استفاده می‌کند.

## ورودی‌ها

| پارامتر | نوع داده | اجباری | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `مدل` | COMBO | بله | `viduq1` | نام مدل |
| `پرامپت` | STRING | بله | - | توضیح متنی برای تولید ویدیو |
| `مدت زمان` | INT | خیر | 5-5 | مدت زمان ویدیوی خروجی بر حسب ثانیه (پیش‌فرض: 5) |
| `بذر` | INT | خیر | 0-2147483647 | دانه (seed) برای تولید ویدیو (0 برای تصادفی) (پیش‌فرض: 0) |
| `نسبت تصویر` | COMBO | خیر | `16:9`<br>`9:16`<br>`1:1` | نسبت تصویر ویدیوی خروجی |
| `رزولوشن` | COMBO | خیر | `1080p` | مقادیر پشتیبانی‌شده ممکن است بسته به مدل و مدت زمان متفاوت باشد |
| `دامنه حرکت` | COMBO | خیر | `auto`<br>`small`<br>`medium`<br>`large` | دامنه حرکت اشیاء در قاب |

**نکته:** فیلد `prompt` اجباری است و نمی‌تواند خالی باشد. پارامتر `duration` در حال حاضر روی ۵ ثانیه ثابت است.
