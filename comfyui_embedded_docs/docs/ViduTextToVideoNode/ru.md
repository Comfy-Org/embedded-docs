Вот перевод документации на русский язык:

Узел Vidu Text To Video Generation создает видео на основе текстовых описаний. Он использует модель генерации видео Vidu для преобразования текстовых подсказок в видеоконтент с настраиваемыми параметрами длительности, соотношения сторон и визуального стиля.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `модель` | COMBO | Да | `viduq1` | Название модели |
| `prompt` | STRING | Да | - | Текстовое описание для генерации видео |
| `duration` | INT | Нет | 5-5 | Длительность выходного видео в секундах (по умолчанию: 5) |
| `seed` | INT | Нет | 0-2147483647 | Сид для генерации видео (0 для случайного) (по умолчанию: 0) |
| `aspect_ratio` | COMBO | Нет | `16:9`<br>`9:16`<br>`1:1` | Соотношение сторон выходного видео |
| `resolution` | COMBO | Нет | `1080p` | Поддерживаемые значения могут различаться в зависимости от модели и длительности |
| `movement_amplitude` | COMBO | Нет | `auto`<br>`small`<br>`medium`<br>`large` | Амплитуда движения объектов в кадре |

**Примечание:** Поле `prompt` является обязательным и не может быть пустым. Параметр `duration` в настоящее время фиксирован на значении 5 секунд.

## Выходные данные

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `output` | VIDEO | Сгенерированное видео на основе текстовой подсказки |
