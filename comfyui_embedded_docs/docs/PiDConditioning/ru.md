## Обзор

Прикрепляет скрытое изображение (latent) и значение сигмы деградации к данным CONDITIONING. Используется для декодирования или апскейлинга PiD (Pixel-in-Detail), позволяя контролировать степень деградации скрытого представления перед обработкой.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `positive` | CONDITIONING | Да | - | Данные conditioning, к которым прикрепляются скрытое изображение и сигма деградации. |
| `latent` | LATENT | Да | - | Скрытое изображение (из VAEEncode или KSampler), прикрепляемое к conditioning. |
| `latent_format` | COMBO | Да | `"flux"`<br>`"sd3"` | Формат скрытого представления. Latent Flux1 и Flux2 определяются автоматически по размерности каналов. SD3 необходимо выбирать вручную (по умолчанию: "flux"). |
| `degrade_sigma` | FLOAT | Да | 0.0 до 1.0 (шаг: 0.01) | Степень применяемой деградации. Значение 0 означает чистое скрытое представление. Увеличьте это значение для подавления шума в поврежденных скрытых выходах (по умолчанию: 0.0). |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `CONDITIONING` | CONDITIONING | Исходные данные conditioning с прикрепленными значениями скрытого изображения и сигмы деградации. |
