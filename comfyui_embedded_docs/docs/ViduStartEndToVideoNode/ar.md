# نظرة عامة

تقوم عقدة توليد الفيديو من البداية إلى النهاية Vidu بإنشاء فيديو عن طريق توليد إطارات بين إطار بداية وإطار نهاية. تستخدم وصفًا نصيًا لتوجيه عملية توليد الفيديو، وتدعم نماذج فيديو متنوعة مع إعدادات مختلفة للدقة وحركة الصورة. تتحقق العقدة من توافق نسب الأبعاد بين إطاري البداية والنهاية قبل المعالجة.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|---------|---------------|--------|---------|-------|
| `النموذج` | COMBO | نعم | `"viduq1"` | اسم النموذج |
| `الإطار الأول` | IMAGE | نعم | - | إطار البداية |
| `إطار النهاية` | IMAGE | نعم | - | إطار النهاية |
| `النص التوجيهي` | STRING | لا | - | وصف نصي لتوليد الفيديو |
| `المدة` | INT | لا | 5-5 | مدة الفيديو الناتج بالثواني (الافتراضي: 5، ثابت عند 5 ثوانٍ) |
| `البذرة` | INT | لا | 0-2147483647 | بذرة توليد الفيديو (0 للعشوائية) (الافتراضي: 0) |
| `الدقة` | COMBO | لا | `"1080p"` | قد تختلف القيم المدعومة حسب النموذج والمدة (الافتراضي: "1080p") |
| `سعة الحركة` | COMBO | لا | `"auto"`<br>`"small"`<br>`"medium"`<br>`"large"` | سعة حركة العناصر في الإطار (الافتراضي: "auto") |

**ملاحظة:** يجب أن تكون نسب الأبعاد لإطاري البداية والنهاية متوافقة (يتم التحقق منها بتسامح نسبة min_rel=0.8، max_rel=1.25).
