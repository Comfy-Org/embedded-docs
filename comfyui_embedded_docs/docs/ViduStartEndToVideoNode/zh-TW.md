## 概述

Vidu 起始至結束影片生成節點透過在起始幀和結束幀之間生成影格來建立影片。它使用文字提示來引導影片生成過程，並支援具有不同解析度和動態設定的各種影片模型。在處理之前，該節點會驗證起始幀和結束幀是否具有相容的長寬比。
