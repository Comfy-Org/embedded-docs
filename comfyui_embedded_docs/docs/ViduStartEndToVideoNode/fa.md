### مرور کلی

گره تولید ویدیو از فریم شروع و پایان Vidu، با تولید فریم‌های میانی بین یک فریم شروع و یک فریم پایان، ویدیو ایجاد می‌کند. این گره از یک توضیح متنی برای هدایت فرآیند تولید ویدیو استفاده می‌کند و از مدل‌های ویدیویی مختلف با تنظیمات تفکیک‌پذیری و حرکت گوناگون پشتیبانی می‌کند. پیش از پردازش، گره اطمینان حاصل می‌کند که فریم‌های شروع و پایان نسبت ابعاد سازگاری دارند.

## ورودی‌ها

| پارامتر | نوع داده | اجباری | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `مدل` | COMBO | بله | `"viduq1"` | نام مدل |
| `فریم شروع` | IMAGE | بله | - | فریم شروع |
| `فریم پایان` | IMAGE | بله | - | فریم پایان |
| `پرامپت` | STRING | خیر | - | توضیح متنی برای تولید ویدیو |
| `مدت زمان` | INT | خیر | 5-5 | مدت زمان ویدیوی خروجی بر حسب ثانیه (پیش‌فرض: 5، ثابت در 5 ثانیه) |
| `بذر` | INT | خیر | 0-2147483647 | دانه (seed) برای تولید ویدیو (0 برای تصادفی) (پیش‌فرض: 0) |
| `رزولوشن` | COMBO | خیر | `"1080p"` | مقادیر پشتیبانی‌شده ممکن است بسته به مدل و مدت زمان متفاوت باشد (پیش‌فرض: "1080p") |
| `دامنه حرکت` | COMBO | خیر | `"auto"`<br>`"small"`<br>`"medium"`<br>`"large"` | دامنه حرکت اشیاء در فریم (پیش‌فرض: "auto") |

**نکته:** فریم‌های شروع و پایان باید نسبت ابعاد سازگاری داشته باشند (با تلورانس نسبت min_rel=0.8، max_rel=1.25 تأیید می‌شود).
