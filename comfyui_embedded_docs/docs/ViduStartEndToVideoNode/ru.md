Вот перевод документации на русский язык:

Узел Vidu Start End To Video Generation создает видео путем генерации кадров между начальным и конечным кадрами. Он использует текстовый запрос для управления процессом генерации видео и поддерживает различные видеомодели с разными настройками разрешения и движения. Узел проверяет, что начальный и конечный кадры имеют совместимые соотношения сторон перед обработкой.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `модель` | COMBO | Да | `"viduq1"` | Название модели |
| `первый кадр` | IMAGE | Да | - | Начальный кадр |
| `конечный кадр` | IMAGE | Да | - | Конечный кадр |
| `описание` | STRING | Нет | - | Текстовое описание для генерации видео |
| `длительность` | INT | Нет | 5-5 | Длительность выходного видео в секундах (по умолчанию: 5, фиксировано на 5 секундах) |
| `сид` | INT | Нет | 0-2147483647 | Сид для генерации видео (0 для случайного) (по умолчанию: 0) |
| `разрешение` | COMBO | Нет | `"1080p"` | Поддерживаемые значения могут различаться в зависимости от модели и длительности (по умолчанию: "1080p") |
| `амплитуда движения` | COMBO | Нет | `"auto"`<br>`"small"`<br>`"medium"`<br>`"large"` | Амплитуда движения объектов в кадре (по умолчанию: "auto") |

**Примечание:** Начальный и конечный кадры должны иметь совместимые соотношения сторон (проверяется с допуском соотношения min_rel=0.8, max_rel=1.25).

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `output` | VIDEO | Сгенерированный видеофайл |
