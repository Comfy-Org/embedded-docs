گره **متن به تصویر Runway** تصاویری را از روی توضیحات متنی با استفاده از مدل Gen 4 شرکت Runway تولید می‌کند. شما می‌توانید یک توضیح متنی ارائه دهید و به‌صورت اختیاری یک تصویر مرجع برای هدایت فرآیند تولید تصویر وارد کنید. این گره ارتباط با API را مدیریت کرده و تصویر تولیدشده را بازمی‌گرداند.

## ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|---------|----------|-------|--------|---------|
| `متن ورودی` | STRING | بله | - | توضیح متنی برای تولید (پیش‌فرض: "") |
| `نسبت تصویر` | COMBO | بله | "16:9"<br>"1:1"<br>"21:9"<br>"2:3"<br>"3:2"<br>"4:5"<br>"5:4"<br>"9:16"<br>"9:21" | نسبت ابعاد تصویر تولیدشده |
| `تصویر مرجع` | IMAGE | خیر | - | تصویر مرجع اختیاری برای هدایت تولید |

**نکته:** تصویر مرجع باید ابعادی حداکثر ۷۹۹۹×۷۹۹۹ پیکسل و نسبت ابعادی بین ۰.۵ تا ۲.۰ داشته باشد. هنگامی که یک تصویر مرجع ارائه می‌شود، فرآیند تولید تصویر را هدایت می‌کند.

## خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-----------|----------|---------|
| `output` | IMAGE | تصویر تولیدشده بر اساس توضیح متنی و تصویر مرجع اختیاری |
