Вот перевод документации на русский язык:

Узел Runway Text to Image генерирует изображения на основе текстовых запросов с использованием модели Gen 4 от Runway. Вы можете указать текстовое описание и при необходимости добавить референсное изображение для управления процессом генерации. Узел обрабатывает взаимодействие с API и возвращает сгенерированное изображение.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `prompt` | STRING | Да | - | Текстовый запрос для генерации (по умолчанию: "") |
| `ratio` | COMBO | Да | "16:9"<br>"1:1"<br>"21:9"<br>"2:3"<br>"3:2"<br>"4:5"<br>"5:4"<br>"9:16"<br>"9:21" | Соотношение сторон для генерируемого изображения |
| `reference_image` | IMAGE | Нет | - | Необязательное референсное изображение для управления генерацией |

**Примечание:** Размеры референсного изображения не должны превышать 7999x7999 пикселей, а соотношение сторон должно находиться в диапазоне от 0,5 до 2,0. При наличии референсного изображения оно используется для управления процессом генерации.

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `output` | IMAGE | Сгенерированное изображение на основе текстового запроса и необязательного референсного изображения |
