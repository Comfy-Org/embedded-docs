# ترجمة وثيقة عقدة LTXV Reference Audio

عقدة **LTXV Reference Audio** تُستخدم لنقل هوية المتحدث في عملية توليد الصوت. تقوم بتشفير مقطع صوتي مرجعي في التكييف (conditioning) الخاص بالنموذج، مما يسمح للصوت المُولّد بتبني خصائص صوت المتحدث. يمكنها أيضًا تطبيق توجيه الهوية (identity guidance)، الذي يُجري خطوة معالجة إضافية لتضخيم تأثير هوية المتحدث.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|----------|---------------|--------|---------|-------|
| `النموذج` | MODEL | نعم | - | النموذج المراد تصحيحه باستخدام توجيه الهوية. |
| `إيجابي` | CONDITIONING | نعم | - | مدخل التكييف الإيجابي. |
| `سلبي` | CONDITIONING | نعم | - | مدخل التكييف السلبي. |
| `الصوت_المرجعي` | AUDIO | نعم | - | المقطع الصوتي المرجعي الذي سيتم نقل هوية المتحدث منه. يُوصى بمدة ~5 ثوانٍ (مدة التدريب). قد تؤدي المقاطع الأقصر أو الأطول إلى تدهور نقل هوية الصوت. |
| `audio_vae` | VAE | نعم | - | مشفر الصوت LTXV Audio VAE لتشفير الصوت المرجعي. |
| `مقياس توجيه الهوية` | FLOAT | لا | 0.0 - 100.0 | قوة توجيه الهوية. يُجري تمريرًا إضافيًا للأمام بدون مرجع في كل خطوة لتضخيم هوية المتحدث. اضبط على 0 لتعطيل (بدون تمرير إضافي). (القيمة الافتراضية: 3.0) |
| `نسبة البداية` | FLOAT | لا | 0.0 - 1.0 | بداية نطاق سيغما (sigma) حيث يكون توجيه الهوية نشطًا. (القيمة الافتراضية: 0.0) |
| `نسبة النهاية` | FLOAT | لا | 0.0 - 1.0 | نهاية نطاق سيغما (sigma) حيث يكون توجيه الهوية نشطًا. (القيمة الافتراضية: 1.0) |

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |

| `سلبي` | CONDITIONING | التكييف السلبي، الذي يحتوي الآن على بيانات الصوت المرجعي المُشفّرة. |
