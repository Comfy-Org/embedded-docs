# ترجمة وثيقة عقدة LTXV Reference Audio

عقدة **LTXV Reference Audio** تُستخدم لنقل هوية المتحدث في عملية توليد الصوت. تقوم بتشفير مقطع صوتي مرجعي في التكييف (conditioning) الخاص بالنموذج، مما يسمح للصوت المُولّد بتبني خصائص صوت المتحدث. يمكنها أيضًا تطبيق توجيه الهوية (identity guidance)، الذي يُجري خطوة معالجة إضافية لتضخيم تأثير هوية المتحدث.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|----------|---------------|--------|---------|-------|
| `model` | MODEL | نعم | - | النموذج المراد تصحيحه باستخدام توجيه الهوية. |
| `positive` | CONDITIONING | نعم | - | مدخل التكييف الإيجابي. |
| `negative` | CONDITIONING | نعم | - | مدخل التكييف السلبي. |
| `reference_audio` | AUDIO | نعم | - | المقطع الصوتي المرجعي الذي سيتم نقل هوية المتحدث منه. يُوصى بمدة ~5 ثوانٍ (مدة التدريب). قد تؤدي المقاطع الأقصر أو الأطول إلى تدهور نقل هوية الصوت. |
| `audio_vae` | VAE | نعم | - | مشفر الصوت LTXV Audio VAE لتشفير الصوت المرجعي. |
| `identity_guidance_scale` | FLOAT | لا | 0.0 - 100.0 | قوة توجيه الهوية. يُجري تمريرًا إضافيًا للأمام بدون مرجع في كل خطوة لتضخيم هوية المتحدث. اضبط على 0 لتعطيل (بدون تمرير إضافي). (القيمة الافتراضية: 3.0) |
| `start_percent` | FLOAT | لا | 0.0 - 1.0 | بداية نطاق سيغما (sigma) حيث يكون توجيه الهوية نشطًا. (القيمة الافتراضية: 0.0) |
| `end_percent` | FLOAT | لا | 0.0 - 1.0 | نهاية نطاق سيغما (sigma) حيث يكون توجيه الهوية نشطًا. (القيمة الافتراضية: 1.0) |

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|-------------|---------------|--------|
| `model` | MODEL | النموذج المُصحّح بوظيفة توجيه الهوية. |
| `positive` | CONDITIONING | التكييف الإيجابي، الذي يحتوي الآن على بيانات الصوت المرجعي المُشفّرة. |
| `negative` | CONDITIONING | التكييف السلبي، الذي يحتوي الآن على بيانات الصوت المرجعي المُشفّرة. |
