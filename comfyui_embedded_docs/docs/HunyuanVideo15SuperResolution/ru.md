Вот перевод документации на русский язык:

Узел HunyuanVideo15SuperResolution подготавливает данные conditioning для процесса суперразрешения видео. Он принимает латентное представление видео и, опционально, начальное изображение, и упаковывает их вместе с шумовой аугментацией и данными CLIP vision в формат, который может быть использован моделью для генерации выходного изображения с более высоким разрешением.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `positive` | CONDITIONING | Да | Н/Д | Входной positive conditioning, который будет модифицирован латентным представлением и данными аугментации. |
| `negative` | CONDITIONING | Да | Н/Д | Входной negative conditioning, который будет модифицирован латентным представлением и данными аугментации. |
| `vae` | VAE | Нет | Н/Д | VAE, используемый для кодирования опционального `start_image`. Обязателен, если указан `start_image`. |
| `start_image` | IMAGE | Нет | Н/Д | Опциональное начальное изображение для направления процесса суперразрешения. Если указано, оно будет увеличено и закодировано в conditioning latent. |
| `clip_vision_output` | CLIP_VISION_OUTPUT | Нет | Н/Д | Опциональные эмбеддинги CLIP vision для добавления в conditioning. |
| `latent` | LATENT | Да | Н/Д | Входное латентное представление видео, которое будет включено в conditioning. |
| `noise_augmentation` | FLOAT | Нет | 0.0 - 1.0 | Сила шумовой аугментации, применяемой к conditioning (по умолчанию: 0.70). |

**Примечание:** Если вы указываете `start_image`, необходимо также подключить `vae` для его кодирования. `start_image` будет автоматически увеличен до размеров, соответствующих входному `latent`.

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `positive` | CONDITIONING | Модифицированный positive conditioning, теперь содержащий объединённое латентное представление, шумовую аугментацию и опциональные данные CLIP vision. |
| `negative` | CONDITIONING | Модифицированный negative conditioning, теперь содержащий объединённое латентное представление, шумовую аугментацию и опциональные данные CLIP vision. |
| `latent` | LATENT | Входное латентное представление передаётся без изменений. |
