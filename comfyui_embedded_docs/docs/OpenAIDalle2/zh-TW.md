# OpenAIDalle2

透過 OpenAI 的 DALL·E 2 端點同步生成圖像。

## 運作方式

此節點連接到 OpenAI 的 DALL·E 2 API，根據文字描述創建圖像。當您提供文字提示時，節點會將其發送到 OpenAI 的伺服器，伺服器會生成相應的圖像並將其返回給 ComfyUI。此節點可在兩種模式下運作：僅使用文字提示的標準圖像生成模式，或同時提供圖像和遮罩時的圖像編輯模式。在編輯模式下，它使用遮罩來確定原始圖像中應修改的部分，同時保持其他區域不變。
