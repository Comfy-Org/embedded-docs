# OpenAIDalle2

Генерирует изображения синхронно через конечную точку DALL·E 2 от OpenAI.

## Принцип работы

Этот узел подключается к API DALL·E 2 от OpenAI для создания изображений на основе текстовых описаний. Когда вы предоставляете текстовый запрос, узел отправляет его на серверы OpenAI, которые генерируют соответствующие изображения и возвращают их в ComfyUI. Узел может работать в двух режимах: стандартная генерация изображения с использованием только текстового запроса или режим редактирования изображения, когда предоставлены как изображение, так и маска. В режиме редактирования маска используется для определения частей исходного изображения, которые следует изменить, сохраняя остальные области без изменений.

## Входные параметры

| Параметр | Тип данных | Тип ввода | По умолчанию | Диапазон | Описание |
|----------|------------|-----------|--------------|----------|----------|
| `prompt` | STRING | обязательный | "" | - | Текстовый запрос для DALL·E |
| `seed` | INT | опциональный | 0 | от 0 до 2147483647 | пока не реализовано в бэкенде |
| `size` | COMBO | опциональный | "1024x1024" | "256x256", "512x512", "1024x1024" | Размер изображения |
| `n` | INT | опциональный | 1 | от 1 до 8 | Количество генерируемых изображений |
| `image` | IMAGE | опциональный | None | - | Опциональное исходное изображение для редактирования |
| `mask` | MASK | опциональный | None | - | Опциональная маска для инпейнтинга (белые области будут заменены) |

## Выходные данные

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `IMAGE` | IMAGE | Сгенерированное или отредактированное изображение(я) от DALL·E 2 |
