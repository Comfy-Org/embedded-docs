Вот перевод документации на русский язык:

Узел SetFirstSigma изменяет последовательность значений сигма, заменяя первое значение сигма в последовательности на пользовательское значение. Он принимает существующую последовательность сигма и новое значение сигма в качестве входных данных, затем возвращает новую последовательность сигма, в которой изменён только первый элемент, при этом все остальные значения сигма остаются неизменными.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `сигмы` | SIGMAS | Да | - | Входная последовательность значений сигма для изменения |
| `сигма` | FLOAT | Да | от 0.0 до 20000.0 | Новое значение сигма, которое будет установлено в качестве первого элемента последовательности (по умолчанию: 136.0) |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `сигмы` | SIGMAS | Изменённая последовательность сигма, в которой первый элемент заменён пользовательским значением сигма |
