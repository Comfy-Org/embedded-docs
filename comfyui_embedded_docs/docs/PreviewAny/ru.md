Узел PreviewAny отображает предварительный просмотр любого типа входных данных в текстовом формате. Он принимает любой тип данных на вход и преобразует их в читаемую строку для просмотра. Узел автоматически обрабатывает различные типы данных, включая строки, числа, логические значения и сложные объекты, пытаясь сериализовать их в формат JSON.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `источник` | ANY | Да | Любой тип данных | Принимает любой тип входных данных для отображения предварительного просмотра |

## Выходы

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `UI Text Display` | STRING | Отображает входные данные, преобразованные в текстовый формат, в пользовательском интерфейсе. Также возвращает текст в виде строкового вывода для дальнейшей обработки. |
