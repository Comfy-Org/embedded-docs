LTXVImgToVideo 節點將輸入圖像轉換為影片生成模型所需的影片潛在表示。它接收單一圖像，使用 VAE 編碼器將其擴展為影格序列，然後應用強度控制的條件處理，以決定在影片生成過程中保留原始圖像內容與進行修改的比例。

## 輸入參數

| 參數名稱 | 資料類型 | 必填 | 數值範圍 | 參數說明 |
|-----------|-----------|----------|-------|-------------|
| `正向` | CONDITIONING | 是 | - | 用於引導影片生成的正向條件提示 |
| `負向` | CONDITIONING | 是 | - | 用於避免影片中出現特定元素的負向條件提示 |
| `vae` | VAE | 是 | - | 用於將輸入圖像編碼至潛在空間的 VAE 模型 |
| `影像` | IMAGE | 是 | - | 要轉換為影片影格的輸入圖像 |
| `寬度` | INT | 否 | 64 至 MAX_RESOLUTION | 輸出影片的寬度（像素）（預設值：768，間隔：32） |
| `高度` | INT | 否 | 64 至 MAX_RESOLUTION | 輸出影片的高度（像素）（預設值：512，間隔：32） |
| `長度` | INT | 否 | 9 至 MAX_RESOLUTION | 生成影片的影格數量（預設值：97，間隔：8） |
| `批次大小` | INT | 否 | 1 至 4096 | 同時生成的影片數量（預設值：1） |
| `強度` | FLOAT | 否 | 0.0 至 1.0 | 控制影片生成過程中對原始圖像的修改程度，1.0 表示保留最多原始內容，0.0 則允許最大程度的修改（預設值：1.0） |
