گره LTXVImgToVideo یک تصویر ورودی را به نمایش نهفته ویدیو برای مدل‌های تولید ویدیو تبدیل می‌کند. این گره یک تصویر واحد را دریافت کرده و با استفاده از رمزگذار VAE آن را به دنباله‌ای از فریم‌ها گسترش می‌دهد، سپس با اعمال شرط‌گذاری همراه با کنترل شدت، تعیین می‌کند که چه مقدار از محتوای تصویر اصلی در طول تولید ویدیو حفظ شود و چه مقدار تغییر کند.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `positive` | CONDITIONING | بله | - | راهنماهای شرط‌گذاری مثبت برای هدایت تولید ویدیو |
| `negative` | CONDITIONING | بله | - | راهنماهای شرط‌گذاری منفی برای جلوگیری از عناصر خاص در ویدیو |
| `vae` | VAE | بله | - | مدل VAE مورد استفاده برای رمزگذاری تصویر ورودی به فضای نهفته |
| `image` | IMAGE | بله | - | تصویر ورودی که به فریم‌های ویدیو تبدیل می‌شود |
| `width` | INT | خیر | ۶۴ تا MAX_RESOLUTION | عرض ویدیوی خروجی بر حسب پیکسل (پیش‌فرض: ۷۶۸، گام: ۳۲) |
| `height` | INT | خیر | ۶۴ تا MAX_RESOLUTION | ارتفاع ویدیوی خروجی بر حسب پیکسل (پیش‌فرض: ۵۱۲، گام: ۳۲) |
| `length` | INT | خیر | ۹ تا MAX_RESOLUTION | تعداد فریم‌های ویدیوی تولید شده (پیش‌فرض: ۹۷، گام: ۸) |
| `batch_size` | INT | خیر | ۱ تا ۴۰۹۶ | تعداد ویدیوهایی که به طور همزمان تولید می‌شوند (پیش‌فرض: ۱) |
| `strength` | FLOAT | خیر | ۰.۰ تا ۱.۰ | کنترل میزان حفظ محتوای تصویر اصلی در اولین فریم ویدیوی تولید شده. مقدار ۱.۰ تصویر اصلی را کاملاً حفظ می‌کند، در حالی که ۰.۰ حداکثر تغییر را امکان‌پذیر می‌سازد (پیش‌فرض: ۱.۰) |

## خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-------------|-----------|-------------|
| `positive` | CONDITIONING | شرط‌گذاری مثبت پردازش‌شده با ماسک فریم ویدیو اعمال‌شده |
| `negative` | CONDITIONING | شرط‌گذاری منفی پردازش‌شده با ماسک فریم ویدیو اعمال‌شده |
| `latent` | LATENT | نمایش نهفته ویدیو شامل فریم‌های رمزگذاری‌شده و ماسک نویز برای تولید ویدیو |
