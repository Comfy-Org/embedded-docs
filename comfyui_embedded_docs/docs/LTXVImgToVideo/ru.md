Вот перевод документации узла **LTXVImgToVideo** на русский язык:

Узел LTXVImgToVideo преобразует входное изображение в скрытое представление видео для моделей генерации видео. Он принимает одно изображение и расширяет его в последовательность кадров с помощью кодировщика VAE, затем применяет conditioning с контролем силы влияния, чтобы определить, какая часть исходного изображения будет сохранена, а какая изменена в процессе генерации видео.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `положительный` | CONDITIONING | Да | - | Позитивные промпты conditioning для направления генерации видео |
| `отрицательный` | CONDITIONING | Да | - | Негативные промпты conditioning для избегания определённых элементов в видео |
| `vae` | VAE | Да | - | Модель VAE, используемая для кодирования входного изображения в скрытое пространство |
| `изображение` | IMAGE | Да | - | Входное изображение, преобразуемое в кадры видео |
| `ширина` | INT | Нет | от 64 до MAX_RESOLUTION | Ширина выходного видео в пикселях (по умолчанию: 768, шаг: 32) |
| `высота` | INT | Нет | от 64 до MAX_RESOLUTION | Высота выходного видео в пикселях (по умолчанию: 512, шаг: 32) |
| `длина` | INT | Нет | от 9 до MAX_RESOLUTION | Количество кадров в сгенерированном видео (по умолчанию: 97, шаг: 8) |
| `размер_пакета` | INT | Нет | от 1 до 4096 | Количество видео, генерируемых одновременно (по умолчанию: 1) |
| `интенсивность` | FLOAT | Нет | от 0.0 до 1.0 | Степень сохранения исходного изображения в первом кадре сгенерированного видео. Значение 1.0 полностью сохраняет исходное изображение, тогда как 0.0 допускает максимальные изменения (по умолчанию: 1.0) |

## Выходные данные

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `положительный` | CONDITIONING | Обработанное позитивное conditioning с применённой маской кадров видео |
| `отрицательный` | CONDITIONING | Обработанное негативное conditioning с применённой маской кадров видео |
| `latent` | LATENT | Скрытое представление видео, содержащее закодированные кадры и маску шума для генерации видео |
