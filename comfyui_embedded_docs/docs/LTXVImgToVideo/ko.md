LTXVImgToVideo 노드는 입력 이미지를 비디오 생성 모델을 위한 비디오 잠재 표현으로 변환합니다. 단일 이미지를 가져와 VAE 인코더를 사용하여 프레임 시퀀스로 확장한 후, 강도 제어를 통해 조건화를 적용하여 비디오 생성 중 원본 이미지 콘텐츠가 보존되거나 수정되는 정도를 결정합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `positive` | CONDITIONING | 예 | - | 비디오 생성을 안내하는 긍정 조건화 프롬프트 |
| `negative` | CONDITIONING | 예 | - | 비디오에서 특정 요소를 회피하기 위한 부정 조건화 프롬프트 |
| `vae` | VAE | 예 | - | 입력 이미지를 잠재 공간으로 인코딩하는 데 사용되는 VAE 모델 |
| `image` | IMAGE | 예 | - | 비디오 프레임으로 변환할 입력 이미지 |
| `width` | INT | 아니요 | 64 ~ MAX_RESOLUTION | 출력 비디오의 가로 픽셀 크기 (기본값: 768, 단계: 32) |
| `height` | INT | 아니요 | 64 ~ MAX_RESOLUTION | 출력 비디오의 세로 픽셀 크기 (기본값: 512, 단계: 32) |
| `length` | INT | 아니요 | 9 ~ MAX_RESOLUTION | 생성된 비디오의 프레임 수 (기본값: 97, 단계: 8) |
| `batch_size` | INT | 아니요 | 1 ~ 4096 | 동시에 생성할 비디오 개수 (기본값: 1) |
| `strength` | FLOAT | 아니요 | 0.0 ~ 1.0 | 생성된 비디오의 첫 번째 프레임에서 원본 이미지 콘텐츠가 얼마나 보존될지 제어합니다. 값이 1.0이면 원본 이미지를 완전히 보존하고, 0.0이면 최대한 수정을 허용합니다 (기본값: 1.0) |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `positive` | CONDITIONING | 비디오 프레임 마스킹이 적용된 처리된 긍정 조건화 |
| `negative` | CONDITIONING | 비디오 프레임 마스킹이 적용된 처리된 부정 조건화 |
| `latent` | LATENT | 비디오 생성을 위한 인코딩된 프레임과 노이즈 마스크를 포함하는 비디오 잠재 표현 |
