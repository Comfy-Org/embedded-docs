Вот перевод документации узла ComfyUI на русский язык:

Узел Morphology применяет к изображениям различные морфологические операции — математические методы, используемые для обработки и анализа форм на изображениях. Он может выполнять такие операции, как эрозия, дилатация, открытие, закрытие и другие, с использованием настраиваемого размера ядра для управления силой эффекта.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `изображение` | IMAGE | Да | - | Входное изображение для обработки |
| `операция` | STRING | Да | `"erode"`<br>`"dilate"`<br>`"open"`<br>`"close"`<br>`"gradient"`<br>`"bottom_hat"`<br>`"top_hat"` | Применяемая морфологическая операция (по умолчанию: "erode") |
| `размер_ядра` | INT | Да | 3-999 | Размер структурирующего элемента (ядра) (по умолчанию: 3). Должен быть нечётным числом. |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `изображение` | IMAGE | Обработанное изображение после применения морфологической операции |
