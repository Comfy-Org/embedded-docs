هذا هو النص المترجم إلى العربية مع الالتزام بالقواعد المحددة:

تقوم عقدة Ideogram V2 بإنشاء الصور باستخدام نموذج الذكاء الاصطناعي Ideogram V2. تستقبل العقدة نصوصًا توجيهية (برومبت) وإعدادات توليد متنوعة لإنشاء الصور عبر خدمة API. تدعم العقدة نسب أبعاد ودقة وخيارات أنماط مختلفة لتخصيص الصور الناتجة.

## المدخلات

| المعامل | نوع البيانات | إجباري | النطاق | الوصف |
|-----------|-----------|----------|-------|-------------|
| `prompt` | STRING | نعم | - | النص التوجيهي لتوليد الصورة (القيمة الافتراضية: سلسلة فارغة) |
| `turbo` | BOOLEAN | لا | - | تفعيل الوضع السريع (توليد أسرع، جودة أقل محتملة) (القيمة الافتراضية: False) |
| `aspect_ratio` | COMBO | لا | "1:1"<br>"16:9"<br>"9:16"<br>"4:3"<br>"3:4"<br>"3:2"<br>"2:3" | نسبة الأبعاد لتوليد الصورة. يتم تجاهلها إذا لم تكن الدقة مضبوطة على AUTO. (القيمة الافتراضية: "1:1") |
| `resolution` | COMBO | لا | "Auto"<br>"1024x1024"<br>"1152x896"<br>"896x1152"<br>"1216x832"<br>"832x1216"<br>"1344x768"<br>"768x1344"<br>"1536x640"<br>"640x1536" | دقة الصورة للتوليد. إذا لم تكن مضبوطة على AUTO، فإنها تتجاوز إعداد نسبة الأبعاد. (القيمة الافتراضية: "Auto") |
| `magic_prompt_option` | COMBO | لا | "AUTO"<br>"ON"<br>"OFF" | تحديد ما إذا كان سيتم استخدام MagicPrompt في التوليد (القيمة الافتراضية: "AUTO") |
| `seed` | INT | لا | 0-2147483647 | البذرة العشوائية للتوليد (القيمة الافتراضية: 0) |
| `style_type` | COMBO | لا | "AUTO"<br>"GENERAL"<br>"REALISTIC"<br>"DESIGN"<br>"RENDER_3D"<br>"ANIME" | نوع النمط للتوليد (لـ V2 فقط) (القيمة الافتراضية: "NONE") |
| `negative_prompt` | STRING | لا | - | وصف ما يجب استبعاده من الصورة (القيمة الافتراضية: سلسلة فارغة) |
| `num_images` | INT | لا | 1-8 | عدد الصور المراد توليدها (القيمة الافتراضية: 1) |

**ملاحظة:** عندما لا تكون `resolution` مضبوطة على "Auto"، فإنها تتجاوز إعداد `aspect_ratio`. المعامل `num_images` له حد أقصى يبلغ 8 صور لكل عملية توليد.
