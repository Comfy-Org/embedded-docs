گره Ideogram V2 با استفاده از مدل هوش مصنوعی Ideogram V2 تصاویر تولید می‌کند. این گره با دریافت توضیحات متنی و تنظیمات مختلف تولید، از طریق یک سرویس API تصاویر را ایجاد می‌کند. این گره از نسبت‌های ابعادی، وضوح‌ها و گزینه‌های سبک مختلف برای سفارشی‌سازی تصاویر خروجی پشتیبانی می‌کند.

## ورودی‌ها

| پارامتر | نوع داده | اجباری | محدوده | توضیحات |
|---------|----------|--------|--------|---------|
| `prompt` | STRING | بله | - | توضیح متن برای تولید تصویر (پیش‌فرض: رشته خالی) |
| `turbo` | BOOLEAN | خیر | - | آیا از حالت توربو استفاده شود (تولید سریع‌تر، کیفیت احتمالی پایین‌تر) (پیش‌فرض: False) |
| `aspect_ratio` | COMBO | خیر | "1:1"<br>"16:9"<br>"9:16"<br>"4:3"<br>"3:4"<br>"3:2"<br>"2:3" | نسبت ابعاد برای تولید تصویر. اگر وضوح روی AUTO تنظیم نشده باشد، نادیده گرفته می‌شود. (پیش‌فرض: "1:1") |
| `resolution` | COMBO | خیر | "Auto"<br>"1024x1024"<br>"1152x896"<br>"896x1152"<br>"1216x832"<br>"832x1216"<br>"1344x768"<br>"768x1344"<br>"1536x640"<br>"640x1536" | وضوح برای تولید تصویر. اگر روی AUTO تنظیم نشده باشد، تنظیم aspect_ratio را بازنویسی می‌کند. (پیش‌فرض: "Auto") |
| `magic_prompt_option` | COMBO | خیر | "AUTO"<br>"ON"<br>"OFF" | تعیین می‌کند که آیا از MagicPrompt در تولید استفاده شود (پیش‌فرض: "AUTO") |
| `seed` | INT | خیر | 0-2147483647 | دانه تصادفی برای تولید (پیش‌فرض: 0) |
| `style_type` | COMBO | خیر | "AUTO"<br>"GENERAL"<br>"REALISTIC"<br>"DESIGN"<br>"RENDER_3D"<br>"ANIME" | نوع سبک برای تولید (فقط V2) (پیش‌فرض: "NONE") |
| `negative_prompt` | STRING | خیر | - | توضیح آنچه باید از تصویر حذف شود (پیش‌فرض: رشته خالی) |
| `num_images` | INT | خیر | 1-8 | تعداد تصاویر برای تولید (پیش‌فرض: 1) |

**توجه:** وقتی `resolution` روی "Auto" تنظیم نشده باشد، تنظیم `aspect_ratio` را بازنویسی می‌کند. پارامتر `num_images` حداکثر محدودیت ۸ تصویر در هر بار تولید دارد.
