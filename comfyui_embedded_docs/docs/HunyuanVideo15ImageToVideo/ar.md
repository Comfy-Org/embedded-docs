# ترجمة وثيقة عقدة HunyuanVideo15ImageToVideo

تقوم عقدة **HunyuanVideo15ImageToVideo** بإعداد بيانات التكييف (conditioning) والفضاء الكامن (latent space) لتوليد الفيديو استنادًا إلى نموذج HunyuanVideo 1.5. تنشئ هذه العقدة تمثيلًا كامنًا أوليًا لتسلسل فيديو، ويمكنها دمج صورة بداية اختيارية أو مخرجات رؤية CLIP لتوجيه عملية التوليد.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|---------|---------------|--------|---------|-------|
| `positive` | CONDITIONING | نعم | - | مطالبات التكييف الإيجابية التي تصف ما يجب أن يحتويه الفيديو. |
| `negative` | CONDITIONING | نعم | - | مطالبات التكييف السلبية التي تصف ما يجب أن يتجنبه الفيديو. |
| `vae` | VAE | نعم | - | نموذج التشفير التلقائي المتغير (VAE) المستخدم لتشفير صورة البداية إلى الفضاء الكامن. |
| `width` | INT | لا | 16 إلى MAX_RESOLUTION | عرض إطارات الفيديو الناتجة بالبكسل. يجب أن يكون قابلاً للقسمة على 16. (الافتراضي: 848) |
| `height` | INT | لا | 16 إلى MAX_RESOLUTION | ارتفاع إطارات الفيديو الناتجة بالبكسل. يجب أن يكون قابلاً للقسمة على 16. (الافتراضي: 480) |
| `length` | INT | لا | 1 إلى MAX_RESOLUTION | العدد الإجمالي للإطارات في تسلسل الفيديو. يجب أن يكون مضاعفًا للرقم 4. (الافتراضي: 33) |
| `batch_size` | INT | لا | 1 إلى 4096 | عدد تسلسلات الفيديو المراد توليدها في دفعة واحدة. (الافتراضي: 1) |
| `start_image` | IMAGE | لا | - | صورة بداية اختيارية لتهيئة توليد الفيديو. إذا تم توفيرها، يتم تشفيرها واستخدامها لتكييف الإطارات الأولى. يتم استخدام أول `length` إطار من الصورة فقط. |
| `clip_vision_output` | CLIP_VISION_OUTPUT | لا | - | تضمينات رؤية CLIP اختيارية لتوفير تكييف بصري إضافي لعملية التوليد. |

**ملاحظة:** عند توفير `start_image`، يتم تغيير حجمها تلقائيًا لتتناسب مع `width` و `height` المحددين باستخدام الاستيفاء ثنائي الخطوط. يتم استخدام أول `length` إطار من دفعة الصور. ثم يتم إضافة الصورة المشفرة إلى كل من التكييف `positive` و `negative` كـ `concat_latent_image` مع `concat_mask` المقابل. يتم تعيين القناع إلى 0.0 للإطارات التي تغطيها صورة البداية وإلى 1.0 للإطارات المتبقية.

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|-------------|---------------|---------|
| `positive` | CONDITIONING | التكييف الإيجابي المعدل، والذي قد يتضمن الآن صورة البداية المشفرة أو مخرجات رؤية CLIP. |
| `negative` | CONDITIONING | التكييف السلبي المعدل، والذي قد يتضمن الآن صورة البداية المشفرة أو مخرجات رؤية CLIP. |
| `latent` | LATENT | موتر كامن فارغ بأبعاد مهيأة لحجم الدفعة المحدد وطول الفيديو والعرض والارتفاع. |
