این گره بر اساس مواد مرجع ارائه‌شده، ویدئویی از یک شخص یا شیء تولید می‌کند. این گره از مدل Wan 2.7 برای ساخت ویدئو از روی یک توضیح متنی استفاده می‌کند و از اجرای تک‌شخصیتی و تعاملات چندشخصیتی پشتیبانی می‌کند. برای عملکرد صحیح، باید حداقل یک ویدئو یا تصویر مرجع ارائه دهید.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `مدل` | COMBO | بله | `"wan2.7-r2v"` | مدل مشخص برای تولید ویدئو. |
| `model.prompt` | STRING | بله | - | توضیحاتی که ویدئو را توصیف می‌کند. برای ارجاع به شخصیت‌های مرجع از شناسه‌هایی مانند 'character1' و 'character2' استفاده کنید. |
| `model.negative_prompt` | STRING | خیر | - | توضیح منفی که مشخص می‌کند در ویدئوی تولیدشده از چه چیزهایی اجتناب شود (پیش‌فرض: خالی). |
| `model.resolution` | COMBO | بله | `"720P"`<br>`"1080P"` | وضوح تصویر ویدئوی خروجی. |
| `model.ratio` | COMBO | بله | `"16:9"`<br>`"9:16"`<br>`"1:1"`<br>`"4:3"`<br>`"3:4"` | نسبت ابعاد ویدئوی خروجی. |
| `model.duration` | INT | بله | 2 تا 10 | طول ویدئوی تولیدشده بر حسب ثانیه (پیش‌فرض: 5). |
| `model.reference_videos` | VIDEO | خیر | - | فهرستی از ویدئوهای مرجع. می‌توانید حداکثر 3 ویدئو اضافه کنید. |
| `model.reference_images` | IMAGE | خیر | - | فهرستی از تصاویر مرجع. می‌توانید حداکثر 5 تصویر اضافه کنید. |
| `بذر` | INT | خیر | 0 تا 2147483647 | دانه (seed) مورد استفاده برای تولید که به کنترل تصادفی بودن خروجی کمک می‌کند (پیش‌فرض: 0). |
| `واترمارک` | BOOLEAN | خیر | - | آیا به نتیجه، واترمارک تولیدشده توسط هوش مصنوعی اضافه شود یا خیر (پیش‌فرض: False). این یک تنظیم پیشرفته است. |

**محدودیت‌های مهم:**
*   باید حداقل یک ویدئو یا تصویر مرجع در ورودی‌های `model.reference_videos` یا `model.reference_images` ارائه دهید.
*   مجموع تعداد ویدئوها و تصاویر مرجع نباید از 5 تجاوز کند.
