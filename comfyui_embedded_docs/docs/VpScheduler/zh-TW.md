VPScheduler 節點旨在根據變異數保持（VP）排程方法生成一系列噪聲級別（sigmas）。此序列對於引導擴散模型中的去噪過程至關重要，可實現對圖像或其他數據類型的受控生成。

## 輸入

| 參數        | 資料類型 | 描述                                                                                          |
|-------------|-------------|--------------------------------------------------------------------------------------------------------------------------------------------------|
| `steps`     | INT         | 指定擴散過程中的步驟數，影響所生成噪聲級別的細粒度。                              |
| `beta_d`    | FLOAT       | 決定整體噪聲級別分佈，影響所生成噪聲級別的變異數。                                 |
| `beta_min`  | FLOAT       | 設定噪聲級別的最小邊界，確保噪聲不會低於特定閾值。                              |
| `eps_s`     | FLOAT       | 調整起始 epsilon 值，微調擴散過程中的初始噪聲級別。                                    |

## 輸出

| 參數      | 資料類型 | 描述                                                                                   |
|-------------|-------------|-----------------------------------------------------------------------------------------------|
| `sigmas`    | SIGMAS      | 基於 VP 排程方法生成的一系列噪聲級別（sigmas），用於引導擴散模型中的去噪過程。 |
