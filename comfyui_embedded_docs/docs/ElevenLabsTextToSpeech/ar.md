# ترجمة وثيقة عقد ElevenLabs Text to Speech

يقوم عقد ElevenLabs لتحويل النص إلى كلام بتحويل النص المكتوب إلى صوت منطوق باستخدام واجهة برمجة تطبيقات ElevenLabs. يتيح لك اختيار صوت معين وضبط خصائص الكلام المختلفة مثل الثبات والسرعة والأسلوب لتوليد مخرجات صوتية مخصصة.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|---------|---------------|--------|---------|-------|
| `voice` | CUSTOM | نعم | غير متاح | الصوت المستخدم في تركيب الكلام. قم بالتوصيل من محدد الصوت أو استنساخ الصوت الفوري. |
| `text` | STRING | نعم | غير متاح | النص المراد تحويله إلى كلام. |
| `stability` | FLOAT | لا | 0.0 - 1.0 | ثبات الصوت. القيم المنخفضة تعطي نطاقًا عاطفيًا أوسع، والقيم الأعلى تنتج كلامًا أكثر اتساقًا ولكن قد يكون رتيبًا (الافتراضي: 0.5). |
| `apply_text_normalization` | COMBO | لا | `"auto"`<br>`"on"`<br>`"off"` | وضع تطبيع النص. 'auto' يترك النظام يقرر، 'on' يطبق التطبيع دائمًا، 'off' يتجاوزه. |
| `model` | DYNAMICCOMBO | لا | `"eleven_multilingual_v2"`<br>`"eleven_v3"` | النموذج المستخدم في تحويل النص إلى كلام. اختيار نموذج يكشف عن معاملاته الخاصة. |
| `language_code` | STRING | لا | غير متاح | رمز اللغة وفقًا لمعيار ISO-639-1 أو ISO-639-3 (مثل 'en' و'es' و'fra'). اتركه فارغًا للكشف التلقائي (الافتراضي: ""). |
| `seed` | INT | لا | 0 - 2147483647 | البذرة لإمكانية إعادة الإنتاج (غير مضمونة الحتمية) (الافتراضي: 1). |
| `output_format` | COMBO | لا | `"mp3_44100_192"`<br>`"opus_48000_192"` | تنسيق إخراج الصوت. |

**معاملات خاصة بالنموذج:**
عند تعيين المعامل `model` إلى `"eleven_multilingual_v2"`، تصبح المعاملات الإضافية التالية متاحة:

* `speed`: سرعة الكلام. 1.0 طبيعي، <1.0 أبطأ، >1.0 أسرع (الافتراضي: 1.0، النطاق: 0.7 - 1.3).
* `similarity_boost`: تعزيز التشابه. القيم الأعلى تجعل الصوت أكثر تشابهًا مع الصوت الأصلي (الافتراضي: 0.75، النطاق: 0.0 - 1.0).
* `use_speaker_boost`: تعزيز التشابه مع صوت المتحدث الأصلي (الافتراضي: False).
* `style`: المبالغة في الأسلوب. القيم الأعلى تزيد من التعبير الأسلوبي ولكنها قد تقلل الثبات (الافتراضي: 0.0، النطاق: 0.0 - 0.2).

عند تعيين المعامل `model` إلى `"eleven_v3"`، تصبح المعاملات الإضافية التالية متاحة:

* `speed`: سرعة الكلام. 1.0 طبيعي، <1.0 أبطأ، >1.0 أسرع (الافتراضي: 1.0، النطاق: 0.7 - 1.3).
* `similarity_boost`: تعزيز التشابه. القيم الأعلى تجعل الصوت أكثر تشابهًا مع الصوت الأصلي (الافتراضي: 0.75، النطاق: 0.0 - 1.0).
