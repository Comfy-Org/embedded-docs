این گره یک LoRA (سازگاری رتبه-پایین) را روی مدل اعمال می‌کند تا رفتار آن را تغییر دهد، اما تنها بر خود مؤلفه مدل تأثیر می‌گذارد. یک فایل LoRA مشخص را بارگذاری کرده و وزن‌های مدل را با شدت معینی تنظیم می‌کند و سایر مؤلفه‌ها مانند رمزگذار متن CLIP را بدون تغییر باقی می‌گذارد.

## ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|---------|----------|-------|--------|---------|
| `مدل` | MODEL | بله | - | مدل پایه‌ای که تنظیمات LoRA روی آن اعمال خواهد شد. |
| `lora_name` | STRING | بله | (فهرست فایل‌های LoRA موجود) | نام فایل LoRA برای بارگذاری و اعمال. گزینه‌ها از فایل‌های موجود در پوشه `loras` پر می‌شوند. |
| `قدرت_مدل` | FLOAT | بله | 100.0- تا 100.0 | شدت تأثیر LoRA بر وزن‌های مدل. مقدار مثبت LoRA را اعمال می‌کند، مقدار منفی معکوس آن را اعمال می‌کند و مقدار صفر هیچ تأثیری ندارد (پیش‌فرض: 1.0). |
