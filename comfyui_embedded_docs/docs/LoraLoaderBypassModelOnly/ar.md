# ترجمة وثيقة عقد ComfyUI

يطبق هذا العقد تقنية LoRA (التكيف منخفض الرتبة) على نموذج لتعديل سلوكه، ولكنه يؤثر فقط على مكون النموذج نفسه. يقوم بتحميل ملف LoRA المحدد ويضبط أوزان النموذج بقوة معينة، مع ترك المكونات الأخرى مثل مشفر النص CLIP دون تغيير.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|---------|--------------|--------|--------|-------|
| `النموذج` | MODEL | نعم | - | النموذج الأساسي الذي سيتم تطبيق تعديلات LoRA عليه. |
| `اسم LoRA` | STRING | نعم | (قائمة بملفات LoRA المتاحة) | اسم ملف LoRA المراد تحميله وتطبيقه. يتم تعبئة الخيارات من الملفات الموجودة في دليل `loras`. |
| `قوة التعديل على النموذج` | FLOAT | نعم | -100.0 إلى 100.0 | قوة تأثير LoRA على أوزان النموذج. القيمة الموجبة تطبق LoRA، والقيمة السالبة تطبق المعكوس، والقيمة 0 ليس لها تأثير (القيمة الافتراضية: 1.0). |

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|------------|--------------|--------|
| `النموذج` | MODEL | النموذج المعدل مع تطبيق تعديلات LoRA على أوزانه. |
