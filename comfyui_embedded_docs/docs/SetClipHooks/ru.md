Узел SetClipHooks позволяет применять пользовательские хуки к модели CLIP, обеспечивая возможность расширенной модификации её поведения. Он может применять хуки к выходам кондиционирования и, опционально, активировать функциональность планирования CLIP. Этот узел создает клонированную копию входной модели CLIP с применением указанных конфигураций хуков.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `clip` | CLIP | Да | - | Модель CLIP, к которой применяются хуки |
| `apply_to_conds` | BOOLEAN | Да | - | Применять ли хуки к выходам кондиционирования (по умолчанию: True) |
| `schedule_clip` | BOOLEAN | Да | - | Активировать ли планирование CLIP (по умолчанию: False) |
| `hooks` | HOOKS | Нет | - | Опциональная группа хуков для применения к модели CLIP |

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `clip` | CLIP | Клонированная модель CLIP с примененными указанными хуками |
