Вот перевод документации на русский язык:

Узел SetClipHooks позволяет применять пользовательские хуки к модели CLIP, обеспечивая расширенные модификации её поведения. Он может применять хуки к выходам кондиционирования и опционально включать функциональность планирования CLIP. Этот узел создает клонированную копию входной модели CLIP с примененными заданными конфигурациями хуков.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `clip` | CLIP | Да | - | Модель CLIP, к которой применяются хуки |
| `apply_to_conds` | BOOLEAN | Да | - | Применять ли хуки к выходам кондиционирования (по умолчанию: True) |
| `schedule_clip` | BOOLEAN | Да | - | Включить ли планирование CLIP (по умолчанию: False) |
| `hooks` | HOOKS | Нет | - | Необязательная группа хуков для применения к модели CLIP |

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `clip` | CLIP | Клонированная модель CLIP с примененными указанными хуками |
