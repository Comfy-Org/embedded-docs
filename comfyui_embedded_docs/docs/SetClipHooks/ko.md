SetClipHooks 노드는 CLIP 모델에 사용자 정의 후크를 적용하여 동작을 고급 수준으로 수정할 수 있도록 합니다. 이 노드는 컨디셔닝 출력에 후크를 적용하고 선택적으로 클립 스케줄링 기능을 활성화할 수 있습니다. 지정된 후크 구성이 적용된 입력 CLIP 모델의 복제본을 생성합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `clip` | CLIP | 예 | - | 후크를 적용할 CLIP 모델 |
| `조건에 적용` | BOOLEAN | 예 | - | 컨디셔닝 출력에 후크를 적용할지 여부 (기본값: True) |
| `clip 스케쥴 사용` | BOOLEAN | 예 | - | 클립 스케줄링을 활성화할지 여부 (기본값: False) |
| `후크` | HOOKS | 아니요 | - | CLIP 모델에 적용할 선택적 후크 그룹 |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `clip` | CLIP | 지정된 후크가 적용된 CLIP 모델의 복제본 |
