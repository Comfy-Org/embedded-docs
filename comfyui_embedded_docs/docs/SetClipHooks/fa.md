گره SetClipHooks به شما امکان می‌دهد هوک‌های سفارشی را روی یک مدل CLIP اعمال کنید و تغییرات پیشرفته‌ای در رفتار آن ایجاد نمایید. این گره می‌تواند هوک‌ها را روی خروجی‌های شرطی‌سازی (conditioning) اعمال کرده و به‌صورت اختیاری قابلیت زمان‌بندی CLIP را فعال کند. این گره یک کپی شبیه‌سازی‌شده از مدل CLIP ورودی با تنظیمات هوک مشخص‌شده ایجاد می‌کند.

## ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|---------|----------|-------|--------|---------|
| `clip` | CLIP | بله | - | مدل CLIP که هوک‌ها روی آن اعمال می‌شوند |
| `اعمال به شرایط` | BOOLEAN | بله | - | آیا هوک‌ها روی خروجی‌های شرطی‌سازی اعمال شوند (پیش‌فرض: True) |
| `زمان‌بندی clip` | BOOLEAN | بله | - | آیا زمان‌بندی CLIP فعال شود (پیش‌فرض: False) |
| `hookها` | HOOKS | خیر | - | گروه هوک اختیاری برای اعمال روی مدل CLIP |
