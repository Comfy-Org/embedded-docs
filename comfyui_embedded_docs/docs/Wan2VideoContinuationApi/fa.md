### مرور کلی

گره ادامه ویدیوی Wan 2.7، یک بخش ویدیویی جدید تولید می‌کند که به طور پیوسته از انتهای یک کلیپ ویدیوی ورودی ادامه می‌یابد. این گره از مدل Wan 2.7 برای合成 ادامه ویدیو بر اساس یک متن راهنما (prompt) استفاده می‌کند و می‌تواند به صورت اختیاری پایان ویدیو را به سمت یک فریم هدف خاص هدایت کند.

## ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
| :--- | :--- | :--- | :--- | :--- |
| `مدل` | COMBO | بله | `"wan2.7-i2v"` | مدل تولید ویدیو برای استفاده. |
| `model.prompt` | STRING | بله | - | راهنمای توصیف عناصر و ویژگی‌های بصری. از زبان انگلیسی و چینی پشتیبانی می‌کند. (پیش‌فرض: رشته خالی) |
| `model.negative_prompt` | STRING | بله | - | راهنمای منفی که موارد اجتناب را توصیف می‌کند. (پیش‌فرض: رشته خالی) |
| `model.resolution` | COMBO | بله | `"720P"`<br>`"1080P"` | وضوح تصویر برای ویدیوی خروجی. |
| `model.duration` | INT | بله | 2 تا 15 | مدت زمان کل خروجی بر حسب ثانیه. مدل ادامه ویدیو را برای پر کردن زمان باقی‌مانده پس از کلیپ ورودی تولید می‌کند. (پیش‌فرض: 5) |
| `کلیپ اول` | VIDEO | بله | - | ویدیوی ورودی برای ادامه از آن. مدت زمان: 2 تا 10 ثانیه. نسبت ابعاد خروجی از این ویدیو استخراج می‌شود. |
| `فریم آخر` | IMAGE | خیر | - | تصویر فریم آخر. ادامه ویدیو به سمت این فریم انتقال می‌یابد. |
| `بذر` | INT | بله | 0 تا 2147483647 | دانه (seed) برای استفاده در تولید. (پیش‌فرض: 0) |
| `گسترش پرامپت` | BOOLEAN | بله | - | آیا راهنما با کمک هوش مصنوعی تقویت شود. (پیش‌فرض: True) |
| `واترمارک` | BOOLEAN | بله | - | آیا یک واترمارک تولیدشده توسط هوش مصنوعی به نتیجه اضافه شود. (پیش‌فرض: False) |

**توجه:** ویدیوی ورودی `first_clip` باید بین 2 تا 10 ثانیه مدت زمان داشته باشد.
