# ترجمة وثيقة عقدة استمرار الفيديو Wan 2.7

عُقدة استمرار الفيديو Wan 2.7 تُنشئ مقطع فيديو جديدًا يستمر بسلاسة من نهاية مقطع فيديو الإدخال. تستخدم العقدة نموذج Wan 2.7 لتوليد الاستمرارية بناءً على موجه نصي، ويمكنها توجيه النهاية نحو إطار هدف محدد بشكل اختياري.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
| :--- | :--- | :--- | :--- | :--- |
| `model` | COMBO | نعم | `"wan2.7-i2v"` | نموذج توليد الفيديو المراد استخدامه. |
| `model.prompt` | STRING | نعم | - | موجه يصف العناصر والخصائص البصرية. يدعم اللغتين الإنجليزية والصينية. (الافتراضي: سلسلة فارغة) |
| `model.negative_prompt` | STRING | نعم | - | موجه سلبي يصف ما يجب تجنبه. (الافتراضي: سلسلة فارغة) |
| `model.resolution` | COMBO | نعم | `"720P"`<br>`"1080P"` | دقة الفيديو الناتج. |
| `model.duration` | INT | نعم | 2 إلى 15 | المدة الإجمالية للإخراج بالثواني. يُنشئ النموذج استمرارية لملء الوقت المتبقي بعد مقطع الإدخال. (الافتراضي: 5) |
| `first_clip` | VIDEO | نعم | - | فيديو الإدخال المراد الاستمرار منه. المدة: 2ث-10ث. تُشتق نسبة العرض إلى الارتفاع للإخراج من هذا الفيديو. |
| `last_frame` | IMAGE | لا | - | صورة الإطار الأخير. ستنتقل الاستمرارية نحو هذا الإطار. |
| `seed` | INT | نعم | 0 إلى 2147483647 | البذرة المستخدمة في التوليد. (الافتراضي: 0) |
| `prompt_extend` | BOOLEAN | نعم | - | ما إذا كان سيتم تحسين الموجه بمساعدة الذكاء الاصطناعي. (الافتراضي: صحيح) |
| `watermark` | BOOLEAN | نعم | - | ما إذا كان سيتم إضافة علامة مائية منشأة بالذكاء الاصطناعي إلى النتيجة. (الافتراضي: خطأ) |

**ملاحظة:** يجب أن تتراوح مدة فيديو الإدخال `first_clip` بين 2 و10 ثوانٍ.

| `output` | VIDEO | استمرارية الفيديو المُنشأة. |
