Вот перевод документации на русский язык в соответствии с вашими правилами:

Узел Wan 2.7 Video Continuation генерирует новый видеосегмент, который плавно продолжает входной видеоклип с его конца. Он использует модель Wan 2.7 для синтеза продолжения на основе текстовой подсказки и может опционально направлять завершение к определенному целевому кадру.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
| :--- | :--- | :--- | :--- | :--- |
| `model` | COMBO | Да | `"wan2.7-i2v"` | Модель генерации видео для использования. |
| `model.prompt` | STRING | Да | - | Подсказка, описывающая элементы и визуальные особенности. Поддерживает английский и китайский языки. (по умолчанию: пустая строка) |
| `model.negative_prompt` | STRING | Да | - | Негативная подсказка, описывающая, чего следует избегать. (по умолчанию: пустая строка) |
| `model.resolution` | COMBO | Да | `"720P"`<br>`"1080P"` | Разрешение выходного видео. |
| `model.duration` | INT | Да | от 2 до 15 | Общая длительность выходного видео в секундах. Модель генерирует продолжение, заполняя оставшееся время после входного клипа. (по умолчанию: 5) |
| `first_clip` | VIDEO | Да | - | Входное видео, которое нужно продолжить. Длительность: от 2 до 10 секунд. Соотношение сторон выходного видео определяется из этого видео. |
| `last_frame` | IMAGE | Нет | - | Изображение последнего кадра. Продолжение будет плавно переходить к этому кадру. |
| `seed` | INT | Да | от 0 до 2147483647 | Зерно для генерации. (по умолчанию: 0) |
| `prompt_extend` | BOOLEAN | Да | - | Следует ли улучшать подсказку с помощью ИИ-ассистента. (по умолчанию: True) |
| `watermark` | BOOLEAN | Да | - | Следует ли добавлять AI-водяной знак к результату. (по умолчанию: False) |

**Примечание:** Длительность входного видео `first_clip` должна составлять от 2 до 10 секунд.

## Выходные параметры

| Имя выходного параметра | Тип данных | Описание |
| :--- | :--- | :--- |
| `output` | VIDEO | Сгенерированное продолжение видео. |
