## 개요

EmptyARVideoLatent 노드는 비디오 생성을 위한 비어 있는 잠재 표현(Latent Representation)을 생성합니다. 지정된 크기, 화면 비율 및 길이를 가진 0으로 채워진 텐서를 제공하여 비디오 생성 프로세스를 초기화하는 데 사용됩니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `width` | INT | 예 | 16 ~ 8192 (단위: 16) | 비디오 프레임의 너비(픽셀 단위, 기본값: 832) |
| `height` | INT | 예 | 16 ~ 8192 (단위: 16) | 비디오 프레임의 높이(픽셀 단위, 기본값: 480) |
| `length` | INT | 예 | 1 ~ 1024 (단위: 4) | 비디오의 프레임 수(기본값: 81) |
| `batch_size` | INT | 예 | 1 ~ 64 | 단일 배치에서 생성할 비디오 수(기본값: 1) |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `LATENT` | LATENT | 0으로 채워진 잠재 텐서로, 지정된 크기, 길이 및 배치 크기를 가진 빈 비디오 잠재 공간을 나타냅니다. 텐서 형태는 [batch_size, 16, lat_t, height/8, width/8]이며, 여기서 lat_t는 길이(length)로부터 계산됩니다. |
