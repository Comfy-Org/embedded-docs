## Обзор

Узел EmptyARVideoLatent создает пустое, незаполненное латентное представление для генерации видео. Он используется для инициализации процесса генерации видео путем создания тензора из нулей с заданными размерами, соотношением сторон и длиной.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `width` | INT | Да | от 16 до 8192 (шаг: 16) | Ширина кадров видео в пикселях (по умолчанию: 832) |
| `height` | INT | Да | от 16 до 8192 (шаг: 16) | Высота кадров видео в пикселях (по умолчанию: 480) |
| `length` | INT | Да | от 1 до 1024 (шаг: 4) | Количество кадров в видео (по умолчанию: 81) |
| `batch_size` | INT | Да | от 1 до 64 | Количество видео для генерации в одном пакете (по умолчанию: 1) |

## Выходные данные

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `LATENT` | LATENT | Латентный тензор, заполненный нулями, представляющий пустое латентное пространство видео с заданными размерами, длиной и размером пакета. Форма тензора: [batch_size, 16, lat_t, height/8, width/8], где lat_t вычисляется на основе длины. |
