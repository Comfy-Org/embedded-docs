گره `ConditioningSetPropertiesAndCombine` داده‌های شرطی‌سازی را با اعمال ویژگی‌های یک ورودی شرطی‌سازی جدید بر روی یک ورودی شرطی‌سازی موجود، تغییر می‌دهد. این گره دو مجموعه شرطی‌سازی را با کنترل شدت شرطی‌سازی جدید و تعیین نحوه اعمال ناحیه شرطی‌سازی، ترکیب می‌کند.

## ورودی‌ها

| پارامتر | نوع داده | نوع ورودی | پیش‌فرض | محدوده | توضیحات |
|---------|----------|-----------|---------|--------|---------|
| `cond` | CONDITIONING | اجباری | - | - | داده‌های شرطی‌سازی اصلی که باید تغییر کنند |
| `cond_NEW` | CONDITIONING | اجباری | - | - | داده‌های شرطی‌سازی جدید که ویژگی‌های قابل اعمال را فراهم می‌کنند |
| `strength` | FLOAT | اجباری | 1.0 | 0.0 - 10.0 | شدت ویژگی‌های شرطی‌سازی جدید را کنترل می‌کند |
| `set_cond_area` | STRING | اجباری | default | ["default", "mask bounds"] | نحوه اعمال ناحیه شرطی‌سازی را تعیین می‌کند |
| `mask` | MASK | اختیاری | - | - | ماسک اختیاری برای تعریف نواحی خاص شرطی‌سازی |
| `hooks` | HOOKS | اختیاری | - | - | توابع هوک اختیاری برای پردازش سفارشی |
| `timesteps` | TIMESTEPS_RANGE | اختیاری | - | - | محدوده گام زمانی اختیاری برای کنترل زمان اعمال شرطی‌سازی |

**توجه:** هنگامی که `mask` ارائه می‌شود، پارامتر `set_cond_area` می‌تواند از "mask bounds" برای محدود کردن اعمال شرطی‌سازی به نواحی ماسک‌شده استفاده کند.
