Узел SplitSigmasDenoise разделяет последовательность значений сигма на две части на основе параметра силы шумоподавления. Он разбивает входные сигмы на последовательности высоких и низких сигм, где точка разделения определяется умножением общего количества шагов на коэффициент шумоподавления. Это позволяет разделить график шума на диапазоны различной интенсивности для специализированной обработки.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `сигмы` | SIGMAS | Да | - | Входная последовательность значений сигма, представляющая график шума |
| `шумоподавление` | FLOAT | Да | 0.0 - 1.0 | Коэффициент силы шумоподавления, определяющий точку разделения последовательности сигм (по умолчанию: 1.0) |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `high_sigmas` | SIGMAS | Первая часть последовательности сигм, содержащая более высокие значения сигма |
| `low_sigmas` | SIGMAS | Вторая часть последовательности сигм, содержащая более низкие значения сигма |
