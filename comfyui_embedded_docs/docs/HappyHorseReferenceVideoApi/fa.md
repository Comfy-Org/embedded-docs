## نمای کلی

این گره با استفاده از مدل HappyHorse، ویدیویی از یک شخص یا شیء بر اساس تصاویر مرجع تولید می‌کند. از ساخت ویدیو با یک شخصیت واحد یا چند شخصیت که با یکدیگر تعامل دارند پشتیبانی می‌کند.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `model` | COMBO | بله | `"happyhorse-1.0-r2v"` | مدل HappyHorse مورد استفاده برای تولید ویدیو. |
| `prompt` | STRING | بله | نامشخص | توضیح متنی ویدیویی که می‌خواهید تولید کنید. برای ارجاع به شخصیت‌های مرجع از شناسه‌هایی مانند 'character1' و 'character2' استفاده کنید. |
| `resolution` | COMBO | بله | `"720P"`<br>`"1080P"` | وضوح تصویر ویدیوی تولید شده. |
| `ratio` | COMBO | بله | `"16:9"`<br>`"9:16"`<br>`"1:1"`<br>`"4:3"`<br>`"3:4"` | نسبت ابعاد ویدیوی تولید شده. |
| `duration` | INT | بله | 3 تا 15 | مدت زمان ویدیوی تولید شده بر حسب ثانیه (پیش‌فرض: 5). |
| `reference_images` | IMAGE | بله | 1 تا 9 | یک یا چند تصویر مرجع از شخص یا شیء مورد نظر برای نمایش در ویدیو. حداقل باید یک تصویر ارائه دهید. |
| `seed` | INT | خیر | 0 تا 2147483647 | مقدار seed برای تولید قابل تکرار (پیش‌فرض: 0). می‌توان seed را طوری تنظیم کرد که پس از هر تولید به‌طور خودکار تغییر کند. |
| `watermark` | BOOLEAN | خیر | True یا False | آیا به ویدیوی نهایی واترمارک تولیدشده با هوش مصنوعی اضافه شود یا خیر (پیش‌فرض: False). |
