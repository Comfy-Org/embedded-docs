## Обзор

Этот узел генерирует видео с изображением человека или объекта на основе референсных изображений с использованием модели HappyHorse. Поддерживает создание видео с одним персонажем или несколькими персонажами, взаимодействующими друг с другом.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `model` | COMBO | Да | `"happyhorse-1.0-r2v"` | Модель HappyHorse, используемая для генерации видео. |
| `prompt` | STRING | Да | Н/Д | Текстовое описание желаемого видео. Используйте идентификаторы, такие как 'character1' и 'character2', для обозначения референсных персонажей. |
| `resolution` | COMBO | Да | `"720P"`<br>`"1080P"` | Разрешение генерируемого видео. |
| `ratio` | COMBO | Да | `"16:9"`<br>`"9:16"`<br>`"1:1"`<br>`"4:3"`<br>`"3:4"` | Соотношение сторон генерируемого видео. |
| `duration` | INT | Да | от 3 до 15 | Длительность генерируемого видео в секундах (по умолчанию: 5). |
| `reference_images` | IMAGE | Да | от 1 до 9 | Одно или несколько референсных изображений человека или объекта для отображения в видео. Необходимо предоставить как минимум одно изображение. |
| `seed` | INT | Нет | от 0 до 2147483647 | Значение зерна для воспроизводимой генерации (по умолчанию: 0). Зерно можно настроить на автоматическое изменение после каждой генерации. |
| `watermark` | BOOLEAN | Нет | True или False | Добавлять ли AI-водяной знак к результирующему видео (по умолчанию: False). |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `VIDEO` | VIDEO | Сгенерированный видеофайл. |
