# SavePointCloud

گره Save Point Cloud یک فایل ابر نقطه‌ای سه‌بعدی را در دایرکتوری خروجی ذخیره می‌کند و به‌صورت اختیاری داده‌های پیش‌نمایش را برای نمایشگر سه‌بعدی فراهم می‌سازد. این گره ضمن مدیریت نام‌گذاری و ذخیره‌سازی فایل، اطلاعات دوربین و مدل را برای اهداف نمایشی عبور می‌دهد.

## ورودی‌ها

| پارامتر | توضیحات | نوع داده | الزامی | محدوده |
|---------|---------|----------|--------|--------|
| `model_3d` | فایل ابر نقطه‌ای (ply.) | FILE3D_POINT_CLOUD_ANY یا FILE3D_PLY | بله | - |
| `filename_prefix` | پیشوند نام فایل ذخیره‌شده (پیش‌فرض: "3d/ComfyUI") | STRING | بله | - |
| `viewport_state` | وضعیت جاری نمای دید شامل اطلاعات دوربین و مدل | LOAD3D | بله | - |
| `model_3d_info` | اطلاعات تکمیلی مدل برای نمایشگر سه‌بعدی | LOAD3D_MODEL_INFO | خیر | - |
| `camera_info` | اطلاعات دوربین برای نمایشگر سه‌بعدی | LOAD3D_CAMERA | خیر | - |
| `width` | عرض نمایش پیش‌نمایش بر حسب پیکسل (پیش‌فرض: 1024) | INT | بله | 1 تا 4096 |
| `height` | ارتفاع نمایش پیش‌نمایش بر حسب پیکسل (پیش‌فرض: 1024) | INT | بله | 1 تا 4096 |

## خروجی‌ها

| نام خروجی | توضیحات | نوع داده |
|-----------|---------|----------|
| `model_3d` | فایل ابر نقطه‌ای ذخیره‌شده | FILE3D_POINT_CLOUD_ANY |
| `model_3d_info` | اطلاعات مدل برای نمایشگر سه‌بعدی | LOAD3D_MODEL_INFO |
| `camera_info` | اطلاعات دوربین برای نمایشگر سه‌بعدی | LOAD3D_CAMERA |
| `width` | عرض نمایش پیش‌نمایش | INT |
| `height` | ارتفاع نمایش پیش‌نمایش | INT |

> این مستند با هوش مصنوعی تهیه شده است. اگر خطایی دیدید یا پیشنهادی برای بهبود دارید، خوشحال می‌شویم مشارکت کنید! [ویرایش در GitHub](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/SavePointCloud/fa.md)
