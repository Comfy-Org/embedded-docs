# SavePointCloud

گره Save Point Cloud یک فایل ابر نقطه‌ای سه‌بعدی را در دایرکتوری خروجی ComfyUI ذخیره می‌کند. همچنین داده‌های ابر نقطه‌ای و اطلاعات اختیاری دوربین و مدل را برای استفاده توسط سایر گره‌های workflow شما عبور می‌دهد.

## ورودی‌ها

| پارامتر | توضیحات | نوع داده | الزامی | محدوده |
|-----------|-------------|-----------|----------|-------|
| `model_3d` | فایل ابر نقطه‌ای (ply.) | FILE3D | بله | - |
| `filename_prefix` | پیشوند نام فایل ذخیره‌شده (پیش‌فرض: "3d/ComfyUI") | STRING | بله | - |
| `viewport_state` | وضعیت حاصل از یک گره نمای سه‌بعدی | LOAD3D | بله | - |
| `model_3d_info` | اطلاعات مدل سه‌بعدی اختیاری برای استفاده پیشرفته | LOAD3DMODELINFO | خیر | - |
| `camera_info` | اطلاعات دوربین اختیاری برای استفاده پیشرفته | LOAD3DCAMERA | خیر | - |
| `width` | عرض تصویر پیش‌نمایش بر حسب پیکسل (پیش‌فرض: 1024) | INT | بله | حداقل: 1، حداکثر: 4096، گام: 1 |
| `height` | ارتفاع تصویر پیش‌نمایش بر حسب پیکسل (پیش‌فرض: 1024) | INT | بله | حداقل: 1، حداکثر: 4096، گام: 1 |

## خروجی‌ها

| نام خروجی | توضیحات | نوع داده |
|-------------|-------------|-----------|
| `model_3d` | فایل ابر نقطه‌ای ذخیره‌شده | FILE3D |
| `model_3d_info` | اطلاعات مربوط به مدل سه‌بعدی | LOAD3DMODELINFO |
| `camera_info` | اطلاعات دوربین برای نمای سه‌بعدی | LOAD3DCAMERA |
| `width` | عرض تصویر پیش‌نمایش | INT |
| `height` | ارتفاع تصویر پیش‌نمایش | INT |

> این مستند با هوش مصنوعی تهیه شده است. اگر خطایی دیدید یا پیشنهادی برای بهبود دارید، خوشحال می‌شویم مشارکت کنید! [ویرایش در GitHub](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/SavePointCloud/fa.md)
