# SavePointCloud

Узел Save Point Cloud сохраняет файл облака точек 3D в выходной каталог ComfyUI. Он также передает данные облака точек и дополнительную информацию о камере и модели для использования другими узлами в вашем рабочем процессе.

## Входные параметры

| Параметр | Описание | Тип данных | Обязательный | Диапазон |
|----------|----------|------------|--------------|----------|
| `model_3d` | Файл облака точек (.ply) | FILE3D | Да | - |
| `filename_prefix` | Префикс для имени сохраняемого файла (по умолчанию: "3d/ComfyUI") | STRING | Да | - |
| `viewport_state` | Состояние из узла 3D-вьюпорта | LOAD3D | Да | - |
| `model_3d_info` | Дополнительная информация о 3D-модели для расширенного использования | LOAD3DMODELINFO | Нет | - |
| `camera_info` | Дополнительная информация о камере для расширенного использования | LOAD3DCAMERA | Нет | - |
| `width` | Ширина изображения предварительного просмотра в пикселях (по умолчанию: 1024) | INT | Да | мин: 1, макс: 4096, шаг: 1 |
| `height` | Высота изображения предварительного просмотра в пикселях (по умолчанию: 1024) | INT | Да | мин: 1, макс: 4096, шаг: 1 |

## Выходные данные

| Имя выхода | Описание | Тип данных |
|------------|----------|------------|
| `model_3d` | Сохраненный файл облака точек | FILE3D |
| `model_3d_info` | Информация о 3D-модели | LOAD3DMODELINFO |
| `camera_info` | Информация о камере для вьюпорта | LOAD3DCAMERA |
| `width` | Ширина изображения предварительного просмотра | INT |
| `height` | Высота изображения предварительного просмотра | INT |
