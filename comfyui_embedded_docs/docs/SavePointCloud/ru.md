# SavePointCloud

Вот перевод документации узла ComfyUI на русский язык в соответствии с вашими правилами.

---

Узел Save Point Cloud сохраняет файл 3D-облака точек в выходной каталог и, при необходимости, предоставляет данные предварительного просмотра для 3D-вьювера. Он обрабатывает именование и сохранение файла, а также передает информацию о камере и модели для целей отображения.

## Входы

| Параметр | Описание | Тип данных | Обязательный | Диапазон |
|----------|----------|------------|--------------|----------|
| `model_3d` | Файл облака точек (.ply) | FILE3D_POINT_CLOUD_ANY или FILE3D_PLY | Да | - |
| `filename_prefix` | Префикс для имени сохраняемого файла (по умолчанию: "3d/ComfyUI") | STRING | Да | - |
| `viewport_state` | Текущее состояние области просмотра, содержащее информацию о камере и модели | LOAD3D | Да | - |
| `model_3d_info` | Дополнительная информация о модели для 3D-вьювера | LOAD3D_MODEL_INFO | Нет | - |
| `camera_info` | Информация о камере для 3D-вьювера | LOAD3D_CAMERA | Нет | - |
| `width` | Ширина окна предварительного просмотра в пикселях (по умолчанию: 1024) | INT | Да | от 1 до 4096 |
| `height` | Высота окна предварительного просмотра в пикселях (по умолчанию: 1024) | INT | Да | от 1 до 4096 |

## Выходы

| Имя выхода | Описание | Тип данных |
|------------|----------|------------|
| `model_3d` | Сохраненный файл облака точек | FILE3D_POINT_CLOUD_ANY |
| `model_3d_info` | Информация о модели для 3D-вьювера | LOAD3D_MODEL_INFO |
| `camera_info` | Информация о камере для 3D-вьювера | LOAD3D_CAMERA |
| `width` | Ширина окна предварительного просмотра | INT |
| `height` | Высота окна предварительного просмотра | INT |
