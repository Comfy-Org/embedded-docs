# SavePointCloud

عقدة حفظ سحابة النقاط تقوم بحفظ ملف سحابة نقاط ثلاثي الأبعاد في دليل مخرجات ComfyUI الخاص بك. كما تقوم بتمرير بيانات سحابة النقاط ومعلومات الكاميرا والنموذج الاختيارية لاستخدامها بواسطة عقد أخرى في سير عملك.

## المدخلات

| المعامل | الوصف | نوع البيانات | إلزامي | النطاق |
|-----------|-------------|-----------|----------|-------|
| `model_3d` | ملف سحابة النقاط (.ply) | FILE3D | نعم | - |
| `filename_prefix` | بادئة اسم الملف المحفوظ (الافتراضي: "3d/ComfyUI") | STRING | نعم | - |
| `viewport_state` | حالة من عقدة منفذ عرض ثلاثي الأبعاد | LOAD3D | نعم | - |
| `model_3d_info` | معلومات نموذج ثلاثي الأبعاد اختيارية للاستخدام المتقدم | LOAD3DMODELINFO | لا | - |
| `camera_info` | معلومات كاميرا اختيارية للاستخدام المتقدم | LOAD3DCAMERA | لا | - |
| `width` | عرض الصورة المعاينة بالبكسل (الافتراضي: 1024) | INT | نعم | min: 1, max: 4096, step: 1 |
| `height` | ارتفاع الصورة المعاينة بالبكسل (الافتراضي: 1024) | INT | نعم | min: 1, max: 4096, step: 1 |

## المخرجات

| اسم المخرج | الوصف | نوع البيانات |
|-------------|-------------|-----------|
| `model_3d` | ملف سحابة النقاط المحفوظ | FILE3D |
| `model_3d_info` | معلومات حول النموذج ثلاثي الأبعاد | LOAD3DMODELINFO |
| `camera_info` | معلومات الكاميرا لمنفذ العرض | LOAD3DCAMERA |
| `width` | عرض الصورة المعاينة | INT |
| `height` | ارتفاع الصورة المعاينة | INT |
