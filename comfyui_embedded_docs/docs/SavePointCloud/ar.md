# SavePointCloud

يقوم عقدة حفظ سحابة النقاط بحفظ ملف سحابة نقاط ثلاثية الأبعاد في دليل الإخراج، مع توفير بيانات معاينة اختيارية لعارض العرض ثلاثي الأبعاد. تتعامل هذه العقدة مع تسمية الملفات وحفظها، بالإضافة إلى تمرير معلومات الكاميرا والنموذج لأغراض العرض.

## المدخلات

| المعامل | الوصف | نوع البيانات | إلزامي | النطاق |
|----------|-------|---------------|---------|--------|
| `model_3d` | ملف سحابة النقاط (.ply) | FILE3D_POINT_CLOUD_ANY أو FILE3D_PLY | نعم | - |
| `filename_prefix` | بادئة اسم الملف المحفوظ (الافتراضي: "3d/ComfyUI") | STRING | نعم | - |
| `viewport_state` | حالة منفذ العرض الحالية التي تحتوي على معلومات الكاميرا والنموذج | LOAD3D | نعم | - |
| `model_3d_info` | معلومات إضافية عن النموذج لعارض العرض ثلاثي الأبعاد | LOAD3D_MODEL_INFO | لا | - |
| `camera_info` | معلومات الكاميرا لعارض العرض ثلاثي الأبعاد | LOAD3D_CAMERA | لا | - |
| `width` | عرض شاشة المعاينة بالبكسل (الافتراضي: 1024) | INT | نعم | 1 إلى 4096 |
| `height` | ارتفاع شاشة المعاينة بالبكسل (الافتراضي: 1024) | INT | نعم | 1 إلى 4096 |

## المخرجات

| اسم المخرج | الوصف | نوع البيانات |
|-------------|-------|---------------|
| `model_3d` | ملف سحابة النقاط المحفوظ | FILE3D_POINT_CLOUD_ANY |
| `model_3d_info` | معلومات النموذج لعارض العرض ثلاثي الأبعاد | LOAD3D_MODEL_INFO |
| `camera_info` | معلومات الكاميرا لعارض العرض ثلاثي الأبعاد | LOAD3D_CAMERA |
| `width` | عرض شاشة المعاينة | INT |
| `height` | ارتفاع شاشة المعاينة | INT |
