Вот перевод документации на русский язык:

Этот узел реализует метод масштабирования эпсилон из исследовательской статьи "Elucidating the Exposure Bias in Diffusion Models" (arxiv.org/abs/2308.15321v6). Он работает путем масштабирования предсказанного шума в процессе сэмплирования, чтобы помочь уменьшить смещение экспозиции, что может привести к улучшению качества генерируемых изображений. В данной реализации используется "равномерное расписание", рекомендованное авторами статьи за его практичность и эффективность.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `model` | MODEL | Да | - | Модель, к которой будет применён патч масштабирования эпсилон. |
| `scaling_factor` | FLOAT | Нет | 0.5 - 1.5 | Коэффициент, на который масштабируется предсказанный шум. Значение больше 1.0 уменьшает шум, а значение меньше 1.0 увеличивает его (по умолчанию: 1.005). |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `model` | MODEL | Модифицированная версия входной модели с применённой функцией масштабирования эпсилон к её процессу сэмплирования. |
