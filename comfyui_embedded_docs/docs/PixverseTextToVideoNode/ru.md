Вот перевод документации на русский язык:

Генерирует видео на основе текстового запроса и различных параметров генерации. Этот узел создает видеоконтент с использованием API PixVerse, позволяя управлять соотношением сторон, качеством, длительностью, стилем движения и другими параметрами.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `prompt` | STRING | Да | - | Запрос для генерации видео (по умолчанию: "") |
| `aspect_ratio` | COMBO | Да | Опции из PixverseAspectRatio | Соотношение сторон для генерируемого видео |
| `quality` | COMBO | Да | Опции из PixverseQuality | Настройка качества видео (по умолчанию: PixverseQuality.res_540p) |
| `duration_seconds` | COMBO | Да | Опции из PixverseDuration | Длительность генерируемого видео в секундах |
| `motion_mode` | COMBO | Да | Опции из PixverseMotionMode | Стиль движения для генерации видео |
| `seed` | INT | Да | от 0 до 2147483647 | Зерно для генерации видео (по умолчанию: 0) |
| `negative_prompt` | STRING | Нет | - | Необязательное текстовое описание нежелательных элементов на изображении (по умолчанию: "") |
| `pixverse_template` | CUSTOM | Нет | - | Необязательный шаблон для влияния на стиль генерации, созданный узлом PixVerse Template |

**Примечание:** При использовании качества 1080p режим движения автоматически устанавливается на обычный, а длительность ограничивается 5 секундами. Для длительности, отличной от 5 секунд, режим движения также автоматически устанавливается на обычный.

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `output` | VIDEO | Сгенерированный видеофайл |
