### نمای کلی

ویدیوهایی بر اساس یک توضیح متنی و پارامترهای مختلف تولید می‌کند. این گره با استفاده از API PixVerse محتوای ویدیویی ایجاد می‌کند و امکان کنترل نسبت تصویر، کیفیت، مدت زمان، سبک حرکت و موارد دیگر را فراهم می‌کند.

## ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|---------|----------|-------|--------|---------|
| `پرامپت` | STRING | بله | - | توضیح متنی برای تولید ویدیو (پیش‌فرض: "") |
| `نسبت تصویر` | COMBO | بله | گزینه‌های PixverseAspectRatio | نسبت تصویر برای ویدیوی تولید شده |
| `کیفیت` | COMBO | بله | گزینه‌های PixverseQuality | تنظیم کیفیت ویدیو (پیش‌فرض: PixverseQuality.res_540p) |
| `مدت زمان (ثانیه)` | COMBO | بله | گزینه‌های PixverseDuration | مدت زمان ویدیوی تولید شده بر حسب ثانیه |
| `حالت حرکت` | COMBO | بله | گزینه‌های PixverseMotionMode | سبک حرکت برای تولید ویدیو |
| `seed` | INT | بله | 0 تا 2147483647 | دانه (seed) برای تولید ویدیو (پیش‌فرض: 0) |
| `پرامپت منفی` | STRING | خیر | - | توضیح متنی اختیاری از عناصر نامطلوب در تصویر (پیش‌فرض: "") |
| `قالب PixVerse` | CUSTOM | خیر | - | یک قالب اختیاری برای تأثیرگذاری بر سبک تولید که توسط گره PixVerse Template ایجاد شده است |

**توجه:** هنگام استفاده از کیفیت 1080p، حالت حرکت به‌طور خودکار روی normal تنظیم می‌شود و مدت زمان به 5 ثانیه محدود می‌گردد. برای مدت‌های غیر از 5 ثانیه نیز حالت حرکت به‌طور خودکار روی normal تنظیم می‌شود.
