Вот перевод документации на русский язык:

Узел PhotoMakerEncode обрабатывает изображения и текст для создания данных кондиционирования для генерации изображений с помощью ИИ. Он принимает эталонное изображение и текстовый запрос, затем создаёт эмбеддинги, которые можно использовать для направления генерации изображения на основе визуальных характеристик эталонного изображения. Узел специально ищет токен "photomaker" в тексте, чтобы определить, куда применить кондиционирование на основе изображения.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `photomaker` | PHOTOMAKER | Да | - | Модель PhotoMaker, используемая для обработки изображения и создания эмбеддингов |
| `изображение` | IMAGE | Да | - | Эталонное изображение, предоставляющее визуальные характеристики для кондиционирования |
| `clip` | CLIP | Да | - | Модель CLIP, используемая для токенизации и кодирования текста |
| `текст` | STRING | Да | - | Текстовый запрос для генерации кондиционирования (по умолчанию: "photograph of photomaker") |

**Примечание:** Если текст содержит слово "photomaker", узел применяет кондиционирование на основе изображения в этой позиции запроса. Если "photomaker" не найден в тексте, узел генерирует стандартное текстовое кондиционирование без влияния изображения.

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `CONDITIONING` | CONDITIONING | Данные кондиционирования, содержащие эмбеддинги изображения и текста для направления генерации изображения |
