Узел `StableCascade_StageC_VAEEncode` обрабатывает изображения через кодировщик VAE для создания латентных представлений для моделей Stable Cascade. Он принимает входное изображение и сжимает его с помощью указанной модели VAE, после чего выводит два латентных представления: одно для этапа C и заполнитель для этапа B. Параметр сжатия определяет, насколько изображение уменьшается перед кодированием.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `image` | IMAGE | Да | - | Входное изображение для кодирования в латентное пространство |
| `vae` | VAE | Да | - | Модель VAE, используемая для кодирования изображения |
| `compression` | INT | Нет | 4-128 | Коэффициент сжатия, применяемый к изображению перед кодированием (по умолчанию: 42) |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `stage_c` | LATENT | Закодированное латентное представление для этапа C модели Stable Cascade |
| `stage_b` | LATENT | Заполнитель латентного представления для этапа B (в настоящее время возвращает нули) |
