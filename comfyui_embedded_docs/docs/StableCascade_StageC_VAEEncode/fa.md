گره `StableCascade_StageC_VAEEncode` تصاویر را از طریق یک رمزگذار VAE پردازش می‌کند تا بازنمایی‌های نهفته (latent) برای مدل‌های Stable Cascade تولید کند. این گره یک تصویر ورودی دریافت کرده و با استفاده از مدل VAE مشخص‌شده آن را فشرده می‌کند، سپس دو بازنمایی نهفته خروجی می‌دهد: یکی برای مرحله C و یک مکان‌نما برای مرحله B. پارامتر فشرده‌سازی (compression) میزان مقیاس‌بندی تصویر قبل از رمزگذاری را کنترل می‌کند.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `image` | IMAGE | بله | - | تصویر ورودی که باید به فضای نهفته رمزگذاری شود |
| `vae` | VAE | بله | - | مدل VAE مورد استفاده برای رمزگذاری تصویر |
| `compression` | INT | خیر | 4-128 | ضریب فشرده‌سازی اعمال‌شده بر تصویر قبل از رمزگذاری (پیش‌فرض: 42) |

## خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-------------|-----------|-------------|
| `stage_c` | LATENT | بازنمایی نهفته رمزگذاری‌شده برای مرحله C مدل Stable Cascade |
| `stage_b` | LATENT | یک بازنمایی نهفته مکان‌نما برای مرحله B (در حال حاضر مقادیر صفر برمی‌گرداند) |
