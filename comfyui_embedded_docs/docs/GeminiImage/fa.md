گره GeminiImage پاسخ‌های متنی و تصویری را از مدل‌های هوش مصنوعی Gemini گوگل تولید می‌کند. این گره به شما امکان می‌دهد ورودی‌های چندوجهی شامل پرامپت‌های متنی، تصاویر و فایل‌ها را برای ایجاد خروجی‌های متنی و تصویری منسجم ارائه دهید. این گره تمام ارتباطات API و تجزیه پاسخ‌ها را با جدیدترین مدل‌های Gemini مدیریت می‌کند.

## ورودی‌ها

| پارامتر | نوع داده | نوع ورودی | پیش‌فرض | محدوده | توضیحات |
|-----------|-----------|------------|---------|-------|-------------|
| `prompt` | STRING | اجباری | "" | - | پرامپت متنی برای تولید |
| `model` | COMBO | اجباری | gemini_2_5_flash_image_preview | `gemini_2_5_flash_image_preview`<br>`gemini_2_5_pro_exp_03_25`<br>`gemini_2_0_flash_exp_image_generation` | مدل Gemini که برای تولید پاسخ‌ها استفاده می‌شود. |
| `seed` | INT | اجباری | 42 | 0 تا 18446744073709551615 | هنگامی که seed روی یک مقدار مشخص ثابت می‌شود، مدل نهایت تلاش خود را برای ارائه پاسخ یکسان به درخواست‌های تکراری به کار می‌گیرد. خروجی قطعی تضمین نمی‌شود. همچنین، تغییر مدل یا تنظیمات پارامترهایی مانند دما (temperature) می‌تواند حتی با استفاده از مقدار seed یکسان، باعث تغییر در پاسخ شود. به‌طور پیش‌فرض، از یک مقدار seed تصادفی استفاده می‌شود. |
| `images` | IMAGE | اختیاری | None | - | تصویر(های) اختیاری برای استفاده به عنوان زمینه (context) مدل. برای گنجاندن چندین تصویر، می‌توانید از گره Batch Images استفاده کنید. |
| `files` | GEMINI_INPUT_FILES | اختیاری | None | - | فایل(های) اختیاری برای استفاده به عنوان زمینه (context) مدل. ورودی‌های گره Gemini Generate Content Input Files را می‌پذیرد. |

**توجه:** این گره شامل پارامترهای پنهان (`auth_token`، `comfy_api_key`، `unique_id`) است که به‌طور خودکار توسط سیستم مدیریت می‌شوند و نیازی به ورودی کاربر ندارند.

## خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-------------|-----------|-------------|
| `IMAGE` | IMAGE | پاسخ تصویری تولید شده از مدل Gemini |
| `STRING` | STRING | پاسخ متنی تولید شده از مدل Gemini |
