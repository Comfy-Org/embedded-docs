Узел `ImageInvert` предназначен для инвертирования цветов изображения, преобразуя значение цвета каждого пикселя в его дополнительный цвет на цветовом круге. Эта операция полезна для создания негативных изображений или визуальных эффектов, требующих инверсии цветов.

## Входные параметры

| Параметр | Тип данных | Описание |
|-----------|-------------|-------------|
| `image`   | `IMAGE`     | Параметр `image` представляет входное изображение, подлежащее инвертированию. Он необходим для указания целевого изображения, цвета которого будут инвертированы, что влияет на выполнение узла и визуальный результат процесса инверсии. |

## Выходные параметры

| Параметр | Тип данных | Описание |
|-----------|-------------|-------------|
| `image`   | `IMAGE`     | На выходе получается инвертированная версия входного изображения, где значение цвета каждого пикселя преобразовано в его дополнительный цвет. |
