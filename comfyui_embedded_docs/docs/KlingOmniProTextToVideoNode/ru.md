Этот узел использует новейшую модель Kling AI для генерации видео по текстовому описанию. Он отправляет ваш запрос на удаленный API и возвращает сгенерированное видео. Узел позволяет управлять длительностью, формой, качеством видео, а также создавать многосценные раскадровки.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `model_name` | COMBO | Да | `"kling-v3-omni"`<br>`"kling-video-o1"` | Конкретная модель Kling для генерации видео (по умолчанию: `"kling-v3-omni"`). |
| `prompt` | STRING | Да | от 0 до 2500 символов | Текстовый запрос, описывающий содержимое видео. Может включать как положительные, так и отрицательные описания. Игнорируется при включенных раскадровках. |
| `aspect_ratio` | COMBO | Да | `"16:9"`<br>`"9:16"`<br>`"1:1"` | Форма или размеры генерируемого видео. |
| `duration` | INT | Да | от 3 до 15 секунд | Длительность видео в секундах (по умолчанию: 5). |
| `resolution` | COMBO | Нет | `"4k"`<br>`"1080p"`<br>`"720p"` | Качество или пиксельное разрешение видео (по умолчанию: `"1080p"`). |
| `storyboards` | DYNAMIC_COMBO | Нет | `"disabled"`<br>`"1 storyboard"`<br>`"2 storyboards"`<br>`"3 storyboards"`<br>`"4 storyboards"`<br>`"5 storyboards"`<br>`"6 storyboards"` | Генерация серии видеосегментов с индивидуальными запросами и длительностью. Игнорируется для модели o1. |
| `generate_audio` | BOOLEAN | Нет | True / False | Создавать ли аудиодорожку для видео (по умолчанию: False). |
| `seed` | INT | Нет | от 0 до 2147483647 | Сид управляет повторным запуском узла; результаты недетерминированы независимо от сида (по умолчанию: 0). |

### Ограничения параметров

- **Ограничения по моделям:**
  - Модель `kling-video-o1` поддерживает только длительность **5 или 10 секунд**.
  - Модель `kling-video-o1` **не** поддерживает генерацию аудио.
  - Модель `kling-video-o1` **не** поддерживает разрешение 4k.
  - Модель `kling-video-o1` **не** поддерживает раскадровки.
- **Ограничения раскадровок:**
  - При включенных раскадровках поле `prompt` игнорируется.
  - Каждая раскадровка требует собственного запроса (от 1 до 512 символов) и длительности.
  - Общая длительность всех раскадровок должна точно равняться глобальному параметру `duration`.
- **Требования к запросу:**
  - Если раскадровки **отключены**, поле `prompt` обязательно (минимум 1 символ).
  - Если раскадровки **включены**, поле `prompt` может быть пустым (0 символов).

| Имя выхода | Тип данных | Описание |

| `output` | VIDEO | Видео, сгенерированное на основе предоставленного текстового запроса и настроек. |
