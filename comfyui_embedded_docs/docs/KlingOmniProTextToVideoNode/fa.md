این گره از جدیدترین مدل هوش مصنوعی Kling برای تولید ویدیو بر اساس توضیحات متنی استفاده می‌کند. درخواست شما را به یک API راه دور ارسال کرده و ویدیوی تولیدشده را بازمی‌گرداند. این گره به شما امکان کنترل طول، شکل، کیفیت و حتی ایجاد استوری‌بوردهای چندصحنه‌ای را می‌دهد.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `model_name` | COMBO | بله | `"kling-v3-omni"`<br>`"kling-video-o1"` | مدل خاص Kling برای تولید ویدیو (پیش‌فرض: `"kling-v3-omni"`). |
| `prompt` | STRING | بله | ۰ تا ۲۵۰۰ کاراکتر | توضیح متنی محتوای ویدیو. می‌تواند شامل توصیفات مثبت و منفی باشد. در صورت فعال بودن استوری‌بورد نادیده گرفته می‌شود. |
| `aspect_ratio` | COMBO | بله | `"16:9"`<br>`"9:16"`<br>`"1:1"` | شکل یا ابعاد ویدیوی تولیدی. |
| `duration` | INT | بله | ۳ تا ۱۵ ثانیه | طول ویدیو بر حسب ثانیه (پیش‌فرض: ۵). |
| `resolution` | COMBO | خیر | `"4k"`<br>`"1080p"`<br>`"720p"` | کیفیت یا وضوح پیکسلی ویدیو (پیش‌فرض: `"1080p"`). |
| `storyboards` | DYNAMIC_COMBO | خیر | `"disabled"`<br>`"1 storyboard"`<br>`"2 storyboards"`<br>`"3 storyboards"`<br>`"4 storyboards"`<br>`"5 storyboards"`<br>`"6 storyboards"` | تولید مجموعه‌ای از بخش‌های ویدیویی با توضیحات و مدت‌زمان مجزا. برای مدل o1 نادیده گرفته می‌شود. |
| `generate_audio` | BOOLEAN | خیر | True / False | آیا برای ویدیو صدا تولید شود (پیش‌فرض: False). |
| `seed` | INT | خیر | ۰ تا ۲۱۴۷۴۸۳۶۴۷ | Seed مشخص می‌کند که گره دوباره اجرا شود یا خیر؛ نتایج صرف‌نظر از seed غیرقطعی هستند (پیش‌فرض: ۰). |

### محدودیت‌ها و قیود پارامترها

- **محدودیت‌های خاص مدل:**
  - مدل `kling-video-o1` فقط از مدت‌زمان‌های **۵ یا ۱۰ ثانیه** پشتیبانی می‌کند.
  - مدل `kling-video-o1` از تولید صدا **پشتیبانی نمی‌کند**.
  - مدل `kling-video-o1` از وضوح 4k **پشتیبانی نمی‌کند**.
  - مدل `kling-video-o1` از استوری‌بورد **پشتیبانی نمی‌کند**.
- **محدودیت‌های استوری‌بورد:**
  - وقتی استوری‌بورد فعال است، فیلد `prompt` نادیده گرفته می‌شود.
  - هر استوری‌بورد به توضیح (۱ تا ۵۱۲ کاراکتر) و مدت‌زمان خاص خود نیاز دارد.
  - مجموع مدت‌زمان همه استوری‌بوردها باید دقیقاً برابر پارامتر `duration` کلی باشد.
- **الزامات توضیح (prompt):**
  - وقتی استوری‌بورد **غیرفعال** است، فیلد `prompt` الزامی است (حداقل ۱ کاراکتر).
  - وقتی استوری‌بورد **فعال** است، فیلد `prompt` می‌تواند خالی باشد (۰ کاراکتر).
