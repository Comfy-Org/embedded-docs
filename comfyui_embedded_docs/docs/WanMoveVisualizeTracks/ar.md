# ترجمة وثيقة WanMoveVisualizeTracks

عقدة WanMoveVisualizeTracks تقوم بتراكب بيانات تتبع الحركة على سلسلة من الصور أو إطارات الفيديو. ترسم تمثيلات بصرية للنقاط المتتبعة، بما في ذلك مسارات حركتها ومواقعها الحالية، مما يجعل بيانات الحركة مرئية وأسهل للتحليل.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|---------|---------------|--------|---------|-------|
| `الصور` | IMAGE | نعم | - | سلسلة الصور المدخلة أو إطارات الفيديو لعرض المسارات عليها. |
| `المسارات` | TRACKS | لا | - | بيانات تتبع الحركة التي تحتوي على مسارات النقاط ومعلومات الرؤية. إذا لم يتم توفيرها، يتم تمرير الصور المدخلة دون تغيير. |
| `دقة الخط` | INT | نعم | 1 - 1024 | عدد الإطارات السابقة المستخدمة عند رسم خط المسار الخلفي لكل مسار (الافتراضي: 24). |
| `حجم الدائرة` | INT | نعم | 1 - 128 | حجم الدائرة المرسومة في الموضع الحالي لكل مسار (الافتراضي: 12). |
| `الشفافية` | FLOAT | نعم | 0.0 - 1.0 | شفافية التراكبات المرسومة للمسارات (الافتراضي: 0.75). |
| `عرض الخط` | INT | نعم | 1 - 128 | عرض الخطوط المستخدمة لرسم مسارات التتبع (الافتراضي: 16). |

**ملاحظة:** إذا كان عدد الصور المدخلة لا يتطابق مع عدد الإطارات في بيانات `tracks` المقدمة، فسيتم تكرار سلسلة الصور لتتناسب مع طول المسار.

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|------------|---------------|---------|
| `IMAGE` | IMAGE | سلسلة الصور مع بيانات تتبع الحركة المرئية كتراكبات. إذا لم يتم توفير `المسارات`، يتم إرجاع الصور المدخلة الأصلية. |
