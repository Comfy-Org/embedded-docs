گره WanMoveVisualizeTracks داده‌های ردیابی حرکت را بر روی یک دنباله از تصاویر یا فریم‌های ویدیویی رسم می‌کند. این گره نمایش‌های بصری از نقاط ردیابی‌شده، شامل مسیر حرکت و موقعیت فعلی آن‌ها را ترسیم کرده و داده‌های حرکت را قابل مشاهده و تحلیل‌پذیر می‌سازد.

## ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `تصاویر` | IMAGE | بله | - | دنباله تصاویر ورودی یا فریم‌های ویدیویی برای نمایش ردیابی‌ها روی آن‌ها. |
| `ترک‌ها` | TRACKS | خیر | - | داده‌های ردیابی حرکت شامل مسیر نقاط و اطلاعات دیدپذیری. در صورت عدم ارائه، تصاویر ورودی بدون تغییر بازگردانده می‌شوند. |
| `وضوح خط` | INT | بله | 1 - 1024 | تعداد فریم‌های قبلی که برای رسم خط دنباله‌کننده مسیر هر ردیاب استفاده می‌شود (پیش‌فرض: 24). |
| `اندازه دایره` | INT | بله | 1 - 128 | اندازه دایره رسم‌شده در موقعیت فعلی هر ردیاب (پیش‌فرض: 12). |
| `شفافیت` | FLOAT | بله | 0.0 - 1.0 | شفافیت لایه‌های ردیابی رسم‌شده (پیش‌فرض: 0.75). |
| `ضخامت خط` | INT | بله | 1 - 128 | عرض خطوط مورد استفاده برای رسم مسیرهای ردیابی (پیش‌فرض: 16). |

**نکته:** اگر تعداد تصاویر ورودی با تعداد فریم‌های داده `tracks` ارائه‌شده مطابقت نداشته باشد، دنباله تصاویر برای هماهنگی با طول ردیابی تکرار می‌شود.
