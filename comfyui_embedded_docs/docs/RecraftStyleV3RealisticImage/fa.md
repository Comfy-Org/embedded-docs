### مرور کلی

این گره یک پیکربندی سبک برای تولید تصاویر واقع‌گرایانه با استفاده از API ریکرفت (Recraft) ایجاد می‌کند. سبک `realistic_image` را انتخاب کرده و به شما امکان می‌دهد یک زیرسبک اختیاری برای تنظیم دقیق خروجی انتخاب کنید.

### ورودی‌ها

| پارامتر | نوع داده | اجباری | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `substyle` | STRING | بله | گزینه‌های متعدد (تعیین‌شده توسط API ریکرفت) | زیرسبک خاصی که باید روی سبک realistic_image اعمال شود. اگر روی "None" تنظیم شود، هیچ زیرسبکی اعمال نخواهد شد. |

### خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-------------|-----------|-------------|
| `recraft_style` | STYLEV3 | یک شیء پیکربندی سبک ریکرفت شامل سبک `realistic_image` و تنظیمات زیرسبک انتخاب‌شده. این خروجی می‌تواند به سایر گره‌های ریکرفت که ورودی سبک می‌پذیرند متصل شود. |
