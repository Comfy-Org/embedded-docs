Этот узел создаёт конфигурацию стиля для генерации реалистичных изображений с использованием API Recraft. Он выбирает стиль `realistic_image` и позволяет задать необязательный подстиль для точной настройки внешнего вида выходного изображения.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `substyle` | STRING | Да | Доступно несколько вариантов (определяется API Recraft) | Конкретный подстиль, применяемый к стилю realistic_image. Если установлено значение "None", подстиль применяться не будет. |

## Выходные параметры

| Имя выходного параметра | Тип данных | Описание |
|-------------|-----------|-------------|
| `recraft_style` | STYLEV3 | Объект конфигурации стиля Recraft, содержащий стиль `realistic_image` и выбранные настройки подстиля. Этот выходной параметр можно подключить к другим узлам Recraft, которые принимают стиль на входе. |
