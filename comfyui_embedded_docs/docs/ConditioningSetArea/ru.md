Этот узел предназначен для изменения информации кондиционирования путем установки определенных областей в контексте кондиционирования. Он позволяет точно манипулировать пространственным расположением элементов кондиционирования, обеспечивая целевые корректировки и улучшения на основе заданных размеров и силы воздействия.

## Входы

| Параметр | Тип данных | Описание |
|-----------|-------------|-------------|
| `CONDITIONING` | CONDITIONING | Данные кондиционирования, подлежащие изменению. Служат основой для применения пространственных корректировок. |
| `width`   | `INT`      | Задает ширину области, устанавливаемой в контексте кондиционирования, определяя горизонтальный масштаб корректировки. |
| `height`  | `INT`      | Определяет высоту устанавливаемой области, влияя на вертикальный охват модификации кондиционирования. |
| `x`       | `INT`      | Горизонтальная начальная точка устанавливаемой области, позиционирующая корректировку в контексте кондиционирования. |
| `y`       | `INT`      | Вертикальная начальная точка для корректировки области, определяющая ее положение в контексте кондиционирования. |
| `strength`| `FLOAT`    | Определяет интенсивность модификации кондиционирования в заданной области, обеспечивая тонкий контроль над воздействием корректировки. |

## Выходы

| Параметр | Тип данных | Описание |
|-----------|-------------|-------------|
| `CONDITIONING` | CONDITIONING | Измененные данные кондиционирования, отражающие заданные настройки и корректировки области. |
