| 參數 | 資料類型 | 說明 |
|-----------|-------------|-------------|
| `CONDITIONING` | CONDITIONING | 待修改的條件資料，作為應用空間調整的基礎。 |
| `寬度`   | `INT`      | 指定在條件設定中設定的區域寬度，影響調整的水平範圍。 |
| `高度`  | `INT`      | 決定設定的區域高度，影響條件修改的垂直範圍。 |

| `Y 座標`       | `INT`      | 區域調整的垂直起始點，確立其在條件設定中的垂直位置。 |
