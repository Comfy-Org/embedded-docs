Узел LatentCrop предназначен для выполнения операций кадрирования (обрезки) латентных представлений изображений. Он позволяет задавать размеры и положение области кадрирования, обеспечивая целенаправленную модификацию латентного пространства.

## Входные параметры

| Параметр | Тип данных | Описание |
|-----------|-------------|-------------|
| `samples` | `LATENT`    | Параметр `samples` представляет латентные представления, подлежащие кадрированию. Он имеет решающее значение для определения данных, над которыми будет выполняться операция обрезки. |
| `width`   | `INT`       | Задает ширину области кадрирования. Напрямую влияет на размеры выходного латентного представления. |
| `height`  | `INT`       | Задает высоту области кадрирования, влияя на размер результирующего обрезанного латентного представления. |
| `x`       | `INT`       | Определяет начальную координату X области кадрирования, влияя на положение обрезки внутри исходного латентного представления. |
| `y`       | `INT`       | Определяет начальную координату Y области кадрирования, задавая положение обрезки внутри исходного латентного представления. |

## Выходные параметры

| Параметр | Тип данных | Описание |
|-----------|-------------|-------------|
| `latent`  | `LATENT`    | Выходные данные представляют собой модифицированное латентное представление с примененным указанным кадрированием. |
