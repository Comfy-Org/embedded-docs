LatentCrop 節點專為對影像的潛在表示執行裁切操作而設計。它允許指定裁切尺寸與位置，從而實現對潛在空間的目標性修改。

## 輸入

| 參數 | 資料類型 | 說明 |
|-----------|-------------|-------------|
| `樣本` | `LATENT`    | `樣本` 參數代表要進行裁切的潛在表示。此參數對於定義將執行裁切操作的資料至關重要。 |
| `寬度`   | `INT`       | 指定裁切區域的寬度。它直接影響輸出潛在表示的尺寸。 |
| `高度`  | `INT`       | 指定裁切區域的高度，影響最終裁切後潛在表示的大小。 |
| `x`       | `INT`       | 決定裁切區域的起始 x 座標，影響裁切在原始潛在表示中的位置。 |
| `y`       | `INT`       | 決定裁切區域的起始 y 座標，設定裁切在原始潛在表示中的位置。 |

## 輸出

| 參數 | 資料類型 | 說明 |
|-----------|-------------|-------------|
| `latent`  | `LATENT`    | 輸出為已套用指定裁切操作的修改後潛在表示。 |
