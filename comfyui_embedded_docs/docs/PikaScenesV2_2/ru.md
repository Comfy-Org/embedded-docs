Вот перевод документации на русский язык:

Узел PikaScenes v2.2 объединяет несколько изображений для создания видео, которое включает объекты из всех исходных изображений. Вы можете загрузить до пяти различных изображений в качестве ингредиентов и сгенерировать высококачественное видео, которое плавно объединит их вместе.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `prompt_text` | STRING | Да | - | Текстовое описание того, что нужно сгенерировать |
| `negative_prompt` | STRING | Да | - | Текстовое описание того, чего следует избегать при генерации |
| `seed` | INT | Да | - | Случайное начальное значение для генерации |
| `resolution` | STRING | Да | - | Разрешение выходного видео |
| `duration` | INT | Да | - | Длительность сгенерированного видео |
| `ingredients_mode` | STRING | Нет | "creative"<br>"precise" | Режим объединения ингредиентов (по умолчанию: "creative") |
| `aspect_ratio` | FLOAT | Нет | 0.4 - 2.5 | Соотношение сторон (ширина / высота) (по умолчанию: 1.778) |
| `image_ingredient_1` | IMAGE | Нет | - | Изображение, которое будет использовано как ингредиент для создания видео |
| `image_ingredient_2` | IMAGE | Нет | - | Изображение, которое будет использовано как ингредиент для создания видео |
| `image_ingredient_3` | IMAGE | Нет | - | Изображение, которое будет использовано как ингредиент для создания видео |
| `image_ingredient_4` | IMAGE | Нет | - | Изображение, которое будет использовано как ингредиент для создания видео |
| `image_ingredient_5` | IMAGE | Нет | - | Изображение, которое будет использовано как ингредиент для создания видео |

**Примечание:** Вы можете предоставить до 5 изображений-ингредиентов, но для генерации видео требуется как минимум одно изображение. Узел будет использовать все предоставленные изображения для создания итоговой видеокомпозиции.

## Выходные данные

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `output` | VIDEO | Сгенерированное видео, объединяющее все входные изображения |
