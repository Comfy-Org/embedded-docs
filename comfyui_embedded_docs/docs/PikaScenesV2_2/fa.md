گره PikaScenes v2.2 با ترکیب چندین تصویر، ویدیویی می‌سازد که اشیاء موجود در تمام تصاویر ورودی را در خود دارد. می‌توانید تا پنج تصویر مختلف را به عنوان مواد اولیه بارگذاری کرده و ویدیویی با کیفیت بالا تولید کنید که آن‌ها را به‌طور یکپارچه ترکیب می‌کند.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|---------|----------|--------|--------|---------|
| `prompt_text` | STRING | بله | - | توضیح متنی آنچه باید تولید شود |
| `negative_prompt` | STRING | بله | - | توضیح متنی آنچه باید در تولید اجتناب شود |
| `seed` | INT | بله | - | مقدار دانه تصادفی برای تولید |
| `resolution` | STRING | بله | - | وضوح خروجی ویدیو |
| `duration` | INT | بله | - | مدت زمان ویدیوی تولید شده |
| `ingredients_mode` | STRING | خیر | "creative"<br>"precise" | حالت ترکیب مواد اولیه (پیش‌فرض: "creative") |
| `aspect_ratio` | FLOAT | خیر | 0.4 - 2.5 | نسبت ابعاد (عرض / ارتفاع) (پیش‌فرض: 1.778) |
| `image_ingredient_1` | IMAGE | خیر | - | تصویری که به عنوان ماده اولیه برای ساخت ویدیو استفاده می‌شود |
| `image_ingredient_2` | IMAGE | خیر | - | تصویری که به عنوان ماده اولیه برای ساخت ویدیو استفاده می‌شود |
| `image_ingredient_3` | IMAGE | خیر | - | تصویری که به عنوان ماده اولیه برای ساخت ویدیو استفاده می‌شود |
| `image_ingredient_4` | IMAGE | خیر | - | تصویری که به عنوان ماده اولیه برای ساخت ویدیو استفاده می‌شود |
| `image_ingredient_5` | IMAGE | خیر | - | تصویری که به عنوان ماده اولیه برای ساخت ویدیو استفاده می‌شود |

**توجه:** می‌توانید تا ۵ تصویر ماده اولیه ارائه دهید، اما برای تولید ویدیو حداقل یک تصویر الزامی است. گره از تمام تصاویر ارائه‌شده برای ایجاد ترکیب نهایی ویدیو استفاده می‌کند.
