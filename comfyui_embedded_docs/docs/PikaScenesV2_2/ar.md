## نظرة عامة

تقوم عقدة PikaScenes v2.2 بدمج صور متعددة لإنشاء فيديو يدمج العناصر من جميع الصور المدخلة. يمكنك تحميل ما يصل إلى خمس صور مختلفة كمكونات، وإنشاء فيديو عالي الجودة يمزجها معًا بسلاسة.

## المدخلات

| المعامل | نوع البيانات | إجباري | النطاق | الوصف |
|-----------|-----------|----------|-------|-------------|
| `prompt_text` | STRING | نعم | - | وصف نصي لما سيتم إنشاؤه |
| `negative_prompt` | STRING | نعم | - | وصف نصي لما يجب تجنبه في الإنشاء |
| `seed` | INT | نعم | - | قيمة البذرة العشوائية للإنشاء |
| `resolution` | STRING | نعم | - | دقة الفيديو الناتج |
| `duration` | INT | نعم | - | مدة الفيديو المُنشأ |
| `ingredients_mode` | STRING | لا | "إبداعي"<br>"دقيق" | وضع دمج المكونات (الافتراضي: "إبداعي") |
| `aspect_ratio` | FLOAT | لا | 0.4 - 2.5 | نسبة العرض إلى الارتفاع (العرض / الارتفاع) (الافتراضي: 1.778) |
| `image_ingredient_1` | IMAGE | لا | - | صورة ستُستخدم كمكون لإنشاء فيديو |
| `image_ingredient_2` | IMAGE | لا | - | صورة ستُستخدم كمكون لإنشاء فيديو |
| `image_ingredient_3` | IMAGE | لا | - | صورة ستُستخدم كمكون لإنشاء فيديو |
| `image_ingredient_4` | IMAGE | لا | - | صورة ستُستخدم كمكون لإنشاء فيديو |
| `image_ingredient_5` | IMAGE | لا | - | صورة ستُستخدم كمكون لإنشاء فيديو |

**ملاحظة:** يمكنك تقديم ما يصل إلى 5 مكونات صور، ولكن يلزم وجود صورة واحدة على الأقل لإنشاء فيديو. ستستخدم العقدة جميع الصور المقدمة لإنشاء الفيديو النهائي.
