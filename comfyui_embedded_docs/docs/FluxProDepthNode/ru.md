Этот узел генерирует изображения, используя карту глубины в качестве направляющей. Он принимает управляющее изображение и текстовый запрос, после чего создаёт новое изображение, которое соответствует как информации о глубине из управляющего изображения, так и описанию из запроса. Узел подключается к внешнему API для выполнения процесса генерации изображений.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `control_image` | IMAGE | Да | - | Управляющее изображение глубины, используемое для направления генерации изображения |
| `prompt` | STRING | Нет | - | Запрос для генерации изображения (по умолчанию: пустая строка) |
| `prompt_upsampling` | BOOLEAN | Нет | - | Выполнять ли апсемплинг запроса. Если активно, автоматически изменяет запрос для более креативной генерации, но результаты становятся недетерминированными (одинаковый seed не даст точно такого же результата). (по умолчанию: False) |
| `skip_preprocessing` | BOOLEAN | Нет | - | Пропускать ли предварительную обработку; установите True, если `control_image` уже содержит карту глубины, и False, если это исходное изображение. (по умолчанию: False) |
| `guidance` | FLOAT | Нет | 1-100 | Сила направляющего воздействия для процесса генерации изображения (по умолчанию: 15) |
| `steps` | INT | Нет | 15-50 | Количество шагов для процесса генерации изображения (по умолчанию: 50) |
| `seed` | INT | Нет | 0-18446744073709551615 | Случайный seed, используемый для создания шума. (по умолчанию: 0) |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `output_image` | IMAGE | Сгенерированное изображение на основе управляющего изображения глубины и запроса |
