این گره با استفاده از یک تصویر کنترل عمق به عنوان راهنما، تصاویر را تولید می‌کند. این گره یک تصویر کنترل و یک متن راهنما دریافت کرده و سپس تصویر جدیدی می‌سازد که هم از اطلاعات عمق تصویر کنترل و هم از توضیحات موجود در متن راهنما پیروی می‌کند. این گره برای انجام فرآیند تولید تصویر به یک API خارجی متصل می‌شود.

## ورودی‌ها

| پارامتر | نوع داده | اجباری | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `control_image` | IMAGE | بله | - | تصویر کنترل عمق که برای هدایت تولید تصویر استفاده می‌شود |
| `prompt` | STRING | خیر | - | متن راهنما برای تولید تصویر (پیش‌فرض: رشته خالی) |
| `prompt_upsampling` | BOOLEAN | خیر | - | آیا بر روی متن راهنما، بالاسری (upsampling) انجام شود یا خیر. در صورت فعال بودن، به‌طور خودکار متن راهنما را برای تولید خلاقانه‌تر تغییر می‌دهد، اما نتایج غیرقطعی هستند (همان seed نتیجه دقیقاً یکسانی تولید نخواهد کرد). (پیش‌فرض: False) |
| `skip_preprocessing` | BOOLEAN | خیر | - | آیا از پیش‌پردازش صرف‌نظر شود یا خیر؛ اگر `control_image` از قبل عمق‌دار (depth-ified) است، روی True تنظیم کنید، اگر یک تصویر خام است، روی False تنظیم کنید. (پیش‌فرض: False) |
| `guidance` | FLOAT | خیر | 1-100 | قدرت هدایت برای فرآیند تولید تصویر (پیش‌فرض: 15) |
| `steps` | INT | خیر | 15-50 | تعداد مراحل برای فرآیند تولید تصویر (پیش‌فرض: 50) |
| `seed` | INT | خیر | 0-18446744073709551615 | دانه تصادفی (seed) که برای ایجاد نویز استفاده می‌شود. (پیش‌فرض: 0) |
