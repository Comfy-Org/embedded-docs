### بررسی اجمالی

این گره یک هوک (hook) ایجاد می‌کند که از مدل LoRA (Low-Rank Adaptation) برای تغییر تنها بخش مدل یک شبکه عصبی استفاده می‌کند. یک فایل checkpoint بارگذاری کرده و آن را با شدت مشخصی روی مدل اعمال می‌کند، در حالی که بخش CLIP را بدون تغییر باقی می‌گذارد. این یک گره آزمایشی است که قابلیت‌های کلاس پایه CreateHookModelAsLora را گسترش می‌دهد.

### ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `ckpt_name` | STRING | بله | گزینه‌های متعدد موجود | فایل checkpoint که به عنوان مدل LoRA بارگذاری می‌شود. گزینه‌های موجود به محتویات پوشه checkpoints بستگی دارد. |
| `strength_model` | FLOAT | بله | 20.0- تا 20.0 | ضریب شدت برای اعمال LoRA روی بخش مدل (پیش‌فرض: 1.0) |
| `prev_hooks` | HOOKS | خیر | - | هوک‌های قبلی اختیاری برای زنجیره‌سازی با این هوک |

### خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-------------|-----------|-------------|
| `hooks` | HOOKS | گروه هوک ایجاد شده شامل تغییرات مدل LoRA |
