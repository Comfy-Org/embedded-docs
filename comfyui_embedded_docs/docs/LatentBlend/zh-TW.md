LatentBlend 節點透過指定的混合因子將兩個潛在樣本進行融合。它接收兩個潛在輸入，並建立一個新的輸出，其中第一個樣本以混合因子加權，第二個樣本則以反向加權。如果輸入樣本具有不同的形狀，系統會自動調整第二個樣本的尺寸以符合第一個樣本的維度。

| `samples2` | LATENT | 是 | - | 要進行混合的第二個潛在樣本 |
