Вот перевод документации на русский язык:

Узел LatentBlend объединяет два латентных сэмпла путем их смешивания с использованием заданного коэффициента смешивания. Он принимает два латентных входа и создает новый выход, где первый сэмпл взвешивается по коэффициенту смешивания, а второй — по обратному значению. Если входные сэмплы имеют разную форму, второй сэмпл автоматически изменяет размер, чтобы соответствовать размерам первого сэмпла.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `samples1` | LATENT | Да | - | Первый латентный сэмпл для смешивания |
| `samples2` | LATENT | Да | - | Второй латентный сэмпл для смешивания |
| `blend_factor` | FLOAT | Да | от 0 до 1 | Управляет соотношением смешивания между двумя сэмплами (по умолчанию: 0.5) |

**Примечание:** Если `samples1` и `samples2` имеют разную форму, `samples2` будет автоматически изменен в размере, чтобы соответствовать размерам `samples1`, с использованием бикубической интерполяции и центрированного кадрирования.

## Выходы

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `latent` | LATENT | Смешанный латентный сэмпл, объединяющий оба входных сэмпла |
