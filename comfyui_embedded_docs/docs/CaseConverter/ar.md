# ترجمة وثيقة عقدة Case Converter

## نظرة عامة

تقوم عقدة محول حالة الأحرف (Case Converter) بتحويل النصوص النصية إلى تنسيقات مختلفة لحالة الأحرف. تأخذ العقدة سلسلة نصية مدخلة وتحولها بناءً على الوضع المحدد، مما ينتج سلسلة نصية مخرجة مع تطبيق تنسيق الحالة المحدد. تدعم العقدة أربعة خيارات مختلفة لتحويل حالة الأحرف لتعديل طريقة كتابة النص بأحرف كبيرة أو صغيرة.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|---------|---------------|--------|---------|-------|
| `سلسلة نصية` | STRING | نعم | - | السلسلة النصية المراد تحويلها إلى تنسيق حالة أحرف مختلف |
| `الوضع` | STRING | نعم | `"UPPERCASE"`<br>`"lowercase"`<br>`"Capitalize"`<br>`"Title Case"` | وضع تحويل حالة الأحرف المطلوب تطبيقه (الافتراضي: `"UPPERCASE"`) |

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|-------------|---------------|---------|
| `output` | STRING | السلسلة النصية المدخلة بعد تحويلها إلى تنسيق الحالة المحدد |
