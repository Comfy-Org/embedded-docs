Узел Case Converter преобразует текстовые строки в различные форматы регистра букв. Он принимает входную строку и преобразует её в соответствии с выбранным режимом, возвращая выходную строку с применённым указанным форматированием регистра. Узел поддерживает четыре различных варианта преобразования регистра для изменения написания вашего текста.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `string` | STRING | Да | - | Текстовая строка для преобразования в другой формат регистра |
| `mode` | STRING | Да | `"UPPERCASE"`<br>`"lowercase"`<br>`"Capitalize"`<br>`"Title Case"` | Режим преобразования регистра (по умолчанию: `"UPPERCASE"`) |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `output` | STRING | Входная строка, преобразованная в указанный формат регистра |
