### گره تبدیل تصویر به ویدیوی Pika

گره Pika Image to Video، یک تصویر و متن راهنما را به نسخه ۲.۲ از API سرویس Pika ارسال می‌کند تا یک ویدیو تولید کند. این گره تصویر ورودی شما را بر اساس توضیحات و تنظیمات ارائه‌شده به فرمت ویدیو تبدیل می‌کند. ارتباط با API را مدیریت کرده و ویدیوی تولیدشده را به عنوان خروجی بازمی‌گرداند.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|---------|----------|--------|--------|---------|
| `image` | IMAGE | بله | - | تصویری که باید به ویدیو تبدیل شود |
| `prompt_text` | STRING | بله | - | توضیحات متنی که تولید ویدیو را هدایت می‌کند |
| `negative_prompt` | STRING | بله | - | متنی که مشخص می‌کند چه مواردی در ویدیو اجتناب شود |
| `seed` | INT | بله | - | مقدار تصادفی seed برای نتایج قابل تکرار |
| `resolution` | STRING | بله | - | تنظیمات رزولوشن ویدیوی خروجی |
| `duration` | INT | بله | - | طول ویدیوی تولیدشده بر حسب ثانیه |

## خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-----------|----------|---------|
| `output` | VIDEO | فایل ویدیوی تولیدشده |
