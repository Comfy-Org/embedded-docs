Вот перевод документации на русский язык:

Узел Pika Image to Video отправляет изображение и текстовый запрос в Pika API версии 2.2 для генерации видео. Он преобразует входное изображение в видеоформат на основе предоставленного описания и настроек. Узел обрабатывает взаимодействие с API и возвращает сгенерированное видео в качестве выходных данных.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `image` | IMAGE | Да | - | Изображение для преобразования в видео |
| `prompt_text` | STRING | Да | - | Текстовое описание, направляющее генерацию видео |
| `negative_prompt` | STRING | Да | - | Текст, описывающий, чего следует избегать в видео |
| `seed` | INT | Да | - | Значение случайного зерна для воспроизводимых результатов |
| `resolution` | STRING | Да | - | Настройка разрешения выходного видео |
| `duration` | INT | Да | - | Длительность сгенерированного видео в секундах |

## Выходные данные

| Имя выходных данных | Тип данных | Описание |
|-------------|-----------|-------------|
| `output` | VIDEO | Сгенерированный видеофайл |
