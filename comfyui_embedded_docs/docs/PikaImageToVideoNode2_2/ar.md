هذه الوثيقة تم إنشاؤها بواسطة الذكاء الاصطناعي. إذا وجدت أي أخطاء أو لديك اقتراحات للتحسين، فلا تتردد في المساهمة! [تحرير على GitHub](https://github.com/Comfy-Org/embedded-docs/blob/main/comfyui_embedded_docs/docs/PikaImageToVideoNode2_2/en.md)

تقوم عقدة "تحويل الصورة إلى فيديو من Pika" بإرسال صورة ونص موجه إلى إصدار 2.2 من واجهة برمجة تطبيقات Pika لإنشاء فيديو. تقوم هذه العقدة بتحويل الصورة المدخلة إلى تنسيق فيديو بناءً على الوصف والإعدادات المقدمة. كما تتولى العقدة عملية الاتصال بواجهة API وتُعيد الفيديو المُنشأ كمخرجات.

## المدخلات

| المعامل | نوع البيانات | إجباري | النطاق | الوصف |
|-----------|-----------|----------|-------|-------------|
| `image` | IMAGE | نعم | - | الصورة المراد تحويلها إلى فيديو |
| `prompt_text` | STRING | نعم | - | النص الوصفي الذي يوجه عملية إنشاء الفيديو |
| `negative_prompt` | STRING | نعم | - | النص الذي يصف ما يجب تجنبه في الفيديو |
| `seed` | INT | نعم | - | قيمة البذرة العشوائية للحصول على نتائج قابلة للتكرار |
| `resolution` | STRING | نعم | - | إعداد دقة الفيديو الناتج |
| `duration` | INT | نعم | - | طول الفيديو المُنشأ بالثواني |

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|-------------|-----------|-------------|
| `output` | VIDEO | ملف الفيديو المُنشأ |
