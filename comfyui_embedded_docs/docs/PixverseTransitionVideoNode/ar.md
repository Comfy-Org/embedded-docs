أنت خبير في الترجمة التقنية متخصص في توثيق عُقد ComfyUI من الإنجليزية إلى العربية.

## قواعد الترجمة

1. **المحتوى الذي يجب عدم ترجمته:**
   - أسماء المعاملات بين علامات الاقتباس الخلفية: `image`, `seed`, `model`
   - أنواع البيانات بالأحرف الكبيرة: IMAGE, STRING, INT, FLOAT, MODEL, CONDITIONING, إلخ
   - القيم في عمود Range: الأرقام، "auto"، أسماء الخيارات
   - الكود، مسارات الملفات

2. **المحتوى الذي يجب ترجمته:**
   - عناوين الأقسام: ## نظرة عامة, ## المدخلات, ## المخرجات
   - جميع النصوص الوصفية والتوضيحية
   - أوصاف المعاملات

3. **جودة الترجمة:**
   - استخدام اللغة العربية الفصحى المعاصرة
   - الحفاظ على نبرة احترافية ولكن سهلة الفهم
   - ضمان الدقة التقنية
   - استخدام المصطلحات التقنية العربية القياسية

4. **التنسيق:**
   - الحفاظ على جميع تنسيقات Markdown
   - الحفاظ على بنية الجداول
   - عدم إضافة أي ملاحظة أو رابط في بداية الوثيقة (سيتم إضافتها تلقائيًا)

ينشئ فيديو انتقالي بين صورتين مدخلتين باستخدام واجهة برمجة تطبيقات PixVerse. يمكنك توفير صورة بداية وصورة نهاية، وتقوم العقدة بإنشاء فيديو سلس ينتقل من صورة إلى أخرى، مسترشدةً بنصك التوجيهي والإعدادات التي اخترتها.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|-----------|-----------|----------|-------|-------------|
| `first_frame` | IMAGE | نعم | - | صورة البداية لفيديو الانتقال |
| `last_frame` | IMAGE | نعم | - | صورة النهاية لفيديو الانتقال |
| `prompt` | STRING | نعم | - | النص التوجيهي لتوليد الفيديو (الافتراضي: سلسلة فارغة) |
| `quality` | COMBO | نعم | `"360p"`<br>`"540p"`<br>`"720p"`<br>`"1080p"` | إعداد جودة الفيديو (الافتراضي: `"540p"`) |
| `duration_seconds` | COMBO | نعم | `5`<br>`8` | مدة الفيديو بالثواني |
| `motion_mode` | COMBO | نعم | `"normal"`<br>`"fast"` | نمط الحركة للانتقال (الافتراضي: `"normal"`) |
| `seed` | INT | نعم | 0 إلى 2147483647 | البذرة لتوليد الفيديو (الافتراضي: 0) |
| `negative_prompt` | STRING | لا | - | وصف نصي اختياري للعناصر غير المرغوب فيها في الصورة (الافتراضي: سلسلة فارغة) |

**ملاحظة حول قيود المعاملات:** عند استخدام جودة 1080p، يتم ضبط نمط الحركة تلقائيًا على `"normal"` وتكون المدة محددة بـ 5 ثوانٍ. بالنسبة لأي مدة غير 5 ثوانٍ، يتم أيضًا ضبط نمط الحركة تلقائيًا على `"normal"`.
