### نمای کلی

یک ویدیوی انتقال بین دو تصویر ورودی با استفاده از API PixVerse تولید می‌کند. شما یک تصویر شروع و یک تصویر پایان ارائه می‌دهید و این گره یک ویدیوی روان ایجاد می‌کند که از یکی به دیگری انتقال می‌یابد و توسط متن راهنما و تنظیمات انتخابی شما هدایت می‌شود.

## ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `فریم اول` | IMAGE | بله | - | تصویر شروع برای ویدیوی انتقال |
| `فریم آخر` | IMAGE | بله | - | تصویر پایان برای ویدیوی انتقال |
| `پرامپت` | STRING | بله | - | راهنمای تولید ویدیو (پیش‌فرض: رشته خالی) |
| `کیفیت` | COMBO | بله | `"360p"`<br>`"540p"`<br>`"720p"`<br>`"1080p"` | تنظیم کیفیت ویدیو (پیش‌فرض: `"540p"`) |
| `مدت زمان (ثانیه)` | COMBO | بله | `5`<br>`8` | مدت زمان ویدیو بر حسب ثانیه |
| `حالت حرکت` | COMBO | بله | `"normal"`<br>`"fast"` | سبک حرکت برای انتقال (پیش‌فرض: `"normal"`) |
| `seed` | INT | بله | 0 تا 2147483647 | دانه (seed) برای تولید ویدیو (پیش‌فرض: 0) |
| `پرامپت منفی` | STRING | خیر | - | توضیح متنی اختیاری از عناصر نامطلوب در تصویر (پیش‌فرض: رشته خالی) |

**نکته در مورد محدودیت‌های پارامتر:** هنگام استفاده از کیفیت 1080p، حالت حرکت به طور خودکار روی `"normal"` تنظیم می‌شود و مدت زمان به 5 ثانیه محدود می‌گردد. برای هر مدت زمانی غیر از 5 ثانیه، حالت حرکت نیز به طور خودکار روی `"normal"` تنظیم می‌شود.
