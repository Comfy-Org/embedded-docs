Вот перевод документации узла OpenAIChatConfig на русский язык:

Узел OpenAIChatConfig позволяет задать дополнительные параметры конфигурации для узла OpenAI Chat. Он предоставляет расширенные настройки, управляющие тем, как модель генерирует ответы, включая поведение при усечении, ограничения длины выходных данных и пользовательские инструкции.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `усечение` | COMBO | Да | `"auto"`<br>`"disabled"` | Стратегия усечения для ответа модели. auto: Если контекст этого и предыдущих ответов превышает размер контекстного окна модели, модель усечет ответ, чтобы он поместился в окно, отбрасывая элементы в середине диалога. disabled: Если ответ модели превысит размер контекстного окна, запрос завершится ошибкой 400 (по умолчанию: "auto") |
| `макс_выходные_токены` | INT | Нет | от 16 до 16384 | Верхняя граница количества токенов, которые могут быть сгенерированы для ответа, включая видимые выходные токены (по умолчанию: 4096) |
| `инструкции` | STRING | Нет | - | Инструкции для модели о том, как генерировать ответ (поддерживается многострочный ввод) |

## Выходные параметры

| Имя выходного параметра | Тип данных | Описание |
|-------------------------|------------|----------|
| `OPENAI_CHAT_CONFIG` | OPENAI_CHAT_CONFIG | Объект конфигурации, содержащий указанные настройки для использования с узлами OpenAI Chat |
