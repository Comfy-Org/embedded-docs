### مرور کلی

گره OpenAIChatConfig به شما امکان می‌دهد گزینه‌های پیکربندی اضافی را برای گره چت OpenAI تنظیم کنید. این گره تنظیمات پیشرفته‌ای را ارائه می‌دهد که نحوه تولید پاسخ توسط مدل را کنترل می‌کند، از جمله رفتار برش (truncation)، محدودیت طول خروجی و دستورالعمل‌های سفارشی.

## ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|---------|----------|--------|--------|----------|
| `truncation` | COMBO | بله | `"auto"`<br>`"disabled"` | استراتژی برش مورد استفاده برای پاسخ مدل. auto: اگر زمینه این پاسخ و پاسخ‌های قبلی از اندازه پنجره زمینه مدل فراتر رود، مدل با حذف آیتم‌های ورودی در وسط مکالمه، پاسخ را برای تطبیق با پنجره زمینه کوتاه می‌کند. disabled: اگر پاسخ مدل از اندازه پنجره زمینه فراتر رود، درخواست با خطای ۴۰۰ ناموفق خواهد بود (پیش‌فرض: "auto") |
| `max_output_tokens` | INT | خیر | ۱۶ تا ۱۶۳۸۴ | کران بالایی برای تعداد توکن‌هایی که می‌توانند برای یک پاسخ تولید شوند، شامل توکن‌های خروجی قابل مشاهده (پیش‌فرض: ۴۰۹۶) |
| `instructions` | STRING | خیر | - | دستورالعمل‌هایی برای مدل در مورد نحوه تولید پاسخ (ورودی چندخطی پشتیبانی می‌شود) |
