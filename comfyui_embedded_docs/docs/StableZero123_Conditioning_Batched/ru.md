Вот перевод документации на русский язык:

Узел StableZero123_Conditioning_Batched обрабатывает входное изображение и генерирует данные кондиционирования для создания 3D-моделей. Он кодирует изображение с помощью моделей CLIP vision и VAE, затем создаёт вложения камеры на основе углов возвышения и азимута для формирования положительного и отрицательного кондиционирования вместе с латентными представлениями для пакетной обработки.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `clip_vision` | CLIP_VISION | Да | - | Модель CLIP vision, используемая для кодирования входного изображения |
| `init_image` | IMAGE | Да | - | Исходное входное изображение для обработки и кодирования |
| `vae` | VAE | Да | - | Модель VAE, используемая для кодирования пикселей изображения в латентное пространство |
| `width` | INT | Нет | 16 до MAX_RESOLUTION | Ширина выходного изображения (по умолчанию: 256, должна быть кратна 8) |
| `height` | INT | Нет | 16 до MAX_RESOLUTION | Высота выходного изображения (по умолчанию: 256, должна быть кратна 8) |
| `batch_size` | INT | Нет | 1 до 4096 | Количество образцов кондиционирования для генерации в пакете (по умолчанию: 1) |
| `elevation` | FLOAT | Нет | -180.0 до 180.0 | Начальный угол возвышения камеры в градусах (по умолчанию: 0.0) |
| `azimuth` | FLOAT | Нет | -180.0 до 180.0 | Начальный азимутальный угол камеры в градусах (по умолчанию: 0.0) |
| `elevation_batch_increment` | FLOAT | Нет | -180.0 до 180.0 | Величина приращения возвышения для каждого элемента пакета (по умолчанию: 0.0) |
| `azimuth_batch_increment` | FLOAT | Нет | -180.0 до 180.0 | Величина приращения азимута для каждого элемента пакета (по умолчанию: 0.0) |

**Примечание:** Параметры `width` и `height` должны быть кратны 8, так как узел внутренне делит эти размеры на 8 для генерации латентного пространства.

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `positive` | CONDITIONING | Положительные данные кондиционирования, содержащие вложения изображения и параметры камеры |
| `negative` | CONDITIONING | Отрицательные данные кондиционирования с нулевыми инициализированными вложениями |
| `latent` | LATENT | Латентное представление обработанного изображения с информацией о пакетной индексации |
