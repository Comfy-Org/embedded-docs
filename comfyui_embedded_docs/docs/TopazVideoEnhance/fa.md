### نمای کلی

گره Topaz Video Enhance از یک API خارجی برای بهبود کیفیت ویدیو استفاده می‌کند. این گره می‌تواند وضوح ویدیو را افزایش دهد، نرخ فریم را از طریق درون‌یابی افزایش دهد و فشرده‌سازی اعمال کند. این گره یک ویدیوی ورودی MP4 را پردازش کرده و نسخه بهبودیافته آن را بر اساس تنظیمات انتخاب شده بازمی‌گرداند.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `ویدیو` | VIDEO | بله | - | فایل ویدیوی ورودی که باید بهبود یابد. |
| `فعال‌سازی بزرگ‌نمایی` | BOOLEAN | بله | - | ویژگی افزایش وضوح ویدیو را فعال یا غیرفعال می‌کند (پیش‌فرض: True). |
| `مدل بزرگ‌نمایی` | COMBO | بله | `"Starlight (Astra) Fast"`<br>`"Starlight (Astra) Creative"`<br>`"Starlight Precise 2.5"` | مدل هوش مصنوعی مورد استفاده برای افزایش وضوح ویدیو. |
| `وضوح بزرگ‌نمایی` | COMBO | بله | `"FullHD (1080p)"`<br>`"4K (2160p)"` | وضوح هدف برای ویدیوی افزایش‌یافته. |
| `سطح خلاقیت` | COMBO | خیر | `"low"`<br>`"middle"`<br>`"high"` | سطح خلاقیت (فقط برای Starlight (Astra) Creative اعمال می‌شود). (پیش‌فرض: "low") |
| `فعال‌سازی درون‌یابی` | BOOLEAN | خیر | - | ویژگی درون‌یابی فریم را فعال یا غیرفعال می‌کند (پیش‌فرض: False). |
| `مدل درون‌یابی` | COMBO | خیر | `"apo-8"` | مدل مورد استفاده برای درون‌یابی فریم (پیش‌فرض: "apo-8"). |
| `ضریب حرکت آهسته` | INT | خیر | 1 تا 16 | ضریب حرکت آهسته اعمال‌شده بر ویدیوی ورودی. به‌عنوان مثال، مقدار 2 خروجی را دو برابر کندتر کرده و مدت زمان را دو برابر می‌کند. (پیش‌فرض: 1) |
| `نرخ فریم خروجی` | INT | خیر | 15 تا 240 | نرخ فریم خروجی. (پیش‌فرض: 60) |
| `حذف فریم‌های تکراری` | BOOLEAN | خیر | - | ورودی را برای فریم‌های تکراری تحلیل کرده و آن‌ها را حذف می‌کند. (پیش‌فرض: False) |
| `حساسیت تشخیص فریم تکراری` | FLOAT | خیر | 0.001 تا 0.1 | حساسیت تشخیص فریم‌های تکراری. (پیش‌فرض: 0.01) |
| `سطح فشرده‌سازی پویا` | COMBO | خیر | `"Low"`<br>`"Mid"`<br>`"High"` | سطح CQP. (پیش‌فرض: "Low") |

**نکته:** حداقل یک ویژگی بهبود باید فعال باشد. اگر هر دو `upscaler_enabled` و `interpolation_enabled` روی `False` تنظیم شوند، گره خطا ایجاد می‌کند. ویدیوی ورودی باید با فرمت MP4 باشد.
