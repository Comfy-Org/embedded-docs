Вот перевод документации на русский язык:

Этот узел Topaz Video Enhance использует внешний API для улучшения качества видео. Он может повышать разрешение видео, увеличивать частоту кадров с помощью интерполяции и применять сжатие. Узел обрабатывает входной MP4-файл и возвращает улучшенную версию на основе выбранных настроек.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `video` | VIDEO | Да | - | Входной видеофайл для улучшения. |
| `upscaler_enabled` | BOOLEAN | Да | - | Включает или отключает функцию повышения разрешения видео (по умолчанию: True). |
| `upscaler_model` | COMBO | Да | `"Starlight (Astra) Fast"`<br>`"Starlight (Astra) Creative"`<br>`"Starlight Precise 2.5"` | Модель ИИ, используемая для повышения разрешения видео. |
| `upscaler_resolution` | COMBO | Да | `"FullHD (1080p)"`<br>`"4K (2160p)"` | Целевое разрешение для улучшенного видео. |
| `upscaler_creativity` | COMBO | Нет | `"low"`<br>`"middle"`<br>`"high"` | Уровень креативности (применяется только к Starlight (Astra) Creative). (по умолчанию: "low") |
| `interpolation_enabled` | BOOLEAN | Нет | - | Включает или отключает функцию интерполяции кадров (по умолчанию: False). |
| `interpolation_model` | COMBO | Нет | `"apo-8"` | Модель, используемая для интерполяции кадров (по умолчанию: "apo-8"). |
| `interpolation_slowmo` | INT | Нет | от 1 до 16 | Коэффициент замедления, применяемый к входному видео. Например, значение 2 делает выходное видео в два раза медленнее и удваивает его длительность. (по умолчанию: 1) |
| `interpolation_frame_rate` | INT | Нет | от 15 до 240 | Выходная частота кадров. (по умолчанию: 60) |
| `interpolation_duplicate` | BOOLEAN | Нет | - | Анализировать входное видео на наличие дублирующихся кадров и удалять их. (по умолчанию: False) |
| `interpolation_duplicate_threshold` | FLOAT | Нет | от 0.001 до 0.1 | Чувствительность обнаружения дублирующихся кадров. (по умолчанию: 0.01) |
| `dynamic_compression_level` | COMBO | Нет | `"Low"`<br>`"Mid"`<br>`"High"` | Уровень CQP. (по умолчанию: "Low") |

**Примечание:** Должна быть включена как минимум одна функция улучшения. Узел выдаст ошибку, если оба параметра `upscaler_enabled` и `interpolation_enabled` установлены в `False`. Входное видео должно быть в формате MP4.

| Имя выходного параметра | Тип данных | Описание |
|-------------|-----------|-------------|
| `video` | VIDEO | Улучшенный выходной видеофайл. |
