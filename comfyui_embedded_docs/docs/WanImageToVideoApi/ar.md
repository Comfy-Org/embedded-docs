بالتأكيد، إليك الترجمة العربية للوثيقة التقنية الخاصة بعقدة ComfyUI مع الالتزام التام بقواعد الترجمة المحددة:

## نظرة عامة

تقوم عقدة "تحويل الصورة إلى فيديو من Wan" بإنشاء فيديو من صورة إدخال واحدة ونص وصفي. تستخدم الصورة المقدمة كإطار أول وتُنشئ سلسلة فيديو بناءً على الوصف، مع خيارات للدقة والمدة والصوت وإعدادات متقدمة أخرى.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|-----------|-----------|----------|-------|-------------|
| `model` | COMBO | نعم | "wan2.5-i2v-preview"<br>"wan2.6-i2v" | النموذج المراد استخدامه (الافتراضي: "wan2.6-i2v") |
| `image` | IMAGE | نعم | - | صورة الإدخال التي تعمل كإطار أول لتوليد الفيديو. مطلوب صورة واحدة بالضبط. |
| `prompt` | STRING | نعم | - | النص الوصفي الذي يصف العناصر والميزات البصرية. يدعم اللغتين الإنجليزية والصينية (الافتراضي: فارغ). |
| `negative_prompt` | STRING | لا | - | النص الوصفي السلبي الذي يصف ما يجب تجنبه (الافتراضي: فارغ). |
| `resolution` | COMBO | لا | "480P"<br>"720P"<br>"1080P" | جودة دقة الفيديو (الافتراضي: "720P"). لا يدعم نموذج Wan 2.6 دقة 480P. |
| `duration` | INT | لا | 5-15 (الخطوة: 5) | مدة الفيديو المُنشأ بالثواني. مدة 15 ثانية مدعومة فقط بواسطة نموذج Wan 2.6 (الافتراضي: 5). |
| `audio` | AUDIO | لا | - | يجب أن يحتوي الصوت على صوت واضح وعالٍ، بدون ضوضاء غريبة أو موسيقى خلفية. عند توفيره، يجب أن تتراوح مدة الصوت بين 3.0 و 29.0 ثانية. |
| `seed` | INT | لا | 0-2147483647 | القيمة الأولية (Seed) المراد استخدامها للتوليد (الافتراضي: 0). |
| `generate_audio` | BOOLEAN | لا | - | إذا لم يتم توفير صوت إدخال، قم بتوليد الصوت تلقائيًا (الافتراضي: False). |
| `prompt_extend` | BOOLEAN | لا | - | ما إذا كان سيتم تحسين النص الوصفي بمساعدة الذكاء الاصطناعي (الافتراضي: True). |
| `watermark` | BOOLEAN | لا | - | ما إذا كان سيتم إضافة علامة مائية منشأة بالذكاء الاصطناعي إلى النتيجة (الافتراضي: False). |
| `shot_type` | COMBO | لا | "single"<br>"multi" | يحدد نوع اللقطة للفيديو المُنشأ، أي ما إذا كان الفيديو لقطة واحدة مستمرة أو لقطات متعددة مع انتقالات. هذا المعامل يؤثر فقط عندما يكون prompt_extend مساوياً لـ True (الافتراضي: "single"). |

**القيود:**

- مطلوب صورة إدخال واحدة بالضبط لتوليد الفيديو.
- لا يدعم نموذج Wan 2.6 (`wan2.6-i2v`) دقة 480P.
- مدة 15 ثانية مدعومة فقط بواسطة نموذج Wan 2.6 (`wan2.6-i2v`).
- عند توفير صوت، يجب أن تتراوح مدته بين 3.0 و 29.0 ثانية.
