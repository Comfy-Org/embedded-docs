Вот перевод документации на русский язык:

Узел Wan Image to Video создает видео на основе одного входного изображения и текстового запроса. Он использует предоставленное изображение в качестве первого кадра и создает видеопоследовательность на основе описания, с возможностью настройки разрешения, длительности, аудио и других расширенных параметров.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `модель` | COMBO | Да | "wan2.5-i2v-preview"<br>"wan2.6-i2v" | Модель для использования (по умолчанию: "wan2.6-i2v") |
| `изображение` | IMAGE | Да | - | Входное изображение, которое служит первым кадром для генерации видео. Требуется ровно одно изображение. |
| `промпт` | STRING | Да | - | Запрос, описывающий элементы и визуальные особенности. Поддерживает английский и китайский языки (по умолчанию: пусто). |
| `негативный_промпт` | STRING | Нет | - | Негативный запрос, описывающий, чего следует избегать (по умолчанию: пусто). |
| `разрешение` | COMBO | Нет | "480P"<br>"720P"<br>"1080P" | Качество разрешения видео (по умолчанию: "720P"). Модель Wan 2.6 не поддерживает 480P. |
| `длительность` | INT | Нет | 5-15 (шаг: 5) | Длительность генерируемого видео в секундах. Длительность 15 секунд поддерживается только моделью Wan 2.6 (по умолчанию: 5). |
| `аудио` | AUDIO | Нет | - | Аудио должно содержать четкий, громкий голос, без посторонних шумов или фоновой музыки. При наличии аудио его длительность должна быть от 3,0 до 29,0 секунд. |
| `сид` | INT | Нет | 0-2147483647 | Сид для генерации (по умолчанию: 0). |
| `сгенерировать_аудио` | BOOLEAN | Нет | - | Если аудиовход не предоставлен, генерировать аудио автоматически (по умолчанию: False). |
| `расширить_промпт` | BOOLEAN | Нет | - | Улучшать ли запрос с помощью ИИ (по умолчанию: True). |
| `водяной_знак` | BOOLEAN | Нет | - | Добавлять ли водяной знак, указывающий на создание ИИ, к результату (по умолчанию: False). |
| `shot_type` | COMBO | Нет | "single"<br>"multi" | Определяет тип кадра для генерируемого видео, то есть, является ли видео одним непрерывным кадром или несколькими кадрами с переходами. Этот параметр действует только когда prompt_extend имеет значение True (по умолчанию: "single"). |

**Ограничения:**

- Для генерации видео требуется ровно одно входное изображение.
- Модель Wan 2.6 (`wan2.6-i2v`) не поддерживает разрешение 480P.
- Длительность 15 секунд поддерживается только моделью Wan 2.6 (`wan2.6-i2v`).
- При наличии аудио его длительность должна быть от 3,0 до 29,0 секунд.

## Выходные данные

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `output` | VIDEO | Сгенерированное видео на основе входного изображения и запроса. |
