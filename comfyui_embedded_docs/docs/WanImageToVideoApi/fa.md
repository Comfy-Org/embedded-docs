گره «Wan Image to Video» از یک تصویر ورودی و یک متن راهنما، ویدئو تولید می‌کند. این گره از تصویر ارائه‌شده به‌عنوان فریم اول استفاده کرده و بر اساس توضیحات داده‌شده، یک دنباله ویدئویی با گزینه‌هایی برای وضوح، مدت‌زمان، صدا و سایر تنظیمات پیشرفته ایجاد می‌کند.

## ورودی‌ها

| پارامتر | نوع داده | اجباری | محدوده | توضیحات |
|---------|----------|--------|--------|---------|
| `مدل` | COMBO | بله | "wan2.5-i2v-preview"<br>"wan2.6-i2v" | مدل مورد استفاده (پیش‌فرض: "wan2.6-i2v") |
| `تصویر` | IMAGE | بله | - | تصویر ورودی که به‌عنوان فریم اول برای تولید ویدئو استفاده می‌شود. دقیقاً یک تصویر مورد نیاز است. |
| `پرامپت` | STRING | بله | - | راهنمایی که عناصر و ویژگی‌های بصری را توصیف می‌کند. از انگلیسی و چینی پشتیبانی می‌کند (پیش‌فرض: خالی). |
| `پرامپت منفی` | STRING | خیر | - | راهنمایی منفی که موارد نامطلوب را توصیف می‌کند (پیش‌فرض: خالی). |
| `وضوح` | COMBO | خیر | "480P"<br>"720P"<br>"1080P" | کیفیت وضوح ویدئو (پیش‌فرض: "720P"). مدل Wan 2.6 از 480P پشتیبانی نمی‌کند. |
| `مدت زمان` | INT | خیر | 5-15 (گام: 5) | مدت‌زمان ویدئوی تولیدشده بر حسب ثانیه. مدت‌زمان ۱۵ ثانیه فقط توسط مدل Wan 2.6 پشتیبانی می‌شود (پیش‌فرض: ۵). |
| `صدا` | AUDIO | خیر | - | صدا باید شامل یک صدای واضح و بلند، بدون نویز اضافی یا موسیقی پس‌زمینه باشد. در صورت ارائه، مدت‌زمان صدا باید بین ۳.۰ تا ۲۹.۰ ثانیه باشد. |
| `بذر` | INT | خیر | 0-2147483647 | دانه (seed) مورد استفاده برای تولید (پیش‌فرض: ۰). |
| `تولید صدا` | BOOLEAN | خیر | - | اگر ورودی صوتی ارائه نشود، به‌صورت خودکار صدا تولید کند (پیش‌فرض: False). |
| `گسترش پرامپت` | BOOLEAN | خیر | - | آیا راهنما با کمک هوش مصنوعی بهبود یابد (پیش‌فرض: True). |
| `واترمارک` | BOOLEAN | خیر | - | آیا یک واترمارک تولیدشده توسط هوش مصنوعی به نتیجه اضافه شود (پیش‌فرض: False). |
| `نوع برداشت` | COMBO | خیر | "single"<br>"multi" | نوع نما را برای ویدئوی تولیدشده مشخص می‌کند، یعنی اینکه ویدئو یک نمای پیوسته است یا چند نما با برش. این پارامتر فقط زمانی اعمال می‌شود که prompt_extend برابر True باشد (پیش‌فرض: "single"). |

**محدودیت‌ها:**

- برای تولید ویدئو دقیقاً یک تصویر ورودی مورد نیاز است.
- مدل Wan 2.6 (`wan2.6-i2v`) از وضوح 480P پشتیبانی نمی‌کند.
- مدت‌زمان ۱۵ ثانیه فقط توسط مدل Wan 2.6 (`wan2.6-i2v`) پشتیبانی می‌شود.
- در صورت ارائه صدا، مدت‌زمان آن باید بین ۳.۰ تا ۲۹.۰ ثانیه باشد.
