Вот перевод документации на русский язык:

Этот узел объединяет основное входное кондиционирование с кондиционированием по умолчанию, используя систему на основе хуков. Он объединяет два источника кондиционирования в единый выходной сигнал, позволяя кондиционированию по умолчанию служить резервным или базовым вариантом, когда основное кондиционирование неполно.

## Входы

| Параметр | Тип данных | Тип входа | По умолчанию | Диапазон | Описание |
|----------|------------|-----------|--------------|----------|----------|
| `cond` | CONDITIONING | Обязательный | - | - | Основное входное кондиционирование для обработки и объединения |
| `cond_DEFAULT` | CONDITIONING | Обязательный | - | - | Данные кондиционирования по умолчанию для объединения с основным кондиционированием |
| `hooks` | HOOKS | Необязательный | - | - | Необязательная конфигурация хуков, управляющая обработкой и объединением данных кондиционирования |

## Выходы

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `CONDITIONING` | CONDITIONING | Объединенные данные кондиционирования, полученные в результате слияния основного кондиционирования и кондиционирования по умолчанию |
