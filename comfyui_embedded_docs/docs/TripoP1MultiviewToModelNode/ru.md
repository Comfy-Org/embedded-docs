## Обзор

Этот узел создает 3D-модель объекта или персонажа на основе 2–4 референсных изображений. Вы предоставляете снимки с разных ракурсов (спереди, слева, сзади, справа), а узел генерирует 3D-сетку в формате GLB. Вид спереди обязателен, остальные три ракурса можно добавлять в любых комбинациях для улучшения результата.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `image` | IMAGE | Да | - | Вид спереди (0°). Обязательный. |
| `image_left` | IMAGE | Нет | - | Вид слева (90°), т.е. левая сторона объекта. |
| `image_back` | IMAGE | Нет | - | Вид сзади (180°). |
| `image_right` | IMAGE | Нет | - | Вид справа (270°), т.е. правая сторона объекта. |
| `output_mode` | COMBO | Да | `"geometry"`<br>`"textured"`<br>`"detailed"` | Режим вывода сгенерированной модели. `"geometry"` создает сырую сетку, `"textured"` добавляет стандартную текстуру, а `"detailed"` создает высокодетализированную текстурированную модель (по умолчанию: `"textured"`). |
| `face_limit` | INT | Нет | -1 до 100000 | Максимальное количество полигонов для выходной сетки. Установите -1 для отсутствия ограничения (по умолчанию: -1). |
| `model_seed` | INT | Нет | 0 до 2147483647 | Сид для воспроизводимой генерации модели. Установите 0 для случайного значения (по умолчанию: 0). |
| `auto_size` | BOOLEAN | Нет | True / False | Автоматически подогнать размер модели под стандартный ограничивающий прямоугольник (по умолчанию: False). |
| `export_uv` | BOOLEAN | Нет | True / False | Экспортировать UV-координаты вместе с моделью (по умолчанию: True). |
| `compress_geometry` | BOOLEAN | Нет | True / False | Сжать геометрические данные для уменьшения размера файла (по умолчанию: False). |

**Примечание:** Необходимо предоставить как минимум 2 изображения: вид спереди (`image`) плюс хотя бы один из остальных ракурсов (`image_left`, `image_back` или `image_right`). Если предоставлено менее 2 изображений, узел выдаст ошибку.

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `model_file` | STRING | Имя файла сгенерированной GLB-модели (только для обратной совместимости). |
| `model_task_id` | MODEL_TASK_ID | Уникальный идентификатор задачи для данного запроса на генерацию модели. |
| `GLB` | FILE3DGLB | Сгенерированная 3D-модель в формате GLB. |
