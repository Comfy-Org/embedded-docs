## نظرة عامة

تقوم هذه العقدة بإنشاء نموذج ثلاثي الأبعاد من 2 إلى 4 صور مرجعية لكائن أو شخصية. يمكنك تقديم صور من زوايا مختلفة (أمامية، يسارية، خلفية، يمينية)، وتقوم العقدة بإنشاء شبكة ثلاثية الأبعاد بتنسيق GLB. المنظر الأمامي مطلوب، ويمكنك إضافة أي مجموعة من المناظر الثلاثة الأخرى اختياريًا للحصول على نتائج أفضل.

## المدخلات

| المعامل | نوع البيانات | مطلوب | النطاق | الوصف |
|-----------|-----------|----------|-------|-------------|
| `image` | IMAGE | نعم | - | المنظر الأمامي (0°). مطلوب. |
| `image_left` | IMAGE | لا | - | المنظر الأيسر (90°)، أي الجانب الأيسر للكائن. |
| `image_back` | IMAGE | لا | - | المنظر الخلفي (180°). |
| `image_right` | IMAGE | لا | - | المنظر الأيمن (270°)، أي الجانب الأيمن للكائن. |
| `output_mode` | COMBO | نعم | `"geometry"`<br>`"textured"`<br>`"detailed"` | وضع الإخراج للنموذج المُنشأ. ينتج `"geometry"` شبكة أولية، ويضيف `"textured"` نسيجًا قياسيًا، وينشئ `"detailed"` نموذجًا عالي التفاصيل مع نسيج (الافتراضي: `"textured"`). |
| `face_limit` | INT | لا | -1 إلى 100000 | الحد الأقصى لعدد الوجوه في شبكة الإخراج. اضبط على -1 لعدم وجود حد (الافتراضي: -1). |
| `model_seed` | INT | لا | 0 إلى 2147483647 | بذرة لتوليد نموذج قابل للتكرار. اضبط على 0 للعشوائية (الافتراضي: 0). |
| `auto_size` | BOOLEAN | لا | True / False | تغيير حجم النموذج تلقائيًا ليتناسب مع صندوق إحاطة قياسي (الافتراضي: False). |
| `export_uv` | BOOLEAN | لا | True / False | تصدير إحداثيات UV مع النموذج (الافتراضي: True). |
| `compress_geometry` | BOOLEAN | لا | True / False | ضغط بيانات الهندسة لتقليل حجم الملف (الافتراضي: False). |

**ملاحظة:** يجب عليك تقديم صورتين على الأقل: المنظر الأمامي (`image`) بالإضافة إلى منظر واحد على الأقل من المناظر الأخرى (`image_left`، `image_back`، أو `image_right`). إذا تم تقديم أقل من صورتين، ستقوم العقدة بإرجاع خطأ.

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|-------------|-----------|-------------|
| `model_file` | STRING | اسم ملف النموذج المُنشأ بتنسيق GLB (للتوافق مع الإصدارات السابقة فقط). |
| `model_task_id` | MODEL_TASK_ID | المعرف الفريد للمهمة لطلب توليد النموذج هذا. |
| `GLB` | FILE3DGLB | النموذج ثلاثي الأبعاد المُنشأ بتنسيق GLB. |
