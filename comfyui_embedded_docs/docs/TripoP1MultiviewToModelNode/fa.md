## مرور کلی

این گره یک مدل سه‌بعدی از ۲ تا ۴ تصویر مرجع از یک شیء یا شخصیت تولید می‌کند. شما تصاویری از زوایای مختلف (جلو، چپ، عقب، راست) ارائه می‌دهید و گره یک مش سه‌بعدی در قالب GLB ایجاد می‌کند. نمای جلو الزامی است و می‌توانید به‌صورت اختیاری هر ترکیبی از سه نمای دیگر را برای نتایج بهتر اضافه کنید.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|---------|----------|--------|--------|---------|
| `image` | IMAGE | بله | - | نمای جلو (۰ درجه). الزامی. |
| `image_left` | IMAGE | خیر | - | نمای چپ (۹۰ درجه)، یعنی سمت چپ سوژه. |
| `image_back` | IMAGE | خیر | - | نمای عقب (۱۸۰ درجه). |
| `image_right` | IMAGE | خیر | - | نمای راست (۲۷۰ درجه)، یعنی سمت راست سوژه. |
| `output_mode` | COMBO | بله | `"geometry"`<br>`"textured"`<br>`"detailed"` | حالت خروجی برای مدل تولیدشده. `"geometry"` یک مش خام، `"textured"` بافت استاندارد اضافه می‌کند و `"detailed"` یک مدل بافت‌دار با جزئیات بالا ایجاد می‌کند (پیش‌فرض: `"textured"`). |
| `face_limit` | INT | خیر | ۱- تا ۱۰۰۰۰۰ | حداکثر تعداد وجه‌های مش خروجی. مقدار ۱- به معنای بدون محدودیت است (پیش‌فرض: ۱-). |
| `model_seed` | INT | خیر | ۰ تا ۲۱۴۷۴۸۳۶۴۷ | دانه (seed) برای تولید مدل قابل تکرار. مقدار ۰ برای تصادفی (پیش‌فرض: ۰). |
| `auto_size` | BOOLEAN | خیر | True / False | اندازه‌گذاری خودکار مدل برای قرارگیری در جعبه مرزی استاندارد (پیش‌فرض: False). |
| `export_uv` | BOOLEAN | خیر | True / False | خروجی مختصات UV به همراه مدل (پیش‌فرض: True). |
| `compress_geometry` | BOOLEAN | خیر | True / False | فشرده‌سازی داده‌های هندسی برای کاهش حجم فایل (پیش‌فرض: False). |

**توجه:** حداقل باید ۲ تصویر ارائه دهید: نمای جلو (`image`) به‌علاوه حداقل یکی از نماهای دیگر (`image_left`، `image_back` یا `image_right`). اگر کمتر از ۲ تصویر ارائه شود، گره خطا خواهد داد.

## خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-----------|----------|---------|
| `model_file` | STRING | نام فایل مدل GLB تولیدشده (فقط برای سازگاری با نسخه‌های قبلی). |
| `model_task_id` | MODEL_TASK_ID | شناسه یکتای وظیفه برای این درخواست تولید مدل. |
| `GLB` | FILE3DGLB | مدل سه‌بعدی تولیدشده در قالب GLB. |
