Вот перевод документации на русский язык:

Узел Reve Image Remix использует API Reve для генерации нового изображения. Он объединяет одно или несколько эталонных изображений с текстовым запросом для создания нового, ремикшированного изображения на основе предоставленного описания.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `reference_images` | IMAGE | Да | от 1 до 6 изображений | Одно или несколько эталонных изображений, используемых в качестве основы для ремикса. Можно добавить от 1 до 6 изображений. |
| `prompt` | STRING | Да | от 1 до 2560 символов | Текстовое описание желаемого изображения. Можно использовать XML-теги `<img>` для ссылки на конкретные изображения по их индексу (например, `<img>0</img>`, `<img>1</img>`). (по умолчанию: пусто) |
| `model` | COMBO | Да | `reve-remix@20250915`<br>`reve-remix-fast@20251030` | Версия модели для ремикса. Каждый вариант модели включает настраиваемые соотношения сторон и масштабирование во время тестирования. |
| `upscale` | COMBO | Нет | `"disabled"`<br>`"enabled"` | Управляет увеличением масштаба сгенерированного изображения. При включении можно выбрать коэффициент увеличения. |
| `remove_background` | BOOLEAN | Нет | `true`<br>`false` | При включении предпринимается попытка удалить фон из сгенерированного изображения. |
| `seed` | INT | Нет | от 0 до 2147483647 | Значение зерна. Изменение этого значения приведет к повторному запуску узла, но результаты остаются недетерминированными независимо от зерна. (по умолчанию: 0) |

**Примечание:** Параметр `model` представляет собой динамическое комбо-поле, которое включает вложенные настройки для `aspect_ratio` (варианты: "auto", "16:9", "9:16", "3:2", "2:3", "4:3", "3:4", "1:1") и `test_time_scaling`. Параметр `upscale` при установке значения "enabled" открывает вложенную настройку `upscale_factor`.

## Выходные данные

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `image` | IMAGE | Новое изображение, сгенерированное процессом ремикса Reve. |
