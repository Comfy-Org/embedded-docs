# ترجمة وثيقة عقدة NormalizeImages

هذه العقدة تقوم بضبط قيم البكسل لصورة الإدخال باستخدام عملية تطبيع رياضية. تطرح قيمة متوسط محددة من كل بكسل ثم تقسم الناتج على انحراف معياري محدد. تُعد هذه خطوة معالجة أولية شائعة لتجهيز بيانات الصورة لنماذج التعلم الآلي الأخرى.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|---------|---------------|--------|---------|-------|
| `image` | IMAGE | نعم | - | صورة الإدخال المراد تطبيعها. |
| `mean` | FLOAT | لا | 0.0 - 1.0 | قيمة المتوسط للتطبيع (القيمة الافتراضية: 0.5). |
| `std` | FLOAT | لا | 0.001 - 1.0 | الانحراف المعياري للتطبيع (القيمة الافتراضية: 0.5). |

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|-------------|---------------|--------|
| `image` | IMAGE | الصورة الناتجة بعد تطبيق عملية التطبيع. |
