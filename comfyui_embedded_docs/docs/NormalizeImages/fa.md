### مرور کلی

این گره با استفاده از یک فرآیند نرمال‌سازی ریاضی، مقادیر پیکسل‌های یک تصویر ورودی را تنظیم می‌کند. این گره یک مقدار میانگین مشخص را از هر پیکسل کم کرده و سپس نتیجه را بر یک انحراف معیار مشخص تقسیم می‌کند. این یک مرحله پیش‌پردازش رایج برای آماده‌سازی داده‌های تصویری جهت استفاده در سایر مدل‌های یادگیری ماشین است.

### ورودی‌ها

| پارامتر | نوع داده | اجباری | محدوده | توضیحات |
|---------|----------|--------|--------|---------|
| `image` | IMAGE | بله | - | تصویر ورودی که باید نرمال‌سازی شود. |
| `mean` | FLOAT | خیر | 0.0 - 1.0 | مقدار میانگین برای نرمال‌سازی (پیش‌فرض: 0.5). |
| `std` | FLOAT | خیر | 0.001 - 1.0 | انحراف معیار برای نرمال‌سازی (پیش‌فرض: 0.5). |
