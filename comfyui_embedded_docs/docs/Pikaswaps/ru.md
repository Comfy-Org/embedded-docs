Вот перевод документации на русский язык:

Узел Pika Swaps заменяет объекты или области в вашем видео на новое изображение. Вы определяете области для замены с помощью маски, и узел бесшовно заменяет указанное содержимое на протяжении всей видеопоследовательности.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `video` | VIDEO | Да | - | Видео, в котором необходимо заменить объект. |
| `image` | IMAGE | Да | - | Изображение, используемое для замены замаскированного объекта в видео. |
| `mask` | MASK | Да | - | Используйте маску для определения областей в видео, подлежащих замене. |
| `prompt_text` | STRING | Да | - | Текстовый запрос, описывающий желаемую замену. |
| `negative_prompt` | STRING | Да | - | Текстовый запрос, описывающий, чего следует избегать при замене. |
| `seed` | INT | Да | от 0 до 4294967295 | Значение случайного зерна для получения воспроизводимых результатов. |

**Примечание:** Для работы этого узла необходимо указать все входные параметры. Параметры `video`, `image` и `mask` работают совместно, определяя операцию замены, где маска указывает, какие области видео будут заменены предоставленным изображением.

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `output` | VIDEO | Обработанное видео с заменённым указанным объектом или областью. |
