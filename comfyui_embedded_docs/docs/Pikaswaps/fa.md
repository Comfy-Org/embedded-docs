گره Pika Swaps اشیاء یا نواحی موجود در ویدیوی شما را با یک تصویر جدید جایگزین می‌کند. شما با استفاده از یک ماسک، نواحی مورد نظر برای جایگزینی را مشخص می‌کنید و این گره به‌طور یکپارچه محتوای مشخص‌شده را در سراسر دنباله ویدیو تعویض می‌کند.

## ورودی‌ها

| پارامتر | نوع داده | اجباری | محدوده | توضیحات |
|---------|----------|--------|--------|---------|
| `video` | VIDEO | بله | - | ویدیویی که قرار است شیء در آن تعویض شود. |
| `image` | IMAGE | بله | - | تصویری که برای جایگزینی شیء ماسک‌شده در ویدیو استفاده می‌شود. |
| `mask` | MASK | بله | - | از ماسک برای تعیین نواحی قابل تعویض در ویدیو استفاده کنید. |
| `prompt_text` | STRING | بله | - | متن راهنما (پرامپت) توصیف‌کننده جایگزینی مورد نظر. |
| `negative_prompt` | STRING | بله | - | متن راهنما (پرامپت) توصیف‌کننده مواردی که باید در جایگزینی اجتناب شود. |
| `seed` | INT | بله | 0 تا 4294967295 | مقدار تصادفی (seed) برای نتایج یکسان و قابل تکرار. |

**توجه:** این گره نیازمند ارائه تمام پارامترهای ورودی است. پارامترهای `video`، `image` و `mask` با هم عمل جایگزینی را تعریف می‌کنند، به‌طوری که ماسک مشخص می‌کند کدام نواحی از ویدیو با تصویر ارائه‌شده جایگزین شوند.
