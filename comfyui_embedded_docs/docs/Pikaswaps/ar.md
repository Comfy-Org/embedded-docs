# ترجمة وثيقة عقدة Pika Swaps

تقوم عقدة Pika Swaps باستبدال الكائنات أو المناطق في الفيديو الخاص بك بصورة جديدة. يمكنك تحديد المناطق المراد استبدالها باستخدام قناع، وتقوم العقدة بتبديل المحتوى المحدد بسلاسة عبر تسلسل الفيديو.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|---------|---------------|--------|---------|-------|
| `video` | VIDEO | نعم | - | الفيديو المراد استبدال كائن فيه. |
| `image` | IMAGE | نعم | - | الصورة المستخدمة لاستبدال الكائن المُقنّع في الفيديو. |
| `mask` | MASK | نعم | - | استخدم القناع لتحديد المناطق في الفيديو المراد استبدالها. |
| `prompt_text` | STRING | نعم | - | نص موجه يصف الاستبدال المطلوب. |
| `negative_prompt` | STRING | نعم | - | نص موجه يصف ما يجب تجنبه في الاستبدال. |
| `seed` | INT | نعم | 0 إلى 4294967295 | قيمة البذرة العشوائية للحصول على نتائج متسقة. |

**ملاحظة:** تتطلب هذه العقدة توفير جميع معاملات الإدخال. تعمل `video` و `image` و `mask` معًا لتحديد عملية الاستبدال، حيث يحدد القناع مناطق الفيديو التي سيتم استبدالها بالصورة المقدمة.

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|------------|---------------|-------|
| `output` | VIDEO | الفيديو المعالج مع استبدال الكائن أو المنطقة المحددة. |
