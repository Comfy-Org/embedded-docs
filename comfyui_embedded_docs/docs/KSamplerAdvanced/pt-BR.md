O nó KSamplerAdvanced foi projetado para aprimorar o processo de amostragem, oferecendo configurações e técnicas avançadas. Seu objetivo é fornecer opções mais sofisticadas para gerar amostras a partir de um modelo, melhorando as funcionalidades básicas do KSampler.

## Entradas

| Parâmetro             | Tipo de Dado | Descrição                                                                                                                                                                                                                                                                                                                                                     |
|----------------------|-------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| `modelo`              | MODEL       | Especifica o modelo a partir do qual as amostras serão geradas, desempenhando um papel crucial no processo de amostragem.                                                                                                                                                                                                                      |
| `adicionar_ruído`          | COMBO[STRING] | Determina se ruído deve ser adicionado ao processo de amostragem, afetando a diversidade e a qualidade das amostras geradas.                                                                                                                                                                                                             |
| `semente_do_ruído`         | INT         | Define a semente para a geração de ruído, garantindo reprodutibilidade no processo de amostragem.                                                                                                                                                                                                                                     |
| `passos`              | INT         | Define o número de etapas a serem executadas no processo de amostragem, impactando o detalhamento e a qualidade da saída.                                                                                                                                                                                                                   |
| `cfg`                | FLOAT       | Controla o fator de condicionamento, influenciando a direção e o espaço do processo de amostragem.                                                                                                                                                                                                                                  |
| `nome_do_sampler`       | COMBO[STRING] | Seleciona o amostrador específico a ser utilizado, permitindo a personalização da técnica de amostragem.                                                                                                                                                                                                                                  |
| `agendador`          | COMBO[STRING] | Escolhe o agendador para controlar o processo de amostragem, afetando a progressão e a qualidade das amostras.                                                                                                                                                                                                                   |
| `positivo`           | CONDITIONING | Especifica o condicionamento positivo para guiar a amostragem em direção a atributos desejados.                                                                                                                                                                                                                                     |
| `negativo`           | CONDITIONING | Especifica o condicionamento negativo para desviar a amostragem de certos atributos.                                                                                                                                                                                                                                     |
| `imagem_latente`       | LATENT      | Fornece a imagem latente inicial a ser usada no processo de amostragem, servindo como ponto de partida.                                                                                                                                                                                                                               |
| `iniciar_no_passo`      | INT         | Determina a etapa inicial do processo de amostragem, permitindo controle sobre a progressão da amostragem.                                                                                                                                                                                                                               |
| `terminar_no_passo`        | INT         | Define a etapa final do processo de amostragem, delimitando o escopo da amostragem.                                                                                                                                                                                                                                         |
| `retornar_com_ruído_restante` | COMBO[STRING] | Indica se a amostra deve ser retornada com ruído residual, afetando a aparência da saída final.                                                                                                                                                                                                                               |

## Saídas

| Parâmetro   | Tipo de Dado | Descrição                                                                                                               |
|-------------|-------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| `latent`    | LATENT      | A saída representa a imagem latente gerada a partir do modelo, refletindo as configurações e técnicas aplicadas. |
