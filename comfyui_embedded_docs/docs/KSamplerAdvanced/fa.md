گره KSamplerAdvanced برای بهبود فرآیند نمونه‌برداری با ارائه پیکربندی‌ها و تکنیک‌های پیشرفته طراحی شده است. هدف این گره ارائه گزینه‌های پیچیده‌تر برای تولید نمونه‌ها از یک مدل، با بهبود قابلیت‌های پایه KSampler می‌باشد.

## ورودی‌ها

| پارامتر | نوع داده | توضیحات |
|---------|-----------|----------|
| `model` | MODEL | مدلی که نمونه‌ها از آن تولید می‌شوند را مشخص می‌کند و نقش مهمی در فرآیند نمونه‌برداری دارد. |
| `add_noise` | COMBO[STRING] | تعیین می‌کند که آیا نویز به فرآیند نمونه‌برداری اضافه شود یا خیر، که بر تنوع و کیفیت نمونه‌های تولیدشده تأثیر می‌گذارد. |
| `noise_seed` | INT | دانه (seed) تولید نویز را تنظیم می‌کند تا تکرارپذیری در فرآیند نمونه‌برداری تضمین شود. |
| `steps` | INT | تعداد گام‌های فرآیند نمونه‌برداری را تعریف می‌کند که بر جزئیات و کیفیت خروجی تأثیر می‌گذارد. |
| `cfg` | FLOAT | عامل شرطی‌سازی را کنترل می‌کند و جهت و فضای فرآیند نمونه‌برداری را تحت تأثیر قرار می‌دهد. |
| `sampler_name` | COMBO[STRING] | نمونه‌بردار خاص مورد استفاده را انتخاب می‌کند و امکان سفارشی‌سازی تکنیک نمونه‌برداری را فراهم می‌آورد. |
| `scheduler` | COMBO[STRING] | زمان‌بند کنترل فرآیند نمونه‌برداری را انتخاب می‌کند که بر پیشرفت و کیفیت نمونه‌ها تأثیر می‌گذارد. |
| `positive` | CONDITIONING | شرطی‌سازی مثبت را برای هدایت نمونه‌برداری به سمت ویژگی‌های مطلوب مشخص می‌کند. |
| `negative` | CONDITIONING | شرطی‌سازی منفی را برای دور کردن نمونه‌برداری از ویژگی‌های خاص مشخص می‌کند. |
| `latent_image` | LATENT | تصویر نهفته اولیه را برای استفاده در فرآیند نمونه‌برداری فراهم می‌کند که به عنوان نقطه شروع عمل می‌کند. |
| `start_at_step` | INT | گام شروع فرآیند نمونه‌برداری را تعیین می‌کند و امکان کنترل بر پیشرفت نمونه‌برداری را فراهم می‌آورد. |
| `end_at_step` | INT | گام پایان فرآیند نمونه‌برداری را تنظیم می‌کند و دامنه نمونه‌برداری را مشخص می‌کند. |
| `return_with_leftover_noise` | COMBO[STRING] | مشخص می‌کند که آیا نمونه با نویز باقی‌مانده بازگردانده شود یا خیر، که بر ظاهر خروجی نهایی تأثیر می‌گذارد. |

## خروجی‌ها

| پارامتر | نوع داده | توضیحات |
|---------|-----------|----------|
| `latent` | LATENT | خروجی نشان‌دهنده تصویر نهفته تولیدشده از مدل است که پیکربندی‌ها و تکنیک‌های اعمال‌شده را منعکس می‌کند. |
