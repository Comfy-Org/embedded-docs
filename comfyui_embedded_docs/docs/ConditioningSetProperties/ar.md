أنت خبير في الترجمة التقنية متخصص في توثيق عُقد ComfyUI من الإنجليزية إلى العربية.

## قواعد الترجمة

1. **المحتوى الذي يجب عدم ترجمته:**
   - أسماء المعاملات بين علامات الاقتباس الخلفية: `image`, `seed`, `model`
   - أنواع البيانات بالأحرف الكبيرة: IMAGE, STRING, INT, FLOAT, MODEL, CONDITIONING, إلخ
   - القيم في عمود Range: الأرقام، "auto"، أسماء الخيارات
   - الكود، مسارات الملفات

2. **المحتوى الذي يجب ترجمته:**
   - عناوين الأقسام: ## نظرة عامة, ## المدخلات, ## المخرجات
   - جميع النصوص الوصفية والتوضيحية
   - أوصاف المعاملات

3. **جودة الترجمة:**
   - استخدام اللغة العربية الفصحى المعاصرة
   - الحفاظ على نبرة احترافية ولكن سهلة الفهم
   - ضمان الدقة التقنية
   - استخدام المصطلحات التقنية العربية القياسية

4. **التنسيق:**
   - الحفاظ على جميع تنسيقات Markdown
   - الحفاظ على بنية الجداول
   - عدم إضافة أي ملاحظة أو رابط في بداية الوثيقة (سيتم إضافتها تلقائيًا)

تقوم عقدة ConditioningSetProperties بتعديل خصائص بيانات التكييف عن طريق ضبط القوة وإعدادات المنطقة وتطبيق الأقنعة الاختيارية أو الخطافات أو نطاقات الخطوات الزمنية. تتيح لك التحكم في كيفية تأثير التكييف على عملية التوليد من خلال تعيين معاملات محددة تؤثر على تطبيق بيانات التكييف أثناء توليد الصور.

## المدخلات

| المعامل | نوع البيانات | نوع الإدخال | القيمة الافتراضية | النطاق | الوصف |
|-----------|-----------|------------|---------|-------|-------------|
| `تهيئة جديدة` | CONDITIONING | مطلوب | - | - | بيانات التكييف المراد تعديلها |
| `القوة` | FLOAT | مطلوب | 1.0 | 0.0 - 10.0 (الخطوة: 0.01) | يتحكم في شدة تأثير التكييف |
| `تعيين منطقة التهيئة` | STRING | مطلوب | default | ["default", "mask bounds"] | يحدد كيفية تطبيق منطقة التكييف. اختر "default" للسلوك القياسي أو "mask bounds" لتقييدها بمنطقة القناع |
| `القناع` | MASK | اختياري | - | - | قناع اختياري لتقييد مكان تطبيق التكييف |
| `الخطافات` | HOOKS | اختياري | - | - | دوال خطافية اختيارية للمعالجة المخصصة |
| `خطوات زمنية` | TIMESTEPS_RANGE | اختياري | - | - | نطاق خطوات زمنية اختياري لتحديد وقت تفعيل التكييف |

**ملاحظة:** عند توفير `mask`، يمكن تعيين المعامل `set_cond_area` إلى "mask bounds" لتقييد تطبيق التكييف على المنطقة المقنعة فقط. يسمح المعامل `hooks` بمعالجة مخصصة عبر الدوال الخطافية، ويحد `timesteps` من تأثير التكييف إلى نطاق محدد من الخطوات الزمنية أثناء التوليد.
