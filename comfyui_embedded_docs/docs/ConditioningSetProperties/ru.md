Вот перевод документации узла ConditioningSetProperties на русский язык:

Узел ConditioningSetProperties изменяет свойства данных кондиционирования путем настройки силы, области применения и применения опциональных масок, хуков или диапазонов временных шагов. Он позволяет управлять тем, как кондиционирование влияет на процесс генерации, задавая определенные параметры, которые воздействуют на применение данных кондиционирования во время генерации изображения.

## Входы

| Параметр | Тип данных | Тип входа | По умолчанию | Диапазон | Описание |
|-----------|-----------|------------|---------|-------|-------------|
| `новое_условие` | CONDITIONING | Обязательный | - | - | Данные кондиционирования для изменения |
| `сила` | FLOAT | Обязательный | 1.0 | 0.0 - 10.0 (шаг: 0.01) | Управляет интенсивностью эффекта кондиционирования |
| `установить_область_условия` | STRING | Обязательный | default | ["default", "mask bounds"] | Определяет способ применения области кондиционирования. Выберите "default" для стандартного поведения или "mask bounds" для ограничения областью маски |
| `маска` | MASK | Опциональный | - | - | Опциональная маска для ограничения области применения кондиционирования |
| `хуки` | HOOKS | Опциональный | - | - | Опциональные функции-хуки для пользовательской обработки |
| `временные_шаги` | TIMESTEPS_RANGE | Опциональный | - | - | Опциональный диапазон временных шагов для ограничения периода активности кондиционирования |

**Примечание:** Если указана `mask`, параметр `set_cond_area` можно установить в значение "mask bounds", чтобы ограничить применение кондиционирования только областью маски. Параметр `hooks` позволяет выполнять пользовательскую обработку с помощью функций-хуков, а `timesteps` ограничивает эффект кондиционирования определенным диапазоном временных шагов в процессе генерации.

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `CONDITIONING` | CONDITIONING | Измененные данные кондиционирования с обновленными свойствами |
