این گره برای تولید داده‌های شرطی‌سازی (conditioning) در کارهای تولید ویدئو طراحی شده است و به‌طور خاص برای استفاده با مدل‌های SVD_img2vid بهینه‌سازی شده است. این گره ورودی‌های مختلفی از جمله تصاویر اولیه، پارامترهای ویدئو و یک مدل VAE دریافت می‌کند تا داده‌های شرطی‌سازی را تولید کند که برای هدایت تولید فریم‌های ویدئو قابل استفاده باشد.

## ورودی‌ها

| پارامتر               | نوع Comfy        | توضیحات |
|----------------------|--------------------|-------------|
| `clip_vision`         | `CLIP_VISION`      | نمایانگر مدل بینایی CLIP است که برای رمزگذاری ویژگی‌های بصری از تصویر اولیه استفاده می‌شود و نقش مهمی در درک محتوا و زمینه تصویر برای تولید ویدئو ایفا می‌کند. |
| `init_image`          | `IMAGE`            | تصویر اولیه‌ای که ویدئو از آن تولید خواهد شد و به عنوان نقطه شروع فرآیند تولید ویدئو عمل می‌کند. |
| `vae`                 | `VAE`              | یک مدل خودرمزگذار متغیر (VAE) که برای رمزگذاری تصویر اولیه به فضای نهان (latent space) استفاده می‌شود و تولید فریم‌های ویدئویی منسجم و پیوسته را تسهیل می‌کند. |
| `width`               | `INT`              | عرض مورد نظر برای فریم‌های ویدئوی تولید شده که امکان سفارشی‌سازی وضوح ویدئو را فراهم می‌کند. |
| `height`              | `INT`              | ارتفاع مورد نظر برای فریم‌های ویدئو که امکان کنترل نسبت تصویر و وضوح ویدئو را فراهم می‌کند. |
| `video_frames`        | `INT`              | تعداد فریم‌هایی که برای ویدئو تولید می‌شود را مشخص می‌کند و طول ویدئو را تعیین می‌نماید. |
| `motion_bucket_id`    | `INT`              | یک شناسه برای دسته‌بندی نوع حرکت اعمال‌شده در تولید ویدئو است و به ایجاد ویدئوهای پویا و جذاب کمک می‌کند. |
| `fps`                 | `INT`              | نرخ فریم بر ثانیه (fps) برای ویدئو که بر نرمی و واقع‌گرایی ویدئوی تولید شده تأثیر می‌گذارد. |
| `augmentation_level`  | `FLOAT`            | پارامتری که سطح افزایش داده (augmentation) اعمال‌شده بر تصویر اولیه را کنترل می‌کند و بر تنوع و تغییرپذیری فریم‌های ویدئوی تولید شده تأثیر می‌گذارد. |

## خروجی‌ها

| پارامتر     | نوع Comfy        | توضیحات |
|---------------|--------------------|-------------|
| `positive`    | `CONDITIONING`     | داده‌های شرطی‌سازی مثبت که شامل ویژگی‌ها و پارامترهای رمزگذاری‌شده برای هدایت فرآیند تولید ویدئو در جهت دلخواه است. |
| `negative`    | `CONDITIONING`     | داده‌های شرطی‌سازی منفی که تضادی با شرطی‌سازی مثبت ایجاد می‌کند و می‌توان از آن برای اجتناب از الگوها یا ویژگی‌های خاص در ویدئوی تولید شده استفاده کرد. |
| `latent`      | `LATENT`           | بازنمایی‌های نهان (latent representations) تولید شده برای هر فریم از ویدئو که به عنوان یک مؤلفه بنیادین برای فرآیند تولید ویدئو عمل می‌کند. |
