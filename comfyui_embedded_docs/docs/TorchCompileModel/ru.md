Узел TorchCompileModel применяет компиляцию PyTorch к модели для оптимизации её производительности. Он создаёт копию входной модели и оборачивает её функциональностью компиляции PyTorch с использованием указанного бэкенда. Это может повысить скорость выполнения модели во время инференса.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `model` | MODEL | Да | - | Модель для компиляции и оптимизации |
| `backend` | STRING | Да | "inductor"<br>"cudagraphs" | Бэкенд компиляции PyTorch для использования при оптимизации |

## Выходные параметры

| Выходной параметр | Тип данных | Описание |
|-------------|-----------|-------------|
| `model` | MODEL | Скомпилированная модель с применённой компиляцией PyTorch |
