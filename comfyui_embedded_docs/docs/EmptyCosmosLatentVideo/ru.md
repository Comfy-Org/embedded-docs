Узел EmptyCosmosLatentVideo создает пустой тензор скрытого видео с заданными размерами. Он генерирует латентное представление, заполненное нулями, которое может использоваться в качестве отправной точки для рабочих процессов генерации видео, с настраиваемыми параметрами ширины, высоты, длины и размера пакета.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `ширина` | INT | Да | от 16 до MAX_RESOLUTION | Ширина скрытого видео в пикселях (по умолчанию: 1280, должна делиться на 16) |
| `высота` | INT | Да | от 16 до MAX_RESOLUTION | Высота скрытого видео в пикселях (по умолчанию: 704, должна делиться на 16) |
| `длина` | INT | Да | от 1 до MAX_RESOLUTION | Количество кадров в скрытом видео (по умолчанию: 121, должно делиться на 8) |
| `размер_пакета` | INT | Нет | от 1 до 4096 | Количество скрытых видео для генерации в одном пакете (по умолчанию: 1) |

## Выходные данные

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `samples` | LATENT | Сгенерированный пустой тензор скрытого видео с нулевыми значениями |
