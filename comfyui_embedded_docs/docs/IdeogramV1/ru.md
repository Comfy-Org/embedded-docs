Вот перевод документации узла ComfyUI **IdeogramV1** на русский язык:

Узел IdeogramV1 генерирует изображения с использованием модели Ideogram V1 через API. Он принимает текстовые подсказки и различные настройки генерации для создания одного или нескольких изображений на основе вашего ввода. Узел поддерживает различные соотношения сторон и режимы генерации для настройки результата.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `prompt` | STRING | Да | - | Подсказка для генерации изображения (по умолчанию: пусто) |
| `turbo` | BOOLEAN | Да | - | Использовать ли турбо-режим (более быстрая генерация, потенциально более низкое качество) (по умолчанию: False) |
| `aspect_ratio` | COMBO | Нет | "1:1"<br>"16:9"<br>"9:16"<br>"4:3"<br>"3:4"<br>"3:2"<br>"2:3" | Соотношение сторон для генерации изображения (по умолчанию: "1:1") |
| `magic_prompt_option` | COMBO | Нет | "AUTO"<br>"ON"<br>"OFF" | Определяет, следует ли использовать MagicPrompt при генерации (по умолчанию: "AUTO") |
| `seed` | INT | Нет | 0-2147483647 | Случайное начальное значение для генерации (по умолчанию: 0) |
| `negative_prompt` | STRING | Нет | - | Описание того, что следует исключить из изображения (по умолчанию: пусто) |
| `num_images` | INT | Нет | 1-8 | Количество генерируемых изображений (по умолчанию: 1) |

**Примечание:** Параметр `num_images` имеет максимальное ограничение — 8 изображений на один запрос генерации.

## Выходные данные

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `output` | IMAGE | Сгенерированное изображение(я) от модели Ideogram V1 |
