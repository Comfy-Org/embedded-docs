گره IdeogramV1 با استفاده از مدل Ideogram V1 از طریق یک API تصاویر تولید می‌کند. این گره با دریافت متن راهنما (prompt) و تنظیمات مختلف تولید، یک یا چند تصویر بر اساس ورودی شما ایجاد می‌کند. این گره از نسبت‌های ابعاد و حالت‌های تولید مختلف برای سفارشی‌سازی خروجی پشتیبانی می‌کند.

## ورودی‌ها

| پارامتر | نوع داده | اجباری | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `prompt` | STRING | بله | - | راهنمای تولید تصویر (پیش‌فرض: خالی) |
| `turbo` | BOOLEAN | بله | - | آیا از حالت توربو استفاده شود (تولید سریع‌تر با کیفیت احتمالی پایین‌تر) (پیش‌فرض: False) |
| `aspect_ratio` | COMBO | خیر | "1:1"<br>"16:9"<br>"9:16"<br>"4:3"<br>"3:4"<br>"3:2"<br>"2:3" | نسبت ابعاد برای تولید تصویر (پیش‌فرض: "1:1") |
| `magic_prompt_option` | COMBO | خیر | "AUTO"<br>"ON"<br>"OFF" | تعیین می‌کند که آیا از MagicPrompt در تولید استفاده شود (پیش‌فرض: "AUTO") |
| `seed` | INT | خیر | 0-2147483647 | مقدار دانه تصادفی برای تولید (پیش‌فرض: 0) |
| `negative_prompt` | STRING | خیر | - | توضیحاتی درباره آنچه باید از تصویر حذف شود (پیش‌فرض: خالی) |
| `num_images` | INT | خیر | 1-8 | تعداد تصاویر برای تولید (پیش‌فرض: 1) |

**توجه:** پارامتر `num_images` حداکثر محدودیت ۸ تصویر در هر درخواست تولید دارد.
