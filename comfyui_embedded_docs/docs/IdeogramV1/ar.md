# ترجمة وثيقة عقد IdeogramV1

## نظرة عامة

يقوم عقد IdeogramV1 بإنشاء الصور باستخدام نموذج Ideogram V1 عبر واجهة برمجة التطبيقات (API). يأخذ العقد نصوصًا توجيهية (prompts) وإعدادات توليد متنوعة لإنشاء صورة واحدة أو أكثر بناءً على مدخلاتك. يدعم العقد نسب أبعاد مختلفة وأوضاع توليد متنوعة لتخصيص المخرجات.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|---------|---------------|--------|--------|-------|
| `الوصف` | STRING | نعم | - | النص التوجيهي لتوليد الصورة (الافتراضي: فارغ) |
| `الوضع_السريع` | BOOLEAN | نعم | - | تفعيل وضع التوربو (توليد أسرع، جودة أقل محتملة) (الافتراضي: False) |
| `نسبة_الأبعاد` | COMBO | لا | "1:1"<br>"16:9"<br>"9:16"<br>"4:3"<br>"3:4"<br>"3:2"<br>"2:3" | نسبة الأبعاد لتوليد الصورة (الافتراضي: "1:1") |
| `خيار_الوصف_السحري` | COMBO | لا | "AUTO"<br>"ON"<br>"OFF" | تحديد ما إذا كان سيتم استخدام MagicPrompt في التوليد (الافتراضي: "AUTO") |
| `البذرة` | INT | لا | 0-2147483647 | قيمة البذرة العشوائية للتوليد (الافتراضي: 0) |
| `الوصف_السلبي` | STRING | لا | - | وصف ما يجب استبعاده من الصورة (الافتراضي: فارغ) |
| `عدد_الصور` | INT | لا | 1-8 | عدد الصور المراد توليدها (الافتراضي: 1) |

**ملاحظة:** معامل `num_images` له حد أقصى يبلغ 8 صور لكل طلب توليد.

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|-------------|---------------|--------|
| `output` | IMAGE | الصورة (الصور) المولدة من نموذج Ideogram V1 |
