Этот узел обнаруживает модели, расположенные в папке `ComfyUI/models/controlnet`, а также считывает модели из дополнительных путей, настроенных в файле extra_model_paths.yaml. В некоторых случаях может потребоваться **обновить интерфейс ComfyUI**, чтобы он смог прочитать файлы моделей из соответствующей папки.

Узел ControlNetLoader предназначен для загрузки модели ControlNet из указанного пути. Он играет ключевую роль в инициализации моделей ControlNet, которые необходимы для применения механизмов управления над сгенерированным контентом или изменения существующего контента на основе управляющих сигналов.

## Входные данные

| Поле               | Тип Comfy         | Описание                                                                                     |
|--------------------|-------------------|----------------------------------------------------------------------------------------------|
| `control_net_name` | `COMBO[STRING]`   | Указывает имя загружаемой модели ControlNet, используемое для поиска файла модели в заданной структуре каталогов. |

## Выходные данные

| Поле           | Тип Comfy      | Описание                                                                                          |
|----------------|----------------|---------------------------------------------------------------------------------------------------|
| `control_net`  | `CONTROL_NET`  | Возвращает загруженную модель ControlNet, готовую к использованию для управления или изменения процессов генерации контента. |
