Вот перевод документации узла ImageAddNoise на русский язык:

Узел ImageAddNoise добавляет случайный шум к входному изображению. Он использует указанное начальное значение генератора случайных чисел для создания согласованных шумовых паттернов и позволяет контролировать интенсивность эффекта шума. Результирующее изображение сохраняет те же размеры, что и входное, но с добавленной визуальной текстурой.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `image` | IMAGE | Да | - | Входное изображение, к которому будет добавлен шум |
| `seed` | INT | Да | от 0 до 18446744073709551615 | Начальное значение генератора случайных чисел, используемое для создания шума (по умолчанию: 0) |
| `strength` | FLOAT | Да | от 0.0 до 1.0 | Управляет интенсивностью эффекта шума (по умолчанию: 0.5) |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `image` | IMAGE | Выходное изображение с применённым добавленным шумом |
