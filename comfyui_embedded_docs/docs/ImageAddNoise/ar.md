# ترجمة وثيقة عقدة ImageAddNoise

تقوم عقدة ImageAddNoise بإضافة ضوضاء عشوائية إلى صورة الإدخال. تستخدم بذرة عشوائية محددة لتوليد أنماط ضوضاء متناسقة وتسمح بالتحكم في شدة تأثير الضوضاء. تحتفظ الصورة الناتجة بنفس أبعاد الصورة المدخلة ولكن مع إضافة نسيج بصري.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|---------|--------------|--------|--------|-------|
| `صورة` | IMAGE | نعم | - | صورة الإدخال التي ستُضاف إليها الضوضاء |
| `البذرة` | INT | نعم | 0 إلى 18446744073709551615 | البذرة العشوائية المستخدمة لإنشاء الضوضاء (الافتراضي: 0) |
| `القوة` | FLOAT | نعم | 0.0 إلى 1.0 | يتحكم في شدة تأثير الضوضاء (الافتراضي: 0.5) |

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|------------|--------------|-------|
| `صورة` | IMAGE | الصورة الناتجة مع تطبيق الضوضاء المضافة |
