### مرور کلی

گره ImageAddNoise نویز تصادفی به یک تصویر ورودی اضافه می‌کند. این گره از یک دانه تصادفی مشخص برای تولید الگوهای نویز یکسان استفاده می‌کند و امکان کنترل شدت اثر نویز را فراهم می‌سازد. تصویر خروجی ابعادی مشابه تصویر ورودی دارد، اما با بافت بصری اضافه‌شده.

### ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|---------|----------|-------|--------|---------|
| `image` | IMAGE | بله | - | تصویر ورودی که نویز به آن اضافه خواهد شد |
| `seed` | INT | بله | 0 تا 18446744073709551615 | دانه تصادفی مورد استفاده برای ایجاد نویز (پیش‌فرض: 0) |
| `strength` | FLOAT | بله | 0.0 تا 1.0 | شدت اثر نویز را کنترل می‌کند (پیش‌فرض: 0.5) |

### خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-----------|----------|---------|
| `image` | IMAGE | تصویر خروجی با نویز اضافه‌شده |
