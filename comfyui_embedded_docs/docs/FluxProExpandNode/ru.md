Расширяет изображение на основе промпта. Этот узел расширяет изображение, добавляя пиксели к верхней, нижней, левой и правой сторонам, одновременно генерируя новое содержимое, соответствующее предоставленному текстовому описанию.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `image` | IMAGE | Да | - | Входное изображение для расширения |
| `prompt` | STRING | Нет | - | Промпт для генерации изображения (по умолчанию: "") |
| `prompt_upsampling` | BOOLEAN | Нет | - | Выполнять ли апсемплинг промпта. Если активно, автоматически модифицирует промпт для более креативной генерации, но результаты недетерминированы (одинаковый сид не даст точно такого же результата). (по умолчанию: False) |
| `top` | INT | Нет | 0-2048 | Количество пикселей для расширения в верхней части изображения (по умолчанию: 0) |
| `bottom` | INT | Нет | 0-2048 | Количество пикселей для расширения в нижней части изображения (по умолчанию: 0) |
| `left` | INT | Нет | 0-2048 | Количество пикселей для расширения в левой части изображения (по умолчанию: 0) |
| `right` | INT | Нет | 0-2048 | Количество пикселей для расширения в правой части изображения (по умолчанию: 0) |
| `guidance` | FLOAT | Нет | 1.5-100 | Сила управления (guidance) для процесса генерации изображения (по умолчанию: 60) |
| `steps` | INT | Нет | 15-50 | Количество шагов для процесса генерации изображения (по умолчанию: 50) |
| `seed` | INT | Нет | 0-18446744073709551615 | Случайный сид, используемый для создания шума. (по умолчанию: 0) |

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `image` | IMAGE | Расширенное выходное изображение |
