Выполняет аутпейнтинг (дорисовку) изображения на основе текстового описания. Этот узел расширяет изображение, добавляя пиксели сверху, снизу, слева и справа, генерируя при этом новое содержимое, соответствующее заданному текстовому описанию.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `изображение` | IMAGE | Да | - | Входное изображение, которое необходимо расширить |
| `запрос` | STRING | Нет | - | Текстовое описание для генерации изображения (по умолчанию: "") |
| `апсемплинг запроса` | BOOLEAN | Нет | - | Выполнять ли апсемплинг (повышение качества) текстового описания. Если активно, автоматически изменяет запрос для более креативной генерации, но результаты становятся недетерминированными (одинаковый seed не даст точно такого же результата). (по умолчанию: False) |
| `верх` | INT | Нет | 0-2048 | Количество пикселей для расширения в верхней части изображения (по умолчанию: 0) |
| `низ` | INT | Нет | 0-2048 | Количество пикселей для расширения в нижней части изображения (по умолчанию: 0) |
| `слева` | INT | Нет | 0-2048 | Количество пикселей для расширения слева от изображения (по умолчанию: 0) |
| `справа` | INT | Нет | 0-2048 | Количество пикселей для расширения справа от изображения (по умолчанию: 0) |
| `направление` | FLOAT | Нет | 1.5-100 | Сила guidance (направляющего воздействия) для процесса генерации изображения (по умолчанию: 60) |
| `шаги` | INT | Нет | 15-50 | Количество шагов для процесса генерации изображения (по умолчанию: 50) |
| `seed` | INT | Нет | 0-18446744073709551615 | Случайный seed, используемый для создания шума. (по умолчанию: 0) |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `изображение` | IMAGE | Расширенное выходное изображение |
