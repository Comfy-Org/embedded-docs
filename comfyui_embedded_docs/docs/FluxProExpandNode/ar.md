# ترجمة وثيقة عقدة توسيع الصورة FluxPro

يقوم هذا العقدة بتوسيع الصورة بناءً على النص الوصفي. تعمل على توسيع الصورة بإضافة بكسلات إلى الجوانب العلوية والسفلية واليمنى واليسرى، مع توليد محتوى جديد يتطابق مع الوصف النصي المقدم.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|---------|--------------|--------|--------|-------|
| `image` | IMAGE | نعم | - | الصورة المدخلة المراد توسيعها |
| `prompt` | STRING | لا | - | النص الوصفي لتوليد الصورة (الافتراضي: "") |
| `prompt_upsampling` | BOOLEAN | لا | - | هل يتم إجراء رفع دقة على النص الوصفي. إذا كان نشطًا، يقوم تلقائيًا بتعديل النص الوصفي لتوليد أكثر إبداعًا، لكن النتائج غير حتمية (نفس البذرة لن تنتج نفس النتيجة تمامًا). (الافتراضي: False) |
| `top` | INT | لا | 0-2048 | عدد البكسلات المراد توسيعها في الجزء العلوي من الصورة (الافتراضي: 0) |
| `bottom` | INT | لا | 0-2048 | عدد البكسلات المراد توسيعها في الجزء السفلي من الصورة (الافتراضي: 0) |
| `left` | INT | لا | 0-2048 | عدد البكسلات المراد توسيعها في الجزء الأيسر من الصورة (الافتراضي: 0) |
| `right` | INT | لا | 0-2048 | عدد البكسلات المراد توسيعها في الجزء الأيمن من الصورة (الافتراضي: 0) |
| `guidance` | FLOAT | لا | 1.5-100 | قوة التوجيه لعملية توليد الصورة (الافتراضي: 60) |
| `steps` | INT | لا | 15-50 | عدد خطوات عملية توليد الصورة (الافتراضي: 50) |
| `seed` | INT | لا | 0-18446744073709551615 | البذرة العشوائية المستخدمة لإنشاء الضوضاء. (الافتراضي: 0) |
