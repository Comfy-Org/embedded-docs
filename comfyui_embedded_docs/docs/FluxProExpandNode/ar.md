# ترجمة وثيقة عقدة توسيع الصورة FluxPro

يقوم هذا العقدة بتوسيع الصورة بناءً على النص الوصفي. تعمل على توسيع الصورة بإضافة بكسلات إلى الجوانب العلوية والسفلية واليمنى واليسرى، مع توليد محتوى جديد يتطابق مع الوصف النصي المقدم.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|---------|--------------|--------|--------|-------|
| `الصورة` | IMAGE | نعم | - | الصورة المدخلة المراد توسيعها |
| `الوصف` | STRING | لا | - | النص الوصفي لتوليد الصورة (الافتراضي: "") |
| `تحسين_الوصف` | BOOLEAN | لا | - | هل يتم إجراء رفع دقة على النص الوصفي. إذا كان نشطًا، يقوم تلقائيًا بتعديل النص الوصفي لتوليد أكثر إبداعًا، لكن النتائج غير حتمية (نفس البذرة لن تنتج نفس النتيجة تمامًا). (الافتراضي: False) |
| `الأعلى` | INT | لا | 0-2048 | عدد البكسلات المراد توسيعها في الجزء العلوي من الصورة (الافتراضي: 0) |
| `الأسفل` | INT | لا | 0-2048 | عدد البكسلات المراد توسيعها في الجزء السفلي من الصورة (الافتراضي: 0) |
| `اليسار` | INT | لا | 0-2048 | عدد البكسلات المراد توسيعها في الجزء الأيسر من الصورة (الافتراضي: 0) |
| `اليمين` | INT | لا | 0-2048 | عدد البكسلات المراد توسيعها في الجزء الأيمن من الصورة (الافتراضي: 0) |
| `الإرشاد` | FLOAT | لا | 1.5-100 | قوة التوجيه لعملية توليد الصورة (الافتراضي: 60) |
| `الخطوات` | INT | لا | 15-50 | عدد خطوات عملية توليد الصورة (الافتراضي: 50) |
| `البذرة` | INT | لا | 0-18446744073709551615 | البذرة العشوائية المستخدمة لإنشاء الضوضاء. (الافتراضي: 0) |
