### نمای کلی

تصویر را بر اساس prompt (دستور متنی) گسترش می‌دهد. این گره با افزودن پیکسل به طرف‌های بالا، پایین، چپ و راست تصویر را بزرگ‌تر کرده و محتوای جدیدی متناسب با توضیحات متنی ارائه‌شده تولید می‌کند.

## ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|---------|----------|-------|--------|---------|
| `image` | IMAGE | بله | - | تصویر ورودی که قرار است گسترش یابد |
| `prompt` | STRING | خیر | - | دستور متنی برای تولید تصویر (پیش‌فرض: "") |
| `prompt_upsampling` | BOOLEAN | خیر | - | آیا بر روی دستور متنی، افزایش وضوح (upsampling) انجام شود. در صورت فعال بودن، به‌طور خودکار دستور متنی را برای تولید خلاقانه‌تر تغییر می‌دهد، اما نتایج غیرقطعی هستند (همان seed نتیجه دقیقاً یکسانی تولید نمی‌کند). (پیش‌فرض: False) |
| `top` | INT | خیر | 0-2048 | تعداد پیکسل‌هایی که به بالای تصویر اضافه می‌شود (پیش‌فرض: 0) |
| `bottom` | INT | خیر | 0-2048 | تعداد پیکسل‌هایی که به پایین تصویر اضافه می‌شود (پیش‌فرض: 0) |
| `left` | INT | خیر | 0-2048 | تعداد پیکسل‌هایی که به چپ تصویر اضافه می‌شود (پیش‌فرض: 0) |
| `right` | INT | خیر | 0-2048 | تعداد پیکسل‌هایی که به راست تصویر اضافه می‌شود (پیش‌فرض: 0) |
| `guidance` | FLOAT | خیر | 1.5-100 | قدرت هدایت برای فرآیند تولید تصویر (پیش‌فرض: 60) |
| `steps` | INT | خیر | 15-50 | تعداد گام‌های فرآیند تولید تصویر (پیش‌فرض: 50) |
| `seed` | INT | خیر | 0-18446744073709551615 | دانه تصادفی (seed) مورد استفاده برای ایجاد نویز (پیش‌فرض: 0) |
