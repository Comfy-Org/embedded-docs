Узел ModelMergeAdd предназначен для слияния двух моделей путем добавления ключевых патчей из одной модели в другую. Этот процесс включает клонирование первой модели и последующее применение патчей из второй модели, что позволяет объединить характеристики или поведение обеих моделей.

## Входные параметры

| Параметр | Тип данных | Описание |
|----------|------------|----------|
| `model1` | `MODEL`    | Первая модель, которая будет клонирована и в которую будут добавлены патчи из второй модели. Служит базовой моделью для процесса слияния. |
| `model2` | `MODEL`    | Вторая модель, из которой извлекаются ключевые патчи и добавляются в первую модель. Вносит дополнительные характеристики или поведение в объединенную модель. |

## Выходные параметры

| Параметр | Тип данных | Описание |
|----------|------------|----------|
| `model`  | MODEL      | Результат слияния двух моделей путем добавления ключевых патчей из второй модели в первую. Эта объединенная модель сочетает характеристики или поведение обеих моделей. |
