Узел PatchModelAddDownscale реализует функциональность Kohya Deep Shrink, применяя операции понижения и повышения разрешения к определенным блокам модели. Он уменьшает разрешение промежуточных признаков во время обработки, а затем восстанавливает их до исходного размера, что позволяет повысить производительность при сохранении качества. Узел обеспечивает точный контроль над тем, когда и как эти операции масштабирования происходят во время выполнения модели.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `model` | MODEL | Да | - | Модель, к которой применяется патч понижения разрешения |
| `block_number` | INT | Нет | 1-32 | Номер конкретного блока, к которому будет применено понижение разрешения (по умолчанию: 3) |
| `downscale_factor` | FLOAT | Нет | 0.1-9.0 | Коэффициент понижения разрешения признаков (по умолчанию: 2.0) |
| `start_percent` | FLOAT | Нет | 0.0-1.0 | Начальная точка процесса шумоподавления, с которой начинается понижение разрешения (по умолчанию: 0.0) |
| `end_percent` | FLOAT | Нет | 0.0-1.0 | Конечная точка процесса шумоподавления, на которой понижение разрешения прекращается (по умолчанию: 0.35) |
| `downscale_after_skip` | BOOLEAN | Нет | - | Применять ли понижение разрешения после пропускных соединений (по умолчанию: True) |
| `downscale_method` | COMBO | Нет | "bicubic"<br>"nearest-exact"<br>"bilinear"<br>"area"<br>"bislerp" | Метод интерполяции, используемый для операций понижения разрешения |
| `upscale_method` | COMBO | Нет | "bicubic"<br>"nearest-exact"<br>"bilinear"<br>"area"<br>"bislerp" | Метод интерполяции, используемый для операций повышения разрешения |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `model` | MODEL | Модифицированная модель с примененным патчем понижения разрешения |
