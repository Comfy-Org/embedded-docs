Узел **ScaleROPE** позволяет изменять вращательное позиционное кодирование (ROPE) модели, применяя отдельные коэффициенты масштабирования и сдвига к её компонентам X, Y и T (время). Это экспериментальный узел продвинутого уровня, используемый для настройки поведения позиционного кодирования модели.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `model` | MODEL | Да | - | Модель, параметры ROPE которой будут изменены. |
| `scale_x` | FLOAT | Нет | 0.0 - 100.0 | Коэффициент масштабирования для компонента X ROPE (по умолчанию: 1.0). |
| `shift_x` | FLOAT | Нет | -256.0 - 256.0 | Значение сдвига для компонента X ROPE (по умолчанию: 0.0). |
| `scale_y` | FLOAT | Нет | 0.0 - 100.0 | Коэффициент масштабирования для компонента Y ROPE (по умолчанию: 1.0). |
| `shift_y` | FLOAT | Нет | -256.0 - 256.0 | Значение сдвига для компонента Y ROPE (по умолчанию: 0.0). |
| `scale_t` | FLOAT | Нет | 0.0 - 100.0 | Коэффициент масштабирования для компонента T (время) ROPE (по умолчанию: 1.0). |
| `shift_t` | FLOAT | Нет | -256.0 - 256.0 | Значение сдвига для компонента T (время) ROPE (по умолчанию: 0.0). |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `model` | MODEL | Модель с применёнными новыми параметрами масштабирования и сдвига ROPE. |
