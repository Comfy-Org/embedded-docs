## Обзор

Создайте 3D-модель по текстовому описанию с помощью API Rodin Gen-2.5. Вы можете выбрать один из режимов качества (Быстрый, Обычный или Экстремально высокий) для баланса между скоростью генерации и качеством результата.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `prompt` | STRING | Да | Макс. 2500 символов | Текстовое описание 3D-модели, которую вы хотите создать. |
| `mode` | COMBO | Да | `"Fast"`<br>`"Regular"`<br>`"Extreme-High"` | Режим качества и скорости генерации. "Fast" — самый быстрый, "Extreme-High" обеспечивает наивысшее качество, но требует больше времени. |
| `material` | COMBO | Да | `"PBR"`<br>`"Matte"`<br>`"Shiny"` | Стиль материала для создаваемой 3D-модели. |
| `geometry_file_format` | COMBO | Да | `"glb"`<br>`"obj"`<br>`"stl"`<br>`"usdz"` | Формат файла для выходной 3D-модели. |
| `texture_mode` | COMBO | Да | `"None"`<br>`"Generated"`<br>`"Generated+HD"` | Режим генерации текстур. "None" — без текстур, "Generated" — создание стандартных текстур, "Generated+HD" — создание текстур высокого разрешения. |
| `seed` | INT | Да | от 0 до 2147483647 | Случайное зерно для воспроизводимых результатов. Использование одного и того же зерна с одинаковыми входными данными приведёт к одинаковому результату. |
| `TAPose` | BOOLEAN | Да | True / False | Применить ли T-позу (руки в стороны) к создаваемой модели. |
| `hd_texture` | BOOLEAN | Да | True / False | Создавать ли текстуры высокого разрешения для модели. |
| `texture_delight` | BOOLEAN | Да | True / False | Применять ли улучшение текстур (повышенное качество текстур) к модели. |
| `addon_highpack` | BOOLEAN | Да | True / False | Создавать ли высокополигональную версию модели в дополнение к стандартной. |
| `bbox_width` | INT | Да | от 1 до 1000 | Ширина ограничивающего прямоугольника в мировых единицах. |
| `bbox_height` | INT | Да | от 1 до 1000 | Высота ограничивающего прямоугольника в мировых единицах. |
| `bbox_length` | INT | Да | от 1 до 1000 | Длина ограничивающего прямоугольника в мировых единицах. |
| `height_cm` | INT | Да | от 1 до 300 | Высота создаваемой модели в сантиметрах. |

**Примечание:** Параметр `prompt` должен содержать от 1 до 2500 символов. Параметр `seed` по умолчанию равен 0 (случайное значение), если не указан иначе.

## Выходные данные

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `model_file` | FILE3DANY | Файл созданной 3D-модели в указанном формате. |
