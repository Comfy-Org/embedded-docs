## نمای کلی

با استفاده از API Rodin Gen-2.5 یک مدل سه‌بعدی از روی توضیح متنی تولید کنید. می‌توانید بین حالت‌های کیفیت مختلف (سریع، معمولی یا فوق‌بالا) یکی را انتخاب کنید تا سرعت تولید و کیفیت خروجی را متعادل سازید.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `prompt` | STRING | بله | حداکثر ۲۵۰۰ کاراکتر | توضیح متنی که مدل سه‌بعدی مورد نظر شما را توصیف می‌کند. |
| `mode` | COMBO | بله | `"Fast"`<br>`"Regular"`<br>`"Extreme-High"` | حالت کیفیت و سرعت تولید. حالت "Fast" سریع‌ترین است، "Extreme-High" بالاترین کیفیت را تولید می‌کند اما زمان بیشتری می‌برد. |
| `material` | COMBO | بله | `"PBR"`<br>`"Matte"`<br>`"Shiny"` | سبک متریال برای مدل سه‌بعدی تولیدشده. |
| `geometry_file_format` | COMBO | بله | `"glb"`<br>`"obj"`<br>`"stl"`<br>`"usdz"` | فرمت فایل برای مدل سه‌بعدی خروجی. |
| `texture_mode` | COMBO | بله | `"None"`<br>`"Generated"`<br>`"Generated+HD"` | حالت تولید بافت. "None" هیچ بافتی تولید نمی‌کند، "Generated" بافت‌های استاندارد و "Generated+HD" بافت‌های با وضوح بالا ایجاد می‌کند. |
| `seed` | INT | بله | ۰ تا ۲۱۴۷۴۸۳۶۴۷ | دانه تصادفی برای نتایج قابل تکرار. استفاده از دانه یکسان با ورودی‌های یکسان، خروجی یکسانی تولید می‌کند. |
| `TAPose` | BOOLEAN | بله | True / False | آیا حالت T-Pose (بازوهای کشیده) روی مدل تولیدشده اعمال شود. |
| `hd_texture` | BOOLEAN | بله | True / False | آیا بافت‌های با وضوح بالا برای مدل تولید شود. |
| `texture_delight` | BOOLEAN | بله | True / False | آیا بهبود بافت (کیفیت بافت افزایش‌یافته) روی مدل اعمال شود. |
| `addon_highpack` | BOOLEAN | بله | True / False | آیا نسخه پرپلیگون مدل علاوه بر نسخه استاندارد تولید شود. |
| `bbox_width` | INT | بله | ۱ تا ۱۰۰۰ | عرض جعبه محصور بر حسب واحدهای جهانی. |
| `bbox_height` | INT | بله | ۱ تا ۱۰۰۰ | ارتفاع جعبه محصور بر حسب واحدهای جهانی. |
| `bbox_length` | INT | بله | ۱ تا ۱۰۰۰ | طول جعبه محصور بر حسب واحدهای جهانی. |
| `height_cm` | INT | بله | ۱ تا ۳۰۰ | ارتفاع مدل تولیدشده بر حسب سانتی‌متر. |

**نکته:** پارامتر `prompt` باید بین ۱ تا ۲۵۰۰ کاراکتر باشد. پارامتر `seed` در صورت عدم مشخص‌شدن، به‌طور پیش‌فرض ۰ (تصادفی) است.
