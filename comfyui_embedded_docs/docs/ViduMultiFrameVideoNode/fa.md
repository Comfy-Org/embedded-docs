| پارامتر | نوع داده | الزامی | محدوده | توضیحات |

| `مدل` | COMBO | بله | `"viduq2-pro"`<br>`"viduq2-turbo"` | مدل Vidu مورد استفاده برای تولید ویدیو. |
| `تصویر شروع` | IMAGE | بله | - | تصویر فریم شروع. نسبت ابعاد باید بین ۱:۴ و ۴:۱ باشد. |

| `فریم‌ها` | DYNAMICCOMBO | بله | `"2"`<br>`"3"`<br>`"4"`<br>`"5"`<br>`"6"`<br>`"7"`<br>`"8"`<br>`"9"` | تعداد انتقال‌های بین فریم‌های کلیدی (۲-۹). انتخاب یک مقدار به‌صورت پویا ورودی‌های مورد نیاز برای هر فریم را نمایش می‌دهد. |

**ورودی‌های فریم (نمایش پویا):**
هنگامی که مقداری برای `frames` انتخاب می‌کنید (مثلاً "۳")، گره مجموعه‌ای از ورودی‌های مورد نیاز برای هر انتقال را نمایش می‌دهد. برای هر فریم `i` از ۱ تا تعداد انتخاب‌شده، باید موارد زیر را ارائه دهید:

* `end_image{i}` (IMAGE): تصویر هدف برای این انتقال. نسبت ابعاد باید بین ۱:۴ و ۴:۱ باشد.
* `prompt{i}` (STRING): توضیح متنی که انتقال به این فریم را هدایت می‌کند (حداکثر ۲۰۰۰ کاراکتر).
* `duration{i}` (INT): مدت زمان بر حسب ثانیه برای این بخش انتقال خاص.
