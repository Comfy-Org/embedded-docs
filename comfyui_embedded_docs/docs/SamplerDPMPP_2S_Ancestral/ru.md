Вот перевод документации на русский язык:

Узел SamplerDPMPP_2S_Ancestral создаёт сэмплер, использующий метод сэмплирования DPM++ 2S Ancestral для генерации изображений. Этот сэмплер сочетает детерминированные и стохастические элементы для получения разнообразных результатов при сохранении некоторой согласованности. Он позволяет контролировать уровень случайности и шума в процессе сэмплирования.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `eta` | FLOAT | Да | 0.0 - 100.0 | Управляет количеством стохастического шума, добавляемого во время сэмплирования (по умолчанию: 1.0) |
| `s_noise` | FLOAT | Да | 0.0 - 100.0 | Управляет масштабом шума, применяемого в процессе сэмплирования (по умолчанию: 1.0) |

## Выходы

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `sampler` | SAMPLER | Возвращает настроенный объект сэмплера, который можно использовать в конвейере сэмплирования |
