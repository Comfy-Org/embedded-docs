Узел ConditioningConcat предназначен для объединения векторов кондиционирования, в частности, для слияния вектора `conditioning_from` с вектором `conditioning_to`. Эта операция является фундаментальной в сценариях, когда информацию кондиционирования из двух источников необходимо объединить в единое представление.

## Входы

| Параметр             | Тип данных Comfy   | Описание |
|-----------------------|--------------------|-------------|
| `conditioning_to`     | `CONDITIONING`     | Представляет основной набор векторов кондиционирования, к которому будут присоединены векторы `conditioning_from`. Служит основой для процесса объединения. |
| `conditioning_from`   | `CONDITIONING`     | Состоит из векторов кондиционирования, которые должны быть присоединены к векторам `conditioning_to`. Этот параметр позволяет интегрировать дополнительную информацию кондиционирования в существующий набор. |

## Выходы

| Параметр            | Тип данных Comfy   | Описание |
|----------------------|--------------------|-------------|
| `conditioning`       | `CONDITIONING`     | Результатом является объединённый набор векторов кондиционирования, полученный путём присоединения векторов `conditioning_from` к векторам `conditioning_to`. |
