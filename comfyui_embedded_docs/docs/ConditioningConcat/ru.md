Узел ConditioningConcat предназначен для конкатенации векторов conditioning, а именно — объединения вектора `conditioning_from` в вектор `conditioning_to`. Эта операция является основополагающей в сценариях, где необходимо объединить информацию conditioning из двух источников в единое представление.

## Входные параметры

| Параметр             | Comfy dtype        | Описание |
|-----------------------|--------------------|-------------|
| `условие_к`     | `CONDITIONING`     | Представляет основной набор векторов conditioning, к которому будут присоединены векторы `условие_от`. Служит базой для процесса конкатенации. |
| `условие_от`   | `CONDITIONING`     | Содержит векторы conditioning, которые необходимо присоединить к векторам `условие_к`. Этот параметр позволяет интегрировать дополнительную информацию conditioning в существующий набор. |

## Выходные параметры

| Параметр            | Comfy dtype        | Описание |
|----------------------|--------------------|-------------|
| `conditioning`       | `CONDITIONING`     | Результат — единый набор векторов conditioning, полученный в результате конкатенации векторов `условие_от` в векторы `условие_к`. |
