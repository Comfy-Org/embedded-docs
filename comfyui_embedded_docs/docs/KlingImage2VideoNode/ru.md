Вот перевод документации на русский язык:

Узел Kling Image to Video генерирует видео из исходного референсного изображения с использованием текстовых подсказок. Он принимает изображение в качестве первого кадра и создает видеопоследовательность на основе положительных и отрицательных текстовых описаний с настраиваемыми параметрами модели, длительности, соотношения сторон и режима генерации.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `start_frame` | IMAGE | Да | - | Референсное изображение, используемое для генерации видео. |
| `prompt` | STRING | Да | - | Положительный текстовый промпт. |
| `negative_prompt` | STRING | Да | - | Отрицательный текстовый промпт. |
| `model_name` | COMBO | Да | `"kling-v2-master"`<br>`"kling-v2-1-master"`<br>`"kling-v2-5-turbo"`<br>`"kling-v2-1"`<br>`"kling-v1-6"`<br>`"kling-v1-5"`<br>`"kling-v1-4"`<br>`"kling-v1-0"` | Модель, используемая для генерации видео (по умолчанию: `"kling-v2-master"`). |
| `cfg_scale` | FLOAT | Да | от 0.0 до 1.0 | Определяет, насколько точно видео следует промпту. Более высокие значения означают более строгое соответствие (по умолчанию: 0.8). |
| `mode` | COMBO | Да | `"std"`<br>`"pro"` | Режим генерации. `"std"` — стандартное качество, `"pro"` — более высокое качество (по умолчанию: `"std"`). |
| `aspect_ratio` | COMBO | Да | `"16:9"`<br>`"9:16"`<br>`"1:1"` | Соотношение сторон генерируемого видео (по умолчанию: `"16:9"`). |
| `duration` | COMBO | Да | `"5"`<br>`"10"` | Длительность генерируемого видео в секундах (по умолчанию: `"5"`). |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `output` | VIDEO | Сгенерированный видеофайл. |
| `video_id` | STRING | Уникальный идентификатор сгенерированного видео. |
| `duration` | STRING | Информация о длительности сгенерированного видео. |
