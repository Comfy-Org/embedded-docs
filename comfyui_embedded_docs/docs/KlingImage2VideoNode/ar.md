بالتأكيد، إليك الترجمة العربية للوثيقة التقنية الخاصة بعقدة ComfyUI، مع الالتزام التام بقواعد الترجمة المحددة:

## عقدة تحويل الصورة إلى فيديو من Kling

تقوم عقدة تحويل الصورة إلى فيديو من Kling بإنشاء فيديو من صورة مرجعية أولية باستخدام مطالبات نصية. تأخذ العقدة صورة كإطار أول وتُنشئ سلسلة فيديو بناءً على أوصاف نصية إيجابية وسلبية، مع خيارات قابلة للتخصيص للنموذج، والمدة، ونسبة العرض إلى الارتفاع، ووضع التوليد.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|-----------|-----------|----------|-------|-------------|
| `start_frame` | IMAGE | نعم | - | الصورة المرجعية المستخدمة لتوليد الفيديو. |
| `prompt` | STRING | نعم | - | المطالبة النصية الإيجابية. |
| `negative_prompt` | STRING | نعم | - | المطالبة النصية السلبية. |
| `model_name` | COMBO | نعم | `"kling-v2-master"`<br>`"kling-v2-1-master"`<br>`"kling-v2-5-turbo"`<br>`"kling-v2-1"`<br>`"kling-v1-6"`<br>`"kling-v1-5"`<br>`"kling-v1-4"`<br>`"kling-v1-0"` | النموذج المستخدم لتوليد الفيديو (الافتراضي: `"kling-v2-master"`). |
| `cfg_scale` | FLOAT | نعم | 0.0 إلى 1.0 | يتحكم في مدى التزام الفيديو بالمطالبة النصية. القيم الأعلى تعني التزامًا أقوى (الافتراضي: 0.8). |
| `mode` | COMBO | نعم | `"std"`<br>`"pro"` | وضع التوليد. `"std"` للجودة القياسية، `"pro"` لجودة أعلى (الافتراضي: `"std"`). |
| `aspect_ratio` | COMBO | نعم | `"16:9"`<br>`"9:16"`<br>`"1:1"` | نسبة العرض إلى الارتفاع للفيديو المُنشأ (الافتراضي: `"16:9"`). |
| `duration` | COMBO | نعم | `"5"`<br>`"10"` | مدة الفيديو المُنشأ بالثواني (الافتراضي: `"5"`). |

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|-------------|-----------|-------------|
| `output` | VIDEO | مخرج الفيديو المُنشأ. |
| `video_id` | STRING | معرف فريد للفيديو المُنشأ. |
| `duration` | STRING | معلومات المدة للفيديو المُنشأ. |
