گره ویدیو از تصویر Kling، با استفاده از یک تصویر مرجع شروع و توصیف‌های متنی، یک ویدیو تولید می‌کند. این گره یک تصویر را به عنوان فریم اول دریافت کرده و بر اساس توصیف‌های متنی مثبت و منفی، یک دنباله ویدیویی ایجاد می‌کند. گزینه‌های قابل تنظیم شامل مدل، مدت زمان، نسبت تصویر و حالت تولید است.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `start_frame` | IMAGE | بله | - | تصویر مرجع برای تولید ویدیو. |
| `prompt` | STRING | بله | - | توصیف متنی مثبت. |
| `negative_prompt` | STRING | بله | - | توصیف متنی منفی. |
| `model_name` | COMBO | بله | `"kling-v2-master"`<br>`"kling-v2-1-master"`<br>`"kling-v2-5-turbo"`<br>`"kling-v2-1"`<br>`"kling-v1-6"`<br>`"kling-v1-5"`<br>`"kling-v1-4"`<br>`"kling-v1-0"` | مدل مورد استفاده برای تولید ویدیو (پیش‌فرض: `"kling-v2-master"`). |
| `cfg_scale` | FLOAT | بله | 0.0 تا 1.0 | میزان تطابق ویدیو با توصیف متنی را کنترل می‌کند. مقادیر بالاتر به معنای پیروی قوی‌تر است (پیش‌فرض: 0.8). |
| `mode` | COMBO | بله | `"std"`<br>`"pro"` | حالت تولید. `"std"` کیفیت استاندارد و `"pro"` کیفیت بالاتر است (پیش‌فرض: `"std"`). |
| `aspect_ratio` | COMBO | بله | `"16:9"`<br>`"9:16"`<br>`"1:1"` | نسبت تصویر ویدیوی تولید شده (پیش‌فرض: `"16:9"`). |
| `duration` | COMBO | بله | `"5"`<br>`"10"` | مدت زمان ویدیوی تولید شده بر حسب ثانیه (پیش‌فرض: `"5"`). |

## خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-------------|-----------|-------------|
| `output` | VIDEO | خروجی ویدیوی تولید شده. |
| `video_id` | STRING | شناسه یکتای ویدیوی تولید شده. |
| `duration` | STRING | اطلاعات مدت زمان ویدیوی تولید شده. |
