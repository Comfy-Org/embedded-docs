# ترجمة وثيقة عقدة ByteDance Text to Video

تقوم عقدة ByteDance Text to Video بإنشاء مقاطع فيديو باستخدام نماذج ByteDance عبر واجهة برمجة تطبيقات (API) بناءً على النصوص التوجيهية. تأخذ العقدة وصفًا نصيًا وإعدادات فيديو متنوعة كمدخلات، ثم تنشئ مقطع فيديو يطابق المواصفات المقدمة. تتعامل العقدة مع الاتصال بواجهة API وتُعيد مقطع الفيديو المُنشأ كمخرجات.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|---------|---------------|--------|---------|-------|
| `model` | STRING | نعم | `"seedance-1-5-pro-251215"`<br>`"seedance-1-0-pro-250528"`<br>`"seedance-1-0-lite-t2v-250428"`<br>`"seedance-1-0-pro-fast-251015"` | نموذج ByteDance المستخدم في التوليد (الافتراضي: `"seedance-1-0-pro-fast-251015"`). |
| `prompt` | STRING | نعم | - | النص التوجيهي المستخدم لتوليد الفيديو. |
| `resolution` | STRING | نعم | `"480p"`<br>`"720p"`<br>`"1080p"` | دقة الفيديو الناتج. |
| `aspect_ratio` | STRING | نعم | `"16:9"`<br>`"4:3"`<br>`"1:1"`<br>`"3:4"`<br>`"9:16"`<br>`"21:9"` | نسبة العرض إلى الارتفاع للفيديو الناتج. |
| `duration` | INT | نعم | 3 إلى 12 | مدة الفيديو الناتج بالثواني (الافتراضي: 5). |
| `seed` | INT | لا | 0 إلى 2147483647 | البذرة المستخدمة في التوليد (الافتراضي: 0). |
| `camera_fixed` | BOOLEAN | لا | - | يحدد ما إذا كان سيتم تثبيت الكاميرا. تُلحق المنصة تعليمة بتثبيت الكاميرا بنصك التوجيهي، ولكنها لا تضمن التأثير الفعلي (الافتراضي: False). |
| `watermark` | BOOLEAN | لا | - | ما إذا كان سيتم إضافة علامة مائية "مُولد بالذكاء الاصطناعي" إلى الفيديو (الافتراضي: False). |
| `generate_audio` | BOOLEAN | لا | - | يتم تجاهل هذا المعامل لأي نموذج باستثناء `seedance-1-5-pro-251215` (الافتراضي: False). |

**قيود المعاملات:**

- يجب أن يحتوي المعامل `prompt` على حرف واحد على الأقل بعد إزالة المسافات البيضاء.
- لا يمكن أن يحتوي المعامل `prompt` على معاملات النص التالية: "resolution"، "ratio"، "duration"، "seed"، "camerafixed"، "watermark".
- يقتصر المعامل `duration` على القيم بين 3 و12 ثانية. بالنسبة لنموذج `seedance-1-5-pro-251215`، فإن الحد الأدنى للمدة المدعومة هو 4 ثوانٍ.
- يقبل المعامل `seed` القيم من 0 إلى 2,147,483,647.
- يكون للمعامل `generate_audio` تأثير فقط عندما يكون `model` مضبوطًا على `seedance-1-5-pro-251215`؛ ويتم تجاهله لجميع النماذج الأخرى.
