Вот перевод документации на русский язык:

Узел ByteDance Text to Video генерирует видео с использованием моделей ByteDance через API на основе текстовых подсказок. На вход подается текстовое описание и различные настройки видео, после чего создается видео, соответствующее указанным параметрам. Узел обрабатывает взаимодействие с API и возвращает сгенерированное видео на выходе.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `model` | STRING | Да | `"seedance-1-5-pro-251215"`<br>`"seedance-1-0-pro-250528"`<br>`"seedance-1-0-lite-t2v-250428"`<br>`"seedance-1-0-pro-fast-251015"` | Модель ByteDance, используемая для генерации (по умолчанию: `"seedance-1-0-pro-fast-251015"`). |
| `prompt` | STRING | Да | - | Текстовая подсказка, используемая для генерации видео. |
| `resolution` | STRING | Да | `"480p"`<br>`"720p"`<br>`"1080p"` | Разрешение выходного видео. |
| `aspect_ratio` | STRING | Да | `"16:9"`<br>`"4:3"`<br>`"1:1"`<br>`"3:4"`<br>`"9:16"`<br>`"21:9"` | Соотношение сторон выходного видео. |
| `duration` | INT | Да | от 3 до 12 | Длительность выходного видео в секундах (по умолчанию: 5). |
| `seed` | INT | Нет | от 0 до 2147483647 | Зерно для генерации (по умолчанию: 0). |
| `camera_fixed` | BOOLEAN | Нет | - | Указывает, следует ли фиксировать камеру. Платформа добавляет инструкцию по фиксации камеры к вашей подсказке, но не гарантирует фактический эффект (по умолчанию: False). |
| `watermark` | BOOLEAN | Нет | - | Добавлять ли водяной знак "Создано AI" к видео (по умолчанию: False). |
| `generate_audio` | BOOLEAN | Нет | - | Этот параметр игнорируется для всех моделей, кроме `seedance-1-5-pro-251215` (по умолчанию: False). |

**Ограничения параметров:**

- Параметр `prompt` должен содержать как минимум 1 символ после удаления пробелов.
- Параметр `prompt` не может содержать следующие текстовые параметры: "resolution", "ratio", "duration", "seed", "camerafixed", "watermark".
- Параметр `duration` ограничен значениями от 3 до 12 секунд. Для модели `seedance-1-5-pro-251215` минимальная поддерживаемая длительность составляет 4 секунды.
- Параметр `seed` принимает значения от 0 до 2 147 483 647.
- Параметр `generate_audio` действует только при установке модели `seedance-1-5-pro-251215`; для всех остальных моделей он игнорируется.

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `output` | VIDEO | Сгенерированный видеофайл |
