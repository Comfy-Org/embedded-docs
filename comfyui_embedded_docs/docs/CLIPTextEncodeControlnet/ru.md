Узел CLIPTextEncodeControlnet обрабатывает текстовый ввод с помощью модели CLIP и объединяет его с существующими данными кондиционирования для создания расширенного вывода кондиционирования для приложений ControlNet. Он токенизирует входной текст, кодирует его через модель CLIP и добавляет полученные эмбеддинги к предоставленным данным кондиционирования в качестве параметров кросс-внимания для ControlNet.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `clip` | CLIP | Да | - | Модель CLIP, используемая для токенизации и кодирования текста |
| `conditioning` | CONDITIONING | Да | - | Существующие данные кондиционирования, которые будут дополнены параметрами ControlNet |
| `text` | STRING | Да | - | Текстовый ввод для обработки моделью CLIP. Поддерживает многострочный текст и динамические промпты |

**Примечание:** Для корректной работы этого узла требуются все три входа (`clip`, `conditioning` и `text`). Вход `text` поддерживает динамические промпты и многострочный текст для гибкой обработки текста.

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `CONDITIONING` | CONDITIONING | Расширенные данные кондиционирования с добавленными параметрами кросс-внимания ControlNet (`cross_attn_controlnet` и `pooled_output_controlnet`), полученными в результате кодирования текста CLIP |
