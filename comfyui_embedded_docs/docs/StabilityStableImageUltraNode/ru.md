Генерирует изображения синхронно на основе текстового запроса и разрешения. Этот узел создает изображения с использованием модели Stable Image Ultra от Stability AI, обрабатывая ваш текстовый запрос и генерируя соответствующее изображение с заданным соотношением сторон и стилем.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `prompt` | STRING | Да | - | Что вы хотите видеть на выходном изображении. Четкий, описательный запрос, который ясно определяет элементы, цвета и объекты, приведет к лучшим результатам. Для контроля веса заданного слова используйте формат `(слово:вес)`, где `слово` — это слово, вес которого вы хотите контролировать, а `вес` — значение от 0 до 1. Например: `Небо было ярким (синим:0.3) и (зеленым:0.8)` передаст небо, которое было синим и зеленым, но больше зеленым, чем синим. |
| `aspect_ratio` | COMBO | Да | `"1:1"`<br>`"16:9"`<br>`"21:9"`<br>`"2:3"`<br>`"3:2"`<br>`"4:5"`<br>`"5:4"`<br>`"9:16"`<br>`"9:21"` | Соотношение сторон генерируемого изображения (по умолчанию: "1:1"). |
| `style_preset` | COMBO | Нет | `"3d-model"`<br>`"analog-film"`<br>`"anime"`<br>`"cinematic"`<br>`"comic-book"`<br>`"digital-art"`<br>`"enhance"`<br>`"fantasy-art"`<br>`"isometric"`<br>`"line-art"`<br>`"low-poly"`<br>`"modeling-compound"`<br>`"neon-punk"`<br>`"origami"`<br>`"photographic"`<br>`"pixel-art"`<br>`"tile-texture"` | Желаемый стиль генерируемого изображения (необязательно). Выберите "None", чтобы не применять предустановку стиля. |
| `seed` | INT | Да | 0 - 4294967294 | Случайное зерно, используемое для создания шума. |
| `image` | IMAGE | Нет | - | Входное изображение для генерации изображения на основе изображения (необязательно). |
| `negative_prompt` | STRING | Нет | - | Текст, описывающий то, что вы не хотите видеть на выходном изображении. Это расширенная функция. |
| `image_denoise` | FLOAT | Нет | 0.0 - 1.0 | Уровень шумоподавления входного изображения; 0.0 дает изображение, идентичное входному, 1.0 — как если бы изображение не было предоставлено вообще (по умолчанию: 0.5). |

**Примечание:** Если входное изображение не предоставлено, параметр `image_denoise` автоматически отключается и игнорируется.

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `output` | IMAGE | Сгенерированное изображение на основе входных параметров. |
