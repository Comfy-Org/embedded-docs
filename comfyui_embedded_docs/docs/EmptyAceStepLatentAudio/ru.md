Вот перевод документации на русский язык:

Узел EmptyAceStepLatentAudio создает пустые латентные аудиосэмплы заданной длительности. Он генерирует пакет тихих аудиолатентов, заполненных нулями, длина которых вычисляется на основе входных секунд и параметров обработки аудио. Этот узел полезен для инициализации рабочих процессов обработки аудио, требующих латентных представлений.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `seconds` | FLOAT | Да | 1.0 - 1000.0 | Длительность аудио в секундах (по умолчанию: 120.0) |
| `batch_size` | INT | Да | 1 - 4096 | Количество латентных изображений в пакете (по умолчанию: 1) |

## Выходы

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `output` | LATENT | Возвращает пустые латентные аудиосэмплы, заполненные нулями. Выходные данные содержат тензор `samples` и поле `type` со значением "audio". |
