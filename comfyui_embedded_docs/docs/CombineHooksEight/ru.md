Узел **Combine Hooks [8]** объединяет до восьми различных групп хуков в одну комбинированную группу. Он принимает несколько входных хуков и объединяет их с помощью функциональности объединения хуков ComfyUI. Это позволяет консолидировать несколько конфигураций хуков для оптимизированной обработки в продвинутых рабочих процессах.

## Входные параметры

| Параметр | Тип данных | Тип входа | По умолчанию | Диапазон | Описание |
|-----------|-----------|------------|---------|-------|-------------|
| `hooks_A` | HOOKS | опциональный | None | - | Первая группа хуков для объединения |
| `hooks_B` | HOOKS | опциональный | None | - | Вторая группа хуков для объединения |
| `hooks_C` | HOOKS | опциональный | None | - | Третья группа хуков для объединения |
| `hooks_D` | HOOKS | опциональный | None | - | Четвертая группа хуков для объединения |
| `hooks_E` | HOOKS | опциональный | None | - | Пятая группа хуков для объединения |
| `hooks_F` | HOOKS | опциональный | None | - | Шестая группа хуков для объединения |
| `hooks_G` | HOOKS | опциональный | None | - | Седьмая группа хуков для объединения |
| `hooks_H` | HOOKS | опциональный | None | - | Восьмая группа хуков для объединения |

**Примечание:** Все входные параметры являются опциональными. Узел объединяет только те группы хуков, которые были предоставлены, игнорируя пустые. Вы можете передать от одной до восьми групп хуков для объединения.

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `HOOKS` | HOOKS | Единая комбинированная группа хуков, содержащая все предоставленные конфигурации хуков |
