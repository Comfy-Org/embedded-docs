Вот перевод документации на русский язык:

Этот узел преобразует сжатое аудиопредставление (латентные сэмплы) обратно в аудиоволновую форму с помощью вариационного автокодировщика (VAE). Он обрабатывает данные небольшими перекрывающимися участками (тайлами) для управления использованием памяти, что делает его подходящим для обработки длинных аудиопоследовательностей.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `samples` | LATENT | Да | Н/Д | Сжатое латентное представление аудио для декодирования. |
| `vae` | VAE | Да | Н/Д | Модель вариационного автокодировщика, используемая для выполнения декодирования. |
| `tile_size` | INT | Да | от 32 до 8192 | Размер каждого обрабатываемого тайла. Аудио декодируется участками такой длины для экономии памяти (по умолчанию: 512). |
| `overlap` | INT | Да | от 0 до 1024 | Количество сэмплов, на которое перекрываются соседние тайлы. Это помогает уменьшить артефакты на границах между тайлами (по умолчанию: 64). |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `output` | AUDIO | Декодированная аудиоволновая форма. |
