### مرور کلی

گره تولید ویدیو از متن Vidu2، یک ویدیو بر اساس توضیحات متنی ایجاد می‌کند. این گره برای تولید محتوای ویدیویی بر اساس prompt شما به یک API خارجی متصل می‌شود و به شما امکان کنترل طول، سبک بصری و فرمت ویدیو را می‌دهد.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|---------|----------|--------|--------|---------|
| `مدل` | COMBO | بله | `"viduq2"` | مدل هوش مصنوعی مورد استفاده برای تولید ویدیو. در حال حاضر، تنها یک مدل در دسترس است. |
| `پرامپت` | STRING | بله | - | توضیحات متنی برای تولید ویدیو، با حداکثر طول ۲۰۰۰ کاراکتر. |
| `مدت زمان` | INT | خیر | ۱ تا ۱۰ | طول ویدیوی تولید شده بر حسب ثانیه. مقدار را می‌توان با استفاده از یک لغزنده تنظیم کرد (پیش‌فرض: ۵). |
| `seed` | INT | خیر | ۰ تا ۲۱۴۷۴۸۳۶۴۷ | عددی برای کنترل تصادفی بودن تولید، که امکان دستیابی به نتایج قابل تکرار را فراهم می‌کند. می‌توان آن را پس از تولید کنترل کرد (پیش‌فرض: ۱). |
| `نسبت تصویر` | COMBO | خیر | `"16:9"`<br>`"9:16"`<br>`"3:4"`<br>`"4:3"`<br>`"1:1"` | نسبت تناسب بین عرض و ارتفاع ویدیو. |
| `رزولوشن` | COMBO | خیر | `"720p"`<br>`"1080p"` | ابعاد پیکسلی ویدیوی تولید شده. این یک پارامتر پیشرفته است. |
| `موسیقی پس‌زمینه` | BOOLEAN | خیر | - | آیا به ویدیوی تولید شده موسیقی پس‌زمینه اضافه شود یا خیر (پیش‌فرض: False). این یک پارامتر پیشرفته است. |
