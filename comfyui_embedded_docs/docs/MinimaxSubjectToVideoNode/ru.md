Вот перевод документации на русский язык:

Генерирует видео синхронно на основе изображения объекта и текстового запроса с использованием API MiniMax. Этот узел принимает изображение объекта и описание для создания видео, которое анимирует или показывает этот объект в соответствии с запросом.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `subject` | IMAGE | Да | - | Изображение объекта для использования в качестве референса при генерации видео |
| `prompt_text` | STRING | Да | - | Текстовый запрос для управления генерацией видео (по умолчанию: пустая строка) |
| `model` | COMBO | Нет | "S2V-01" | Модель для генерации видео (по умолчанию: "S2V-01") |
| `seed` | INT | Нет | от 0 до 18446744073709551615 | Случайное зерно, используемое для создания шума (по умолчанию: 0) |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `output` | VIDEO | Сгенерированное видео на основе входного изображения объекта и запроса |
