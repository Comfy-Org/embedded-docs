### نمای کلی

این گره با استفاده از API مینی‌مکس (MiniMax)، بر اساس یک تصویر سوژه و یک متن راهنما، به‌صورت همزمان ویدیو تولید می‌کند. این گره یک تصویر از سوژه و یک توضیح دریافت کرده و ویدیویی می‌سازد که سوژه را مطابق با متن راهنما به حرکت درآورده یا به نمایش می‌گذارد.

### ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|---------|----------|-------|--------|---------|
| `subject` | IMAGE | بله | - | تصویر سوژه برای ارجاع در تولید ویدیو |
| `prompt_text` | STRING | بله | - | متن راهنما برای هدایت تولید ویدیو (پیش‌فرض: رشته خالی) |
| `model` | COMBO | خیر | "S2V-01" | مدل مورد استفاده برای تولید ویدیو (پیش‌فرض: "S2V-01") |
| `seed` | INT | خیر | 0 تا 18446744073709551615 | دانه تصادفی برای ایجاد نویز (پیش‌فرض: 0) |

### خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-----------|----------|---------|
| `output` | VIDEO | ویدیوی تولید شده بر اساس تصویر سوژه و متن راهنما |
