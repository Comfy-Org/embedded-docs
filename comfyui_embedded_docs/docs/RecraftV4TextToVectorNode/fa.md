گره «تبدیل متن به بردار Recraft V4» تصاویر گرافیک برداری مقیاس‌پذیر (SVG) را بر اساس توضیحات متنی تولید می‌کند. این گره از طریق یک API خارجی به مدل‌های Recraft V4 یا Recraft V4 Pro برای تولید تصویر متصل می‌شود. خروجی این گره یک یا چند تصویر SVG بر اساس prompt شما است.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|---------|----------|--------|--------|----------|
| `پرامپت` | STRING | بله | نامشخص | prompt برای تولید تصویر. حداکثر ۱۰,۰۰۰ کاراکتر. |
| `پرامپت منفی` | STRING | خیر | نامشخص | توضیحات متنی اختیاری از عناصر نامطلوب در تصویر. |
| `مدل` | COMBO | بله | `"recraftv4"`<br>`"recraftv4_pro"` | مدل مورد استفاده برای تولید. انتخاب مدل، گزینه‌های موجود `size` را تغییر می‌دهد. |
| `size` | COMBO | بله | برای `recraftv4`: `"1024x1024"`, `"1152x896"`, `"896x1152"`, `"1216x832"`, `"832x1216"`, `"1344x768"`, `"768x1344"`, `"1536x640"`, `"640x1536"`<br>برای `recraftv4_pro`: `"2048x2048"`, `"2304x1792"`, `"1792x2304"`, `"2432x1664"`, `"1664x2432"`, `"2688x1536"`, `"1536x2688"`, `"3072x1280"`, `"1280x3072"` | اندازه تصویر تولید شده. گزینه‌های موجود به `مدل` انتخاب شده بستگی دارد. مقدار پیش‌فرض برای `recraftv4` برابر `"1024x1024"` و برای `recraftv4_pro` برابر `"2048x2048"` است. |
| `تعداد` | INT | بله | ۱ تا ۶ | تعداد تصاویر برای تولید (پیش‌فرض: ۱). |
| `بذر` | INT | بله | ۰ تا ۱۸۴۴۶۷۴۴۰۷۳۷۰۹۵۵۱۶۱۵ | seed برای تعیین اینکه گره دوباره اجرا شود؛ نتایج واقعی بدون توجه به seed غیرقطعی هستند. |
| `کنترل‌های Recraft` | CUSTOM | خیر | نامشخص | کنترل‌های اضافی اختیاری بر روی تولید از طریق گره «کنترل‌های Recraft». |

**نکته:** پارامتر `size` یک ورودی پویا است که گزینه‌های موجود آن بر اساس `model` انتخاب شده تغییر می‌کند. مقدار `seed` نتایج قابل تکرار از API خارجی را تضمین نمی‌کند.
