Узел Recraft V4 Text to Vector генерирует иллюстрации в формате Scalable Vector Graphics (SVG) на основе текстового описания. Он подключается к внешнему API для использования модели Recraft V4 или Recraft V4 Pro для генерации изображений. Узел выводит одно или несколько SVG-изображений в соответствии с вашим запросом.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `prompt` | STRING | Да | Н/Д | Запрос для генерации изображения. Максимум 10 000 символов. |
| `negative_prompt` | STRING | Нет | Н/Д | Необязательное текстовое описание нежелательных элементов на изображении. |
| `model` | COMBO | Да | `"recraftv4"`<br>`"recraftv4_pro"` | Модель, используемая для генерации. Выбор модели изменяет доступные параметры `size`. |
| `size` | COMBO | Да | Для `recraftv4`: `"1024x1024"`, `"1152x896"`, `"896x1152"`, `"1216x832"`, `"832x1216"`, `"1344x768"`, `"768x1344"`, `"1536x640"`, `"640x1536"`<br>Для `recraftv4_pro`: `"2048x2048"`, `"2304x1792"`, `"1792x2304"`, `"2432x1664"`, `"1664x2432"`, `"2688x1536"`, `"1536x2688"`, `"3072x1280"`, `"1280x3072"` | Размер генерируемого изображения. Доступные варианты зависят от выбранной `model`. По умолчанию: `"1024x1024"` для `recraftv4` и `"2048x2048"` для `recraftv4_pro`. |
| `n` | INT | Да | от 1 до 6 | Количество генерируемых изображений (по умолчанию: 1). |
| `seed` | INT | Да | от 0 до 18446744073709551615 | Зерно для определения необходимости повторного запуска узла; фактические результаты недетерминированы независимо от зерна. |
| `recraft_controls` | CUSTOM | Нет | Н/Д | Необязательные дополнительные элементы управления генерацией через узел Recraft Controls. |

**Примечание:** Параметр `size` является динамическим входным параметром, доступные варианты которого изменяются в зависимости от выбранной `model`. Значение `seed` не гарантирует воспроизводимых результатов от внешнего API.

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `output` | SVG | Сгенерированное изображение (изображения) в формате Scalable Vector Graphics (SVG). |
