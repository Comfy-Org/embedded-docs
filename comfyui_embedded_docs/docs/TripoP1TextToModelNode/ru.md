## Обзор

Этот узел генерирует 3D-модель на основе текстового описания с использованием API Tripo P1. Он оптимизирован для создания низкополигональных сеток, готовых к использованию в играх, со стабильной топологией, что делает его подходящим для приложений реального времени.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `prompt` | STRING | Да | До 1024 символов | Текстовое описание 3D-модели, которую вы хотите сгенерировать. |
| `negative_prompt` | STRING | Нет | До 255 символов | Текстовое описание того, чего вы не хотите видеть в сгенерированной модели. |
| `output_mode` | DICT | Да | См. описание | Управляет качеством выходной модели и настройками текстуры. Этот параметр представляет собой словарь со следующими ключами:<br><br>`texture_quality`: STRING, Диапазон: `"standard"`<br>`pbr`: BOOLEAN, по умолчанию: True<br>`texture`: BOOLEAN, по умолчанию: True<br>`subdivision`: INT, по умолчанию: 0, Диапазон: от 0 до 2<br>`texture_size`: INT, по умолчанию: 2048, Диапазон: от 512 до 4096 (должно быть степенью двойки)<br>`texture_format`: STRING, Диапазон: `"png"`<br>`texture_clean`: BOOLEAN, по умолчанию: False<br>`texture_seamless`: BOOLEAN, по умолчанию: False<br><br>По умолчанию: `{"texture_quality": "standard", "pbr": True, "texture": True, "subdivision": 0, "texture_size": 2048, "texture_format": "png", "texture_clean": False, "texture_seamless": False}` |
| `image_seed` | INT | Нет |  | Значение зерна для генерации изображения, используемое для управления случайностью. По умолчанию: 42. |
| `face_limit` | INT | Нет |  | Максимальное количество полигонов для сгенерированной сетки. Значение -1 означает отсутствие ограничения. По умолчанию: -1. |
| `model_seed` | INT | Нет |  | Значение зерна для генерации модели, используемое для управления случайностью. |
| `auto_size` | BOOLEAN | Нет |  | Если включено, узел автоматически определит оптимальный размер модели. По умолчанию: False. |
| `export_uv` | BOOLEAN | Нет |  | Если включено, модель будет содержать UV-координаты для наложения текстур. По умолчанию: True. |
| `compress_geometry` | BOOLEAN | Нет |  | Если включено, геометрия будет сжата для уменьшения размера файла. По умолчанию: False. |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `model_file` | STRING | Путь к файлу сгенерированной 3D-модели (только для обратной совместимости). |
| `model task_id` | MODEL_TASK_ID | Уникальный идентификатор задачи для запроса на генерацию модели. |
| `GLB` | FILE3DGLB | Сгенерированная 3D-модель в формате GLB. |
