## نظرة عامة

تُنشئ هذه العقدة نموذجًا ثلاثي الأبعاد من وصف نصي باستخدام واجهة برمجة تطبيقات Tripo P1. وهي مُحسَّنة لإنشاء شبكات منخفضة المضلعات جاهزة للألعاب ذات طوبولوجيا مستقرة، مما يجعلها مناسبة للتطبيقات في الوقت الفعلي.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|-----------|-----------|----------|-------|-------------|
| `prompt` | STRING | نعم | حتى 1024 حرفًا | الوصف النصي للنموذج ثلاثي الأبعاد الذي ترغب في إنشائه. |
| `negative_prompt` | STRING | لا | حتى 255 حرفًا | وصف نصي لما لا ترغب في ظهوره في النموذج المُنشأ. |
| `output_mode` | DICT | نعم | انظر الوصف | يتحكم في جودة النموذج المُخرج وإعدادات النسيج. هذا المعامل هو قاموس بالمفاتيح التالية:<br><br>`texture_quality`: STRING، النطاق: `"standard"`<br>`pbr`: BOOLEAN، القيمة الافتراضية: True<br>`texture`: BOOLEAN، القيمة الافتراضية: True<br>`subdivision`: INT، القيمة الافتراضية: 0، النطاق: 0 إلى 2<br>`texture_size`: INT، القيمة الافتراضية: 2048، النطاق: 512 إلى 4096 (يجب أن يكون من مضاعفات العدد 2)<br>`texture_format`: STRING، النطاق: `"png"`<br>`texture_clean`: BOOLEAN، القيمة الافتراضية: False<br>`texture_seamless`: BOOLEAN، القيمة الافتراضية: False<br><br>القيمة الافتراضية: `{"texture_quality": "standard", "pbr": True, "texture": True, "subdivision": 0, "texture_size": 2048, "texture_format": "png", "texture_clean": False, "texture_seamless": False}` |
| `image_seed` | INT | لا |  | قيمة بذرة لتوليد الصورة، تُستخدم للتحكم في العشوائية. القيمة الافتراضية: 42. |
| `face_limit` | INT | لا |  | الحد الأقصى لعدد الأسطح للشبكة المُنشأة. القيمة -1 تعني عدم وجود حد. القيمة الافتراضية: -1. |
| `model_seed` | INT | لا |  | قيمة بذرة لتوليد النموذج، تُستخدم للتحكم في العشوائية. |
| `auto_size` | BOOLEAN | لا |  | إذا تم التفعيل، ستحدد العقدة تلقائيًا الحجم الأمثل للنموذج. القيمة الافتراضية: False. |
| `export_uv` | BOOLEAN | لا |  | إذا تم التفعيل، سيتضمن النموذج إحداثيات UV لتعيين النسيج. القيمة الافتراضية: True. |
| `compress_geometry` | BOOLEAN | لا |  | إذا تم التفعيل، سيتم ضغط الهندسة لتقليل حجم الملف. القيمة الافتراضية: False. |
