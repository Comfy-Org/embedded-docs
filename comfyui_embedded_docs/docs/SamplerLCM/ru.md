Вот перевод документации на русский язык:

Узел SamplerLCM предоставляет семплер LCM (Latent Consistency Model) с настраиваемыми параметрами шума на каждом шаге. Он позволяет контролировать шум, применяемый на каждом этапе сэмплирования, обеспечивая тонкую настройку процесса сэмплирования.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `s_noise` | FLOAT | Да | 0.0 до 64.0 (шаг: 0.01) | Множитель шума на первом шаге. Значение 1.0 соответствует масштабу шума, используемому при обучении модели. (по умолчанию: 1.0) |
| `s_noise_end` | FLOAT | Да | 0.0 до 64.0 (шаг: 0.01) | Множитель шума на последнем шаге. Установите равным `s_noise` для постоянного графика шума. (по умолчанию: 1.0) |
| `noise_clip_std` | FLOAT | Да | 0.0 до 10.0 (шаг: 0.01) | Ограничивает шум на каждом шаге в пределах +/- N стандартных отклонений. Значение 0 отключает ограничение. (по умолчанию: 0.0) |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `SAMPLER` | SAMPLER | Настроенный объект семплера LCM, готовый к использованию в рабочем процессе сэмплирования. |
