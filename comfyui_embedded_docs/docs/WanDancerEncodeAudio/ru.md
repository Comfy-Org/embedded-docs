## Обзор

Этот узел обрабатывает аудиовход для извлечения признаков, которые можно использовать для управления моделью генерации видео. Он анализирует аудио для определения темпа, битов и других музыкальных характеристик, затем упаковывает эту информацию в формат, подходящий для conditioning модели видео, что позволяет синхронизировать сгенерированное видео с аудио.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `audio` | AUDIO | Да | - | Аудиовход для анализа и кодирования. |
| `video_frames` | INT | Да | Мин: 1, Макс: 268435456 (MAX_RESOLUTION), Шаг: 4 | Количество кадров в целевом видео. Используется для расчета частоты кадров для синхронизации (по умолчанию: 149). |
| `audio_inject_scale` | FLOAT | Да | Мин: 0.0, Макс: 10.0, Шаг: 0.01 | Масштаб для аудиопризнаков при внедрении в модель видео (по умолчанию: 1.0). |

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `fps_string` | AUDIO_ENCODER_OUTPUT | Словарь, содержащий обработанные аудиопризнаки, рассчитанную частоту кадров (fps) и масштаб внедрения аудио. Этот выход используется для conditioning модели генерации видео. |
| `fps_string` | STRING | Текстовая строка, описывающая рассчитанную частоту кадров (fps) на основе длины аудио и количества кадров видео. Эта строка предназначена для использования в промпте модели видео. |
