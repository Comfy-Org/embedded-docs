## نمای کلی

این گره یک ورودی صوتی را پردازش می‌کند تا ویژگی‌هایی را استخراج کند که می‌توان از آنها برای هدایت یک مدل تولید ویدیو استفاده کرد. این گره صدا را برای تشخیص تمپو، ضرب‌آهنگ و سایر ویژگی‌های موسیقایی تحلیل می‌کند و سپس این اطلاعات را در قالبی مناسب برای شرطی‌سازی مدل ویدیو بسته‌بندی می‌کند و امکان همگام‌سازی ویدیوی تولید شده با صدا را فراهم می‌سازد.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `audio` | AUDIO | بله | - | ورودی صوتی که باید تحلیل و رمزگذاری شود. |
| `video_frames` | INT | بله | حداقل: ۱، حداکثر: ۲۶۸۴۳۵۴۵۶ (MAX_RESOLUTION)، گام: ۴ | تعداد فریم‌های ویدیوی هدف. برای محاسبه نرخ فریم جهت همگام‌سازی استفاده می‌شود (پیش‌فرض: ۱۴۹). |
| `audio_inject_scale` | FLOAT | بله | حداقل: ۰.۰، حداکثر: ۱۰.۰، گام: ۰.۰۱ | مقیاس ویژگی‌های صوتی هنگام تزریق به مدل ویدیو (پیش‌فرض: ۱.۰). |

## خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-------------|-----------|-------------|
| `audio_encoder_output` | AUDIO_ENCODER_OUTPUT | یک دیکشنری شامل ویژگی‌های پردازش‌شده صوتی، نرخ فریم محاسبه‌شده (fps) و مقیاس تزریق صدا. این خروجی برای شرطی‌سازی مدل تولید ویدیو استفاده می‌شود. |
| `fps_string` | STRING | یک رشته متنی که نرخ فریم محاسبه‌شده (fps) را بر اساس طول صدا و تعداد فریم‌های ویدیو توصیف می‌کند. این رشته برای استفاده در پرامپت مدل ویدیو در نظر گرفته شده است. |
