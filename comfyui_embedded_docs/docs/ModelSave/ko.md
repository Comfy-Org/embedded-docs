# ModelSave 노드

ModelSave 노드는 학습되거나 수정된 모델을 컴퓨터 저장소에 저장합니다. 모델을 입력으로 받아 지정된 파일 이름으로 파일에 기록합니다. 이를 통해 작업을 보존하고 향후 프로젝트에서 모델을 재사용할 수 있습니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `모델` | MODEL | 예 | - | 디스크에 저장할 모델 |
| `파일명 접두사` | STRING | 예 | - | 저장할 모델 파일의 파일 이름 및 경로 접두사 (기본값: "diffusion_models/ComfyUI") |
| `prompt` | PROMPT | 아니요 | - | 워크플로우 프롬프트 정보 (자동으로 제공됨) |
| `extra_pnginfo` | EXTRA_PNGINFO | 아니요 | - | 추가 워크플로우 메타데이터 (자동으로 제공됨) |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| *없음* | - | 이 노드는 출력 값을 반환하지 않습니다 |
