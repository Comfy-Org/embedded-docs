Вот перевод документации на русский язык:

Узел ModelSave сохраняет обученные или измененные модели в память вашего компьютера. Он принимает модель на вход и записывает ее в файл с указанным вами именем. Это позволяет сохранять результаты вашей работы и повторно использовать модели в будущих проектах.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `model` | MODEL | Да | - | Модель, которую необходимо сохранить на диск |
| `filename_prefix` | STRING | Да | - | Префикс имени файла и пути для сохраняемого файла модели (по умолчанию: "diffusion_models/ComfyUI") |
| `prompt` | PROMPT | Нет | - | Информация о подсказке рабочего процесса (предоставляется автоматически) |
| `extra_pnginfo` | EXTRA_PNGINFO | Нет | - | Дополнительные метаданные рабочего процесса (предоставляются автоматически) |

## Выходы

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| *Нет* | - | Этот узел не возвращает никаких выходных значений |
