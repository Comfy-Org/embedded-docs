بالتأكيد، إليك الترجمة العربية للوثيقة التقنية الخاصة بعقدة `ModelSave` في ComfyUI، مع الالتزام التام بقواعد الترجمة المحددة:

عقدة حفظ النموذج (ModelSave) تقوم بحفظ النماذج المُدرّبة أو المُعدّلة على وحدة التخزين في جهاز الكمبيوتر الخاص بك. تستقبل العقدة نموذجًا كمدخل، وتقوم بكتابته في ملف باسم الملف الذي تحدده. يتيح لك ذلك الحفاظ على عملك وإعادة استخدام النماذج في مشاريع مستقبلية.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|-----------|-----------|----------|-------|-------------|
| `model` | MODEL | نعم | - | النموذج المراد حفظه على القرص الصلب |
| `filename_prefix` | STRING | نعم | - | بادئة اسم الملف والمسار لملف النموذج المحفوظ (الافتراضي: "diffusion_models/ComfyUI") |
| `prompt` | PROMPT | لا | - | معلومات تعليمات سير العمل (يتم توفيرها تلقائيًا) |
| `extra_pnginfo` | EXTRA_PNGINFO | لا | - | بيانات وصفية إضافية لسير العمل (يتم توفيرها تلقائيًا) |
