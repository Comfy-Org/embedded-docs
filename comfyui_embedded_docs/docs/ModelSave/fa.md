گره ModelSave مدل‌های آموزش‌دیده یا تغییر یافته را در حافظه رایانه شما ذخیره می‌کند. این گره یک مدل را به عنوان ورودی دریافت کرده و آن را با نام فایل مشخص‌شده توسط شما در یک فایل ذخیره می‌کند. این کار به شما امکان می‌دهد کار خود را حفظ کرده و مدل‌ها را در پروژه‌های آینده مجدداً استفاده کنید.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|---------|----------|--------|--------|---------|
| `model` | MODEL | بله | - | مدلی که باید روی دیسک ذخیره شود |
| `filename_prefix` | STRING | بله | - | پیشوند مسیر و نام فایل برای فایل مدل ذخیره‌شده (پیش‌فرض: "diffusion_models/ComfyUI") |
| `prompt` | PROMPT | خیر | - | اطلاعات راهنمای گردش کار (به‌طور خودکار ارائه می‌شود) |
| `extra_pnginfo` | EXTRA_PNGINFO | خیر | - | فراداده اضافی گردش کار (به‌طور خودکار ارائه می‌شود) |
