# GeminiVideoOmni

با استفاده از مدل Gemini Omni Flash گوگل، یک ویدیو همراه با صدا از روی یک提示 متنی تولید کنید. برای راهنمایی یا ویرایش نتیجه، می‌توانید به‌صورت اختیاری تصاویر و/یا ویدیوهای مرجع ارائه دهید. طول مورد نظر (۳ تا ۱۰ ثانیه) و نسبت تصویر (۱۶:۹ یا ۹:۱۶) را مستقیماً در提示 خود描述 کنید.

## ورودی‌ها

| پارامتر | توضیحات | نوع داده | الزامی | محدوده |
|-----------|-------------|-----------|----------|-------|
| `model` | مدل ویدیویی Gemini که برای تولید ویدیو استفاده می‌شود. | MODEL | بله | "Omni Flash" |
| `seed` | Seed مشخص می‌کند که آیا گره باید دوباره اجرا شود یا خیر؛ نتایج صرف‌نظر از seed قطعی نیستند (پیش‌فرض: ۴۲). | INT | بله | ۰ تا ۲۱۴۷۴۸۳۶۴۷ |
| `prompt` |提示 متنی که ویدیوی مورد نظر برای تولید را توصیف می‌کند. باید حداقل یک کاراکتر غیرفضایی پس از حذف فاصله‌های ابتدا و انتها داشته باشد. | STRING | بله | حداقل ۱ کاراکتر پس از حذف فاصله‌های خالی |
| `images` | تصاویر مرجع اختیاری برای راهنمایی تولید ویدیو. حداکثر ۱۴ تصویر. | IMAGE | خیر | چندین تصویر مجاز است (حداکثر ۱۴) |
| `videos` | ویدیوهای مرجع اختیاری برای راهنمایی یا ویرایش تولید ویدیو. حداکثر ۳ ویدیو، هر کدام تا ۱۰ ثانیه. | VIDEO | خیر | چندین ویدیو مجاز است (حداکثر ۳، هر کدام حداکثر ۱۰ ثانیه) |
| `temperature` | کنترل تصادفی بودن در تولید (پیش‌فرض: ۱.۰). | FLOAT | خیر | ۰.۰ تا ۲.۰ |
| `top_p` | پارامتر نمونه‌گیری هسته (پیش‌فرض: ۰.۹۵). | FLOAT | خیر | ۰.۰ تا ۱.۰ |

## خروجی‌ها
