# Google Gemini Omni (ویدیو)

با استفاده از مدل Gemini Omni Flash گوگل، یک ویدیو همراه با صدا از روی یک متن راهنما (پرامپت) تولید کنید. به‌صورت اختیاری می‌توانید تصاویر و/یا ویدیوهای مرجع را برای هدایت یا ویرایش نتیجه ارائه دهید. طول مورد نظر (۳ تا ۱۰ ثانیه) و نسبت تصویر (۱۶:۹ یا ۹:۱۶) را مستقیماً در پرامپت توضیح دهید.

## ورودی‌ها

| پارامتر | توضیحات | نوع داده | الزامی | محدوده |
|---------|---------|----------|--------|--------|
| `model` | مدل ویدیویی Gemini که برای تولید ویدیو استفاده می‌شود. | MODEL | بله | "Omni Flash" |
| `seed` | Seed مشخص می‌کند که آیا گره باید دوباره اجرا شود یا خیر؛ نتایج صرف‌نظر از seed غیرقطعی هستند (پیش‌فرض: ۴۲). | INT | بله | ۰ تا ۲۱۴۷۴۸۳۶۴۷ |
| `prompt` | متن راهنما (پرامپت) که ویدیوی مورد نظر را توصیف می‌کند. باید حداقل یک کاراکتر غیرفضایی پس از حذف فاصله‌های ابتدا و انتها داشته باشد. | STRING | بله | حداقل ۱ کاراکتر پس از حذف فاصله‌ها |
| `images` | تصاویر مرجع اختیاری برای هدایت تولید ویدیو. حداکثر ۱۴ تصویر. | IMAGE | خیر | چند تصویر مجاز (حداکثر ۱۴) |
| `videos` | ویدیوهای مرجع اختیاری برای هدایت یا ویرایش تولید ویدیو. حداکثر ۳ ویدیو، هرکدام تا ۱۰ ثانیه. | VIDEO | خیر | چند ویدیو مجاز (حداکثر ۳، هرکدام حداکثر ۱۰ ثانیه) |
| `temperature` | میزان تصادفی بودن در تولید را کنترل می‌کند (پیش‌فرض: ۱.۰). | FLOAT | خیر | ۰.۰ تا ۲.۰ |
| `top_p` | پارامتر نمونه‌گیری هسته (پیش‌فرض: ۰.۹۵). | FLOAT | خیر | ۰.۰ تا ۱.۰ |

## خروجی‌ها
