# Google Gemini Omni (Video)

Этот узел генерирует видео со звуком на основе текстового запроса с использованием модели Google Gemini Omni Flash. Вы можете дополнительно предоставить эталонные изображения или видео для направления или редактирования результата.

## Входные параметры

| Параметр | Описание | Тип данных | Обязательный | Диапазон |
|-----------|-------------|-----------|----------|-------|
| `model` | Модель Gemini для генерации видео. | COMBO | Да | `"Omni Flash"` |
| `seed` | Управляет повторным запуском узла; результаты недетерминированы независимо от значения seed. (по умолчанию: 42) | INT | Да | от 0 до 2147483647 |

**Примечание:** Параметр `model` является динамическим выпадающим списком, который расширяется дополнительными входами при выборе "Omni Flash". Эти дополнительные входы включают:
- `prompt` (STRING, обязательный): Текстовый запрос, описывающий желаемое видео. Укажите желаемую длительность (3-10 секунд) и соотношение сторон (16:9 или 9:16) непосредственно в запросе.
- `images` (IMAGE, необязательный): Эталонные изображения для направления генерации видео. Максимум 14 изображений.
- `videos` (VIDEO, необязательный): Эталонные видео для направления генерации видео. Максимум 3 видео, каждое длительностью не более 10 секунд.
- `temperature` (FLOAT, необязательный, по умолчанию: 1.0): Управляет случайностью генерации.
- `top_p` (FLOAT, необязательный, по умолчанию: 0.95): Управляет семплированием по ядру (nucleus sampling).

## Выходные параметры

| Имя выхода | Описание | Тип данных |
|-------------|-------------|-----------|
| `video` | Сгенерированное видео со звуком. | VIDEO |
| `text` | Текстовый ответ модели, если он есть. | STRING |
