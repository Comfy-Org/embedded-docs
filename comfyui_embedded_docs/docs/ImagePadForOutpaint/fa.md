این گره برای آماده‌سازی تصاویر جهت فرآیند نقاشی بیرونی (outpainting) با افزودن حاشیه در اطراف آن‌ها طراحی شده است. ابعاد تصویر را به‌گونه‌ای تنظیم می‌کند که با الگوریتم‌های نقاشی بیرونی سازگار باشد و تولید نواحی گسترش‌یافته تصویر را فراتر از مرزهای اصلی تسهیل می‌کند.

## ورودی‌ها

| پارامتر | نوع داده | توضیحات |
|-----------|-------------|-------------|
| `image` | `IMAGE` | ورودی `image` تصویر اصلی است که برای نقاشی بیرونی آماده می‌شود و به عنوان پایه عملیات حاشیه‌گذاری عمل می‌کند. |
| `left` | `INT` | میزان حاشیه‌ای که باید به سمت چپ تصویر اضافه شود را مشخص می‌کند و بر ناحیه گسترش‌یافته برای نقاشی بیرونی تأثیر می‌گذارد. |
| `top` | `INT` | میزان حاشیه‌ای که باید به بالای تصویر اضافه شود را تعیین می‌کند و بر گسترش عمودی برای نقاشی بیرونی اثر می‌گذارد. |
| `right` | `INT` | میزان حاشیه‌ای که باید به سمت راست تصویر اضافه شود را تعریف می‌کند و بر گسترش افقی برای نقاشی بیرونی تأثیر می‌گذارد. |
| `bottom` | `INT` | میزان حاشیه‌ای که باید به پایین تصویر اضافه شود را نشان می‌دهد و به گسترش عمودی برای نقاشی بیرونی کمک می‌کند. |
| `feathering` | `INT` | نرمی انتقال بین تصویر اصلی و حاشیه اضافه‌شده را کنترل می‌کند و یکپارچگی بصری را برای نقاشی بیرونی بهبود می‌بخشد. |

## خروجی‌ها

| پارامتر | نوع داده | توضیحات |
|-----------|-------------|-------------|
| `image` | `IMAGE` | خروجی `image` تصویر حاشیه‌گذاری‌شده را نشان می‌دهد که برای فرآیند نقاشی بیرونی آماده است. |
| `mask` | `MASK` | خروجی `mask` نواحی تصویر اصلی و حاشیه اضافه‌شده را مشخص می‌کند که برای هدایت الگوریتم‌های نقاشی بیرونی مفید است. |
