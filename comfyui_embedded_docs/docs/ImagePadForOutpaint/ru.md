Этот узел предназначен для подготовки изображений к процессу аутпейнтинга (достраивания изображения за его пределами) путем добавления отступов вокруг них. Он корректирует размеры изображения, чтобы обеспечить совместимость с алгоритмами аутпейнтинга, облегчая создание расширенных областей изображения за пределами исходных границ.

## Входные параметры

| Параметр | Тип данных | Описание |
|-----------|-------------|-------------|
| `image`   | `IMAGE`     | Входной параметр `image` — это основное изображение, подготавливаемое для аутпейнтинга, которое служит основой для операций добавления отступов. |
| `left`    | `INT`       | Определяет величину отступа, добавляемого к левой стороне изображения, влияя на расширяемую область для аутпейнтинга. |
| `top`     | `INT`       | Определяет величину отступа, добавляемого к верхней части изображения, влияя на вертикальное расширение для аутпейнтинга. |
| `right`   | `INT`       | Определяет величину отступа, добавляемого к правой стороне изображения, влияя на горизонтальное расширение для аутпейнтинга. |
| `bottom`  | `INT`       | Указывает величину отступа, добавляемого к нижней части изображения, внося вклад в вертикальное расширение для аутпейнтинга. |
| `feathering` | `INT` | Управляет плавностью перехода между исходным изображением и добавленным отступом, улучшая визуальную интеграцию для аутпейнтинга. |

## Выходные параметры

| Параметр | Тип данных | Описание |
|-----------|-------------|-------------|
| `image`   | `IMAGE`     | Выходной параметр `image` представляет собой изображение с отступами, готовое к процессу аутпейнтинга. |
| `mask`    | `MASK`      | Выходной параметр `mask` указывает области исходного изображения и добавленных отступов, что полезно для направления алгоритмов аутпейнтинга. |
