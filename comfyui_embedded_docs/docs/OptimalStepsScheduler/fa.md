## ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|---------|----------|-------|--------|----------|
| `model_type` | COMBO | بله | "FLUX"<br>"Wan"<br>"Chroma" | نوع مدل انتشار برای محاسبه سطح نویز |
| `گام‌ها` | INT | بله | 3-1000 | تعداد کل مراحل نمونه‌برداری برای محاسبه (پیش‌فرض: 20) |
| `کاهش نویز` | FLOAT | خیر | 0.0-1.0 | قدرت نویززدایی را کنترل می‌کند که تعداد مراحل مؤثر را تنظیم می‌کند (پیش‌فرض: 1.0) |

**نکته:** هنگامی که `denoise` کمتر از 1.0 تنظیم شود، گره مراحل مؤثر را به صورت `steps * denoise` محاسبه می‌کند. اگر `denoise` برابر 0.0 باشد، گره یک تنسور خالی برمی‌گرداند.
