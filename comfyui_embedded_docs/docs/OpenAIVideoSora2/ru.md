Вот перевод документации на русский язык:

Узел OpenAIVideoSora2 генерирует видео с использованием моделей Sora от OpenAI. Он создает видеоконтент на основе текстовых подсказок и необязательных входных изображений, после чего возвращает сгенерированное видео. Узел поддерживает различную длительность и разрешение видео в зависимости от выбранной модели.

**УВЕДОМЛЕНИЕ ОБ УСТАРЕВАНИИ:** OpenAI прекратит поддержку API Sora v2 в сентябре 2026 года. Этот узел будет удален из ComfyUI в указанное время.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `модель` | COMBO | Да | "sora-2"<br>"sora-2-pro" | Модель OpenAI Sora для генерации видео (по умолчанию: "sora-2") |
| `промпт` | STRING | Да | - | Направляющий текст; может быть пустым, если присутствует входное изображение (по умолчанию: пусто) |
| `размер` | COMBO | Да | "720x1280"<br>"1280x720"<br>"1024x1792"<br>"1792x1024" | Разрешение генерируемого видео (по умолчанию: "1280x720") |
| `длительность` | COMBO | Да | 4<br>8<br>12 | Длительность генерируемого видео в секундах (по умолчанию: 8) |
| `изображение` | IMAGE | Нет | - | Необязательное входное изображение для генерации видео |
| `сид` | INT | Нет | от 0 до 2147483647 | Зерно для определения необходимости повторного запуска узла; фактические результаты недетерминированы независимо от зерна (по умолчанию: 0) |

**Ограничения:**

- Модель "sora-2" поддерживает только разрешения "720x1280" и "1280x720"
- При использовании параметра `image` поддерживается только одно входное изображение
- Результаты недетерминированы независимо от значения зерна

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `output` | VIDEO | Сгенерированное видео |
