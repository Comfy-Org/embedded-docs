### مرور کلی

گره OpenAIVideoSora2 با استفاده از مدل‌های Sora شرکت OpenAI ویدئو تولید می‌کند. این گره بر اساس توضیحات متنی و تصاویر ورودی اختیاری، محتوای ویدئویی ایجاد کرده و خروجی ویدئوی تولیدشده را بازمی‌گرداند. این گره بسته به مدل انتخابی، از مدت‌زمان و وضوح‌های مختلف ویدئو پشتیبانی می‌کند.

**اطلاعیه عدم پشتیبانی:** OpenAI در سپتامبر ۲۰۲۶ سرویس API مدل Sora v2 را متوقف خواهد کرد. این گره نیز در آن زمان از ComfyUI حذف خواهد شد.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|---------|----------|--------|--------|---------|
| `model` | COMBO | بله | "sora-2"<br>"sora-2-pro" | مدل OpenAI Sora برای تولید ویدئو (پیش‌فرض: "sora-2") |
| `prompt` | STRING | بله | - | متن راهنما؛ در صورت وجود تصویر ورودی می‌تواند خالی باشد (پیش‌فرض: خالی) |
| `size` | COMBO | بله | "720x1280"<br>"1280x720"<br>"1024x1792"<br>"1792x1024" | وضوح تصویر برای ویدئوی تولیدشده (پیش‌فرض: "1280x720") |
| `duration` | COMBO | بله | 4<br>8<br>12 | مدت‌زمان ویدئوی تولیدشده بر حسب ثانیه (پیش‌فرض: ۸) |
| `image` | IMAGE | خیر | - | تصویر ورودی اختیاری برای تولید ویدئو |
| `seed` | INT | خیر | ۰ تا ۲۱۴۷۴۸۳۶۴۷ | دانه (seed) برای تعیین اجرای مجدد گره؛ نتایج واقعی بدون توجه به مقدار seed غیرقطعی هستند (پیش‌فرض: ۰) |

**محدودیت‌ها:**

- مدل "sora-2" تنها از وضوح‌های "720x1280" و "1280x720" پشتیبانی می‌کند
- هنگام استفاده از پارامتر image تنها یک تصویر ورودی پشتیبانی می‌شود
- نتایج بدون توجه به مقدار seed غیرقطعی هستند
