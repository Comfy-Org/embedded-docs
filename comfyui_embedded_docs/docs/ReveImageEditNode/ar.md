تتيح لك عقدة Reve Image Edit تعديل صورة موجودة بناءً على وصف نصي. تستخدم العقدة واجهة برمجة تطبيقات Reve API لتفسير تعليماتك وتطبيق التغييرات المطلوبة على الصورة التي تقدمها.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|-----------|-----------|----------|-------|-------------|
| `الصورة` | IMAGE | نعم | - | الصورة المراد تعديلها. |
| `تعليمات التعديل` | STRING | نعم | - | وصف نصي لكيفية تعديل الصورة. الحد الأقصى 2560 حرفًا. |
| `النموذج` | MODEL | نعم | `"reve-edit@20250915"`<br>`"reve-edit-fast@20251030"` | إصدار النموذج المستخدم للتعديل. |
| `model.aspect_ratio` | COMBO | لا | `"auto"`<br>`"16:9"`<br>`"9:16"`<br>`"3:2"`<br>`"2:3"`<br>`"4:3"`<br>`"3:4"`<br>`"1:1"` | نسبة العرض إلى الارتفاع للصورة المعدلة. عند تعيينها على "auto"، يتم تحديد نسبة العرض إلى الارتفاع تلقائيًا. |
| `model.test_time_scaling` | FLOAT | لا | - | عامل القياس أثناء الاختبار للنموذج. قد تؤدي القيم الأعلى إلى تحسين الجودة ولكنها تزيد من وقت المعالجة. |
| `تكبير` | COMBO | لا | `"disabled"`<br>`"enabled"` | يتحكم في ما إذا كان سيتم تحسين دقة الصورة المولدة. |
| `upscale.upscale_factor` | FLOAT | لا | - | العامل الذي يتم به تحسين دقة الصورة عند تفعيل خيار تحسين الدقة. |
| `إزالة الخلفية` | BOOLEAN | لا | - | يتحكم في ما إذا كان سيتم إزالة الخلفية من الصورة المولدة. |
| `البذرة` | INT | لا | 0 إلى 2147483647 | تتحكم البذرة في ما إذا كان يجب إعادة تشغيل العقدة؛ النتائج غير حتمية بغض النظر عن البذرة. (الافتراضي: 0) |

**ملاحظة:** معامل `upscale.upscale_factor` يكون ذا صلة فقط عندما يتم تعيين معامل `upscale` على `"enabled"`.
