گره StableZero123_Conditioning یک تصویر ورودی و زاویه‌های دوربین را پردازش می‌کند تا داده‌های شرطی‌سازی و نمایش‌های نهفته برای تولید مدل سه‌بعدی ایجاد کند. این گره از یک مدل بینایی CLIP برای رمزگذاری ویژگی‌های تصویر استفاده می‌کند، آن‌ها را با اطلاعات جاسازی‌شده دوربین بر اساس زاویه ارتفاع و آزیموت ترکیب می‌کند، و شرطی‌سازی مثبت و منفی را به همراه یک نمایش نهفته برای وظایف تولید سه‌بعدی پایین‌دستی تولید می‌کند.

## ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `clip_vision` | CLIP_VISION | بله | - | مدل بینایی CLIP که برای رمزگذاری ویژگی‌های تصویر استفاده می‌شود |
| `init_image` | IMAGE | بله | - | تصویر ورودی که باید پردازش و رمزگذاری شود |
| `vae` | VAE | بله | - | مدل VAE که برای رمزگذاری پیکسل‌ها به فضای نهفته استفاده می‌شود |
| `width` | INT | بله | 16 تا MAX_RESOLUTION | عرض خروجی برای نمایش نهفته (پیش‌فرض: ۲۵۶، باید بر ۸ بخش‌پذیر باشد) |
| `height` | INT | بله | 16 تا MAX_RESOLUTION | ارتفاع خروجی برای نمایش نهفته (پیش‌فرض: ۲۵۶، باید بر ۸ بخش‌پذیر باشد) |
| `batch_size` | INT | بله | 1 تا 4096 | تعداد نمونه‌هایی که در هر دسته تولید می‌شود (پیش‌فرض: ۱) |
| `elevation` | FLOAT | بله | ۱۸۰.۰- تا ۱۸۰.۰ | زاویه ارتفاع دوربین بر حسب درجه (پیش‌فرض: ۰.۰) |
| `azimuth` | FLOAT | بله | ۱۸۰.۰- تا ۱۸۰.۰ | زاویه آزیموت دوربین بر حسب درجه (پیش‌فرض: ۰.۰) |

**توجه:** پارامترهای `width` و `height` باید بر ۸ بخش‌پذیر باشند، زیرا گره آن‌ها را به‌طور خودکار بر ۸ تقسیم می‌کند تا ابعاد نمایش نهفته را ایجاد کند.

## خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-------------|-----------|-------------|
| `positive` | CONDITIONING | داده شرطی‌سازی مثبت که ویژگی‌های تصویر و جاسازی‌های دوربین را ترکیب می‌کند |
| `negative` | CONDITIONING | داده شرطی‌سازی منفی با ویژگی‌های مقداردهی‌اولیه‌شده با صفر |
| `latent` | LATENT | نمایش نهفته با ابعاد [batch_size, 4, height//8, width//8] |
