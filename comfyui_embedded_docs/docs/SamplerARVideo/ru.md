# Sampler AR Video

Узел Sampler AR Video предоставляет специализированный метод сэмплирования для авторегрессионных видеомоделей, таких как модели, использующие техники Causal Forcing или Self-Forcing. Он управляет всеми параметрами, связанными с авторегрессионным (AR) циклом, непосредственно в рабочем процессе, что упрощает настройку генерации видеокадров моделью по одному шагу за раз.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `num_frame_per_block` | INT | Да | от 1 до 64 | Количество кадров на авторегрессионный блок. Значение 1 означает, что модель генерирует по одному кадру за раз (покадрово), а значение 3 — что генерируются три кадра вместе (блочно). Этот параметр должен соответствовать режиму обучения контрольной точки. По умолчанию: 1. |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `SAMPLER` | SAMPLER | Настроенный объект сэмплера, использующий функцию сэмплирования "ar_video" с заданными авторегрессионными параметрами. |
