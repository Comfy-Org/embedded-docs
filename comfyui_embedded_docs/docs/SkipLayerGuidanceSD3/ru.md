Вот перевод документации на русский язык:

Узел SkipLayerGuidanceSD3 улучшает проработку детальной структуры, применяя дополнительный набор бесклассового направляющего воздействия (classifier-free guidance) с пропущенными слоями. Эта экспериментальная реализация вдохновлена методом Perturbed Attention Guidance и работает путем избирательного обхода определенных слоев в процессе негативного кондиционирования для улучшения структурных деталей в генерируемом результате.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `модель` | MODEL | Да | - | Модель, к которой применяется направляющее воздействие с пропуском слоев |
| `слои` | STRING | Да | - | Разделенный запятыми список индексов слоев для пропуска (по умолчанию: "7, 8, 9") |
| `масштаб` | FLOAT | Да | 0.0 - 10.0 | Сила эффекта направляющего воздействия с пропуском слоев (по умолчанию: 3.0) |
| `начальный_процент` | FLOAT | Да | 0.0 - 1.0 | Начальная точка применения направляющего воздействия в процентах от общего числа шагов (по умолчанию: 0.01) |
| `конечный_процент` | FLOAT | Да | 0.0 - 1.0 | Конечная точка применения направляющего воздействия в процентах от общего числа шагов (по умолчанию: 0.15) |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `модель` | MODEL | Модифицированная модель с примененным направляющим воздействием с пропуском слоев |
