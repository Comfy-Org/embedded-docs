Узел SV3D_Conditioning подготавливает данные conditioning для генерации 3D-видео с использованием модели SV3D. Он принимает исходное изображение и обрабатывает его через кодировщики CLIP vision и VAE для создания позитивного и негативного conditioning, а также латентного представления. Узел генерирует последовательности углов возвышения и азимута камеры для создания многокадрового видео на основе указанного количества кадров.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `clip_vision` | CLIP_VISION | Да | - | Модель CLIP vision, используемая для кодирования входного изображения |
| `init_image` | IMAGE | Да | - | Исходное изображение, служащее отправной точкой для генерации 3D-видео |
| `vae` | VAE | Да | - | Модель VAE, используемая для кодирования изображения в латентное пространство |
| `width` | INT | Нет | 16 до MAX_RESOLUTION | Ширина выходных кадров генерируемого видео (по умолчанию: 576, должна делиться на 8) |
| `height` | INT | Нет | 16 до MAX_RESOLUTION | Высота выходных кадров генерируемого видео (по умолчанию: 576, должна делиться на 8) |
| `video_frames` | INT | Нет | 1 до 4096 | Количество кадров для генерации видеопоследовательности (по умолчанию: 21) |
| `elevation` | FLOAT | Нет | -90.0 до 90.0 | Угол возвышения камеры в градусах для 3D-вида (по умолчанию: 0.0) |

## Выходные данные

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `positive` | CONDITIONING | Данные позитивного conditioning, содержащие эмбеддинги изображения и параметры камеры для генерации |
| `negative` | CONDITIONING | Данные негативного conditioning с обнуленными эмбеддингами для контрастной генерации |
| `latent` | LATENT | Пустой латентный тензор с размерами, соответствующими указанному количеству кадров и разрешению |
