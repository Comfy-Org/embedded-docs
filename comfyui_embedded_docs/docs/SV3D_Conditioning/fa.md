گره SV3D_Conditioning داده‌های شرطی‌سازی را برای تولید ویدیوی سه‌بعدی با استفاده از مدل SV3D آماده می‌کند. این گره یک تصویر اولیه دریافت کرده و آن را از طریق رمزگذارهای CLIP vision و VAE پردازش می‌کند تا شرطی‌سازی مثبت و منفی به همراه یک نمایش نهفته ایجاد کند. این گره بر اساس تعداد فریم‌های ویدیویی مشخص‌شده، دنباله‌های ارتفاع و آزیموت دوربین را برای تولید ویدیوی چندفریمی تولید می‌کند.

## ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `clip vision` | CLIP_VISION | بله | - | مدل vision CLIP که برای رمزگذاری تصویر ورودی استفاده می‌شود |
| `تصویر اولیه` | IMAGE | بله | - | تصویر اولیه که به عنوان نقطه شروع برای تولید ویدیوی سه‌بعدی عمل می‌کند |
| `vae` | VAE | بله | - | مدل VAE که برای رمزگذاری تصویر به فضای نهفته استفاده می‌شود |
| `عرض` | INT | خیر | 16 تا MAX_RESOLUTION | عرض خروجی برای فریم‌های ویدیویی تولیدشده (پیش‌فرض: ۵۷۶، باید بر ۸ بخش‌پذیر باشد) |
| `ارتفاع` | INT | خیر | 16 تا MAX_RESOLUTION | ارتفاع خروجی برای فریم‌های ویدیویی تولیدشده (پیش‌فرض: ۵۷۶، باید بر ۸ بخش‌پذیر باشد) |
| `فریم‌های ویدیو` | INT | خیر | 1 تا 4096 | تعداد فریم‌هایی که برای دنباله ویدیو تولید می‌شود (پیش‌فرض: ۲۱) |
| `ارتفاع زاویه` | FLOAT | خیر | 90.0- تا 90.0 | زاویه ارتفاع دوربین بر حسب درجه برای نمای سه‌بعدی (پیش‌فرض: ۰.۰) |

## خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-------------|-----------|-------------|
| `positive` | CONDITIONING | داده‌های شرطی‌سازی مثبت شامل جاسازی‌های تصویر و پارامترهای دوربین برای تولید |
| `negative` | CONDITIONING | داده‌های شرطی‌سازی منفی با جاسازی‌های صفرشده برای تولید تقابلی |
| `latent` | LATENT | یک تنسور نهفته خالی با ابعاد منطبق بر تعداد فریم‌های ویدیویی و وضوح مشخص‌شده |
