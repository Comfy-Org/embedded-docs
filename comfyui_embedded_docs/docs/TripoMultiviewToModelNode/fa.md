### نمای کلی

این گره با استفاده از API سرویس Tripo، مدل‌های سه‌بعدی را به‌صورت همزمان تولید می‌کند. برای این کار، حداکثر چهار تصویر از زوایای مختلف یک شیء را پردازش می‌کند. برای ایجاد یک مدل سه‌بعدی کامل با گزینه‌های بافت و متریال، به یک تصویر از نمای جلو و حداقل یک نمای دیگر (چپ، پشت یا راست) نیاز است.

### ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `تصویر` | IMAGE | بله | - | تصویر نمای جلوی شیء (الزامی) |
| `تصویر چپ` | IMAGE | خیر | - | تصویر نمای چپ شیء |
| `تصویر پشت` | IMAGE | خیر | - | تصویر نمای پشت شیء |
| `تصویر راست` | IMAGE | خیر | - | تصویر نمای راست شیء |
| `نسخه مدل` | COMBO | خیر | گزینه‌های متعدد موجود | نسخه مدل مورد استفاده برای تولید |
| `جهت` | COMBO | خیر | گزینه‌های متعدد موجود | تنظیم جهت‌گیری برای مدل سه‌بعدی (پیش‌فرض: "default") |
| `تکسچر` | BOOLEAN | خیر | - | آیا بافت برای مدل تولید شود (پیش‌فرض: True) |
| `PBR` | BOOLEAN | خیر | - | آیا متریال PBR (رندر مبتنی بر فیزیک) تولید شود (پیش‌فرض: True) |
| `بذر مدل` | INT | خیر | - | دانه تصادفی برای تولید مدل (پیش‌فرض: 42) |
| `بذر تکسچر` | INT | خیر | - | دانه تصادفی برای تولید بافت (پیش‌فرض: 42) |
| `کیفیت تکسچر` | COMBO | خیر | `"standard"`<br>`"detailed"` | سطح کیفیت برای تولید بافت (پیش‌فرض: "standard") |
| `تراز تکسچر` | COMBO | خیر | `"original_image"`<br>`"geometry"` | روش تراز کردن بافت‌ها روی مدل (پیش‌فرض: "original_image") |
| `محدودیت وجه` | INT | خیر | 1- تا 500000 | حداکثر تعداد وجه‌ها در مدل تولید شده. برای عدم محدودیت، مقدار 1- را تنظیم کنید (پیش‌فرض: 1-) |
| `چهارضلعی` | BOOLEAN | خیر | - | این پارامتر منسوخ شده و هیچ کاری انجام نمی‌دهد (پیش‌فرض: False) |
| `کیفیت هندسه` | COMBO | خیر | `"standard"`<br>`"detailed"` | سطح کیفیت برای تولید هندسه (پیش‌فرض: "standard") |

**توجه:** تصویر جلو (`image`) همیشه الزامی است. برای پردازش چندنمایی، حداقل یک تصویر نمای دیگر (`image_left`، `image_back` یا `image_right`) باید ارائه شود.

### خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-------------|-----------|-------------|
| `model_file` | STRING | مسیر فایل یا شناسه مدل سه‌بعدی تولید شده (فقط برای سازگاری با نسخه‌های قبلی) |
| `model task_id` | MODEL_TASK_ID | شناسه وظیفه برای پیگیری فرآیند تولید مدل |
| `GLB` | FILE3DGLB | فایل مدل سه‌بعدی تولید شده در قالب GLB |
