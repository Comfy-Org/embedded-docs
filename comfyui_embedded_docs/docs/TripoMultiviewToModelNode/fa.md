### نمای کلی

این گره با استفاده از API سرویس Tripo، مدل‌های سه‌بعدی را به‌صورت همزمان تولید می‌کند. برای این کار، حداکثر چهار تصویر از زوایای مختلف یک شیء را پردازش می‌کند. برای ایجاد یک مدل سه‌بعدی کامل با گزینه‌های بافت و متریال، به یک تصویر از نمای جلو و حداقل یک نمای دیگر (چپ، پشت یا راست) نیاز است.

### ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `image` | IMAGE | بله | - | تصویر نمای جلوی شیء (الزامی) |
| `image_left` | IMAGE | خیر | - | تصویر نمای چپ شیء |
| `image_back` | IMAGE | خیر | - | تصویر نمای پشت شیء |
| `image_right` | IMAGE | خیر | - | تصویر نمای راست شیء |
| `model_version` | COMBO | خیر | گزینه‌های متعدد موجود | نسخه مدل مورد استفاده برای تولید |
| `orientation` | COMBO | خیر | گزینه‌های متعدد موجود | تنظیم جهت‌گیری برای مدل سه‌بعدی (پیش‌فرض: "default") |
| `texture` | BOOLEAN | خیر | - | آیا بافت برای مدل تولید شود (پیش‌فرض: True) |
| `pbr` | BOOLEAN | خیر | - | آیا متریال PBR (رندر مبتنی بر فیزیک) تولید شود (پیش‌فرض: True) |
| `model_seed` | INT | خیر | - | دانه تصادفی برای تولید مدل (پیش‌فرض: 42) |
| `texture_seed` | INT | خیر | - | دانه تصادفی برای تولید بافت (پیش‌فرض: 42) |
| `texture_quality` | COMBO | خیر | `"standard"`<br>`"detailed"` | سطح کیفیت برای تولید بافت (پیش‌فرض: "standard") |
| `texture_alignment` | COMBO | خیر | `"original_image"`<br>`"geometry"` | روش تراز کردن بافت‌ها روی مدل (پیش‌فرض: "original_image") |
| `face_limit` | INT | خیر | 1- تا 500000 | حداکثر تعداد وجه‌ها در مدل تولید شده. برای عدم محدودیت، مقدار 1- را تنظیم کنید (پیش‌فرض: 1-) |
| `quad` | BOOLEAN | خیر | - | این پارامتر منسوخ شده و هیچ کاری انجام نمی‌دهد (پیش‌فرض: False) |
| `geometry_quality` | COMBO | خیر | `"standard"`<br>`"detailed"` | سطح کیفیت برای تولید هندسه (پیش‌فرض: "standard") |

**توجه:** تصویر جلو (`image`) همیشه الزامی است. برای پردازش چندنمایی، حداقل یک تصویر نمای دیگر (`image_left`، `image_back` یا `image_right`) باید ارائه شود.

### خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-------------|-----------|-------------|
| `model_file` | STRING | مسیر فایل یا شناسه مدل سه‌بعدی تولید شده (فقط برای سازگاری با نسخه‌های قبلی) |
| `model task_id` | MODEL_TASK_ID | شناسه وظیفه برای پیگیری فرآیند تولید مدل |
| `GLB` | FILE3DGLB | فایل مدل سه‌بعدی تولید شده در قالب GLB |
