## قواعد الترجمة

1. **المحتوى الذي يجب عدم ترجمته:**
   - أسماء المعاملات بين علامات الاقتباس الخلفية: `image`, `seed`, `model`
   - أنواع البيانات بالأحرف الكبيرة: IMAGE, STRING, INT, FLOAT, MODEL, CONDITIONING, إلخ
   - القيم في عمود Range: الأرقام، "auto"، أسماء الخيارات
   - الكود، مسارات الملفات

2. **المحتوى الذي يجب ترجمته:**
   - عناوين الأقسام: ## نظرة عامة, ## المدخلات, ## المخرجات
   - جميع النصوص الوصفية والتوضيحية
   - أوصاف المعاملات

3. **جودة الترجمة:**
   - استخدام اللغة العربية الفصحى المعاصرة
   - الحفاظ على نبرة احترافية ولكن سهلة الفهم
   - ضمان الدقة التقنية
   - استخدام المصطلحات التقنية العربية القياسية

4. **التنسيق:**

   - الحفاظ على بنية الجداول

تقوم هذه العقدة بإنشاء نماذج ثلاثية الأبعاد بشكل متزامن باستخدام واجهة برمجة تطبيقات Tripo من خلال معالجة ما يصل إلى أربع صور تُظهر زوايا مختلفة لجسم ما. تتطلب صورة أمامية وصورة إضافية واحدة على الأقل (يسار، خلف، أو يمين) لإنشاء نموذج ثلاثي الأبعاد كامل مع خيارات القوام والمواد.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|-----------|-----------|----------|-------|-------------|
| `الصورة` | IMAGE | نعم | - | الصورة الأمامية للجسم (مطلوبة) |
| `الصورة اليسرى` | IMAGE | لا | - | الصورة اليسرى للجسم |
| `الصورة الخلفية` | IMAGE | لا | - | الصورة الخلفية للجسم |
| `الصورة اليمنى` | IMAGE | لا | - | الصورة اليمنى للجسم |
| `إصدار_النموذج` | COMBO | لا | خيارات متعددة متاحة | إصدار النموذج المستخدم في الإنشاء |
| `التوجيه` | COMBO | لا | خيارات متعددة متاحة | إعداد الاتجاه للنموذج ثلاثي الأبعاد (الافتراضي: "default") |
| `الملمس` | BOOLEAN | لا | - | ما إذا كان سيتم إنشاء قوام للنموذج (الافتراضي: True) |
| `PBR` | BOOLEAN | لا | - | ما إذا كان سيتم إنشاء مواد PBR (العرض الفيزيائي) (الافتراضي: True) |
| `بذرة_النموذج` | INT | لا | - | البذرة العشوائية لإنشاء النموذج (الافتراضي: 42) |
| `بذرة_الملمس` | INT | لا | - | البذرة العشوائية لإنشاء القوام (الافتراضي: 42) |
| `جودة_الملمس` | COMBO | لا | `"standard"`<br>`"detailed"` | مستوى جودة إنشاء القوام (الافتراضي: "standard") |
| `محاذاة_الملمس` | COMBO | لا | `"original_image"`<br>`"geometry"` | طريقة محاذاة القوام مع النموذج (الافتراضي: "original_image") |
| `حد_الوجه` | INT | لا | -1 إلى 500000 | الحد الأقصى لعدد الأسطح في النموذج المُنشأ. اضبط على -1 لعدم وجود حد (الافتراضي: -1) |
| `رباعي` | BOOLEAN | لا | - | هذه المعلمة مهملة ولا تؤدي أي وظيفة (الافتراضي: False) |
| `جودة الهندسة` | COMBO | لا | `"standard"`<br>`"detailed"` | مستوى جودة إنشاء الهندسة (الافتراضي: "standard") |

**ملاحظة:** الصورة الأمامية (`image`) مطلوبة دائمًا. يجب توفير صورة زاوية إضافية واحدة على الأقل (`image_left`، `image_back`، أو `image_right`) لمعالجة العرض المتعدد.

## المخرجات
