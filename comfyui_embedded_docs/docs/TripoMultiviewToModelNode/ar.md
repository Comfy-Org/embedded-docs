## قواعد الترجمة

1. **المحتوى الذي يجب عدم ترجمته:**
   - أسماء المعاملات بين علامات الاقتباس الخلفية: `image`, `seed`, `model`
   - أنواع البيانات بالأحرف الكبيرة: IMAGE, STRING, INT, FLOAT, MODEL, CONDITIONING, إلخ
   - القيم في عمود Range: الأرقام، "auto"، أسماء الخيارات
   - الكود، مسارات الملفات

2. **المحتوى الذي يجب ترجمته:**
   - عناوين الأقسام: ## نظرة عامة, ## المدخلات, ## المخرجات
   - جميع النصوص الوصفية والتوضيحية
   - أوصاف المعاملات

3. **جودة الترجمة:**
   - استخدام اللغة العربية الفصحى المعاصرة
   - الحفاظ على نبرة احترافية ولكن سهلة الفهم
   - ضمان الدقة التقنية
   - استخدام المصطلحات التقنية العربية القياسية

4. **التنسيق:**
   - الحفاظ على جميع تنسيقات Markdown
   - الحفاظ على بنية الجداول
   - عدم إضافة أي ملاحظة أو رابط في بداية الوثيقة (سيتم إضافتها تلقائيًا)

تقوم هذه العقدة بإنشاء نماذج ثلاثية الأبعاد بشكل متزامن باستخدام واجهة برمجة تطبيقات Tripo من خلال معالجة ما يصل إلى أربع صور تُظهر زوايا مختلفة لجسم ما. تتطلب صورة أمامية وصورة إضافية واحدة على الأقل (يسار، خلف، أو يمين) لإنشاء نموذج ثلاثي الأبعاد كامل مع خيارات القوام والمواد.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|-----------|-----------|----------|-------|-------------|
| `image` | IMAGE | نعم | - | الصورة الأمامية للجسم (مطلوبة) |
| `image_left` | IMAGE | لا | - | الصورة اليسرى للجسم |
| `image_back` | IMAGE | لا | - | الصورة الخلفية للجسم |
| `image_right` | IMAGE | لا | - | الصورة اليمنى للجسم |
| `model_version` | COMBO | لا | خيارات متعددة متاحة | إصدار النموذج المستخدم في الإنشاء |
| `orientation` | COMBO | لا | خيارات متعددة متاحة | إعداد الاتجاه للنموذج ثلاثي الأبعاد (الافتراضي: "default") |
| `texture` | BOOLEAN | لا | - | ما إذا كان سيتم إنشاء قوام للنموذج (الافتراضي: True) |
| `pbr` | BOOLEAN | لا | - | ما إذا كان سيتم إنشاء مواد PBR (العرض الفيزيائي) (الافتراضي: True) |
| `model_seed` | INT | لا | - | البذرة العشوائية لإنشاء النموذج (الافتراضي: 42) |
| `texture_seed` | INT | لا | - | البذرة العشوائية لإنشاء القوام (الافتراضي: 42) |
| `texture_quality` | COMBO | لا | `"standard"`<br>`"detailed"` | مستوى جودة إنشاء القوام (الافتراضي: "standard") |
| `texture_alignment` | COMBO | لا | `"original_image"`<br>`"geometry"` | طريقة محاذاة القوام مع النموذج (الافتراضي: "original_image") |
| `face_limit` | INT | لا | -1 إلى 500000 | الحد الأقصى لعدد الأسطح في النموذج المُنشأ. اضبط على -1 لعدم وجود حد (الافتراضي: -1) |
| `quad` | BOOLEAN | لا | - | هذه المعلمة مهملة ولا تؤدي أي وظيفة (الافتراضي: False) |
| `geometry_quality` | COMBO | لا | `"standard"`<br>`"detailed"` | مستوى جودة إنشاء الهندسة (الافتراضي: "standard") |

**ملاحظة:** الصورة الأمامية (`image`) مطلوبة دائمًا. يجب توفير صورة زاوية إضافية واحدة على الأقل (`image_left`، `image_back`، أو `image_right`) لمعالجة العرض المتعدد.

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|-------------|-----------|-------------|
| `model_file` | STRING | مسار الملف أو معرف النموذج ثلاثي الأبعاد المُنشأ (للتوافق مع الإصدارات السابقة فقط) |
| `model task_id` | MODEL_TASK_ID | معرف المهمة لتتبع عملية إنشاء النموذج |
| `GLB` | FILE3DGLB | ملف النموذج ثلاثي الأبعاد المُنشأ بتنسيق GLB |
