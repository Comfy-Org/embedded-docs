Этот узел синхронно генерирует 3D-модели с помощью API Tripo, обрабатывая до четырех изображений, показывающих объект с разных ракурсов. Для создания полноценной 3D-модели с опциями текстуры и материалов требуется фронтальное изображение и как минимум один дополнительный ракурс (слева, сзади или справа).

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `image` | IMAGE | Да | - | Изображение объекта спереди (обязательно) |
| `image_left` | IMAGE | Нет | - | Изображение объекта слева |
| `image_back` | IMAGE | Нет | - | Изображение объекта сзади |
| `image_right` | IMAGE | Нет | - | Изображение объекта справа |
| `версия_модели` | COMBO | Нет | Доступно несколько вариантов | Версия модели, используемая для генерации |
| `ориентация` | COMBO | Нет | Доступно несколько вариантов | Настройка ориентации 3D-модели (по умолчанию: "default") |
| `текстура` | BOOLEAN | Нет | - | Создавать ли текстуры для модели (по умолчанию: True) |
| `PBR` | BOOLEAN | Нет | - | Создавать ли PBR-материалы (физически корректный рендеринг) (по умолчанию: True) |
| `сид_модели` | INT | Нет | - | Случайное зерно для генерации модели (по умолчанию: 42) |
| `сид_текстуры` | INT | Нет | - | Случайное зерно для генерации текстуры (по умолчанию: 42) |
| `качество_текстуры` | COMBO | Нет | `"standard"`<br>`"detailed"` | Уровень качества генерации текстур (по умолчанию: "standard") |
| `выравнивание_текстуры` | COMBO | Нет | `"original_image"`<br>`"geometry"` | Метод выравнивания текстур относительно модели (по умолчанию: "original_image") |
| `лимит_лиц` | INT | Нет | от -1 до 500000 | Максимальное количество полигонов в сгенерированной модели. Установите -1 для отсутствия ограничения (по умолчанию: -1) |
| `квад` | BOOLEAN | Нет | - | Этот параметр устарел и не выполняет никаких действий (по умолчанию: False) |
| `качество геометрии` | COMBO | Нет | `"standard"`<br>`"detailed"` | Уровень качества генерации геометрии (по умолчанию: "standard") |

**Примечание:** Фронтальное изображение (`image`) всегда обязательно. Для многовидовой обработки необходимо предоставить как минимум одно дополнительное изображение (`image_left`, `image_back` или `image_right`).

## Выходные данные

| Имя выходного параметра | Тип данных | Описание |
|-------------------------|------------|----------|
| `model_file` | STRING | Путь к файлу или идентификатор сгенерированной 3D-модели (только для обратной совместимости) |
| `model task_id` | MODEL_TASK_ID | Идентификатор задачи для отслеживания процесса генерации модели |
| `GLB` | FILE3DGLB | Сгенерированный файл 3D-модели в формате GLB |
