Вот перевод документации на русский язык:

Узел ImageRGBToYUV преобразует цветные изображения RGB в цветовое пространство YUV. Он принимает RGB-изображение на вход и разделяет его на три отдельных канала: Y (яркость), U (синяя проекция) и V (красная проекция). Каждый выходной канал возвращается в виде отдельного полутонового изображения, представляющего соответствующий компонент YUV.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `изображение` | IMAGE | Да | - | Входное RGB-изображение для преобразования в цветовое пространство YUV |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `Y` | IMAGE | Компонент яркости (светимости) цветового пространства YUV |
| `U` | IMAGE | Компонент синей проекции цветового пространства YUV |
| `V` | IMAGE | Компонент красной проекции цветового пространства YUV |
