Узел LatentComposite предназначен для смешивания или объединения двух латентных представлений в единый выходной результат. Этот процесс необходим для создания составных изображений или признаков путем контролируемого комбинирования характеристик входных латентных представлений.

## Входные параметры

| Параметр       | Тип данных | Описание |
|----------------|------------|----------|
| `образцы_для`   | `LATENT`   | Латентное представление 'samples_to', на которое будет наложено 'samples_from'. Служит основой для операции композиции. |
| `образцы_из` | `LATENT`   | Латентное представление 'samples_from', которое будет наложено на 'samples_to'. Вносит свои признаки или характеристики в итоговый составной результат. |
| `x`            | `INT`      | Координата X (горизонтальное положение), по которой латентное представление 'samples_from' будет размещено на 'samples_to'. Определяет горизонтальное выравнивание композиции. |
| `y`            | `INT`      | Координата Y (вертикальное положение), по которой латентное представление 'samples_from' будет размещено на 'samples_to'. Определяет вертикальное выравнивание композиции. |
| `перо`      | `INT`      | Логическое значение, указывающее, следует ли изменить размер латентного представления 'samples_from' для соответствия 'samples_to' перед композицией. Это может повлиять на масштаб и пропорции итогового составного результата. |

## Выходные параметры

| Параметр | Тип данных | Описание |
|----------|------------|----------|
| `latent` | `LATENT`   | Выходной результат представляет собой составное латентное представление, объединяющее признаки обоих латентных представлений 'samples_to' и 'samples_from' на основе указанных координат и опции изменения размера. |
