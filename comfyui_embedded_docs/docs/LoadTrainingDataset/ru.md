Вот перевод документации на русский язык:

Этот узел загружает закодированный обучающий набор данных, который ранее был сохранён на диск. Он выполняет поиск и чтение всех файлов-сегментов данных из указанной папки в выходном каталоге ComfyUI, после чего возвращает объединённые латентные векторы и данные кондиционирования для использования в рабочих процессах обучения.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `folder_name` | STRING | Да | Н/Д | Имя папки, содержащей сохранённый набор данных, расположенной внутри выходного каталога ComfyUI (по умолчанию: "training_dataset"). |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `latents` | LATENT | Список латентных словарей, где каждый словарь содержит ключ `"samples"` с тензором. |
| `conditioning` | CONDITIONING | Список списков кондиционирования, где каждый внутренний список содержит данные кондиционирования для соответствующего образца. |
