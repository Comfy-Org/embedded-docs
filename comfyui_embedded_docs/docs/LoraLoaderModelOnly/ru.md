Этот узел обнаруживает модели, расположенные в папке `ComfyUI/models/loras`, а также считывает модели из дополнительных путей, указанных в файле extra_model_paths.yaml. Иногда может потребоваться **обновить интерфейс ComfyUI**, чтобы он смог прочитать файлы моделей из соответствующей папки.

Данный узел специализируется на загрузке LoRA-модели без необходимости использования CLIP-модели, фокусируясь на улучшении или модификации заданной модели на основе параметров LoRA. Он позволяет динамически регулировать силу воздействия модели через параметры LoRA, обеспечивая точный контроль над поведением модели.

## Входные данные

| Поле              | Тип Comfy         | Описание                                                                                   |
|-------------------|-------------------|-----------------------------------------------------------------------------------------------|
| `model`           | `MODEL`           | Базовая модель для модификаций, к которой будут применены корректировки LoRA.                   |
| `lora_name`       | `COMBO[STRING]`   | Имя загружаемого LoRA-файла, определяющее корректировки, применяемые к модели.      |
| `strength_model`  | `FLOAT`           | Определяет интенсивность корректировок LoRA: более высокие значения указывают на более сильные модификации. |

## Выходные данные

| Поле    | Тип данных | Описание                                                              |
|---------|-------------|--------------------------------------------------------------------------|
| `model` | `MODEL`     | Модифицированная модель с применёнными корректировками LoRA, отражающая изменения в поведении или возможностях модели. |
