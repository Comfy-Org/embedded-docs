# ترجمة وثيقة عقدة ComfyUI

## عقدة حفظ مجموعة بيانات الصور والنصوص إلى مجلد

تقوم عقدة حفظ مجموعة بيانات الصور والنصوص إلى مجلد بحفظ قائمة من الصور وتعليقاتها النصية المقابلة في مجلد محدد داخل دليل الإخراج الخاص بـ ComfyUI. لكل صورة يتم حفظها كملف PNG، يتم إنشاء ملف نصي مطابق بنفس الاسم الأساسي لتخزين تعليقها التوضيحي. وهذا مفيد لإنشاء مجموعات بيانات منظمة من الصور المُنشأة وأوصافها.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|---------|---------------|--------|---------|-------|
| `images` | IMAGE | نعم | - | قائمة الصور المراد حفظها. |
| `texts` | STRING | نعم | - | قائمة التعليقات النصية المراد حفظها. |
| `folder_name` | STRING | لا | - | اسم المجلد لحفظ الصور فيه (داخل دليل الإخراج). (الافتراضي: "dataset") |
| `filename_prefix` | STRING | لا | - | بادئة لأسماء ملفات الصور المحفوظة. (الافتراضي: "image") |

**ملاحظة:** مدخلات `images` و `texts` هي قوائم. تتوقع العقدة أن يتطابق عدد التعليقات النصية مع عدد الصور المقدمة. سيتم حفظ كل تعليق نصي في ملف `.txt` مطابق للصورة المرتبطة به.
