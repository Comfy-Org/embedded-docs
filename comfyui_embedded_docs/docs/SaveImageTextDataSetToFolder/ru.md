Вот перевод документации на русский язык:

Узел "Сохранить изображения и текстовый набор данных в папку" сохраняет список изображений и соответствующие им текстовые подписи в указанную папку внутри выходного каталога ComfyUI. Для каждого изображения, сохраненного в формате PNG, создается соответствующий текстовый файл с тем же базовым именем для хранения его подписи. Это полезно для создания организованных наборов данных из сгенерированных изображений и их описаний.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `images` | IMAGE | Да | - | Список изображений для сохранения. |
| `texts` | STRING | Да | - | Список текстовых подписей для сохранения. |
| `folder_name` | STRING | Нет | - | Имя папки для сохранения изображений (внутри выходного каталога). (по умолчанию: "dataset") |
| `filename_prefix` | STRING | Нет | - | Префикс для имен файлов сохраняемых изображений. (по умолчанию: "image") |

**Примечание:** Входные параметры `images` и `texts` являются списками. Узел ожидает, что количество текстовых подписей будет соответствовать количеству предоставленных изображений. Каждая подпись будет сохранена в файле `.txt`, соответствующем своему парному изображению.

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| - | - | Этот узел не имеет выходных параметров. Он сохраняет файлы непосредственно в файловую систему. |
