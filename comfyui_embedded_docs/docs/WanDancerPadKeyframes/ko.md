# 개요

이 노드는 긴 비디오 생성 과정의 특정 구간에 대한 키프레임 시퀀스를 준비합니다. 입력 이미지 배치와 오디오 트랙을 받아, 오디오 길이를 기준으로 전체 비디오의 총 프레임 수를 계산한 후, 선택한 구간에 입력 이미지를 키프레임으로 분배하고 나머지는 빈 프레임으로 채웁니다. 또한 해당 구간의 오디오 부분을 추출합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 여부 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `images` | IMAGE | 예 | 이미지 배치 | 키프레임으로 분배할 입력 이미지입니다. |
| `segment_length` | INT | 예 | 1 ~ 10000 | 이 구간의 프레임 길이입니다(기본값: 149). |
| `segment_index` | INT | 예 | 0 ~ 100 | 현재 구간의 인덱스입니다(0은 첫 번째, 1은 두 번째 등을 의미하며 기본값: 0). |
| `audio` | AUDIO | 예 | 오디오 데이터 | 전체 출력 프레임 수를 계산하고 구간 오디오를 추출하는 데 사용할 오디오입니다. |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `keyframes_sequence` | IMAGE | 지정된 구간에 대한 패딩 처리된 키프레임 시퀀스입니다. |
| `keyframes_mask` | MASK | 유효 프레임을 나타내는 마스크입니다(키프레임 위치는 1, 패딩 위치는 0). |
| `audio_segment` | AUDIO | 이 비디오 구간에 해당하는 오디오 구간입니다. |
