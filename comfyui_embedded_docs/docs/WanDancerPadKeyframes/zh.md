## ## 概述

此节点为较长视频生成过程中的特定片段准备关键帧序列。它接收一批输入图像和一条音轨，根据音频时长计算完整视频应包含的总帧数，然后将输入图像作为关键帧分布到所选片段中，其余部分用空白帧填充。同时，它还会提取该片段对应的音频部分。

## ## 输入

| 参数 | 数据类型 | 是否必填 | 范围 | 描述 |
|-----------|-----------|----------|-------|-------------|
| `images` | IMAGE | 是 | 图像批次 | 要作为关键帧分布的输入图像。 |
| `segment_length` | INT | 是 | 1 至 10000 | 此片段的帧长度（默认值：149）。 |
| `segment_index` | INT | 是 | 0 至 100 | 当前片段索引（0 表示第一个，1 表示第二个，以此类推，默认值：0）。 |
| `audio` | AUDIO | 是 | 音频数据 | 用于计算总输出帧数并提取片段音频的音频。 |

## ## 输出

| 输出名称 | 数据类型 | 描述 |
|-------------|-----------|-------------|
| `keyframes_sequence` | IMAGE | 指定片段的填充关键帧序列。 |
| `keyframes_mask` | MASK | 指示有效帧的遮罩（1 表示关键帧位置，0 表示填充位置）。 |
| `audio_segment` | AUDIO | 此视频片段的音频片段。 |
