## Обзор

Этот узел подготавливает последовательность ключевых кадров для определенного сегмента более длительного процесса генерации видео. Он принимает пакет входных изображений и аудиодорожку, вычисляет общее количество кадров, которое должно быть в полном видео, на основе длительности аудио, а затем распределяет входные изображения в качестве ключевых кадров по выбранному сегменту, заполняя остальную часть пустыми кадрами. Также извлекается соответствующий фрагмент аудио для этого сегмента.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `images` | IMAGE | Да | Пакет изображений | Входные изображения, которые будут распределены в качестве ключевых кадров. |
| `segment_length` | INT | Да | от 1 до 10000 | Длина этого сегмента в кадрах (по умолчанию: 149). |
| `segment_index` | INT | Да | от 0 до 100 | Какой это сегмент (0 — первый, 1 — второй и т.д., по умолчанию: 0). |
| `audio` | AUDIO | Да | Аудиоданные | Аудио для расчета общего количества выходных кадров и извлечения аудио сегмента. |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `keyframes_sequence` | IMAGE | Заполненная последовательность ключевых кадров для указанного сегмента. |
| `keyframes_mask` | MASK | Маска, указывающая на валидные кадры (1 — позиции ключевых кадров, 0 — заполненные позиции). |
| `audio_segment` | AUDIO | Аудио сегмент для этого видео сегмента. |
