## مرور کلی

این گره یک دنباله از فریم‌های کلیدی را برای بخش مشخصی از یک فرآیند طولانی‌تر تولید ویدیو آماده می‌کند. این گره یک دسته از تصاویر ورودی و یک آهنگ صوتی دریافت می‌کند، تعداد کل فریم‌های مورد نیاز ویدیوی کامل را بر اساس مدت زمان صدا محاسبه می‌کند و سپس تصاویر ورودی را به‌عنوان فریم‌های کلیدی در بخش انتخاب شده توزیع می‌کند و باقی را با فریم‌های خالی پر می‌کند. همچنین بخش متناظر از صدا را برای آن بخش استخراج می‌کند.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|---------|----------|--------|--------|----------|
| `images` | IMAGE | بله | دسته تصاویر | تصاویر ورودی که به‌عنوان فریم‌های کلیدی توزیع می‌شوند. |
| `segment_length` | INT | بله | ۱ تا ۱۰۰۰۰ | طول این بخش بر حسب فریم (پیش‌فرض: ۱۴۹). |
| `segment_index` | INT | بله | ۰ تا ۱۰۰ | شماره این بخش (۰ برای اولی، ۱ برای دومی و غیره، پیش‌فرض: ۰). |
| `audio` | AUDIO | بله | داده صوتی | صدا برای محاسبه تعداد کل فریم‌های خروجی و استخراج صدای بخش. |

## خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-----------|----------|----------|
| `keyframes_sequence` | IMAGE | دنباله فریم‌های کلیدی پرشده برای بخش مشخص‌شده. |
| `keyframes_mask` | MASK | ماسک نشان‌دهنده فریم‌های معتبر (۱ برای موقعیت‌های فریم کلیدی، ۰ برای موقعیت‌های پرشده). |
| `audio_segment` | AUDIO | بخش صوتی مربوط به این بخش ویدیو. |
