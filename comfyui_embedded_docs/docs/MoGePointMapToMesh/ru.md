## Обзор

Этот узел преобразует карту точек MoGe в трехмерную сетку. Он принимает геометрические данные, созданные узлом оценки глубины MoGe, и триангулирует их в сетку с UV-координатами и опциональной текстурой.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `moge_geometry` | MOGE_GEOMETRY | Да | Н/Д | Геометрические данные MoGe, содержащие карты точек, глубину и опционально исходное изображение. |
| `batch_index` | INT | Да | от 0 до 4096 | Индекс изображения в пакетной геометрии MoGe для преобразования в сетку. Количество вершин для каждого изображения различается, поэтому пакеты нельзя объединить в одну сетку MESH (по умолчанию: 0). |
| `decimation` | INT | Да | от 1 до 8 | Шаг вершин; 1 = полное разрешение (по умолчанию: 1). |
| `discontinuity_threshold` | FLOAT | Да | от 0.0 до 1.0 | Отбрасывает пиксели, чей разброс глубины в окрестности 3x3 превышает эту долю. 0 = отключено (по умолчанию: 0.04). |
| `texture` | BOOLEAN | Да | True/False | Передает исходное изображение в качестве текстуры baseColor (по умолчанию: True). |

## Выходные данные

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `MESH` | MESH | Трехмерная сетка с вершинами, гранями, UV-координатами и опциональной текстурой из исходного изображения. |
