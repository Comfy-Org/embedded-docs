Узел `CLIPTextEncodeKandinsky5` подготавливает текстовые подсказки для использования с моделью Kandinsky 5. Он принимает два отдельных текстовых входа, токенизирует их с помощью предоставленной модели CLIP и объединяет в единый выходной условный сигнал. Этот выходной сигнал используется для управления процессом генерации изображения.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `clip` | CLIP | Да | | Модель CLIP, используемая для токенизации и кодирования текстовых подсказок. |
| `clip_l` | STRING | Да | | Основная текстовая подсказка. Этот вход поддерживает многострочный текст и динамические подсказки. |
| `qwen25_7b` | STRING | Да | | Дополнительная текстовая подсказка. Этот вход поддерживает многострочный текст и динамические подсказки. |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `CONDITIONING` | CONDITIONING | Объединенные условные данные, сгенерированные из обеих текстовых подсказок, готовые к передаче в модель Kandinsky 5 для генерации изображения. |
