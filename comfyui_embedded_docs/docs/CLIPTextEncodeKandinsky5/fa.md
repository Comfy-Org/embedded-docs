گره CLIPTextEncodeKandinsky5، پرامپت‌های متنی را برای استفاده با مدل Kandinsky 5 آماده می‌کند. این گره دو ورودی متنی مجزا دریافت می‌کند، آن‌ها را با استفاده از یک مدل CLIP ارائه‌شده توکن‌سازی می‌کند و در یک خروجی شرطی‌سازی واحد ترکیب می‌نماید. این خروجی برای هدایت فرآیند تولید تصویر استفاده می‌شود.

## ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|---------|----------|-------|--------|---------|
| `clip` | CLIP | بله | | مدل CLIP مورد استفاده برای توکن‌سازی و رمزگذاری پرامپت‌های متنی. |
| `clip_l` | STRING | بله | | پرامپت متنی اصلی. این ورودی از متن چندخطی و پرامپت‌های پویا پشتیبانی می‌کند. |
| `qwen25_7b` | STRING | بله | | یک پرامپت متنی ثانویه. این ورودی از متن چندخطی و پرامپت‌های پویا پشتیبانی می‌کند. |

## خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-----------|----------|---------|
| `CONDITIONING` | CONDITIONING | داده‌های شرطی‌سازی ترکیبی حاصل از هر دو پرامپت متنی، آماده برای تغذیه به مدل Kandinsky 5 جهت تولید تصویر. |
