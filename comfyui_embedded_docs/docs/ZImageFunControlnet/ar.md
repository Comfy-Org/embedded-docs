تطبق عقدة ZImageFunControlnet شبكة تحكم متخصصة للتأثير على عملية توليد الصور أو تحريرها. تستخدم العقدة نموذجًا أساسيًا، وتصحيح نموذج، ومشفرًا تلقائيًا متغيرًا (VAE)، مما يتيح لك ضبط قوة تأثير التحكم. يمكن لهذه العقدة العمل مع صورة أساسية، وصورة لإعادة التلوين، وقناع لإجراء تعديلات أكثر استهدافًا.

## المدخلات

| المعامل | نوع البيانات | إجباري | النطاق | الوصف |
|-----------|-----------|----------|-------|-------------|
| `النموذج` | MODEL | نعم | - | النموذج الأساسي المستخدم في عملية التوليد. |
| `تصحيح النموذج` | MODEL_PATCH | نعم | - | نموذج تصحيح متخصص يطبق توجيه شبكة التحكم. |
| `vae` | VAE | نعم | - | المشفر التلقائي المتغير المستخدم لتشفير وفك تشفير الصور. |
| `القوة` | FLOAT | نعم | -10.0 إلى 10.0 | قوة تأثير شبكة التحكم. تطبق القيم الموجبة التأثير، بينما يمكن للقيم السالبة عكسه (القيمة الافتراضية: 1.0). |
| `الصورة` | IMAGE | لا | - | صورة أساسية اختيارية لتوجيه عملية التوليد. |
| `صورة التلوين` | IMAGE | لا | - | صورة اختيارية تُستخدم خصيصًا لإعادة تلوين المناطق المحددة بواسطة قناع. |
| `القناع` | MASK | لا | - | قناع اختياري يحدد مناطق الصورة التي يجب تحريرها أو إعادة تلوينها. |

**ملاحظة:** يُستخدم معامل `inpaint_image` عادةً مع `mask` لتحديد محتوى إعادة التلوين. قد يتغير سلوك العقدة بناءً على المدخلات الاختيارية المقدمة (مثل استخدام `image` للتوجيه أو استخدام `image` و`mask` و`inpaint_image` لإعادة التلوين).

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|-------------|-----------|-------------|
| `النموذج` | MODEL | النموذج مع تطبيق تصحيح شبكة التحكم، جاهز للاستخدام في خط أنابيب أخذ العينات. |
| `positive` | CONDITIONING | التكييف الإيجابي، الذي قد يتم تعديله بواسطة مدخلات شبكة التحكم. |
| `negative` | CONDITIONING | التكييف السلبي، الذي قد يتم تعديله بواسطة مدخلات شبكة التحكم. |
