Узел ImageYUVToRGB преобразует изображения из цветового пространства YUV в цветовое пространство RGB. Он принимает три отдельных входных изображения, представляющих каналы Y (яркость), U (синяя проекция) и V (красная проекция), и объединяет их в одно изображение RGB с помощью преобразования цветового пространства.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `Y` | IMAGE | Да | - | Входное изображение канала Y (яркость) |
| `U` | IMAGE | Да | - | Входное изображение канала U (синяя проекция) |
| `V` | IMAGE | Да | - | Входное изображение канала V (красная проекция) |

**Примечание:** Все три входных изображения (Y, U и V) должны подаваться вместе и иметь совместимые размеры для корректного преобразования.

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `output` | IMAGE | Преобразованное изображение RGB |
