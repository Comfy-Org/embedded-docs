بالتأكيد، إليك الترجمة العربية للوثيقة التقنية الخاصة بعقدة `ImageOnlyCheckpointSave` في ComfyUI، مع الالتزام التام بقواعد الترجمة المحددة:

تقوم عقدة `ImageOnlyCheckpointSave` بحفظ ملف نقطة تفتيش (checkpoint) يحتوي على نموذج (model) ومشفر رؤية CLIP (CLIP vision encoder) وشبكة VAE. تقوم هذه العقدة بإنشاء ملف بصيغة safetensors مع بادئة اسم الملف المحددة، وتخزينه في دليل الإخراج. صُممت هذه العقدة خصيصًا لحفظ مكونات النموذج المتعلقة بالصور معًا في ملف نقطة تفتيش واحد.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|-----------|-----------|----------|-------|-------------|
| `model` | MODEL | نعم | - | النموذج المراد حفظه في نقطة التفتيش |
| `clip_vision` | CLIP_VISION | نعم | - | مشفر رؤية CLIP المراد حفظه في نقطة التفتيش |
| `vae` | VAE | نعم | - | شبكة VAE (التشفير التلقائي المتغير) المراد حفظها في نقطة التفتيش |
| `filename_prefix` | STRING | نعم | - | البادئة لاسم ملف الإخراج (الافتراضي: "checkpoints/ComfyUI") |
| `prompt` | PROMPT | لا | - | معامل مخفي لبيانات طلب سير العمل |
| `extra_pnginfo` | EXTRA_PNGINFO | لا | - | معامل مخفي لبيانات وصفية إضافية بصيغة PNG |
