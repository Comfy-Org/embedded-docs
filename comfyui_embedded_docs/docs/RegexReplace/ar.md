# عقد استبدال التعبيرات النمطية (RegexReplace)

يقوم عقدة استبدال التعبيرات النمطية بالبحث عن النصوص واستبدالها في السلاسل النصية باستخدام أنماط التعبيرات النمطية. تتيح لك هذه العقدة البحث عن أنماط نصية واستبدالها بنص جديد، مع خيارات للتحكم في كيفية عمل مطابقة الأنماط بما في ذلك حساسية حالة الأحرف، والمطابقة متعددة الأسطر، وتحديد عدد عمليات الاستبدال.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|---------|---------------|--------|---------|-------------|
| `سلسلة نصية` | STRING | نعم | - | السلسلة النصية المدخلة للبحث والاستبدال داخلها |
| `نمط_التعبير_النمطي` | STRING | نعم | - | نمط التعبير النمطي للبحث عنه في السلسلة النصية المدخلة |
| `استبدال` | STRING | نعم | - | النص البديل لاستبدال الأنماط المطابقة به |
| `غير_حساس_لحالة_الحرف` | BOOLEAN | لا | - | عند التفعيل، يجعل مطابقة الأنماط تتجاهل اختلافات حالة الأحرف (الافتراضي: True) |
| `متعدد_الأسطر` | BOOLEAN | لا | - | عند التفعيل، يغير سلوك ^ و $ للمطابقة في بداية/نهاية كل سطر بدلاً من بداية/نهاية السلسلة النصية بأكملها (الافتراضي: False) |
| `مطابقة_الكل` | BOOLEAN | لا | - | عند التفعيل، ستطابق النقطة (.) أي حرف بما في ذلك أحرف السطر الجديد. عند التعطيل، لن تطابق النقطة أحرف السطر الجديد (الافتراضي: False) |
| `عدد` | INT | لا | 0-100 | الحد الأقصى لعدد عمليات الاستبدال. اضبط على 0 لاستبدال جميع التكرارات (الافتراضي). اضبط على 1 لاستبدال أول تطابق فقط، و2 لأول تطابقين، وهكذا (الافتراضي: 0) |

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|-------------|---------------|-------------|
| `output` | STRING | السلسلة النصية المعدلة مع تطبيق الاستبدالات المحددة |
