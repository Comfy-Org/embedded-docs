گره RegexReplace با استفاده از الگوهای عبارت باقاعده (Regular Expression)، متن را در رشته‌ها جستجو و جایگزین می‌کند. این گره به شما امکان می‌دهد الگوهای متنی را جستجو کرده و با متن جدید جایگزین کنید، با گزینه‌هایی برای کنترل نحوه تطبیق الگو از جمله حساسیت به حروف بزرگ و کوچک، تطبیق چندخطی و محدود کردن تعداد جایگزینی‌ها.

## ورودی‌ها

| پارامتر | نوع داده | اجباری | محدوده | توضیحات |
|---------|----------|--------|--------|---------|
| `string` | STRING | بله | - | رشته متنی ورودی که در آن جستجو و جایگزینی انجام می‌شود |
| `regex_pattern` | STRING | بله | - | الگوی عبارت باقاعده برای جستجو در رشته ورودی |
| `replace` | STRING | بله | - | متن جایگزین برای جایگزینی الگوهای تطبیق‌یافته |
| `case_insensitive` | BOOLEAN | خیر | - | در صورت فعال‌سازی، تطبیق الگو تفاوت حروف بزرگ و کوچک را نادیده می‌گیرد (پیش‌فرض: True) |
| `multiline` | BOOLEAN | خیر | - | در صورت فعال‌سازی، رفتار ^ و $ را تغییر می‌دهد تا در ابتدا/انتهای هر خط تطبیق داده شوند، نه فقط ابتدا/انتهای کل رشته (پیش‌فرض: False) |
| `dotall` | BOOLEAN | خیر | - | در صورت فعال‌سازی، کاراکتر نقطه (.) با هر کاراکتری از جمله کاراکتر خط جدید تطبیق می‌یابد. در صورت غیرفعال‌سازی، نقطه با خطوط جدید تطبیق نمی‌یابد (پیش‌فرض: False) |
| `count` | INT | خیر | 0-100 | حداکثر تعداد جایگزینی‌ها. برای جایگزینی همه موارد، مقدار 0 را تنظیم کنید (پیش‌فرض). برای جایگزینی فقط اولین تطبیق، مقدار 1، برای دو تطبیق اول مقدار 2 و غیره (پیش‌فرض: 0) |

|-----------|----------|---------|
| `output` | STRING | رشته اصلاح‌شده با جایگزینی‌های مشخص‌شده اعمال‌شده |
