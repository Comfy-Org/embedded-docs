Этот узел предназначен для обработки и подготовки данных для использования в моделях StableZero123, с фокусом на приведение входных данных к определенному формату, совместимому и оптимизированному для этих моделей.

## Входные параметры

| Параметр             | Comfy dtype        | Описание |
|-----------------------|--------------------|-------------|
| `clip_vision`         | `CLIP_VISION`      | Обрабатывает визуальные данные для согласования с требованиями модели, улучшая понимание моделью визуального контекста. |
| `init_image`          | `IMAGE`            | Служит начальным изображением для модели, задавая базовый уровень для дальнейших операций с изображениями. |
| `vae`                 | `VAE`              | Интегрирует выходные данные вариационного автоэнкодера, облегчая способность модели генерировать или изменять изображения. |
| `width`               | `INT`              | Задает ширину выходного изображения, позволяя динамически изменять размер в соответствии с потребностями модели. |
| `height`              | `INT`              | Определяет высоту выходного изображения, обеспечивая настройку размеров выходных данных. |
| `batch_size`          | `INT`              | Управляет количеством изображений, обрабатываемых за один пакет, оптимизируя вычислительную эффективность. |
| `elevation`           | `FLOAT`            | Регулирует угол возвышения для рендеринга 3D-модели, улучшая пространственное понимание модели. |
| `azimuth`             | `FLOAT`            | Изменяет азимутальный угол для визуализации 3D-модели, улучшая восприятие моделью ориентации. |

## Выходные параметры

| Параметр     | Тип данных | Описание |
|---------------|--------------|-------------|
| `positive`    | `CONDITIONING` | Генерирует положительные векторы условий, способствуя усилению положительных признаков модели. |
| `negative`    | `CONDITIONING` | Создает отрицательные векторы условий, помогая модели избегать определенных признаков. |
| `latent`      | `LATENT`     | Формирует скрытые представления, облегчая более глубокое понимание данных моделью. |
