Вот перевод документации для узла CLIPTextEncodeLumina2:

Узел CLIP Text Encode для Lumina2 кодирует системный промпт и пользовательский промпт с помощью модели CLIP в эмбеддинг, который может направлять диффузионную модель на генерацию определенных изображений. Он объединяет предопределенный системный промпт с вашим пользовательским текстовым промптом и обрабатывает их через модель CLIP для создания данных кондиционирования для генерации изображений.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `system_prompt` | STRING | Да | `"superior"`<br>`"alignment"` | Lumina2 предоставляет два типа системных промптов: "superior" генерирует изображения с превосходным выравниванием изображения и текста; "alignment" генерирует высококачественные изображения с максимальной степенью выравнивания изображения и текста. |
| `user_prompt` | STRING | Да | Н/Д | Текст для кодирования. Поддерживает многострочный ввод и динамические промпты. |
| `clip` | CLIP | Да | Н/Д | Модель CLIP, используемая для кодирования текста. |

**Примечание:** Входной параметр `clip` является обязательным и не может быть None. Если входной параметр clip недействителен, узел выдаст ошибку, указывающую на то, что контрольная точка может не содержать валидную модель CLIP или текстового энкодера.

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `CONDITIONING` | CONDITIONING | Кондиционирование, содержащее закодированный текст, используемый для направления диффузионной модели. |
