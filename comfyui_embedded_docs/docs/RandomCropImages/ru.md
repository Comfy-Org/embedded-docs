Узел "Random Crop Images" (Случайная обрезка изображений) случайным образом выбирает прямоугольный участок из каждого входного изображения и обрезает его до заданной ширины и высоты. Это часто используется для аугментации данных с целью создания вариаций обучающих изображений. Случайное положение обрезки определяется значением seed, что позволяет воспроизвести ту же самую обрезку.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `image` | IMAGE | Да | - | Изображение, которое будет обрезано. |
| `width` | INT | Нет | 1 - 8192 | Ширина области обрезки (по умолчанию: 512). |
| `height` | INT | Нет | 1 - 8192 | Высота области обрезки (по умолчанию: 512). |
| `seed` | INT | Нет | 0 - 18446744073709551615 | Число, используемое для управления случайным положением обрезки (по умолчанию: 0). |

**Примечание:** Параметры `width` и `height` должны быть меньше или равны размерам входного изображения. Если указанный размер превышает размер изображения, обрезка будет ограничена границами изображения.

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `image` | IMAGE | Результирующее изображение после применения случайной обрезки. |
