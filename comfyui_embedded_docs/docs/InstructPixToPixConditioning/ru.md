Узел InstructPixToPixConditioning подготавливает данные кондиционирования для редактирования изображений InstructPix2Pix, объединяя положительные и отрицательные текстовые подсказки с данными изображения. Он обрабатывает входные изображения через VAE-энкодер для создания латентных представлений и присоединяет эти латентные данные как к положительному, так и к отрицательному кондиционированию. Узел автоматически обрабатывает размеры изображения, обрезая их до значений, кратных 8 пикселям, для обеспечения совместимости с процессом VAE-кодирования.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `positive` | CONDITIONING | Да | - | Положительные данные кондиционирования, содержащие текстовые подсказки и настройки для желаемых характеристик изображения |
| `negative` | CONDITIONING | Да | - | Отрицательные данные кондиционирования, содержащие текстовые подсказки и настройки для нежелательных характеристик изображения |
| `vae` | VAE | Да | - | Модель VAE, используемая для кодирования входных изображений в латентные представления |
| `pixels` | IMAGE | Да | - | Входное изображение, подлежащее обработке и кодированию в латентное пространство |

**Примечание:** Размеры входного изображения автоматически корректируются путем обрезки до ближайшего значения, кратного 8 пикселям по ширине и высоте, для обеспечения совместимости с процессом VAE-кодирования.

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `positive` | CONDITIONING | Положительные данные кондиционирования с присоединенным латентным представлением изображения |
| `negative` | CONDITIONING | Отрицательные данные кондиционирования с присоединенным латентным представлением изображения |
| `latent` | LATENT | Пустой латентный тензор с теми же размерами, что и закодированное изображение |
