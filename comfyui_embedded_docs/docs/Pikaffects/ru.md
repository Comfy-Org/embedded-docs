Вот перевод документации на русский язык:

Узел Pikaffects создаёт видео с различными визуальными эффектами, применяемыми к входному изображению. Он использует API генерации видео Pika для преобразования статичных изображений в анимированные видео с определёнными эффектами, такими как плавление, взрыв или левитация. Для доступа к сервису Pika узлу требуется ключ API и токен аутентификации.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `image` | IMAGE | Да | - | Исходное изображение, к которому применяется эффект Pikaffect. |
| `pikaffect` | COMBO | Да | "Cake-ify"<br>"Crumble"<br>"Crush"<br>"Decapitate"<br>"Deflate"<br>"Dissolve"<br>"Explode"<br>"Eye-pop"<br>"Inflate"<br>"Levitate"<br>"Melt"<br>"Peel"<br>"Poke"<br>"Squish"<br>"Ta-da"<br>"Tear" | Конкретный визуальный эффект, применяемый к изображению (по умолчанию: "Cake-ify"). |
| `prompt_text` | STRING | Да | - | Текстовое описание, направляющее генерацию видео. |
| `negative_prompt` | STRING | Да | - | Текстовое описание того, чего следует избегать в сгенерированном видео. |
| `seed` | INT | Да | от 0 до 4294967295 | Значение случайного зерна для воспроизводимых результатов. |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `output` | VIDEO | Сгенерированное видео с применённым эффектом Pikaffect. |
