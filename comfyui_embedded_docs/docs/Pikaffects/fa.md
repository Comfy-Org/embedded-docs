### مرور کلی

گره Pikaffects ویدیوهایی با افکت‌های بصری متنوع بر روی یک تصویر ورودی تولید می‌کند. این گره از API تولید ویدیوی Pika برای تبدیل تصاویر ثابت به ویدیوهای متحرک با افکت‌های خاص مانند ذوب شدن، انفجار یا شناور شدن استفاده می‌کند. برای دسترسی به سرویس Pika، این گره به یک کلید API و توکن احراز هویت نیاز دارد.

### ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|---------|----------|--------|--------|---------|
| `image` | IMAGE | بله | - | تصویر مرجع برای اعمال Pikaffect. |
| `pikaffect` | COMBO | بله | "Cake-ify"<br>"Crumble"<br>"Crush"<br>"Decapitate"<br>"Deflate"<br>"Dissolve"<br>"Explode"<br>"Eye-pop"<br>"Inflate"<br>"Levitate"<br>"Melt"<br>"Peel"<br>"Poke"<br>"Squish"<br>"Ta-da"<br>"Tear" | افکت بصری خاص برای اعمال روی تصویر (پیش‌فرض: "Cake-ify"). |
| `prompt_text` | STRING | بله | - | توضیحات متنی برای هدایت تولید ویدیو. |
| `negative_prompt` | STRING | بله | - | توضیحات متنی از مواردی که باید در ویدیوی تولید شده اجتناب شود. |
| `seed` | INT | بله | 0 تا 4294967295 | مقدار تصادفی seed برای نتایج قابل تکرار. |

| `output` | VIDEO | ویدیوی تولید شده با Pikaffect اعمال شده. |
