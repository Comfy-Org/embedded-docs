بالتأكيد، إليك الترجمة العربية للوثيقة التقنية الخاصة بعقدة ComfyUI مع الالتزام التام بقواعد الترجمة المحددة:

عقدة Pikaffects تقوم بإنشاء مقاطع فيديو مع تطبيق تأثيرات بصرية متنوعة على صورة الإدخال. تستخدم العقدة واجهة برمجة تطبيقات Pika لتوليد الفيديو لتحويل الصور الثابتة إلى مقاطع فيديو متحركة ذات تأثيرات محددة مثل الذوبان أو الانفجار أو الطفو. تتطلب العقدة مفتاح واجهة برمجة تطبيقات ورمز توثيق للوصول إلى خدمة Pika.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|-----------|-----------|----------|-------|-------------|
| `image` | IMAGE | نعم | - | الصورة المرجعية لتطبيق تأثير Pikaffect عليها. |
| `pikaffect` | COMBO | نعم | "Cake-ify"<br>"Crumble"<br>"Crush"<br>"Decapitate"<br>"Deflate"<br>"Dissolve"<br>"Explode"<br>"Eye-pop"<br>"Inflate"<br>"Levitate"<br>"Melt"<br>"Peel"<br>"Poke"<br>"Squish"<br>"Ta-da"<br>"Tear" | التأثير البصري المحدد لتطبيقه على الصورة (الافتراضي: "Cake-ify"). |
| `prompt_text` | STRING | نعم | - | وصف نصي يوجه عملية توليد الفيديو. |
| `negative_prompt` | STRING | نعم | - | وصف نصي لما يجب تجنبه في الفيديو المُنشأ. |
| `seed` | INT | نعم | 0 إلى 4294967295 | قيمة البذرة العشوائية للحصول على نتائج قابلة للتكرار. |

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|-------------|-----------|-------------|
| `output` | VIDEO | الفيديو المُنشأ مع تطبيق تأثير Pikaffect. |
