Этот узел предназначен для увеличения разрешения изображений с использованием указанной модели апскейла. Он эффективно управляет процессом апскейла, адаптируя изображение к соответствующему устройству, оптимизируя использование памяти и применяя модель апскейла блочным способом для предотвращения потенциальных ошибок нехватки памяти.

## Входы

| Параметр         | Comfy dtype       | Описание                                                                 |
|-------------------|-------------------|----------------------------------------------------------------------------|
| `upscale_model`   | `UPSCALE_MODEL`   | Модель апскейла, используемая для увеличения разрешения изображения. Имеет решающее значение для определения алгоритма апскейла и его параметров. |
| `image`           | `IMAGE`           | Изображение, подлежащее увеличению разрешения. Этот вход необходим для определения исходного содержимого, которое будет подвергнуто процессу апскейла. |

## Выходы

| Параметр | Тип данных | Описание                                        |
|-----------|-------------|----------------------------------------------------|
| `image`   | `IMAGE`     | Увеличенное изображение, обработанное моделью апскейла. Этот выход является результатом операции апскейла, демонстрируя улучшенное разрешение или качество. |
