### 概述

此节点为 AR（自回归）视频模型准备图像到视频的生成配置。它接收起始图像，使用 VAE 将其编码到潜在空间中，并将编码后的图像存储在模型配置中。这使得视频采样过程能够将该图像用作第一帧，从而无需独立的图像到视频模型架构即可有效初始化生成过程。

### 输入

| 参数 | 数据类型 | 是否必需 | 范围 | 描述 |
|-----------|-----------|----------|-------|-------------|
| `model` | MODEL | 是 | - | 用于生成的 AR 视频模型。 |
| `vae` | VAE | 是 | - | 用于将起始图像编码到潜在空间的 VAE 模型。 |
| `start_image` | IMAGE | 是 | - | 将作为生成视频第一帧的初始图像。 |
| `width` | INT | 是 | 16 到 8192（步长：16） | 生成视频帧的宽度（默认值：832）。 |
| `height` | INT | 是 | 16 到 8192（步长：16） | 生成视频帧的高度（默认值：480）。 |
| `length` | INT | 是 | 1 到 1024（步长：4） | 生成视频的总帧数（默认值：81）。 |
| `batch_size` | INT | 是 | 1 到 64 | 单次批处理中生成的视频序列数量（默认值：1）。 |

### 输出

| 输出名称 | 数据类型 | 描述 |
|-------------|-----------|-------------|

| `LATENT` | LATENT | 一个空的潜在张量，其维度与视频生成过程所需尺寸一致。 |
