## Обзор

Этот узел подготавливает конфигурацию для генерации видео из изображения с использованием AR (авторегрессионных) видеомоделей. Он принимает начальное изображение, кодирует его в скрытое пространство с помощью VAE и сохраняет закодированное изображение в конфигурации модели. Это позволяет процессу семплирования видео использовать изображение в качестве первого кадра, эффективно задавая начальные условия генерации без необходимости в отдельной архитектуре модели "изображение-в-видео".

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `model` | MODEL | Да | - | AR видеомодель, используемая для генерации. |
| `vae` | VAE | Да | - | Модель VAE, используемая для кодирования начального изображения в скрытое пространство. |
| `start_image` | IMAGE | Да | - | Исходное изображение, которое будет служить первым кадром генерируемого видео. |
| `width` | INT | Да | от 16 до 8192 (шаг: 16) | Ширина кадров генерируемого видео (по умолчанию: 832). |
| `height` | INT | Да | от 16 до 8192 (шаг: 16) | Высота кадров генерируемого видео (по умолчанию: 480). |
| `length` | INT | Да | от 1 до 1024 (шаг: 4) | Общее количество кадров в генерируемом видео (по умолчанию: 81). |
| `batch_size` | INT | Да | от 1 до 64 | Количество видеопоследовательностей, генерируемых за один пакет (по умолчанию: 1). |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `MODEL` | MODEL | Клонированная модель с закодированным начальным изображением, сохраненным в ее конфигурации для генерации видео. |
| `LATENT` | LATENT | Пустой тензор скрытого пространства с правильными размерами для процесса генерации видео. |
