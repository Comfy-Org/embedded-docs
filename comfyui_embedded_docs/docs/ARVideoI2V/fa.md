## نمای کلی

این گره یک تنظیمات تولید ویدئو از تصویر را برای مدل‌های ویدئویی AR (خودرگرسیو) آماده می‌کند. یک تصویر شروع دریافت کرده، آن را با استفاده از VAE به فضای نهان رمزگذاری می‌کند و تصویر رمزگذاری‌شده را در پیکربندی مدل ذخیره می‌کند. این امکان را فراهم می‌کند که فرآیند نمونه‌گیری ویدئو از تصویر به عنوان فریم اول استفاده کند و عملاً تولید را بدون نیاز به معماری جداگانه مدل تصویر به ویدئو، آغاز نماید.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|---------|----------|--------|--------|---------|
| `model` | MODEL | بله | - | مدل ویدئوی AR که برای تولید استفاده می‌شود. |
| `vae` | VAE | بله | - | مدل VAE که برای رمزگذاری تصویر شروع به فضای نهان استفاده می‌شود. |
| `start_image` | IMAGE | بله | - | تصویر اولیه که به عنوان فریم اول ویدئوی تولیدشده عمل می‌کند. |
| `width` | INT | بله | 16 تا 8192 (گام: 16) | عرض فریم‌های ویدئوی تولیدشده (پیش‌فرض: 832). |
| `height` | INT | بله | 16 تا 8192 (گام: 16) | ارتفاع فریم‌های ویدئوی تولیدشده (پیش‌فرض: 480). |
| `length` | INT | بله | 1 تا 1024 (گام: 4) | تعداد کل فریم‌ها در ویدئوی تولیدشده (پیش‌فرض: 81). |
| `batch_size` | INT | بله | 1 تا 64 | تعداد دنباله‌های ویدئویی که در یک دسته تولید می‌شوند (پیش‌فرض: 1). |

## خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-----------|----------|---------|
| `MODEL` | MODEL | مدل کلون‌شده با تصویر شروع رمزگذاری‌شده ذخیره‌شده در پیکربندی آن برای تولید ویدئو. |
| `LATENT` | LATENT | یک تنسور نهان خالی با ابعاد صحیح برای فرآیند تولید ویدئو. |
