بالتأكيد، إليك الترجمة العربية للوثيقة مع الالتزام التام بقواعد الترجمة المحددة:

## نظرة عامة

تُحضّر هذه العقدة إعدادًا لتوليد فيديو من صورة لنماذج الفيديو ذاتية الانحدار (AR). تبدأ بصورة أولية، وتقوم بتشفيرها إلى الفضاء الكامن باستخدام VAE، ثم تخزّن الصورة المشفرة في تكوين النموذج. يتيح ذلك لعملية أخذ عينات الفيديو استخدام الصورة كإطار أول، مما يغذي عملية التوليد بفعالية دون الحاجة إلى بنية نموذج منفصلة لتحويل الصورة إلى فيديو.

## المدخلات

| المعامل | نوع البيانات | إجباري | النطاق | الوصف |
|-----------|-----------|----------|-------|-------------|
| `model` | MODEL | نعم | - | نموذج الفيديو ذاتي الانحدار (AR) الذي سيُستخدم في التوليد. |
| `vae` | VAE | نعم | - | نموذج VAE المستخدم لتشفير الصورة الأولية إلى الفضاء الكامن. |
| `start_image` | IMAGE | نعم | - | الصورة الأولية التي ستُستخدم كإطار أول للفيديو المُولّد. |
| `width` | INT | نعم | 16 إلى 8192 (خطوة: 16) | عرض إطارات الفيديو المُولّد (الافتراضي: 832). |
| `height` | INT | نعم | 16 إلى 8192 (خطوة: 16) | ارتفاع إطارات الفيديو المُولّد (الافتراضي: 480). |
| `length` | INT | نعم | 1 إلى 1024 (خطوة: 4) | العدد الإجمالي للإطارات في الفيديو المُولّد (الافتراضي: 81). |
| `batch_size` | INT | نعم | 1 إلى 64 | عدد تسلسلات الفيديو المراد توليدها في دفعة واحدة (الافتراضي: 1). |

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|-------------|-----------|-------------|
| `MODEL` | MODEL | النموذج المُستنسخ مع تخزين صورة البداية المشفرة في تكوينه لتوليد الفيديو. |
| `LATENT` | LATENT | موتر كامن فارغ بأبعاد صحيحة لعملية توليد الفيديو. |
