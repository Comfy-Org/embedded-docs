## 概述

此節點為 AR（自迴歸）影片模型準備圖像轉影片的生成設定。它接收起始圖像，使用 VAE 將其編碼到潛在空間中，並將編碼後的圖像儲存在模型的配置中。這使得影片取樣過程能夠將該圖像作為第一幀使用，從而有效引導生成過程，無需使用獨立的圖像轉影片模型架構。
