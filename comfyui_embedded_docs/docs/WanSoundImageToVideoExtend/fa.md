گره `WanSoundImageToVideoExtend` با تولید فریم‌های اضافی، یک ویدئوی نهفته (latent) موجود را گسترش می‌دهد. این کار به‌صورت اختیاری با راهنمایی صدا، یک تصویر مرجع و یک ویدئوی کنترلی انجام می‌شود. این گره یک ویدئوی نهفته اولیه دریافت کرده و با استفاده از شرایط‌دهی (conditioning) و نشانه‌های صوتی ارائه‌شده برای تأثیرگذاری بر محتوای جدید، یک دنباله ویدئویی طولانی‌تر تولید می‌کند.

## ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `مثبت` | CONDITIONING | بله | - |提示‌های شرطی مثبت که مشخص می‌کنند ویدئو چه محتوایی باید داشته باشد |
| `منفی` | CONDITIONING | بله | - |提示‌های شرطی منفی که مشخص می‌کنند ویدئو از چه محتوایی باید پرهیز کند |
| `vae` | VAE | بله | - | رمزگذار خودکار متغیر (VAE) که برای رمزگذاری و رمزگشایی فریم‌های ویدئو استفاده می‌شود |
| `طول` | INT | بله | 1 تا MAX_RESOLUTION | تعداد کل فریم‌هایی که برای دنباله ویدئو تولید می‌شود (پیش‌فرض: 77، گام: 4) |
| `ویدیو latent` | LATENT | بله | - | نمایش نهفته ویدئوی اولیه که به عنوان نقطه شروع برای گسترش عمل می‌کند |
| `خروجی انکودر صوتی` | AUDIOENCODEROUTPUT | خیر | - | جاسازی‌های صوتی اختیاری که می‌توانند بر اساس ویژگی‌های صدا بر تولید ویدئو تأثیر بگذارند |
| `تصویر مرجع` | IMAGE | خیر | - | تصویر مرجع اختیاری که راهنمایی بصری برای تولید ویدئو ارائه می‌دهد |
| `کنترل ویدیو` | IMAGE | خیر | - | ویدئوی کنترلی اختیاری که می‌تواند حرکت و سبک ویدئوی تولید شده را هدایت کند |
