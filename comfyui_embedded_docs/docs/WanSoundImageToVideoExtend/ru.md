Вот перевод документации на русский язык:

Узел WanSoundImageToVideoExtend расширяет существующий видеолатент, генерируя дополнительные кадры, опционально направляемые аудио, референсным изображением и управляющим видео. Он принимает начальный видеолатент и создает более длинную видеопоследовательность, используя предоставленные условия и аудиосигналы для влияния на новый контент.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `positive` | CONDITIONING | Да | - | Позитивные условия, направляющие, что должно быть включено в видео |
| `negative` | CONDITIONING | Да | - | Негативные условия, указывающие, чего следует избегать в видео |
| `vae` | VAE | Да | - | Вариационный автоэнкодер, используемый для кодирования и декодирования видеокадров |
| `length` | INT | Да | от 1 до MAX_RESOLUTION | Общее количество кадров для генерации видеопоследовательности (по умолчанию: 77, шаг: 4) |
| `video_latent` | LATENT | Да | - | Исходное представление видеолатента, служащее отправной точкой для расширения |
| `audio_encoder_output` | AUDIOENCODEROUTPUT | Нет | - | Опциональные аудиовекторы, которые могут влиять на генерацию видео на основе звуковых характеристик |
| `ref_image` | IMAGE | Нет | - | Опциональное референсное изображение, обеспечивающее визуальное направление для генерации видео |
| `control_video` | IMAGE | Нет | - | Опциональное управляющее видео, которое может направлять движение и стиль генерируемого видео |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `positive` | CONDITIONING | Обработанные позитивные условия с примененным видеоконтекстом |
| `negative` | CONDITIONING | Обработанные негативные условия с примененным видеоконтекстом |
| `latent` | LATENT | Сгенерированное представление видеолатента, содержащее расширенную видеопоследовательность |
