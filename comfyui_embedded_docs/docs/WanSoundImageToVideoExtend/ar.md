# ترجمة وثيقة عقدة WanSoundImageToVideoExtend

تعمل عقدة WanSoundImageToVideoExtend على توسيع كامن الفيديو الموجود عن طريق توليد إطارات إضافية، مع إمكانية الاسترشاد بالصوت وصورة مرجعية وفيديو تحكم. تأخذ هذه العقدة كامن فيديو ابتدائي وتنتج تسلسل فيديو أطول، مستخدمةً التكييف والإشارات الصوتية المقدمة للتأثير على المحتوى الجديد.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|----------|---------------|--------|---------|---------|
| `positive` | CONDITIONING | نعم | - | مطالبات التكييف الإيجابية التي توجه محتوى الفيديو |
| `negative` | CONDITIONING | نعم | - | مطالبات التكييف السلبية التي تحدد ما يجب أن يتجنبه الفيديو |
| `vae` | VAE | نعم | - | المشفر التلقائي المتغير المستخدم لتشفير وفك تشفير إطارات الفيديو |
| `length` | INT | نعم | 1 إلى MAX_RESOLUTION | العدد الإجمالي للإطارات المراد توليدها لتسلسل الفيديو (الافتراضي: 77، الخطوة: 4) |
| `video_latent` | LATENT | نعم | - | تمثيل كامن الفيديو الابتدائي الذي يعمل كنقطة بداية للتوسيع |
| `audio_encoder_output` | AUDIOENCODEROUTPUT | لا | - | تضمينات صوتية اختيارية يمكنها التأثير على توليد الفيديو بناءً على خصائص الصوت |
| `ref_image` | IMAGE | لا | - | صورة مرجعية اختيارية توفر توجيهاً بصرياً لتوليد الفيديو |
| `control_video` | IMAGE | لا | - | فيديو تحكم اختياري يمكنه توجيه الحركة والأسلوب للفيديو المولد |

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|-------------|---------------|---------|
| `positive` | CONDITIONING | التكييف الإيجابي المعالج مع تطبيق سياق الفيديو |
| `negative` | CONDITIONING | التكييف السلبي المعالج مع تطبيق سياق الفيديو |
| `latent` | LATENT | تمثيل كامن الفيديو المولد الذي يحتوي على تسلسل الفيديو الموسع |
