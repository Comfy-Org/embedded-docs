Масштабирование изображения в синхронном режиме. Улучшает заданное растровое изображение с помощью инструмента «четкое масштабирование», увеличивая разрешение изображения, делая его более четким и чистым.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `image` | IMAGE | Да | - | Входное изображение для масштабирования. Принимает пакет изображений. |

## Выходы

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `image` | IMAGE | Масштабированное изображение с улучшенным разрешением и четкостью. Возвращает пакет изображений, если на вход был подан пакет. |
