# ترجمة وثيقة عقدة SaveWEBM

عقدة SaveWEBM تحفظ سلسلة من الصور كملف فيديو بصيغة WEBM. تستقبل العقدة صور إدخال متعددة وتقوم بترميزها إلى فيديو باستخدام أحد برامج الترميز VP9 أو AV1 مع إعدادات جودة قابلة للتعديل ومعدل إطارات. يتم حفظ ملف الفيديو الناتج في دليل الإخراج مع بيانات وصفية تتضمن معلومات الموجه (prompt).

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|---------|---------------|--------|---------|-------|
| `الصور` | IMAGE | نعم | - | سلسلة صور الإدخال لترميزها كإطارات فيديو |
| `بادئة اسم الملف` | STRING | لا | - | بادئة اسم ملف الإخراج (الافتراضي: "ComfyUI") |
| `الترميز` | COMBO | نعم | "vp9"<br>"av1" | برنامج الترميز المستخدم لتشفير الفيديو |
| `معدل الإطارات في الثانية` | FLOAT | لا | 0.01-1000.0 | معدل الإطارات لفيديو الإخراج (الافتراضي: 24.0) |
| `crf` | FLOAT | لا | 0-63.0 | إعداد الجودة حيث تعني قيمة crf الأعلى جودة أقل مع حجم ملف أصغر، وقيمة crf الأقل تعني جودة أعلى مع حجم ملف أكبر (الافتراضي: 32.0) |

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|-------------|---------------|-------|
| `ui` | PREVIEW | معاينة الفيديو التي تعرض ملف WEBM المحفوظ |
