Узел SaveWEBM сохраняет последовательность изображений в видеофайл формата WEBM. Он принимает несколько входных изображений и кодирует их в видео с использованием кодека VP9 или AV1 с настраиваемыми параметрами качества и частоты кадров. Полученный видеофайл сохраняется в выходной каталог с метаданными, включающими информацию о запросе.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `изображения` | IMAGE | Да | - | Последовательность входных изображений для кодирования в видеокадры |
| `префикс_имени_файла` | STRING | Нет | - | Префикс для имени выходного файла (по умолчанию: "ComfyUI") |
| `кодек` | COMBO | Да | "vp9"<br>"av1" | Видеокодек, используемый для кодирования |
| `fps` | FLOAT | Нет | 0.01-1000.0 | Частота кадров выходного видео (по умолчанию: 24.0) |
| `crf` | FLOAT | Нет | 0-63.0 | Настройка качества: более высокое значение crf означает более низкое качество и меньший размер файла, более низкое значение crf означает более высокое качество и больший размер файла (по умолчанию: 32.0) |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `ui` | PREVIEW | Предварительный просмотр видео, показывающий сохраненный файл WEBM |
