Реализует метод эпсилон-масштабирования (Epsilon Scaling) из научной статьи "Elucidating the Exposure Bias in Diffusion Models". Этот метод повышает качество выборки путем масштабирования предсказанного шума в процессе сэмплирования. Используется равномерное расписание для смягчения эффекта смещения экспозиции в диффузионных моделях.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `model` | MODEL | Да | - | Модель, к которой применяется эпсилон-масштабирование |
| `scaling_factor` | FLOAT | Нет | 0.5 - 1.5 | Коэффициент масштабирования предсказанного шума (по умолчанию: 1.005) |

## Выходные данные

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `model` | MODEL | Модель с примененным эпсилон-масштабированием |
