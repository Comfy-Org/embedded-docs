گره بهبود پوست تصویر Magnific Image Skin Enhancer، پردازش تخصصی مبتنی بر هوش مصنوعی را روی تصاویر پرتره اعمال می‌کند تا ظاهر پوست را بهبود بخشد. این گره سه حالت مجزا برای اهداف مختلف بهبود ارائه می‌دهد: حالت خلاقانه برای جلوه‌های هنری، حالت وفادار برای حفظ ظاهر اصلی، و حالت انعطاف‌پذیر برای بهبودهای هدفمند مانند نورپردازی یا واقع‌گرایی. این گره تصویر را برای پردازش به یک API خارجی ارسال می‌کند و نتیجه بهبودیافته را بازمی‌گرداند.

## ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|---------|----------|-------|--------|---------|
| `image` | IMAGE | بله | - | تصویر پرتره برای بهبود. |
| `sharpen` | INT | خیر | 0 تا 100 | شدت وضوح (پیش‌فرض: 0). |
| `smart_grain` | INT | خیر | 0 تا 100 | شدت دانه‌بندی هوشمند (پیش‌فرض: 2). |
| `mode` | COMBO | بله | `"creative"`<br>`"faithful"`<br>`"flexible"` | حالت پردازش مورد استفاده. `"creative"` برای بهبود هنری، `"faithful"` برای حفظ ظاهر اصلی، و `"flexible"` برای بهینه‌سازی هدفمند است. |
| `skin_detail` | INT | خیر | 0 تا 100 | سطح بهبود جزئیات پوست. این ورودی فقط زمانی در دسترس و ضروری است که `mode` روی `"faithful"` تنظیم شده باشد (پیش‌فرض: 80). |
| `optimized_for` | COMBO | خیر | `"enhance_skin"`<br>`"improve_lighting"`<br>`"enhance_everything"`<br>`"transform_to_real"`<br>`"no_make_up"` | هدف بهینه‌سازی بهبود. این ورودی فقط زمانی در دسترس و ضروری است که `mode` روی `"flexible"` تنظیم شده باشد. |

**محدودیت‌ها:**

* گره دقیقاً یک تصویر ورودی دریافت می‌کند.
* تصویر ورودی باید حداقل ارتفاع و عرض ۱۶۰ پیکسل داشته باشد.
* نسبت ابعاد تصویر ورودی باید بین ۱:۳ و ۳:۱ باشد (اعتبارسنجی غیرسختگیرانه).
* پارامتر `skin_detail` فقط زمانی فعال است که `mode` روی `"faithful"` تنظیم شده باشد.
* پارامتر `optimized_for` فقط زمانی فعال است که `mode` روی `"flexible"` تنظیم شده باشد.
