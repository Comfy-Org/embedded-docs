گره **Kling Image(First Frame) to Video with Audio** از مدل هوش مصنوعی Kling برای تولید یک ویدیوی کوتاه از یک تصویر شروع و یک توضیح متنی استفاده می‌کند. این گره یک دنباله ویدیویی ایجاد می‌کند که با تصویر ارائه‌شده آغاز می‌شود و می‌تواند به‌صورت اختیاری شامل صدای تولیدشده توسط هوش مصنوعی برای همراهی با تصاویر باشد.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|---------|----------|--------|--------|---------|
| `model_name` | COMBO | بله | `"kling-v2-6"` | نسخه خاص مدل هوش مصنوعی Kling که برای تولید ویدیو استفاده می‌شود. |
| `start_frame` | IMAGE | بله | - | تصویری که به‌عنوان فریم اول ویدیوی تولیدشده عمل می‌کند. تصویر باید حداقل ۳۰۰×۳۰۰ پیکسل باشد و نسبت ابعادی بین ۱:۲.۵ تا ۲.۵:۱ داشته باشد. |
| `prompt` | STRING | بله | - | توضیح متنی مثبت. این محتوای ویدیویی را که می‌خواهید تولید کنید توصیف می‌کند. توضیح باید بین ۱ تا ۲۵۰۰ کاراکتر باشد. |
| `mode` | COMBO | بله | `"pro"` | حالت عملیاتی برای تولید ویدیو. |
| `duration` | COMBO | بله | `5`<br>`10` | طول ویدیوی تولیدشده بر حسب ثانیه. |
| `generate_audio` | BOOLEAN | خیر | - | در صورت فعال بودن، گره صدا را برای همراهی با ویدیو تولید می‌کند. در صورت غیرفعال بودن، ویدیو بی‌صدا خواهد بود. (پیش‌فرض: True) |

## خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-----------|----------|---------|
| `video` | VIDEO | فایل ویدیویی تولیدشده که بسته به ورودی `generate_audio` ممکن است شامل صدا باشد. |
