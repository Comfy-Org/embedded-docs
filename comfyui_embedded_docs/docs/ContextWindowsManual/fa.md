### مرور کلی

گره "پنجره‌های زمینه (دستی)" به شما امکان می‌دهد تا به‌صورت دستی پنجره‌های زمینه را برای مدل‌ها در طول نمونه‌برداری پیکربندی کنید. این گره بخش‌های زمینه هم‌پوشان با طول، هم‌پوشانی و الگوهای زمان‌بندی مشخص ایجاد می‌کند تا داده‌ها را در بخش‌های قابل مدیریت پردازش کرده و در عین حال پیوستگی بین بخش‌ها را حفظ کند. این گره گزینه‌های پیشرفته‌ای برای کنترل نحوه اعمال پنجره‌های زمینه، از جمله به‌هم‌ریختگی نویز، حفظ شرطی‌سازی و رفع پنجره‌های علی ارائه می‌دهد.

## ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `model` | MODEL | بله | - | مدلی که در طول نمونه‌برداری پنجره‌های زمینه روی آن اعمال می‌شود. |
| `context_length` | INT | خیر | 1+ | طول پنجره زمینه (پیش‌فرض: 16). |
| `context_overlap` | INT | خیر | 0+ | هم‌پوشانی پنجره زمینه (پیش‌فرض: 4). |
| `context_schedule` | COMBO | خیر | `STATIC_STANDARD`<br>`UNIFORM_STANDARD`<br>`UNIFORM_LOOPED`<br>`BATCHED` | گام پنجره زمینه. |
| `context_stride` | INT | خیر | 1+ | گام پنجره زمینه؛ فقط برای زمان‌بندی‌های یکنواخت قابل اعمال است (پیش‌فرض: 1). |
| `closed_loop` | BOOLEAN | خیر | - | آیا حلقه پنجره زمینه بسته شود؛ فقط برای زمان‌بندی‌های حلقه‌ای قابل اعمال است (پیش‌فرض: False). |
| `fuse_method` | COMBO | خیر | `PYRAMID`<br>`LIST_STATIC` | روش مورد استفاده برای ادغام پنجره‌های زمینه (پیش‌فرض: PYRAMID). |
| `dim` | INT | خیر | 0-5 | بعدی که پنجره‌های زمینه روی آن اعمال می‌شود (پیش‌فرض: 0). |
| `freenoise` | BOOLEAN | خیر | - | آیا به‌هم‌ریختگی نویز FreeNoise اعمال شود، ترکیب پنجره را بهبود می‌بخشد (پیش‌فرض: False). |
| `cond_retain_index_list` | STRING | خیر | - | فهرستی از ایندکس‌های نهفته برای نگهداری در تنسورهای شرطی‌سازی برای هر پنجره؛ برای مثال تنظیم این مقدار به '0' از تصویر شروع اولیه برای هر پنجره استفاده می‌کند (پیش‌فرض: ""). |
| `split_conds_to_windows` | BOOLEAN | خیر | - | آیا شرطی‌سازی‌های چندگانه (ایجاد شده توسط ConditionCombine) بر اساس ایندکس ناحیه به هر پنجره تقسیم شود (پیش‌فرض: False). |
| `causal_window_fix` | BOOLEAN | خیر | - | آیا یک فریم رفع علی به پنجره‌های زمینه با ایندکس غیرصفر اضافه شود (پیش‌فرض: True). |

**محدودیت‌های پارامتر:**

- `context_stride` فقط زمانی استفاده می‌شود که زمان‌بندی‌های یکنواخت انتخاب شده باشند
- `closed_loop` فقط برای زمان‌بندی‌های حلقه‌ای قابل اعمال است
- `dim` باید بین 0 تا 5 (شامل) باشد
- `cond_retain_index_list` یک رشته شامل فهرستی از ایندکس‌های صحیح جدا شده با کاما را دریافت می‌کند (مثلاً "0,1,2")
