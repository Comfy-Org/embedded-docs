Вот перевод документации на русский язык в соответствии с вашими правилами:

Узел «Контекстные окна (вручную)» позволяет вручную настраивать контекстные окна для моделей во время семплирования. Он создает перекрывающиеся сегменты контекста с заданной длиной, перекрытием и шаблонами планирования для обработки данных управляемыми блоками, сохраняя при этом непрерывность между сегментами. Этот узел предоставляет расширенные возможности для управления применением контекстных окон, включая перемешивание шума, сохранение кондиционирования и исправление каузальных окон.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `model` | MODEL | Да | - | Модель, к которой применяются контекстные окна во время семплирования. |
| `context_length` | INT | Нет | 1+ | Длина контекстного окна (по умолчанию: 16). |
| `context_overlap` | INT | Нет | 0+ | Перекрытие контекстного окна (по умолчанию: 4). |
| `context_schedule` | COMBO | Нет | `STATIC_STANDARD`<br>`UNIFORM_STANDARD`<br>`UNIFORM_LOOPED`<br>`BATCHED` | Шаг контекстного окна. |
| `context_stride` | INT | Нет | 1+ | Шаг контекстного окна; применяется только для равномерных расписаний (по умолчанию: 1). |
| `closed_loop` | BOOLEAN | Нет | - | Замыкать ли цикл контекстного окна; применяется только для зацикленных расписаний (по умолчанию: False). |
| `fuse_method` | COMBO | Нет | `PYRAMID`<br>`LIST_STATIC` | Метод слияния контекстных окон (по умолчанию: PYRAMID). |
| `dim` | INT | Нет | 0-5 | Измерение, к которому применяются контекстные окна (по умолчанию: 0). |
| `freenoise` | BOOLEAN | Нет | - | Применять ли перемешивание шума FreeNoise, улучшает смешивание окон (по умолчанию: False). |
| `cond_retain_index_list` | STRING | Нет | - | Список индексов скрытых состояний, которые необходимо сохранить в тензорах кондиционирования для каждого окна; например, установка значения '0' будет использовать начальное изображение для каждого окна (по умолчанию: ""). |
| `split_conds_to_windows` | BOOLEAN | Нет | - | Разделять ли множественные кондиционирования (созданные узлом ConditionCombine) на каждое окно на основе индекса региона (по умолчанию: False). |
| `causal_window_fix` | BOOLEAN | Нет | - | Добавлять ли каузальный фиксированный кадр к контекстным окнам с ненулевым индексом (по умолчанию: True). |

**Ограничения параметров:**

- `context_stride` используется только при выборе равномерных расписаний
- `closed_loop` применяется только к зацикленным расписаниям
- `dim` должен быть в диапазоне от 0 до 5 включительно
- `cond_retain_index_list` ожидает строку, содержащую список целочисленных индексов, разделенных запятыми (например, "0,1,2")
