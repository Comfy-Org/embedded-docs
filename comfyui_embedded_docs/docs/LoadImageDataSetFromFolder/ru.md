Вот перевод документации на русский язык:

Этот узел загружает несколько изображений из указанной подпапки в директории ввода ComfyUI. Он сканирует выбранную папку на наличие распространённых типов файлов изображений и возвращает их в виде списка, что полезно для пакетной обработки или подготовки наборов данных.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `folder` | STRING | Да | *Доступно несколько опций* | Папка для загрузки изображений. Опциями являются подпапки, находящиеся в основной директории ввода ComfyUI. |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `images` | IMAGE | Список загруженных изображений. Узел загружает все допустимые файлы изображений (PNG, JPG, JPEG, WEBP), найденные в выбранной папке. |
