# معاينة سحابة النقاط

تتيح لك عقدة معاينة سحابة النقاط عرض ملف سحابة نقاط ثلاثي الأبعاد داخل واجهة ComfyUI. تقوم بحفظ سحابة النقاط في ملف مؤقت وعرضها في نافذة معاينة ثلاثية الأبعاد، مع تمرير بيانات النموذج وإعدادات منفذ العرض لمزيد من المعالجة.

## المدخلات

| المعامل | الوصف | نوع البيانات | إلزامي | النطاق |
|-----------|-------------|-----------|----------|-------|
| `model_3d` | ملف سحابة النقاط (.ply) | FILE3D | نعم | - |
| `model_3d_info` | معلومات حول النموذج ثلاثي الأبعاد | LOAD3DMODELINFO | لا | - |
| `viewport_state` | حالة منفذ العرض الحالية | LOAD3D | نعم | - |
| `camera_info` | معلومات الكاميرا للعرض ثلاثي الأبعاد | LOAD3DCAMERA | لا | - |
| `width` | عرض نافذة المعاينة (الافتراضي: 1024) | INT | نعم | 1 إلى 4096 |
| `height` | ارتفاع نافذة المعاينة (الافتراضي: 1024) | INT | نعم | 1 إلى 4096 |

## المخرجات

| اسم المخرج | الوصف | نوع البيانات |
|-------------|-------------|-----------|
| `model_3d` | بيانات نموذج سحابة النقاط | FILE3D |
| `model_3d_info` | معلومات حول النموذج ثلاثي الأبعاد | LOAD3DMODELINFO |
| `camera_info` | معلومات الكاميرا للعرض ثلاثي الأبعاد | LOAD3DCAMERA |
| `width` | عرض نافذة المعاينة | INT |
| `height` | ارتفاع نافذة المعاينة | INT |
