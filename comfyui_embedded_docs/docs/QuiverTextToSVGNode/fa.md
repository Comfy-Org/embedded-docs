گره Quiver Text to SVG با استفاده از مدل‌های Quiver AI، یک تصویر گرافیک برداری مقیاس‌پذیر (SVG) از توضیحات متنی تولید می‌کند. می‌توانید به‌صورت اختیاری تصاویر مرجع و دستورالعمل‌های سبک را برای هدایت فرآیند تولید ارائه دهید.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|---------|----------|--------|--------|---------|
| `prompt` | STRING | بله | نامشخص | توضیحات متنی از خروجی SVG مورد نظر. این دستورالعمل اصلی برای تولید است. |
| `instructions` | STRING | خیر | نامشخص | راهنمایی سبک یا قالب‌بندی اضافی. این یک پارامتر پیشرفته و اختیاری است. |
| `reference_images` | IMAGE | خیر | ۰ تا ۴ تصویر | حداکثر ۴ تصویر مرجع برای هدایت تولید. این یک ورودی اختیاری است. |
| `model` | COMBO | بله | `"Quiver SVG v1"`<br>`"Quiver SVG v1 Max"`<br>`"Quiver SVG v1 Preview"` | مدل مورد استفاده برای تولید SVG. گزینه‌های موجود توسط API Quiver تعیین می‌شوند. |
| `seed` | INT | بله | ۰ تا ۲۱۴۷۴۸۳۶۴۷ | دانه (seed) برای تعیین اینکه آیا گره باید دوباره اجرا شود؛ نتایج واقعی بدون توجه به seed غیرقطعی هستند. پیش‌فرض: ۰. |

**نکته:** ورودی `reference_images` حداکثر ۴ تصویر را می‌پذیرد. اگر تعداد بیشتری ارائه شود، گره خطا ایجاد می‌کند.
