### تقسیم تصویر به فهرست کاشی‌ها

گره «تقسیم تصویر به فهرست کاشی‌ها» یک تصویر ورودی واحد را به مجموعه‌ای از بخش‌های مستطیلی کوچک‌تر و هم‌پوشان به نام کاشی تقسیم می‌کند. این گره یک فهرست دسته‌ای از این کاشی‌ها ایجاد می‌کند که می‌توانند توسط سایر گره‌ها به‌صورت جداگانه پردازش شوند. اندازه هر کاشی و میزان هم‌پوشانی بین آن‌ها قابل تنظیم است.

## ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|---------|----------|-------|--------|---------|
| `image` | IMAGE | بله | - | تصویر ورودی که قرار است به کاشی‌ها تقسیم شود. |
| `tile_width` | INT | بله | 64 تا 1048576 | عرض هر کاشی خروجی بر حسب پیکسل (پیش‌فرض: 1024). |
| `tile_height` | INT | بله | 64 تا 1048576 | ارتفاع هر کاشی خروجی بر حسب پیکسل (پیش‌فرض: 1024). |
| `overlap` | INT | بله | 0 تا 4096 | تعداد پیکسل‌هایی که کاشی‌های مجاور روی هم هم‌پوشانی دارند (پیش‌فرض: 128). |
