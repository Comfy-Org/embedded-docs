Вот перевод документации на русский язык:

Узел Split Image into List of Tiles разделяет одно входное изображение на серию меньших, перекрывающихся прямоугольных областей, называемых тайлами. Он создает пакетный список этих тайлов, которые могут быть обработаны по отдельности другими узлами. Можно задать размер каждого тайла и величину перекрытия между ними.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `image` | IMAGE | Да | - | Входное изображение, которое будет разделено на тайлы. |
| `tile_width` | INT | Да | от 64 до 1048576 | Ширина каждого выходного тайла в пикселях (по умолчанию: 1024). |
| `tile_height` | INT | Да | от 64 до 1048576 | Высота каждого выходного тайла в пикселях (по умолчанию: 1024). |
| `overlap` | INT | Да | от 0 до 4096 | Количество пикселей, на которое будут перекрываться соседние тайлы (по умолчанию: 128). |

## Выходные параметры

| Имя выходного параметра | Тип данных | Описание |
|-------------|-----------|-------------|
| `image` | IMAGE | Пакетный список, содержащий все отдельные тайлы изображения. |
