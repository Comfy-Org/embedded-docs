گره Ideogram V3 با استفاده از مدل Ideogram V3 تصاویر تولید می‌کند. این گره هم از تولید تصویر معمولی بر اساس توضیحات متنی و هم از ویرایش تصویر در صورت ارائه همزمان تصویر و ماسک پشتیبانی می‌کند. کنترل‌های مختلفی برای نسبت ابعاد، وضوح، سرعت تولید و تصاویر مرجع شخصیت اختیاری ارائه می‌دهد.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|---------|----------|--------|--------|---------|
| `پرامپت` | STRING | بله | - | توضیحات برای تولید یا ویرایش تصویر (پیش‌فرض: خالی) |
| `تصویر` | IMAGE | خیر | - | تصویر مرجع اختیاری برای ویرایش تصویر |
| `ماسک` | MASK | خیر | - | ماسک اختیاری برای نقاشی درون‌تصویری (نواحی سفید جایگزین می‌شوند) |
| `نسبت ابعاد` | COMBO | خیر | "1:1"<br>"16:9"<br>"9:16"<br>"4:3"<br>"3:4"<br>"3:2"<br>"2:3" | نسبت ابعاد برای تولید تصویر. اگر وضوح روی Auto تنظیم نشده باشد نادیده گرفته می‌شود (پیش‌فرض: "1:1") |
| `وضوح` | COMBO | خیر | "Auto"<br>"1024x1024"<br>"1152x896"<br>"896x1152"<br>"1216x832"<br>"832x1216"<br>"1344x768"<br>"768x1344"<br>"1536x640"<br>"640x1536" | وضوح برای تولید تصویر. اگر روی Auto تنظیم نشده باشد، تنظیم aspect_ratio را نادیده می‌گیرد (پیش‌فرض: "Auto") |
| `گزینه MagicPrompt` | COMBO | خیر | "AUTO"<br>"ON"<br>"OFF" | تعیین می‌کند که آیا از MagicPrompt در تولید استفاده شود (پیش‌فرض: "AUTO") |
| `بذر` | INT | خیر | 0-2147483647 | دانه تصادفی برای تولید (پیش‌فرض: 0) |
| `تعداد تصاویر` | INT | خیر | 1-8 | تعداد تصاویر برای تولید (پیش‌فرض: 1) |
| `سرعت رندر` | COMBO | خیر | "DEFAULT"<br>"TURBO"<br>"QUALITY" | تعادل بین سرعت تولید و کیفیت را کنترل می‌کند (پیش‌فرض: "DEFAULT") |
| `تصویر کاراکتر` | IMAGE | خیر | - | تصویری که به عنوان مرجع شخصیت استفاده می‌شود |
| `ماسک کاراکتر` | MASK | خیر | - | ماسک اختیاری برای تصویر مرجع شخصیت |

**محدودیت‌های پارامتر:**

- هنگامی که هر دو `image` و `mask` ارائه شوند، گره به حالت ویرایش تغییر می‌کند
- اگر تنها یکی از `image` یا `mask` ارائه شود، خطا رخ می‌دهد
- `character_mask` نیازمند وجود `character_image` است
- پارامتر `aspect_ratio` زمانی که `resolution` روی "Auto" تنظیم نشده باشد نادیده گرفته می‌شود
- نواحی سفید در ماسک در طول نقاشی درون‌تصویری جایگزین می‌شوند
- ماسک شخصیت و تصویر شخصیت باید اندازه یکسانی داشته باشند
