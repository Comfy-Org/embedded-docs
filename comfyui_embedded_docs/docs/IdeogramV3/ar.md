# ترجمة وثيقة عقدة Ideogram V3

عقدة Ideogram V3 تُنشئ الصور باستخدام نموذج Ideogram V3. تدعم العقدة كلاً من توليد الصور العادي من النصوص الوصفية وتحرير الصور عند توفير كل من الصورة والقناع. توفر العقدة تحكمات متنوعة لنسبة الأبعاد، الدقة، سرعة التوليد، وصور مرجعية اختيارية للشخصيات.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|----------|---------------|--------|---------|-------|
| `prompt` | STRING | نعم | - | النص الوصفي لتوليد الصورة أو تحريرها (الافتراضي: فارغ) |
| `image` | IMAGE | لا | - | صورة مرجعية اختيارية لتحرير الصورة |
| `mask` | MASK | لا | - | قناع اختياري للرسم الداخلي (سيتم استبدال المناطق البيضاء) |
| `aspect_ratio` | COMBO | لا | "1:1"<br>"16:9"<br>"9:16"<br>"4:3"<br>"3:4"<br>"3:2"<br>"2:3" | نسبة الأبعاد لتوليد الصورة. يتم تجاهلها إذا لم تكن الدقة مضبوطة على تلقائي (الافتراضي: "1:1") |
| `resolution` | COMBO | لا | "Auto"<br>"1024x1024"<br>"1152x896"<br>"896x1152"<br>"1216x832"<br>"832x1216"<br>"1344x768"<br>"768x1344"<br>"1536x640"<br>"640x1536" | دقة توليد الصورة. إذا لم تكن مضبوطة على تلقائي، فإنها تتجاوز إعداد نسبة الأبعاد (الافتراضي: "Auto") |
| `magic_prompt_option` | COMBO | لا | "AUTO"<br>"ON"<br>"OFF" | تحديد ما إذا كان سيتم استخدام MagicPrompt في التوليد (الافتراضي: "AUTO") |
| `seed` | INT | لا | 0-2147483647 | البذرة العشوائية للتوليد (الافتراضي: 0) |
| `num_images` | INT | لا | 1-8 | عدد الصور المراد توليدها (الافتراضي: 1) |
| `rendering_speed` | COMBO | لا | "DEFAULT"<br>"TURBO"<br>"QUALITY" | يتحكم في المفاضلة بين سرعة التوليد والجودة (الافتراضي: "DEFAULT") |
| `character_image` | IMAGE | لا | - | صورة لاستخدامها كمرجع للشخصية |
| `character_mask` | MASK | لا | - | قناع اختياري لصورة مرجع الشخصية |

**قيود المعاملات:**

- عند توفير كل من `image` و `mask`، تتحول العقدة إلى وضع التحرير
- إذا تم توفير واحد فقط من `image` أو `mask`، سيحدث خطأ
- `character_mask` يتطلب وجود `character_image`
- يتم تجاهل معامل `aspect_ratio` عندما لا تكون `resolution` مضبوطة على "Auto"
- سيتم استبدال المناطق البيضاء في القناع أثناء الرسم الداخلي
- يجب أن يكون قناع الشخصية وصورة الشخصية بنفس الحجم
