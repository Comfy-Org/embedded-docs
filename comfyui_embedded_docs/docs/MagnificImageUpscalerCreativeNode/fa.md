این گره از سرویس Magnific AI برای بزرگ‌نمایی و بهبود خلاقانه تصویر استفاده می‌کند. به شما امکان می‌دهد با یک توضیح متنی، بهبود تصویر را هدایت کنید، سبک خاصی را برای بهینه‌سازی انتخاب کنید، و جنبه‌های مختلف فرآیند خلاقانه مانند جزئیات، شباهت به تصویر اصلی و قدرت سبک‌پردازی را کنترل کنید. این گره یک تصویر بزرگ‌نمایی‌شده با ضریب انتخابی شما (2x، 4x، 8x یا 16x) و با حداکثر اندازه خروجی ۲۵.۳ مگاپیکسل ارائه می‌دهد.

**محدودیت‌ها:**

* تصویر `image` ورودی باید دقیقاً یک تصویر باشد.
* تصویر ورودی باید حداقل ارتفاع و عرض ۱۶۰ پیکسل داشته باشد.
* نسبت ابعاد تصویر ورودی باید بین ۱:۳ و ۳:۱ باشد.
* اندازه نهایی خروجی (ابعاد ورودی ضرب‌در `scale_factor`) نمی‌تواند از ۲۵,۳۰۰,۰۰۰ پیکسل تجاوز کند. اگر `auto_downscale` غیرفعال باشد و این محدودیت نقض شود، گره یک خطا ایجاد می‌کند.
