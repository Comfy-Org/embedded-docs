Узел PairConditioningCombine объединяет две пары данных кондиционирования (позитивные и негативные) в одну пару. Он принимает на вход две отдельные пары кондиционирования и объединяет их с использованием внутренней логики комбинирования кондиционирования ComfyUI. Этот узел является экспериментальным и в основном используется для расширенных рабочих процессов манипуляции кондиционированием.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `positive_A` | CONDITIONING | Да | - | Первый вход позитивного кондиционирования |
| `negative_A` | CONDITIONING | Да | - | Первый вход негативного кондиционирования |
| `positive_B` | CONDITIONING | Да | - | Второй вход позитивного кондиционирования |
| `negative_B` | CONDITIONING | Да | - | Второй вход негативного кондиционирования |

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `positive` | CONDITIONING | Объединенный выход позитивного кондиционирования |
| `negative` | CONDITIONING | Объединенный выход негативного кондиционирования |
