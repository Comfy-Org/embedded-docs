Узел ImageFlip выполняет отражение изображений по различным осям. Он может отражать изображения вертикально по оси X или горизонтально по оси Y. Для выполнения отражения узел использует операции torch.flip в зависимости от выбранного метода.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `image` | IMAGE | Да | - | Входное изображение для отражения |
| `flip_method` | STRING | Да | "x-axis: vertically"<br>"y-axis: horizontally" | Направление отражения (по умолчанию: "x-axis: vertically") |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `image` | IMAGE | Результирующее отражённое изображение |
