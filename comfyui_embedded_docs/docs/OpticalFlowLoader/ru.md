## Обзор

Загружает модель оптического потока из папки `models/optical_flow/`. В настоящее время поддерживается только формат RAFT-large от torchvision, который используется узлом VOIDWarpedNoise. ComfyUI не загружает веса оптического потока автоматически; необходимо вручную разместить файл контрольной точки в директории `models/optical_flow/`.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `model_name` | STRING | Да | Список файлов в папке `models/optical_flow/` | Модель оптического потока для загрузки. Файлы должны быть размещены в папке `optical_flow`. В настоящее время поддерживается только `raft_large.pth` от torchvision. |

## Выходные данные

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `OPTICAL_FLOW` | MODEL | Загруженная модель оптического потока, обернутая в ModelPatcher для использования с другими узлами. |
