Узел PreviewImage предназначен для создания временных изображений для предварительного просмотра. Он автоматически генерирует уникальное временное имя файла для каждого изображения, сжимает изображение до указанного уровня и сохраняет его во временную директорию. Эта функция особенно полезна для создания превью изображений в процессе обработки без изменения исходных файлов.

## Входные параметры

| Параметр | Тип данных | Описание |
|-----------|-------------|-------------|
| `images`  | `IMAGE`     | Параметр `images` определяет изображения, которые будут обработаны и сохранены как временные превью. Это основной входной параметр узла, определяющий, какие изображения будут участвовать в процессе создания предварительного просмотра. |

## Выходные параметры

Узел не имеет выходных типов.
