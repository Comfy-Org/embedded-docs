Узел BetaSamplingScheduler генерирует последовательность уровней шума (сигм) для процесса сэмплирования с использованием алгоритма бета-планирования. Он принимает модель и параметры конфигурации для создания настраиваемого графика шума, управляющего процессом шумоподавления при генерации изображений. Данный планировщик позволяет точно настраивать траекторию снижения шума с помощью параметров alpha и beta.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `model` | MODEL | Да | - | Модель, используемая для сэмплирования, предоставляющая объект сэмплирования модели |
| `steps` | INT | Да | от 1 до 10000 | Количество шагов сэмплирования, для которых генерируются сигмы (по умолчанию: 20) |
| `alpha` | FLOAT | Да | от 0.0 до 50.0 | Параметр alpha для бета-планировщика, управляющий кривой планирования (по умолчанию: 0.6) |
| `beta` | FLOAT | Да | от 0.0 до 50.0 | Параметр beta для бета-планировщика, управляющий кривой планирования (по умолчанию: 0.6) |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `SIGMAS` | SIGMAS | Последовательность уровней шума (сигм), используемая в процессе сэмплирования |
