Вот перевод документации на русский язык:

Узел SolidMask генерирует однородную маску с заданным значением по всей своей площади. Он предназначен для создания масок определённых размеров и интенсивности, что полезно в различных задачах обработки изображений и работы с масками.

## Входные параметры

| Параметр | Тип данных | Описание |
|-----------|-------------|-------------|
| `значение`   | FLOAT       | Задаёт значение интенсивности маски, влияя на её общий вид и применимость в последующих операциях. |
| `ширина`   | INT         | Определяет ширину создаваемой маски, напрямую влияя на её размер и соотношение сторон. |
| `высота`  | INT         | Задаёт высоту создаваемой маски, влияя на её размер и соотношение сторон. |

## Выходные параметры

| Параметр | Тип данных | Описание |
|-----------|-------------|-------------|
| `mask`    | MASK        | Выдаёт однородную маску с указанными размерами и значением. |
