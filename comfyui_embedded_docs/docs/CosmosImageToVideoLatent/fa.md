گره CosmosImageToVideoLatent، نمایش‌های نهفته (latent) ویدئو را از تصاویر ورودی ایجاد می‌کند. این گره یک فضای نهفته ویدئویی خالی تولید کرده و به‌صورت اختیاری تصاویر شروع و/یا پایان را در فریم‌های ابتدایی و/یا انتهایی دنباله ویدئو رمزگذاری می‌کند. هنگامی که تصاویر ارائه می‌شوند، ماسک‌های نویز متناظر نیز ایجاد می‌شوند تا مشخص کنند کدام بخش‌های فضای نهفته باید در طول تولید حفظ شوند.

## ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|---------|----------|-------|--------|---------|
| `vae` | VAE | بله | - | مدل VAE مورد استفاده برای رمزگذاری تصاویر به فضای نهفته |
| `عرض` | INT | بله | 16 تا MAX_RESOLUTION | عرض ویدئوی خروجی بر حسب پیکسل (پیش‌فرض: 1280) |
| `ارتفاع` | INT | بله | 16 تا MAX_RESOLUTION | ارتفاع ویدئوی خروجی بر حسب پیکسل (پیش‌فرض: 704) |
| `طول` | INT | بله | 1 تا MAX_RESOLUTION | تعداد فریم‌های دنباله ویدئو (پیش‌فرض: 121) |
| `اندازه دسته` | INT | بله | 1 تا 4096 | تعداد دسته‌های نهفته برای تولید (پیش‌فرض: 1) |
| `تصویر شروع` | IMAGE | خیر | - | تصویر اختیاری برای رمزگذاری در ابتدای دنباله ویدئو |
| `تصویر پایان` | IMAGE | خیر | - | تصویر اختیاری برای رمزگذاری در انتهای دنباله ویدئو |

**نکته:** هنگامی که نه `start_image` و نه `end_image` ارائه نشوند، گره یک فضای نهفته خالی بدون هیچ ماسک نویزی برمی‌گرداند. هنگامی که هر یک از تصاویر ارائه شود، بخش‌های متناظر فضای نهفته رمزگذاری و ماسک‌گذاری می‌شوند.
