Узел CosmosImageToVideoLatent создает скрытые представления видео на основе входных изображений. Он генерирует пустой скрытый слой видео и, опционально, кодирует начальное и/или конечное изображения в первый и/или последний кадры видеопоследовательности. Если изображения предоставлены, узел также создает соответствующие маски шума, указывающие, какие части скрытого слоя должны быть сохранены во время генерации.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `vae` | VAE | Да | - | Модель VAE, используемая для кодирования изображений в скрытое пространство |
| `ширина` | INT | Да | 16 до MAX_RESOLUTION | Ширина выходного видео в пикселях (по умолчанию: 1280) |
| `высота` | INT | Да | 16 до MAX_RESOLUTION | Высота выходного видео в пикселях (по умолчанию: 704) |
| `длина` | INT | Да | 1 до MAX_RESOLUTION | Количество кадров в видеопоследовательности (по умолчанию: 121) |
| `размер_пакета` | INT | Да | 1 до 4096 | Количество генерируемых пакетов скрытых слоев (по умолчанию: 1) |
| `начальное_изображение` | IMAGE | Нет | - | Опциональное изображение для кодирования в начале видеопоследовательности |
| `конечное_изображение` | IMAGE | Нет | - | Опциональное изображение для кодирования в конце видеопоследовательности |

**Примечание:** Если не указаны ни `start_image`, ни `end_image`, узел возвращает пустой скрытый слой без маски шума. Если указано хотя бы одно изображение, соответствующие участки скрытого слоя кодируются и маскируются соответствующим образом.

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `latent` | LATENT | Сгенерированное скрытое представление видео с опционально закодированными изображениями и соответствующими масками шума |
