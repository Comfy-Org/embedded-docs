Узел SamplerEulerAncestral создаёт сэмплер Эйлера с предками (Euler Ancestral) для генерации изображений. Этот сэмплер использует особый математический подход, сочетающий интегрирование Эйлера с методами семплирования предков для создания вариаций изображений. Узел позволяет настроить поведение семплирования, регулируя параметры, управляющие случайностью и размером шага в процессе генерации.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `eta` | FLOAT | Нет | 0.0 - 100.0 | Управляет размером шага и стохастичностью процесса семплирования (по умолчанию: 1.0). Это расширенный параметр. |
| `s_noise` | FLOAT | Нет | 0.0 - 100.0 | Управляет количеством шума, добавляемого во время семплирования (по умолчанию: 1.0). Это расширенный параметр. |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `sampler` | SAMPLER | Возвращает настроенный сэмплер Эйлера с предками, который можно использовать в конвейере семплирования. |
