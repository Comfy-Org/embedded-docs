نسخة مبسطة من عقدة SkipLayerGuidanceDiT التي تقوم فقط بتعديل المسار غير المشروط أثناء عملية إزالة الضوضاء. تطبق هذه العقدة توجيه تخطي الطبقات على طبقات محول محددة في نماذج DiT (محول الانتشار) عن طريق تخطي طبقات معينة بشكل انتقائي أثناء المسار غير المشروط بناءً على معاملات التوقيت والطبقات المحددة.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|---------|--------------|--------|---------|-------|
| `model` | MODEL | نعم | - | النموذج المراد تطبيق توجيه تخطي الطبقات عليه |
| `double_layers` | STRING | لا | - | قائمة مفصولة بفواصل لمؤشرات طبقات الكتلة المزدوجة المراد تخطيها (الافتراضي: "7, 8, 9") |
| `single_layers` | STRING | لا | - | قائمة مفصولة بفواصل لمؤشرات طبقات الكتلة المفردة المراد تخطيها (الافتراضي: "7, 8, 9") |
| `start_percent` | FLOAT | لا | 0.0 - 1.0 | النسبة المئوية لبدء عملية إزالة الضوضاء عند بدء توجيه تخطي الطبقات (الافتراضي: 0.0) |
| `end_percent` | FLOAT | لا | 0.0 - 1.0 | النسبة المئوية لانتهاء عملية إزالة الضوضاء عند توقف توجيه تخطي الطبقات (الافتراضي: 1.0) |

**ملاحظة:** يتم تطبيق توجيه تخطي الطبقات فقط عندما يحتوي كل من `double_layers` و `single_layers` على مؤشرات طبقات صالحة. إذا كان كلا المعاملين فارغين، تقوم العقدة بإرجاع النموذج الأصلي دون تغيير. يكون توجيه تخطي الطبقات نشطًا فقط عندما تقع قيمة سيجما لخطوة إزالة الضوضاء الحالية بين `start_percent` و `end_percent` (يتم تحويلها داخليًا إلى قيم سيجما).
