نسخه ساده‌شده گره SkipLayerGuidanceDiT که تنها در طول فرآیند نویززدایی، عبور غیرشرطی را تغییر می‌دهد. این گره با رد کردن انتخابی لایه‌های مشخصی در طول عبور غیرشرطی بر اساس پارامترهای زمان‌بندی و لایه تعیین‌شده، راهنمایی رد شدن لایه را به لایه‌های ترانسفورمر خاصی در مدل‌های DiT (ترانسفورمر انتشار) اعمال می‌کند.

## ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|---------|----------|-------|--------|---------|
| `model` | MODEL | بله | - | مدلی که راهنمایی رد شدن لایه به آن اعمال می‌شود |
| `double_layers` | STRING | خیر | - | فهرست جدا شده با کاما از ایندکس‌های لایه بلوک دوتایی برای رد شدن (پیش‌فرض: "7, 8, 9") |
| `single_layers` | STRING | خیر | - | فهرست جدا شده با کاما از ایندکس‌های لایه بلوک تکی برای رد شدن (پیش‌فرض: "7, 8, 9") |
| `start_percent` | FLOAT | خیر | 0.0 - 1.0 | درصد شروع فرآیند نویززدایی که راهنمایی رد شدن لایه آغاز می‌شود (پیش‌فرض: 0.0) |
| `end_percent` | FLOAT | خیر | 0.0 - 1.0 | درصد پایان فرآیند نویززدایی که راهنمایی رد شدن لایه متوقف می‌شود (پیش‌فرض: 1.0) |

**توجه:** راهنمایی رد شدن لایه تنها زمانی اعمال می‌شود که هر دو `double_layers` و `single_layers` حاوی ایندکس‌های لایه معتبر باشند. اگر هر دو خالی باشند، گره مدل اصلی را بدون تغییر بازمی‌گرداند. راهنمایی رد شدن لایه تنها زمانی فعال است که مقدار سیگمای گام نویززدایی فعلی بین `start_percent` و `end_percent` (که به‌صورت داخلی به مقادیر سیگما تبدیل می‌شوند) قرار داشته باشد.
