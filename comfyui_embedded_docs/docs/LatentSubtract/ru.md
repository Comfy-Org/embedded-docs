Узел LatentSubtract предназначен для вычитания одного латентного представления из другого. Эта операция может использоваться для манипуляции или изменения характеристик выходных данных генеративных моделей путем удаления признаков или атрибутов, представленных в одном латентном пространстве, из другого.

## Входные данные

| Параметр     | Тип данных | Описание |
|--------------|-------------|-------------|
| `samples1`   | `LATENT`    | Первый набор латентных семплов, из которого производится вычитание. Служит основой для операции вычитания. |
| `samples2`   | `LATENT`    | Второй набор латентных семплов, который будет вычтен из первого набора. Эта операция может изменить выходные данные генеративной модели путем удаления атрибутов или признаков. |

## Выходные данные

| Параметр | Тип данных | Описание |
|-----------|-------------|-------------|
| `latent`  | `LATENT`    | Результат вычитания второго набора латентных семплов из первого. Это модифицированное латентное представление может использоваться для дальнейших задач генерации. |
