Этот узел предназначен для настройки временного аспекта кондиционирования путем задания определенного диапазона временных шагов. Он обеспечивает точный контроль над начальной и конечной точками процесса кондиционирования, позволяя выполнять более целенаправленную и эффективную генерацию.

## Входы

| Параметр | Тип данных | Описание |
| --- | --- | --- |
| `CONDITIONING` | CONDITIONING | Вход кондиционирования представляет текущее состояние процесса генерации, которое этот узел изменяет, задавая определенный диапазон временных шагов. |
| `начало` | `FLOAT` | Параметр start задает начало диапазона временных шагов в процентах от общего процесса генерации, что позволяет точно настроить момент начала эффектов кондиционирования. |
| `конец` | `FLOAT` | Параметр end определяет конечную точку диапазона временных шагов в процентах, обеспечивая точный контроль над продолжительностью и завершением эффектов кондиционирования. |

## Выходы

| Параметр | Тип данных | Описание |
| --- | --- | --- |
| `CONDITIONING` | CONDITIONING | Выходные данные представляют собой модифицированное кондиционирование с примененным указанным диапазоном временных шагов, готовое к дальнейшей обработке или генерации. |
