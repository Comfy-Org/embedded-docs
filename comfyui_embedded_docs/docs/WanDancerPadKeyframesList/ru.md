## Обзор

Этот узел принимает последовательность изображений и опциональную аудиодорожку, разделяя их на заданное количество дополненных сегментов. Он предназначен для подготовки последовательностей ключевых кадров для генерации видео, где каждый сегмент дополняется до единообразной длины, а соответствующая маска указывает, какие кадры являются валидными.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `images` | IMAGE | Да | N/A | Входная последовательность изображений для разделения на сегменты. |
| `segment_length` | INT | Да | от 1 до 10000 | Длина каждого сегмента в кадрах (по умолчанию: 149). |
| `num_segments` | INT | Да | от 1 до 100 | Количество дополненных сегментов для вывода в виде списков (по умолчанию: 1). |
| `audio` | AUDIO | Нет | N/A | Аудиодорожка для нарезки под каждый выводимый сегмент. |

## Выходные данные

| Имя выходного параметра | Тип данных | Описание |
|-------------------------|------------|----------|
| `keyframes_sequence` | IMAGE | Список дополненных последовательностей ключевых кадров, по одной для каждого сегмента. |
| `keyframes_mask` | MASK | Список масок, указывающих валидные кадры для каждого сегмента. |
| `audio_segment` | AUDIO | Список аудиосегментов, по одному для каждого видеосегмента. |
