Узел **VOIDInpaintConditioning** подготавливает данные conditioning, необходимые для инпейнтинга (восстановления изображений) с использованием моделей CogVideoX. Он принимает исходное видео и предварительно обработанную квадромаску (quadmask), кодирует их через VAE и объединяет в 32-канальный сигнал conditioning, который модель использует для заполнения замаскированных областей.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `positive` | CONDITIONING | Да | - | Положительное conditioning, дополняемое информацией о латентном представлении инпейнтинга |
| `negative` | CONDITIONING | Да | - | Отрицательное conditioning, дополняемое информацией о латентном представлении инпейнтинга |
| `vae` | VAE | Да | - | Модель VAE, используемая для кодирования маски и замаскированного видео в латентное пространство |
| `video` | IMAGE | Да | - | Кадры исходного видео в формате [T, H, W, 3] |
| `quadmask` | MASK | Да | - | Предварительно обработанная квадромаска из узла VOIDQuadmaskPreprocess в формате [T, H, W] |
| `width` | INT | Да | от 16 до MAX_RESOLUTION (шаг: 8) | Ширина, к которой изменяются видео и маска (по умолчанию: 672) |
| `height` | INT | Да | от 16 до MAX_RESOLUTION (шаг: 8) | Высота, к которой изменяются видео и маска (по умолчанию: 384) |
| `length` | INT | Да | от 1 до MAX_RESOLUTION (шаг: 1) | Количество обрабатываемых пиксельных кадров. Для CogVideoX-Fun-V1.5 (patch_size_t=2) latent_t должно быть четным — длины, дающие нечетное latent_t, округляются вниз (например, 49 → 45) (по умолчанию: 45) |
| `batch_size` | INT | Да | от 1 до 64 | Размер пакета для выходного шумового латентного представления (по умолчанию: 1) |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `positive` | CONDITIONING | Положительное conditioning с добавленной информацией о латентном представлении инпейнтинга |
| `negative` | CONDITIONING | Отрицательное conditioning с добавленной информацией о латентном представлении инпейнтинга |
| `latent` | LATENT | Тензор шумового латентного представления, заполненный нулями, размерностью [batch_size, 16, latent_t, latent_h, latent_w] |
