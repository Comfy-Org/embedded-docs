گره VOIDInpaintConditioning داده‌های شرطی‌سازی مورد نیاز برای inpainting با مدل‌های CogVideoX را آماده می‌کند. این گره یک ویدئوی منبع و یک quadmask از پیش‌پردازش‌شده را دریافت کرده، آن‌ها را از طریق VAE رمزگذاری می‌کند و در یک سیگنال شرطی‌سازی ۳۲ کاناله ترکیب می‌کند که مدل از آن برای پر کردن نواحی پوشیده‌شده استفاده می‌کند.

## ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|---------|----------|-------|--------|---------|
| `positive` | CONDITIONING | بله | - | شرطی‌سازی مثبت که با اطلاعات نهان inpainting تقویت می‌شود |
| `negative` | CONDITIONING | بله | - | شرطی‌سازی منفی که با اطلاعات نهان inpainting تقویت می‌شود |
| `vae` | VAE | بله | - | مدل VAE مورد استفاده برای رمزگذاری ماسک و ویدئوی ماسک‌شده به فضای نهان |
| `video` | IMAGE | بله | - | فریم‌های ویدئوی منبع [T, H, W, 3] |
| `quadmask` | MASK | بله | - | quadmask از پیش‌پردازش‌شده از گره VOIDQuadmaskPreprocess [T, H, W] |
| `width` | INT | بله | ۱۶ تا MAX_RESOLUTION (گام: ۸) | عرضی که ویدئو و ماسک به آن تغییر اندازه می‌دهند (پیش‌فرض: ۶۷۲) |
| `height` | INT | بله | ۱۶ تا MAX_RESOLUTION (گام: ۸) | ارتفاعی که ویدئو و ماسک به آن تغییر اندازه می‌دهند (پیش‌فرض: ۳۸۴) |
| `length` | INT | بله | ۱ تا MAX_RESOLUTION (گام: ۱) | تعداد فریم‌های پیکسلی برای پردازش. برای CogVideoX-Fun-V1.5 (patch_size_t=2)، latent_t باید زوج باشد — طول‌هایی که latent_t فرد تولید می‌کنند به پایین گرد می‌شوند (مثلاً ۴۹ → ۴۵) (پیش‌فرض: ۴۵) |
| `batch_size` | INT | بله | ۱ تا ۶۴ | اندازه دسته برای خروجی نهان نویز (پیش‌فرض: ۱) |
