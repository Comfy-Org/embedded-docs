عقدة VOIDInpaintConditioning تقوم بإعداد بيانات التكييف اللازمة لعملية إعادة التلوين (inpainting) باستخدام نماذج CogVideoX. تأخذ هذه العقدة فيديو مصدرًا وقناعًا رباعيًا معالجًا مسبقًا، وتقوم بترميزهما عبر VAE، ثم تدمجهما في إشارة تكييف ذات 32 قناة يستخدمها النموذج لملء المناطق المقنعة.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|---------|--------------|--------|---------|-------|
| `positive` | CONDITIONING | نعم | - | التكييف الإيجابي الذي سيتم توسيعه بمعلومات الكامن الخاصة بإعادة التلوين |
| `negative` | CONDITIONING | نعم | - | التكييف السلبي الذي سيتم توسيعه بمعلومات الكامن الخاصة بإعادة التلوين |
| `vae` | VAE | نعم | - | نموذج VAE المستخدم لترميز القناع والفيديو المقنع إلى الفضاء الكامن |
| `video` | IMAGE | نعم | - | إطارات الفيديو المصدر بتنسيق [T, H, W, 3] |
| `quadmask` | MASK | نعم | - | القناع الرباعي المعالج مسبقًا من عقدة VOIDQuadmaskPreprocess بتنسيق [T, H, W] |
| `width` | INT | نعم | 16 إلى MAX_RESOLUTION (الخطوة: 8) | العرض الذي سيتم تغيير حجم الفيديو والقناع إليه (الافتراضي: 672) |
| `height` | INT | نعم | 16 إلى MAX_RESOLUTION (الخطوة: 8) | الارتفاع الذي سيتم تغيير حجم الفيديو والقناع إليه (الافتراضي: 384) |
| `length` | INT | نعم | 1 إلى MAX_RESOLUTION (الخطوة: 1) | عدد إطارات البكسل المراد معالجتها. بالنسبة لنموذج CogVideoX-Fun-V1.5 (حجم_التصحيح_الزمني=2)، يجب أن يكون البعد الزمني الكامن (latent_t) زوجيًا — يتم تقريب الأطوال التي تنتج latent_t فرديًا إلى الأسفل (مثل 49 → 45) (الافتراضي: 45) |
| `batch_size` | INT | نعم | 1 إلى 64 | حجم الدفعة لإخراج الكامن الضوضائي (الافتراضي: 1) |

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|------------|--------------|--------|
| `positive` | CONDITIONING | التكييف الإيجابي مع إضافة معلومات الكامن الخاصة بإعادة التلوين |
| `negative` | CONDITIONING | التكييف السلبي مع إضافة معلومات الكامن الخاصة بإعادة التلوين |
| `latent` | LATENT | موتر كامن ضوضائي مملوء بالأصفار ذو شكل [batch_size, 16, latent_t, latent_h, latent_w] |
