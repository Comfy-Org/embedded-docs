عقدة VOIDInpaintConditioning تُعد بيانات التكييف اللازمة لعملية الرسم الداخلي (inpainting) باستخدام نماذج CogVideoX. تستقبل العقدة فيديو مصدرًا وقناعًا رباعيًا مُعالَجًا مسبقًا، وتقوم بترميزهما عبر المشفر التلقائي المتغير (VAE)، ثم تدمجهما في إشارة تكييف ذات 32 قناة يستخدمها النموذج لملء المناطق المُقنّعة.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|---------|--------------|--------|---------|-------|
| `positive` | CONDITIONING | نعم | - | التكييف الإيجابي الذي سيتم توسيعه بمعلومات الكامن الخاصة بالرسم الداخلي |
| `negative` | CONDITIONING | نعم | - | التكييف السلبي الذي سيتم توسيعه بمعلومات الكامن الخاصة بالرسم الداخلي |
| `vae` | VAE | نعم | - | نموذج المشفر التلقائي المتغير المستخدم لترميز القناع والفيديو المُقنّع إلى الفضاء الكامن |
| `video` | IMAGE | نعم | - | إطارات الفيديو المصدر [T, H, W, 3] |
| `quadmask` | MASK | نعم | - | القناع الرباعي المُعالَج مسبقًا من عقدة VOIDQuadmaskPreprocess [T, H, W] |
| `width` | INT | نعم | 16 إلى MAX_RESOLUTION (خطوة: 8) | العرض الذي سيتم تغيير حجم الفيديو والقناع إليه (الافتراضي: 672) |
| `height` | INT | نعم | 16 إلى MAX_RESOLUTION (خطوة: 8) | الارتفاع الذي سيتم تغيير حجم الفيديو والقناع إليه (الافتراضي: 384) |
| `length` | INT | نعم | 1 إلى MAX_RESOLUTION (خطوة: 1) | عدد إطارات البكسل المراد معالجتها. بالنسبة لنموذج CogVideoX-Fun-V1.5 (حجم الرقعة الزمنية patch_size_t=2)، يجب أن يكون البعد الزمني الكامن (latent_t) زوجيًا — يتم تقريب الأطوال التي تُنتج latent_t فرديًا إلى الأسفل (مثل 49 → 45) (الافتراضي: 45) |
| `batch_size` | INT | نعم | 1 إلى 64 | حجم الدفعة لمخرج الضوضاء الكامنة (الافتراضي: 1) |

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|------------|--------------|-------|
| `negative` | CONDITIONING | التكييف الإيجابي مع إضافة معلومات الكامن الخاصة بالرسم الداخلي |
| `latent` | CONDITIONING | التكييف السلبي مع إضافة معلومات الكامن الخاصة بالرسم الداخلي |
| `latent` | LATENT | موتر ضوضاء كامن مملوء بالأصفار ذو شكل [batch_size, 16, latent_t, latent_h, latent_w] |
