این گره یک تانسور نهفته خالی ایجاد می‌کند که به‌طور خاص برای استفاده با مدل HunyuanVideo 1.5 قالب‌بندی شده است. با تخصیص یک تانسور از صفرها با تعداد کانال صحیح و ابعاد فضایی مناسب برای فضای نهفته مدل، یک نقطه شروع خالی برای تولید ویدیو ایجاد می‌کند.

## ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `عرض` | INT | بله | - | عرض فریم ویدیو بر حسب پیکسل. |
| `ارتفاع` | INT | بله | - | ارتفاع فریم ویدیو بر حسب پیکسل. |
| `طول` | INT | بله | - | تعداد فریم‌های دنباله ویدیو. |
| `اندازه دسته` | INT | خیر | - | تعداد نمونه‌های ویدیویی برای تولید در یک دسته (پیش‌فرض: ۱). |

**نکته:** ابعاد فضایی تانسور نهفته تولید شده با تقسیم `width` و `height` ورودی بر ۱۶ محاسبه می‌شود. بعد زمانی (فریم‌ها) به صورت `((length - 1) // 4) + 1` محاسبه می‌شود.
