Узел LatentCompositeMasked предназначен для смешивания двух латентных представлений в заданных координатах, с возможностью использования маски для более контролируемого композитинга. Этот узел позволяет создавать сложные латентные изображения путём наложения частей одного изображения на другое, с возможностью изменения размера исходного изображения для идеального соответствия.

## Входные параметры

| Параметр | Тип данных | Описание |
|-----------|-------------|-------------|
| `destination` | `LATENT`    | Латентное представление, на которое будет наложено другое латентное представление. Служит базовым слоем для операции композитинга. |
| `source` | `LATENT`    | Латентное представление, которое будет наложено на целевое изображение. Этот исходный слой может быть изменён в размере и позиционирован в соответствии с заданными параметрами. |
| `x` | `INT`       | Координата X в целевом латентном представлении, куда будет помещён источник. Позволяет точно позиционировать исходный слой. |
| `y` | `INT`       | Координата Y в целевом латентном представлении, куда будет помещён источник, обеспечивая точное позиционирование наложения. |
| `resize_source` | `BOOLEAN` | Логический флаг, указывающий, следует ли изменить размер исходного латентного представления до размеров целевого перед композитингом. |
| `mask` | `MASK`     | Необязательная маска, которая может использоваться для управления смешиванием источника с целевым изображением. Маска определяет, какие части источника будут видны в итоговом композите. |

## Выходные параметры

| Параметр | Тип данных | Описание |
|-----------|-------------|-------------|
| `latent`  | `LATENT`    | Результирующее латентное представление после наложения источника на целевое изображение, с возможностью использования маски для выборочного смешивания. |
