### 概述

LatentCompositeMasked 節點旨在將兩個潛在表示（latent representations）在指定座標上混合在一起，並可選擇使用遮罩進行更精確的合成控制。此節點能透過將一幅圖像的一部分疊加到另一幅圖像上，並可調整來源圖像大小以完美契合，從而創建複雜的潛在圖像。

### 輸入

| 參數 | 資料類型 | 描述 |
|-----------|-------------|-------------|
| `destination` | `LATENT`    | 作為合成操作底層的潛在表示，另一個潛在表示將被合成到其上。 |
| `source` | `LATENT`    | 要合成到目標上的潛在表示。此來源圖層可根據指定參數調整大小和位置。 |
| `x` | `INT`       | 來源將被放置於目標潛在表示中的 x 座標，用於精確定位來源圖層。 |
| `y` | `INT`       | 來源將被放置於目標潛在表示中的 y 座標，可實現準確的疊加定位。 |
| `調整來源大小` | `BOOLEAN` | 布林標誌，指示在合成前是否應將來源潛在表示調整為與目標相同的尺寸。 |
| `遮罩` | `MASK`     | 可選的遮罩，用於控制來源疊加到目標上的混合效果。遮罩定義了最終合成中來源的哪些部分將可見。 |

### 輸出

| 參數 | 資料類型 | 描述 |
|-----------|-------------|-------------|
| `latent`  | `LATENT`    | 將來源合成到目標上後得到的潛在表示，可能使用了遮罩進行選擇性混合。 |
