LatentCompositeMasked 節點專為將兩個潛在表徵在指定座標位置進行混合而設計，可選擇性地使用遮罩來實現更精確的合成控制。此節點能將來源圖像的部分內容疊加至目標圖像上，並具備調整來源圖像尺寸的功能，以實現完美融合。

## 輸入參數

| 參數名稱 | 資料類型 | 描述 |
|-----------|-------------|-------------|
| `destination` | `LATENT`    | 作為合成操作基礎圖層的潛在表徵，其他潛在表徵將被合成至此目標之上 |
| `source` | `LATENT`    | 將被合成至目標之上的潛在表徵，此來源圖層可根據指定參數進行尺寸調整和定位 |
| `x` | `INT`       | 來源圖層在目標潛在表徵中的 X 軸座標位置，用於實現精確定位 |
| `y` | `INT`       | 來源圖層在目標潛在表徵中的 Y 軸座標位置，確保疊加定位的準確性 |
| `調整來源大小` | `BOOLEAN` | 布林標誌，指示在合成前是否應將來源潛在表徵調整為與目標尺寸相匹配 |
| `遮罩` | `MASK`     | 可選遮罩，用於控制來源與目標的混合效果，定義來源圖層在最終合成中可見的區域 |
