Вот перевод документации на русский язык:

Кодирует текст и задаёт conditioning разрешения для PixArt Alpha. Этот узел обрабатывает текстовый ввод и добавляет информацию о ширине и высоте для создания данных conditioning, предназначенных специально для моделей PixArt Alpha. Не применяется к моделям PixArt Sigma.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `ширина` | INT | Да | 0 до MAX_RESOLUTION | Размер ширины для conditioning разрешения (по умолчанию: 1024) |
| `высота` | INT | Да | 0 до MAX_RESOLUTION | Размер высоты для conditioning разрешения (по умолчанию: 1024) |
| `текст` | STRING | Да | - | Текстовый ввод для кодирования, поддерживает многострочный ввод и динамические подсказки |
| `clip` | CLIP | Да | - | Модель CLIP, используемая для токенизации и кодирования |

## Выходы

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `CONDITIONING` | CONDITIONING | Закодированные данные conditioning с текстовыми токенами и информацией о разрешении |
