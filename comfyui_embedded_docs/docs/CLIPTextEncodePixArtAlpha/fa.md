### نمای کلی

متن را رمزگذاری کرده و شرط‌گذاری وضوح را برای PixArt Alpha تنظیم می‌کند. این گره ورودی متن را پردازش کرده و اطلاعات عرض و ارتفاع را برای ایجاد داده‌های شرط‌گذاری مخصوص مدل‌های PixArt Alpha اضافه می‌کند. این گره برای مدل‌های PixArt Sigma قابل استفاده نیست.

### ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|---------|----------|-------|--------|----------|
| `width` | INT | بله | 0 تا MAX_RESOLUTION | بعد عرض برای شرط‌گذاری وضوح (پیش‌فرض: 1024) |
| `height` | INT | بله | 0 تا MAX_RESOLUTION | بعد ارتفاع برای شرط‌گذاری وضوح (پیش‌فرض: 1024) |
| `text` | STRING | بله | - | ورودی متنی برای رمزگذاری، پشتیبانی از ورودی چندخطی و اعلان‌های پویا |
| `clip` | CLIP | بله | - | مدل CLIP مورد استفاده برای توکن‌سازی و رمزگذاری |

### خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-----------|----------|---------|
| `CONDITIONING` | CONDITIONING | داده‌های شرط‌گذاری رمزگذاری‌شده با توکن‌های متنی و اطلاعات وضوح |
