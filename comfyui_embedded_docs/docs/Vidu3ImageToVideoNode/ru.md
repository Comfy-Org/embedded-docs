Узел генерации видео из изображения Vidu Q3 создаёт видеопоследовательность на основе входного изображения. Он использует модель Vidu Q3 для анимации изображения, опционально направляемой текстовой подсказкой, и выводит видеофайл.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `model` | COMBO | Да | `"viduq3-pro"`<br>`"viduq3-turbo"` | Модель для генерации видео. |
| `model.resolution` | COMBO | Да | `"720p"`<br>`"1080p"`<br>`"2K"` (только viduq3-pro) | Разрешение выходного видео. Доступные варианты зависят от выбранной модели. |
| `model.duration` | INT | Да | от 1 до 16 | Длительность выходного видео в секундах (по умолчанию: 5). |
| `model.audio` | BOOLEAN | Да | `True` / `False` | При включении создаёт видео со звуком (включая диалоги и звуковые эффекты) (по умолчанию: False). |
| `image` | IMAGE | Да | - | Изображение, используемое в качестве начального кадра создаваемого видео. |
| `prompt` | STRING | Нет | - | Необязательная текстовая подсказка для генерации видео (максимум 2000 символов) (по умолчанию: пусто). |
| `seed` | INT | Нет | от 0 до 2147483647 | Значение зерна для контроля случайности генерации (по умолчанию: 1). |

**Примечание:** Изображение `image` должно иметь соотношение сторон от 1:4 до 4:1 (от портретного до ландшафтного). Подсказка `prompt` необязательна, но не может превышать 2000 символов. Варианты параметра `model.resolution` зависят от выбранной модели `model`: `"viduq3-pro"` поддерживает `"720p"`, `"1080p"` и `"2K"`; `"viduq3-turbo"` поддерживает `"720p"` и `"1080p"`.

## Выходные данные

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `output` | VIDEO | Сгенерированный видеофайл. |
