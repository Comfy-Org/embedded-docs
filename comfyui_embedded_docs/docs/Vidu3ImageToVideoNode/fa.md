گره تولید ویدئو از تصویر Vidu Q3، یک دنباله ویدئویی را با شروع از یک تصویر ورودی ایجاد می‌کند. این گره از مدل Vidu Q3 برای پویانمایی تصویر استفاده می‌کند و به‌صورت اختیاری می‌تواند توسط یک متن راهنما هدایت شود و در نهایت یک فایل ویدئویی خروجی می‌دهد.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `مدل` | COMBO | بله | `"viduq3-pro"`<br>`"viduq3-turbo"` | مدل مورد استفاده برای تولید ویدئو. |
| `model.resolution` | COMBO | بله | `"720p"`<br>`"1080p"`<br>`"2K"` (فقط viduq3-pro) | وضوح تصویر ویدئوی خروجی. گزینه‌های موجود به مدل انتخاب‌شده بستگی دارد. |
| `model.duration` | INT | بله | 1 تا 16 | مدت زمان ویدئوی خروجی بر حسب ثانیه (پیش‌فرض: 5). |
| `model.audio` | BOOLEAN | بله | `True` / `False` | در صورت فعال‌سازی، ویدئو با صدا (شامل دیالوگ و افکت‌های صوتی) خروجی می‌دهد (پیش‌فرض: False). |
| `تصویر` | IMAGE | بله | - | تصویری که به‌عنوان فریم شروع ویدئوی تولیدشده استفاده می‌شود. |
| `پرامپت` | STRING | خیر | - | یک متن راهنمای اختیاری برای تولید ویدئو (حداکثر ۲۰۰۰ کاراکتر) (پیش‌فرض: خالی). |
| `seed` | INT | خیر | 0 تا 2147483647 | یک مقدار دانه (seed) برای کنترل تصادفی‌بودن تولید (پیش‌فرض: 1). |

**توجه:** تصویر `image` باید نسبت تصویری بین ۱:۴ و ۴:۱ (پرتره تا منظره) داشته باشد. متن راهنما `prompt` اختیاری است اما نمی‌تواند از ۲۰۰۰ کاراکتر تجاوز کند. گزینه‌های `model.resolution` به مدل انتخاب‌شده `model` بستگی دارد: `"viduq3-pro"` از `"720p"`، `"1080p"` و `"2K"` پشتیبانی می‌کند؛ `"viduq3-turbo"` از `"720p"` و `"1080p"` پشتیبانی می‌کند.

## خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-------------|-----------|-------------|
| `output` | VIDEO | فایل ویدئویی تولیدشده. |
