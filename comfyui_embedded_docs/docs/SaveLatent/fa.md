گره SaveLatent، تنسورهای نهفته (latent tensors) را به‌عنوان فایل روی دیسک ذخیره می‌کند تا بعداً استفاده یا به اشتراک گذاشته شوند. این گره نمونه‌های نهفته را دریافت کرده و آن‌ها را با ابرداده‌های اختیاری شامل اطلاعات پرامپت، در دایرکتوری خروجی ذخیره می‌کند. گره به‌طور خودکار نام‌گذاری و سازمان‌دهی فایل‌ها را انجام می‌دهد و ساختار داده‌های نهفته را حفظ می‌کند.

## ورودی‌ها

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|---------|----------|-------|--------|---------|
| `samples` | LATENT | بله | - | نمونه‌های نهفته که باید روی دیسک ذخیره شوند |
| `filename_prefix` | STRING | خیر | - | پیشوند نام فایل خروجی (پیش‌فرض: "latents/ComfyUI") |
| `prompt` | PROMPT | خیر | - | اطلاعات پرامپت برای درج در ابرداده (پارامتر پنهان) |
| `extra_pnginfo` | EXTRA_PNGINFO | خیر | - | اطلاعات اضافی PNG برای درج در ابرداده (پارامتر پنهان) |

## خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-----------|----------|---------|
| `ui` | UI | اطلاعات موقعیت فایل را برای نمونه‌های نهفته ذخیره‌شده در رابط ComfyUI ارائه می‌دهد |
