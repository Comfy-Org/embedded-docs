عقدة حفظ الكمون (SaveLatent) تقوم بحفظ موترات الكمون على القرص كملفات لاستخدامها لاحقًا أو للمشاركة. تأخذ العقدة عينات كمون وتحفظها في دليل الإخراج مع بيانات وصفية اختيارية تتضمن معلومات المطالبة. تتعامل العقدة تلقائيًا مع تسمية الملفات وتنظيمها مع الحفاظ على بنية بيانات الكمون.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|---------|--------------|--------|--------|-------|
| `samples` | LATENT | نعم | - | عينات الكمون المراد حفظها على القرص |
| `filename_prefix` | STRING | لا | - | البادئة لاسم ملف الإخراج (الافتراضي: "latents/ComfyUI") |
| `prompt` | PROMPT | لا | - | معلومات المطالبة لتضمينها في البيانات الوصفية (معامل مخفي) |
| `extra_pnginfo` | EXTRA_PNGINFO | لا | - | معلومات PNG إضافية لتضمينها في البيانات الوصفية (معامل مخفي) |

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|------------|--------------|--------|
| `ui` | UI | يوفر معلومات موقع الملف للكمون المحفوظ في واجهة ComfyUI |
