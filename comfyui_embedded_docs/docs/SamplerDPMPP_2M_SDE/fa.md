گره SamplerDPMPP_2M_SDE یک نمونه‌بردار DPM++ 2M SDE برای مدل‌های انتشار ایجاد می‌کند. این نمونه‌بردار از حل‌کننده‌های معادلات دیفرانسیل مرتبه دوم به همراه معادلات دیفرانسیل تصادفی برای تولید نمونه‌ها استفاده می‌کند. انواع مختلف حل‌کننده و گزینه‌های مدیریت نویز را برای کنترل فرآیند نمونه‌برداری ارائه می‌دهد.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `نوع حل‌گر` | STRING | بله | `"midpoint"`<br>`"heun"` | نوع حل‌کننده معادله دیفرانسیل مورد استفاده در فرآیند نمونه‌برداری |
| `eta` | FLOAT | بله | 0.0 - 100.0 | تصادفی بودن فرآیند نمونه‌برداری را کنترل می‌کند (پیش‌فرض: 1.0) |
| `s_noise` | FLOAT | بله | 0.0 - 100.0 | میزان نویز اضافه‌شده در طول نمونه‌برداری را کنترل می‌کند (پیش‌فرض: 1.0) |
| `دستگاه نویز` | STRING | بله | `"gpu"`<br>`"cpu"` | دستگاهی که محاسبات نویز روی آن انجام می‌شود. وقتی روی "cpu" تنظیم شود، نمونه‌بردار از تولید نویز مبتنی بر CPU استفاده می‌کند؛ وقتی روی "gpu" تنظیم شود، برای عملکرد بالقوه سریع‌تر از تولید نویز مبتنی بر GPU استفاده می‌کند |
