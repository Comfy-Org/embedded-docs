Вот перевод документации на русский язык:

Узел SamplerDPMPP_2M_SDE создаёт сэмплер DPM++ 2M SDE для диффузионных моделей. Этот сэмплер использует решатели дифференциальных уравнений второго порядка со стохастическими дифференциальными уравнениями для генерации образцов. Он предоставляет различные типы решателей и опции обработки шума для управления процессом сэмплирования.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `solver_type` | STRING | Да | `"midpoint"`<br>`"heun"` | Тип решателя дифференциальных уравнений, используемого в процессе сэмплирования |
| `eta` | FLOAT | Да | 0.0 - 100.0 | Управляет стохастичностью процесса сэмплирования (по умолчанию: 1.0) |
| `s_noise` | FLOAT | Да | 0.0 - 100.0 | Управляет количеством шума, добавляемого во время сэмплирования (по умолчанию: 1.0) |
| `noise_device` | STRING | Да | `"gpu"`<br>`"cpu"` | Устройство, на котором выполняются вычисления шума. При значении "cpu" сэмплер использует генерацию шума на основе CPU; при значении "gpu" — генерацию шума на основе GPU для потенциально более высокой производительности |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `sampler` | SAMPLER | Настроенный объект сэмплера, готовый к использованию в конвейере сэмплирования |
