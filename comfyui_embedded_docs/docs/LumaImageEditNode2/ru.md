## Обзор

Этот узел редактирует существующее изображение с помощью текстовой подсказки, используя модель Luma UNI-1. Он принимает исходное изображение и описание желаемого изменения, после чего генерирует новую отредактированную версию изображения.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `source` | IMAGE | Да | - | Исходное изображение для редактирования. |
| `prompt` | STRING | Да | 1–6000 символов | Описание желаемого изменения. По умолчанию: "" (пустая строка). |
| `model` | MODEL | Да | `"uni-1"`<br>`"uni-1-max"` | Модель, используемая для редактирования. |
| `seed` | INT | Да | от 0 до 2147483647 | Зерно определяет, должен ли узел выполниться повторно; результаты недетерминированы независимо от значения зерна. По умолчанию: 0. |

**Ограничения параметров:**
- `prompt` должен содержать от 1 до 6000 символов.
- Параметр `model` представляет собой динамический комбинированный список, который при установке значения `"uni-1"` или `"uni-1-max"` предоставляет дополнительные подпараметры (такие как `style`, `web_search` и `image_ref`). Подпараметр `image_ref` принимает не более 8 ссылок на изображения.

## Выходные данные

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `image` | IMAGE | Отредактированное изображение, сгенерированное моделью Luma UNI-1. |
