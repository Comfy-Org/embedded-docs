بالتأكيد، إليك الترجمة العربية للوثيقة مع الالتزام التام بالقواعد المحددة:

عقدة WanFirstLastFrameToVideo تنشئ تهيئة فيديو من خلال الجمع بين إطارات البداية والنهاية مع مطالبات نصية. تقوم بتوليد تمثيل كامن لتوليد الفيديو عن طريق ترميز الإطارين الأول والأخير، وتطبيق أقنعة لتوجيه عملية التوليد، ودمج ميزات رؤية CLIP عند توفرها. تقوم هذه العقدة بإعداد كل من التهيئة الإيجابية والسلبية لنماذج الفيديو لتوليد تسلسلات متماسكة بين نقاط البداية والنهاية المحددة.

## المدخلات

| المعامل | نوع البيانات | إجباري | النطاق | الوصف |
|-----------|-----------|----------|-------|-------------|
| `positive` | CONDITIONING | نعم | - | تهيئة نصية إيجابية لتوجيه توليد الفيديو |
| `negative` | CONDITIONING | نعم | - | تهيئة نصية سلبية لتوجيه توليد الفيديو |
| `vae` | VAE | نعم | - | نموذج VAE المستخدم لترميز الصور إلى الفضاء الكامن |
| `width` | INT | نعم | 16 إلى MAX_RESOLUTION | عرض الفيديو الناتج (الافتراضي: 832، الخطوة: 16) |
| `height` | INT | نعم | 16 إلى MAX_RESOLUTION | ارتفاع الفيديو الناتج (الافتراضي: 480، الخطوة: 16) |
| `length` | INT | نعم | 1 إلى MAX_RESOLUTION | عدد الإطارات في تسلسل الفيديو (الافتراضي: 81، الخطوة: 4) |
| `batch_size` | INT | نعم | 1 إلى 4096 | عدد مقاطع الفيديو المراد توليدها في وقت واحد (الافتراضي: 1) |
| `clip_vision_start_image` | CLIP_VISION_OUTPUT | لا | - | ميزات رؤية CLIP المستخرجة من صورة البداية |
| `clip_vision_end_image` | CLIP_VISION_OUTPUT | لا | - | ميزات رؤية CLIP المستخرجة من صورة النهاية |
| `start_image` | IMAGE | لا | - | صورة الإطار البادئ لتسلسل الفيديو |
| `end_image` | IMAGE | لا | - | صورة الإطار الختامي لتسلسل الفيديو |

**ملاحظة:** عند توفير كل من `start_image` و `end_image`، تقوم العقدة بإنشاء تسلسل فيديو ينتقل بين هذين الإطارين. معاملات `clip_vision_start_image` و `clip_vision_end_image` اختيارية، ولكن عند توفيرها، يتم دمج ميزات رؤية CLIP الخاصة بها وتطبيقها على كل من التهيئة الإيجابية والسلبية. يتم اقتصاص `start_image` إلى أول إطارات `length`، ويتم اقتصاص `end_image` إلى آخر إطارات `length` قبل المعالجة.

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|-------------|-----------|-------------|
| `positive` | CONDITIONING | تهيئة إيجابية مع ترميز إطار الفيديو المطبق وميزات رؤية CLIP |
| `negative` | CONDITIONING | تهيئة سلبية مع ترميز إطار الفيديو المطبق وميزات رؤية CLIP |
| `latent` | LATENT | موتر كامن فارغ بأبعاد تتطابق مع معاملات الفيديو المحددة |
