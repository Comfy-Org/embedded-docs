### **WanFirstLastFrameToVideo**

گره `WanFirstLastFrameToVideo` با ترکیب فریم‌های شروع و پایان با پرامپت‌های متنی، شرایط‌گذاری ویدئو را ایجاد می‌کند. این گره با رمزگذاری فریم‌های اول و آخر، اعمال ماسک برای هدایت فرآیند تولید، و استفاده از ویژگی‌های بینایی CLIP (در صورت وجود)، یک نمایش نهفته برای تولید ویدئو ایجاد می‌کند. این گره شرایط‌گذاری مثبت و منفی را برای مدل‌های ویدئویی آماده می‌کند تا توالی‌های منسجمی بین نقاط شروع و پایان مشخص شده تولید کنند.

### **ورودی‌ها**

| پارامتر | نوع داده | ضروری | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `positive` | CONDITIONING | بله | - | شرایط‌گذاری متن مثبت برای هدایت تولید ویدئو |
| `negative` | CONDITIONING | بله | - | شرایط‌گذاری متن منفی برای هدایت تولید ویدئو |
| `vae` | VAE | بله | - | مدل VAE مورد استفاده برای رمزگذاری تصاویر به فضای نهفته |
| `width` | INT | بله | 16 تا MAX_RESOLUTION | عرض ویدئوی خروجی (پیش‌فرض: 832، گام: 16) |
| `height` | INT | بله | 16 تا MAX_RESOLUTION | ارتفاع ویدئوی خروجی (پیش‌فرض: 480، گام: 16) |
| `length` | INT | بله | 1 تا MAX_RESOLUTION | تعداد فریم‌های توالی ویدئو (پیش‌فرض: 81، گام: 4) |
| `batch_size` | INT | بله | 1 تا 4096 | تعداد ویدئوهایی که به طور همزمان تولید می‌شوند (پیش‌فرض: 1) |
| `clip_vision_start_image` | CLIP_VISION_OUTPUT | خیر | - | ویژگی‌های بینایی CLIP استخراج شده از تصویر شروع |
| `clip_vision_end_image` | CLIP_VISION_OUTPUT | خیر | - | ویژگی‌های بینایی CLIP استخراج شده از تصویر پایان |
| `start_image` | IMAGE | خیر | - | تصویر فریم شروع برای توالی ویدئو |
| `end_image` | IMAGE | خیر | - | تصویر فریم پایان برای توالی ویدئو |

**توجه:** هنگامی که هر دو `start_image` و `end_image` ارائه شوند، گره یک توالی ویدئویی ایجاد می‌کند که بین این دو فریم انتقال می‌یابد. پارامترهای `clip_vision_start_image` و `clip_vision_end_image` اختیاری هستند، اما در صورت ارائه، ویژگی‌های بینایی CLIP آن‌ها به هم متصل شده و بر روی هر دو شرایط‌گذاری مثبت و منفی اعمال می‌شوند. `start_image` به اولین `length` فریم و `end_image` به آخرین `length` فریم قبل از پردازش برش داده می‌شود.

### **خروجی‌ها**

| نام خروجی | نوع داده | توضیحات |
|-------------|-----------|-------------|
| `positive` | CONDITIONING | شرایط‌گذاری مثبت با رمزگذاری فریم ویدئو و ویژگی‌های بینایی CLIP اعمال شده |
| `negative` | CONDITIONING | شرایط‌گذاری منفی با رمزگذاری فریم ویدئو و ویژگی‌های بینایی CLIP اعمال شده |
| `latent` | LATENT | تنسور نهفته خالی با ابعاد مطابق با پارامترهای ویدئوی مشخص شده |
