این گره با استفاده از سرویس هوش مصنوعی Bria، پس‌زمینه ویدیو را حذف می‌کند. ویدیوی ورودی را پردازش کرده و پس‌زمینه اصلی را با یک رنگ ثابت دلخواه شما جایگزین می‌کند. این عملیات از طریق یک API خارجی انجام می‌شود و نتیجه به‌صورت یک فایل ویدیویی جدید بازگردانده می‌شود.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `video` | VIDEO | بله | نامعتبر | فایل ویدیویی ورودی که پس‌زمینه آن حذف خواهد شد. |
| `background_color` | STRING | بله | `"Black"`<br>`"White"`<br>`"Gray"`<br>`"Red"`<br>`"Green"`<br>`"Blue"`<br>`"Yellow"`<br>`"Cyan"`<br>`"Magenta"`<br>`"Orange"` | رنگ ثابتی که به‌عنوان پس‌زمینه جدید برای ویدیوی خروجی استفاده می‌شود. |
| `seed` | INT | خیر | 0 تا 2147483647 | مقدار دانه‌ای (seed) که تعیین می‌کند آیا گره باید دوباره اجرا شود. نتایج بدون توجه به مقدار seed غیرقطعی هستند. (پیش‌فرض: 0) |

**توجه:** ویدیوی ورودی باید مدت‌زمان ۶۰ ثانیه یا کمتر داشته باشد.

## خروجی‌ها

| نام خروجی | نوع داده | توضیحات |
|-------------|-----------|-------------|
| `output` | VIDEO | فایل ویدیویی پردازش‌شده که پس‌زمینه آن حذف و با رنگ انتخاب‌شده جایگزین شده است. |
