Этот узел удаляет фон из видео с помощью сервиса Bria AI. Он обрабатывает входное видео и заменяет исходный фон на сплошной цвет по вашему выбору. Операция выполняется через внешний API, а результат возвращается в виде нового видеофайла.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `video` | VIDEO | Да | Н/Д | Входной видеофайл, из которого будет удалён фон. |
| `background_color` | STRING | Да | `"Black"`<br>`"White"`<br>`"Gray"`<br>`"Red"`<br>`"Green"`<br>`"Blue"`<br>`"Yellow"`<br>`"Cyan"`<br>`"Magenta"`<br>`"Orange"` | Сплошной цвет, который будет использоваться в качестве нового фона выходного видео. |
| `seed` | INT | Нет | от 0 до 2147483647 | Значение зерна, которое определяет, следует ли перезапускать узел. Результаты не являются детерминированными независимо от значения зерна. (по умолчанию: 0) |

**Примечание:** Длительность входного видео не должна превышать 60 секунд.

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `output` | VIDEO | Обработанный видеофайл с удалённым фоном, заменённым на выбранный цвет. |
