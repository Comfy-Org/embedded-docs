Вот перевод документации на русский язык:

Этот узел сохраняет подготовленный обучающий набор данных на жесткий диск вашего компьютера. Он принимает закодированные данные, которые включают скрытые представления изображений (latents) и соответствующие им текстовые условия (conditioning), и организует их в несколько меньших файлов, называемых шардами (shards), для упрощения управления. Узел автоматически создает папку в вашем выходном каталоге и сохраняет как файлы данных, так и файл метаданных, описывающий набор данных.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `latents` | LATENT | Да | Н/Д | Список словарей скрытых представлений из узла MakeTrainingDataset. |
| `conditioning` | CONDITIONING | Да | Н/Д | Список списков условий из узла MakeTrainingDataset. |
| `folder_name` | STRING | Нет | Н/Д | Имя папки для сохранения набора данных (внутри выходного каталога). (по умолчанию: "training_dataset") |
| `shard_size` | INT | Нет | от 1 до 100000 | Количество образцов в одном файле шарда. (по умолчанию: 1000) |

**Примечание:** Количество элементов в списке `latents` должно точно совпадать с количеством элементов в списке `conditioning`. Узел выдаст ошибку, если эти количества не совпадают.

## Выходные параметры

Этот узел не создает никаких выходных данных. Его функция заключается в сохранении файлов на ваш диск.
