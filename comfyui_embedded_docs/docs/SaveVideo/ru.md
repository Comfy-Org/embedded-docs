Узел SaveVideo сохраняет входное видео в директорию вывода ComfyUI. Он позволяет указать префикс имени файла, формат видео и кодек для сохранения. Узел автоматически обрабатывает именование файлов с инкрементальным счётчиком и может включать метаданные рабочего процесса в сохранённое видео.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `video` | VIDEO | Да | - | Видео для сохранения. |
| `filename_prefix` | STRING | Нет | - | Префикс для сохраняемого файла. Может включать информацию о форматировании, такую как %date:yyyy-MM-dd% или %Empty Latent Image.width% для включения значений из узлов (по умолчанию: "video/ComfyUI"). |
| `format` | COMBO | Нет | Доступно несколько вариантов | Формат для сохранения видео (по умолчанию: "auto"). |
| `codec` | COMBO | Нет | Доступно несколько вариантов | Кодек для использования в видео (по умолчанию: "auto"). |

## Выходные данные

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| *Нет выходов* | - | Этот узел не возвращает никаких выходных данных. |
