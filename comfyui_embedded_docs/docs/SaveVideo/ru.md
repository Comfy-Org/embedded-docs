Вот перевод документации узла SaveVideo на русский язык:

Узел SaveVideo сохраняет входное видео в ваш выходной каталог ComfyUI. Он позволяет указать префикс имени файла, формат видео и кодек для сохраняемого файла. Узел автоматически обрабатывает именование файлов с увеличением счетчика и может включать метаданные рабочего процесса в сохраненное видео.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `video` | VIDEO | Да | - | Видео для сохранения. |
| `filename_prefix` | STRING | Нет | - | Префикс для имени сохраняемого файла. Может включать информацию форматирования, например %date:yyyy-MM-dd% или %Empty Latent Image.width%, для подстановки значений из узлов (по умолчанию: "video/ComfyUI"). |
| `format` | COMBO | Нет | `"auto"`<br>`"mp4"`<br>`"webm"`<br>`"mkv"`<br>`"gif"` | Формат для сохранения видео (по умолчанию: "auto"). |
| `codec` | COMBO | Нет | `"auto"`<br>`"h264"`<br>`"h265"`<br>`"vp9"`<br>`"av1"`<br>`"prores"` | Кодек для использования в видео (по умолчанию: "auto"). |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| *Нет выходов* | - | Этот узел не возвращает никаких выходных данных. |
