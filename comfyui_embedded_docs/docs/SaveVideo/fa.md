گره SaveVideo محتوای ویدیوی ورودی را در دایرکتوری خروجی ComfyUI شما ذخیره می‌کند. این گره به شما امکان می‌دهد پیشوند نام فایل، فرمت ویدیو و کدک فایل ذخیره‌شده را مشخص کنید. این گره به‌طور خودکار نام‌گذاری فایل را با افزایش شمارنده مدیریت می‌کند و می‌تواند فراداده‌های گردش کار را در ویدیوی ذخیره‌شده بگنجاند.

## ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|---------|----------|--------|--------|---------|
| `video` | VIDEO | بله | - | ویدیویی که باید ذخیره شود. |
| `filename_prefix` | STRING | خیر | - | پیشوند فایل برای ذخیره‌سازی. ممکن است شامل اطلاعات قالب‌بندی مانند `%date:yyyy-MM-dd%` یا `%Empty Latent Image.width%` برای درج مقادیر گره‌ها باشد (پیش‌فرض: `"video/ComfyUI"`). |
| `format` | COMBO | خیر | `"auto"`<br>`"mp4"`<br>`"webm"`<br>`"mkv"`<br>`"gif"` | فرمت ذخیره‌سازی ویدیو (پیش‌فرض: `"auto"`). |
| `codec` | COMBO | خیر | `"auto"`<br>`"h264"`<br>`"h265"`<br>`"vp9"`<br>`"av1"`<br>`"prores"` | کدک مورد استفاده برای ویدیو (پیش‌فرض: `"auto"`). |
