# ترجمة وثيقة عقد SaveVideo

عقد **SaveVideo** يقوم بحفظ محتوى الفيديو المدخل إلى دليل الإخراج الخاص بـ ComfyUI. يتيح لك تحديد بادئة اسم الملف، تنسيق الفيديو، وبرنامج الترميز للملف المحفوظ. يقوم العقد تلقائيًا بإدارة تسمية الملفات مع زيادة العداد ويمكنه تضمين بيانات سير العمل الوصفية في الفيديو المحفوظ.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|-----------|-----------|----------|-------|-------------|
| `video` | VIDEO | نعم | - | الفيديو المراد حفظه. |
| `filename_prefix` | STRING | لا | - | بادئة اسم الملف المراد حفظه. قد تتضمن معلومات تنسيق مثل %date:yyyy-MM-dd% أو %Empty Latent Image.width% لتضمين قيم من العقد (الافتراضي: "video/ComfyUI"). |
| `format` | COMBO | لا | `"auto"`<br>`"mp4"`<br>`"webm"`<br>`"mkv"`<br>`"gif"` | تنسيق حفظ الفيديو (الافتراضي: "auto"). |
| `codec` | COMBO | لا | `"auto"`<br>`"h264"`<br>`"h265"`<br>`"vp9"`<br>`"av1"`<br>`"prores"` | برنامج الترميز المستخدم للفيديو (الافتراضي: "auto"). |

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|-------------|-----------|-------------|
| *لا يوجد مخرجات* | - | لا يقوم هذا العقد بإرجاع أي بيانات إخراج. |
