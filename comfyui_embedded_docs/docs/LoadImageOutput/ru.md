Вот перевод документации на русский язык:

Узел LoadImageOutput загружает изображения из папки output. При нажатии кнопки обновления он обновляет список доступных изображений и автоматически выбирает первое, что упрощает последовательный просмотр сгенерированных изображений.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `image` | COMBO | Да | Доступно несколько вариантов | Загружает изображение из папки output. Включает опцию загрузки и кнопку обновления для обновления списка изображений. При нажатии кнопки обновления узел обновляет список изображений и автоматически выбирает первое, что позволяет легко выполнять последовательный просмотр. |

## Выходы

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `image` | IMAGE | Загруженное изображение из папки output |
| `mask` | MASK | Маска, связанная с загруженным изображением |
