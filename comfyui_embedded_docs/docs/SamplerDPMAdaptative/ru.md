Узел SamplerDPMAdaptative реализует адаптивный сэмплер DPM (диффузионная вероятностная модель), который автоматически регулирует размер шагов в процессе сэмплирования. Он использует контроль ошибок на основе допусков для определения оптимального размера шагов, балансируя вычислительную эффективность и точность сэмплирования. Такой адаптивный подход помогает поддерживать качество, потенциально сокращая количество необходимых шагов.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `order` | INT | Да | 2-3 | Порядок метода сэмплера (по умолчанию: 3) |
| `rtol` | FLOAT | Да | 0.0-100.0 | Относительный допуск для контроля ошибок (по умолчанию: 0.05) |
| `atol` | FLOAT | Да | 0.0-100.0 | Абсолютный допуск для контроля ошибок (по умолчанию: 0.0078) |
| `h_init` | FLOAT | Да | 0.0-100.0 | Начальный размер шага (по умолчанию: 0.05) |
| `pcoeff` | FLOAT | Да | 0.0-100.0 | Пропорциональный коэффициент для управления размером шага (по умолчанию: 0.0) |
| `icoeff` | FLOAT | Да | 0.0-100.0 | Интегральный коэффициент для управления размером шага (по умолчанию: 1.0) |
| `dcoeff` | FLOAT | Да | 0.0-100.0 | Дифференциальный коэффициент для управления размером шага (по умолчанию: 0.0) |
| `accept_safety` | FLOAT | Да | 0.0-100.0 | Коэффициент безопасности для принятия шага (по умолчанию: 0.81) |
| `eta` | FLOAT | Да | 0.0-100.0 | Параметр стохастичности (по умолчанию: 0.0) |
| `s_noise` | FLOAT | Да | 0.0-100.0 | Коэффициент масштабирования шума (по умолчанию: 1.0) |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `sampler` | SAMPLER | Возвращает настроенный экземпляр адаптивного DPM сэмплера |
