Вот перевод документации узла GetImageSize на русский язык:

Узел GetImageSize извлекает размеры и информацию о пакете из входного изображения. Он возвращает ширину, высоту и размер пакета изображения, а также отображает эту информацию в виде текста прогресса на интерфейсе узла. Исходные данные изображения передаются без изменений.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `image` | IMAGE | Да | - | Входное изображение, из которого извлекается информация о размере |

## Выходы

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `width` | INT | Ширина входного изображения в пикселях |
| `height` | INT | Высота входного изображения в пикселях |
| `batch_size` | INT | Количество изображений в пакете |
