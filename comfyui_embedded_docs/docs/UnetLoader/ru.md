Узел `UNETLoader` предназначен для загрузки U-Net моделей по имени, упрощая использование предварительно обученных архитектур U-Net в системе.

Этот узел обнаружит модели, расположенные в папке `ComfyUI/models/diffusion_models`.

## Входные параметры

| Параметр      | Тип данных    | Описание |
|---------------|---------------|----------|
| `unet_name`   | COMBO[STRING] | Указывает имя загружаемой U-Net модели. Это имя используется для поиска модели в заданной структуре каталогов, что позволяет динамически загружать различные U-Net модели. |
| `weight_dtype`| ...           | 🚧  fp8_e4m3fn fp9_e5m2  |

## Выходные параметры

| Параметр | Тип данных | Описание |
|----------|------------|----------|
| `model`  | MODEL      | Возвращает загруженную U-Net модель, позволяя использовать её для дальнейшей обработки или логического вывода в системе. |
