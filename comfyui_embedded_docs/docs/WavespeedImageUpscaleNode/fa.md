### نمای کلی

گره افزایش وضوح تصویر WaveSpeed از یک سرویس هوش مصنوعی خارجی برای افزایش رزولوشن و کیفیت تصویر استفاده می‌کند. این گره یک عکس ورودی واحد دریافت کرده و آن را به رزولوشن هدف بالاتری مانند 2K، 4K یا 8K افزایش می‌دهد و نتیجه‌ای شارپ‌تر و با جزئیات بیشتر تولید می‌کند.

### ورودی‌ها

| پارامتر | نوع داده | الزامی | محدوده | توضیحات |
|-----------|-----------|----------|-------|-------------|
| `مدل` | STRING | بله | `"SeedVR2"`<br>`"Ultimate"` | مدل هوش مصنوعی مورد استفاده برای افزایش وضوح. گزینه‌های "SeedVR2" و "Ultimate" سطوح مختلف کیفیت و قیمت‌گذاری را ارائه می‌دهند. |
| `تصویر` | IMAGE | بله | | تصویر ورودی که باید وضوح آن افزایش یابد. |
| `وضوح هدف` | STRING | بله | `"2K"`<br>`"4K"<br>`"8K"` | رزولوشن خروجی مورد نظر برای تصویر افزایش‌یافته. |

**توجه:** این گره دقیقاً به یک تصویر ورودی نیاز دارد. ارائه یک دسته از تصاویر منجر به خطا خواهد شد.
