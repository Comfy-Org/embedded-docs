Узел WaveSpeed Image Upscale использует внешний AI-сервис для повышения разрешения и качества изображения. Он принимает одно входное изображение и увеличивает его до более высокого целевого разрешения, например 2K, 4K или 8K, создавая более чёткий и детализированный результат.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `model` | STRING | Да | `"SeedVR2"`<br>`"Ultimate"` | Модель AI для увеличения разрешения. "SeedVR2" и "Ultimate" предлагают разные уровни качества и ценообразования. |
| `image` | IMAGE | Да | | Входное изображение для увеличения разрешения. |
| `target_resolution` | STRING | Да | `"2K"`<br>`"4K"`<br>`"8K"` | Желаемое выходное разрешение для увеличенного изображения. |

**Примечание:** Этот узел требует ровно одно входное изображение. Передача пакета изображений приведёт к ошибке.

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `image` | IMAGE | Увеличенное выходное изображение с высоким разрешением. |
