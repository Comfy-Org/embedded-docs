Этот узел предназначен для изменения поведения модели при семплировании путем применения дискретной стратегии семплирования. Он позволяет выбирать различные методы семплирования, такие как epsilon, v_prediction, lcm или x0, и опционально корректирует стратегию шумоподавления модели на основе настройки коэффициента шума при нулевой выборке (zsnr).

## Входы

| Параметр | Тип данных | Python dtype     | Описание |
|-----------|--------------|-------------------|-------------|
| `model`   | MODEL     | `torch.nn.Module` | Модель, к которой будет применена дискретная стратегия семплирования. Этот параметр имеет решающее значение, поскольку определяет базовую модель, которая будет модифицирована. |
| `sampling`| COMBO[STRING] | `str`           | Определяет метод дискретного семплирования, применяемый к модели. Выбор метода влияет на то, как модель генерирует образцы, предлагая различные стратегии семплирования. |
| `zsnr`    | `BOOLEAN`   | `bool`           | Логический флаг, который при включении корректирует стратегию шумоподавления модели на основе коэффициента шума при нулевой выборке. Это может повлиять на качество и характеристики генерируемых образцов. |

## Выходы

| Параметр | Тип данных | Python dtype     | Описание |
|-----------|-------------|-------------------|-------------|
| `model`   | MODEL     | `torch.nn.Module` | Модифицированная модель с примененной дискретной стратегией семплирования. Теперь эта модель способна генерировать образцы с использованием указанного метода и настроек. |
