Узел KarrasScheduler предназначен для генерации последовательности уровней шума (сигм) на основе расписания шума, предложенного Каррасом и соавт. (2022). Этот планировщик полезен для управления процессом диффузии в генеративных моделях, позволяя выполнять точную настройку уровней шума, применяемых на каждом этапе генерации.

## Входные параметры

| Параметр    | Тип данных | Описание                                                                                      |
|-------------|-------------|------------------------------------------------------------------------------------------------|
| `шаги`     | INT         | Определяет количество шагов в расписании шума, влияя на детализацию генерируемой последовательности сигм. |
| `максимальное_сигма` | FLOAT       | Максимальное значение сигмы в расписании шума, задающее верхнюю границу уровней шума.                    |
| `минимальное_сигма` | FLOAT       | Минимальное значение сигмы в расписании шума, задающее нижнюю границу уровней шума.                    |
| `ро`       | FLOAT       | Параметр, управляющий формой кривой расписания шума, влияющий на то, как уровни шума изменяются от sigma_min до sigma_max. |

## Выходные параметры

| Параметр  | Тип данных | Описание                                                                 |
|-----------|-------------|-----------------------------------------------------------------------------|
| `sigmas`  | SIGMAS      | Сгенерированная последовательность уровней шума (сигм) в соответствии с расписанием шума Карраса и соавт. (2022). |
