## Обзор

Узел Not выполняет логическую операцию НЕ (NOT) над любым входным значением. Он возвращает True, если входное значение считается ложным (например, 0, пустая строка, None, False), и возвращает False, если входное значение является истинным. Для определения истинности используются стандартные правила Python.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `value` | ANY | Да | Любое значение | Входное значение для инвертирования. Принимается любой тип данных, оценка выполняется по правилам истинности Python. |

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `output` | BOOLEAN | Логическая инверсия входного значения. Возвращает True, если входное значение ложно, и False, если входное значение истинно. |
