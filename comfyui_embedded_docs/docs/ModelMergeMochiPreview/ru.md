Этот узел объединяет две модели ИИ на основе блоков с точным контролем над различными компонентами модели. Он позволяет смешивать модели, регулируя веса интерполяции для отдельных участков, включая позиционные частоты, слои внедрения и отдельные блоки трансформера. Процесс объединения комбинирует архитектуры и параметры обеих входных моделей в соответствии с заданными значениями весов.
