# ترجمة وثيقة عقدة ComfyUI

هذه العقدة تقوم بخلط قائمة من الصور وقائمة من النصوص معًا، مع الحفاظ على اقتراناتها الأصلية. تستخدم بذرة عشوائية لتحديد ترتيب الخلط، مما يضمن خلط نفس قوائم الإدخال بنفس الطريقة في كل مرة يتم فيها إعادة استخدام البذرة.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|---------|---------------|--------|---------|-------|
| `images` | IMAGE | نعم | - | قائمة الصور المراد خلطها. |
| `texts` | STRING | نعم | - | قائمة النصوص المراد خلطها. |
| `seed` | INT | لا | 0 إلى 18446744073709551615 | البذرة العشوائية. يتم تحديد ترتيب الخلط بواسطة هذه القيمة (الافتراضي: 0). |

**ملاحظة:** يجب أن تكون مدخلات `images` و `texts` قوائم بنفس الطول. ستقوم العقدة بإقران الصورة الأولى مع النص الأول، والصورة الثانية مع النص الثاني، وهكذا، قبل خلط هذه الأزواج معًا.

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|------------|---------------|-------|
| `texts` | IMAGE | قائمة الصور بعد الخلط. |
| `texts` | STRING | قائمة النصوص بعد الخلط، مع الحفاظ على اقتراناتها الأصلية مع الصور. |
