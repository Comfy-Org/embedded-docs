Этот узел перемешивает список изображений и список текстов вместе, сохраняя их парные связи. Для определения порядка перемешивания используется случайное начальное число (seed), что гарантирует одинаковое перемешивание одних и тех же входных списков при повторном использовании того же seed.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `images` | IMAGE | Да | - | Список изображений для перемешивания. |
| `texts` | STRING | Да | - | Список текстов для перемешивания. |
| `seed` | INT | Нет | от 0 до 18446744073709551615 | Случайное начальное число. Порядок перемешивания определяется этим значением (по умолчанию: 0). |

**Примечание:** Входные данные `images` и `texts` должны быть списками одинаковой длины. Узел сопоставит первое изображение с первым текстом, второе изображение со вторым текстом и так далее, после чего перемешает эти пары вместе.

## Выходные данные

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `images` | IMAGE | Перемешанный список изображений. |
| `texts` | STRING | Перемешанный список текстов с сохранением исходных парных связей с изображениями. |
