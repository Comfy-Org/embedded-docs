Узел ModelMergeSimple предназначен для слияния двух моделей путем смешивания их параметров на основе заданного коэффициента. Этот узел упрощает создание гибридных моделей, объединяющих сильные стороны или характеристики обеих входных моделей.

Параметр `ratio` определяет коэффициент смешивания между двумя моделями. Когда это значение равно 1, выходная модель на 100% состоит из `model1`, а когда это значение равно 0, выходная модель на 100% состоит из `model2`.

## Входные параметры

| Параметр | Тип данных | Описание |
|-----------|-------------|-------------|
| `модель1`  | `MODEL`     | Первая модель для слияния. Служит базовой моделью, на которую накладываются патчи из второй модели. |
| `модель2`  | `MODEL`     | Вторая модель, чьи патчи накладываются на первую модель с учетом заданного коэффициента. |
| `соотношение`   | `FLOAT`     | Когда это значение равно 1, выходная модель на 100% состоит из `модель1`, а когда это значение равно 0, выходная модель на 100% состоит из `модель2`. |

## Выходные параметры

| Параметр | Тип данных | Описание |
|-----------|-------------|-------------|
| `model`   | MODEL     | Результирующая объединенная модель, включающая элементы обеих входных моделей в соответствии с заданным коэффициентом. |
