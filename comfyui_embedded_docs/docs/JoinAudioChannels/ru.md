Узел "Объединить аудиоканалы" объединяет два отдельных монофонических аудиовхода в один стереофонический аудиовыход. Он принимает левый и правый каналы, обеспечивает совместимость их частот дискретизации и длины, после чего объединяет их в двухканальную аудиоволновую форму.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|----------|------------|--------------|----------|----------|
| `audio_left` | AUDIO | Да | | Монофонические аудиоданные, которые будут использоваться в качестве левого канала результирующего стереофонического аудио. |
| `audio_right` | AUDIO | Да | | Монофонические аудиоданные, которые будут использоваться в качестве правого канала результирующего стереофонического аудио. |

**Примечание:** Оба входных аудиопотока должны быть монофоническими (одноканальными). Если они имеют разную частоту дискретизации, канал с более низкой частотой будет автоматически передискретизирован до более высокой. Если аудиопотоки имеют разную длину, они будут обрезаны до длины более короткого.

## Выходы

| Имя выхода | Тип данных | Описание |
|------------|------------|----------|
| `audio` | AUDIO | Результирующее стереофоническое аудио, содержащее объединённые левый и правый каналы. |
